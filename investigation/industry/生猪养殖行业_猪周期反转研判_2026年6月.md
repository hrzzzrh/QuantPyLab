# 生猪养殖行业深度研究：猪周期反转研判

> **研究日期**: 2026年6月7日
> **核心问题**: 猪周期当前处于什么阶段？何时能够反转？

---

## Executive Summary

- **核心判断**: 当前猪周期正处于"价格底+政策底"已现但"市场底"未至的磨底阶段。能繁母猪去化偏慢叠加养殖效率持续提升，本轮周期底部将显著长于历史。**8月前后可能出现阶段性拐点（季节性+去化共振），但真正周期级别的反转大概率在2027年Q1左右**，高度取决于2026年下半年的产能去化力度。
- **关键数据**: 2026年Q1能繁母猪存栏3904万头（高于新目标3750万头约4%）；猪价4月触底8.6元/kg刷新十年新低；19家上市猪企Q1合计亏损约76亿元；猪粮比4.1:1处于过度下跌一级预警。
- **核心矛盾**: "能繁降但供给不降"——PSY从2021年的15.5头飙升至2025年的23.93头，生产效率提升完全对冲了产能去化。
- **反转信号**: 淘汰母猪/生猪价比率需跌破0.65、能繁母猪累计去化需达到约10%以上、出栏均重需明显回落。
- **投资窗口**: 当前已是布局养殖板块的左侧窗口期（历史规律：全行业深度亏损+产能加速去化期是最佳买点），但需对底部持续时间保持耐心。

## 1. Background & Context

### 1.1 猪周期的运行逻辑

猪周期的本质是**产能周期的时间错配**：猪价上涨→养殖盈利→补栏能繁母猪（产能扩张）→10个月后生猪集中出栏→供大于求→猪价下跌→跌破成本→淘汰母猪（产能收缩）→供给不足→猪价再度回升。

中国猪周期历史上约3-4年完成一轮，但2022年开启的本轮周期出现两个显著变化：
1. **形态异化**：不再呈现单一峰谷，而是在2022年10月和2024年9月分别出现两个价格高点，形成"M型"走势
2. **底部拉长**：低猪价持续时间和亏损深度均超过历史常态

### 1.2 当前周期溯源

| 时间节点 | 事件 | 能繁母猪存量 |
|----------|------|-------------|
| 2021年 | 非洲猪瘟后产能高速扩张，制度调控元年（目标4100万头） | ~4300万头 |
| 2023年7月-2024年4月 | 深亏倒逼去化，能繁累计下降6.67% | 从4300→4000万头 |
| 2024年2月 | 调控目标下调至3900万头 | - |
| 2024年10月 | 新一轮猪周期启动 | - |
| 2025年8月 | 能繁存栏近期高点4038万头 | 4038万头 |
| 2025年7月起 | 产能去化阶段开启 | 逐月下降 |
| 2026年Q1 | 猪价跌破9元/kg创十年新低 | 3904万头 |
| 2026年5月14日 | 调控目标再下调至3750万头 | - |

## 2. Deep Dive Analysis

### 2.1 供给端：去化慢、效率高、体重大的三重压制

#### 2.1.1 能繁母猪去化进度严重滞后

截至2026年Q1末，全国能繁母猪存栏3904万头，虽然环比下降1.43%（减57万头），但距离新的3750万头正常保有量目标**仍需去化约150万头**。Q1仅完成调减任务的19%，进度偏慢。

按旧标准3900万头，当前为目标量的100.1%（绿色区间）；但按新标准3750万头，实为目标量的**104%**，已踏入黄色预警区间。

#### 2.1.2 "存栏降但供给增"的核心悖论

这是本轮周期最核心的结构性问题：

| 指标 | 2021年 | 2025年 | 增幅 |
|------|--------|--------|------|
| MSY（年提供出栏肥猪数） | 15.5头 | 23.93头 | **+54.4%** |
| PSY（年提供断奶仔猪数） | 19.9头 | 23.4头 | +17.6% |
| 能繁母猪存栏 | ~4300万头 | ~4000万头 | -7.0% |
| 年度生猪出栏 | 6.71亿头 | 7.20亿头 | **+7.2%** |

**能繁母猪数量下降了7%，但总出栏量反而增长了7.2%**——生产效率飞跃完全对冲了产能去化。

根据Mysteel测算，2026年行业PSY继续提升不超过0.5头，全程成活率同比提升1-1.62个百分点，全年MSY将维持18-18.5头区间。即便能繁母猪继续小幅下降，最终出栏量仍可能持平甚至增长。

#### 2.1.3 出栏均重居高不下

截至2026年5月底，生猪出栏均重123.21kg，为近三年同期最高水平。高体重进一步放大了实际猪肉供给量——即便出栏头数不变，出肉量也在增加。

前期压栏和二育猪源尚未完全出清，预计5月下旬至6月集中释放，继续增厚阶段性供给压力。

#### 2.1.4 供给压力何时实质性缓解？

按10个月繁育传导时滞推算：
- **2025年8月**能繁母猪4038万头（近期高点）→ 对应**2026年6月**出栏高峰
- **2025年9月起**能繁母猪趋势性下降 → 对应**2026年7月起**出栏量边际减少
- 8月对应2025年10月母猪3990万头，去化加速 → **8月标猪供应能力或大幅减少**

中邮证券预估：2026年Q3-Q4生猪供应环比减少，但供大于求格局难以逆转。2027年Q1产能去化或到位。

### 2.2 需求端：结构性与周期性双重疲软

- 2025年人均家庭猪肉消费26.6kg，同比降5.4%，**连续第二年下滑**。牛肉、禽肉、水产对猪肉的替代效应持续加大。
- 猪肉占中国肉类消费总量的比重持续下降。
- 2026年6月为传统淡季，高温天气抑制鲜肉消费，屠宰企业开工率低迷。
- 冻品库存率29.79%，处于近年高位，挤占鲜肉消费空间。
- 下半年（Q3-Q4）消费有望季节性回暖，叠加产能去化传导，为猪价提供一定底部支撑。

USDA预测2026年中国猪肉消费需求仍偏弱，猪肉进口量将继续下降。

### 2.3 成本与利润：全行业深度亏损持续倒逼去化

| 指标 | 数值（截至2026年5月底） |
|------|------------------------|
| 生猪均价 | 约9.5-10.14元/kg |
| 自繁自养利润 | **-264.08元/头** |
| 外购仔猪利润 | -213.6元/头 |
| 猪粮比 | **4.1:1**（过度下跌一级预警） |
| 头部企业完全成本 | 11.6-12元/kg |
| 自繁自养5个月出栏成本 | 12.01元/kg |

猪价已跌破十年新低，**即使是成本控制最优的温氏（11.6元/kg），每公斤生猪刚性亏损仍达1.77元**。行业整体每头亏损超400元，远超中小养殖户承受极限。

但需要注意的是：**深度亏损本身是去产能的催化剂，而非反转的充分条件**。历史上，只有亏损持续至现金流断裂，才会触发大范围产能出清。

### 2.4 产业格局：规模化再造猪周期特征

#### 2.4.1 规模化率已达70%，去化意愿分化

非洲猪瘟后，大型集团借助资本力量逆周期扩张，中小散户加速退出，行业规模化率飙升至约70%。这带来了三个关键变化：

1. **去化抗性强**：集团企业资金池更深、融资渠道更多元。牧原Q1虽亏12亿但账上仍有142.7亿现金，负债率降至50.73%
2. **不会集中出清**：大企业即便亏损也会维持产能以等待竞争对手先倒下，导致去化呈"温水煮蛙"式缓慢递减
3. **产业话语权集中**：本次产能调控方案首次将10万头以上大型集团企业纳入监测名单，头部企业去化意愿直接决定全行业去化节奏

#### 2.4.2 上市猪企Q1全景：越卖越亏、负债飙升

| 企业 | Q1归母净利润 | 资产负债率(Q1末) | 关键动态 |
|------|-------------|-------------------|---------|
| 牧原股份 | -12.15亿 | 50.73%↓ | 已去化母猪13.6%，现金142.7亿 |
| 温氏股份 | -10.70亿 | 53.14%↑ | 短期借款激增40倍储备现金 |
| 新希望 | -8.98亿 | 70.85% | 短债缺口明显 |
| 神农集团 | -6.48亿 | - | - |
| 天邦食品 | 大额亏损 | >80% | 负债率触及红线 |
| 新五丰 | 大额亏损 | >80% | 负债率触及红线 |
| **19家合计** | **~76亿** | 9家超60% | 仅牧原负债率下降 |

经营性现金流普遍骤降超90%，部分企业已开始计提资产减值、淘汰高成本产能。**现金流紧张正倒逼企业加速淘汰低效产能**，但距离全面出清仍有距离。

#### 2.4.3 淘汰母猪价格比率：产能尚未进入深度出清

淘汰母猪价格/商品猪价格比值已达76.8%（历史数据显示>75%为扛亏损/扩产区间），表明**行业尚未出现恐慌性淘汰**。历史上该比率需跌破65%并持续一段时间，才意味着产能出清彻底完成、反转可期。

### 2.5 政策信号：史上最强产能调控已落地

2026年5月14日发布的《生猪产能综合调控实施方案（2026年修订）》是**生猪调控史上力度最大的一轮政策**：

| 调控维度 | 旧方案（2024年修订） | 新方案（2026年修订） |
|----------|---------------------|---------------------|
| 能繁母猪保有量 | 3900万头 | **3750万头** |
| 绿色区间 | 92%-105% | **92%-103%（收窄）** |
| 黄色/红色区间 | 同步放宽 | **同步压缩（更早触发干预）** |
| 大型企业监管 | 无强制 | **10万头以上企业纳入监测名单** |
| 核心逻辑 | 防大起大落 | **供需适配+逆周期调节** |

4月政治局会议**首次将"稳定生猪价格"纳入宏观稳民生重点**，信号意义极强。

但历史经验表明：**政策底≠市场底**。2021年9月首次推出产能调控后，市场仍磨底4-5个月至2022年3月才见底；2024年2月下调目标后，也需要2-3个月才确认市场底。预计本轮政策底到市场底的传导期约**3-4个月**，即2026年Q3-Q4。

### 2.6 历史周期参照：本轮底部将持续更长

| 周期特征 | 前两轮去化 | 本轮去化 |
|----------|-----------|----------|
| 累计去化幅度 | 39.40%（首轮）/ 6.67%（二轮） | **仅3.41%**（截至2026年3月） |
| 去化速度 | 快速集中 | **温和缓慢** |
| 底部持续时间 | 4-6个月 | 预计**6-12个月** |
| 规模化程度 | 较低 | **已至70%** |

本轮去化在幅度、速度和节奏上均显著慢于前两轮，卓创资讯判断本轮底部磨底时间将"相对更长"，需要耐心等待。

## 3. 周期反转研判

### 3.1 综合时间线

```
2026年 H1
  ┌─ Q1: 猪价暴跌至十年新低(8.6元/kg)，全行业深亏76亿
  ├─ Q2: 5月猪价企稳(9.5-10元/kg)，政策底确立
  │       6月端午备货+出栏量边际下降，有望确认阶段性底部
  │       但仍受出栏均重高、冻品库存高、消费淡季三重压制
  │
2026年 H2
  ├─ Q3: ★ 供给压力趋势性缓解（去化效果开始传导至出栏端）
  │       8月可能迎来季节性拐点与产能去化共振
  │       猪价大概率"底部企稳、缓慢修复"，不会急涨
  │       产能去化持续推进（政策+市场双重驱动）
  ├─ Q4: 消费季节性回暖，猪价温和反弹
  │       但四季度出栏量仍大概率偏高（4月仔猪环比增加）
  │       上行空间有限，大概率维持12-14元/kg区间
  │
2027年
  └─ Q1: ★★ 大概率确认市场底
          产能去化积累到位 + 前期高位存栏惯性释放完毕
          若下半年去化超预期，可能提前至2026年Q4
          周期高度取决于2026年去化力度
```

### 3.2 反转的核心触发条件

以下信号齐备时即为反转确认点：

1. **能繁母猪降至3750万头以下**（当前还需去化约150万头，按当前月均去化20万头计算约需7-8个月，即2026年底至2027年初）
2. **淘汰母猪/生猪价比率跌破0.65**，进入深度去化区间
3. **生猪出栏均重从123kg明显回落**，前期压栏猪源充分出清
4. **中小散户出现集中退出潮**或个别高负债猪企出现债务违约事件
5. **猪价不再对利空敏感**，市场抛压衰竭

### 3.3 各机构观点汇总

| 机构 | 反转时点判断 | 核心逻辑 |
|------|-------------|---------|
| 中邮证券 | **2027年Q1** | 三四季度供应减少但难逆供需，需等产能去化到位 |
| 中信证券 | **2027年** | 2026产能逐步去化，2027年猪价景气或可期 |
| 开源证券 | 拐点条件积聚中 | 行业冰点已现，否极泰来可期 |
| 长城证券 | **2026年H2趋势性缓解** | 6月有望季节性拐点与去化效果共振 |
| 长江证券 | 产能加速去化期 | 当前是最佳布局窗口 |
| 中国养猪网 | **8月可能迎来拐点** | 2025年8月后母猪存栏去化，传导至8月标猪供应大减 |
| 广发期货 | 暂无强势反弹基础 | 政策底+价格底已现，需要时间底 |
| 格林大华 | 等待产能集中大量去化 | 当前去化温和，未出现怀孕母猪被淘汰 |
| USDA | 2026年猪肉生产稳定 | 更高屠宰量被更轻胴体重量抵消 |
| 王祖力(农科院) | H2可能改善但幅度有限 | 产能去化幅度小，消费弱 |

## 4. Risk Assessment

### 4.1 上行风险（反转提前/幅度更大）

- **疫病冲击**：非洲猪瘟/口蹄疫等大规模爆发导致被动性产能出清，加速反转
- **政策加码超预期**：中央/地方出台更强力去产能措施（如强制淘汰、环保关停）
- **消费超预期复苏**：宏观经济提振带动餐饮/家庭猪肉消费大幅回暖
- **中小散户集中退出**：持续亏损迫使高成本农户加速退场
- **饲料价格大涨**：玉米/豆粕价格飙升加速高成本产能退出

### 4.2 下行风险（反转延后/幅度更弱）

- **规模化企业持续扛亏**：大企业资金储备充足，去化持续缓慢
- **PSY继续提升**：生产效率进一步冲抵能繁母猪去化效果
- **消费持续低迷**：经济下行+替代蛋白渗透，猪肉消费量持续萎缩
- **冻品库存压制**：高库存持续分流鲜肉需求，弱化价格反弹动力
- **二育扰乱节奏**：二次育肥阶段性扰乱供需，拉长底部周期
- **去化不及预期**：若下半年去化仍然偏慢，反转可能推迟至2027年下半年

| 指标 | 当前值 | 反转阈值 | 监测频率 |

| 能繁母猪存栏 | 3904万头 | <3750万头 | 月度 |
| 淘汰母猪/生猪价比 | 76.8% | <65% | 周度 |
| 猪粮比 | 4.1:1 | >6:1 | 周度 |

## 5. Synthesis & Conclusion

### 5.1 核心结论

**猪周期反转不会在短期内发生，但已进入"黎明前的黑暗"阶段。**

1. **短期（2026年6-7月）**：供强需弱格局延续，猪价仍将在9-11元/kg区间弱势震荡磨底。端午节备货提振有限，高温淡季+高体重出栏+高冻品库存三重压力下，趋势性上涨行情难以出现。

2. **中期（2026年H2）**：8月前后可能迎来年内重要拐点——能繁母猪自2025年8月高点的去化效果开始传导至出栏端，叠加季节性消费回暖，猪价有望从底部缓慢抬升。但受制于产能去化不彻底，反弹空间有限，大概率在12-14元/kg区间。

3. **长期（2027年）**：若2026年下半年政策+市场双轮驱动去化取得实质性进展，**2027年Q1前后大概率确认本轮周期市场底并开启新一轮上行**。周期高度取决于2026年剩余时间的去化力度——去化越彻底，反转空间越大。

### 5.2 投资策略建议

> ⚠️ 免责声明：以下仅为基于公开信息的行业分析，不构成具体投资建议。

- **当前阶段（左侧布局期）**：历史规律表明，在全行业深度亏损+产能加速去化的阶段，是布局养殖板块的最佳时机。建议关注具备绝对成本优势和充裕现金流的龙头企业（如牧原股份、温氏股份），其在周期反转时业绩弹性最大。
- **等待确认信号**：当淘汰母猪/生猪价比率跌破65%+猪粮比持续回升+出栏均重明显下降时，可视为反转确认信号，适合增加配置。
- **规避高负债标的**：资产负债率超70%的企业存在因债务压力被迫出清甚至违约的风险，反转前可能先被淘汰。

### 5.3 关于"规模化改变周期特征"的长期判断

本轮猪周期标志着行业从"散户主导的大起大落"时代进入"规模企业主导的慢性去化"新阶段。**未来的猪周期波动幅度或趋于平坦、周期长度或趋于延长**，单纯依赖历史规律进行择时将面临更大挑战。核心竞争要素将从"赌周期"转向"拼成本"，具备持续降本能力的企业将成为穿越周期的最终赢家。

## 六、持续跟踪框架

本章分为两部分：**6.1-6.6 为具体跟踪事项**（围绕猪周期反转研判的核心变量组织），**6.7-6.11 为指导原则**（当框架之外的新信息出现时，如何发掘、评估并纳入研判）。

### 6.1 供给端：产能去化的进度条

供给端是决定反转时点的核心矛盾。跟踪逻辑：**能繁母猪存栏变化（10 个月后）→ 生猪出栏量变化 → 猪价方向**。

#### 6.1.1 能繁母猪——最核心的领先指标

| 具体指标 | 当前基准 | 数据来源 | 频率 | 为什么关注 |
|---------|----------|---------|------|---------|
| 全国能繁母猪存栏（万头）及环比变化 | 3,904 万头（Q1 官方）；5月样本环比 **-1.10%**（涌益）/ **-1.32%**（钢联），为去化开启以来最快 | 农业农村部/国家统计局；涌益/钢联 | 月度 | **反转的核心引擎**，领先出栏 10 个月。目标值 3,750 万头。样本数据显示产能去化在 5 月**显著加速** |
| 月均去化速度（万头/月） | Q1 约 20 万头/月；5月样本加速至约 **40-50 万头/月** | 自行计算 | 月度 | 若维持 5 月速度，3-4 个月可达标（2026 年 9-10 月） |

| 能繁母猪淘汰量（万头） | — | Mysteel、卓创资讯 | 月度 | 加速淘汰是去化提速的直接证据 |

**关键判据**：反转确认的第一前提是能繁母猪降至 3,750 万头以下。若月均去化速度从当前的 20 万头/月上升至 30-40 万头/月，反转时间表将提前至 2026 年 Q4；若降至 10 万头/月以下，反转将推迟至 2027 H2 甚至更晚。

> **【6月跟踪 2026.6.30】** ★★★ 核心信号：5月样本能繁去化加速至 -1.1%~-1.32%（月均 40-50 万头），较 Q1 的月均 20 万头速度**翻倍**。不过这是样本数据，最终以官方发布为准。样本加速需连续 3 个月确认（6-7 月）后触发研判修正。同时，4月样本去化 -0.70%（涌益），显示去化在逐月加速——4月 > Q1，5月 > 4月。

**重大数据修正风险**：历史上农业农村部的能繁母猪存栏数据会根据实际情况进行回溯性修正。若出现重大修正（如某月一次性上调/下调超 50 万头），需立即重新评估去化进度和反转时间表。

#### 6.1.2 生产效率——去化效果的"对冲因子"

> **【6月跟踪 2026.6.30】** 温氏披露 PSY 已从 26.3→28 头（+6.5%），规划每年提升 1 头以上、未来突破 32 头。**结构性逆风在加剧**：若全行业 PSY 从 23.4→25 头（+6.8%），要维持同等出栏量，能繁需从 3,904→3,650 万头（-6.5%）。即 3,750 万头目标可能仍不够，需降至约 3,650 万头才能实现真正的供给收缩。此为当前研判中最大的上行风险（对反转延后而言）。

### 6.7 框架之外：如何发掘新信息

猪周期的分析框架相对成熟（去化→出栏→价格），但以下信息源可能提供框架之外的洞察：

1. **定期扫描行业数据平台**（每周一次）：中国养猪网、Mysteel 农产品、卓创资讯、猪易网。关注日度/周度猪价、淘汰母猪价、仔猪价、出栏均重。
2. **上市猪企月度销售简报**（每月 5-10 日发布）：牧原/温氏/新希望/正邦/天邦等。关注**出栏量环比变化、销售均价、能繁母猪存栏变动**。出栏量环比趋势比绝对数更重要。
3. **农业农村部月度新闻发布会**：通常公布能繁母猪存栏环比、定点屠宰量等核心数据。同时关注其措辞变化——"产能偏高"vs"产能基本合理"vs"产能偏紧"反映官方判断的转变。
4. **饲料数据作为领先指标**：仔猪料、母猪料销量环比变化领先能繁母猪存栏变化 1-2 个月。若母猪料销量持续下降，意味着更多养殖户在淘汰母猪。
5. **生猪调运数据**：关注北方主产区→南方主销区的调运量变化，反映区域性供需平衡状态。
6. **逆向信号**：猪价在淡季突然大涨（可能是二育扰动而非真正反转）、大型猪企在巨亏中仍公告扩产（说明去化意愿弱）、能繁母猪存栏不降反升（去化进程被打断）。

### 6.8 信号评估：噪声 vs 趋势的区分原则

猪价和产能数据的波动性极高，区分信号与噪声尤为重要：

1. **日度/周度猪价波动多为噪声**：端午节前备货、周末屠宰量变化、天气影响调运等造成短期波动，不改变供需基本面。只有**趋势性变化**（连续 2 周以上同向）才构成信号。
2. **月度的能繁母猪存栏环比变化是核心信号**：单月小幅波动（±0.5% 以内）可能是统计误差或季节性因素，但**连续 3 个月同向变化**大概率是趋势。当前 2025 年 8 月以来能繁已连续 8 个月下降，去化趋势已确立——关键看降速。
3. **出栏均重的变化方向比绝对值更重要**：单周波动可能受短期出栏节奏影响，但 2-3 周的持续下降趋势意味着压栏猪源正在出清。
4. **淘汰母猪/生猪价比率是高质量的去化信号**：这个比率比能繁母猪存栏数据更"真实"——它反映的是养殖户的实际淘汰意愿而非统计估算。比率趋势性下行是最可靠的去化加速信号。
5. **猪粮比是政策干预的前瞻指标**：猪粮比低于 5:1 触发一级预警，低于 4:1 触发收储等措施。关注发改委预警等级变化和收储公告。

| 层级 | 类型 | 代表来源 | 使用原则 |
|------|------|---------|---------|
| **T1** | 官方统计与政策 | 农业农村部、国家统计局、国家发改委、海关总署 | 能繁母猪/屠宰量/猪粮比等核心数据的权威来源；注意数据可能回溯修正 |
| **T2** | 权威行业数据 | Mysteel 农产品、卓创资讯、中国养猪网、猪易网 | 淘汰母猪比率/出栏均重/冻品库存等高频数据的核心来源 |
| **T3** | 公司披露 | 牧原/温氏/新希望/正邦/天邦月度销售简报+季报/年报 | 出栏量/均价/能繁变化/现金流/负债率 |
| **T4** | 券商与独立研究 | 中邮/中信/开源/长江/长城证券；USDA FAS；农科院（王祖力） | 关注其方法论和时间表判断而非结论数字 |
| **T5** | 行业媒体 | 期货日报、36氪、21经济网、财联社 | 核对原始来源 |

**交叉验证原则**：猪周期分析中最容易犯的错误是依赖单一数据源。能繁母猪数据必须用淘汰比率和仔猪价格交叉验证。若农业农村部数据显示能繁在降、但淘汰比率不降反升——说明实际去化力度弱于官方数据。

### 6.11 更新纪律

猪周期研判的最大风险是**过度关注短期波动而忽视结构性变化**。当前周期的核心结构性问题（PSY 提升对冲去化、规模化企业扛亏）是影响反转时点的第一性原理，不要因为单周猪价波动而改变对中长期趋势的判断。

| 更新类型 | 触发条件 | 动作 |
|---------|---------|------|
| **数据刷新** | 月度能繁母猪存栏发布、猪企月度销售简报 | 更新对应数据，计算去化速度和达标时间 |
| **信号标注** | 淘汰比率单周降至 70% 以下、出栏均重单周下降 >2 kg、猪粮比单周升至 >5:1 | 加注"待验证"，连续确认后升级 |
| **研判修正** | 连续 3 个月月均去化速度显著高于或低于 20 万头（如 >30 万头或 <10 万头） | **修订反转时间表**：加速则提前至 2026 Q4，减速则推迟至 2027 H2 |
| **结构性修订** | (1) 农业农村部**回溯性大幅修正**能繁母猪存量数据；(2) PSY 数据出现 >1.5 头的年度提升；(3) 大规模非洲猪瘟/口蹄疫爆发；(4) 新希望/天邦/新五丰出现**实质性债务违约** | 重写核心研判章节 |
| **定期审视** | 每月第二个周五（月度数据集中发布后） | 重新检查：(1) 反转信号清单中哪些已亮起、哪些仍未出现；(2) 跟踪事项是否需要增减 |

> **一条原则**：猪周期的反转是"慢变量"驱动的事件——能繁母猪 10 个月的传导时滞决定了趋势一旦形成不会轻易扭转。但在关键窗口期（去化加速），边际变化值得高度关注。

> **6月30日跟踪记录**：本次为数据刷新 + 信号标注级别更新。5月样本能繁去化加速（-1.1%~-1.32%）尚未满足"连续3个月"的研判修正触发条件，但淘汰比率跌破 75% 已触发信号标注（'待验证'状态）。**两个核心变量**：能繁母猪去化速度（当前在加速）和 PSY 提升速度（当前也在加速）——二者方向相反，但都对反转时间表产生显著影响。**反转时间表上调概率在增加但待确认**。下次定期审视窗口：7月第二个周五（7月10日前后），重点验证 6-7 月样本去化是否持续。
