## 三、逻辑证伪警报（出现任一信号，必须重新评估投资逻辑）

| 证伪信号 | 判断标准 | 后果 |
|:---|:---|:---|
| **IPM市占率跌破40%** | 年报/半年报IPM出货量<2亿颗×国产IPM总量，或竞品（格力新元/芯能）进入格力/美的核心供应链 | 核心护城河受损，评级下调至"卖出" |
| **SiC连续3季亏损扩大** | 士兰明镓连续3个季度经营性亏损扩大（每季>1.5亿），且无减亏趋势 | SiC投资逻辑证伪，下调盈利预测50%+ |
| **毛利率连续两季下滑超2pct** | 2026Q2+Q3毛利率均<17% | 判断毛利率企稳的逻辑错误，全面下调评级 |
| **有息负债/权益突破100%** | 资产负债表中（短期+长期借款）/归母权益>100% | 财务风险不可接受，流动性危机概率大幅上升 |
| **董事长/核心管理层大规模减持** | 陈向东等七人合计减持超过1%股份 | 内部人对公司前景悲观，系统性折价 |
| **12英寸模拟IC线停工或延期超6个月** | 公司公告或厦门海沧当地新闻 | 中长期增长逻辑受损，估值中枢下移 |
| **车规客户出现重大质量事故** | 媒体报道/客户公告士兰微车规产品召回 | 车规认证壁垒受损，车规业务增长逻辑终结 |

## 五、跟踪清单（简易操作版）

每次季报后按此清单逐项打钩：

- [ ] 营收YoY是否在基准区间（15-20%）？
- [ ] 毛利率是否环比改善？
- [ ] 经营现金流是否>净利润？
- [ ] 有息负债是否在可控范围（<100%权益）？
- [ ] IPM出货是否按20-30%增速推进？
- [ ] SiC产线是否按计划爬坡？
- [ ] 研发费用率是否持续下降？
- [ ] 白电排产数据是否回暖（季节性因素外）？
- [ ] 功率半导体行业是否维持涨价趋势？
- [ ] 证伪警报是否触发？（见第三节）

**全部绿灯 → 维持基准假设和目标价**
**1-2项黄灯 → 警惕，密切跟踪**
**任意红灯 → 立即重新评估投资逻辑**
