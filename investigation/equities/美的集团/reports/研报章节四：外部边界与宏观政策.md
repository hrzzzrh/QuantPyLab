# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月26日

---

## 一、宏观环境变量

### 1.1 国内经济：增长有顶、复苏不均

| 指标 | 2025 | 2026E | 2027E | 方向 |
| :--- | :---: | :---: | :---: | :--- |
| GDP增速 | 5.0% | 4.6-4.7% | 4.3-4.5% | ↓ 温和放缓 |
| CPI通胀 | 0.0% | 0.6-1.3% | 1.0% | ↑ 微幅回升 |
| 房地产投资 | -12% | -13% | — | ↓ 仍在探底 |
| 社会消费品零售 | ~3.5% | ~2.5% | — | ↓ 弱于预期 |
| 出口增速 | 5.1% | ~3.0% | — | ↓ 外需减速 |
| 制造业投资 | 高个位数 | 高个位数 | — | → 政策托底 |

**对美的集团的影响**：

1. **GDP增速下行至4.6-4.7%区间**：经济增速放缓对家电消费形成系统性压制。但家电作为生活必需品（尤其空调在高温下的刚性需求属性），其需求弹性远小于可选消费品（如汽车）。美的的"刚需+更新"双属性使其在经济放缓中具备一定防御性。

2. **房地产持续拖累（投资-13%）**：新房交付下降直接影响空调/冰箱/洗衣机的配套需求。但经过三年下行，地产对家电需求的边际负面影响已大幅减弱——当前新增配套需求仅占家电总需求的约20-25%（2020年前为35-40%），存量更新已成为主导力量。

3. **消费端疲弱（零售仅+2.5%）**：2026年4月数据显示家电零售同比-15.1%、家具-10.4%，以旧换新政策效应消退后的需求真空期正在显现。**这与模块二的需求前置审计结论一致——2025年补贴创造的"超买"正在被2026年的"少买"所抵消。**

4. **通胀回升但极为温和（0.6% CPI）**：当前仍是近通缩环境，央行无需加息。对美的而言，低通胀环境有利于维持较低的融资成本（短期借款439亿），也为后续可能的降息提供了空间。

### 1.2 人民币汇率：渐进升值，海外收入受扰

- 中国经常账户盈余预计超GDP的4%，理论上支撑人民币升值。但资本外流（海外直接投资、证券投资）部分抵消升值压力。
- 花旗预测2026年人民币对美元温和升值，贸易加权汇率基本稳定。
- **对美的的影响**：
  - 负向：海外收入1,959亿（占42.9%），人民币升值1%≈利润损失约10-15亿元
  - 正向：进口原材料（芯片、零部件）成本下降；海外产能以本地货币计价，天然对冲
  - 美的财务费用2025年为净收入59亿（利息+汇兑收益），说明其对汇率风险管理有效

### 1.3 全球利率环境：降息周期提供流动性支撑

- 美联储2025年9月重启降息，2026年仍处于降息通道
- 全球流动性宽松支撑资产价格，对美的估值形成底部支撑
- 但中东地缘冲突推升油价，可能延缓全球降息节奏

## 二、地缘政治风险

### 2.1 中美贸易摩擦：已从"突发事件"变为"常态化变量"

| 时间 | 事件 | 对家电出口的影响 |
| :--- | :--- | :--- |
| 2018年 | 美国对华空调/冰箱/洗衣机加征25%关税 | 中国对美空调出口从20.6%→不足8% |
| 2025年Q4 | 特朗普与习近平韩国会晤，达成一年贸易休战 | 关税暂缓升级 |
| 2026年2月 | 美国最高法院推翻大规模关税 | 对华10%芬太尼关税+10%对等关税取消 |
| 2026年4月 | 美国新规：含>15%金属的成品征25%统一关税 | 钢铁密集型家电（冰箱/洗衣机）成本上升 |
| 2026年5月 | 中美贸易和投资委员会成立，讨论关税调整 | 边际缓和信号 |

**核心判断：中美贸易摩擦已不再是美的出口的"系统性风险"，而是已被管理和定价的"已知变量"**

证据：
1. 中国对美家电直接出口占总出口的比例已从2018年的27%以上降至不足15%（对美的而言更低，推测<10%）
2. 美的的海外产能（泰国、越南、印尼、巴西、埃及）已形成"中国技术+海外制造"的双轨模式
3. 东南亚出口到美国虽有关税，但远低于从中国直接出口的税率
4. 海尔Q1 2026北美市场的崩塌（收入降近10%）恰恰反证了"不依赖北美"这一战略的优势

### 2.2 中东地缘冲突：伊朗战争的外溢效应（已发生根本性转折）

**当前局势**（截至2026年6月26日）：**美伊停战协议已于6月17日正式签署生效，霍尔木兹海峡重新开放。** 这是跟踪手册中列为最高优先级催化剂的重大基本面转折。

**关键时间线**：
- 6月14日：美国总统特朗普宣布美伊协议"现在完成"，正式批准开放霍尔木兹海峡，下令解除美国海军对伊朗港口的海上封锁
- 6月15日：伊朗最高国家安全委员会正式确认伊美停战谅解备忘录达成
- 6月17日：双方远程签署谅解备忘录，设立60天谈判期，目标达成涵盖制裁解除和伊朗核计划的全面协议
- 6月18日：霍尔木兹海峡正式重新开放，伊朗宣布60天内不收取通行费
- 6月20日：特朗普声明60天期限结束后也不会收取任何通行费；美军解除针对伊朗港口的海上运输限制
- 6月24日：单日78艘船舶通过霍尔木兹海峡，创战争爆发以来最高纪录；美国能源部长确认24小时内72艘船只、2000万桶石油通过海峡，"完全恢复冲突前石油运输"
- 6月25日：新华网报道，本月霍尔木兹海峡的船舶日均通航量已恢复至冲突前水平的约57%

对美的集团的三大正向影响（全部转为利好）：

| 影响维度 | 战前状态（6月初） | 当前状态（6月26日） | 对美的的影响 |
| :--- | :--- | :--- | :--- |
| 金属供应 | 霍尔木兹海峡封锁→铝库存20年最低 | **海峡重开**→中东铝供应恢复预期，铝价大跌 | **成本端重大利好**（见下文） |
| 物流运费 | 海运费用高位→中东出口订单取消 | 运费回落趋势确立 | 中东出口恢复，海外收入改善 |
| 能源成本 | 油价高位→塑料/制冷剂成本高 | 油价回落至战前水平 | 间接降低原材料成本 |

**铜价维度**（最新数据，截至2026年6月25日）：
- LME铜3月期约 **$13,194/吨**（6月25日现货结算价），较6月5日（$13,731）**下降3.9%**
- 6月25日铜价一度触及七周低点，后因美元走弱反弹至$13,211
- 铜价回落主因：美联储鹰派加息预期→美元走强+风险资产承压；美伊停战→地缘风险溢价消退
- 沪铜主力对应当前约9.7-10.0万元/吨区间，较高点回落约6-8%
- Aegis报告指出铜价"自伊朗战争期间涨幅全部回吐"
- 铜占空调整机成本22-25%，铜价每下降1万元/吨，单台空调成本降低200-300元

**铝价维度**（重大利好——铝价已从4年新高暴跌至3个月低点）：
- LME铝3月期约 **$3,132/吨**（6月25日现货结算价，6月26日回升至$3,143），较6月2日高点（$3,855）**暴跌18.7%**
- 铝价走势时间线：6月2日$3,855（4年新高）→ 6月15日$3,420（停战协议宣布）→ 6月23日$3,264（海峡重开）→ 6月25日$3,132（硅谷美联储鹰派+中东供应回归预期）
- Bloomberg报道铝价"因中东供应回归暴跌7%"，创下四周连跌
- 铝库存：LME库存308,225吨（6月25日），较6月高点略有回升但仍处低位
- **国内铝价对应约2.2-2.3万元/吨，较6月高点（2.5-2.6万元/吨）回落10-12%**
- 上次研报中判断的"铝价$3,855创4年新高是最大风险边际恶化"——**这一核心风险已基本解除**

**成本压力测试（大幅改善）**：若铜价维持$13,200/吨、铝价$3,150/吨运行，美的原材料成本压力将显著缓解。按原材料占比82.26%计算：
- 铝价较6月高点下降18.7%，铜价下降3.9%
- 铝/铜合计占白电物料成本约30-35%
- **粗略估计：综合原材料成本较6月初的极端水平下降约5-7%**
- 此前估计"铝价若维持$3,800-4,000，毛利率压缩3-4ppt"的情景已不成立
- **新评估：毛利率压缩幅度预计收窄至1.5-2.5ppt**（仍有压力，但可控）

**剩余风险**：
1. 伊朗副外长仍将黎巴嫩停战作为最终协议的前置条件，若黎巴嫩冲突激化可能影响后续谈判
2. 霍尔木兹海峡通航虽在恢复，但仍需扫雷作业，完全正常化需1-2个月
3. 协议为60天临时停战+谈判框架，最终协议尚未签署，存在破裂风险（但概率较低，双方均有达成全面协议的政治需求）
4. 6月25日伊朗革命卫队通过无线电要求船舶"仅经许可的指定航线通行"并警告"偏离将承担后果"，显示局势仍存反复可能

### 2.3 供应链重构（China+1）：不可逆但已入价

- 全球家电制造向东南亚（越南、泰国）、墨西哥、东欧转移是大趋势
- 美的是全球产能布局最激进的企业之一（29个海外研发中心+43个制造基地）
- 短期有建设成本，但长期构建了不可复制的竞争壁垒

## 三、监管与产业政策

### 3.1 以旧换新政策：从"跑量"到"提质"，边际支撑减弱

| 对比维度 | 2025年政策 | 2026年政策 | 变化 |
| :--- | :--- | :--- | :--- |
| 补贴品类 | 12类 | 6类 | **减半** |
| 补贴比例 | 20%（1级）/15%（2级） | 15%（仅1级） | **缩水** |
| 空调限购 | 每人3件 | **每人1件** | **大幅收紧** |
| 单件上限 | 2,000元 | 1,500元 | -25% |
| 全年预算 | 约834亿 | 约700亿 | -16% |

**对美的的影响**：正面有限，负面可控
- 补贴品类缩减中剔除的品类（灶具、烟机、微波炉等）对美的影响较小，核心品类（空调、冰箱、洗衣机、热水器）仍在补贴范围内
- 补贴比例下降+预算缩减意味着政策对需求的拉动边际减弱约25-30%
- 但政策强调对一级能效的倾斜，有利于美的的产品结构（一级能效占比行业领先）

### 3.2 "十五五"规划导向：制造业升级的确定性

- 十五五规划（2026-2030）核心方向："新质生产力"、科技自立自强、高端制造
- 美的的ToB业务（机器人与自动化、楼宇科技、工业技术）完美契合国家战略
- 数据中心的AI基建、国产替代、设备更新补贴为楼宇科技提供独立于地产的增长来源
- 人形机器人关节模组、具身智能等前沿布局享受政策红利

### 3.3 环保与能效新规：推升行业门槛

- 2026年6月冰箱新能效标准实施，加速淘汰低效产品
- 空调新能效标准向更高APF值演进
- 环保冷媒（R290/R32）替代进程加速
- **本质影响**：环保/能效门槛提高→小品牌退出→行业集中度提升→美的受益

## 【模块四核心结论】（重大更新）

1. **宏观环境温和不利但非灾难性**：GDP 4.6-4.7%、消费疲弱、地产拖累持续，但家电的刚性需求属性和存量更新主导的特征使美的具备一定防御性。**5月社零同比转负（-0.6%，四年来首次），家电类-15.6%（较4月-15.1%进一步恶化），消费端压力正在系统性加剧。** 5月空调全渠道零售额同比-10.3%，6月行业排产-18.9%，7月内销排产-15.6%——零售下行斜率在Q2进一步陡化。

2. **原材料成本出现戏剧性反转——铝价暴跌18.7%是本次更新的最大利好**：美伊停战协议6月17日正式签署，霍尔木兹海峡重开，铝价从6月2日的$3,855（4年新高）暴跌至6月25日的$3,132（3个月低点），降幅高达18.7%。铜价同步回落3.9%至$13,194。此前研报中视为"最大成本风险边际恶化"的铝价已降至美伊战争前水平。**毛利率压缩幅度从此前估计的3-4ppt收窄至1.5-2.5ppt。** 但仍需注意6月25日伊朗革命卫队无线电警告事件显示局势并非完全稳定。

3. **地缘风险大幅消退——跟踪手册最高优先级催化剂已触发**：美伊停战协议的签署带来了铝价回落、运费下降、中东市场恢复三重利好。霍尔木兹海峡日均通航量已恢复至战前约57%（6月24日78艘创新高）。虽然仍需关注黎巴嫩停战等后续条件，但战争威胁已从"高概率尾部风险"降级为"低概率残余风险"。

4. **最大长期不确定性仍是2027年国补退坡**：若以旧换新在2027年完全退出，内销量可能面临15%+的断崖式下跌。美伊停战的好处主要体现在成本端，无法弥补国内需求的结构性下行。但美的的海外（43%）和ToB（27%）结构已为最坏情况做好了缓冲准备。2026年Q2零售数据的加速恶化（5月家电类-15.6%）使这一风险的时间线更加紧迫。

5. **新增关注点：厄尔尼诺高温预期**：NOAA预测2026年发生中等及以上强度厄尔尼诺事件概率达82%，7月全国或迎大范围持续性高温热浪，有望拉动空调旺季零售修复，成为下半年内销核心增长变量。但需等待7月末至8月实际气温验证。

*（本章节完。整体判断：宏观与地缘环境偏向逆风，成本端铝价创4年新高是最显著的风险边际恶化。但美伊停战谈判取得进展提供了一个重要的潜在正面催化。短期核心矛盾是：成本上行斜率（铝价）> 价格传导斜率（终端均价同比-3%），毛利率压缩风险加剧。）*
