# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月06日

---

## 一、宏观环境变量

### 1.1 国内经济：增长有顶、复苏不均

| 指标 | 2025 | 2026E | 2027E | 方向 |
| :--- | :---: | :---: | :---: | :--- |
| GDP增速 | 5.0% | 4.6-4.7% | 4.3-4.5% | ↓ 温和放缓 |
| CPI通胀 | 0.0% | 0.6-1.3% | 1.0% | ↑ 微幅回升 |
| 房地产投资 | -12% | -13% | — | ↓ 仍在探底 |
| 社会消费品零售 | ~3.5% | ~2.5% | — | ↓ 弱于预期 |
| 出口增速 | 5.1% | ~3.0% | — | ↓ 外需减速 |
| 制造业投资 | 高个位数 | 高个位数 | — | → 政策托底 |

**对美的集团的影响**：

1. **GDP增速下行至4.6-4.7%区间**：经济增速放缓对家电消费形成系统性压制。但家电作为生活必需品（尤其空调在高温下的刚性需求属性），其需求弹性远小于可选消费品（如汽车）。美的的"刚需+更新"双属性使其在经济放缓中具备一定防御性。

2. **房地产持续拖累（投资-13%）**：新房交付下降直接影响空调/冰箱/洗衣机的配套需求。但经过三年下行，地产对家电需求的边际负面影响已大幅减弱——当前新增配套需求仅占家电总需求的约20-25%（2020年前为35-40%），存量更新已成为主导力量。

3. **消费端疲弱（零售仅+2.5%）**：2026年4月数据显示家电零售同比-15.1%、家具-10.4%，以旧换新政策效应消退后的需求真空期正在显现。**这与模块二的需求前置审计结论一致——2025年补贴创造的"超买"正在被2026年的"少买"所抵消。**

4. **通胀回升但极为温和（0.6% CPI）**：当前仍是近通缩环境，央行无需加息。对美的而言，低通胀环境有利于维持较低的融资成本（短期借款439亿），也为后续可能的降息提供了空间。

### 1.2 人民币汇率：渐进升值，海外收入受扰

- 中国经常账户盈余预计超GDP的4%，理论上支撑人民币升值。但资本外流（海外直接投资、证券投资）部分抵消升值压力。
- 花旗预测2026年人民币对美元温和升值，贸易加权汇率基本稳定。
- **对美的的影响**：
  - 负向：海外收入1,959亿（占42.9%），人民币升值1%≈利润损失约10-15亿元
  - 正向：进口原材料（芯片、零部件）成本下降；海外产能以本地货币计价，天然对冲
  - 美的财务费用2025年为净收入59亿（利息+汇兑收益），说明其对汇率风险管理有效

### 1.3 全球利率环境：降息周期提供流动性支撑

- 美联储2025年9月重启降息，2026年仍处于降息通道
- 全球流动性宽松支撑资产价格，对美的估值形成底部支撑
- 但中东地缘冲突推升油价，可能延缓全球降息节奏

## 二、地缘政治风险

### 2.1 中美贸易摩擦：已从"突发事件"变为"常态化变量"

| 时间 | 事件 | 对家电出口的影响 |
| :--- | :--- | :--- |
| 2018年 | 美国对华空调/冰箱/洗衣机加征25%关税 | 中国对美空调出口从20.6%→不足8% |
| 2025年Q4 | 特朗普与习近平韩国会晤，达成一年贸易休战 | 关税暂缓升级 |
| 2026年2月 | 美国最高法院推翻大规模关税 | 对华10%芬太尼关税+10%对等关税取消 |
| 2026年4月 | 美国新规：含>15%金属的成品征25%统一关税 | 钢铁密集型家电（冰箱/洗衣机）成本上升 |
| 2026年5月 | 中美贸易和投资委员会成立，讨论关税调整 | 边际缓和信号 |

**核心判断：中美贸易摩擦已不再是美的出口的"系统性风险"，而是已被管理和定价的"已知变量"**

证据：
1. 中国对美家电直接出口占总出口的比例已从2018年的27%以上降至不足15%（对美的而言更低，推测<10%）
2. 美的的海外产能（泰国、越南、印尼、巴西、埃及）已形成"中国技术+海外制造"的双轨模式
3. 东南亚出口到美国虽有关税，但远低于从中国直接出口的税率
4. 海尔Q1 2026北美市场的崩塌（收入降近10%）恰恰反证了"不依赖北美"这一战略的优势

### 2.2 中东地缘冲突：伊朗战争的外溢效应

**当前局势**（截至2026年6月6日）：2026年2月28日起的美伊战争已持续近100天。4月8日达成临时停火后，双方在5月下旬曾就60天停火延期+核谈判初步谅解备忘录达成框架，但特朗普要求修改条款、伊朗同样提出修订意见，协议至今未正式签署。6月初局势再度恶化——伊朗无人机袭击科威特机场（造成1死63伤）、美军击落4架伊朗无人机并打击伊朗沿海雷达站、霍尔木兹海峡通行量仍严重受限（近两月仅约1,000次商业通行 vs 战前每日100+艘次）。

对美的集团的三重冲击：

| 冲击维度 | 具体影响 | 对美的的影响 |
| :--- | :--- | :--- |
| 能源成本 | 国际油价高位震荡→塑料/制冷剂成本持续承压 | 原材料成本全线承压 |
| 金属供应 | 霍尔木兹海峡封锁持续→铝库存降至近20年最低 | 铝价3,800+美元/吨，创4年新高 |
| 物流运费 | 海运费用持续高位→订单延迟/取消 | 中东出口量3月下调超50万台，4月外销-12% |

**铜价维度**（截至2026年6月5日，更直接影响美的）：
- LME铜3月期约13,731美元/吨（6月5日结算价），较5月24日时的~13,600美元微幅上涨
- 本周铜价一度突破$13,900后回落至$14,000下方，主要受美债收益率回升和美元走强压制
- ING分析师指出铜价近期为"获利了结"，但长期结构性需求（电气化、AI基建、电网投资）仍支撑铜价底部
- 美国商务部预计6月30日前就是否将关税扩展至精炼铜做出决定，可能引发下半年铜价大幅波动
- 沪铜主力约10.3-10.6万元/吨区间，较2025年均价上涨18.6%
- 铜占空调整机成本22-25%，每吨铜价上涨1万元，单台空调成本增加200-300元

**铝价维度**（重点关注——较上次更新发生重大变化）：
- LME铝现货价6月2日触及**$3,855/吨，创4年新高**（超越2022年3月的$3,851），6月3日回落至$3,797
- LME铝库存持续下降至335,450吨（6月3日），其中已注销仓单80,825吨（占比24.1%）——意味着近1/4的注册库存已被预定提走
- 花旗上调2026H2铝目标价至4,000美元/吨，牛市情景看$4,000-4,200
- 国内铝价对应约2.5-2.6万元/吨，较上次更新（2.4万元/吨）明显抬升
- **铝库存处于20年最低水平，霍尔木兹海峡封锁使中东铝供应受阻，叠加全球能源转型对铝的增量需求，铝价中枢已系统性上移**

**压力测试（更新）**：若铜价维持$13,700/吨、铝价$3,800/吨运行，美的原材料成本将同比增长约10-14%（此前估计8-12%）。按原材料占比82.26%计算，对应总营业成本增加约8-12%（此前6-10%），需提价4-6%才能完全覆盖（此前3-5%）。铝价的超预期上涨是本次更新中最大的成本端边际恶化。

**积极因素**：
1. 美的已开展大宗商品套期保值业务，历史上（2021年铜价涨40%时）净利率仅下降不到1个百分点
2. 美的2026年1月已率先提价约6%，行业跟随，但4月终端零售均价同比-3%（Q1 -1%），说明在弱需求下成本传导能力边际减弱
3. 铝价上涨反而有利于减少"铝代铜"带来的竞争威胁（低成本品牌无法通过降级材料规避成本）
4. 铜价短期回落至$14,000下方，部分缓解成本压力

### 2.3 供应链重构（China+1）：不可逆但已入价

- 全球家电制造向东南亚（越南、泰国）、墨西哥、东欧转移是大趋势
- 美的是全球产能布局最激进的企业之一（29个海外研发中心+43个制造基地）
- 短期有建设成本，但长期构建了不可复制的竞争壁垒

## 三、监管与产业政策

### 3.1 以旧换新政策：从"跑量"到"提质"，边际支撑减弱

| 对比维度 | 2025年政策 | 2026年政策 | 变化 |
| :--- | :--- | :--- | :--- |
| 补贴品类 | 12类 | 6类 | **减半** |
| 补贴比例 | 20%（1级）/15%（2级） | 15%（仅1级） | **缩水** |
| 空调限购 | 每人3件 | **每人1件** | **大幅收紧** |
| 单件上限 | 2,000元 | 1,500元 | -25% |
| 全年预算 | 约834亿 | 约700亿 | -16% |

**对美的的影响**：正面有限，负面可控
- 补贴品类缩减中剔除的品类（灶具、烟机、微波炉等）对美的影响较小，核心品类（空调、冰箱、洗衣机、热水器）仍在补贴范围内
- 补贴比例下降+预算缩减意味着政策对需求的拉动边际减弱约25-30%
- 但政策强调对一级能效的倾斜，有利于美的的产品结构（一级能效占比行业领先）

### 3.2 "十五五"规划导向：制造业升级的确定性

- 十五五规划（2026-2030）核心方向："新质生产力"、科技自立自强、高端制造
- 美的的ToB业务（机器人与自动化、楼宇科技、工业技术）完美契合国家战略
- 数据中心的AI基建、国产替代、设备更新补贴为楼宇科技提供独立于地产的增长来源
- 人形机器人关节模组、具身智能等前沿布局享受政策红利

### 3.3 环保与能效新规：推升行业门槛

- 2026年6月冰箱新能效标准实施，加速淘汰低效产品
- 空调新能效标准向更高APF值演进
- 环保冷媒（R290/R32）替代进程加速
- **本质影响**：环保/能效门槛提高→小品牌退出→行业集中度提升→美的受益

## 四、综合风险矩阵

| 风险类别 | 烈度 | 影响方向 | 美的对冲能力 |
| :--- | :---: | :---: | :---: |
| 铜铝价格持续高位 | **高**（上调） | 毛利率压缩3-4ppt | 套保+提价传导（但终端弱需求制约） |
| 人民币升值 | 中 | 海外收入折算损失 | 本地化产能对冲、财务套保 |
| 中美贸易再升级 | 中（已入价） | 对美出口减少 | 不到10%收入依赖美国 |
| 中东地缘长期化 | 中高 | 运费+原材料双升 | 全球多点布局缓冲 |
| 房地产持续下滑 | 中（边际减弱） | 新配套需求下降 | 存量更新已为主导（75%+） |
| 国内消费进一步走弱 | 中高 | 国内销量承压 | 海外42.9%收入对冲 |
| 以旧换新2027年退坡 | 高（远期） | 需求断崖（-15%+） | 模块五将定量测算 |
| 碳关税/ESG合规 | 低（中期） | 出口门槛提高 | 39家国家级绿色工厂 |
| **美伊停战协议达成** | **正（重大利好）** | 铝价回落+运费下降+中东市场恢复 | 三大压力的同步释放 |

## 【模块四核心结论】

1. **宏观环境温和不利但非灾难性**：GDP 4.6-4.7%、消费疲弱、地产拖累持续，但家电的刚性需求属性和存量更新主导的特征使美的具备一定防御性。2026年4月零售数据（空调-14%量/-15%额）较Q1（-13%）进一步恶化，消费端的压力正在加剧。

2. **原材料成本是2026年最紧迫的利润变量——铝价恶化是本次更新的最大边际变化**：铜价维持$13,700/吨高位（同比+18%），**铝价突破$3,855/吨创4年新高**，较上次研报更新时的$3,570上涨约8%，已触及跟踪手册中">4,000需警惕成本传导压力"警戒线的门槛区域。美的的套保能力+年初提价6%提供了一定缓冲，但终端零售均价4月同比-3%说明成本传导链条在弱需求下正在受阻。毛利率压缩幅度从此前估计的2-3ppt上调至3-4ppt。

3. **地缘风险仍在高位但边际变化积极**：美伊停战谈判已进入"接近达成"的最后阶段——双方初步同意60天停火延期+开启核谈判，但关键条款（伊朗铀浓缩时间表、冻结资金释放额度）仍存分歧。霍尔木兹海峡通行量已跌至正常水平的不足5%。若协议达成，将触发铝价回落+运费下降+中东市场恢复的三重利好。但鉴于双方在核心条款上仍僵持，停战协议的时间表高度不确定。

4. **最大的长期不确定性仍是2027年国补退坡**：若以旧换新在2027年完全退出，内销量可能面临15%+的断崖式下跌。但美的的海外（43%）和ToB（27%）结构已经为此做好了缓冲准备。2026年政策的持续性已被确认。

*（本章节完。整体判断：宏观与地缘环境偏向逆风，成本端铝价创4年新高是最显著的风险边际恶化。但美伊停战谈判取得进展提供了一个重要的潜在正面催化。短期核心矛盾是：成本上行斜率（铝价）> 价格传导斜率（终端均价同比-3%），毛利率压缩风险加剧。）*
