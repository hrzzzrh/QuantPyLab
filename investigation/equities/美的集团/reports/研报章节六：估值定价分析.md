# 研报章节六：估值定价分析

**研究日期**：2026年06月26日

## 一、价值锚定：适配性方法选择

### 1.1 方法论选择理由

美的集团的商业模式特征决定了估值方法的优先级：

| 估值方法 | 适配度 | 理由 |
| :--- | :---: | :--- |
| **PE（市盈率）** | ★★★★★ | 盈利稳定、现金流充沛，是最合理的主锚 |
| **DCF（现金流折现）** | ★★★★☆ | 自由现金流充沛（2025年FCF 422亿），作为验证锚 |
| PB-ROE | ★★★☆☆ | 适用于ROE下行期评估底部支撑，但PB受权益基数扩张影响 |
| PEG | ★★★☆☆ | 利润增速9-11%（中性），PE约11-14x，PEG<1.5合理 |
| SOTP | ★★★★☆ | 适用于ToC/ToB业务差异化的估值逻辑 |
| PS | ★★☆☆☆ | 不适用于成熟制造企业 |
| EV/EBITDA | ★★★☆☆ | 作为辅助参考，排除折旧摊销影响 |

**选择路径**：以PE为主锚（对标历史分位和同业），SOTP为结构验证，DCF/FCF yield为安全边际验证。

### 1.2 历史分位审计

| 指标 | 当前值（6月26日） | 6月6日值 | 5年中位数 | 分位 | 10分位 | 90分位 |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| PE(TTM) | 13.53x | 14.07x | 13.36x | ~52% | 12.07x | 17.17x |
| PB | 2.57x | 2.68x | 2.70x | ~25% | 2.43x | 4.22x |

**分位审计结论**：当前PE处于中位数附近（13.53 vs 13.36），已从6月6日的14.07显著回落，估值从"略偏贵"进入"合理偏低位"。考虑到：
1. A股市场整体估值环境的变迁——2019-2021年的极端估值不可重现
2. 美的已从"纯粹家电股"向"科技+全球化集团"转型，理应享受一定的估值中枢上移
3. 海外可比公司（大金、江森自控、西门子等）的PE普遍在15-22x之间
4. **当前PE(2026E)仅约12.4x（78.55 ÷ 6.35），已逼近近5年10分位（12.07x），安全边际正在累积**

**估值中枢参考**：给予美的合理PE中枢区间 **13-15x**（较历史中位数持平或小幅溢价，体现ToB/全球化溢价），对应基准市值区间 **6,240-7,200亿元**。

## 二、全变量对冲审计矩阵

### 2.1 审慎思考：估值影响力过滤

在进行估值调节前，先区分核心变量与噪声：

| 变量 | 是否真正影响估值中枢？ | 判断 |
| :--- | :--- | :--- |
| 国内家电行业短期下滑 | ❌ 已体现在利润预测中 | 进E，不进PE |
| 铜铝成本上涨 | ❌ 已体现在毛利率假设中 | 进E，不进PE |
| 关税不确定性 | ❌ 产能已全球化对冲 | 进E（小幅），不进PE |
| 以旧换新政策缩减 | ❌ 已在保守情景中定价 | 进E，不进PE |
| ToB转型带来的增长潜力 | ✅ 改变长期赛道宽度 | **进PE（+溢价）** |
| 全球化品牌价值提升 | ✅ 改变业务可预见性 | **进PE（+溢价）** |
| 治理透明度与股东回报 | ✅ 降低尾部风险 | **进PE（+溢价）** |
| A股流动性/风格偏好 | ⚠️ 阶段性但不可持续 | 微弱影响 |

### 2.2 正式审计矩阵

| 审计维度 | 核心审计因子 | 审慎思考（估值影响力过滤） | 调节建议 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质（DNA）** | 全域效率文化、"敢破敢立"的组织变革力（24hr响应机制/极度扁平化）、8座灯塔工厂的智造壁垒、内生孵化能力（从0打造COLMO、华凌防御线） | **Yes（溢价）**：美的独特的"制造业效率文化"是A股极其稀缺的护城河，体现在净利率从8.5%→9.8%、ROIC在重资产化中仍维持12%+。这不是"平庸的优秀"，而是"卓越的韧性"。 | **+5%** | 这种效率文化为长期增长提供了"确定性的基本面"，使投资者愿意为未来利润支付比同业更高的倍数。 |
| **宏观：政策与战略对齐** | "十五五"新质生产力（人形机器人/具身智能）、设备更新（楼宇科技受益）、AI基建（数据中心液冷）、国产替代（电梯/压缩机） | **部分Yes（微弱溢价）**：政策方向确实利好，但"政策红利"≠"定价权"。"十五五"对多数公司的影响为零。但美的的ToB业务在数据中心的真实放量（Carrier/特灵同行验证了赛道宽度）**属于那极少数的例外**。 | **+3%** | 数据中心液冷+AI基建是真实的产业趋势而非概念，美的从ToC能力延伸至ToB领域有技术壁垒，可享受赛道溢价。 |
| **宏观：地缘与外部约束** | 北美敞口<10%、中东运费飙涨、全球供应链重构成本、人民币升值对海外收入折算影响 | **部分Yes（微弱折价）**：全球化在带来增长的同时也引入了地缘波动的"脆弱性溢价"。海尔Q1北美崩塌是前车之鉴。美的虽然敞口小，但"全球化的代价"客观存在。 | **-3%** | 地缘不确定性增加了盈利预测的波动区间，体现在PE倍数上应给予适度折让。 |
| **市场：流动性与风格** | A股对"高股息+低估值"蓝筹的风格偏好、港股上市带来的双重定价（H股折价约15-20%）、外资持股比例** | **No（零影响）**：所谓的"高股息风格"是市场的短期叙事，不能作为改变长期估值中枢的依据。H股折价更多反映的是流动性差异而非基本面分歧。 | **0%** | "风格契合"不能成为赋予估值溢价的理由——它是短期资金博弈的结果，随风而逝。估值应基于基本面，而非市场叙事。 |
| **综合调节系数** | — | — | **+5%** | **基准市值×(1+5%)→修正目标市值** |

### 2.3 修正市值计算

**基准市值选择**：

| 方法 | 参数 | 基准市值 |
| :--- | :--- | :---: |
| PE法（中性EPS 6.35×合理PE 13x） | 13x × 480亿 = 6,240亿 | 6,240亿 |
| PE法（中性EPS 6.35×合理PE 15x） | 15x × 480亿 = 7,200亿 | 7,200亿 |
| FCF Yield法（中性FCF 440亿/5-6% yield） | 440亿÷5.5% = 8,000亿 | 8,000亿 |
| PEG法（中性增速9.2%×1.5 PEG = 13.8x） | 13.8x × 480亿 = 6,624亿 | 6,624亿 |
| **基准市值中枢** | 综合三种方法 | **约6,600亿** |

**修正目标市值** = 6,600亿 × (1 + 5%) = **6,930亿**

**隐含PE** = 6,930亿 ÷ 480亿 = **14.4x**（位于13-15x合理区间内，经得起回测）

## 三、目标价与盈亏比

### 3.1 目标价计算

当前（2026-06-26）：
- 不复权市价：**78.55元**
- 总市值：**约5,980亿元**
- 总股本：约76.13亿股（新增H股可转债172亿港元尚未转股，暂不计入稀释）

| 情景 | 目标市值 | 目标价（不复权） | 较现价涨幅 | 隐含PE | 概率 |
| :--- | :---: | :---: | :---: | :---: | :---: |
| 保守（Bear） | 5,350亿 | **70.3元** | -10.5% | 13.2x | 25% |
| 中性（Base） | **6,930亿** | **91.0元** | **+15.9%** | **14.4x** | 55% |
| 乐观（Bull） | 8,300亿 | **109.0元** | +38.8% | 15.7x | 20% |

**概率加权目标价** = 70.3 × 25% + 91.0 × 55% + 109.0 × 20% = **89.4元**

### 3.2 盈亏比分析

- 现价：78.55元
- 目标价（中性）：91.0元
- 保守情景价格：70.3元
- **盈亏比** = (91.0 - 78.55) / (78.55 - 70.3) = 12.45 / 8.25 ≈ **1.51**

盈亏比1.51，较上次更新（0.49）大幅改善！核心原因：1）EPS从6.21上修至6.35；2）基准市值中枢从6,400亿上修至6,600亿；3）当前价格从81.69元回落至78.55元。三重因素叠加使盈亏比从极度不理想的0.49提升至1.51。**当前价格的风险收益比已进入"可接受"区间（盈亏比>1.0），但仍未达到"理想建仓"标准（盈亏比>2.0对应约75元以下）。**

## 四、逻辑证伪与极限压力测试

### 4.1 黑天鹅情景：三重打击同时发生

**定义**："国内需求断崖 + 海外贸易壁垒全面升级 + 铜价突破15万元/吨"同时发生

在此极端情景下：
- 国内家电零售全年下滑20%+（连续两年国补透支）
- 美国对所有中国产品征100%关税，且东南亚转口被堵死
- 铜价受供应冲击突破15万元/吨

**极限利润测算**：收入下滑30%至3,200亿，毛利率降至21%，归母净利润约100亿。EPS ≈ 1.31元。

### 4.2 极限支撑位（地狱价格）

在"行业生存危机"模式下，市场给予的PE将极度压缩（类似2018年美的PE曾低至9.5x）：

- **PE极限压缩**（9.5x）：100亿 × 9.5x = 950亿市值 → **12.5元/股**
- **PB清算价值**（1.5x归母权益下限）：2,232亿×1.5 = 3,348亿 → **44.0元/股**
- **股息安全垫**（按每股4.3元/年，5%股息率要求）：**86元/股**（当前价格78.55元低于此线，股息率约5.5%，正向持有收益）

**综合极限支撑位**：约**40-45元**（接近1.5x PB或净资产清算价值）

**风险判断**：当前78.55元距离极限支撑位约44%的下行空间，距离中性目标价91.0元约15.9%的上行空间。风险收益比较6月6日（盈亏比0.49）已大幅改善。
