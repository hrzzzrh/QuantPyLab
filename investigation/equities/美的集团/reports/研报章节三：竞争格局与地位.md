## 一、竞争格局扫描

### 1.1 行业集中度：寡头格局稳固，集中度持续提升

| 品类 | 竞争格局 | CR3/CR5 | 美的排名 | 数据来源/时间 |
| :--- | :--- | :---: | :---: | :--- |
| 家用空调 | 美的>格力>海尔 | 全渠道CR2≈50% | **第1**（全渠道~29%） | 奥维+产业在线/2025H1 |
| 中央空调 | 格力>美的 | CR3≈50% | 第2 | 产业在线/2025 |
| 冰箱 | 海尔>美的>海信 | CR3≈68%（内销） | 第2（19.7%内销） | 华经/2025 |
| 洗衣机 | 海尔>美的 | CR3≈70% | 第2 | 奥维云网/2025 |

| 机器人（中国） | 发那科>埃斯顿...>库卡 | 国产份额58% | 库卡第4-5 | MIR/2024 |

### 1.2 三巨头2025年业绩对比

| 指标 | 美的集团 | 海尔智家 | 格力电器 |
| :--- | :---: | :---: | :---: |
| 营业收入 | **4,585亿** (+12.1%) | 3,023亿 (+5.7%) | 1,704亿 (**-9.9%**) |
| 归母净利润 | **439.5亿** (+14.0%) | 195.5亿 (+4.4%) | 290.0亿 (**-9.9%**) |
| 毛利率 | 26.4% | ~30% | ~30% |
| 净利率 | 9.8% | 6.5% | 17.0% |
| ROE（加权） | 19.7% | ~16% | ~24% |
| 海外收入占比 | **42.9%** | >50% | ~15% |
| 研发费用 | **177.9亿** | 101.0亿 | ~70亿 |
| 合同负债 | **469.9亿** | 85.4亿 | ~120亿 |

**核心观察：**
1. **营收与利润全面领先**：美的收入是海尔的1.5倍、格力的2.7倍，且增速最快（+12.1%），净利润同样最高（439.5亿），**在三巨头中实现了唯一的收入与利润双位数增长**。
2. **格力深陷单一品类陷阱**：空调占总收入约78%，在2025年行业下滑中遭受最大冲击，收入/利润双降近10%。
3. **海尔盈利能力垫底**：净利率仅6.5%，高端化与全球化的费用拖累明显，Q4净利润环比暴跌58.5%。
4. **美的合同负债碾压级领先**：美的合同负债469.9亿，是海尔的5.5倍、格力的约4倍，体现对下游强大的"先款后货"议价权。

### 1.3 新兴挑战者：小米的"侧翼进攻"

小米家电2025年Q1大家电业务收入翻倍增长，空调/冰箱出货量同比暴涨超65%。但小米的核心局限在于：

- **制造端劣势**：小米家电部分产线依赖代工，成本控制力不如美的的垂直整合
- **高端化缺失**：万元级以上产品仍是短板，品牌溢价能力有限
- **线下渠道薄弱**：小米之家数量虽多，但需要塞入的商品品类太多，专业化家电销售能力不足

**美的的应对**：重启华凌品牌作为线上性价比防线（"拼成本"），同时COLMO+东芝守高端。这种"上打下压"的双线策略在历史上曾成功抵御多轮价格战。

## 二、护城河识别

### 2.1 成本规模护城河（★★★★★，最强壁垒）

美的的规模效应是中国家电行业最深的护城河：

- **收入规模**：4,585亿，几乎是海尔的1.5倍、格力的2.7倍
- **核心零部件自制**：美芝压缩机全球市场份额第一（空调压缩机），威灵电机全球前三
- **垂直整合**：从上游核心零部件（压缩机、电机、芯片）到下游渠道零售的全价值链覆盖
- **灯塔工厂**：拥有8座世界级灯塔工厂，智能制造水平全球领先
- **成本优势量化**：美的能在华凌品牌"杀价"的同时仍守住利润，而小米代工模式难以匹敌

### 2.2 品牌组合护城河（★★★★☆，正在拓宽）

| 品牌层级 | 品牌 | 定位 | 2025年进展 |
| :--- | :--- | :--- | :--- |
| 超高端 | COLMO | AI科技家电 | 品牌店>1,600家，零售额同比双位数增长 |
| 高端 | 东芝（白电） | 艺术品家电 | 冰箱+20%、洗衣机+25%、厨电+60% |
| 大众 | 美的 | 全品类综合 | 国内线上/线下销售额行业第一 |
| 性价比 | 华凌 | 线上年轻平价 | 对标小米的防御性品牌 |
| 海外 | Midea/Carrier/Toshiba | 组合出海 | OBM占比45%+ |

**品牌溢价验证**：COLMO与东芝双品牌零售额从2022年的超100亿增长至2024年的超170亿，2025年上半年增长势头延续，带动智能家居毛利率从2021年的25.1%提升至2025H1的28.5%。

### 2.3 渠道网络护城河（★★★★★，难以复制）

- **线上**：美的系产品在京东、天猫、抖音、拼多多等主流平台家电销售额均行业第一，电商销售额占智能家居内销收入的55%以上。
- **线下**：600家Mall店、超3,000家门店数智化改造、下沉市场零售规模突破300亿（行业第一）。
- **海外**：50个国家自营分公司、1.6万家数字化终端网点、51个海外官网/近200个社媒账号。
- **渠道数字化**：美云销APP简化、线上线下一盘货、T+3柔性制造降低库存周转（客户库存周转效率改善10%+）。

### 2.4 全球化先发护城河（★★★★☆，仍在加速）

- 海外研发中心29个、制造基地43个（遍布20+国家）
- 50国本地化自营业务覆盖——这是格力完全不具备、海尔仅在少数区域具备的能力
- 产能布局：泰国（空调）、越南、印尼、巴西等多地产能已形成关税缓冲垫
- 品牌赞助：曼城+哈兰德+巴萨（2026/2027起），年触达全球数亿球迷

### 2.5 技术研发护城河（★★★☆☆，硬件强但软件弱）

- 研发投入178亿（绝对值最高），累计专利超1.1万件
- 在压缩机、电机、热泵等核心零部件具备全球竞争力
- 8座灯塔工厂体现智能制造的技术转化能力
- **短板**：软件和互联网生态能力弱于小米，AI/IoT生态靠"开放合作"（与华为/vivo/蔚来/比亚迪结盟）而非自建闭环

## 三、产业链定价权

### 3.1 对上游：压款能力极强

| 指标 | 美的集团 | 行业含义 |
| :--- | :---: | :--- |
| 应付票据及应付账款 | 1,308亿 | 占用供应商资金规模巨大 |
| 前五大供应商集中度 | 6.02% | 供应商高度分散，依赖美的订单 |
| 原材料占比 | 82.26% | 但通过规模集采+套保有效管理成本波动 |

**解读**：1,308亿的应付账款意味着美的平均占用供应商资金约2-3个月，这构成了巨额的无息负债（负的Cash Conversion Cycle），是家电行业最强的盈利模式之一。

### 3.2 对下游：先款后货的绝对控制力

| 指标 | 美的 | 海尔 | 海信 |
| :--- | :---: | :---: | :---: |
| 合同负债 | **469.9亿** | 85.4亿 | 16.2亿 |

合同负债（预收经销商的款项）469.9亿是海尔（85.4亿）的5.5倍、海信（16.2亿）的29倍。这种悬殊的差距反映了美的在渠道端的"不可替代性"——经销商愿意先付款再提货，因为在终端，美的的产品就是最好卖的。

### 3.3 价格传导能力：2026年提价验证

2026年1月，美的空调率先提价约6%，随后行业多家品牌跟随。在原材料（铜铝）高位运行、国补缩减的双重压力下，美的能够率先提价且被市场接受，验证了其定价权的真实存在。

**模块一关联**：毛利率从2021年22.5%提至2025年26.7%（+4.2ppt），正是定价权+产品升级的综合体现。

## 四、竞争格局演变趋势

### 4.1 结构性机会：行业出清下的份额集中

| 趋势 | 表现 | 影响 |
| :--- | :--- | :--- |
| 成本上升+需求收缩 | 中小品牌生存空间被压缩 | 美的Q1内销份额+5ppt |
| 能效标准升级 | 新国标淘汰低效产品 | 头部品牌技术储备变现 |
| 价格战加剧 | 2025年空调均价下降约5% | 小品牌利润率滑向亏损 |
| 国补政策向一级能效倾斜 | 补贴退出后，中低端市场萎缩 | 高端产品竞争壁垒提升 |

### 4.2 潜在威胁：小米的"生态颠覆"

- 小米以IOT生态+互联网模式切入家电，线上价格优势明显
- 但通过模块一/二的深入分析，小米对美的的威胁被市场**高估**：
  1. 家电的核心壁垒是**制造+渠道**，不是软件——小米代工模式难以突破美的的规模壁垒
  2. 美的通过华凌品牌直接"拼成本"的防御非常有效（美的一体化制造成本 < 小米代工成本）
  3. 高端市场（万元+）美的有COLMO/东芝，小米几乎无份额
  4. 美的已与华为等行业头部结盟AI生态，并非"被动挨打"

## 【外部数据交叉验证】

1. **真实订单验证**：Q1 2026美的空调内销出货+18.18%，远超行业均值（3月行业+5.6%），验证了B端集中度提升的真实性。

2. **供应链生态穿透**：应付账款1,308亿的规模，供应商依赖度高（集中度仅6.02%），美的的订单是上游企业的"生命线"，具有极强的砍价能力。

3. **技术代差实证**：COLMO已构建全屋智控、空气、用水等五大系统，厨房电器零售额同比+60%，产品创新并非停留在PPT层面。8座灯塔工厂是全球制造业的顶级认证（中国家电行业最多）。

4. **实物产能校验**：固定资产从335亿增至450亿（+34.2%），在建工程转固说明前期产能项目正在落地，不存在"纸上产能"问题。
