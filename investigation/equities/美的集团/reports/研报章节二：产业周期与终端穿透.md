# 研报章节二：产业周期与终端穿透

**研究日期**：2026年06月26日

---

## 一、产业生命周期与库存节奏

### 1.1 周期定位：成熟期存量博弈

中国家电行业（以白电为核心）已完成从增量渗透到存量替代的历史性跨越。中国制造的白色家电已超过全球市场的60%，中国拥有全球80%的家用空调产能及90%的空调压缩机产能。行业生命周期处于**成熟期后半段**：

- **渗透率饱和**：空调城市百户拥有量已超160台，冰箱/洗衣机接近100台/百户
- **增长动力切换**：从新增购房配套（地产驱动）转向更新换代（存量替换驱动）
- **行业增速下行**：2018年前两位数增长→2023-2025年个位数增长→2026年预计下滑
- **竞争模式演化**：从"做大蛋糕"的增量竞争，转为"抢别人碗里的肉"的存量博弈

**关键识别**：家电行业不是衰退期，而是"总量稳定+结构升级"的成熟期。品类内体现在：高端化替代低端（COLMO/卡萨帝份额提升）、智能化替代非智能（AI冰箱渗透率57%+）、大容量替代小容量（501L+冰箱占比近50%）。品类间体现在：干衣机、洗碗机、扫地机器人等新兴品类仍在渗透率提升阶段。

### 1.2 库存动力学：主动去库阶段

**当前库存周期定位**：产业链处于**主动去库存**阶段

| 信号 | 数据 | 解读 |
| :--- | :--- | :--- |
| 2025年底渠道库存 | 高位 | 连续16个月国补刺激的"抢装"后遗症 |
| 2026年Q1空调零售 | 量同比-13%，额同比-13.8% | 终端需求大幅回落 |
| 2026年Q1空调产量 | 同比+0.7% | 生产已经开始收缩 |
| 4月白电排产 | 同比-3.6% | 厂商主动控产 |
| 美的内销出货 | 美的Q1空调出货+18% | **龙头逆势增长**，份额挤压效应明显 |
| 2025年末库存 | 空调1,441万台（同比-16.5%） | 对比上年同期已大幅去化 |

**核心辨析——"牛鞭效应"放大性评估**：

奥维云网推总数据显示，2026年Q1空调终端零售量同比-13%，但产业在线3月空调内销出货量同比+5.62%（美的内销+18.18%）。**出货与零售形成近20个百分点的剪刀差**，背后有两层含义：

1. **渠道补库而非真实需求**：出货好于零售，说明渠道在旺季前补库存（经销商对高温预期和成本上涨的备货行为），而非终端消费的真实回暖。
2. **龙头份额集中**：美的（+18.18%）、格力（+1.61%）、海尔（+7.78%）三家内销增速均好于行业零售端，表明中小品牌的份额正在被蚕食——这是存量博弈中的"胜者为王"逻辑。

**牛鞭效应判断**：当前处于牛鞭效应的**反弹阶段**——2024-2025年国补刺激终端"超额消费"（向上放大），2026年补贴缩减后终端需求回落（向下放大）。由于渠道备货行为叠加，终端的微小下滑在出厂端被显著放大。这一过程预计将持续2-3个季度后在2026H2趋于稳定。

美的集团在海外市场的核心竞争力在于：1）50国本地化运营；2）泰国、越南、印尼、巴西等多地产能布局已形成缓冲垫；3）OBM占比突破45%，品牌溢价逐步兑现。**模块一发现的海外收入+15.92%、海外毛利26.60%高于国内的数据，在全球出口承压背景下恰恰验证了其全球化布局的α属性。**

### 2.3 动能溯源与透支审计

#### 增长动能分类

| 驱动因素 | 类别 | 持续性评估 | 影响力 |
| :--- | :--- | :--- | :--- |
| 以旧换新补贴（2024-2025） | 外生-政策干预 | **已消退**（2026年补贴缩减近60%） | 2025年贡献约4,395亿消费 |
| 以旧换新补贴（2026） | 外生-政策干预 | **弱化中**（品类从12类→6类，补贴比例20%→15%） | 第一批625亿元（YoY -23%） |
| 全球高温气候 | 外生-自然 | **持续** | 不可逆的长期趋势 |
| 新兴市场城镇化 | 内生-结构 | **长期确定** | 非洲/东南亚空调普及率<40% |
| 地产竣工 | 外生-宏观 | **仍在筑底** | 2024年新开工持续下降 |
| 产品结构升级 | 内生-创新 | **持续** | 高端/智能/节能占比提升 |
| 出海品牌溢价 | 内生-战略 | **逐步兑现** | OBM占比45%→60%的路径 |

#### 需求前置（跨期断层）审计

**强制审计结论：当前增长存在显著的"跨期需求移位"（Intertemporal Demand Displacement）风险**

**证据链**：

1. **2025年透支规模**：商务部数据显示2025年1-11月家电以旧换新超1.28亿台，补贴约827亿元，带动消费4,395亿元。这是在"20%补贴+12个品类+空调可补贴3件"的宽松政策下实现的——远远超过"正常"更新需求。

2. **2026年政策断崖**：2026年政策从12个品类缩至6个（剔除灶具、烟机、微波炉、净水器、洗碗机、电饭煲），补贴比例从20%降至15%（仅一级能效），空调从每人3件降至1件，单件上限从2000元降至1500元。第一批资金625亿（YoY-23%），全年预计700亿左右。

3. **达到使用年限的存量规模**：据券商测算，2025年末空调/冰箱/洗衣机/电视达到安全使用年限需要更新的规模分别为6,881万/16,605万/7,428万/14,391万台——基数庞大，但更新意愿受补贴力度影响极大。

4. **最悲观情景下的需求断崖**：若2027年国补完全退坡，券商测算四大家电更新需求量将同比-19.4%，内销量-15.4%。这意味着补贴创造的不是"增量的增量"，而是"时间的转移"。

**对美的的影响评估**：需求前置对美的的冲击存在三重对冲——1）龙头的份额提升能力（Q1美的内销份额+5ppt）；2）海外收入占比已达43%，且海外增速+16%远超国内+9.4%；3）ToB业务（1,228亿+17.5%）不依赖消费补贴。**这印证了模块一发现的"全球化和ToB转型是应对国内周期下行的核心对冲"的判断。**

## 三、供需边际与压力传导

### 3.1 供需平衡审计

**国内家电供需平衡表**：

| 维度 | 2024-2025年 | 2026年Q1 | 趋势 |
| :--- | :--- | :--- | :--- |
| 终端需求 | 国补刺激下超常增长 | 大幅回落（空调零售-13%） | ↓ |
| 渠道库存 | 高位积累 | 主动去化中 | ↓ |
| 厂商出货 | 跟随国补放量 | 龙头逆势增长，中小下滑 | 分化 |
| 产能利用率 | 旺季冲高至88% | 空调3月约72% | ↓ |
| 新增产能 | 泰国/越南/巴西建厂 | 仍在建设 | → |

**供需格局判断**：**从紧平衡向阶段性过剩滑落**。2026年Q1空调零售量-13%的同时，龙头出货量仍在小幅增长，意味着渠道库存将继续累积——这是典型的需求收缩期"渠道吸库存"现象。预计2026年H2进入被动去库存阶段，届时可能出现价格战的边际恶化。

### 3.2 压力传导特征分析

**传导路径一：补贴退坡→终端需求收缩→渠道去库存→出厂量下滑→价格竞争加剧**

- 传导时滞：约2-3个季度（当前处于该路径的"渠道去库存"阶段）
- 放大倍数：终端需求下降1%→渠道去库存叠加→出厂量下降3-5%（牛鞭效应）
- 美的的缓冲：份额提升（+5ppt）可以部分对冲总量下滑

**传导路径二：铜铝涨价→制造成本上升→终端提价→消费抑制**

- 铜价高位运行，铜铝占白电物料成本约30-35%
- 2026年Q1，品牌终端均价已有所上调（空调均价+6.7%、冰箱均价+5.6%），但提价幅度不足以完全覆盖成本压力
- 美的原材料成本占比82.26%，是成本最敏感的环节

**传导路径三：中东地缘冲突→运费飙升→出口订单延迟/取消→短期海外收入承压**

- 中东占中国家用空调出口约18%，是核心增量市场
- 2026年3月出口排产已下调超50万台
- 美的海外产能（泰国+越南）可部分对冲，但运力瓶颈仍制约交付

**波动放大效应评估**：

美的集团面临的是典型的"微笑曲线两端受压"：一端是制造成本上升（铜铝），另一端是终端价格承压（国补退坡+竞争加剧）。但美的的规模优势（原材料集采议价权）、品牌溢价（COLMO/东芝高端化）和全球化弹性（42.9%海外收入）构成了强大的缓冲垫，使其在产业链压力传导中处于有利位置。

## 三(A)、ToB机器人业务的独立周期分析

### 工业机器人行业周期定位

**周期定位：成长期中段**

与家电的成熟期不同，工业机器人行业仍处于成长的黄金期：
- 2024年全球工业机器人年安装量54.2万台，中国29.5万台（占54%）
- 中国机器人密度567台/万工人（全球第三），仍有提升空间
- 国产品牌市占率从2023年47%跃升至2024年58%（摩根士丹利数据）
- IFR预测2025-2028年全球年安装量CAGR 7%，2028年突破70万台
- MIR预测2026年中国市场继续维持两位数增长

**库卡在中国的竞争定位：**
- 库卡是全球四大工业机器人品牌之一（与发那科、ABB、安川并列）
- 在中国市场，库卡面临国产替代的"双向挤压"：既要与发那科竞争高端，也要应对埃斯顿等国产品牌的价格压力
- 但库卡在汽车行业（尤其是德系车企）、重型负载领域仍具有品牌和技术优势

**模块一关联**：美的机器人与自动化业务收入310亿（+8.05%），增速低于行业整体（中国+13.6%），说明库卡在中国市场的份额面临压力。这需要后续模块中深入验证。
