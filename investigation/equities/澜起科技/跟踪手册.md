## 2. 关键外部变量 (Exogenous Watchlist)

1.  **北美及国内 CSP 资本开支**：关注 NVIDIA 季度财报中关于“运力”芯片的叙述。
2.  **美国 BIS “带宽限制”政策补丁**：重点监控是否出现针对“互连带宽（GB/s）”的新增出口红线。
3.  **Intel Granite Rapids 交付节奏**：英特尔 CPU 的延期将直接推迟 DDR5 接口芯片的爆发点。
4.  **DDR5 现货/合约价走势**：内存颗粒价格过高可能抑制通用服务器的采购意愿。

## 3. 逻辑证伪警报 (Invalidation Triggers) —— **重点！**

若出现以下任一信号，需立即下调评级或止损：

*   **警报 A (市场份额流失)**：连续两季在英特尔新平台中的 RCD 份额被瑞萨（Renesas）反超，或总市占率跌破 30%。
*   **警报 B (价格战烈度失控)**：DDR5 RCD 毛利率连续两季下滑并**跌破 60%**。
*   **警报 C (互连芯片禁令落地)**：美国 BIS 正式限制 PCIe 6.0 规格芯片出口中国，且国内 AEC 业务无法弥补缺口。
*   **警报 D (核心人员流失)**：核心研发副总裁或 SerDes 团队负责人离职。
