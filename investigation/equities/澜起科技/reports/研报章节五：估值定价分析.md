## 二、价值锚定：适配性估值方法选择

### 2.1 方法论选择与理由

| 估值方法 | 是否采用 | 理由 |
| :--- | :---: | :--- |
| **PE（市盈率）** | ✅ 主方法 | 公司稳定盈利，利润质量高（全部费用化研发），适合PE定价 |
| **PEG（成长性估值）** | ✅ 交叉验证 | 澜起处于高增长阶段，PEG可验证PE是否合理 |
| **PB-ROE** | ⚠️ 辅助参考 | 公司资产极轻（Fabless），ROE主要由净利率驱动，PB参考价值有限 |
| **DCF（现金流折现）** | ❌ 不适合 | 半导体周期波动大，远期FCF预测可靠性低 |
| **SOTP（分部估值）** | ✅ 合理性检验 | 运力新品与传统DDR5业务的成长性差异大，分部估值提供参考 |
| **PS（市销率）** | ⚠️ 辅助参考 | 利润率已稳定在40%+，PS可作为极端场景下的底线估值 |
| **EV/EBITDA** | ⚠️ 辅助参考 | 公司几乎零负债+高现金，EV显著低于市值，EV/EBITDA偏低 |

**核心方法：PE (2026E) + PEG交叉验证 + SOTP合理性检验**

## 四、逻辑纯净性校验 (Consistency Guardrails)

### 4.1 E/PE 分离校验

| 因子 | 归入 E（利润预测） | 归入 PE（估值倍数） | 理由 |
| :--- | :---: | :---: | :--- |
| DDR5第4代RCD出货占比提升 | ✅ | - | 直接贡献毛利率和净利润 |
| 运力新品(MRCD/MDB等)放量 | ✅ | - | 具体、可计算的增量收入 |
| 国产DDR5生态（CXMT）的长期红利 | - | ✅ (+5%) | 趋势确定但无法精确量化具体金额 |
| 平台化能力的估值溢价 | - | ✅ (+8%) | 关乎信心和远期确定性 |
| H股融资后现金闲置 | - | ✅ (-3%) | 不直接影响利润，但影响ROE和对管理层的信心 |
| 美国管制的供应链风险 | - | ✅ (-4%) | 无法量化为具体成本 |

**校验通过** ✅：具体的、可算的增量进E；不可算的、关乎信心的因素进PE。

### 4.2 锚点回测

修正后的隐含PE 88x（2026E），与以下参照对比：

| 参照 | PE倍数 | 澜起88x的合理性 |
| :--- | ---: | :--- |
| 澜起自身历史中位数 | 79.7x | 偏高9分位，但当前基本面（增长/格局/新品）远超历史任何时期，溢价合理 |
| Rambus当前 | ~65x | Rambus增速仅澜起的一半，澜起溢价35%合理 |
| Astera Labs当前 | ~162x | Astera单一赛道却享受极端PE，澜起不应与之比肩但也不必被其压制 |
| A股半导体设计板块中位数 | ~60-70x | 澜起属于板块内的"最优等生"，应享受估值溢价 |

**回测结论**：88x 2026E PE在可比框架内处于"合理偏高"位置，有逻辑支撑但安全边际有限。

### 5.2 盈亏比计算

```
当前股价（不复权）：210.27元
概率加权目标价：223元
保守场景价格：106元

上行空间 = (223 - 210.27) / 210.27 = +6.1%
下行风险 = (210.27 - 106) / 210.27 = -49.6%

盈亏比 = 6.1% / 49.6% = 0.12x
```

**盈亏比解读**：当前股价（210元）已非常接近Base Case目标价（223元），上行空间仅约6%。盈亏比0.12x表明风险收益特征不理想——一个悲观场景的出现可导致近50%的回撤，而正常情景的上涨空间有限。

**但是**，这并不一定意味着"立即卖出"。需要结合以下两点：
1. 盈亏比计算中的"保守场景价格106元"是基于熊市×低PE的极端组合，发生概率仅20%；
2. 若公司持续超预期，Base Case会上调，目标价也随之抬升——澜起正处于"业绩可能持续超预期"的阶段。

## 六、逻辑证伪与极限压力测试 (Falsification & Stress Test)

1. **代工停摆**：台积电/联电等停止为澜起代工 → 所有产品线停产
2. **EDA断供**：Cadence/Synopsys停止提供设计工具 → 即使转向中芯国际代工，设计能力也受限
3. **客户流失**：三星/SK海力士/美光出于政治风险考虑，暂停/减少向澜起采购
4. **H股暴跌**：国际投资者恐慌性抛售
