# 研报章节四：外部边界与宏观政策

**研究日期**：2026年05月26日

**前序模块核心结论回顾**：
1. 模块一：2023年营收暴跌37.7%，2024-2025年V型复苏。毛利率从46.4%回升至62.2%，ROIC从2.71%回升至15.28%。
2. 模块二：2026-2027年"黄金窗口期"（AI服务器28%+增长），但2028年面临供过于求风险。
3. 模块三：内存接口芯片全球第一（36.8%），PCIe Retimer全球第二（10.9%），但Astera Labs在PCIe赛道遥遥领先（75.7%毛利率）。

---

## 一、宏观环境变量

### 1.1 利率与折现率环境

当前全球利率环境对高估值成长股总体偏友好：
- **美联储**：2025年下半年已启动降息周期，2026年大概率维持宽松基调。无风险利率下行有利于DCF估值模型中的远期现金流折现——这对于处于成长扩张期的澜起科技（大量价值在远期）构成估值支撑。
- **中国央行**：2025-2026年持续定向降准、LPR下调，以支持科技企业和制造业投资。流动性宽松有助于提升科创板整体估值水位。

**对澜起科技的具体影响**：公司的估值弹性对利率敏感度较高。当前PE TTM约13倍（处于历史极低分位，P10=55.2），利率下行可能带动PE均值回归。但由于A股科创板估值更多受市场情绪和流动性驱动，利率传导的时滞和幅度存在较大不确定性。

### 1.2 汇率损益分析

| 维度 | 影响分析 |
|:---|:---|
| **收入端** | 公司营收以美元和韩元结算为主（主要客户为三星/SK海力士/美光），人民币贬值有利于收入端折算，人民币升值则侵蚀收入。2025年人民币总体偏弱，客观上利好营收。 |
| **成本端** | 晶圆代工费用以美元计价，与收入端币种基本匹配，形成自然对冲。 |
| **汇率风险敞口** | 公司持有约85亿元货币资金，其中相当比例为美元和港元（H股募集资金）。若人民币大幅升值，将产生汇兑损失。2025年财务费用中汇兑损益的具体金额需关注，但公司可通过外汇远期等工具对冲。 |

### 1.3 通胀传导效率

全球半导体产业链通胀压力正在缓解。2023-2024年经历过一轮晶圆代工涨价周期后，2025-2026年台积电等代工厂涨价幅度趋缓（约5-10%年涨）。澜起科技通过产品子代升级（ASP提升15-20%）和产品结构优化（高毛利运力芯片占比提升），有效对冲了上游成本上涨。**通胀对公司的负面影响整体可控。**

## 二、地缘政治风险

### 2.1 美国对华半导体管制：系统性升级进行时

2026年，美国对华半导体管制进入"立法化+多边化"新阶段，是澜起科技面临的最核心外部风险：

| 管制工具 | 核心内容 | 对澜起科技的潜在影响 |
|:---|:---|:---|
| **MATCH法案**（2026年4月） | 强制盟友150天内对齐管制标准，禁运DUV光刻机及关键设备 | 间接影响：限制中国先进制程芯片制造能力，可能影响公司在中国本土的代工选择 |
| **EDA出口管制升级**（2026年5月） | 限制3D991/3E991类EDA软件对华出口 | 潜在影响：如果管制范围扩展至先进制程IP/PDK，可能限制公司向更先进工艺节点迁移的能力 |
| **《AI OVERWATCH法案》等6项法案** | 建立AI芯片出口"全域管制框架"，动态围堵中国算力获取 | 间接利好：管制越严，国产替代需求越强，作为国内唯一PCIe 5.0 Retimer供应商受益 |

### 2.2 公司的应对策略评估

澜起科技在地缘政治上的风险管理展现出较好的前瞻性：

1. **"绕开美国市场"的主动选择**：公司在北美市场的直接销售收入占比不足1%，主动避免了出口管制的直接冲击。核心客户集中于韩国（三星、SK海力士）和亚洲区域，有效规避了中美科技冲突的正面波及。

2. **暂缓AI芯片研发**：2024年Q4公司决定暂缓投入长达5年的AI算力芯片研发项目（IPO募投项目，原计划投入5亿元），将资源转移至PCIe Switch芯片。这一决策具有深刻的战略意义——AI芯片的先进制程（7nm以下）极度依赖台积电代工，在中美脱钩背景下存在供应链被切断的风险；而PCIe互连芯片可用更成熟的工艺节点实现，供应链安全性更高。

3. **A+H双平台架构**：2026年2月港股上市，不仅拓宽了融资渠道，更建立了"离岸资本平台"，为海外并购、引入国际人才、规避单一市场风险提供了战略弹性。

4. **新加坡亚太总部+首尔研发中心**："产业链跟随"策略使得公司在物理上嵌入韩国DRAM产业生态，即使中美关系进一步恶化，也不影响与核心客户的日常协作。

### 2.3 地缘风险的"双向不对称性"

经过审慎分析，澜起科技面临的地缘政治风险呈现**非对称性**——负面冲击可控，但正面受益可期：

**负面冲击（有上限）**：
- 最坏情况是中国芯片设计公司被全面限制使用美国EDA工具——这会影响先进制程设计，但澜起科技的内存接口芯片目前使用成熟工艺（台积电12nm/16nm等），受先进制程制裁的冲击有限；
- 晶圆代工被限制——公司Fabless模式下高度依赖台积电/中芯国际等代工厂，但内存接口芯片属于"非军事、非AI训练"的通用芯片品类，被直接列入实体清单的概率较低。

**正面受益（有空间）**：
- 信创/国产替代政策倾斜：在中国半导体自主可控的大背景下，澜起科技作为大陆唯一的PCIe 5.0 Retimer供应商和内存接口芯片全球龙头，将获得政策性采购优先；
- 中国大陆云厂商（阿里、腾讯、字节等）的AI资本开支增长，产生的国产替代需求直接利好澜起。

## 三、监管与产业政策

### 3.1 中国半导体产业政策：持续加码

| 政策方向 | 具体措施 | 对澜起科技的影响 |
|:---|:---|:---|
| **"十四五"集成电路专项** | 重点扶持EDA、芯片设计、先进封装等环节 | 澜起作为科创板龙头设计公司，享受研发费用加计扣除、人才引进补贴等优惠 |
| **大基金三期** | 规模约3000亿元，重点投向设备和材料 | 间接利好：制造能力的提升为设计公司提供更稳定的本土代工选项 |
| **国产替代采购政策** | 国企/政府项目优先采购国产芯片 | 直接利好：加快PCIe Retimer等产品在国内政企市场的渗透 |
| **科创板支持** | 允许未盈利企业上市、放宽股权激励 | 澜起2019年首批上市受益，但当前影响边际递减（已盈利） |

### 3.2 行业准入壁垒

内存接口芯片行业存在天然的"JEDEC标准+客户认证"双重准入壁垒，政策端不构成额外的限制。相反，中国政府对集成电路设计行业的扶持政策（进口设备免税、人才落户、研发补贴等）持续降低了公司的运营成本。

## 四、模块四核心结论

1. **利率环境对估值偏友好**，但科创板估值更多受市场情绪主导，需配合模块六的估值定价综合分析。

2. **地缘政治风险是公司长期最核心的外部变量**：美国对华半导体管制的"立法化+多边化"趋势不可逆转，但澜起科技的主动应对（绕开美国市场、暂缓AI芯片、A+H双平台、韩国生态嵌入）将负面冲击控制在可承受范围内。公司产品使用成熟工艺节点，受先进制程制裁的波及面有限。

3. **国产替代是确定性利好**：中美科技脱钩越深，澜起科技作为大陆唯一PCIe 5.0 Retimer供应商和内存接口芯片全球龙头的战略价值越凸显。中国云厂商的国产化采购正在打开一个"政策红利+市场红利"的双重窗口。

4. **需持续关注的三个政策变量**：①EDA工具出口管制是否扩展至IP授权领域（将直接影响先进SerDes IP的获取）；②美国是否将中国Fabless设计公司列入实体清单（目前概率低但非零）；③MATCH法案及后续立法的落实进度与盟友配合度。
