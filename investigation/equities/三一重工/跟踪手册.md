- 5月三大龙头（三一+5%、徐工+3-5%、柳工+5%）集体提价，**终结近3年价格战**

## 6. 跟踪检查清单 (Checklist)

### 6.1 每日检查项

- [ ] 股价是否触及关键支撑/阻力位（18.65元年线、19.50元阻力）
- [ ] 成交量是否异常（超过100万手或低于40万手）
- [ ] 北向资金流向（净流入/流出超过1亿元需关注）
- [ ] 人民币汇率（美元兑人民币中间价，突破6.80需高度关注）

### 6.2 每周检查项

- [ ] 周K线形态（是否形成止跌企稳信号）
- [ ] 原材料价格（铁矿石、钢材价格走势）
- [ ] 🆕 欧盟起重机反倾销调查进展（月度关注→周度关注）
- [ ] 主力资金流向（大单净流入/流出）

### 6.3 每月检查项

- [ ] 月度挖掘机销量数据（国内+出口）
- [ ] 🆕 提价终端传导效果验证（经销商实际成交价变化）
- [ ] 公司公告（订单、重大合同、产能进展）
- [ ] 欧盟/英国贸易政策动态

### 6.4 每季度检查项

- [ ] 季报业绩（营收、利润、毛利率、现金流）
- [ ] 应收账款与坏账计提情况
- [ ] 海外业务收入占比与增速
- [ ] 德国工厂产能利用率
- [ ] 全面更新估值模型

### 7.2 触发更新条件

**立即更新条件**：
- 欧盟发布整机关税裁定

- 月度销量数据发布
- 重大合同公告
