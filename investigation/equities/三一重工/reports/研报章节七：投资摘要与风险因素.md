## 1. 投资摘要 (Investment Summary)

## 3. 研究结论象限图 (Final Evaluation Matrix)

### 3.1 象限定位

基于"确定性"与"预期收益空间"两个维度，将三一重工定位于：

```
                    高预期收益空间
                           ▲
                           │
        第二象限           │           第一象限
        (Speculative       │           (High-Conviction Buy)
         Growth)            │            高确定性 + 高预期收益
        低确定性 + 高收益   │            ★ 核心重仓建议
                           │
◄──────────────────────────┼──────────────────────────►
低确定性                   │                    高确定性
                           │
        第四象限           │           第三象限
        (Avoid/Wait)       │           (Defensive Value)
        低确定性 + 低收益   │            高确定性 + 低预期收益
                           │            ★ 防御/底仓建议
                           │
                           ▼
                    低预期收益空间
```

**三一重工定位**：**第三象限偏第一象限 (Defensive Value → High-Conviction Buy 过渡)** —— **防御/底仓 → 适度增持**

### 3.2 定位理由

**确定性评估（高，较此前更高）**：
- ✅ 国内复苏超预期：4月挖机销量+29.8%，1-4月累计+22.2%
- ✅ 价格战正式终结：三巨头集体提价5%，行业从价格战切换至价值竞争
- ✅ 现金流质量：TTM经营现金流191亿元，覆盖净利润2.3倍

**预期收益空间评估（低→中改善）**：
- ⚠️ 估值已从高估切换至低估：PE_TTM从65%分位降至25.8%分位
- ⚠️ 目标价18.50元与现价基本持平，但估值回归中枢（PE 25x）对应约20%上行空间
- ⚠️ 最大不确定性：Q2汇率走势、欧盟起重机反倾销调查

### 3.3 投资策略建议

**适合的投资者类型**：
- ✅ 追求稳定分红的价值投资者（股息率2.2%）
- ✅ 对工程机械行业有长期配置需求的机构
- ✅ 能够承受短期波动、看重长期基本面的投资者

**不适合的投资者类型**：
- ❌ 追求高弹性收益的投机者
- ❌ 对回撤容忍度较低的投资者
- ❌ 希望短期（1-3个月）获得显著收益的投资者

**仓位建议**：
- **防御性底仓**：占总仓位5-10%，作为穿越周期的防御性配置
- **非进攻性品种**：不建议作为高弹性的进攻品种重仓配置
- **动态调整**：若股价回调至18-19元区间，可适度加仓至10-15%
