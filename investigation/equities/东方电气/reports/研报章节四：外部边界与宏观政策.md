# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月24日
**公司名称**：东方电气股份有限公司 (600875.SH)

---

## 一、宏观环境变量

### 1.1 利率环境与估值折现率

2025年中国人民银行实施适度宽松的货币政策，5月将7天期逆回购利率从1.50%调降至1.40%。当前10年期国债收益率在1.85%附近波动，与政策利率（OMO 1.40%）的利差约45BP，处于历史正常区间（40-70BP）的下限附近。

对估值的影响：低利率环境理论上支撑高PE估值（折现率低）。但当前10Y国债1.85%已处历史低位区间的下边界，市场普遍预期2026年利率中枢小幅上移至1.6-2.0%区间。假设2026年OMO再降10-20BP至1.20-1.30%，10Y国债中枢约1.8%，对东方电气DCF估值的WACC影响约30-50BP（对应估值弹性±5-8%）。**利率端整体偏正面但进一步大幅宽松空间有限，对估值上修弹性不大。**

### 1.2 汇率损益

境外营收占比仅6.54%（51.39亿，同比-7.71%），汇率直接影响有限。但公司2025年财务费用同比+164%至1.18亿，主因汇兑净损失增加。2025年国际市场新生效合同超140亿元（占新签~12%），随着出海加速，汇率风险敞口将逐步扩大。

### 1.3 通胀与原材料传导

电力装备主要原材料为钢材（特种钢、硅钢、合金钢）、铜等。2025年大宗商品价格总体平稳，钢材价格较2022年高点显著回落，有利于公司毛利率修复。大型电站设备交付周期长达12-24个月，原材料价格波动存在6-12个月的传导滞后期。2026年原材料端大概率保持平稳，不会成为毛利率的主要拖累因素。

## 二、地缘政治风险评估

### 2.1 海外业务与地缘敏感度

| 维度 | 现状 | 风险评估 |
|:---|:---|:---|
| 海外营收占比 | 6.54%（51亿），2025年-7.7% | 低依赖度 |
| 海外新签占比 | ~12%（140亿/1172亿） | 中等，趋势上升 |
| 主要市场 | "一带一路"沿线（中亚、中东、东南亚、非洲） | 地缘风险可控 |
| 涉美业务 | 极少 | 制裁风险极低 |
| 海外营收地域 | 传统EPC项目为主 | 波动性大 |

**核心判断**：东方电气海外业务高度集中于"一带一路"沿线，避免了欧美市场的贸易摩擦风险。公司是典型的"内循环为主+一带一路出海"模式，中长期受益于新兴市场能源基建需求（全球仍有7.6亿人无电可用），地缘政治风险整体可控。

### 2.2 供应链重构风险 (China+1)

电力装备属于资本密集型重工业，生产高度依赖国内完整的重装产业链（大型铸锻件、特种钢材、精密加工），短期内不具备向海外转移制造基地的经济可行性。东方电气的制造基地集中在中国（四川德阳、新疆等），不受"供应链重构"趋势的直接影响。

### 2.3 贸易摩擦与出口管制

| 风险类型 | 评估 |
|:---|:---|
| 关税/反倾销 | 极低（产品不出口欧美） |
| 技术出口管制 | 极低（核心技术均为自主研发） |
| 制裁/次级制裁 | 低（无涉俄罗斯等制裁热点地区大额业务） |
| CFIUS/投资审查 | 不适用（无在美投资） |
| 知识产权纠纷 | 低（燃机/核电均为正向自主研发） |

**关键优势**：G50重型燃机100%正向自主研发，不存在与GE/西门子/三菱的知识产权纠纷风险。相较于上海电气（与安萨尔多合作）和哈尔滨电气（与三菱合作），东方电气的自主可控属性使其在技术封锁环境下享有独特优势。

## 三、监管与产业政策

### 3.1 国家发展战略契合度：极高

| 政策/规划 | 核心内容 | 对东方电气的影响 |
|:---|:---|:---|
| **"十五五"规划纲要**（2026.3） | "非化石能源十年倍增行动"，核电1.1亿kW、海上风电1亿kW+、抽水蓄能1亿kW | 直接利好核电/风电/水电装备需求 |
| **《电力装备行业稳增长方案》**(2025-2026) | 龙头企业年均营收增速目标10%+，传统电力装备年均6%+ | 政策明确支持，提供增长底线 |
| **《新一代煤电升级方案》**(2025-2027) | 深调峰/快调节/宽负荷/高韧性/低排放六维升级 | 煤电改造订单替代新建订单，平滑周期波动 |
| **碳达峰/碳中和**（2030） | 2030年非化石能源发电量占比超50% | 加速火电→新能源切换，利好核电/风电/储能 |
| **国资委央企考核** | "一利五率"考核体系 | 推动公司改善经营现金流、ROE等核心指标 |

**政策红利量化**：
- 核电：2022-2025年已核准41台机组，总投资超8,000亿，东方电气占设备份额约33%（每台~13亿订单），对应在手+潜在订单约380亿
- 抽水蓄能：2030年目标装机1.2亿kW（当前约0.67亿kW），对应增量投资约3,000亿，东方电气水电机组市占率约40%
- 煤电改造：2025-2030年市场空间约800亿，东方电气占25-30%份额

### 3.2 行业准入壁垒

| 壁垒类型 | 强度 | 说明 |
|:---|:---|:---|
| 核安全资质（NNSA） | ★★★★★ | 核级设备制造许可证，仅有少数央企持有 |
| 大型工程资质 | ★★★★ | EPC总包资质、海外工程经验积累 |
| 技术专利壁垒 | ★★★★ | G50燃机13年正向研发积累，不可复制 |
| 客户关系壁垒 | ★★★★ | 与五大发电集团的60年+绑定关系 |
| 资本壁垒 | ★★★ | 单台核电机组设备投资200亿，重资产门槛 |

### 3.3 环保约束与ESG

- 正面：主力产品（核电、风电、水电）属于"绿色能源装备"，受益于全球能源转型趋势
- 中性：煤电设备仍有较大体量（营收占31%），但在"双碳"目标下市场份额逐步被新能源产品替代
- 负面风险：风电叶片回收、退役火电机组处理等环境责任尚未形成明确成本计提机制

### 3.4 税收政策

- 高新技术企业享受15%的所得税优惠税率
- 研发费用加计扣除政策（100%加计）持续受益，2025年研发费用32.61亿对应税收减免约4.9亿
- 风电/光伏装备出口退税政策支持海外竞争力

## 四、外部环境综合评分卡

| 维度 | 评估 | 评分（-5至+5） |
|:---|:---|:---|
| 利率/折现率 | 低利率环境支撑估值，但进一步宽松空间有限 | +1 |
| 汇率 | 境外占比低，汇兑损失可控但在增长 | -1 |
| 通胀/原材料 | 钢材价格回落，有利于毛利率 | +2 |
| 地缘政治风险 | "一带一路"集中于非敏感区域，自主可控降低制裁风险 | +1 |
| 贸易摩擦 | 基本不受欧美贸易壁垒影响 | +2 |
| 供应链重构 | 重装制造业不受China+1影响 | +2 |
| 国家战略契合度 | 核电/燃机/抽蓄均为"十五五"重点方向 | +5 |
| 政策监管 | 电力装备稳增长方案提供直接支持 | +4 |
| **综合** | **政策面极度友好，外部风险可控** | **+16** |

**核心结论**：东方电气处于中国产业政策的绝对核心位置——"十五五"规划的能源安全+装备制造自主可控两大战略在此交汇。公司海外业务的地缘政治敏感度极低（无欧美业务），且核心产品均为正向自主研发（不受技术封锁限制）。最大的外部不确定性来自国内煤电订单的退潮节奏，但这属于产业周期的内生变量而非外部冲击。
