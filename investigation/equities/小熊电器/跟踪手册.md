# 小熊电器 (002959) 投研跟踪手册 (买方修订版)

**更新日期**：2026年04月08日
**核心逻辑**：利润优先 + 出海对冲 + AI 战略转型 (待验证)

---

## 1. 核心逻辑验证点 (Thesis Validators)

每季财报需重点核对的定量指标：
- [ ] **总营收增速**：**核心红线 0%**。若总营收出现同比下滑，说明出海增量无法对冲内需萎缩。
- [ ] **海外营收占比**：观察是否向 20% 以上稳步迈进。
- [ ] **综合毛利率**：警戒线 34%。关注提价策略在原材料暴涨背景下的执行力。
- [ ] **AI 新品贡献度**：监控“AI 智享系列”在 618、双 11 等大促中的 GMV 占比。

## 2. 关键外部变量 (Exogenous Watchlist)

需实时关注的外部数据：
- [ ] **铜/塑料现货价**：关注 2026Q2 是否出现见顶回落信号。
- [ ] **美国 Section 122 关税**：监控 7 月份关税到期后的政策演向。
- [ ] **百度文心大模型合作**：关注 AI 语音交互在小家电上的真实用户活跃度 (DAU)。

## 3. 逻辑证伪警报 (Invalidation Triggers) —— **重点修正**

一旦出现以下信号，必须触发减仓或清仓：
- [ ] **信号 A (核心证伪)**：总营收连续两季出现 **5% 以上的同比下滑**。
- [ ] **信号 B (盈利失守)**：扣非净利润出现同比负增长（说明利润优先战略失效）。
- [ ] **信号 C (机构撤离)**：2026Q1/Q2 基金持仓占比出现断崖式下跌（显示主流资金放弃博弈）。
- [ ] **信号 D (大客户流失)**：罗曼智能丢失飞利浦或松下的核心订单份额。

## 4. 关键时间节点 (Calendar)

- **2026年4月下旬**：2026 一季报披露（**关键点：营收是否破位**）。
- **2026年6月**：618 大促观察（验证 AI 新品接受度）。
