## 一、价值锚定：适配性方法选择

### 1.1 方法论筛选

先惠技术的商业模式特征：高成长装备+结构件双轮驱动、项目制收入确认、利润刚进入加速释放期、自由现金流刚转正但稳定性不足。

| 估值方法 | 适配性 | 理由 |
|:---|:---|:---|
| **PE（市盈率）** | ★★★★★ 首选 | 利润增长确定性强，EPS可预测度高，行业横向可比性佳 |
| **PEG（成长性估值）** | ★★★★ 辅助 | 高增速匹配高PE的合理性检验 |
| **SOTP（分部估值）** | ★★★★ 辅助 | 业务属性差异大（装备 vs 结构件），分部定价更精准 |
| **DCF（现金流折现）** | ★★ 参考 | 自由现金流波动大（2022-2023大额为负→2024暴增→2025回归），FCFF预测可靠性低 |
| **PB-ROE** | ★★ 参考 | ROE 16.5%合理，但轻资产服务属性使得PB的行业可比性弱 |
| **EV/EBITDA** | ★★★ 参考 | 可排除折旧和资本结构差异，但行业可比公司数据不全 |

**核心锚定**：以PE为估值主锚、PEG为成长性约束、SOTP为下限校验。

## 三、估值全变量对冲审计矩阵（强制性输出）

### 3.1 估值影响力过滤

在进入正式审计前，先按框架要求进行**审慎思考**：前序模块发掘的利好/利空中，哪些是真正能改变估值中枢的因子？哪些只是"平庸的关联"？

**评估为"平庸关联"（对估值影响=0）的因子**：
- 公司是科创板智能制造龙头 → 标签本身不产生溢价，业绩才是定价锚
- 固态电池设备布局 → 远期期权，尚未进入估值折现期
- 中国制造业升级大趋势 → 宏观叙事不影响个股定价，同行业公司享有相同的贝塔

**评估为"估值锚点"（需计入调节）的因子**：
- 客户集中度（77%来自CATL）→ 构成确定性的折价（见内部DNA维度）
- 海外高毛利持续性与可持续性 → 影响长期盈利中枢假设
- 固态电池设备布局 → 虽远期但属于稀缺的"正确期权"

### 3.3 修正估值

**修正后PE**：24.9x
**修正目标市值**：5.07亿 × 24.9 = **126.2亿**
**修正目标价（不复权）**：126.2 / 1.2644 = **99.82元**

在全面审计后，**目标价从基准120.30元下修至99.82元**。核心压力来自：CATL单客户依赖的估值折价（-15%）是最具说服力的调节项，其他因子正负相抵后整体偏负。

**价格下行后的贝叶斯更新——关键变化**：
- 5月11日：当前价103.60元 vs 修正目标价99.82元 → 高估3.6%，**盈亏比-0.04（不利）**
- 5月27日：当前价88.25元 vs 修正目标价99.82元 → **低估13.1%，盈亏比+0.56**
- **6月3日**：当前价82.25元 vs 修正目标价99.82元 → **低估21.4%，盈亏比+1.07**
- **价格从103.60→88.25→82.25的三阶段回调已充分释放双顶风险，盈亏比从负值→+0.56→+1.07，风险收益比已进入高性价比区间（>1.0）**
- **价格回调已将CATL依赖折价（-15%）大幅消化。当前82.25元对应的隐含PE仅20.5x（2026E），已显著低于修正PE 24.9x**，意味着市场对CATL依赖的定价折价已经过度反映，估值修复空间充分。

## 五、逻辑证伪与极限压力测试

### 5.1 "一击毙命"黑天鹅情景

**定义**：宁德时代因战略调整（如转向自研PACK线或引入新核心供应商），将先惠技术在模组/PACK设备和结构件的供应份额削减50%。

**影响路径**：
1. 设备订单断崖→2026H2起新签订单归零→2027年设备营收从17亿骤降至5亿
2. 结构件出货减半→从26亿降至13亿
3. 叠加产能闲置带来的固定成本无法摊销→毛利率崩溃
4. **地狱情景2027年营收**：结构件13亿+设备5亿=18亿
5. **地狱情景2027年净利润**：18亿 × (15%综合GM - 15%固定费率) × 0.85 = **接近于零或微亏**

**地狱价格**：基于重资产清算价值
- 每股净资产（2025年末）：18.25元
- 扣除商誉7.57亿（CATL关系崩塌必然伴随商誉全额减值）→调整后BVPS = (23.08 - 7.57) / 1.2644 = 12.27元
- 扣除应收账款减值30%（CATL不续约→部分应收可能无法回收）→再减2.63元 → **极限底价约9.64元/股**
- 对应市值：12.2亿

**这是概率极低（<5%）的尾部风险，但一旦发生，股价将从103元跌至约10元，跌幅约90%。**

### 5.2 极限支撑位审计

- 净现金价值（货币资金8.11亿 - 有息负债6.24亿）：1.87亿 → 每股约1.48元（底线支撑）
- 谨慎支撑：PB 1.0x → 18.25元/股（所有资产按账面价值清算）
- 乐观支撑：PB 2.0x → 36.50元/股（历史最低PB 2.10x附近）
