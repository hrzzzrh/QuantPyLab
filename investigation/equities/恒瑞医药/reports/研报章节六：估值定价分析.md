## 一、价值锚定：适配性方法选择

### 1.1 方法论选择理由

恒瑞医药属于**高成长+高盈利创新药平台型公司**，现金充裕（405亿）、零负债、创新药占比持续提升。适用的估值方法：

| 方法 | 适用性 | 选用理由 |
| :--- | :--- | :--- |
| **PE（市盈率）** | ★★★★★ 首选 | 盈利稳定、可预测、行业通用。当前PE处于历史低分位，有均值回归逻辑 |
| **DCF（现金流折现）** | ★★★★ 验证 | 高确定性现金流（OCF/NP 1.46x），适合作为内在价值底线 |
| **PB-ROE** | ★★ 参考 | 资产较轻，大量现金拖累了PB的有效性。仅作极端情境参考 |
| **PEG** | ★★★ 辅助 | 20%的利润增速对应PEG 2.0-2.5，处于合理偏贵区间 |
| **EV/EBITDA** | ★★★ 辅助 | 剔除冗余现金影响后，反映真实经营资产估值 |

### 1.2 历史估值分位审计

**核心发现**：当前PE 45.87处于**所有时间窗口的历史最低区域**（仅略高于1年最低43.16）。这意味着市场正在以"最悲观"的预期为恒瑞定价。

但必须谨慎解读：A股历史PE包含2020-2021年医药板块整体泡沫（恒瑞PE一度高达105x）。10年中位数61.85不应简单作为"合理估值"锚点——历史分位在A股这类新兴市场中的参考意义需要打折（如框架所强调）。

**更合理的锚点**：
- 2021年后医药板块去泡沫完成，2023-2025年PE运行区间约50-70x
- 当前45.87低于去泡沫后的常态区间下限

## 二、核心：估值全变量对冲审计矩阵

### 2.1 基准市值计算

**PE法（首选）**：

| 情景 | 2026E EPS | 适用PE | 目标价（元/股，不复权）| 目标市值（亿）| 相对现价涨跌幅 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 保守 | 1.13 | 42x | 47.46 | 3,150 | -15.4% |
| **中性** | **1.40** | **52x** | **72.80** | **4,832** | **+29.8%** |
| 乐观 | 1.66 | 58x | 96.28 | 6,390 | +71.6% |

**PE倍数选择逻辑**：
- 42x（保守）：接近历史5年最低33x，对应增长停滞/GLP-1失败情景
- 52x（中性）：低于历史5年中位数57x但高于当前45.87。考虑到9号文政策红利+创新药占比提升+零杠杆财务质量，适当溢价合理
- 58x（乐观）：略高于5年中位数，反映GLP-1爆款+全球化的估值溢价

**DCF法（验证）**：

| 参数 | 假设值 | 逻辑 |
| :--- | :--- | :--- |
| 基期FCF | 100亿（2026E）| 基于93亿NP×1.07转换率（低于2025年1.46x，保守化处理）|
| 5年CAGR | 18%→15%→12%→10%→8% | 逐步下滑的增长曲线 |
| WACC | 8.5% | 零负债+β约0.9+股权风险溢价6%+无风险利率2.5% |
| 终值增长率 | 3.0% | 老龄化驱动的长期增长 |
| 净现金 | 405亿 | 2026Q1数据 |

计算结果：内在价值约**3,400亿**，对应**51.22元/股**。

DCF估值（51.22元）低于PE法（72.80元），差距来自：
1. DCF对未来5年增速的保守假设（均值回归逻辑）
2. PE法隐含了对"政策红利+管线期权"的估值溢价
3. 两者差距≈市场的"创新期权价值"

### 2.2 全变量对冲审计矩阵

以中性基准市值**4,832亿**（72.80元）为底色，进行定性转定量调节：

| 审计维度 | 核心审计因子 | **审慎思考（估值影响力过滤）** | 调节建议 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质** | 管线纵深壁垒（24款已上市+100+临床）；商业化体系不可复制（9,000人、22,000家医院）；战略定力（50年只做药，不跨界） | **溢价。** 管线纵深确实提供了抗单品风险的能力；商业化体系在中国药企中独一无二。这些是能改变估值中枢的稀缺特质。但要警惕"平庸的优秀"——管线虽多，缺乏全球重磅炸弹 | **+5%** | 管线矩阵的确定性溢价，抵消单品爆款的缺乏。类似于"宽基指数"的估值逻辑——低波动高确定性 |
| **宏观：政策与战略对齐** | 9号文自主定价；产业定位升级为"支柱"；创新药不集采；商保目录落地 | **溢价。** 这是真正能改变估值中枢的政策。从"集采受害标的"变为"创新受益标的"的认知切换尚未被市场充分定价。9号文标志着制度拐点 | **+8%** | 政策从"控费优先"→"创新优先"，这是行业估值体系的重估，非个股事件。恒瑞作为最大受益者，享受β+α |
| **宏观：地缘与外部约束** | BIOSECURE已立法（2027年后生效）；FDA数据禁令潜在风险；全球供应链重构 | **折价。** BIOSECURE不直接冲击恒瑞（非CXO），但地缘政治不确定性抑制了海外估值溢价。市场对"中国药企出海"的信任折价是真实存在的 | **-3%** | 将无法量化的地缘风险转化为PE倍数的系统性折价。但恒瑞"BD+NewCo"模式的隔离效果可部分对冲 |
| **市场：流动性与风格** | A+H双上市带来的流动性提升；当前市场风格偏防御（红利/价值）；创新药板块资金回流趋势 | **中性偏正。** H股上市确实带来增量资金（南下资金），但港股估值低于A股形成锚定压制。创新药板块经过2年调整已具吸引力，资金回流趋势正在形成 | **+2%** | 双重上市提升国际可见度，港股定价可能逐渐上拉A股估值而非下拉 |
| **综合调节系数** | — | — | **+12%** | — |

### 2.4 锚点回测

修正后隐含PE = 5,412 / 93 = **58.2x**（2026E）
- vs 历史5年中位数 56.88x → **略高于中位数，合理**
- vs 当前实际PE 45.87x → **显著高于当前，需要逻辑支撑**

**逻辑支撑**：
1. 9号文是制度性拐点（不是短期事件），市场尚未完成估值重定价
2. 创新药占比从58%→65%的结构升级，盈利质量系统性提升
3. 恒瑞的"宽基指数"特性（管线分散、抗风险）在不确定性环境中应享受确定性溢价
4. 零杠杆+405亿现金的安全垫赋予了其他药企不具备的下行保护

## 四、逻辑证伪与极限压力测试

### 4.1 黑天鹅场景——"一击毙命"事件

**场景定义**：GLP-1瑞普泊肽被NMPA拒绝批准（安全性质疑或数据不达标），同时核心抗肿瘤品种遭遇临床失败或严重不良反应召回。

**后果推演**：
- 创新药增长引擎熄火，2026E创新药收入增速降至10%以下
- 市场对管线兑现能力的信任崩溃，PE压缩至35-40x（接近历史最低33x）
- BD交易对手方退出或重新谈判
- 极端目标市值：1,500-2,000亿（对应股价22.60-30.12元）

### 4.2 极限支撑位审计

**基于净现金价值**：
- 净现金：405亿
- 每股净现金：405/66.37 = 6.10元
- 仿制药清算价值：约118亿收入×1x PS = 118亿 → 1.78元/股
- 创新药管线残值：约1,500亿（BD市场可比交易估值）→ 22.60元/股
- **极限支撑价**：6.10 + 1.78 + 22.60 ≈ **30.48元**

即在最极端情景下（公司被完全清算），理论底价约为30元（对应PE约22x，PB约3.2x）。

| 指标 | 数值 |

恒瑞当前估值处于10年历史最低分位，市场定价隐含了对"创新药高增长不可持续"的悲观预期。但基本面的现实是：创新药占比突破58%（2026Q1 62%）、净利率持续上行（28%）、9号文开启政策黄金期。PE从45.87修复至58x（仍低于10年中位数62x）的路径依赖以下逻辑兑现：（1）GLP-1获批并开启放量；（2）2026年全年创新药增长超30%的目标达成；（3）BD交易持续验证管线价值。若任一条件落空，上行空间将显著压缩。
