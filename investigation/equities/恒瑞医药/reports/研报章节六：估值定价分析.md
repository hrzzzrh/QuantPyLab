## 一、价值锚定：适配性方法选择

### 1.1 方法论选择理由

恒瑞医药属于**高成长+高盈利创新药平台型公司**，现金充裕（405亿）、零负债、创新药占比持续提升。适用的估值方法：

| 方法 | 适用性 | 选用理由 |
| :--- | :--- | :--- |
| **PE（市盈率）** | ★★★★★ 首选 | 盈利稳定、可预测、行业通用。当前PE处于历史低分位，有均值回归逻辑 |
| **DCF（现金流折现）** | ★★★★ 验证 | 高确定性现金流（OCF/NP 1.46x），适合作为内在价值底线 |
| **PB-ROE** | ★★ 参考 | 资产较轻，大量现金拖累了PB的有效性。仅作极端情境参考 |
| **PEG** | ★★★ 辅助 | 20%的利润增速对应PEG 2.0-2.5，处于合理偏贵区间 |
| **EV/EBITDA** | ★★★ 辅助 | 剔除冗余现金影响后，反映真实经营资产估值 |

**核心发现**：PE在6月8日触及**37.55**——1年和3年窗口的绝对历史最低值（此前最低均为38.12），距5年/10年最低33.33仅11.2%空间。随后在6月12日反弹至39.64。自2021年医药板块泡沫破裂以来，恒瑞PE从未在1年和3年窗口内降至37.55水平。当前39.64虽较37.55有所回升，但仍处于所有时间窗口的历史极端低位。

但必须谨慎解读：A股历史PE包含2020-2021年医药板块整体泡沫（恒瑞PE一度高达105x）。10年中位数61.85不应简单作为"合理估值"锚点——历史分位在A股这类新兴市场中的参考意义需要打折（如框架所强调）。

**更合理的锚点**：
- 2021年后医药板块去泡沫完成，2023-2025年PE运行区间约50-70x
- 当前39.64远低于去泡沫后的常态区间下限
- 6月8日PE触及37.55后反弹至39.64，表明此区域有估值买盘支撑
- 距5年/10年最低33.33仍有约15%下行空间（以39.64计），但PE越低、压缩空间越小、反弹概率越大

## 二、核心：估值全变量对冲审计矩阵

### 2.1 基准市值计算

**PE法（首选）**：

| 情景 | 2026E EPS | 适用PE | 目标价（元/股，不复权）| 目标市值（亿）| 相对现价涨跌幅 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 保守 | 1.18 | 42x | 49.56 | 3,289 | +0.7% |
| **中性** | **1.57** | **52x** | **81.64** | **5,419** | **+65.9%** |
| 乐观 | 1.85 | 58x | 107.30 | 7,121 | +118.1% |

**PE倍数选择逻辑**：
- 42x（保守）：接近历史5年最低33x，对应增长停滞/GLP-1失败情景
- 52x（中性）：低于历史5年中位数57x但远高于当前39.64。考虑到9号文政策红利+创新药占比提升+零杠杆财务质量，适当溢价合理
- 58x（乐观）：略高于5年中位数，反映GLP-1爆款+全球化的估值溢价

**DCF法（验证）**：

| 参数 | 假设值 | 逻辑 |
| :--- | :--- | :--- |
| 基期FCF | 110亿（2026E）| 基于104亿NP×1.06转换率（低于2025年1.46x，保守化处理）|
| 5年CAGR | 18%→15%→12%→10%→8% | 逐步下滑的增长曲线 |
| WACC | 8.5% | 零负债+β约0.9+股权风险溢价6%+无风险利率2.5% |
| 终值增长率 | 3.0% | 老龄化驱动的长期增长 |
| 净现金 | 405亿 | 2026Q1数据 |

计算结果：内在价值约**3,740亿**，对应**56.34元/股**。

DCF估值（56.34元）低于PE法（81.64元），差距来自：
1. DCF对未来5年增速的保守假设（均值回归逻辑）
2. PE法隐含了对"政策红利+管线期权"的估值溢价
3. 两者差距≈市场的"创新期权价值"

### 2.2 全变量对冲审计矩阵

以中性基准市值**5,419亿**（81.64元）为底色，进行定性转定量调节：

| 审计维度 | 核心审计因子 | **审慎思考（估值影响力过滤）** | 调节建议 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质** | 管线纵深壁垒（24款已上市+100+临床）；商业化体系不可复制（9,000人、22,000家医院）；战略定力（50年只做药，不跨界） | **溢价。** 管线纵深确实提供了抗单品风险的能力；商业化体系在中国药企中独一无二。这些是能改变估值中枢的稀缺特质。但要警惕"平庸的优秀"——管线虽多，缺乏全球重磅炸弹 | **+5%** | 管线矩阵的确定性溢价，抵消单品爆款的缺乏。类似于"宽基指数"的估值逻辑——低波动高确定性 |
| **内部：BD/MNC背书** | BMS deal（全球Top5药企，6亿首付）；验证管线全球价值及BD能力 | **溢价。** MNC最高级别背书直接验证了恒瑞管线的国际竞争力。6亿首付金额在中国药企出海BD中属于第一梯队，信号效应强于金额本身。BMS的尽调深度是市场无法复制的"免费"质量审计 | **+3%** | MNC背书是最具确定性的质量信号，有助于缩小A股与国际创新药估值体系的折价 |
| **宏观：政策与战略对齐** | 9号文自主定价；产业定位升级为"支柱"；创新药不集采；商保目录落地 | **溢价。** 这是真正能改变估值中枢的政策。从"集采受害标的"变为"创新受益标的"的认知切换尚未被市场充分定价。9号文标志着制度拐点 | **+8%** | 政策从"控费优先"→"创新优先"，这是行业估值体系的重估，非个股事件。恒瑞作为最大受益者，享受β+α |
| **宏观：地缘与外部约束** | BIOSECURE已立法（2027年后生效）；BINSA法案（6月2日提交，两党正式提案，点名BMS-恒瑞deal）；FDA数据禁令潜在风险；全球供应链重构 | **折价。** 5月26日时地缘风险评估基于Moolenaar"建议函"（无法律效力），折价-3%。6月2日BINSA作为正式两党立法提案提交，风险等级发生质变。但三对冲因素仍在：（1）BMS deal大概率受祖父条款保护；（2）Kailera模式天然隔离GLP-1全球价值；（3）欧洲/日本BD不受影响。综合评估：地缘折价加深但非致命。 | **-5%**（较5月-3%加深）| BINSA从"建议函"升级为"正式法案"，增加了不确定性折价。但需防范"过度折价"——恒瑞99%收入在国内，BD收入的"归零"概率极低，当前市场可能已过度定价此风险 |
| **市场：流动性与风格** | A+H双上市带来的流动性提升；当前市场风格偏防御（红利/价值）；创新药板块资金回流趋势 | **中性偏正。** H股上市确实带来增量资金（南下资金），但港股估值低于A股形成锚定压制。创新药板块经过2年调整已具吸引力，资金回流趋势正在形成 | **+2%** | 双重上市提升国际可见度，港股定价可能逐渐上拉A股估值而非下拉 |

### 2.4 锚点回测

修正后隐含PE = 6,123 / 104 = **58.9x**（2026E）

**逻辑支撑**：

2. 创新药占比从58%→65%的结构升级，盈利质量系统性提升

## 四、逻辑证伪与极限压力测试

### 4.1 黑天鹅场景——"一击毙命"事件

**场景定义**：GLP-1瑞普泊肽被NMPA拒绝批准（安全性质疑或数据不达标），同时核心抗肿瘤品种遭遇临床失败或严重不良反应召回。

**后果推演**：
- 创新药增长引擎熄火，2026E创新药收入增速降至10%以下
- 市场对管线兑现能力的信任崩溃，PE压缩至35-40x（接近历史最低33x）
- BD交易对手方退出或重新谈判
- 极端目标市值：1,500-2,000亿（对应股价22.60-30.12元）

### 4.2 极限支撑位审计

**基于净现金价值**：
- 净现金：405亿
- 每股净现金：405/66.37 = 6.10元
- 仿制药清算价值：约118亿收入×1x PS = 118亿 → 1.78元/股
- 创新药管线残值：约1,500亿（BD市场可比交易估值）→ 22.60元/股
- **极限支撑价**：6.10 + 1.78 + 22.60 ≈ **30.48元**

即在最极端情景下（公司被完全清算），理论底价约为30元（对应PE约22x，PB约3.2x）。

6月8日PE触及37.55——1年和3年窗口的绝对历史最低值，距5年/10年最低33.33仅11.2%空间。随后6月12日放量反弹至48.49元（PE 39.64），技术面出现本轮调整的首个有效底部信号（地量473K+放量1003K反弹+收复48.00趋势线）。正面催化剂持续积累：ASCO 2026公布了瑞康曲妥珠单抗CRC III期阳性结果并获优先审评、ADA 2026确认瑞普泊肽三项优异数据、SHR4640（首个国产URAT1抑制剂）首批全国发货、三个新分子获临床试验批件（HRS-6209/HRS-8080/HRS-9813）、2026医保预申报窗口于6月10日关闭。负面因素方面，BINSA法案面临国会立法效率低下的现实阻碍（选举年+历史最低立法产出率），其在2026年内通过的概率已从30%下调至约25%；且RA Capital、PhRMA等大型机构正积极游说反对。目标价92.24元维持不变（EPS及调节系数未调整），当前48.49元对应上涨空间+90.2%。当前估值已属历史极端水平，基本面催化剂（GLP-1审批+中报+BMS交割）密集集中于未来2-4个月，且技术面已出现底部信号——这可能是近年来最接近"估值底+技术底+催化剂"三重共振的配置时点。
