# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月06日
**标的**：恒瑞医药（600276.SH / 1276.HK）
**目标年份**：2026年（当年）

**前序核心结论回顾**：
- 模块一：创新药占比58%，净利率28%，零负债，OCF质量极优
- 模块二：创新药是"总量收缩中的结构性牛市"，GLP-1赛道面临供给洪峰
- 模块三：管线纵深国内第一，商业化体系为不可复制壁垒，但全球化能力弱于百济

## 一、宏观环境变量

### 1.1 人口结构与需求基本盘

中国正处于不可逆的老龄化加速期：

| 指标 | 数值 | 趋势 |
| :--- | :--- | :--- |
| 65岁以上人口 | 2.24亿（15.9%）| 持续上升，预计2030年超20% |
| 年医疗总诊疗人次 | 101.1亿（2024年）| +20.07% YoY |
| 居民人均可支配收入 | 43,377元 | +5.0%（名义增长）|
| 癌症年新发病例 | ~482万（2022年数据）| 随老龄化持续攀升 |

**对恒瑞的影响**：人口老龄化是最确定的长期宏观利好。癌症发病率与年龄强相关（老年发病率是中年人的5-10倍），慢性病（糖尿病、心血管疾病、自免疾病）同样是老龄化的"影子需求"。恒瑞横跨肿瘤与慢病的管线矩阵，恰好全面覆盖了老龄化带来的药品需求增量。

### 1.2 利率与折现率对估值的影响

当前中国处于宽松货币周期：
- LPR（贷款市场报价利率）持续下行的趋势支撑长期资产估值
- 对DCF估值的影响：无风险利率下降 → WACC下降 → 长期高增长标的的终值（Terminal Value）占比上升
- 恒瑞作为"长期确定性增长"型标的，是利率下行的核心受益者

**量化估算**：若10年期国债收益率从2.5%降至2.0%，恒瑞DCF估值中的终值将提升约8-12%。

### 1.3 汇率损益

恒瑞的海外收入几乎全部来自BD授权（33.92亿，占总营收约10.7%），以美元/欧元计价：
- **人民币贬值（当前情境）**：BD收入折算为人民币增加，利好利润表
- **人民币升值**：BD收入折算减少，但影响可控（占比仅10%）
- H股价格波动：恒瑞A+H两地上市，港股价格受汇率和全球流动性影响

**结论**：汇率对恒瑞经营的影响轻微，不足以构成核心变量。

## 二、监管与产业政策——历史性利好

### 2.1 产业地位升级：从"新兴"到"支柱"

2026年政府工作报告将生物医药的定位从"新兴产业"升级为**"新兴支柱产业"**，与集成电路、航空航天、低空经济并列。这是自2015年药品审评审批改革以来，中央层面对医药产业给出的最高战略定位。

**政策信号解读**：
- "支柱"意味着政策目标从"规范行业"转向"做大产业"
- 生物医药被纳入国家战略竞争的核心赛道
- 预期后续将有系统性产业扶持政策出台（税收优惠、专项基金、人才计划等）

### 2.2 药品价格形成机制改革（国务院9号文）

2026年4月14日，国务院办公厅印发《关于健全药品价格形成机制的若干意见》（9号文），实现了从"单一控价"到"多元平衡"的战略转变：

| 政策要点 | 具体内容 | 对恒瑞的影响 |
| :--- | :--- | :--- |
| **创新药自主定价** | 高水平创新药可"自主合理定价"，在上市初期保持价格相对稳定，接受社会监督和同行评议 | **核心利好**：定价权部分回归企业。此前"灵魂砍价"的预期被正式打破。恒瑞的创新药将享有"价格稳定期"保护 |
| **医保谈判更趋平衡** | "兼顾患者获益和鼓励创新"，不再单纯追求最低价，综合首发价格合理性、临床价值等因素 | 恒瑞每年有多款创新药进入医保谈判，政策转向将改善"以价换量"的性价比 |
| **创新药不集采** | "政策口径清晰：创新药不集采，集采非创新" | 仿制药继续承压，但创新药获得制度性安全垫 |
| **多元支付体系** | 基本医保+商业健康保险创新药品目录+公益慈善，"1+3+N"多层次保障 | 拓宽创新药支付渠道，特别是高价创新药的支付天花板将被打开 |
| **商保创新药目录** | 2025年12月首次发布商业健康保险创新药品目录 | 历史性突破——商保正式纳入创新药支付体系 |
| **真实世界研究** | 建立药品医保价值评估制度，允许基于真实世界数据调整价格 | 为已上市创新药提供价格上调路径（罕见但有理论上可能）|

**核心判断**：9号文是**中国创新药产业的政策拐点**。它标志着监管层正式承认"高价值创新必须获得高回报"，从"控费优先"转向"提质+控费+创新"并重。对恒瑞这种创新药收入占比58%+的企业而言，政策环境从未如此友好。

### 2.3 审评审批持续提速

- 创新药IND审批时限：目标压缩至**30个工作日**
- 2025年NMPA批准创新药76个（同比+58%），创历史新高
- 创新药从获批上市到纳入医保目录：平均周期从5年缩短至**1年左右**
- CDE突破性治疗品种、优先审评品种的认定数量和速度均显著提升

恒瑞的瑞康曲妥珠单抗（HER2 ADC）已获**11次CDE突破性治疗品种认定**，为获批加速提供了绿色通道。

### 2.4 2026年医保目录调整机制——预申报与转轨通道

2026年5月9日，国家医保局发布《2026年国家基本医疗保险、工伤保险和生育保险药品目录调整工作方案（征求意见稿）》，引入两项突破性机制：

**（1）"预申报"制度（首次引入）**
完成CDE技术审评但尚未正式获批的创新药，可预先申报进入医保目录。这意味着创新药可在获得NMPA批文之前就启动医保谈判流程，大幅缩短"获批→进保"的时间窗口。对恒瑞管线中处于审评末期的品种（如瑞康曲妥珠单抗等），预申报机制有望使新药从获批到纳入医保的时间压缩至半年以内，显著加速商业化放量节奏。

**（2）商保→医保"转轨"通道**
2025年已纳入商业健康保险创新药品目录的药品，可直接申报进入国家医保目录。此为商保目录与医保目录之间的制度性"桥接"，为已在商保市场获得初步验证的创新药提供了快速进入医保的绿色通道。

**（3）时间线前移**
新版目录计划于11月底前公布，较以往惯例提前约1个月，有利于新纳入品种次年1月及时执行新价格。

**对恒瑞的影响**：三项机制均利好恒瑞。预申报制度解决了创新药"获批后等进保"的真空期痛点，商保转轨通道为恒瑞已布局商保目录的品种提供了制度便利。整体而言，2026年医保目录调整机制进一步强化了"创新药友好型"政策取向，恒瑞作为国内最大创新药管线持有者，是最大受益方之一。

### 2.5 集采常态化——双刃剑评估

**对恒瑞的双重影响**：

| 影响层面 | 具体表现 |
| :--- | :--- |
| **利空（仿制药业务）** | 恒瑞仍有约116亿仿制药收入（约占营收37%），每轮集采都面临降价压力。2025年31个集采项目，价格平均降幅70%。但仿制药利润占比已大幅缩小。 |
| **利好（创新药竞争）** | 集采压低仿制药价格→压缩me-too创新药的套利空间→加速淘汰低水平重复→**利好真正做差异化创新的头部企业如恒瑞** |

**趋势预判**：集采对恒瑞的边际影响已从"负面冲击"转为"中性偏利好"。2021-2022年集采冲击高峰期，恒瑞净利率从17%一度下探，但2023年后创新药放量完全覆盖并超越了仿制药减量。

## 三、地缘政治风险

### 3.1 BIOSECURE法案——已立法但影响有限

| 时间线 | 事件 |
| :--- | :--- |
| 2024年1月 | BIOSECURE首次提出，点名药明康德、药明生物、华大基因等5家公司 |
| 2024年12月 | 未纳入FY25国防授权法案，但持续在国会推进 |
| 2025年12月18日 | **特朗普签署FY26 NDAA，BIOSECURE作为第851节正式成为法律** |

**法案核心内容**：
- 禁止联邦机构采购"受关注生物技术公司"的设备或服务
- 禁止与使用这些公司服务的实体签订/续签联邦合同
- 删除具体公司名称，改为依赖国防部1260H名单
- 生效时间：OMB需2026年12月前公布名单→FAR修订（1年内）→生效（60-90天后），**实质生效可能在2027-2028年**
- 设有过渡期至2032年

**对恒瑞的影响评估**：
- **直接影响：极低**。恒瑞不属于CDMO/CRO，不提供外包服务，不是BIOSECURE的打击对象
- **间接影响：有限**。恒瑞的创新药若未来寻求FDA批准，需依赖自身临床数据。当前恒瑞仅20+项国际临床试验，在FDA申请上市的品种极少
- **溢出风险：可控**。最坏情境——若"受关注实体"名单扩大至创新药企，恒瑞可能面临美国市场准入障碍。但恒瑞的核心业务99%在中国，且海外策略是"BD授权而非自主销售"，地缘政治风险已通过NewCo模式（Kailera为美国注册公司）部分对冲

### 3.2 COINS法案与生物技术投资管制——Moolenaar致函财政部事件（新增风险）

2026年5月21日，美国众议院"中国共产党战略竞争特设委员会"主席John Moolenaar（R-MI）致函财政部长Scott Bessent，要求将**生物技术**增列为《COINS法案》的"禁止技术"（prohibited technology），限制美国资本和技术流入中国生物技术领域。信函**明确点名**BMS-恒瑞152亿美元战略合作，以及礼来与中国药企的合作协议。

**COINS法案背景**：
- 2025年12月由特朗普签署为法律（NDAA第LXXXV章），目前覆盖**半导体、AI、量子技术**三个领域
- Section 809授权财政部可将管辖范围扩展至任何"助力外国对手军事/情报/网络能力"的技术
- 财政部需在450天内（约2027年3月）发布实施细则
- **当前状态：生物技术尚未被纳入管辖范围**

**Moolenaar的核心论据**：
1. 2025年中美跨境BD授权交易总额达**1360亿美元**，5000万美元以上交易中中国企业占比从2020年的**0%** 飙升至**48%**
2. 指控中国推行"国家主导的蓄意战略"以主导全球生物技术（涵盖新药研发、生物药制造、临床研究）
3. BMS-恒瑞协议涉及知识产权和研发know-how向中方转移（共同开发+全球商业化选择权），构成国家安全隐忧
4. 类比稀土和半导体——若不限制，美国将在创新药物领域形成对华战略依赖

**美国国会近期反华生物技术立法动向汇总**：

| 时间 | 事件 | 性质 | 状态 |
| :--- | :--- | :--- | :--- |
| 2025年12月 | BIOSECURE法案随NDAA签署为法律 | 限制联邦采购中国生物技术设备/服务 | 已立法，2026年12月公布首批名单 |
| 2025年12月 | COINS法案随NDAA签署为法律 | 限制对中国半导体/AI/量子技术的美国投资 | 已立法，生物技术未被纳入 |
| 2026年4月29日 | 众议院拨款委员会通过FDA中国数据禁令报告语言 | 禁止FDA接受中国临床数据用于IND | 报告语言（非法律），待全院表决 |
| 2026年5月21日 | Moolenaar致函财政部要求将生物技术纳入COINS | 建议将生物技术列为"禁止技术" | 建议函（非法律），Treasury未回应 |
| **2026年6月2日** | **BINSA法案正式提交国会（两党）** | **生物技术纳入COINS审查，点名BMS-恒瑞deal** | **正式立法提案，预计纳入2027年PDUFA** |

**对恒瑞的影响分析**：

| 影响维度 | 短期（2026-2027） | 长期（2028+） |
| :--- | :--- | :--- |
| **BMS deal交割** | 几乎无影响。COINS不覆盖生物技术，HSR反垄断审查独立于COINS，合同义务受法律保护 | 适用祖父条款，极低风险 |
| **未来美国BD交易** | 中度不确定性。Treasury尚未启动规则制定，大选年进度缓慢 | 若纳管则显著受限，但欧洲/日本BD不受影响 |
| **Kailera/NewCo模式** | 无影响。Kailera为美国注册公司，COINS限制的是"对华投资" | 实际上构成天然对冲——资金留在美国体系内 |
| **GSK等欧洲合作** | 完全无影响。COINS仅约束美国资本 | 可能反而受益（欧洲药企填补美国留下的BD真空） |

**概率情景分析**：

| 情景 | 概率 | 对恒瑞BD估值的影响 |
| :--- | :---:| :--- |
| **什么都不做**（Treasury沉默/驳回） | 35% | 无影响。当前估值假设基线 |
| **局部限制**（仅限基因数据/生物制造等子领域，不涉及药物IP许可） | 40% | 低影响。恒瑞BD交易核心是药物IP许可，非制造或数据业务 |
| **全面纳入**（所有药物IP许可+共同开发均需审查） | 20% | 重大影响。未来美国BD通道收缩，但已签合同受祖父条款保护 |
| **立即执行**（2026年内出台限制） | 5% | 严重影响。但考虑到450天规则制定周期+Treasury沉默，概率极低 |

**三个关键对冲因素**：
1. **已签合同锁定**：BMS（152亿美元）、GSK（120亿美元）等已签署协议受法律保护。即使COINS未来纳管生物技术，祖父条款几乎肯定适用
2. **BD伙伴多元化**：恒瑞的主要BD伙伴中GSK（英国）、Merck KGaA（德国）不受COINS约束。且英国MHRA正在扩大接受中国临床数据，"弃美入欧"提供了替代路径
3. **业务底线极度本土化**：恒瑞99%收入来自中国。与百济神州（67%海外收入）形成对比——最极端情境下恒瑞仍可依靠国内市场+欧洲BD维持高增长，底线远高于市场恐惧

**核心判断**：Moolenaar信函是真实的政策风险升级信号，但其法律效力为零（仅为建议函），且BMS deal在最坏情境下受祖父条款保护。真正需要关注的是Treasury在2027年规则制定中是否采纳，而非当前的政治噪音。

### 3.2A BINSA法案——从致函建议到正式立法提案（2026年6月2日重大升级）

2026年6月2日，Moolenaar联合民主党众议员Debbie Dingell（D-MI）正式向国会提交**《生物技术投资国家安全法案》（Biotech Investment National Security Act，简称BINSA）**，作为对COINS法案的修订提案。这标志着生物技术投资限制从"个人政见函"升级为"正式两党立法提案"，风险等级发生质变。

**BINSA核心条款**：

| 条款 | 内容 |
| :--- | :--- |
| **覆盖范围** | 药物开发、生物制品制造、临床研发（农业生物技术、工业发酵、基础学术研究被明确排除）|
| **受审查交易类型** | 美方对华药物IP许可交易、合资企业、股权投资，以及对"受关注外国主体"的技术许可 |
| **审查主体** | 美国财政部（Treasury），须在立法通过后1年内发布实施细则 |
| **新增条款** | 国防部长须在60天内评估美资流入中国生物技术领域对国家安全的负面影响 |
| **立法路径** | 作为COINS法案修订案推进；业界预计可能被纳入PDUFA重新授权法案（2027年通过）|

**BINSA vs 此前Moolenaar信函的本质区别**：

| 维度 | 5月21日信函 | 6月2日BINSA法案 |
| :--- | :--- | :--- |
| 法律效力 | 零（建议函）| 如通过则具有完全法律约束力 |
| 两党支持 | 单人（共和党）| 两党（Moolenaar+民主党Dingell）|
| 点名程度 | 提及BMS-恒瑞deal | **专门点名BMS-恒瑞152亿美元和辉瑞-信达105亿美元交易** |
| 审查机制 | 无 | 建立Treasury主动审查+国防部评估双重机制 |
| 通过概率 | N/A | 中高（两党对华强硬共识+纳入PDUFA立法车辆）|

**工业界反应分化**：

| 阵营 | 代表 | 立场 |
| :--- | :--- | :--- |
| **支持脱钩** | Ginkgo Bioworks（Jason Kelly）、Curie Bio关联投资者 | 公开支持。Ginkgo与中国合成生物学企业存在直接竞争 |
| **反对脱钩** | RA Capital（Peter Kolchinsky）、Alnylam创始人John Maraganore、AstraZeneca、Pfizer等大型药企 | 强烈反对。Kolchinsky发表长篇博客论证BINSA将伤害患者并促使美国更依赖外国生物技术 |
| **谨慎观望** | PhRMA、BIO行业协会 | 未公开表态，但BMS、Lilly等成员企业正在游说反对 |

**BINSA通过概率与时间线评估**：

| 情景 | 概率 | 时间窗口 |
| :--- | :---:| :--- |
| **完整通过**（生物技术全面纳入COINS审查） | 30% | 最早2027年Q3-Q4（随PDUFA重新授权通过）|
| **削弱版本通过**（仅涵盖基因数据/生物制造等子领域，排除药物IP许可） | 35% | 同期 |
| **立法搁浅**（工业界游说成功阻止纳入PDUFA） | 35% | 2028年后可能重启 |

**关键对冲因素**：
1. **President Trump签署需权衡**：虽为China hawk，但Trump多次强调降低药价、促进美国制药回流。BINSA可能因伤害美国药企管线补充能力而面临行政分支阻力
2. **祖父条款覆盖范围成关键博弈点**：若BINSA通过，BMS-恒瑞deal（2026年5月签署）能否获得祖父条款保护将取决于具体立法文本中的生效日期条款——这是BMS/恒瑞法律团队的核心关切
3. **Kailera模式的结构性优势凸显**：Kailera为美国注册公司，其管线资产已不在中国境内。即使BINSA通过，Kailera旗下瑞普泊肽的FDA申报路径不受影响，GLP-1的全球商业价值实质上已被隔离保护

| :--- | :--- | :--- |
| **BMS deal交割** | 几乎无影响。HSR反垄断审查独立于BINSA | 适用祖父条款概率高，但需关注具体立法文本 |

### 3.3 FDA拟禁中国临床试验数据——持续发酵

2026年4月29日，美国众议院拨款委员会以35:25通过FY2027拨款法案报告，其中包含禁止FDA接受中国临床数据用于IND申请的语言。

**法律状态与推进路径**：
1. **当前效力**：仅为委员会报告语言（Report Language），非约束性法律条款，不具有直接法律强制力。但其作为拨款指导文件，对FDA预算执行有实质性引导作用。
2. **适用范围**：明确限定于IND阶段，不涉及NDA/BLA的审评。
3. **立法路径**：尚需完成以下步骤——众议院全院表决→参议院审议→两院协调→总统签署，流程耗时可能跨越数年。
4. **缓冲安排**：FDA获一年缓冲期发布相关指导文件，短期内不会立即执行。

**实际落地概率评估**：中等偏低。尽管国会两党对华强硬已成共识，但完全切断中国临床数据通道将损害美国药企利益（大量MNC在中国开展临床试验），工业界游说力量不可忽视。然而，政策信号已足够明确——中国创新药的FDA申报路径正面临结构性收紧。

**英美分化格局**：与美国收紧趋势形成鲜明对照，英国MHRA反向操作，计划扩大接受中国临床数据的范围，表明全球监管格局并非铁板一块。中国创新药企可能转向以EMA/MHRA为跳板进入欧洲市场，形成"弃美入欧"的替代路径。

**对恒瑞的影响**：
- 短期：IND阶段风险尚不紧迫，恒瑞当前FDA申报品种极少
- 中期：若立法成型，恒瑞需通过海外临床平台（Kailera/波士顿研发中心）开展独立临床试验，成本将显著上升
- 对策空间：英国MHRA的开放态度提供了替代路径，恒瑞可考虑以欧洲上市为优先目标

### 3.4 全球供应链重构（China+1）

- 美欧日印韩"生物制药联盟"已于2024年成立，旨在构建去中国化的生物制药供应链
- 韩国三星生物和Lonza在大分子CDMO领域加速扩产
- 印度CDMO企业营收增速10%+，但知识产权保护和合规产能仍有短板

**对恒瑞的影响**：供应链重构主要冲击CXO/CDMO赛道，对恒瑞的直接冲击有限。恒瑞的上游原料供应链以内为主（连云港、成都等自有基地），非贸易敏感品类。

## 五、外部边界结论

恒瑞当前面临**"内外分化急剧加剧"的外部环境**：

- **国内**：政策环境处于历史最友好时期。9号文、产业支柱定位、商保目录落地、审评提速、医保预申报（5月31日正式发布方案，6月1日启动申报）——这是一套完整的"创新药友好型"制度组合。
- **国际**：地缘政治风险从"信号级"升级为"立法级"。BINSA法案（6月2日提交）是继BIOSECURE之后，第二个直接针对中美生物技术合作链条的国会立法提案，且明确点名BMS-恒瑞152亿美元合作。与5月21日Moolenaar信函相比，BINSA实现了三重质变：（1）从个人建议函→两党正式法案；（2）从"呼吁Treasury行动"→"直接修改法律建立审查机制"；（3）从共和党单边→两党联署。虽然完整通过概率仅30%（RA Capital/大型药企强烈游说反对），但已改变市场对恒瑞BD收入的"尾部风险"定价——从"极不可能"调整为"不可忽视的中概率事件"。

**关键变量的三层解析（更新）**：
1. **已签合同层**：BMS（152亿美元）、GSK（120亿美元）等已签署协议受法律保护。但在BINSA情境下，祖父条款的覆盖范围（是否涵盖co-development阶段的知识产权持续转移）成为核心博弈点。BMS 6亿美元首付和已确认BD收入的锁定性依然较高
2. **未来BD层**：美国BD通道面临"立法不确定性导致的寒蝉效应"——即使BINSA最终被削弱或搁浅，美方药企可能在立法明朗化前（2027年中）暂缓新的对华BD谈判。但欧洲/日本药企不受影响，且英国MHRA正在扩大接受中国数据——"弃美入欧"路径正在成型。恒瑞作为中国最大创新药管线持有者，若美国买家退场则对欧洲买家的议价力反而增强
3. **业务底线层**：恒瑞99%收入来自中国本土，对外部地缘冲击的敏感度远低于百济神州（67%海外收入）。极端情境下（BINSA完整通过+未来美国BD归零），恒瑞仍可依赖国内创新药市场维持15-20%增长。**红线**：若辉瑞-信达deal也被纳入审查，则BINSA将通过超党派共识快速推进

**核心理念**：如果把创新药产业看作"在国内定价、到海外变现"的双轮模型，那么国内的政策轮保持加速转动（医保+9号文+商保），但海外的变现轮面临从"摩擦"到"可能锁死"的结构性风险。恒瑞的"BD授权+NewCo"轻资产模式（Kailera）在此情境下的前瞻性被进一步验证——GLP-1的全球商业价值已被隔离在Kailera美国实体中，不受BINSA直接影响。2026年6-7月是关键观察窗口：BMS deal的HSR审查结果+GLP-1瑞普泊肽审批决定+2026医保申报窗口，三重催化剂密集交汇。
