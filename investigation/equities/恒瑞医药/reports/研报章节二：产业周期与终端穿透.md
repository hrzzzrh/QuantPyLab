# 研报章节二：产业周期与终端穿透

**研究日期**：2026年05月26日
**标的**：恒瑞医药（600276.SH / 1276.HK）
**目标年份**：2026年（当年）

**前序核心结论回顾（模块一）**：
恒瑞已完成从仿制药龙头到创新药平台的结构性转变（创新药占比突破58%），净利率持续扩张（17%→28%），OCF质量极优（1.46x），但面临资本效率低下（405亿现金闲置）和创新药占比仍需提升的挑战。这些财务特征决定了模块二的切入点——必须穿透创新药终端需求的真实动能。

---

## 一、产业生命周期与库存节奏

### 1.1 行业周期定位：创新药处于成长期，全行业处于调整期

中国医药行业正经历"总量调整 + 结构升级"的双重变局：

| 维度 | 整体医药行业 | 创新药细分 |
| :--- | :--- | :--- |
| **周期阶段** | 成熟期（存量博弈）| **成长期（渗透率驱动）** |
| **关键信号** | 2025年全终端市场约20033亿元，同比**下降1.4%**（罕见总量收缩）；规模以上医药制造业利润下降2.8% | 2025年创新药市场约1.22万亿，增速仍在双位数；获批创新药76个（同比+58%），创历史新高 |
| **增长动能** | 人口老龄化驱动的刚性需求 vs 集采降价导致的"量增价减" | 创新药审评加速（纳入ICH、优先审评）、医保"以价换量"、商保补充支付 |

**核心判断**：医药行业整体正处于"以价换量"的存量洗牌期，但创新药是当前唯一具备高确定性的成长板块。恒瑞身处创新药赛道，受益于结构性增长，但其仿制药业务（约116亿收入）承受着集采降价的持续压力。

### 1.2 库存动力学：从被动去库到主动补库的切换

- **仿制药/原料药环节**：处于**被动去库阶段**。集采常态化导致医疗机构减少仿制药库存，"以量换价"模式下企业按约定量精准供应，传统的渠道压货模式被彻底打破。2025年集采31个项目，省级联盟占87%，覆盖面持续扩大。
- **创新药环节**：处于**主动补库初期**。创新药获批放量驱动企业扩充产能。恒瑞在上海、广东、天津、苏州等地的产能建设持续进行。ADC CDMO领域尤为明显——药明合联营收增速47%，产能排期至2027年。

**牛鞭效应辨析**：当前创新药的上量更多源于真实临床需求释放（老龄化、医保准入、学术推广），而非渠道压货的放大效应。证据包括：
1. 恒瑞的创新药收入季度分布均匀（2025年Q1-Q4：创新药季度收入稳定增长），无年末突击压货痕迹
2. 应收账款增长（2025年末51.47亿）低于营收增长（+13%），说明终端回款质量良好

## 二、终端需求穿透与跨期断层审计

### 2.1 最终产品画像：两大核心终端

恒瑞的创新药最终负载于以下终端形态：

| 产品线 | 最终终端 | 收入占比（2026Q1）| 核心场景 |
| :--- | :--- | :--- | :--- |
| 抗肿瘤产品 | 癌症患者治疗 | 73.2%（创新药内部）| 三甲医院肿瘤科、DTP药房 |
| 非肿瘤产品（代谢/自免/神经）| 慢性病患者管理 | 26.8%（创新药内部）| 内分泌科、风湿免疫科、零售 |

### 2.2 抗肿瘤终端景气度审计

**市场规模与增长**：
- 2025年中国抗肿瘤药物市场规模约3187亿元（5年CAGR约11.4%）
- 从2020年的1858亿元净增1329亿元，5年累计涨幅71.5%
- 预计2026年突破3500亿元

**增长动能源头审计**：

| 动能来源 | 性质 | 持续性评估 |
| :--- | :--- | :--- |
| 人口老龄化（65岁以上人口2.24亿，占15.9%）| **内生** | 确定、持续、不可逆。老年癌症发病率是中年人的5-10倍 |
| 诊疗可及性提升（总诊疗人次101.1亿，+20.07%）| **半内生** | 分级诊疗+医保覆盖扩大，具有政策惯性 |
| 治疗范式升级（化疗占比从78%降至43.8%，靶向/免疫疗法占比47.1%）| **内生** | 技术替代驱动，不可逆 |
| 医保"以价换量"（国家医保已纳入199种创新药，累计支出5048亿元）| **外生/政策** | 持续但边际减弱（2025新增36种抗癌药，平均降价63%）|
| 商保补充支付（商业险在抗癌药支付中占比预计从12%升至18-20%）| **外生/政策** | 中期增量 |

**跨期断层审计——需求前置风险评估**：

- **PD-1抑制剂市场**：存在一定程度的**需求前置**。国产PD-1年治疗费用已从进口药30万+降至5万以内，医保纳入后短期内需求被提前释放。恒瑞卡瑞利珠单抗面临的挑战是：PD-1市场已趋于饱和，新增适应症（联用方案的推进）将是维持增长的唯一路径。
- **HER2 ADC（瑞康曲妥珠单抗）**：需求前置风险较低。该品类国内渗透率尚处早期（对标DS-8201在日本渗透率），医保纳入将带来的是增量释放而非存量转移。
- **整体评估**：恒瑞抗癌管线丰富（超30款肿瘤创新药），单产品需求前置风险可被管线迭代所消化。不存在"一锤子买卖"式的系统性透支。

### 2.3 代谢/GLP-1终端景气度审计

**市场规模与结构性需求**：
- 中国成人超重/肥胖率：50.7%（2018年，预计2030年升至61.3%），相当于约7亿肥胖/超重人群
- 2026年中国减重药物市场从"野蛮生长"进入"合规化2.0阶段"
- 全球GLP-1市场正处爆发期，2025年全球GLP-1市场超500亿美元

**恒瑞GLP-1管线的终端定位**：
1. **瑞普泊肽（HRS9531，GLP-1/GIP双靶点）**：注射剂型减重NDA已获受理，预计2026年获批。口服版中国II期减重数据：26周平均体重下降最高12.1%（优于司美格鲁肽的约10%），且未观察到平台期。计划推进肥胖症III期。
2. **HRS-7535（口服小分子GLP-1）**：2型糖尿病和减重适应症均已进入III期，赛道差异化明显。
3. **Kailera全球平台**：恒瑞以"NewCo"模式将GLP-1海外权益授予Kailera（纳斯达克上市，代码KLRA），总里程碑高达60亿美元。

**竞争烈度评估（跨期断层风险的核心来源）**：

| 风险因子 | 评估 |
| :--- | :--- |
| 司美格鲁肽专利已到期，中瑞FTA数据保护触发 | 专利已到期，但因中瑞FTA数据保护，仿制药审批暂停，最早2027年4月才可能大规模涌入。窗口期较此前预期延长6-12个月。风险等级：中高 |
| 礼来替尔泊肽已进医保 | 疗效最强，月治疗费已降至300+元，对恒瑞定价形成锚定 |
| 信达玛仕度肽已上市 | 国产先发优势 + 渠道覆盖 |
| 辉瑞埃诺格鲁肽进口 | 2026年入华，新竞争者 |

**新增关键洞察——司美格鲁肽仿制药审批延迟**：2026年3月20日司美格鲁肽核心专利到期后，由于中瑞FTA（自由贸易协定）中的药品试验数据保护条款（保护期6年），瑞士子公司Novo Nordisk Pharma AG持有的司美格鲁肽在中国触发了数据保护，多家仿制药企业（九源基因、丽珠集团、联邦制药等）的上市申请审批状态显示为"暂停"。这意味着仿制药最早可能要到2027年4月才能获批，而非此前预期的2026年Q3。这一变化**显著延长了恒瑞GLP-1双靶点创新药的竞争窗口期**。走2.4类改良型生物制品路径的企业（如齐鲁、石药）可能不受此影响，但其临床数据需独立验证。

**跨期断层审计结论**：
- **存在显著的"竞争前置"风险，但不存在"需求前置"风险**。7亿超重人群的临床需求是真实且几乎未被满足的（当前GLP-1药物覆盖率极低），但供给端的竞品洪峰（预计2027-2028年）将在仿制药获批后将市场从"蓝海"压入"红海"。
- 恒瑞的差异化在于：（1）双靶点（GLP-1/GIP）提供更优减重数据；（2）口服+注射双剂型布局；（3）Kailera全球化变现。商业化窗口期相对充裕但不可松懈——需利用延长的窗口期加速品牌建设和处方引流。

## 三、供需边际与压力传导

### 3.1 行业整体供需平衡

| 供给端 | 需求端 | 供需边际 |
| :--- | :--- | :--- |
| 2025年NMPA批准76个创新药（+58%），管线加速释放 | 创新药市场年增长约8-10% | **供给增速 > 需求增速**，竞争加剧不可避免 |
| CDMO产能CAGR>20%扩张，但结构分化（ADC/大分子缺、小分子过）| 研发支出持续增加（恒瑞87亿、百济141亿）| 高端产能（ADC、CGT）仍紧缺，低端产能过剩 |
| GLP-1赛道：20+仿制药+5+创新药涌入 | 7亿超重人群，但自费为主 | **短期供过于求风险显著** |

### 3.2 价格压力传导特征

**波动放大效应分析**：
- 集采价格降幅（平均70%）对仿制药利润的冲击是**即时且不可逆的**。恒瑞的仿制药业务（约116亿）每一轮集采都将面临新一轮降价。
- 但恒瑞的"对冲机制"正在生效——创新药收入（163亿）增速（+26%）远超仿制药减量，净利率不降反升（17%→28%）。

**传导时滞与弹性**：
- 医保谈判降价的**传导时滞约为3-6个月**（协议签署→医院进药→处方放量），期间存在"以价换量"的验证窗口。
- 恒瑞的博弈筹码包括：（1）**品种数量优势**（24款1类创新药），单一品种集采冲击可被管线宽度分散；（2）**学术推广壁垒**（全国医院覆盖网络），并非单纯价格竞争。

### 3.3 供需边际预判

**核心预判**：2026-2027年创新药供需边际将从"紧平衡"向"选择性过剩"滑落。

- **抗肿瘤赛道**：有一定程度的拥挤（PD-1/L1类已有10+产品），但恒瑞的优势在于管线深度（ADC、双抗、PARP等差异化靶点）而非单品依赖。
- **代谢/减重赛道**：2026年下半年至2027年是"供给洪峰期"（司美仿制药+多款创新药集中上市），短期供给冲击猛烈。但长期看，7亿肥胖/超重人口的渗透率（当前<1%）决定了天花板极高，**优胜劣汰后的幸存者将享受巨大市场**。
- **恒瑞的相对优势**：恒瑞GLP-1双靶点（GIP/GLP-1）是目前国产进度最快的之一，且借Kailera打开了全球化想象空间。但**时间窗口紧张**——必须在2026-2027年窗口期内建立品牌心智和处方习惯。

## 四、强制复核项确认

基于以上分析，逐项确认：

1. **我是否已经穿透并引用了该行业最终终端产品的未来两年景气度预测？**
   → 是。已分析抗肿瘤药物（2026E 3500亿+）和减重/代谢药物（渗透率<1%，长期巨大空间）的终端景气度。

2. **我是否审计了当前增长逻辑中是否存在由于刺激政策导致的"跨期需求移位"？**
   → 是。已识别PD-1市场存在一定程度的医保前置释放效应，但恒瑞管线宽度可消化；减重市场不存在需求前置，但存在**供给前置（竞争超前于需求释放）**。

3. **我是否基于行业产能与终端需求，分析了供需边际的演变方向及其对公司的压力传导特征？**
   → 是。供需边际正从紧平衡向选择性过剩演变，恒瑞的关键风险窗口为2026-2027年GLP-1赛道供给洪峰。但管线纵深和学术推广壁垒为其提供了缓冲。

## 五、产业周期结论

恒瑞所处的创新药行业正处于**"总量收缩中的结构性牛市"**——整体医药市场萎缩（-1.4%），但创新药市场以双位数增速逆势扩张。这一"剪刀差"恰恰构成恒瑞的alpha来源：仿制药的集采冲击已基本被消化（净利率恢复至28%），创新药管线正进入密集兑现期（2026-2028年预计约53项成果获批）。

然而，最大的外部不确定性来自**GLP-1赛道的供给洪峰**。司美格鲁肽专利到期、20+仿制药涌入、替尔泊肽降价、信达已占先机，恒瑞必须在窗口期（2027-2028年）内完成品牌建设和处方引流，否则将被卷入纯粹的价格消耗战。

**产业周期评级**：创新药成长期，但细分赛道竞争烈度快速攀升。恒瑞的管线纵深是最大护城河——不依赖单一爆款，而是凭借24款已上市创新药的矩阵叠加效应。
