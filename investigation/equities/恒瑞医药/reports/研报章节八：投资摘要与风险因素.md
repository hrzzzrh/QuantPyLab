# 研报章节八：投资摘要与风险因素

**研究日期**：2026年06月14日
**标的**：恒瑞医药（600276.SH / 1276.HK）

---

## 一、投资摘要

### 核心投资逻辑

恒瑞医药正处于**"创新转型兑现 + 政策环境质变 + 估值历史低位"三重拐点的交汇处**。

**1. 创新转型已过临界点。** 创新药收入占比从2022年约38%跃升至2025年58.34%（2026Q1进一步升至61.69%），创新药已成为业务主体。2025年营收316亿（+13%）、归母净利77亿（+22%），增速和质量均显著改善。2026-2028年预计约53项创新成果获批，管线兑现密度为历史最高。

**2. 政策环境发生质变。** 2026年国务院9号文确立"市场主导+分类施策"的药品定价框架，创新药获得自主定价权和价格稳定期，创新药明确不纳入集采。同时产业定位从"新兴"升级为"支柱"，制度红利刚刚开启。这一政策拐点的深远意义尚未被市场充分定价。

**3. 估值处于历史极端低位。** 当前PE（TTM）39.64x，6月8日一度触及37.55——1年和3年窗口的绝对历史最低值，距5年/10年最低33.33x仅11.2%空间。随后技术性反弹至48.49元。市场以接近"历史最极端悲观"的预期为恒瑞定价，但基本面显示：净利率持续扩张（17%→28%）、OCF质量极优（1.46x）、零杠杆（405亿净现金）。估值修复至52-58x（仍低于10年中位数62x）具有极强逻辑支撑。

**4. 盈利弹性持续释放。** 销售费用率连续5年下降（36%→27%），每降1个百分点可释放约3.2亿税前利润。净利率从2021年17%升至2026Q1的28%，仍有上行空间。

**5. 多赛道布局的"宽基指数"特性提供确定性溢价。** 24款已上市创新药横跨肿瘤、代谢、自免、神经科学，不依赖单一爆款。这种"东方不亮西方亮"的结构，在不确定性上升的市场环境中具有稀缺价值。

**6. BMS 152亿美元战略合作——BD收入从"期权性收益"升级为"持续性增长引擎"。** 恒瑞与全球Top5药企达成史上最大BD交易，6亿美元首付+深度共同开发模式，标志着研发能力获得MNC最高级别体系化背书。BMS合作的确定性付款（9.5亿美元锁定）将BD收入来源从一次性"卖青苗"偶然收益，升级为可持续、可预期的持续性增长引擎。

## 二、风险因素

按对股价的潜在影响烈度排序：

### 第一梯队：致命风险（影响烈度★★★★★）

**1. GLP-1瑞普泊肽获批失败或上市后严重不达预期**
- 触发条件：NMPA拒绝批准（安全性质疑）、上市后出现严重不良反应、竞品价格战导致放量远低于预期
- 影响：非肿瘤创新药增长引擎断裂，2026E创新药增速从30%降至10-15%，EPS从1.40降至1.15-1.20
- 概率：15-20%

**2. 核心肿瘤品种遭遇重大临床/监管挫折**
- 触发条件：瑞康曲妥珠单抗关键适应症审评被拒、已上市品种出现严重安全性问题导致召回
- 影响：市场对管线质量的信任崩溃，PE从45x压缩至35x，市值缩水25-35%
- 概率：5-10%

### 第二梯队：重大风险（影响烈度★★★★）

**3. BD收入骤降**
- 触发条件：GSK/默沙东等交易对手因自身战略调整终止合作、Kailera临床失败、新BD交易断档
- 影响：BD收入从35亿降至10-15亿，净利润直接减少10-15亿（-10-16%）
- 概率：10-15%（BMS 9.5亿美元确定付款锁定）
- 子风险——**BMS交易未获HSR反垄断批准**：若BMS合作因美国HSR（Hart-Scott-Rodino）审查被否决，BD收入将失去最大确定性来源，净利润影响约8-10亿

**4. 中美地缘政治升级至创新药领域——BINSA法案（6月2日重大升级）**
- 触发条件：BINSA法案通过并纳入PDUFA重新授权（2027年）；或Treasury在2027年3月COINS实施细则中纳入生物技术
- **当前状态**: 6月2日两党（Moolenaar+Dingell）正式提交BINSA法案，明确点名BMS-恒瑞deal。从5月21日"建议函"升级为"正式立法提案"，两党联署增强了通过概率
- 影响：若完整通过且未设祖父条款→未来美国BD通道显著收缩，已签BMS合同面临追溯审查风险；若削弱版通过（仅限基因数据/生物制造）→对恒瑞影响有限
- 对冲：Kailera模式天然隔离GLP-1全球价值；欧洲/日本BD不受影响；恒瑞99%收入在国内
- 概率：完整通过30%；削弱版35%；搁浅35%

### 第三梯队：中等风险（影响烈度★★★）

**5. 销售费用率下降停滞甚至反弹**
- 触发条件：基层慢病布局需要大规模投入、创新药竞争加剧推高学术推广成本
- 影响：净利率从28%回落至25%，EPS从1.40降至1.30
- 概率：20-25%

**6. 仿制药加速下滑**
- 触发条件：新一轮集采降价幅度超预期（>80%）、仿制药品种被踢出医院采购目录
- 影响：仿制药收入从118亿降至100亿以下，拖累营收2-3%
- 概率：15-20%

**7. 研发资本化比例持续上升引发利润质量担忧**
- 触发条件：资本化率从20%升至30%+，未来摊销压力增大
- 影响：市场对利润真实性的信任下降，PE折价
- 概率：10-15%

### 第四梯队：低概率尾部风险（影响烈度★★）

**8. 管理层变动（孙飘扬退休/离任）**
**9. 大规模商誉/资产减值（当前商誉≈0，风险极低）**
**10. 汇率大幅波动对BD收入的折算影响**

## 三、研究结论象限图

基于"确定性"与"预期收益空间"两维评估：

```
预期收益高 ↑
            │
    第二象限    │    第一象限 ★
  (Speculative  │  (High-Conviction
    Growth)     │      Buy)
            │
  低确定性 ←──┼───→ 高确定性
            │
    第四象限    │    第三象限
   (Avoid/Wait) │  (Defensive Value)
            │
预期收益低 ↓
```

### 恒瑞医药定位：**第一象限（High-Conviction Buy）**

**定位理由**：

| 维度 | 评估 |
| :--- | :--- |
| **确定性** | ★★★★★（极高）。管线兑现路径清晰（24款已上市+53项预计2026-2028获批），创新药占比突破62%（Q1 2026）验证了转型成功，政策环境确定性利好（9号文+支柱产业+医保预申报），ADA 2026进一步验证GLP-1管线数据。BMS 152亿美元战略合作为研发能力提供MNC最高级别背书。BINSA法案增加地缘不确定性，但三条对冲路径（祖父条款+Kailera隔离+欧洲BP）在极端情境下仍有效 |
| **预期收益空间** | ★★★★★（极高）。估值创1年/3年历史新低（PE 37.55于6月8日，此后反弹至39.64），现价48.49元对应中性目标价92.24元，上涨空间+90.2%，盈亏比极端非对称（保守情景目标价49.56元仍高于现价）。催化剂密集：GLP-1获批（6-7月关键窗口）、中报超预期、BMS首付款到账等。6月12日技术面出现底部信号 |
| **综合定位** | **核心重仓候选。** 高确定性来自管线纵深+政策拐点+财务质量三重保障；高预期收益来自创历史新低的估值分位。与5月26日相比，估值非对称性因股价进一步下跌而增强，但需警惕BINSA立法进程。适合作为医药板块核心配置，仓位建议5-10% |

**与同类标的对比**：
- vs 百济神州（第二象限）：百济全球化成长性更高但盈利确定性低（净利14亿对应PE 360x），适合进攻型配置
- vs 信达生物（第一/二象限边界）：信达本土商业化效率高但管线厚度不足，确定性和收益空间均略逊于恒瑞
