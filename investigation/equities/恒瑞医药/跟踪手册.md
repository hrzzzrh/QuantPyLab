## 三、逻辑证伪警报（Invalidation Triggers）

**以下信号出现时，投资逻辑已死，必须无条件止损或下调评级：**

| 序号 | 证伪信号 | 烈度 | 触发行动 |
| :--- | :--- | :--- | :--- |
| **红灯1** | 瑞康曲妥珠单抗（HER2 ADC）被NMPA拒绝批准核心适应症（乳腺癌一线/结直肠癌），或出现严重安全性问题导致撤回 | **致命** | 立即清仓。该品种是市场对恒瑞ADC平台信心的核心锚点，失败意味着管线质量遭到根本性质疑 |
| **红灯2** | GLP-1瑞普泊肽获批后6个月内市占率低于5%（即放量严重不达预期，被信达/诺和/礼来完全压制）| **致命** | 减持至半仓，下调至第三象限。GLP-1是增长新引擎，失败意味着"非肿瘤转型"叙事破产 |
| **红灯3** | 创新药收入连续两季YoY增速低于10%（当前25%+）| **致命** | 清仓。创新药增长是唯一估值支柱，失速意味着所有PE倍数的支撑崩塌 |
| **红灯4** | 经营现金流连续两季为负，且应收账款暴增（>30% YoY）| **严重** | 减持至半仓。说明"卖药不赚钱"，利润是账面游戏 |
| **红灯5** | 美国将恒瑞列入实体清单或BIOSECURE名单扩展至创新药企 | **严重** | 减持至半仓。海外变现通道被切断，估值逻辑需重写 |
| **黄灯1** | 销售费用率反弹至30%以上且创新药增速未同步提升 | **警惕** | 重新评估商业化效率，下调净利率预测 |
| **黄灯2** | 研发资本化率升至30%以上 | **警惕** | 重新评估利润质量，可能导致PE折价 |
| **黄灯3** | GSK/MSD等核心BD交易对手终止合作 | **警惕** | 重新评估管线外部验证信号 |

## 五、仓位管理与行动框架

| 情景 | 条件 | 行动 |
| :--- | :--- | :--- |
| **建仓区间** | 股价 50-56元（当前处于此区间）| 分批建仓至目标仓位的50-70% |
| **加仓区间** | 股价回调至 51-53元（不跌破51.63低点）| 加仓至目标仓位的100% |
| **减仓观察** | 股价快速拉升至 70元以上但无基本面催化剂 | 减仓至50%，等待回调 |
| **止盈区间** | 股价达到 80元（目标价81.53附近）| 分批止盈，保留30-50%仓位享受长期趋势 |
| **止损区间** | 跌破 48元（即有效跌破长期上升趋势线）| 无条件清仓，承认逻辑失败 |
| **催化剂加仓** | GLP-1获批 + 股价在55-60区间 | 快速加至满仓，催化剂驱动的主升浪可能启动 |
