## 3. 外部边界监控 (Exogenous Watchlist)

* [ ] **USMCA 2026日落审查 (2026年7月)**：**最高优先级！** 关注FEOC条款的最终适用范围和对中资控股工厂的影响。直接影响墨西哥工厂的利润和全球化战略。
* [x] **墨西哥关税应对**：确认可通过PROSEC计划获得0-7%优惠税率。名义关税的实质冲击有限。
* [x] **汇率管理**：2026Q1汇兑损失0.91亿元。公司已全面启动套期保值。**2026Q2验证**套保策略的实际对冲效果。
* [ ] **GE/三菱/西门子的季度订单数据**：关注三大客户每季度的燃机订单指引和积压量变化。若出现连续两季订单增速放缓至<10%，需重新评估大型零部件的增长斜率。
* [ ] **全球数据中心资本开支**：关注微软、谷歌、亚马逊、Meta的季度资本开支指引。若出现大规模削减（如连续两季削减>15%），则燃机需求逻辑可能松动。
* [ ] **人民币兑美元汇率**：关注USD/CNY是否突破6.70（持续走强）或7.10（走弱）。即使有套保，极端汇率波动仍会影响外销利润折算。

## 4. 逻辑证伪警报 (Invalidation Triggers)

**警报 A（合规毁损——致命级）**：
USMCA正式通过宽泛版"FEOC排除条款"，明确拒绝中国资本控股工厂的零关税待遇，且墨西哥工厂无法通过法人架构调整规避。
→ **触发动作：减持50-70%，重新评级至"回避"**

**警报 B（增长断裂——严重级）**：
燃机/机床订单出现以下任一信号：①GE等客户单季燃机新增订单同比下降>20%；②豪迈合同负债连续两季下降>10%；③2026H1中报机床营收增速大幅回落至<30%（vs.中性预测55%）。
→ **触发动作：减持30-50%，等待进一步数据验证**

**警报 C（盈利质量恶化——严重级）**：
毛利率连续两季跌破32%（vs.当前33.56%），且无法用产品组合偏移合理解释（即各板块毛利率均在下滑）。
→ **触发动作：减持20-30%，重新评估成本结构和定价权**

**警报 D（趋势走坏——技术性）**：
股价有效跌破MA120（除权后约52.31元，对应除权前约75.85元），且伴随成交量放大（>10万手/日）。
→ **触发动作：减持30%，技术面止损**

**警报 E（估值极端——警示级）**：
PE-TTM突破33x（5年历史最高值）且无基本面的超预期事件支撑（如超大型订单公告）。
→ **触发动作：不追高，已持有者考虑部分止盈**
