# 豪迈科技 (002595) 投后跟踪手册 (Operational Checklist)

*   [ ] **机床外销增速**：季度营收同比是否维持在 80% 以上。
*   [ ] **机床毛利率**：是否稳定在 25% 以上并向 30% 靠拢。
*   [ ] **模具市占率**：关注米其林、固特异等巨头是否调整模具自给比例。

*   [ ] **汇率变动**：USD/CNY 是否跌破 6.8（人民币大幅升值警报）。
*   [ ] **政策动态**：国家对“工业母机”增值税加计抵减清单的年度审核情况。
*   [ ] **地缘政治**：美国针对墨西哥产地零部件的转口贸易审查政策。

## 3. 逻辑证伪警报 (Invalidation Triggers)
*   **警报 A**：机床业务出现重大质量事故或大规模退货（核心竞争力崩塌）。
*   **警报 B**：燃机缸体订单连续两个季度环比下滑（算力驱动逻辑终结）。
*   **警报 C**：股价跌破 MA60（81.89 元）且三个交易日内无法收回（趋势走坏）。

## 4. 关键时间节点 (Calendar)
