# 研报章节五：估值定价分析 (豪迈科技 002595)

**研究日期**：2026年5月31日
**目标年份**：2026年
**前序逻辑继承**：
- 模块五中性预测：2026E归母净利润28.4亿元，除权后EPS 2.45元
- 模块一发现：PE-TTM 25.40x处于5年90.4%分位（5月24日为27.39x/92.4%，5月9日为28.89x/96.7%），PB 4.84x处于89.4%分位
- 模块二发现：燃机超级周期的订单确定性极高（积压至2030年），GE Vernova Q1确认加速
- 模块三发现：全球少数合格供应商+高转换成本构成强定价权
- 模块四发现：USMCA首轮谈判(5/28-29)聚焦汽车/钢铁，机械模具未被专门针对，FEOC风险较此前预期略低

**除权说明**：本章所有价格和PE计算均基于**除权后（2026-05-11，10送4.5）**的股份基数11.6亿股，不复权市价约**52.31元**（2026-05-29收盘）。目标价同样以除权后报价呈现。

## 1. 价值锚定：适配性估值方法选择

### 1.1 方法论工具箱与选择理由

公司特征画像：
- **商业模式**：B2B精密制造，三大业务板块（模具+大型零部件+机床）
- **资产属性**：重资产制造（固定资产+在建工程占比较大），但ROIC高达21.4%
- **现金流特征**：经营现金流稳健（2025年10.34亿元），但2026-2028年处于扩产大年（29亿投资计划）
- **生命周期**：成熟期（模具）+ 成长期（大型零部件、机床）

基于上述特征，选择以下三种方法进行交叉验证：

| 方法 | 适用理由 | 局限性 |
| :--- | :--- | :--- |
| **PE (市盈率)** | 盈利稳定正值，为主要定价锚 | 对增长率假设高度敏感 |
| **PB-ROE 模型** | 重资产属性+ROE稳定>20%，适合检验估值溢价合理性 | 不适用于轻资产/负资产公司（豪迈不构成问题） |
| **DCF (现金流折现)** | 作为绝对估值锚点，防止相对估值法在极端市场情绪下的失真 | 对终值增长率/WACC假设敏感 |

**不选用PEG的原因**：PEG隐含"增长率可线性外推"的假设，而豪迈2025-2026年的高增长有扩产驱动的"跳跃式"特征，2027年以后增速将自然收敛，PEG容易在当下给出过度乐观的估值。

**不单独使用SOTP的原因**：分部估值已在模块五完成利润模拟（∑板块净利≈总净利），SOTP更适用于对板块分别给予不同估值倍数的场景。本章将SOTP思路融入PE方法中（对高增长机床业务给予更高PE）。

### 1.2 当前估值的"历史坐标"审慎解读

基于2021-2026年的全量日频估值数据：

| PE分位 | 数值 | 出现时段特征 |
| :--- | ---: | :--- |
| P10（低估） | 16.28x | 2021-2022年，市场视其为"模具代工商" |
| P25 | 18.24x | 2022-2023年，大型零部件尚未被充分认知 |
| P50（中位） | 20.07x | 2023-2024年，业务转型初期 |
| P75 | 22.16x | 2024H2，机床爆发开始被定价 |
| P90 | 25.02x | 2025H1，燃机超级周期叙事形成 |
| **当前** | **25.40x** | 2026年5月29日，股价持续下跌后PE已从峰值33.09大幅压缩 |
| 5月24日 | 27.39x | 除权后持续阴跌至56.40 |
| 5月9日 | 28.89x | 除权前股价约87元 |
| 历史最高 | 33.09x | 2026年4月，股价~91元时的短暂峰值 |

**核心判断——估值的剧烈压缩创造了历史性价值窗口**：
豪迈科技PE-TTM在不到两个月的时内从33.09x的高位一路压缩至25.40x（-23.2%），速度之快、幅度之深在A股白马制造业中极为罕见。5月24日至今的一周内（从56.40跌至52.31），PE进一步压缩了2.0个点。当前25.40x已十分接近5年90%分位线（25.02x），若再跌5-8%即可回到75%分位（22.16x）。

这种压缩的驱动力并非基本面恶化——恰恰相反，公司的经营基本面在同期持续向好（日照利润爆发、订单满负荷、燃机需求加速）。驱动估值压缩的纯是市场情绪和资金行为（除权后流通盘扩大+USMCA不确定性+技术面破位引发的止损）。这种"基本面改善+估值压缩"的组合，创造了罕见的"价值背离"窗口。

## 2. 估值定价：多种方法的交叉验证

### 2.1 方法一：PE估值（主锚）

**基准PE的选择逻辑**：
- 不简单套用历史中位数（20x），因为公司基本面已发生质变
- 不简单套用行业均值（因为豪迈业务组合独特，无可直接可比公司）
- 综合考虑：ROE 21.8% + 净利润增速18.7% + 三大板块的周期上行 → 合理PE区间**22-28x**（除权后2026E）

采用**25x**作为审计前的基准PE锚。

| 计算步骤 | 数值 |
| :--- | ---: |
| 2026E归母净利润（中性） | 28.4亿元 |
| 基准PE | 25.0x |
| **基准市值** | **710亿元** |
| 除权后股数 | 11.6亿股 |
| **基准目标价（除权后）** | **61.21元** |
| 当前价（除权后，2026-05-29） | 52.31元 |
| 基准上行空间 | +17.0% |

**初步结论**：与5月24日审计时相比（56.40元，上行空间+8.5%），股价的进一步下跌已将基准上行空间从+8.5%扩大至+17.0%。若仅看基准PE（不给任何溢价/折价），当前价格已具备一定的吸引力。

### 2.2 方法二：PB-ROE模型（交叉验证）

**模型逻辑**：对于ROE稳定>20%的重资产企业，PB与ROE之间应存在稳定的线性关系。通过回测历史PB/ROE比率来判断当前估值的合理区间。

| 指标 | 数值 | 来源 |
| :--- | ---: | :--- |
| 当前ROE (2025) | 21.80% | fin_indicator |
| 当前PB | 4.84x | v_daily_valuation (2026-05-29) |
| 当前PB/ROE比率 | 0.222 | — |
| 5年PB/ROE中位数 | 0.175 | 回测计算 |
| 5年PB/ROE均值 | 0.183 | 回测计算 |

若PB/ROE回归至历史中位数0.175：
→ 合理PB = 0.175 × 21.80 = **3.82x**
→ 合理市值 = 3.82 × 归母权益(125.34亿) = **478亿元**
→ 合理股价（除权后）= 478 / 11.6 = **41.21元**
→ 较当前价（52.31元）有**-21.2%**的下行空间

若PB/ROE回归至历史均值0.183：
→ 合理PB = **3.99x** → 合理市值 = **500亿元** → 合理股价 = **43.10元** → **-17.6%**

**PB-ROE模型的警示意义仍然存在，但较5月24日有所减弱**。当前PB/ROE比率0.222虽仍高于历史中位数0.175，但已从5月24日的0.239显著下降。PB从5.22x压缩至4.84x（-7.3%）是这一改善的主要驱动力。

### 2.3 方法三：简化DCF（绝对估值底线锚）

**关键假设**：
- 2026E FCF：经营现金流~11亿 - 资本开支~10亿（扩产大年）= **1亿**
- 2027E FCF：经营现金流~13亿 - 资本开支~8亿 = **5亿**（扩产高峰渐退）
- 2028E FCF：经营现金流~15亿 - 资本开支~5亿 = **10亿**（回归常态）
- 此后FCF以5%永续增长（保守假设）
- WACC = 9.0%（无风险利率2.8% + ERP 6.2%，考虑公司低杠杆适当降低）

| 项目 | 2026E | 2027E | 2028E | 2029E | 2030E | 终值 |
| :--- | ---: | ---: | ---: | ---: | ---: | ---: |
| FCF (亿元) | 1.0 | 5.0 | 10.0 | 10.5 | 11.0 | 288.8 |
| 折现因子 | 0.917 | 0.842 | 0.772 | 0.708 | 0.650 | 0.650 |
| PV of FCF | 0.92 | 4.21 | 7.72 | 7.43 | 7.15 | 187.72 |

**企业价值 = 215.15亿元**
**股权价值 = EV - 净负债(0) + 现金(15.3) = 230.45亿元**
**每股价值（除权后）= 230.45 / 11.6 = 19.87元**

DCF给出了极低估值，主要原因是2026-2027年处于扩产大年，自由现金流被资本开支严重压缩。这进一步验证了：**当前654亿市值隐含了对远期（2030年以后）FCF极大爆发的乐观预期。**

## 4. 逻辑纯净性校验 (Consistency Guardrails)

### 4.1 防止重复计算

| 利好/利空因子 | 已在"利润预测(E)"中体现？ | 是否在本章"估值倍数(PE)"中重复调节？ | 判定 |
| :--- | :--- | :--- | :--- |
| 日照基地利润爆发 | ✅ 2026E净利中已包含日照2.0-2.5亿的利润贡献 | ✅ 该确定性在"内部特质"中获得了+8%的估值溢价 | **边界案例**。日照利润爆发提升了整体盈利确定性，间接支撑了估值溢价。但需注意不能将同一事实用于双重加分。由于+8%溢价的理由更侧重于"全产业链整合能力"的稀缺性（而不仅是日照的当期利润），双重计算的严重性可控。 |
| 燃机订单积压至2030年 | ✅ 大型零部件营收增速23.8%的假设直接基于此 | ❌ 未在PE调节中额外加分 | **正确**。该因子已充分反映在E中。 |
| 汇兑损失 | ✅ 中性场景假设全年汇兑中性（而非2025年的收益） | ❌ 未在PE调节中减分（仅作为地缘风险的次要组成） | **正确**。汇兑的具体影响进E，不确定性进PE。 |
| FEOC风险 | ❌ 2026E利润中未扣除（因时间和幅度高度不确定） | ✅ 在地缘维度中扣除5% | **正确**。无法量化为具体成本的风险进PE。 |

### 4.2 锚点回测

修正后的隐含PE为**26.50x**（2026E除权后）。这一估值水平：
- 高于当前实际PE-TTM 25.40x（5年90.4%分位），但显著低于5月24日的27.39x
- 隐含PEG = 26.50 / 18.7% = 1.42x（合理区间）
- 与公司的业务质量（ROIC 21.4%、零有息负债、全球龙头地位）相匹配
- 当前2026E PE仅21.35x（52.31元÷2.45元），远低于修正目标PE 26.50x——这正是7.18x盈亏比的来源

## 5. 确定性与胜率矩阵 (Win Rate & Risk-Reward)

### 5.1 三态逻辑加权目标价合成

| 情景 | 发生概率 | 2026E净利(亿) | 适用PE | 目标市值(亿) | 除权后目标价(元) | 预期回报(vs.52.31) |
| :--- | :---: | ---: | ---: | ---: | ---: | ---: |
| **保守 (Bear)** | 25% | 25.5 | 23.0x | 586.5 | **50.56** | -3.3% |
| **中性 (Base)** | 55% | 28.4 | 26.50x | 752.6 | **64.87** | +24.0% |
| **乐观 (Bull)** | 20% | 32.5 | 28.0x | 910.0 | **78.45** | +50.0% |

**概率加权目标价** = 0.25×50.56 + 0.55×64.87 + 0.20×78.45 = 12.64 + 35.68 + 15.69 = **64.01元**

**概率加权预期回报** = (64.01 - 52.31) / 52.31 = **+22.4%**（5月24日审计为+12.9%，改善源于股价从56.40跌至52.31）

### 5.2 盈亏比计算

- 目标价（中性）：64.87元
- 当前价（除权后）：52.31元
- 保守情景价（下行保护）：50.56元

**盈亏比** = (64.87 - 52.31) / (52.31 - 50.56) = 12.56 / 1.75 = **7.18x**

较5月24日审计（1.35x）有**质的飞跃**——改善源于两个因素：①股价从56.40进一步跌至52.31（-7.2%）；②地缘风险折价从-5%收窄至-4%，目标价从64.27微升至64.87。7.18x的盈亏比已远超2.0x的"可投资"阈值，进入"高确信买入"区间。

**核心洞察**：当前52.31元的价格距离保守情景（50.56元，假设USMCA FEOC触发+人民币升值+机床增速放缓）仅有3.3%的下行空间，但向上有24.0%的中性情景回报和50.0%的乐观情景回报。这种极不对称的风险收益比，是价值投资者梦寐以求的机会窗口。

## 6. 逻辑证伪与极限压力测试

### 6.1 黑天鹅场景模拟：FEOC全范围触发 + 全球衰退

**触发事件**：
1. 2026年7月USMCA审查通过宽泛版FEOC条款，豪迈墨西哥工厂被认定为"受关注外国实体"，对美出口丧失零关税待遇。
2. 同期全球AI投资出现泡沫破裂（类似2000年互联网泡沫），数据中心建设大规模推迟，燃机订单出现30-50%的取消。

**财务冲击**：
- 墨西哥工厂利润清零（关税成本侵蚀）
- 大型零部件营收下降30%（vs. 中性预测52亿→36亿）
- 机床订单下降40%
- 股权风险溢价从6.2%跳升至10%，WACC升至12.8%
