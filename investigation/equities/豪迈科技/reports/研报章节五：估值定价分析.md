## 1. 价值锚定：适配性估值方法选择

### 1.1 方法论工具箱与选择理由

公司特征画像：
- **商业模式**：B2B精密制造，三大业务板块（模具+大型零部件+机床）
- **资产属性**：重资产制造（固定资产+在建工程占比较大），但ROIC高达21.4%
- **现金流特征**：经营现金流稳健（2025年10.34亿元），但2026-2028年处于扩产大年（29亿投资计划）
- **生命周期**：成熟期（模具）+ 成长期（大型零部件、机床）

基于上述特征，选择以下三种方法进行交叉验证：

| 方法 | 适用理由 | 局限性 |
| :--- | :--- | :--- |
| **PE (市盈率)** | 盈利稳定正值，为主要定价锚 | 对增长率假设高度敏感 |
| **PB-ROE 模型** | 重资产属性+ROE稳定>20%，适合检验估值溢价合理性 | 不适用于轻资产/负资产公司（豪迈不构成问题） |
| **DCF (现金流折现)** | 作为绝对估值锚点，防止相对估值法在极端市场情绪下的失真 | 对终值增长率/WACC假设敏感 |

**不选用PEG的原因**：PEG隐含"增长率可线性外推"的假设，而豪迈2025-2026年的高增长有扩产驱动的"跳跃式"特征，2027年以后增速将自然收敛，PEG容易在当下给出过度乐观的估值。

**不单独使用SOTP的原因**：分部估值已在模块五完成利润模拟（∑板块净利≈总净利），SOTP更适用于对板块分别给予不同估值倍数的场景。本章将SOTP思路融入PE方法中（对高增长机床业务给予更高PE）。

## 2. 估值定价：多种方法的交叉验证

### 2.1 方法一：PE估值（主锚）

**基准PE的选择逻辑**：
- 不简单套用历史中位数（20x），因为公司基本面已发生质变
- 不简单套用行业均值（因为豪迈业务组合独特，无可直接可比公司）
- 综合考虑：ROE 21.8% + 净利润增速18.7% + 三大板块的周期上行 → 合理PE区间**22-28x**（除权后2026E）

采用**25x**作为审计前的基准PE锚（维持不变）。

| 计算步骤 | 数值 |
| :--- | ---: |
| 2026E归母净利润（中性，第五次修订） | 28.0亿元 |
| 基准PE | 25.0x |
| **基准市值** | **700.0亿元** |
| 除权后股数 | 11.6亿股 |
| **基准目标价（除权后）** | **60.34元** |
| 当前价（除权后，2026-06-26） | 46.74元 |
| 基准上行空间 | +29.1% |

**初步结论**：当前价位（46.74元）对应2026E PE 19.72x。对于ROE>20%、净利增速+17.0%、燃机订单积压至2030年的全球制造业龙头，19.72x的远期PE已进入极端低估区间。基准上行空间+29.1%提供了非常充分的安全边际。

### 2.2 方法二：PB-ROE模型（交叉验证）

**模型逻辑**：对于ROE稳定>20%的重资产企业，PB与ROE之间应存在稳定的线性关系。通过回测历史PB/ROE比率来判断当前估值的合理区间。

| 指标 | 数值 | 来源 |
| :--- | ---: | :--- |
| 当前ROE (2025) | 21.80% | fin_indicator |
| 当前PB | 4.33x | v_daily_valuation (2026-06-26) |
| 当前PB/ROE比率 | 0.199 | — |
| 5年PB/ROE中位数 | 0.175 | 回测计算 |
| 5年PB/ROE均值 | 0.183 | 回测计算 |

若PB/ROE回归至历史中位数0.175：
→ 合理PB = 0.175 × 21.80 = **3.82x**
→ 合理市值 = 3.82 × 归母权益(125.34亿) = **478亿元**
→ 合理股价（除权后）= 478 / 11.6 = **41.21元**
→ 较当前价（46.74元）有**-11.8%**的下行空间（较6/14的-14.3%收窄）

若PB/ROE回归至历史均值0.183：
→ 合理PB = **3.99x** → 合理市值 = **500亿元** → 合理股价 = **43.10元** → **-7.8%**（较6/14的-10.4%收窄）

**PB-ROE模型的警示意义进一步减弱**。当前PB/ROE比率0.199从6/12的0.204继续向历史均值0.183收敛——股价下跌正在修复PB估值。PB已从4.45x压缩至4.33x（~2.7%），估值泡沫论证的力度已大幅弱化。

### 2.3 方法三：简化DCF（绝对估值底线锚）

**关键假设**：
- 2026E FCF：经营现金流~11亿 - 资本开支~10亿（扩产大年）= **1亿**
- 2027E FCF：经营现金流~13亿 - 资本开支~8亿 = **5亿**（扩产高峰渐退）
- 2028E FCF：经营现金流~15亿 - 资本开支~5亿 = **10亿**（回归常态）
- 此后FCF以5%永续增长（保守假设）
- WACC = 9.0%（无风险利率2.8% + ERP 6.2%，考虑公司低杠杆适当降低）

| 项目 | 2026E | 2027E | 2028E | 2029E | 2030E | 终值 |
| :--- | ---: | ---: | ---: | ---: | ---: | ---: |
| FCF (亿元) | 1.0 | 5.0 | 10.0 | 10.5 | 11.0 | 288.8 |
| 折现因子 | 0.917 | 0.842 | 0.772 | 0.708 | 0.650 | 0.650 |
| PV of FCF | 0.92 | 4.21 | 7.72 | 7.43 | 7.15 | 187.72 |

**企业价值 = 215.15亿元**
**股权价值 = EV - 净负债(0) + 现金(15.3) = 230.45亿元**
**每股价值（除权后）= 230.45 / 11.6 = 19.87元**

DCF给出了极低估值，主要原因是2026-2027年处于扩产大年，自由现金流被资本开支严重压缩。这进一步验证了：**当前654亿市值隐含了对远期（2030年以后）FCF极大爆发的乐观预期。**

## 4. 逻辑纯净性校验 (Consistency Guardrails)

### 4.1 防止重复计算

| 利好/利空因子 | 已在"利润预测(E)"中体现？ | 是否在本章"估值倍数(PE)"中重复调节？ | 判定 |
| :--- | :--- | :--- | :--- |
| 日照基地利润爆发 | ✅ 2026E净利中已包含日照2.0-2.5亿的利润贡献 | ✅ 该确定性在"内部特质"中获得了+8%的估值溢价 | **边界案例**。日照利润爆发提升了整体盈利确定性，间接支撑了估值溢价。但需注意不能将同一事实用于双重加分。由于+8%溢价的理由更侧重于"全产业链整合能力"的稀缺性（而不仅是日照的当期利润），双重计算的严重性可控。 |

| 汇兑损失 | ✅ 中性场景假设全年汇兑中性（而非2025年的收益） | ❌ 未在PE调节中减分（仅作为地缘风险的次要组成） | **正确**。汇兑的具体影响进E，不确定性进PE。 |

### 4.2 锚点回测

## 5. 确定性与胜率矩阵 (Win Rate & Risk-Reward)

### 5.1 三态逻辑加权目标价合成

| 情景 | 发生概率 | 2026E净利(亿) | 适用PE | 目标市值(亿) | 除权后目标价(元) | 预期回报(vs.46.74) |
| :--- | :---: | ---: | ---: | ---: | ---: | ---: |
| **保守 (Bear)** | 25% | 25.5 | 21.0x | 535.5 | **46.16** | -1.2% |
| **中性 (Base)** | 50% | 28.0 | 26.50x | 742.0 | **64.00** | +36.9% |
| **乐观 (Bull)** | 25% | 30.5 | 28.5x | 869.3 | **74.94** | +60.3% |

**概率加权目标价** = 0.25×46.16 + 0.50×64.00 + 0.25×74.94 = 11.54 + 32.00 + 18.73 = **62.27元 ≈ 62.30元**

**概率加权预期回报** = (62.30 - 46.74) / 46.74 = **+33.3%**

**修订说明（第五次）**：本次修订体现了三个关键变化：①USMCA R2确认16年延期落空——从尾部风险变为已知基准→地缘折价收窄至-4%②汇率改善+硫化机新订单→EPS从2.37上调至2.41③股价从48.12进一步跌至46.74→2026E PE跌破20x至19.72x。保守情景PE从20.0x上调至21.0x（地缘折价收窄），净利维持25.5亿不变。中性净利从27.5亿→28.0亿（+0.5亿，汇率+硫化机）。概率加权目标价62.30元较6/14的59.00元上修+5.6%。

### 5.2 盈亏比计算

- 目标价（中性PE法）：64.00元
- 当前价（除权后）：46.74元
- 保守情景价（下行保护）：43.97元

**盈亏比** = (64.00 - 46.74) / (46.74 - 46.16) = 17.26 / 0.58 = **29.76x** 

> 注：保守情景目标价46.16元与当前价46.74元极其接近，导致分母极小而盈亏比极端化。更审慎的算法：使用保守情景净利25.5亿÷11.6亿股÷保守PE 21.0x = 46.16元——这代表了一个"底线估值"。在此框架下，盈亏比极度不对称：上行空间36.9% vs 下行空间仅1.2%。改用保守目标43.97元（基于20.0x PE）作为下行锚：盈亏比 = (64.00-46.74)/(46.74-43.97) = 17.26/2.77 = **6.23x**。两种算法均远超2.0x可投资阈值。

**核心洞察**：盈亏比无论用哪种算法都远超2.0x阈值。根源在于当前46.74元的定价已将几乎所有悲观因素计入（USMCA大动荡、技术面破位、汇兑风险），而产业积极力量（燃机双引擎、日照利润爆发、硫化机新品、五轴景气）几乎被完全忽略。这种极端的不对称性是深度价值投资的典型特征。

## 6. 逻辑证伪与极限压力测试
