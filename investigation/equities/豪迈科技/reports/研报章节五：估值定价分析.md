## 1. 价值锚定：适配性估值方法选择

### 1.1 方法论工具箱与选择理由

公司特征画像：
- **商业模式**：B2B精密制造，三大业务板块（模具+大型零部件+机床）
- **资产属性**：重资产制造（固定资产+在建工程占比较大），但ROIC高达21.4%
- **现金流特征**：经营现金流稳健（2025年10.34亿元），但2026-2028年处于扩产大年（29亿投资计划）
- **生命周期**：成熟期（模具）+ 成长期（大型零部件、机床）

基于上述特征，选择以下三种方法进行交叉验证：

| 方法 | 适用理由 | 局限性 |
| :--- | :--- | :--- |
| **PE (市盈率)** | 盈利稳定正值，为主要定价锚 | 对增长率假设高度敏感 |
| **PB-ROE 模型** | 重资产属性+ROE稳定>20%，适合检验估值溢价合理性 | 不适用于轻资产/负资产公司（豪迈不构成问题） |
| **DCF (现金流折现)** | 作为绝对估值锚点，防止相对估值法在极端市场情绪下的失真 | 对终值增长率/WACC假设敏感 |

**不选用PEG的原因**：PEG隐含"增长率可线性外推"的假设，而豪迈2025-2026年的高增长有扩产驱动的"跳跃式"特征，2027年以后增速将自然收敛，PEG容易在当下给出过度乐观的估值。

**不单独使用SOTP的原因**：分部估值已在模块五完成利润模拟（∑板块净利≈总净利），SOTP更适用于对板块分别给予不同估值倍数的场景。本章将SOTP思路融入PE方法中（对高增长机床业务给予更高PE）。

### 1.2 当前估值的"历史坐标"审慎解读

基于2021-2026年的全量日频估值数据：

| PE分位 | 数值 | 出现时段特征 |
| :--- | ---: | :--- |
| P10（低估） | 16.28x | 2021-2022年，市场视其为"模具代工商" |
| P25 | 18.24x | 2022-2023年，大型零部件尚未被充分认知 |
| P50（中位） | 20.07x | 2023-2024年，业务转型初期 |
| P75 | 22.16x | 2024H2，机床爆发开始被定价 |
| P90 | 25.02x | 2025H1，燃机超级周期叙事形成 |
| **当前** | **24.66x** | 2026年6月5日，股价持续下跌后PE已从峰值33.09大幅压缩 |
| 5月31日 | 25.40x | MA120正在测试 |
| 5月9日 | 28.89x | 除权前股价约87元 |
| 历史最高 | 33.09x | 2026年4月，股价~91元时的短暂峰值 |

**核心判断——估值的剧烈压缩继续深化，但盈利预测同步下修**：
豪迈科技PE-TTM在不到三个月的时间内从33.09x的高位一路压缩至24.66x（-25.5%）。5月31日至今的一周内（从52.31跌至50.79），PE进一步压缩了0.74个点。当前24.66x略低于5年90%分位线（25.31x）。

但需同步注意的是：章节八初版曾将2026E净利下修至25.5亿，其核心问题在于将基建期支出与调试运行成本的会计影响混同。经三次修订后，当前2026E PE = 50.79 / (27.5/11.6) = **21.43x**。该估值对于ROE>20%、净利增速+14.9%的全球制造业龙头而言，具备显著吸引力。

## 2. 估值定价：多种方法的交叉验证

### 2.1 方法一：PE估值（主锚）

**基准PE的选择逻辑**：
- 不简单套用历史中位数（20x），因为公司基本面已发生质变
- 不简单套用行业均值（因为豪迈业务组合独特，无可直接可比公司）
- 综合考虑：ROE 21.8% + 净利润增速18.7% + 三大板块的周期上行 → 合理PE区间**22-28x**（除权后2026E）

采用**25x**作为审计前的基准PE锚（较此前26.50x下调，反映三爬坡对短期盈利可见度的削弱）。

| 计算步骤 | 数值 |
| :--- | ---: |
| 2026E归母净利润（中性，第三次修订） | 27.5亿元 |
| 基准PE | 25.0x |
| **基准市值** | **687.5亿元** |
| 除权后股数 | 11.6亿股 |
| **基准目标价（除权后）** | **59.27元** |
| 当前价（除权后，2026-06-05） | 50.79元 |
| 基准上行空间 | +16.7% |

**初步结论**：当前价位（50.79元）对应2026E PE 21.43x。对于ROE>20%、净利增速+14.9%的全球制造业龙头，21.43x的远期PE已进入低估区间。基准上行空间+16.7%，叠加全变量审计矩阵的+6%调节后，预期回报具备较强吸引力。

### 2.2 方法二：PB-ROE模型（交叉验证）

**模型逻辑**：对于ROE稳定>20%的重资产企业，PB与ROE之间应存在稳定的线性关系。通过回测历史PB/ROE比率来判断当前估值的合理区间。

| 指标 | 数值 | 来源 |
| :--- | ---: | :--- |
| 当前ROE (2025) | 21.80% | fin_indicator |
| 当前PB | 4.70x | v_daily_valuation (2026-06-05) |
| 当前PB/ROE比率 | 0.222 | — |
| 5年PB/ROE中位数 | 0.175 | 回测计算 |
| 5年PB/ROE均值 | 0.183 | 回测计算 |

若PB/ROE回归至历史中位数0.175：
→ 合理PB = 0.175 × 21.80 = **3.82x**
→ 合理市值 = 3.82 × 归母权益(125.34亿) = **478亿元**
→ 合理股价（除权后）= 478 / 11.6 = **41.21元**
→ 较当前价（50.79元）有**-18.8%**的下行空间

若PB/ROE回归至历史均值0.183：
→ 合理PB = **3.99x** → 合理市值 = **500亿元** → 合理股价 = **43.10元** → **-15.1%**

**PB-ROE模型的警示意义仍然存在，但较5月24日有所减弱**。当前PB/ROE比率0.222虽仍高于历史中位数0.175，但已从5月24日的0.239显著下降。PB从5.22x压缩至4.84x（-7.3%）是这一改善的主要驱动力。

### 2.3 方法三：简化DCF（绝对估值底线锚）

**关键假设**：
- 2026E FCF：经营现金流~11亿 - 资本开支~10亿（扩产大年）= **1亿**
- 2027E FCF：经营现金流~13亿 - 资本开支~8亿 = **5亿**（扩产高峰渐退）
- 2028E FCF：经营现金流~15亿 - 资本开支~5亿 = **10亿**（回归常态）
- 此后FCF以5%永续增长（保守假设）
- WACC = 9.0%（无风险利率2.8% + ERP 6.2%，考虑公司低杠杆适当降低）

| 项目 | 2026E | 2027E | 2028E | 2029E | 2030E | 终值 |
| :--- | ---: | ---: | ---: | ---: | ---: | ---: |
| FCF (亿元) | 1.0 | 5.0 | 10.0 | 10.5 | 11.0 | 288.8 |
| 折现因子 | 0.917 | 0.842 | 0.772 | 0.708 | 0.650 | 0.650 |
| PV of FCF | 0.92 | 4.21 | 7.72 | 7.43 | 7.15 | 187.72 |

**企业价值 = 215.15亿元**
**股权价值 = EV - 净负债(0) + 现金(15.3) = 230.45亿元**
**每股价值（除权后）= 230.45 / 11.6 = 19.87元**

DCF给出了极低估值，主要原因是2026-2027年处于扩产大年，自由现金流被资本开支严重压缩。这进一步验证了：**当前654亿市值隐含了对远期（2030年以后）FCF极大爆发的乐观预期。**

## 4. 逻辑纯净性校验 (Consistency Guardrails)

### 4.1 防止重复计算

| 利好/利空因子 | 已在"利润预测(E)"中体现？ | 是否在本章"估值倍数(PE)"中重复调节？ | 判定 |
| :--- | :--- | :--- | :--- |
| 日照基地利润爆发 | ✅ 2026E净利中已包含日照2.0-2.5亿的利润贡献 | ✅ 该确定性在"内部特质"中获得了+8%的估值溢价 | **边界案例**。日照利润爆发提升了整体盈利确定性，间接支撑了估值溢价。但需注意不能将同一事实用于双重加分。由于+8%溢价的理由更侧重于"全产业链整合能力"的稀缺性（而不仅是日照的当期利润），双重计算的严重性可控。 |
| 燃机订单积压至2030年 | ✅ 大型零部件营收增速23.8%的假设直接基于此 | ❌ 未在PE调节中额外加分 | **正确**。该因子已充分反映在E中。 |
| 汇兑损失 | ✅ 中性场景假设全年汇兑中性（而非2025年的收益） | ❌ 未在PE调节中减分（仅作为地缘风险的次要组成） | **正确**。汇兑的具体影响进E，不确定性进PE。 |
| FEOC风险 | ❌ 2026E利润中未扣除（因时间和幅度高度不确定） | ✅ 在地缘维度中扣除5% | **正确**。无法量化为具体成本的风险进PE。 |

### 4.2 锚点回测

修正后的隐含PE为**26.50x**（2026E除权后）。这一估值水平：
- 显著高于当前实际PE-TTM 24.66x（5年89.0%分位），也高于5月31日的25.40x
- 隐含PEG = 26.50 / 6.6%（修正后净利增速）= 4.02x（偏高，反映三爬坡期盈利增速放缓）
- 与公司的业务质量（ROIC 21.4%、零有息负债、全球龙头地位）相匹配，但26.50x的PE要求2027-2028年盈利增速回归至15%+方能消化
- 当前2026E PE约21.43x（50.79元÷2.37元），低于修正目标PE 26.50x——这是12.03元安全边际的来源，对应4.95x盈亏比

## 5. 确定性与胜率矩阵 (Win Rate & Risk-Reward)

### 5.1 三态逻辑加权目标价合成

| 情景 | 发生概率 | 2026E净利(亿) | 适用PE | 目标市值(亿) | 除权后目标价(元) | 预期回报(vs.50.79) |
| :--- | :---: | ---: | ---: | ---: | ---: | ---: |
| **保守 (Bear)** | 25% | 25.5 | 22.0x | 561.0 | **48.36** | -4.8% |
| **中性 (Base)** | 50% | 27.5 | 26.50x | 728.8 | **62.82** | +23.7% |
| **乐观 (Bull)** | 25% | 30.0 | 28.0x | 840.0 | **72.41** | +42.6% |

**概率加权目标价** = 0.25×48.36 + 0.50×62.82 + 0.25×72.41 = 12.09 + 31.41 + 18.10 = **61.60元**

**概率加权预期回报** = (61.60 - 50.79) / 50.79 = **+21.3%**

### 5.2 盈亏比计算

- 目标价（中性PE法）：62.82元
- 当前价（除权后）：50.79元
- 保守情景价（下行保护）：48.36元

**盈亏比** = (62.82 - 50.79) / (50.79 - 48.36) = 12.03 / 2.43 = **4.95x**

**核心洞察**：在修复了内部数值不一致后，盈亏比从2.26x跃升至4.95x——远超2.0x可投资阈值，接近5/31的7.18x水平。当前50.79元距离保守情景（48.36元）仅4.8%下行空间，却享有23.7%的中性情景上行空间。这种高度不对称的风险收益比，配得上"买入"评级。
