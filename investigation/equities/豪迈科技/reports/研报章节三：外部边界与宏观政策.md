# 研报章节三：外部边界与宏观政策 (豪迈科技 002595)

**研究日期**：2026年6月7日
**目标年份**：2026年
**核心逻辑**：豪迈科技的全球化布局正经历从"贸易型出口"向"制造型本地化"的范式切换。USMCA 2026日落审查的FEOC条款是当前最具杀伤力的外部风险，但公司通过PROSEC计划、多层法人架构和套期保值已构建了有实效的防御体系。

---

## 1. 宏观环境变量：折现率、汇率与通胀的三重传导

### 1.1 利率环境与估值折现率

2026年上半年，全球主要央行进入降息周期的节奏出现分化：

- **美联储**：2026年5月联邦基金利率维持在3.75-4.00%。市场预期年内还有1-2次降息，但通胀韧性（核心PCE仍在2.8%附近）限制了降息空间。对中国制造业出口企业而言，美国利率的下行边际有利于全球资本开支周期的延续，间接支撑了燃机、轮胎模具等资本品需求。
- **中国人民银行**：LPR 1年期3.10%，5年期3.60%。适度的宽松政策有利于降低企业融资成本，但豪迈科技几乎零有息负债的财务结构使其对国内利率变动的敏感度极低。
- **对估值的影响**：理论上，无风险利率下行应推动DCF估值的折现率下移从而提升合理PE。但当前豪迈PE-TTM处于92.4%的历史分位，利率下行对估值的边际推升空间已被充分甚至过度定价。**利率端的利好不是当前估值的核心支撑逻辑**。

### 1.2 汇率损益：利润表的"钟摆"效应

汇率是豪迈科技利润表中波动最大的单一变量。公司外销收入占比估计在**40-50%**之间（以美元和欧元结算为主），汇兑损益通过"财务费用"科目直接冲击净利润。

**2026Q1的汇兑冲击回顾**：
- Q1财务费用为净支出0.62亿元，相比2025Q1净收益0.29亿元，单季汇兑负面冲击约**0.91亿元**。
- 2026年4月，人民币兑美元从6.89阶段性走强至6.83附近，升幅约0.87%。这一波动导致Q1末存量外币资产的折算损失。
- 若剔除该汇兑因素，2026Q1主营业务利润增速约为**+17-20%**（vs. 名义净利增速-0.83%）。

**套期保值的攻守转换**：
公司已在2026年4月业绩说明会上确认将全面启动套期保值策略。这意味着后续季度的利润表对汇率波动的敏感度将显著降低。但需注意两个局限：
1. 套期保值只能锁定未来现金流，对存量外币资产（应收账款、货币资金）的折算损益无效。
2. 套保比例通常为60-80%，仍会留有一定的风险敞口。

**情景模拟**：假设2026全年USD/CNY均值在6.80-6.90区间波动（vs. 2025年均值约7.05），套保后的净汇兑影响约在±0.3亿元以内，对全年30亿+的利润体量影响有限。

### 1.3 通胀传导效率

- **原材料端**：豪迈科技的主要原材料为钢材、铸铁、合金等，大宗商品属性较强。2025-2026年，全球钢材价格处于中低位震荡（热轧卷板约3,800-4,200元/吨），对成本端压力不大。
- **能源端**：大型铸造业务（日照基地）对电力消耗较大。2026年中国工业电价保持稳定，没有出现显著上涨。相比欧洲竞争对手（电力成本约为中国的2-3倍），豪迈的能源成本优势在通胀压力下反而被放大。
- **人工端**：公司所在的山东潍坊地区属于制造业重镇，劳动力供应相对充裕。2026年制造业工资增速约5-8%，属于可控范围。
- **通胀传导能力评估**：公司对下游客户的议价能力较强（全球少数合格供应商地位），但B2B长期合同通常包含成本联动条款，成本上涨可以部分转嫁。综合来看，通胀对公司的利润侵蚀能力**较弱**，不属于核心风险变量。

## 2. 地缘政治风险：全球化布局的"压力测试"

### 2.1 墨西哥关税与PROSEC对冲机制

2026年，墨西哥对1,463项商品（含机械、模具、钢铁制品）加征**25%-50%**的名义进口关税，这对在墨西哥设厂的中国制造商普遍构成冲击。

**豪迈的防御路径——PROSEC计划**：
- PROSEC（Programa de Promoción Sectorial，行业推广计划）是墨西哥政府为鼓励特定制造业发展而设立的关税优惠机制。汽车、机械、电子等行业的企业可申请获得0-7%的优惠税率。
- 豪迈墨西哥工厂作为已实际投产的制造实体（非简单的组装/贴标），具备申请PROSEC的完整资质。对于汽车和机械行业，关税可从名义的35-50%降至**0-7%**。
- **结论**：名义关税对豪迈墨西哥工厂的实质冲击**有限**。公司通过"深度本地化制造"（而非简单转口贸易）已建立了实质性的关税护城河。

### 2.2 USMCA 2026日落审查：FEOC条款——墨加正式请求续期，风险进一步边际缓和（2026年6月7日审计更新）

2026年7月1日，USMCA（美墨加协定）将迎来首次"日落审查"（Joint Review）。这是当前豪迈科技面临的最重大地缘政治风险。

**审查时间线与最新进展**：

| 时间节点 | 事件 | 影响评估 |
| :--- | :--- | :--- |
| 2026年3月5日 | USTR与墨西哥经济部长正式启动双边技术讨论 | 进程开启 |
| 2026年3月16日 | 首轮美墨双边技术谈判 | 初步交换立场 |
| 2026年5月20日 | 15名参议院民主党人致函USTR，要求遏制中国在墨西哥投资 | 政治压力上升 |
| **2026年5月28-29日** | **美墨首轮正式双边谈判在墨西哥城完成** | **关键进展：聚焦汽车/钢铁，机械模具未被专门针对** |
| 2026年6月16-17日 | 第二轮谈判（华盛顿DC），新增农业+公平竞争议题 | 关注"经济安全"议题是否扩大化 |
| 2026年7月1日 | 日落审查截止日（非硬截止） | 若未达成一致则自动进入10年年度审查期 |
| 2026年7月20日当周 | 第三轮谈判（墨西哥城） | 可能为最终轮 |

**首轮谈判及后续关键解读（6月7日更新）**：

1. **谈判完成了**：USTR于5月29日发布正式声明，确认首轮谈判"聚焦于汽车原产地规则、钢铁和铝、经济安全"。美方目标为"减少对墨贸易逆差并强化美国供应链"。

2. **机械/模具未被列为优先议题**：谈判的核心战场是**汽车**行业——美方提出50%美国成分+82%北美成分的要求，远超当前USMCA的75%北美成分标准。钢铁和铝也被列为重点。但是，机械、模具、铸造等工业中间品并未出现在首轮谈判的官方议题列表中。这对豪迈是**边际正面信号**。

3. **关税将成为新常态**：USTR Greer于5月26日明确表示"打算在对墨西哥和加拿大有大量贸易逆差的情况下保持关税"。这意味着USMCA不会回到完全零关税状态，但特定产品的优惠税率仍将存在。PROSEC计划的0-7%优惠税率预计将继续有效。

4. **FEOC概念尚未成为官方谈判核心**：United Steelworkers等工会在其立场文件中明确要求"收紧FEOC条款，防止中国资本通过墨西哥获得零关税待遇"。但USTR的官方声明中并未将FEOC单独列为谈判议题。这是一个重要的信号——FEOC目前更多是政治修辞而非技术性谈判文本。

5. **加拿大也被纳入博弈**：加拿大贸易部长LeBlanc于6月3日赴华盛顿会见USTR Greer，正式请求USMCA续期16年并提交了具体提案。三方博弈格局增加了复杂性但也分散了美方的谈判火力。

6. **墨西哥正式请求续期（6月1日新增）**：墨西哥经济部长Ebrard于6月1日正式致函USTR和加拿大，请求USMCA续期16年。至此，墨加两国均已正式表态支持续期——三国均有维护框架的政治诉求。

**最差情景概率修正**（经6月7日审计）：

| 情景 | 触发条件 | 对豪迈的影响 | 6月7日概率 | 5月31日概率 | 变化 |
| :--- | :--- | :--- | :---: | :---: | :---: |
| **最佳** | FEOC范围局限于"敏感行业"（汽车/电池/EV），机械/模具/铸造不在列 | 几乎无影响 | ~30% | ~25% | **+5pct** |
| **中性** | FEOC覆盖一般制造业，但墨西哥工厂通过多层法人架构被认定为"非中国资本控股" | 有限影响 | ~40% | ~40% | 不变 |
| **最差** | FEOC宽泛适用，且穿透认定所有中资控股工厂 | 墨西哥工厂失去零关税 | **~30%** | ~35% | **-5pct** |

**概率修正理由（6月7日更新）**：墨加两国于6月1-3日相继正式请求USMCA续期16年，表明三方均有维护框架的政治诉求，降低了美国单方面退出或激进修订的概率。CSIS 3月报告评估"painful extension"为基准情景。机械/模具/铸造行业的FEOC优先级进一步降低。但后续两轮谈判（6/16-17和7/20）尚未完成，"经济安全"概念的宽泛性仍需警惕。

### 2.3 供应链重构 (China+1) 的双刃剑效应

"China+1"（在中国之外建立备用供应链）的全球趋势对豪迈产生了**非对称的利好**：

- **正面效应**：豪迈的墨西哥和埃及工厂恰好是跨国客户实现"供应链多元化"的理想选择。公司通过全球化产能布局，反而成为客户降低供应链集中风险的合作伙伴。这正是之前研报中所述的"地缘免疫型白马"的核心逻辑。
- **成本侧影响**：海外产能的运营成本（人工、管理、合规）高于国内，但公司通过输出高附加值产线（而非低端组装）维持了足够的利润空间。墨西哥工厂产销两旺的现状证明了这一模式的商业可行性。

## 3. 监管与产业政策

### 3.1 国家发展规划契合度

豪迈科技的业务布局与当前中国产业政策的战略方向高度吻合：

| 政策领域 | 具体政策/法规 | 出台时间 | 对豪迈的影响 |
| :--- | :--- | :--- | :--- |
| **产业链安全** | 《工业和供应链安全保障条例》 | 2026年3月31日发布，4月起实施 | 强化关键制造环节自主性审计，利好已实现核心部件自制的豪迈机床业务 |
| **高端装备** | 《中国制造2025》战略 | 持续推进 | 五轴数控机床列为重点攻关领域 |
| **绿色制造** | 双碳政策体系 | 持续推进 | 电加热硫化机（零碳排放）作为绿色轮胎制造装备，受益于节能减排政策 |
| **新能源** | 风电/光伏装机规划 | 持续推进 | 大型零部件业务中的风电铸件受益于风电装机增长 |

**核心判断**：豪迈科技是少数在多个政策维度上同时受益的制造业标的。其业务布局与"自主可控+绿色制造+高端装备"三重政策方向均高度契合。但需注意，政策契合度高不等于可以享受估值溢价——政策红利最终需要通过订单和利润来验证。

### 3.2 行业准入与环保约束

- **铸造行业的环保门槛**：大型铸造属于高能耗、高排放行业，中国近年来持续收紧铸造行业的环保标准。但豪迈日照基地作为新建项目，在建设之初就按最高环保标准设计（粉尘排放、废气处理等），环保合规成本已在项目预算中。更严格的环保执法反而构成对中小铸造竞争对手的**供给侧出清效应**，从长期看有利于头部企业份额提升。
- **机床行业的技术准入**：五轴联动数控机床涉及军工和航空航天等敏感领域，存在出口管制和技术转让限制。豪迈目前机床业务主要面向国内市场和民用领域，短期内不受出口管制影响。但若未来拓展海外机床销售，需关注瓦森纳协定等国际技术管制框架。

## 总结：外部风险的真实轮廓

经过全维度审计，豪迈科技面临的外部风险呈现典型的"金字塔"结构：

- **塔尖（低频高烈度，需持续监控）**：USMCA FEOC条款若宽泛适用，将构成单一最大外部风险。但7月审查前存在高度不确定性，且公司有多层对冲工具。
- **塔身（中频中烈度，已主动管理）**：人民币汇率波动为公司利润表的常态波动源，但随着套期保值全面启动，边际影响将趋于收敛。
- **塔基（高频低烈度，基本免疫）**：原材料价格波动、利率变动、国内环保政策收紧等对公司影响有限。

**核心判断**：豪迈科技的全球化布局已从"被动应对地缘风险"进入"主动利用地缘分化获利"的阶段。墨西哥和埃及工厂的存在不是成本负担，而是战略性实物资产——它们使得公司能够在全球贸易壁垒上升的时代，反而获得竞争对手无法复制的市场准入优势。
