# 研报章节三：竞争格局与地位

**研究日期：2026年07月01日**
**承接前序模块核心发现**：模块一发现Q1 2026利润增长失速；模块二确认三大业务终端市场景气度整体向上，但风电铸件竞争加剧。本模块从竞争维度评估公司在博弈场中的生存能力与获利空间。

---

## 一、轮胎模具：全球龙头，寡头格局稳固

### 1.1 全球市场集中度

| 指标 | 数据 |
|:---|:---|
| 2024年全球轮胎模具市场 | ~16.3亿美元 |
| 2025-2031年CAGR | 3.11% |
| CR3（前三大市占率） | ~41% |
| CR5（前五大市占率） | ~53% |
| 豪迈科技全球市占率 | **32.75%**（2024年） |
| 豪迈科技国内市占率 | **55%+** |
| 覆盖全球Top 75轮胎企业 | 66家 |
| 出口覆盖国家 | 79个 |

**竞争梯队**：
- **第一梯队（>10%全球份额）**：豪迈科技（Himile）——独一档
- **第二梯队（3-8%）**：Dynamic Design（韩国）、巨轮智能（中国）、天阳模具（中国）、Hankook Precision Works（韩国）
- **第三梯队（<3%）**：山东万通、安徽大道模具、King Machine、A-Z、HERBERT、SeYoung TMS、Shinko Mold等

### 1.2 与主要竞争对手的量化对比

| 指标 | 豪迈科技 | 巨轮智能 |
|:---|:---|:---|
| 2025年营收(亿元) | 110.78 | 7.88 |
| 轮胎模具收入(亿元) | 55.09 | 2.97 |
| 2025年净利润(亿元) | 23.93 | -2.32 |
| ROE | 21.80% | -29.85% |
| 毛利率 | 33.56% | 10.37% |
| 净利率 | 21.62% | -29.47% |
| 资产负债率 | 18.49% | 48.37% |
| 模具业务毛利率 | 39.98% | ~10-15%(估) |

**差距根源分析**：
1. **规模效应鸿沟**：豪迈科技模具营收是巨轮智能的18.5倍，巨大规模差异带来采购成本、制造费用分摊的碾压性优势。
2. **自制设备壁垒**：豪迈科技自成立起便自制专用机床（电火花机床、五轴加工中心），模具生产设备自给率高，而竞争对手依赖外购进口设备，资本开支和维护成本差异巨大。
3. **客户覆盖广度**：豪迈覆盖全球66/75顶级轮胎企业，而巨轮等主要服务国内市场。轮胎企业供应商认证周期长（通常2-3年），转换成本极高，形成强锁定效应。
4. **技术密度**：豪迈在电加热模具、3D打印钢片、智能化设计系统等前沿领域持续投入，研发费用6.56亿元（占营收5.94%），而巨轮智能研发费用规模极低且持续亏损。

### 1.3 护城河识别

| 护城河维度 | 强度 | 核心依据 |
|:---|:---|:---|
| **规模成本壁垒** | ★★★★★ | 全球最大产能，自制设备降低CAPEX，规模碾压竞争对手 |
| **转换成本** | ★★★★☆ | 轮胎企业供应商认证2-3年+模具定制化强绑定 |
| **全球布局壁垒** | ★★★★☆ | 泰国/墨西哥/美国本地化产能，关税规避+近距离服务 |
| **专利/技术壁垒** | ★★★☆☆ | 电加热技术、3D打印工艺有专利保护，但非垄断性 |
| **品牌溢价** | ★★★☆☆ | Continental连续6年"最佳供应商"，但模具差异化有限 |

**综合护城河评级：极宽（Wide Moat）**。在轮胎模具这个特定赛道，豪迈科技具备近乎垄断性的竞争地位。全球市占率32.75%且有继续提升趋势（主要竞争对手巨轮智能持续亏损收缩），未来有望进一步巩固"一超多强"格局。

## 二、大型零部件：绑定头部客户，燃气轮机差异化突围

### 2.1 竞争格局特征

大型零部件（铸造+精加工）是一个分散度较高的市场，涉及风电铸件、燃气轮机铸件、工程机械铸件等多个细分领域。主要竞争对手包括：

- **风电铸件**：日月股份、吉鑫科技、永冠新材、山东龙马等
- **燃气轮机铸件**：行业壁垒更高，竞争对手较少，豪迈科技凭借与GE/西门子/三菱的长期合作占据先发优势

### 2.2 豪迈科技的差异化竞争

| 维度 | 豪迈科技 | 传统铸件企业 |
|:---|:---|:---|
| 铸造+精加工一体化 | 全流程覆盖（毛坯→成品） | 多限于铸造或精加工单一环节 |
| 客户绑定深度 | GE/西门子/三菱/东方电气/上海电气等头部企业长期战略合作 | 客户相对分散 |
| 燃气轮机技术壁垒 | 可生产1000MW超超临界汽轮机缸体等高端铸件 | 多数企业不具备 |
| 产能扩张速度 | 2025年6.5万吨新产能投产+7万吨备案中 | 扩张节奏受行业景气度制约 |

**竞争护城河**：中等宽度。客户认证壁垒高（GE等对供应商审核周期长达3-5年），且燃气轮机铸件的技术门槛（高温合金、大型复杂腔体铸造）显著高于普通风电铸件。但风电铸件领域竞争激烈（日月股份等专业化企业产能规模大），毛利率从2024年的25.67%降至2025年的21.58%即是竞争加剧的直接体现。

### 2.3 产业链定价权

- **对上游（钢材/铸造材料）**：议价权有限，材料成本占比22.91%，随行就市
- **对下游（GE/西门子等）**：燃气轮机零部件供需缺口巨大（OEM产能已售罄至2028年），当前处于**供方市场**，议价权较强
- **对下游（风电整机商）**：风电铸件供应相对充裕，议价权较燃气轮机弱

## 三、数控机床：后发追赶者，差异化路径清晰

### 3.1 竞争格局

| 梯队 | 代表企业 | 特征 |
|:---|:---|:---|
| 第一梯队（国际） | DMG MORI、Mazak、Okuma | 高端五轴市场主导，技术最领先 |
| 第二梯队（国内龙头） | 科德数控、海天精工、纽威数控 | 国产五轴领军，规模快速增长 |
| 第三梯队（追赶者） | **豪迈科技**、创世纪、国盛智科 | 特定领域差异化竞争 |

豪迈科技2025年数控机床营收9.68亿元（+142.59%），而科德数控2024年营收约6.05亿元（+33.85%）。豪迈的机床营收增速和绝对规模已经在接近甚至超过部分专业机床企业，但这是在"自产自用+外销"的混合模式下实现的，外销部分的竞争力尚需时间验证。

### 3.2 差异化优势

| 维度 | 豪迈科技模式 | 科德数控等专业化模式 |
|:---|:---|:---|
| 技术来源 | 20+年内部自用验证后才外溢 | 创立之初即面向外部市场 |
| 全产业链 | 铸造→热处理→加工→装配→检测全覆盖 | 多依赖外购核心部件 |
| 应用验证 | 在自用生产线上大规模验证可靠性 | 需客户试用逐步验证 |
| 业务协同 | 与模具/铸件业务共享客户渠道 | 独立开拓市场 |

**核心观点**：豪迈科技的机床业务本质是"内部工具外溢"模式——20多年的自用验证提供了工业级可靠性背书，这是纯机床初创企业无法复制的。但也存在品牌认知度不足（在外部机床市场仍需积累）、销售渠道建设尚处早期等挑战。

## 四、交叉验证：供应链与订单证据

### 4.1 真实订单验证

- **轮胎模具**：2025年内销增速32.30%（远超外销17.62%），反映中国轮胎企业加速出海建厂（赛轮轮胎、玲珑轮胎等）带来的模具需求。中国轮胎企业在东南亚、墨西哥等地建厂，豪迈泰国、豪迈墨西哥两大海外基地直接受益。
- **燃气轮机**：GE Vernova产能售罄至2028年、西门子能源2025年燃机销量194台（接近2024年两倍）、三菱重工计划两年产能翻倍——全球三大OEM均满产满销，且豪迈科技与三者均有供应关系，订单景气度极高。2025年豪迈科技年报披露6.5万吨铸件产能已投产+7万吨备案扩产，是**产能端对订单景气的物理验证**。
- **机床**：2025年机床库存同比+121%，生产量同比+154%，说明公司正在积极备货、扩产以满足订单增长；自产自用设备从2024年的1.91亿元增至2025年的4.59亿元（+140.25%），说明公司自身也在加速设备更新，侧面验证机床产品的可靠性。

### 4.2 供应商端渗透验证

2025年前五大供应商采购额19.77亿元（占比26.13%），其中关联方采购12.60%（可能指向豪迈集团内部原材料或设备供应），供应商集中度较高但属于制造业常见特征。

## 五、竞争地位总结

| 业务板块 | 竞争地位 | 核心优势 | 主要风险 |
|:---|:---|:---|:---|
| 轮胎模具 | **全球绝对龙头**（市占率32.75%） | 规模效应、客户锁定、自制设备、全球布局 | 行业增速有限（3-6%CAGR） |
| 大型零部件 | **燃机铸件领先、风电铸件中游** | 头部客户绑定、燃气轮机技术壁垒、一体化服务 | 风电铸件竞争加剧，毛利率下行 |
| 数控机床 | **差异化追赶者** | 20+年自用验证、全产业链、业务协同 | 品牌认知度不足、规模不经济风险 |
