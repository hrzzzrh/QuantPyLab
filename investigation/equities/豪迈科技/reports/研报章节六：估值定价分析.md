# 研报章节六：估值定价分析（定稿修订）

**研究日期：2026年07月01日**
**当前股价（不复权）**：46.49元（2026年7月1日收盘）
**当前市值**：539.28亿元（11.6亿股）
**目标年份**：2026年（Q3，锚定当年EPS）

**逻辑继承**：
- 模块五定稿预测：2026E EPS **2.28元**，净利润**26.5亿**（+10.7%）
- 三态EPS：保守1.97 / 中性2.28 / 乐观2.57
- ROE维持~20%水平
- 29.23亿扩产项目2026年不进营业成本，费用端影响可控（-0.9亿净利）

## 一、价值锚定：适配性方法选择

| 方法 | 权重 | 选择理由 |
|:---|:---|:---|
| **PE（市盈率）** | 60% | 盈利稳定正值，A股制造业最常用定价锚。为主方法 |
| **PB-ROE** | 25% | ROE长期>20%，重资产属性，适合交叉验证 |
| **PEG** | 15% | 正常化利润增速~15%，辅助判断估值透支程度 |

### 1.1 历史分位审计

| PE TTM分位 | 数值 | 当前(22.6x)位置 |
|:---|:---|:---|
| P10（低估） | 17.3x | — |
| P25 | 19.3x | — |
| **P50（中位）** | **21.6x** | — |
| P75 | 24.3x | — |
| P90（高估） | 27.8x | — |
| **当前** | **22.6x** | 中位线上方5%，P50-P75之间 |
| **2026E PE (46.49÷2.28)** | **20.4x** | **低于历史中位数21.6x** |

**核心观察**：当前PE-TTM 22.6x处于历史中位略上方，但**2026E远期PE仅20.4x，已低于历史中位数**。这意味着市场在按"豪迈回到平庸年代"定价，而产业证据（燃机超级周期+毛利率修复+扩产推进）指向相反方向。

### 1.2 基准PE选择

**为什么是24x而非旧版的26.5x或初版的23x？**

- 旧版26.5x（PEG 1.56）隐含了对28亿净利润的乐观预期配合，利润基数虚高导致PE被摊薄
- 初版23x过度保守，忽视了一个ROE 20%+的全球制造业龙头的估值溢价应高于平庸企业
- **24x**：在历史中位数21.6x基础上给予约11%的合理溢价。溢价来自：ROE从17%→22%的结构性提升、三个增长引擎（vs 过去仅模具单一驱动）、数控机床期权价值

| 计算步骤 | 数值 |
|:---|:---|
| 2026E归母净利润（定稿） | 26.5亿 |
| 基准PE | **24.0x** |
| **基准市值** | **636.0亿** |
| 除权后股数 | 11.6亿股 |
| **基准目标价** | **54.83元** |
| 当前价 | 46.49元 |
| 基准上行空间 | +17.9% |

### 1.3 PB-ROE校验

| 参数 | 数值 |
|:---|:---|
| 2026E ROE | 26.5÷128.0 ≈ 20.7% |
| 当前PB | 4.30x |
| 隐含回报率 | 20.7%÷4.30 = 4.8% |
| 5年PB/ROE历史中位数比率 | 0.175 |
| → 合理PB | 0.175×20.7 = **3.62x** |
| → 合理市值 | 3.62×128.0 = **463亿** |
| → 合理股价（除权后） | **39.91元** |

PB-ROE模型提示当前估值偏高（隐含回报仅4.8%）。但需注意：1) 历史中位PB/ROE比率对应的时期（2021-2023年）公司尚未展现当前的增长引擎组合；2) 2026E ROE因扩产稀释暂时回调，2027-2028年有望回升至22%+。PB-ROE更多反映"当前ROE的静态定价"，而非"成长性溢价"。此处取其警示含义（不宜给予过度PE溢价），但不过度依赖。

## 二、估值全变量对冲审计矩阵

### 2.1 审慎调节

| 审计维度 | 核心因子 | 审慎思考 | 调节 | 深度理由 |
|:---|:---|:---|:---:|:---|
| **内部：企业DNA** | 全球轮胎模具龙头(32.75%)+ROIC>20%+无杠杆+内生孵化(CNC从自用到外销) | **稀缺性溢价成立**。但这种稀缺性已在PE从23→24x的提升中体现。不应再额外叠加 | **+2%** | 较旧版+5%收窄，将部分溢价转入基准PE(24x) |
| **内部：扩产执行力** | 日照Q1净利4.6倍增长，实证了跨过盈亏平衡点的非对称利润弹性 | 这是**已实现的利润弹性**，属于E的范畴，不额外进PE | **0%** | 已在26.5亿净利中体现 |
| **宏观：机床国产替代** | 五轴机床国产化率59.5%→78%，国家战略级长逻辑 | 5-10年级别结构性趋势，非短期主题。豪迈"自用外溢"模式独特 | **+2%** | 政策长逻辑改善远期天花板，给予小幅估值溢价 |
| **宏观：汇率风险** | 人民币升值趋势确定 | **不调整PE**。汇率是具体可计算的钱，已体现在E的财务费用预测(+1.0亿) | **0%** | 严格遵循"可计算进E，不可计算进PE"原则 |
| **宏观：中美关税** | 综合关税33-57%，USTR提议CNC额外12.5% | 海外产能(泰/墨/美)已大幅对冲。但关税若扩展至模具品类+收紧转口审查，风险不可忽视 | **-2%** | 较旧版-3%收窄。USMCA R2确认机械/模具非攻击目标，关税焦虑已部分消退 |
| **市场：流动性与风格** | 11.6亿股(送转后)，自由流通市值~530亿 | 送转后流动性改善利于机构配置。当前A股偏好"业绩确定性+制造业龙头" | **+1%** | 小幅溢价 |

**综合调节**：+2%+0%+2%+0%-2%+1% = **+3%**

**调节幅度较旧版（+15%）、新版（+6%）均大幅收窄**。核心理由：旧版+15%过于慷慨（将多种非稀缺因素折算为估值溢价），新版+6%也偏多。在PB-ROE已提示估值偏高的背景下，估值矩阵的调节应保持审慎。**真正改变估值中枢的只有"机床国产替代"这一个政策长逻辑（+2%）和"寡头垄断稀缺性"（+2%）——合计+4%，被关税尾部折价-2%和已部分计入基准PE的溢价稀释后，仅剩+3%。**

### 2.2 修正目标市值

修正目标市值 = 636.0亿 × (1+3%) = **655.1亿**

## 三、三态目标价合成

| 情景 | EPS(元) | PE | 修正后目标价(元) | 概率 |
|:---|:---|:---|:---|:---|
| 保守(Bear) | 1.97 | 21x | **42.7** | 20% |

- **概率加权目标价**：42.7×0.2 + 56.5×0.55 + 71.7×0.25 = 8.54 + 31.08 + 17.93 = **57.5元**

### 3.1 盈亏比计算

| 参数 | 数值 |
|:---|:---|
| 加权目标价 | 57.5元 |
| 当前价 | 46.49元 |

### 3.2 锚点回测

修正后中性PE = 56.5÷2.28 = **24.8x**。该PE：
- 处于历史P75（24.3x）上方约2%，高于中位数（21.6x）约15%
- 隐含PEG = 24.8÷15.0（正常化增速）= **1.65**
- 对于ROE>20%+零有息负债+三个增长引擎的全球制造业龙头，24.8x是合理偏保守的定价

## 四、与历史两版估值的关键差异

| 指标 | 旧版6/26 | 初版7/1 | **定稿7/1** | 差异归因 |
|:---|:---|:---|:---|:---|
| EPS | 2.41 | 2.19 | **2.28** | 财务费用+1.0亿(vs →0.2,+1.5)取中 |
| 基准PE | 25x | 23x | **24x** | 中等溢价，历史中位+11% |
| 综合调节 | +15% | +6% | **+3%** | 严格过滤，仅保留稀缺性+政策长逻辑 |
| 目标价(加权) | 62.30 | 54.50 | **57.5** | — |
| 上涨空间 | +33.3% | +17.2% | **+23.7%** | — |
| 盈亏比 | 6.23x | 1.87x | **2.90x** | — |

**定稿的核心修正**：
1. **旧版高估了利润**（28→26.5亿）和估值溢价（+15%→+3%），目标价虚高
2. **初版低估了毛利率修复**（25.4→26.5亿），且PE选择过于保守（23x）
3. 定稿取两者之长：利润取旧版会计洞察+初版数据审慎的折中（26.5亿）、PE取有据可依的合理溢价（24x）、估值调节严格过滤（+3%）

## 六、估值总结

| 项目 | 数据 |
|:---|:---|
| 当前股价（不复权） | 46.49元 |
| 2026E EPS（定稿） | 2.28元 |
| 2026E PE（当前） | 20.4x（低于历史中位21.6x） |
| **概率加权目标价** | **57.5元** |

| 保守下限 | 42.7元（-8.2%） |
| 周期下行底 | ~35元（PE 15x） |


**结论**：当前46.49元对应2026E PE 20.4x——低于历史中位数21.6x。对于一个ROE>20%、零有息负债、三引擎驱动的全球制造业龙头而言，这是一个**价值错配**。目标价57.5元，盈亏比2.90:1，建议在44-47元区间分批建仓。旧版62.30元利润假设偏乐观，初版54.50元对毛利率修复力度估计不足。**26.5亿利润+24x PE+3%调节=57.5元**是对两版缺陷修正后的最优估计。
