# 研报章节二：产业周期与终端穿透

**研究日期：2026年07月01日**
**承接模块一核心发现**：豪迈科技2025年营收110.78亿元，三大业务（轮胎模具53.7%、大型零部件35.8%、数控机床8.7%）均处于上行期，但Q1 2026利润增长失速（营收+17%，净利-0.83%），需判断是产业周期拐点还是阶段性扰动。

---

## 一、轮胎模具：终需稳健增长，EV驱动结构性升级

### 1.1 产业链位置与周期定位

轮胎模具是轮胎硫化成型的关键工装，处于"钢材/铸件→模具制造→轮胎生产→汽车OEM/替换市场"链条的中上游。需求驱动因素按重要性排序：**轮胎花纹迭代频率 > 轮胎产能扩张 > 汽车保有量增长**。

**生命周期判断**：轮胎模具处于**成熟期后半段的结构性增长阶段**。全球市场CAGR约3-6%，属于非爆发式增长赛道。但两大结构性因素正在扩大有效市场空间：1) EV轮胎对模具精度的要求大幅提升，推动模具更换周期缩短；2) 轮胎花纹更新频率加快（从过去3-5年一次变为1-2年一次），增加了模具的替换性需求。

### 1.2 终端需求穿透：全球轮胎市场

| 指标 | 数据 |
|:---|:---|
| 2025年全球轮胎市场规模 | ~2,500亿美元 |
| 2025-2035年CAGR | 5.4% |
| 2025年全球轮胎模具市场 | ~16-19亿美元 |
| 2025-2031年模具CAGR | 3.1%-6.3% |
| EV轮胎增速 | 10-22% CAGR（远超整体） |
| 替换市场占比 | 70%+ |

**终端景气度审计**：全球轮胎市场维持5%+增速，核心驱动力为：
1. **发展中国家汽车保有量增长**：亚太地区CAGR 6.0%，中国、印度、东盟贡献增量
2. **EV轮胎需求爆发**：EV轮胎增速22% CAGR，且EV轮胎对低滚阻、低噪音、高载荷的要求迫使轮胎企业不断开发新花纹，直接带动模具需求
3. **替换市场稳定性**：替换需求占比70%+，具有消费品的抗周期属性，不直接受新车销量波动影响

### 1.3 动能溯源与透支审计

**增长动能来源**：主要由内生力量驱动（EV技术更迭、轮胎花纹迭代加速），**未发现显著的政策刺激透支风险**。轮胎模具作为消耗性工装，其需求有轮胎花纹更新这个"硬约束"支撑——只要轮胎企业持续推出新花纹，模具需求就不会断档。而当下正是EV轮胎设计的集中研发期，对模具的测试、小批量、量产需求是真实的终端拉动。

**跨期断层风险评估**：**低风险**。EV渗透率仍在持续提升（中国2025年新能源乘用车渗透率超50%），全球范围内EV轮胎的标准化设计远未完成，未来3-5年模具需求有坚实的技术迭代支撑。

### 1.4 库存动力学

轮胎模具为"以销定产"的订单式生产，不存在渠道库存的概念。轮胎企业自身的模具库存管理较为精益（模具单价高、占地面积大），因此该环节不存在显著的牛鞭效应风险。

## 二、大型零部件：风电稳健，燃气轮机爆发

### 2.1 风电铸件：高基数下的增速换挡

**2025年回顾**：中国风电新增装机120GW（+51% YoY），创历史新高。受政策抢装潮驱动，其中陆风110GW、海风6.6GW。整机招标量143.82GW，核准项目160GW，储备充足。

**2026年展望**：
| 预测来源 | 2026年新增装机 | 增速 |
|:---|:---|:---|
| 国能能源研究院 | 130GW | +10% |
| 中金公司 | 130-140GW | +8-17% |
| 国金证券 | 132GW | +10% |
| CWEA秘书长 | 120GW | 持平 |

**核心判断**：2026年风电装机维持在120-140GW高位，但增速从+51%回落至+10%左右。**高基数红利消退，但绝对量依然庞大**。"十五五"期间，陆风年均110-120GW，海风有望从2026年的10-12GW增至后期的年均20GW+。

**对豪迈科技的影响**：大型零部件风电铸件业务营收约39亿元（占35.8%），受益于行业高装机量，但需注意：
1. 2025年铸件业务毛利率从25.67%降至21.58%，同比下降4.09pct，表明**竞争加剧**（风机价格虽然回升，但上游零部件环节的议价权尚未同步改善）
2. 整机商"反内卷"后风机毛利率修复的传导效应有6-12个月滞后期，预计2026H2铸件毛利率有望企稳

**需求前置/透支审计**：2025年存在抢装因素（部分2026年需求被前置），但"十五五"规划将年均风电装机目标提升至120GW+，"沙戈荒"大基地合计约500GW进入集中投产期，**中长期需求有政策顶层设计的制度保障，跨期断层风险可控**。

### 2.2 燃气轮机零部件：AI驱动的超级周期

这是豪迈科技三个业务中最具预期差的增长极：

| 指标 | 数据 |
|:---|:---|
| 2025年全球燃气轮机市场 | ~202.5亿美元 |
| 2025-2031年CAGR | 6.44% |
| 2025年全球燃机订单 | 100.3GW（创历史新高） |
| 全球年产能 | 仅60-70GW |
| GE Vernova产能 | 已售罄至2028年，2029年仅剩10% |
| 西门子能源2025年销量 | 194台（几乎是2024年的两倍） |
| 三菱重工 | 计划两年内产能翻倍 |
| AI数据中心驱动力 | 2025-2028年美国数据中心总功耗从8.9GW升至111.3GW |

**供需缺口巨大**：全球燃机订单100GW vs 产能60-70GW，缺口持续扩大。GE、西门子、三菱三大OEM均满产满销，产能扩张速度远跟不上需求。这意味着燃机OEM将全力扩充供应链，为上游零部件供应商（如豪迈科技的铸件和精加工业务）带来**订单可见度极高的结构性增量**。

**终端景气度审计**：超强。AI数据中心电力需求是长周期（5-10年）结构性趋势，而非短期政策刺激。即使考虑部分项目延迟，燃机供应链的景气度至少延续至2030年。

## 三、数控机床：国产替代加速，五轴机床爆发

### 3.1 市场空间与增速

| 指标 | 2025年 | 2026年E | 2030年E | CAGR |
|:---|:---|:---|:---|:---|
| 中国数控机床市场规模 | 1,224亿元 | 1,304亿元 | 1,597亿元 | 5.3% |
| 其中五轴机床 | 129亿元 | 157亿元 | 352亿元 | 22.2% |
| 五轴机床国产化率 | 59.5% | — | 78%+ | — |

### 3.2 周期定位与驱动力

**生命周期**：五轴数控机床处于**成长期中段**（渗透率约10.5%→22%的加速阶段）。核心驱动力：
1. **航空航天下游**（占比33.7%）：国产大飞机、军用航空发动机对高端加工设备需求刚性
2. **新能源汽车一体化压铸模具**：超大吨位压铸机配套的大型五轴机床需求爆发
3. **设备更新政策**：贷款贴息+《机械行业稳增长工作方案》持续落地
4. **国产替代**：从2020年的18%提升至2025年的59.5%，自主可控战略持续推进

### 3.3 豪迈科技的独特优势

豪迈科技的机床业务并非"从零开始"。公司自成立之初便自制电火花机床及三/四/五轴机床用于轮胎模具生产，积累了20+年的机床自用经验。当前推向市场的产品（立式五轴加工中心、卧式车铣复合、五轴激光加工中心等）是在大规模内部使用验证后的外溢，**可靠性经过了工业级考验，这是与纯初创型机床企业最本质的区别**。

2025年数控机床营收9.68亿元（+142.59%），虽然体量尚小，但增速远超行业均值，正在经历从"内部工具"到"外部产品"的质变。

### 3.4 供需边际预判

**供给端**：科德数控、海天精工、纽威数控等国产龙头在加速扩产，供给竞争加剧。但豪迈科技的差异化在于：1) 自研自用20+年的机床制造经验；2) 全产业链布局（铸造→热处理→加工→装配→检测）；3) 与轮胎模具业务的协同效应。短期竞争压力有限，但长期需关注专业化机床企业的规模效应。

## 四、强制复核项确认

1. **终端产品景气度预测**：已穿透三大业务的终端市场——轮胎市场（5.4% CAGR）、风电装机（2026年130GW）、燃气轮机（结构性短缺至2030年）、五轴机床（22% CAGR），均有明确景气度数据支撑。

2. **跨期需求移位审计**：轮胎模具无显著透支风险（EV驱动真实）；风电存在2025年抢装前置但"十五五"政策保障中长期需求；燃气轮机是长期结构性趋势（AI数据中心需求）；机床受设备更新政策推动但长期由国产替代驱动。

3. **供需边际与压力传导**：
   - 轮胎模具：需求稳健增长，公司为全球龙头具备议价权，**压力传导可控**
   - 大型零部件：风电铸件竞争加剧，毛利率下行4.09pct，**正在经历压力传导**；燃气轮机零部件需求旺盛，**议价权提升**
   - 数控机床：行业高增长+国产替代窗口，**供给扩张暂未形成过剩**
