# 特锐德（300001）跟踪手册

**最后更新日期**：2026年04月08日

## 1. 核心逻辑验证点 (Thesis Validators)

需随季报及月度运营数据重点验证：
- [ ] **充电电量增长**：特来电月度充电量同比增速是否维持在 **40% 以上**。
- [ ] **盈利质量连续性**：季度净利率是否稳定在 **8% 以上**，毛利率是否呈现阶梯式回升。
- [ ] **出海逻辑兑现**：海外合同结算进度，2026年海外营收占比是否顺利突破 **12%**。
- [ ] **能源交易贡献**：虚拟电厂（VPP）业务是否产生实质性收入，单度电非电费收益是否突破 **0.05元**。
- [ ] **竞争对手动态**：监控星星充电（万帮数字能源）港股 IPO 进程及其在核心城市的单桩平均服务费水平变动。

## 2. 关键外部变量 (Exogenous Watchlist)

- [ ] **800V车型渗透率**：关注国内新发布车型中 800V 高压平台占比（预期 25%+）。
- [ ] **沙特“2030愿景”基建进度**：关注中东地区对中国电力设备的准入政策变动。
- [ ] **电力现货市场改革**：各省份（尤其是山东、广东）对虚拟电厂参与现货交易的结算规则细化。

## 3. 逻辑证伪警报 (Invalidation Triggers)

**若出现以下信号，必须承认投资逻辑已死，启动止损或下调评级：**
- [ ] **市占率大幅下滑**：特来电公共充电终端保有量或电量份额跌破 **20%**。
- [ ] **盈利能力逆转**：综合毛利率连续两个季度下滑超过 **3 个百分点**。
- [ ] **重大安全事故**：充电网遭受严重黑客攻击或发生系统性起火事件，导致政府监管介入。
- [ ] **融资环境突变**：资产负债率重回 **70% 以上** 且经营性现金流转负。
