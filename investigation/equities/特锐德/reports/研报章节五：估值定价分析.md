# 研报章节五：估值定价分析

**研究日期**：2026年04月08日

## 1. 价值锚定：适配性方法选择

针对特锐德“制造+运营”的复合属性，本报告采用 **分部估值法 (SOTP)** 为主，**历史市盈率 (PE) 还原法** 为辅进行交叉验证。

### 1.1 分部估值 (SOTP) 模型
锚定 2026 年预测利润：
*   **变配电制造板块**：预测净利润 8.0 亿元（税后）。参考电力设备行业平均水平，给予 **18倍 PE**。估值 = 144 亿元。
*   **充电网运营板块**：预测归母净利润 6.0 亿元（税后，已扣除少数股东权益）。考虑到其作为行业第一的规模效应、5300万用户流量及虚拟电厂的溢价，给予 **45倍 PE**。估值 = 270 亿元。
*   **基准总市值** = 144 + 270 = **414 亿元**。

### 1.2 历史分位审计
*   **当前 PE (TTM)**：26.06倍，处于上市以来历史分位 **<10%** 位置。
*   **结论**：即使不考虑充电桩的成长溢价，当前股价也已跌破其传统电力设备资产的估值底线，具备极高的安全边际。

## 3. 最终定价建议与胜率矩阵

*   **修正目标市值** = 414亿 * 1.08 = **447.1 亿元**。
*   **最终目标价 (2026E)** = 447.1 / 10.56 = **42.34 元**。
*   **现价 (2026-04-08)** = 28.52 元。
*   **预期收益空间**：**+48.5%**。

### 确定性与胜率
*   **中性场景 (P=60%)**：实现 1.55元 EPS，估值锚定 27x PE，价格 42.0 元。
*   **保守场景 (P=20%)**：实现 1.15元 EPS（受价格战影响），估值维持 25x PE，价格 28.8 元。
*   **盈亏比** = (42.34 - 28.52) / (28.52 - 28.8) ≈ **安全边际依然显著，盈亏比极具吸引力**。

## 4. 逻辑证伪与极限压力测试

### 4.1 黑天鹅场景：网络安全风险
*   **事件描述**：若特来电充电网遭受大规模网络攻击，导致核心城市补能瘫痪，政府可能收回运营资质。
*   **影响**：充电网业务估值瞬间归零。

### 4.2 极限支撑位审计 (地狱价格)
*   **测算底稿**：基于 2025 年净资产约 85 亿元，若市场极端非理性，给予 **1.2x PB**。
*   **地狱价格** = 85 * 1.2 / 10.56 ≈ **9.66 元**。
*   **风险边界**：距离现价有约 66% 的理论极端回撤空间（但在盈利增长 30% 背景下，发生概率极低）。

**最终定价结论**：特锐德目前处于“**业绩拐点已现、估值尚在谷底**”的错配期。45.08 元的目标价仅是将其 PE 倍数还原至 30 倍左右的合理水平，尚未计入其虚拟电厂业务规模化后的想象力溢价。
