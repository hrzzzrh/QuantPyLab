# 研报章节五：估值定价分析

**研究日期**：2026年04月23日

## 1. 价值锚定：从“估值修复”到“安全边际审计”

针对特锐德当前的内卷现状，本报告引入“价值地板”测算逻辑。

### 1.1 极限压力测试下的“价值地板”
*   **制造板块估值**：预测 2026 年净利润 8.2 亿元，给予 **15倍 PE**（电力设备二线溢价）。估值 = 123 亿元。
*   **充电网运营资产清算价值**：特来电净资产约 60 亿元，按 **1x PB** 计价。估值 = 60 亿元。
*   **公允支撑位 (Support)**：考虑其行业龙头地位，给予 20x 设备 PE 及 1.5x 运营 PB，支撑位约为 **22.0 元**。

### 1.2 分部估值 (SOTP) 模型 (Revised)
基于 2026 年调整后利润 13.2 亿元（EPS 1.25 元）：
*   **变配电制造板块**：净利 7.4 亿，给予 **18倍 PE**。估值 = 133 亿元。
*   **充电网运营板块**：净利 5.8 亿，因 VPP 确定性增加及 800V 优势，给予 **35倍 PE**。估值 = 203 亿元。
*   **基准目标市值** = 133 + 203 = **336 亿元**。

### 1.2 历史分位审计
*   **当前 PE (TTM)**：26.39倍（截至2026-04-23），处于上市以来历史分位 **<10%** 位置。
*   **结论**：即使不考虑充电桩的成长溢价，当前股价也已跌破其传统电力设备资产的估值底线，具备极高的安全边际。

## 3. 最终定价建议与胜率矩阵 (Revised)

*   **基准目标价 (2026E)** = 336 / 10.56 = **31.82 元**。
*   **溢价期权**：沙特大单持续性 (3元) + 虚拟电厂爆发 (5元) + 800V 流量垄断 (3元)。
*   **修正目标价范围**：**38.50 - 42.80 元**。
*   **现价 (2026-04-24)** = 31.32 元。

### 3.1 胜率与概率审计矩阵

基于当前 31.32 元的股价，我们对未来 12 个月的收益/风险比进行如下概率审计：

| 场景 | 核心触发变量 | 目标定价 | 发生概率 | 期望收益 |
| :--- | :--- | :---: | :---: | :---: |
| **乐观 (Bull)** | 沙特工厂提前达产 + VPP 利润贡献 > 1亿 | 42.80 元 | 25% | +36.6% |
| **中性 (Base)** | **海外平稳交付 + 国内内卷维持现状** | **31.82 元** | **55%** | **+1.6%** |
| **悲观 (Bear)** | **应收账款大幅计提 + 特来电价格战转亏** | **22.00 元** | **20%** | **-29.8%** |

*   **综合期望收益率** = (36.6% * 0.25) + (1.6% * 0.55) + (-29.8% * 0.20) = **4.07%**。
*   **盈亏比 (Reward-to-Risk)** = (42.80 - 31.32) / (31.32 - 22.00) ≈ **1.23**。

**结论**：目前的期望收益率（约 4%）极低，盈亏比虽大于 1 但并不具吸引力。这验证了我们在《章节七》中给出的 **“Hold (持有/观望)”** 评级。当前股价处于典型的“逻辑真空期”，向下的地板（22元）和向上的期权（42.8元）张力并存，**赔率并不优厚**。

## 4. 逻辑证伪与极限压力测试

### 4.1 黑天鹅场景：网络安全风险
*   **事件描述**：若特来电充电网遭受大规模网络攻击，导致核心城市补能瘫痪。
*   **影响**：充电网业务估值瞬间归零。

### 4.2 极限支撑位审计 (地狱价格)
*   **测算底稿**：基于 2025 年净资产约 85 亿元，给予 **1.5x PB**（行业出清底线）。
*   **地狱价格** = 85 * 1.5 / 10.56 ≈ **12.07 元**。

**最终定价结论**：特锐德目前正处于从“变压器工厂”向“充电网运营巨头”及“虚拟电厂运营商”身份切换的阵痛尾声。57.17 元的目标价反映了其作为能源互联网核心枢纽的合理定价。
