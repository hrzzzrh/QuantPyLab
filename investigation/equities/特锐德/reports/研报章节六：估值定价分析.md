## 一、价值锚定：适配性估值方法选择

### 1.1 方法论筛选

特锐德的商业模式特征决定了估值方法的选择：

| 估值方法 | 适用性 | 理由 | 权重 |
| :--- | :---: | :--- | :---: |
| **PE（市盈率）** | ★★★★★ | 公司已稳定盈利，利润增速可预测，PE是A股最主流的估值语言 | 40% |
| **SOTP（分部估值）** | ★★★★★ | 两大业务属性差异巨大（工业制造 v.s. 平台运营），必须分开估值 | 40% |
| **PB-ROE** | ★★★☆☆ | 作为重资产公司的辅助验证，尤其对电力设备板块 | 10% |
| **DCF** | ★★★☆☆ | 长期视角参考，但充电运营的远期中性和折现率假设过于敏感 | 10% |
| **PEG** | ★★☆☆☆ | PEG对"成长性"的依赖高度主观，且易被低基数效应扭曲 | 0% |

### 1.2 为什么不用PEG？

特锐德过去5年净利润增速约64%（低基数效应显著），若直接套用PEG=1得到64倍PE显然是荒谬的。PEG对于处于"利润拐点"的公司（从一个很低的利润基数转向正常化利润）完全不适用，会严重高估合理估值。

**关键判断**：虽然当前25.5x处于历史P10，看似"极其便宜"，但不能简单套用历史分位。2020-2022年公司的利润基数极低（ROE仅3%-7%），导致PE虚高。**2023年以后公司利润正常化（ROE 7.5%-15.7%），PE运行区间收窄至23-40倍**——这才是合理的估值参考系。

当前25.5x在"正常化PE区间"中仍处于**偏低位置**（接近下限23x）。近30日PE在24.1-26.6之间窄幅波动，表明市场定价进入僵持阶段——多空双方均在等待2026H1中报（8月）的业绩验证。

**基准市值** = 1.52元 × 19倍 × 10.56亿股 = **305亿元**
**基准目标价** = **28.9元**（不复权市价，较当前-4.5%）

注意：基准目标价略低于当前股价——这不是"看空"信号，而是PE方法在盈利预测下修后的自然结果。后续全变量审计矩阵将通过定性因子的溢价调节修正这一底价。

### 6.2 盈亏比计算

**公式**：盈亏比 = (目标价 - 现价) / (现价 - 保守场景价格)

盈亏比 = (31.8 - 30.24) / (30.24 - 17.1) = 1.56 / 13.14 = **0.12**

**解读（重大变化）**：盈亏比从0.93大幅下降至0.12，上行动能有限而下行风险可观。这意味着当前价位**已不再是"合理偏低估"**，而是接近**公允甚至略偏高估**。以30.24元买入，1年预期收益仅约+5.2%（概率加权），而一旦落入保守情景（概率25%），潜在亏损达43%。投资性价比边际走弱。

**操作含义的转变**：
- **此前（5/9）**：盈亏比0.93，适合底仓配置
- **当前（5/11修订后）**：盈亏比0.12，建议等待更好的入场点。若股价回落至MA60附近（约28.5-29元），盈亏比可恢复至约0.3；若回落至MA120附近（约27.6元），盈亏比约0.5——届时将重新具备底仓配置的性价比。
