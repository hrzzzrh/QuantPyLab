# 研报章节四：外部边界与宏观政策

**研究日期**：2026年05月29日

---

## 一、逻辑继承：前序模块的外部变量红线

本节分析的外部变量，将直接影响模块五（盈利预测）中的销量假设（地缘关税是否阻断出口）、价格假设（关税叠加拿价能力）、成本假设（汇率/大宗商品传导）以及模块六（估值）中的风险溢价调节。

前序模块已识别出以下外部相关信号：
1. 公司全球CCL第二，出口占比约20%（外销收入约26亿/半年），海外资产190.9亿（占总资产6.4%）
2. 高端产品受美系终端客户（NVIDIA/AWS/Google）认证驱动，地缘政治直接影响供应链卡位
3. 泰国工厂即将投产（2026年7月），是规避关税的重要战略支点

## 二、中美贸易与关税环境

### 2.1 当前关税格局（2026年5月）

| 关税类型 | 当前税率 | 变动风险 | 对CCL/PCB影响 |
| :--- | :--- | :--- | :--- |
| Section 122 对等关税（中国） | 15% | 2026.11到期，可能延期或调整 | PCB/CCL直接纳入 |
| Section 232 半导体关税 | 25% | **可能升至50%（6月USTR结束咨询）** | PCB受影响（HS分类涵盖） |
| Section 232 铜制品 | 25%（2026.4生效） | — | 铜箔进口成本上升 |
| MFN基准关税 | 0-3% | 稳定 | 影响有限 |
| **叠加坡关税** | **40-43%**（122+232合计，可能升至65%） | **重大风险** | PCB出口美国成本显著上升 |

**关键节点**：
- **2026年6月**：USTR对半导体232关税是否升至50%的咨询期结束
- **2026年11月**：吉隆坡经贸磋商联合安排（暂停24%对等关税等）到期
- **2026年11月1日**：美国《2026数字供应链透明度法案》生效，要求对华出口超50万美元的供应商提供ESG审计报告

### 2.2 对生益科技的实质影响

**直接影响**（直接对美出口）：
- 公司2025H1外销收入约26亿（占总营收20.5%），其中对美出口占比估算在5-8%
- 若关税全面叠加，对美出口的PCB/CCL成本将上升40-65%，倒逼美国客户转向东南亚/台系供应商

**间接影响**（通过客户间接传导）：
- 公司大量产品通过下游PCB客户间接出口美国
- 中国PCB厂商若被高关税打击，订单可能转移至东南亚，而生益科技泰国工厂恰可承接这一需求

**战略缓冲——泰国工厂**：
- 2026年1月已封顶，计划2026年7月投产
- 年产能1200万m²高性能CCL + 2340万米粘结片
- 获BOI 8年企业所得税豁免 + 设备进口关税免征
- 定位汽车电子、AI服务器、芯片载板
- **这是对冲中美关税风险的最重要物理资产**

### 2.3 中美脱钩的深层影响

清华CISS研究报告指出：半导体设备领域的中美脱钩已进入**不可逆阶段**。中国"50%国产设备强制令"导致美国设备商在华东订单断崖式下滑30.6%。在AI芯片方面，美国采取"以费促管"模式——允许NVIDIA H200/MI325X在缴纳25%"技术准入费"后出口中国。这种"精准切割"（设备彻底断链、高利润芯片收费准入）意味着：

- **对生益科技的利好**：中国AI算力芯片供应虽受限但未断流，国内AI服务器仍能增长，公司作为大陆唯一NVIDIA认证CCL供应商的战略价值凸显
- **对生益科技的压力**：若管制进一步收紧，国产GPU（华为昇腾等）需要更多国产配套材料，公司需要加速从"NVIDIA供应链"向"国产GPU供应链"的切换准备

## 三、国内产业政策红利

### 3.1 "十五五"规划纲要中的核心定位

| 政策维度 | 具体内容 | 对生益科技影响 |
| :--- | :--- | :--- |
| 产业定位 | 集成电路列十大新产业新赛道**首位** | CCL作为半导体封装关键材料，受全链支持 |
| 技术攻关 | 全链条推动关键核心技术攻关（含先进材料） | 可能获得研发补贴和税收优惠 |
| 国产化目标 | 2030年半导体整体自给率70%，12英寸硅片100%国产 | 高端电子材料国产替代加速 |
| 专项资金 | 大基金三期超3000亿（重点投向设备、材料、EDA） | 上游供应商获益，间接利好 |
| 设备材料国产化 | 12英寸产线国产设备覆盖率目标60%，光刻胶自给率50% | 促进国内电子材料生态成熟 |

### 3.2 《电子信息制造业2025-2026年稳增长行动方案》

直接关联公司当前经营的政策文本：
- 目标：电子制造业增加值年均增速约7%
- 强调服务器产业规模超4000亿元
- 支持AI、先进计算、光通信等前沿技术
- 鼓励外商在华设产线和研发中心

### 3.3 《印制电路板行业规范条件（2025年版）》

**这是对CCL行业最直接的结构性政策**：
- "禁止低端扩产 + 资金倒逼研发"双闸门机制
- 80%以上新建产能导向高端领域（20层+高多层、HDI、IC载板）
- 低端FR-4产能扩张受限 → 有利于龙头企业巩固市场份额
- 高端产品享受政策倾斜 → 与生益科技松山湖二厂定位高度一致

### 3.4 税收优惠政策

集成电路和软件企业享受：
- 线宽<28nm企业：10年免征所得税
- 先进封装测试企业：特定税收优惠
- 关键原材料（含封装载板）生产企业：进口税收优惠
- 研发费用加计扣除比例提高

**生益科技的具体受益点**：公司布局的封装基板材料（FC-CSP/FC-BGA用），属于国家鼓励的关键原材料范畴，可能享受进口设备关税减免和研发费用加计扣除政策。

## 四、汇率损益与大宗商品价格传导

### 4.1 汇率影响分析

| 维度 | 现状 | 影响 |
| :--- | :--- | :--- |
| 外销收入占比 | ~20% | 人民币贬值利好出口收入 |
| 境外资产 | ~6.4%（190.9亿） | 汇率波动影响报表折算 |
| 进口原材料成本 | 铜箔/树脂/玻纤布部分依赖进口 | 人民币贬值增加采购成本 |
| 2025H1其他综合收益 | +31.14%（汇率变动收益） | 近期汇率变动对公司偏正面 |

**净影响判断**：公司进出口大致平衡，汇率影响中性偏正面（海外资产收益+出口产品价格竞争力提升）。

### 4.2 大宗商品价格传导

| 原材料 | 价格趋势（2026） | 公司对冲手段 |
| :--- | :--- | :--- |
| 铜箔（HVLP4） | 严重短缺，持续涨价 | 多供应商策略 + 涨价向下游传导 |
| 玻纤布（Q布/电子布） | Q布紧缺但H2放量，普通电子布涨价 | 菲利华一供 + 二供储备 |
| 环氧树脂 | 受原油/化工品涨价推动，持续高位 | 自供率60% + 三菱化学合作 |
| 碳氢树脂 | 东材科技扩产，供给改善 | 关联公司（东材为M9独家树脂商） |

**行业特色**：CCL行业具有向下游传导成本的定价权（验证见模块三）。公司调研确认"涨价没有停止的迹象，还要继续"。在当前供需紧平衡下，原材料涨价反而成为公司提价的正当理由，构成**利润增厚因子**。

## 五、地缘政治与供应链重构

### 5.1 "China+1"供应链转移趋势

全球PCB产业正在向东南亚（尤其泰国）大规模转移：
- 过去两年泰国落地PCB投资项目超100个，总投资逾1700亿泰铢
- 大批项目2025年起陆续投产
- 台光电、联茂等台系CCL厂商也在扩产泰国

**对生益科技的影响**：
- **挑战**：若下游PCB客户大规模转移至泰国/东南亚，国内CCL需求可能被分流
- **机遇**：公司泰国工厂即将投产，可直接服务东南亚客户群，形成"中国+泰国"双基地模式
- **判断**：公司泰国产能的落地时机恰到好处，可以在供应链转移中保持客户黏性

### 5.2 技术标准分裂风险

美中在半导体领域的"硅幕"正在落下：
- 中国推进RISC-V架构 + 先进封装标准（ACC 1.0）
- 西方主导EUV + UCIe标准
- 未来可能出现两套互不兼容的技术生态

**对CCL行业的影响**：
- 短期内，NVIDIA/AMD/Intel体系仍是全球主流，高端CCL需求仍以美系标准为主
- 长期来看，若中国构建独立技术生态，需要独立的CCL供应体系——这进一步强化了生益科技作为"大陆唯一NVIDIA M9认证CCL供应商"的战略价值
- 关键风险：若美国加强对华AI芯片出口管制，可能导致国内AI算力建设放缓

### 5.3 美伊地缘冲突与其他风险

中东局势持续紧张，直接影响：
- 全球原油/化工品价格上涨 → 环氧树脂等原材料成本上行
- 海运保险费率上升 → 公司出口物流成本增加
- 但公司2025年报已提及"高层密集走访供应商争取资源"应对成本波动

## 六、模块四核心结论

1. **中美关税是最大外部不确定性**。当前叠加关税40%（可能升至65%），直接冲击PCB/CCL对美出口。**泰国工厂（2026年7月投产）是公司最重要的对冲资产**，能否成功运营将决定公司在贸易摩擦中的生存能力。

2. **中国产业政策高度契合公司发展方向**。"十五五"将集成电路/先进材料列为优先攻关领域，PCB行业规范条件"限低端、促高端"与公司松山湖二厂定位一致。政策端构成**明确的中期利好**。

3. **中美技术脱钩在半导体设备领域已不可逆**，但AI芯片出口管制处于"收费准入"阶段而非全面禁运。中国AI算力产业仍将高速增长，生益科技作为大陆唯一NVIDIA认证的CCL供应商，在"国产替代+美系供应链双轨并行"中具有独特价值。

4. **大宗商品价格高企是双刃剑**——一方面推高公司成本，另一方面为提价提供正当性。在当前紧平衡格局下，**涨价红利大于成本压力**。

5. **地缘政治对公司估值的实质影响**：关税隐含折价（-5%到-10%）、国产替代溢价（+5%）、泰国工厂期权重估（+3%到+5%）。具体量化将在模块六的"全变量对冲审计矩阵"中体现。

6. **关键预警信号**：2026年11月关税安排到期 + 6月USTR半导体关税咨询结束 + 美国《数字供应链透明度法案》生效。这三个时点若出现超预期负面政策，将对公司出口业务构成实质性冲击。
