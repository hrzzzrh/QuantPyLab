# 研报章节四：外部边界与宏观政策

**研究日期**：2026年05月28日

---

## 一、前序模块核心结论回顾

**模块一红线**：1）增长引擎换挡至AIDC；2）毛利率修复至25.9%高位；3）存货累积（+22.3% vs营收+5.7%）；4）经营现金流改善但历史不稳。

**模块二红线**：1）光伏2026年需求断层（-52%）；2）风电稳健增长（9% CAGR）；3）AIDC爆发（30-50% CAGR）；4）2026Q1订单加速（+64.5%）。

**模块三红线**：1）干式变压器国内市占率第一（14%），但行业供过于求长期化；2）AIDC综合方案弱于Eaton/Delta/Vertiv，竞争优势在场景先发而非技术绝对领先；3）海外收入占比从32%向50%+跃进，2026Q1海外订单占比67%；4）国际认证（349项）为最深护城河。

## 二、宏观环境变量

### 2.1 铜价超级周期：成本端的持续考验

铜占变压器材料成本约30%，其价格走势直接影响公司毛利率。

**现状与趋势：**
- 2025年初至2026年初，铜价从不足7万元/吨攀升至10万元/吨以上，涨幅接近50%
- 当前铜价在10.2-10.7万元/吨区间高位震荡（2026年5月28日）
- 全球铜精矿现货加工费（TC）已跌破-100美元/吨，矿端供应极度紧张
- 国家电网"十五五"规划投资4万亿元（较"十四五"+40%），2026年投资约8000亿元（+10%）
- AI数据中心用铜量是传统数据中心的3倍

**对金盘科技的影响：**
- 2025年公司毛利率逆势提升至25.9%，说明定价传导能力较强（产品提价+结构升级对冲了铜价上涨）
- 但行业"铝代铜"趋势已在中小型充电站等低端场景加速蔓延（全铜变压器26.5万元 vs 全铝11.3万元），若铜价持续高位，可能侵蚀中低端产品的需求
- 公司非晶合金带材产线投产，非晶变压器用材与传统硅钢+铜不同，可部分规避铜价压力
- 关键判断：铜价预计中长期维持高位（超级周期），公司毛利率持续大幅提升空间有限

**通胀传导效率：** 中国变压器出口均价同比上涨约三分之一，说明成本传导在海外市场较为顺畅（欧美客户愿支付20%溢价抢货）。国内市场价格竞争激烈，传导效率较低。

### 2.2 汇率损益：海外业务的双刃剑

人民币在过去一年升值约6,000个基点，对出口企业形成显著压力。

**对金盘科技的具体影响：**
- 2026Q1公司汇兑损失约1,700万元（来自券商研报测算），叠加可转债利息计提1,100万元和股权激励1,400万元，合计非经常性拖累约4,200万元
- 剔除这些因素后，Q1实际净利润约1.54亿元（同比+44%），显示主营业务韧性远超报表数字
- 海外收入占比迅速提升（2026Q1约40.6%），汇率敏感度持续放大
- 公司已布局马来西亚工厂（本地化生产），天然对冲部分汇率风险（美元收入-马币成本）

**趋势研判：** 人民币长期升值基础仍牢固（制造业竞争力增强、人民币国际化推进），但央行不愿看到单边升值预期。出口企业需通过外汇套保、人民币结算、合同调价机制等管理汇率风险。金盘科技作为出口占比快速攀升的企业，汇率管理能力将成为利润稳定性的关键变量。

### 2.3 利率与折现率：估值端的双面效应

中国当前处于宽松货币政策周期，十年期国债收益率处于历史低位（约1.7-1.9%），有利于成长股估值扩张。但若通胀压力显现或美联储政策转向，利率上行将对高PE股票（金盘科技当前PE约56-73x）形成较大估值压力。

## 三、地缘政治与贸易壁垒

### 3.1 美国关税政策：短期利好、中期风险

2026年4月2日，特朗普签署总统公告，将Section 232金属关税重构为四层结构。对变压器行业核心影响：

| 时期 | 关税规则 | 对金盘科技的实质影响 |
| :--- | :--- | :--- |
| **旧规则（-2026.4）** | 50%关税仅针对金属含量部分 | 复杂、不可预测 |
| **新规则 Tier III（2026.4-2027.12）** | **15%**统一税率，按完税全价计征 | **显著优于旧规则**：干式变压器关税从约$10,125降至$6,750（-33%） |
| **2028年起** | 自动切换至Tier I **25%**税率 | 较当前15%跳升67%，压力骤增 |

**关键判断：** 2026-2027年是中国变压器出口美国的**战略窗口期**。关税从50%（仅金属）降至15%（全价），实质利好中国变压器出口。但2028年起税率跳升至25%，届时马来西亚工厂的产能（可规避关税）将成为核心战略资产。

**行业背景：** 全球变压器严重短缺。Goldman Sachs将美国变压器进口依赖度从57%上调至60%，短缺可能持续至2028年以后。美国本土产能增速仅2-4%/年，而需求增速9-12%/年。中国对美变压器出口2026年1-2月飙升182%。短缺的刚性需求赋予了中国供应商较强的定价和谈判筹码。

### 3.2 欧洲：严格标准下的结构性机遇

欧洲电网升级投资规模约1.2万亿欧元。2025年中国变压器对欧出口额同比+42.2%，达18.37亿美元。

**机遇：** 欧洲电网老化（多次大停电）、AI算力爆发、新能源并网三重驱动，需求极度旺盛。金盘科技已取得CE/UKCA/DNV/BV/TUV等欧洲认证，并通过海外营销中心（德国）覆盖欧洲市场，首台80MVA液浸式变压器已成功交付欧洲。

**壁垒：** 欧盟《能源相关产品生态设计要求》对变压器能效、噪音设定严格门槛，新进入者认证周期长（3-5年）。这对已获认证的金盘科技构成护城河，但也限制其产品线扩展速度。

### 3.3 马来西亚工厂：地缘对冲的核心资产

中美关系持续紧张背景下，马来西亚工厂是金盘科技全球化战略的关键棋子：

- **关税规避**：马来西亚制造可规避美国对华关税（Section 232/301），直接进入美国市场
- **供应链弹性**：满足全球客户"China+1"的供应链多元化需求
- **成本优化**：享受当地劳动力成本优势，同时靠近东南亚新兴市场
- **风险分散**：降低对中国单一生产基地的依赖

**但需注意**：美国有可能将关税扩展至中国企业的海外产能（如针对转口贸易的限制），且马来西亚工厂的规模和管理能力尚在爬坡期。

## 四、监管与产业政策

### 4.1 国内政策：新能源退坡与AIDC加持

**不利因素：**
- **"136号文"取消强制配储**（2025年2月）：新能源项目不再强制要求配套储能，储能行业短期承压
- **分布式光伏新规**：导致2025年抢装→2026年需求断层（1-4月新增装机同比-52%）
- **光伏行业整体进入调整期**：CPIA预测2026年新增装机180-240GW（较2025年315GW大幅下滑）

**有利因素：**
- **国家电网"十五五"4万亿投资**：较"十四五"+40%，2026年投资约8,000亿（+10%），特高压和配电网改造是重点
- **AIDC被列为国家战略性新兴产业**：算力基础设施获政策重点支持
- **中国制造2025/新质生产力**：高端电力装备制造享受税收优惠和研发补贴
- **海外产能建设获政策鼓励**：企业出海享受出口信贷和保险支持

### 4.2 海外政策：机遇与风险并存

| 地区 | 核心政策 | 影响 |
| :--- | :--- | :--- |
| **美国** | 电网升级投资1,360亿美元（2026-2035）；2026年750亿美元输电扩建计划 | 极大利好变压器需求 |
| **欧洲** | EU电网现代化1.2万亿欧元；REPowerEU新能源目标 | 持续拉动中国变压器出口 |
| **中东** | 沙特"2030愿景"巨额基建；中国出口沙特变压器2025年超60亿元（+80%） | 新兴增量市场 |
| **东南亚** | 经济发展+电网扩建+中国产能外溢 | 马来西亚工厂辐射 |

| :--- | :--- | :--- |
