## 一、价值锚定：适配性方法选择

### 1.1 估值方法选择与理由

| 方法 | 适用性 | 理由 |
|:---|:---:|:---|
| **PE（市盈率）** | ★★★★★ | 公司盈利稳定、可预测，为成长型制造业最通用的估值方法 |
| **PB-ROE** | ★★★★ | 高ROE（31%+）公司的合理估值应结合ROE中枢评估 |
| **PEG** | ★★★ | 增长正在换挡（从30%+降至~19%），PEG的适用性下降 |
| **DCF（现金流折现）** | ★★★ | 现金流极好（OCF/NP > 1.2），但高增长阶段假设波动大 |
| **PS（市销率）** | ★★ | 辅助参考，净利率15%+使PS与PE高度相关 |
| **EV/EBITDA** | ★★ | 公司无大量有息负债，杠杆率较低，EV/EBITDA参考意义有限 |

**选择组合**：以**PE估值法为主锚**（直接反映盈利能力和增长预期），**PB-ROE法为辅助校验**（控制杠杆率变化对PE的影响），**DCF/PS为交叉验证**。

**历史分位的参考价值审慎评估**：

1. **2021-2022年的极端高估（PE 80-133x）不具备参考意义**：那是新能源泡沫+流动性泛滥双重叠加的非理性阶段，不应纳入合理估值区间锚定
2. **2018年的极端低估（PE 7.5-17x）对应行业政策毁灭性冲击**：531新政后光伏行业全面崩溃，当前的行业基本面远优于彼时
3. **2019-2020年的PE 15-31x区间是较合理的参照**：行业复苏期，与当前"增速换挡"环境更可比
4. **A股市场环境发生深刻变化**：注册制改革、外资持续流入、机构化程度提升，A股估值中枢整体上移，单纯套用历史分位需保守使用

**结论**：剔除非理性极端值后，阳光电源合理的PE中枢在**18-25x**区间。低于18x对应极度悲观（光伏崩盘重现2018），高于25x需要极强的增长来支撑。

## 二、核心：估值全变量对冲审计矩阵（强制性输出）

### 2.1 基准市值确定

**基准市值**：以中性情景2026E EPS 7.79元 × 基准PE 21倍 = **163.6元**，对应市值 **3,359亿元**。

（基准PE 21倍的选择逻辑：2024-2025年均PE约16x，2026年储能高增长赋予小幅溢价，21x对应PEG≈1.1，兼顾增速换挡和行业地位。）

## 三、强制性：逻辑纯净性校验

### 3.1 E（盈利预测）与PE（估值倍数）的分配原则

| 因素 | 进入E还是PE？ | 理由 |
|:---|:---|:---|
| 储能销量增长60%+ | → **E** | 可量化为具体收入/利润增加 |
| 逆变器行业销量-9% | → **E** | 可量化为出货量下降和收入减少 |
| 储能价格战致毛利率-2% | → **E** | 可量化为单位利润变化 |
| 欧盟政策不确定性 | → **PE** | 不可量化具体金额，但影响长期信心 |
| 全球能源转型确定性格局 | → **PE** | 关乎折现率和风险溢价 |
| AIDC/氢能业务期权 | → **PE**（暂不纳入） | 无确定收入，仅影响长期想象力 |
| 公司战略聚焦与资本纪律 | → **PE** | 关乎治理溢价 |

### 3.2 隐含综合PE（21.4倍）的合理性校验

| 参考系 | PE值 | 阳光电源21.4倍的位置 |
|:---|:---:|:---|
| 2024年均PE | 15.8x | 高于（但2024年增长7.8%，2026E增长19%） |
| 2025年均PE | 16.4x | 高于（增速提升赋予溢价） |
| 2026年Q1高基数下利润-40%后的PE | 24.3x | 低于（市场隐含Q2-Q4恢复） |
| 剔除极端值的历史中枢 | 18-25x | **处于区间中下部** |
| 阳光电源2021年泡沫期 | 84-134x | 远低于 |
| A股储能可比（宁德时代） | ~20-25x | 接近 |
| 全球逆变器可比（SMA） | ~12-15x（萎靡） | 高于（但SMA无储能增长） |
| A股光伏制造（通威/隆基） | ~10-15x | 高于（储能赋予溢价合理） |

**结论**：21.4倍的隐含PE处于合理区间。既体现了储能高增长的溢价，又充分打折了逆变器下行和地缘风险的担忧。

### 4.2 盈亏比计算

- **现价（2026年5月22日不复权收盘价）**：164.51元
- **概率加权目标价**：164.3元
- **潜在上行空间**：-0.13%（当前已小幅高于加权目标价）

## 五、逻辑证伪与极限压力测试

### 5.1 "一击毙命"黑天鹅场景

**定义**：欧盟通过CSA 2.0，将阳光电源正式列入"高风险供应商"（HRV）清单。

**连锁冲击**：
1. **直接损失**：欧洲市场（约250-300亿元收入、100-120亿元毛利）在36个月内全部丧失
2. **估值崩溃**：市场将阳光电源从"全球成长龙头"重新定价为"局限于中国+发展中国家的区域公司"
3. **可比公司参照**：SMA（德国老牌逆变器商）在2018-2020年中国竞争者涌入后，PE从25x跌至10x以下，市值蒸发超80%
4. **连锁反应**：其他发达国家（日本、韩国、澳大利亚）可能跟进限制

**地狱价格估算**：
- 假设海外收入减半（丧失欧洲），净利润降至60-70亿元
- 市场给予"受限制造业"估值：**PE 12倍**
- **极端目标价**：70 / 20.53 × 12 ≈ **40.9元**（不复权）
- **相对于现价跌幅**：-70.7%

### 5.2 极限支撑位审计（基于净现金+重资产清算）

|:---|:---:|:---:|

**极端支撑位**：
- **重置成本支撑**（PB=1）：22.71元 → 市值466亿
- **净现金支撑**：8.45元 → 市值173亿
- **清算价值支撑**（流动资产-总负债的清算折扣）：~12-15元

**概率评估**：CSA 2.0将阳光电源列入HRV的概率约为15-20%（中欧贸易依存度高，全面脱钩概率不大但风险真实存在）。
