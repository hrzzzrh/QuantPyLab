# 研报章节六：估值定价分析

**研究日期：2026年05月23日**

## 一、价值锚定：适配性方法选择与基准定价

### 1.1 方法论筛选理由

三环集团是一家**多业务驱动的成长型制造企业**，具有以下特征：
- 正向盈利、现金流充沛（适用PE、EV/EBITDA、DCF）
- 资产密集度较高（适用PB-ROE交叉验证）
- 业务板块属性差异大（适用SOTP分部估值）
- 处于成长期但增速波动大（PEG参考价值有限，不宜作为主要锚）

**选择组合**：**PE（主锚）+ PB-ROE（副锚）+ SOTP（交叉验证）**，辅以DCF作为理论上下限参考。不采用PS（高净利率下PS失真）、PEG（MLCC周期性强，单年增速误导性高）。

### 1.2 PE估值：历史分位审计与前瞻锚定

**历史分位数据（2018年至今，2032个交易日）**：

| 分位 | PE-TTM | 对应市值（亿元） |
|:---|:---|:---|
| Min | 19.81 | ~570 |
| P10 | 26.45 | ~760 |
| P25 | 30.02 | ~860 |
| **P50（中位）** | **36.34** | **~1040** |
| P75 | 41.29 | ~1180 |
| P90 | 49.64 | ~1420 |
| **当前（2026/5/22）** | **76.62** | **~2204** |

当前PE 76.62处于**七年历史最高位**，远超P90分位（49.64），比中位数（36.34）高出111%。

**历史分位参考的局限性审慎评估**：A股是新兴市场，公司和市场环境剧烈变化，历史分位不能简单套用。三环集团当前面临的三个结构性变化使历史PE区间参考意义大打折扣：
1. **商业模式转变**：从精品利基（插芯70%份额、50%+毛利率）转向规模化竞争（MLCC 2.5%份额、42%毛利率），盈利质量下降，理论上估值中枢应**下移**而非上移
2. **SOFC新业务期权**：SOFC隔膜板从0到近20亿年收入，属于历史上不存在的增量价值，理论上应给予**额外溢价**
3. **政策环境质变**："十五五"规划将电子元器件列为国家级战略产业，估值体系中的政策溢价因子显著提升

**综合判断**：SOFC期权+政策溢价的正向效应，与商业模式退化（毛利率↓、ROE↓）的负向效应形成对冲。以历史中位数PE 36倍为基准中枢，给予以下调整：

| 调整因子 | 方向 | 幅度 | 理由 |
|:---|:---|:---|:---|
| SOFC高增长期权 | 上调 | +8-12x | 2026E贡献净利润7-8亿（+30%增量），高毛利（56%）永久性地提升盈利结构 |
| MLCC周期拐点+国产替代 | 上调 | +2-4x | 近期增长加速，但持续性取决于技术突破 |
| 毛利率/ROE结构性退化 | 下调 | -5-8x | 从50%+毛利率衰退至42%，ROE从20%+降至12-14% |
| 政策溢价提升 | 上调 | +2-3x | "十五五"+供应链安全法规定性改变 |
| 利率高企环境 | 下调 | -3-5x | 全球"Higher for Longer"压制高PE成长股 |
| **综合调整** | — | **+4~6x** | — |

**调整后合理PE区间**：36.34（历史中位）+ 4~6 = **40-42倍**

**基准PE定价**：
- 2026E EPS（中性）= 2.05元
- 合理PE × EPS = 40~42 × 2.05 = **82.0 ~ 86.1元**
- 对应市值 = 1,571 ~ 1,650亿元

> 当前不复权市价115.00元，对应2026E Forward PE 56.1x，显著高于合理PE区间。

### 1.3 PB-ROE交叉验证

ROE与PB存在理论上的线性关系：合理PB = ROE / (Ke - g)，其中Ke为权益成本（取9%），g为永续增长率（取4%）。

| 假设 | 2026E ROE | 隐含合理PB | 2026E BVPS | 合理股价 |
|:---|:---|:---|:---|:---|
| 保守 | 13.0% | 2.60× | 12.8元 | 33.3元 |
| **中性** | **15.0%** | **3.00×** | **13.0元** | **39.0元** |
| 乐观 | 17.0% | 3.40× | 13.2元 | 44.9元 |

PB-ROE模型的定价结果（33-45元）显著低于PE模型（82-86元），**暴露了一个关键矛盾**：公司的ROE水平（12-15%）难以支撑当前9.84倍的PB。即便用最激进的假设（ROE回到历史17%+），合理PB也仅3.4倍。

**这一矛盾的核心含义**：当前市场给予三环集团的PB定价隐含了ROE将大幅回归历史高位（20%+）的预期，而这一假设与公司从轻资产利基向重资产规模转型的现实严重背离。

## 二、估值全变量对冲审计矩阵

### 2.1 估值影响力过滤——审慎思考

在以下审计中，我将严格区分"平庸的关联"与"定价的锚点"：

| 潜在因子 | 估值影响力判断 | 过滤结论 |
|:---|:---|:---|
| 公司深耕陶瓷55年 | 行业knowhow积累是真实的DNA优势，支撑技术追赶 | **少数可溢价因子** |
| "十五五"政策利好 | 利好全行业，不能成为三环集团相对于风华高科的溢价理由 | **零影响（行业共性）** |

## 三、强制性逻辑纯净性校验

### 3.1 区分"进E"与"进PE"的因子

| 因子 | 应进哪里 | 理由 |
|:---|:---|:---|
| SOFC隔膜板2026年增量收入（+18亿） | **E（利润预测）** | 具体可计算的增量收入→增量利润 |
| MLCC价格拐点毛利率回升 (+1.5ppt) | **E（利润预测）** | 具体的毛利率改善→可量化的利润增厚 |
| 政府补贴（~4.5亿） | **E（利润预测）** | 虽非经常性，但持续且可预测，应体现在利润中 |
| "十五五"政策支持 | **进E和PE均有，主要进PE（行业性溢价）** | 政策带来的具体收入增量进E；政策带来的"信心溢价"进PE |
| 粉体自研的技术DNA | **PE（估值倍数）** | 不可量化的竞争优势→关乎长期信心的估值溢价 |
| 地缘断供风险 | **PE（估值倍数）** | 不可量化的尾风险→折价 |
| 毛利率长期下滑趋势 | **PE（估值倍数）** | 结构性退化→持续压低估值中枢 |

### 3.2 锚点回测

修正市值1,300亿 ÷ 2026E EPS 2.05 = **隐含PE 33.1倍**

这一水平：
- 低于历史中位数PE 36.34倍，与商业模式退化（毛利率42% vs 历史50%+）的事实一致
- 高于SOTP隐含PE 32.4倍，反映了一定程度的成长性溢价
- 显著高于PB-ROE隐含价格（39元/股），但PE法和PB-ROE法在ROE<15%时出现系统性偏差是正常的（PB-ROE低估了ROE回升预期的期权价值）

**综合来看，33倍PE是商业模式退化与成长性溢价博弈后的合理均衡水平。**

### 4.2 盈亏比计算

- 现价（不复权市价）：**115.00元**
- 概率加权目标价：**70.00元**
- 保守场景价格：**53.10元**

**盈亏比 = (70.00 - 115.00) / (115.00 - 53.10) = -45.00 / 61.90 = -0.73**

**盈亏比为负**，意味着当前价格不仅低于目标价，且下行空间（至保守场景）大于上行空间（至目标价）。从风险收益比角度，**当前价位不具备买入价值**。

如需达到>2:1的正向盈亏比，买入价格需要降至约**73元以下**（对应PE约36倍2026E）。

## 五、逻辑证伪与极限压力测试

### 5.1 黑天鹅场景模拟

**"一击毙命"事件：日本政府加入美国阵营，对华实施先进陶瓷粉体和精密设备出口管制**

场景描述：
- 日本Sakai化学等核心粉体供应商被禁止向中国MLCC企业供货
- 三环集团自研粉体在中低端规格可自给，但10μF以上高容MLCC所需纳米级粉体无法替代
- 日本/德国精密烧结炉、流延机等设备禁运，产能扩张停滞

在这种场景下：
- MLCC产线利用率下降30-50%，MLCC收入萎缩40%
- 高容产品无法量产，被迫退守中低端市场
- 利润大幅下滑60%+

**"地狱价格"计算（基于清算/重置价值）**：

**地狱价格区间：3.68元（净现金）~ 20元（考虑持续经营价值+品牌+客户关系等无形资产）**

这一计算表明，即使在最极端的情景下（黑天鹅→业务崩溃），三环集团的股价也有坚实的净资产支撑（68亿净现金）。但当前115元的股价距离这一"地狱支撑位"有**极远的距离**，意味着估值泡沫破灭时的回撤幅度可能极为惨烈。
