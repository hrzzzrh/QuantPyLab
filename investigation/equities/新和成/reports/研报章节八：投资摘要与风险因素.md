## 一、投资摘要 (Investment Summary)

### 1.1 公司速写

浙江新和成是全球精细化工领域的多赛道龙头——维生素VA/VE全球第一/前二、蛋氨酸全球第三/中国第一、PPS全球第二/国内唯一全系列供应商、柠檬醛/芳樟醇等香料中间体全球前三。公司以"化工+"和"生物+"双平台，构建了营养品（66%营收）、香精香料（17%）、新材料（10%）、原料药四大板块的多元化对冲结构。2025年营收222.51亿（+3%），归母净利润67.64亿（+15.3%），ROE 21.87%，经营现金流87.97亿远超净利润，资产负债率仅27.73%。

### 1.2 核心投资逻辑

**三层驱动，逐级验证**：

1. **第一层（确定性高）——蛋氨酸量增**：公司蛋氨酸产能从2022年的15万吨跃升至2025年的55万吨（全球第三），18万吨液体蛋氨酸与中石化合资项目已投产，7万吨固蛋技改完成。全球蛋氨酸需求年增5%-6%（年增量10万吨+），叠加中国豆粕减量替代的政策驱动，量增逻辑确定性接近满分（5分）。

2. **第二层（中等确定性）——维生素价格中枢上移**：2026年Q1-Q3全行业多厂集中检修（新和成VE线6-9月检修10-12周、浙江医药3个月、北沙10-11周）形成历史罕见的供给真空，VA/VE价格较年初分别上涨67%和89%。寡头格局（CR3>60%）和下游价格不敏感（维生素占饲料成本仅0.15%-0.51%）支撑提价持续性。万华化学VA新产能（1万吨）是主要不确定性来源。

3. **第三层（成长期权）——新材料/香料价值重估**：PPS四期推进+天津尼龙66产业链（预计2027年投产）+生物基PPS亮相，新材料板块有望在未来3年从21亿成长为50亿+营收体量。全球PPS市场CAGR 8.4%，新能源车/5G/AI驱动需求。香料板块受益于全球天然化和功能性升级趋势（CAGR 6.5%）。

## 二、风险因素（按对股价影响烈度排序）

### 🔴 高风险（可能导致20%+下跌）

**R1. 蛋氨酸地缘溢价全面消退（烈度：极高，确定性：大幅上升）**
- **触发条件**：美伊和平协议签署→霍尔木兹海峡恢复通航（华盛顿邮报5/25报道框架已"达成95%"）→赢创新加坡+住友合计约20%全球产能恢复→蛋氨酸价格从39-40元/kg回落至20-25元/kg
- **影响量化**：若蛋氨酸全年均价降至25元/kg（接近保守场景），净利润较中性预测减少约19亿，EPS减少约0.62元
- **概率评估**：**60%**（显著上升，原评估30%）。这是当前最大的下行风险

**R2. 维生素价格超预期下行（烈度：高，新增）**
- **触发条件**：VA跌破80元/kg、VE跌破75元/kg——当前VA已至83-85、VE 84-86，距此仅有5-10%空间
- **影响量化**：若VA全年均价降至70、VE降至68（保守场景），营收较中性减少约25亿，EPS减少约0.35元
- **概率评估**：40%。下游去库存+万华化学/巴斯夫/新进入者产能释放叠加

### 🟡 中风险

**R3. 万华化学VA产能冲击（烈度：中）**
- 万华化学1万吨VA产能已全产业链贯通（2025年3月），全球产能第二。若其快速通过国际大客户认证并大幅提升产量，将加剧VA供给过剩

**R4. 饲料需求提前达峰（烈度：中）**
- 中国饲料总需求预计2028年达峰，若养殖业去产能加速或疫情爆发，达峰可能提前

### 🟢 低风险

**R5. 人民币持续升值**：Q1已产生1.07亿汇兑损失（去年同期+1.03亿收益），若USDCNY跌破7.0将进一步侵蚀利润

**R6. 存货跌价风险**：2025年末医药化工类库存量同比暴涨88.87%，若产品价格下行需进一步计提减值

**R7. 新业务盈利不达预期**：天津尼龙项目投资巨大，若下游需求不达预期可能拖累整体ROIC

## 三、研究结论象限图 (Final Evaluation Matrix)

基于"确定性"与"预期收益空间"二维矩阵，将新和成定位如下：

```
预期收益空间
    ↑
 高 │  第二象限              │  第一象限 ★
    │  (Speculative Growth)   │  (High-Conviction Buy)
    │                         │
    │                         │    ★ 新和成
    │                         │   (确定性：中高)
    │                         │   (预期收益：中高)
    │                         │   目标价42.07元
 中 │─────────────────────────│  上行空间+30.3%
    │                         │   盈亏比1.73:1
    │                         │
    │  第四象限              │  第三象限
 低 │  (Avoid/Wait)           │  (Defensive Value)
    │                         │
    └─────────────────────────┴──────→
     低                      中                  高
                          确定性
```

**定位结果**：**从第一象限（High-Conviction Buy）移至第一象限与第三象限的交界处**。

**判断依据**：
- **确定性从中高下调至中等**：核心利润来源（维生素/蛋氨酸）受价格周期波动影响加大——VA/VE已从高点回落约20%，中东和平协议使蛋氨酸面临额外不确定性。多业务对冲结构仍提供保护，但不足以完全对冲短期价格下行风险。
- **预期收益从中高下调至中等**：18.4%的上行空间在12个月维度仍有一定吸引力，但盈亏比恶化至0.62:1后风险收益失衡。
- **配置建议调整**：从"核心持仓（Core Holding）"调整为"**观望等待（Wait for Better Entry）**"。当前不建议新增仓位，已在仓的投资者可持有但不加仓。建议等待25-28元区间（PE 11-13x，盈亏比重建至>1.5:1）再考虑介入。
