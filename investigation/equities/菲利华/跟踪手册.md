## 一、核心逻辑验证点（Thesis Validators）

以下指标需随每次财报（季报/半年报/年报）定量验证。任何一项持续恶化将触发逻辑重新评估。

### 1.1 Q布（石英电子布）——最关键的单一验证指标

| 验证指标 | 2025实际 | 2026E (Base) | 2026H1需达到 | 红灯阈值 |
| :--- | :--- | :--- | :--- | :--- |
| Q布季度营收 | ~0.25亿（平均） | Q1:1.0 Q2:2.5 Q3:3.0 Q4:3.8 | Q2≥1.5亿 | Q2<1.0亿 → 量产严重滞后 |
| Q布毛利率 | ~65% | 60-65% | ≥55% | <50% → 定价权丧失 |
| Q布客户 | 英伟达（认证中） | 英伟达+台光电子+生益科技 | 至少2家批量客户 | 英伟达认证转批量失败 |
| 中益新材项目进度 | 2026Q1开工 | Q3试产/Q4量产 | Q2完成设备安装 | Q4仍未能试产 |

## 三、逻辑证伪警报（Invalidation Triggers）

以下信号一旦出现，必须承认投资逻辑的核心支柱已崩塌，直接触发**止损或降仓**，不得以"长期看好"为由死扛。

| 级别 | 证伪信号 | 触发后的行动 |
| :--- | :--- | :--- |
| 🔴 **止损（立即离场）** | ①英伟达下一代芯片平台明确宣布不采用Q布方案 | 彻底清仓。Q布故事归零，公司退回传统石英材料估值（约30x PE，对应股价~30-35元） |
| 🔴 **止损（立即离场）** | ②Q布连续两个季度营收<0.5亿（远低于Base case预期的2.5亿/季） | 立刻减仓至观察仓。量产失败意味着核心增长逻辑崩塌 |
| 🟠 **降仓（减半）** | ③日东纺宣布2027Q1福岛工厂提前量产，且中材科技Q布通过英伟达认证 | 竞争格局从"双寡头"变为"三国杀"，估值溢价消失。减半仓位 |
| 🟠 **降仓（减半）** | ④Q布均价从270元跌破180元（降幅>33%） | 行业从"供给短缺"逆转为"供过于求"。Q布盈利预测需腰斩 |
| 🟡 **警惕（暂持）** | ⑤连续两个季度OCF/NP<50% | 现金利润转化率恶化至危险水平，质疑营收质量 |
| 🟡 **警惕（暂持）** | ⑥毛利率连续两季环比下滑超3个百分点 | 竞争加剧或成本失控的信号 |
| 🟡 **警惕（暂持）** | ⑦ROIC连续三季低于7% | 巨额资本开支未能创造价值，管理层资本配置能力受质疑 |

## 五、定期跟踪checklist

### 每季度（随财报）

- [ ] Q布季度出货量/营收/毛利率是否符合Base case轨道
- [ ] 应收/存货/OCF三项核心财务质量指标
- [ ] ROIC是否触底回升（>8%为改善信号，>10%为确认信号）
- [ ] 分业务毛利率是否稳定或改善
- [ ] 重大合同/认证/项目里程碑更新

### 每月

- [ ] Q布行业价格（是否稳定在250-300元/米区间）
- [ ] 英伟达/半导体行业重大新闻
- [ ] 竞争对手扩产/认证动态
- [ ] 高纯石英砂价格趋势

### 事件驱动

- [ ] 美国BIS出口管制更新
- [ ] 中国半导体/航天产业政策变化
