## 一、价值锚定：适配性方法选择与基准市值

### 1.1 方法选择逻辑

菲利华的商业模式特征为：1）处于高成长转型期（Q布从0→1爆发），盈利处于非线性跃升阶段；2）业务板块属性差异显著（高成长Q布 vs 成熟半导体石英 vs 稳定航空航天 vs 竞争性光伏）；3）重资产投入期（大规模资本开支尚未完全转化为利润）。

基于以上特征，选择三种估值方法进行交叉验证：

| 方法 | 适用理由 | 局限性 |
| :--- | :--- | :--- |
| **PE（市盈率）** | 公司已盈利且处于高速增长期；市场最常用的可比估值框架 | 对PE倍数的选取高度主观；历史分位参考意义受业务结构剧变影响 |
| **PEG（成长性估值）** | 适用于高增长公司，将PE与增速挂钩 | 对增速预测极其敏感；忽略了业务质量差异 |
| **SOTP（分部估值）** | 各板块属性差异大、适用不同估值倍数 | 期间费用分摊有一定主观性 |

暂不适用DCF——公司处于CAPEX高峰期（自由现金流极低），远期终端价值假设的误差会严重扭曲结论。

### 1.2 历史分位审计与当前估值定位

| 统计量 | PE TTM | PB |
| :--- | :--- | :--- |
| 当前值 | **151.62x** | **14.52x** |
| 近5年中位数 | 55.38x | 7.26x |
| 近5年75分位 | 67.82x | 9.14x |
| 近5年90分位 | 116.26x | 11.00x |
| 全历史最大值 | 169.74x | 15.42x |

当前PE和PB均处于历史**约98-99分位**的极端位置。按传统历史分位框架判断，股价严重高估。

**但是——历史分位的参考意义需要谨慎评估**（框架原则）。菲利华正处于"业务结构剧变期"：2025年Q布尚为"故事"（营收占比5%），2026年Q布成为"现实"（预测占比31%）。历史PE分位是在"菲利华=传统石英材料公司"的框架下形成的，而当前市场正在为"菲利华=Q布龙头+航空航天王者+半导体国产替代先锋"重新定价。**简单套用历史分位将严重低估公司的估值中枢上移。**

### 1.3 基准市盈率锚定

采用"可比公司参考 + 增长调整 + 历史中枢上修"的三维锚定法：

**(1) 可比公司参考**：

| 可比公司 | 赛道 | 2026E PE | PEG | 特征 |
| :--- | :--- | :--- | :--- | :--- |
| 中材科技 | 玻纤+特种电子布 | ~25x | ~0.8 | 风电基本盘+电子布弹性 |
| 石英股份 | 高纯石英砂 | ~18x | ~0.6 | 光伏砂基本盘，增速放缓 |
| 宏和科技 | 超薄电子布 | ~40x | ~0.5 | 纯电子布标的，Q布概念 |
| 菲利华（本报告） | 石英材料+Q布+航天 | — | — | 独一无二的赛道组合 |

菲利华无法找到完全可比的公司——"Q布+航空航天石英纤维+半导体石英材料"的三位一体组合在中国A股具有唯一性。这赋予了估值溢价的基础。

**(2) 增长调整（PEG框架）**：
- 2026E EPS增速（vs 2025）：+119%
- 2024-2026E CAGR：~75%
- 参考PEG=0.5-0.8的合理性：对于高确定性（认证壁垒+产能可见）的新兴材料公司，0.6-0.8的PEG是A股成长股的常见区间
- 隐含合理PE：75% × 0.7 = **52.5x 2026E PE**

但PEG框架隐含"增速回归均值"的假设——需考虑Q布高增长的可持续性（2027E增速约60%+）。

**(3) 综合锚定**：

| 维度 | 隐含2026E PE |
| :--- | :--- |
| 历史中枢上修（考虑业务结构升级+50%溢价） | 55x × 1.5 = 82.5x |
| PEG框架（0.7 × 75% CAGR） | 52.5x |
| 可比公司溢价（中材科技25x + 稀缺性溢价80%） | 45x |
| **综合判断——中性情景合理PE** | **60-70x** |

**基准市值** = 2026E归母净利润（Base Case）× 合理PE中枢  
= 9.7亿 × 65x = **630亿元**（对应股价120.7元/股）

**当前市值（672亿）较基准市值溢价约6.7%**——市场定价基本合理，但安全边际不足。

## 二、估值全变量对冲审计矩阵

**审计原则**：将630亿基准市值作为"底色"，通过定性转量化审计得出修正目标市值。

| 审计维度 | 核心审计因子 | **审慎思考（估值影响力过滤）** | 调节幅度 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质 (DNA)** | **技术宗教般的研发信仰**：60年专注石英，14%研发费用率，237名硕士+11名博士，八年攻克M9 Q布 | **辨析**：这种"长期主义"在A股极为稀缺，能赋予确定性溢价。多数A股公司追求短期利润最大化，而菲利华的战略定力符合"十倍股"的气质基础 | **+10%** | 60年技术积淀转化为多赛道的认证壁垒（航空航天5-8年、半导体2-3年），大幅降低了"被颠覆"的概率。长期确定性隐含更低的折现率 |
| | **资本配置效率（ROIC下滑）**：ROIC从16.16%→7.46%，大规模CAPEX尚未贡献等比例利润 | **辨析**：这是真实的效率隐忧，不能以"投资未来"为由回避。但需考虑：80亿CAPEX对应的是Q布（毛利率65%）和合成石英（毛利率57%）的高回报项目。ROIC可能已在2025年触底 | **-3%** | ROIC趋势是衡量管理层资本配置能力的核心指标。若2026年ROIC无法回升至10%+，将验证"投资效率低下"的怀疑 |
| **宏观：政策与战略对齐** | **三重国家级战略共振**：半导体自主可控+航空航天强国+新材料战略矿种 | **辨析**：政策红利是真实的，但需要过滤"是否直接转化为定价权"。高纯石英矿的战略管制→矿源壁垒→竞争格局改善→毛利率支撑——这是一条清晰的传导链。50亿专项基金→10%税率→直接增厚EPS | **+8%** | 政策环境降低了公司的"政治风险折现率"。军品订单的预算确定性提供反周期特征。但补贴退坡风险需在中长期关注 |
| **宏观：地缘与外部约束** | **中美科技脱钩双刃剑**：出口管制倒逼国产替代（利好），但存在"长臂管辖"风险（Q布若被纳入管制清单） | **辨析**：利空因子的概率极低——石英材料是基础耗材而非核心技术，纳入管制的政治成本远高于收益。国产替代的利好却是确定性事件 | **+3%** | 净影响偏正面——地缘风险更多是国产替代的催化剂而非菲利华自身的风险源。但需为Q布管制风险保留少量折价 |
| **市场：流动性与风格** | **高PE成长股审美**：当前市场风格偏好AI/半导体相关的高弹性标的，菲利华完美契合 | **辨析**：这是双刃剑——当市场风格切换（如转向红利/价值），高PE标的将首当其冲。但AI算力+半导体国产替代的故事在未来12个月内有极强的市场号召力。风格溢价有基本面支撑（Q布业绩爆发），非纯炒作 | **+5%** | 流动性与风格维度的溢价最不稳定，一旦市场转向，这部分溢价会率先消失。但当前至2026H2，AI算力叙事大概率仍为市场主线 |

## 三、逻辑纯净性校验（Consistency Guardrails）

### 3.1 E（盈利）与 PE（估值）的分离审计

| 利好/利空因子 | 已计入 E（盈利预测） | 已计入 PE（估值调节） | 是否存在重复计算？ |
| :--- | :--- | :--- | :--- |
| Q布产能爬坡 | ✅（P×Q模型中Q=400万米） | ❌ | 无重复 |
| Q布价格维持高位 | ✅（P×Q模型中P=270元/米） | ❌ | 无重复 |
| 政策补贴降税 | ✅（有效税率从8%→10%） | ✅（+8%政策溢价） | ⚠️ 需拆解：税率影响已进E，但政策稳定性、矿源壁垒等无法量化的信心因素进PE，整体无重复 |
| 航空航天订单确定性 | ❌（未单独拆出预测） | ✅（+10% DNA溢价） | 无重复——军品订单确定性是"不可量化入E"的信心因素 |
| 国产替代趋势 | ✅（隐含在材料业务15%增速中） | ⚠️（部分重叠在+8%政策溢价中） | ⚠️ 有轻度重叠——国产替代"量"的贡献已计入E中的15%增速，"信心/溢价"的贡献计入PE调节中的+8%。调节幅度已考虑了该重叠 |
| 市场风格/流动性 | ❌（不可计入E） | ✅（+5%风格溢价） | 无重复 |

**锚点回测**：修正后隐含2026E PE = 775亿/9.7亿 = **79.9x**。该PE显著高于近5年历史75分位（67.82x），但低于90分位（116.26x）和当前PE（151.62x）。在公司业务结构发生根本性变化（Q布从0→1，毛利率从47%→53.5%）的背景下，79.9x的隐含PE具备商业逻辑支撑：
- Q布是毛利率65%+、渗透率从0起步、全球仅3家供应商的超级赛道
- 航空航天石英纤维是国内垄断的军品资产（通常享有30-40%估值溢价）
- 79.9x PE对应2025→2026E的119%增速，隐含PEG仅0.67——处于成长股的合理区间

## 五、逻辑证伪与极限压力测试

### 5.1 黑天鹅场景模拟："一击毙命"事件

**场景定义**：英伟达因技术路线变更或地缘政治原因，宣布Rubin Ultra平台放弃Q布方案，改用PTFE基材或其他替代材料。

**影响路径**：
- Q布需求预期瞬间崩塌（从1800万米→0）
- 菲利华12亿元的Q布扩产投资全部沦为沉没成本
- 市场对公司从"AI算力核心标的"重新定价为"传统石英材料公司"
- PE估值从80x向历史中枢50x回归

**地狱价格计算**：
- 2026E除Q布外归母净利：材料4.63亿 + 制品1.11亿 ≈ 5.74亿（来自SOTP分部模拟）
- 清算情景下PE：30x（仅传统石英材料+航天业务）
- 极限市值：5.74 × 30 = **172亿**
- 极限股价：172亿/5.22亿 = **33.0元/股**（较当前-74.3%）

**发生概率评估**：<5%（Q布在技术上对高频高速PCB有明显优势，且英伟达已投入大量研发资源适配）

### 5.2 次级风险场景

**场景B**：日东纺提前完成福岛工厂扩产（2027Q1而非2027H2），且中材科技Q布通过英伟达认证，供给缺口提前弥合，Q布价格从270元/米暴跌至150元/米。

- 2026E NP从9.7亿下调至约7亿（Q布毛利腰斩）
- 合理PE从65x下调至45x（竞争恶化+增长降速）
- 股价：7/5.22 × 45 = **60.3元**（较当前-53%）

**发生概率评估**：15-20%

## 六、估值结论

1. **当前股价（128.66元）处于"公允偏贵"区间**——基准市值630亿叠加+23%审计调节后目标市值775亿（148.5元），上行空间仅15.4%，而黑天鹅/竞争恶化情景下下行空间高达53-74%

2. **最佳投资时点不在当前**——建议等待以下信号之一再建仓：
   - 股价回调至100-110元区间（对应2026E PE 54-59x，回归历史中位数，提供足够安全边际）
   - Q2财报确认Q布量产进度超预期（触发对标Bull case的重估）

3. **估值矛盾的核心**：菲利华的"故事"（Q布龙头+航空航天王者）确实极具吸引力，但当前股价已经充分甚至过度反映了这一叙事的乐观预期。市场为公司定价时隐含了"Q布放量+竞争格局不变+行业景气持续"三重要假设同时成立，任何一项出现偏差都可能导致估值的大幅回调。

4. **如果持有**：以148.5元为第一目标位（2026年年报验证Q布放量），196元为第二目标位（需要Bull case全面兑现）。止损位设在100元（对应2026E PE 54x，跌破历史中位数意味着市场开始质疑Q布逻辑）。
