# 研报章节四：外部边界与宏观政策

**研究日期**：2026年05月27日

---

> **前序模块红线回顾**：
> 本模块继承以下核心结论——(1) 药明康德72%收入来自美国客户，地缘政治是系统性核心风险；(2) BIOSECURE Act已于2025年12月18日签署成法；(3) 全球CDMO市场景气度处于上行周期，多肽/TIDES赛道最为强劲；(4) 公司竞争护城河（转换成本+一体化）深厚但面临政治力量的"爆破解构"风险。

## 一、地缘政治风险：BIOSECURE Act全景解构

### 1.1 立法历程与当前状态

| 时间节点 | 关键事件 | 市场影响 |
|:---|:---|:---|
| 2024年1月 | BIOSECURE Act草案首次流出，点名药明康德 | 股价暴跌，业务短期承压 |
| 2024年5月 | 修订版增加药明生物；增设至2032年的脱钩期限 | 市场初步消化，但观望情绪浓厚 |
| 2024年9月 | 众议院高票通过 | 下跌再度加速 |
| 2024年12月 | **未能进入FY25 NDAA，立法进程受挫** | 反弹，但不确定性的阴影未散 |
| 2025年9月 | 修订版纳入FY26 NDAA修正案，**取消直接点名药明康德** | 利好，但系"换个方式追剿" |
| 2025年12月18日 | **BIOSECURE Act作为FY26 NDAA第851条正式签署成法** | 靴子落地，但执法路径模糊 |

### 1.2 现行BIOSECURE Act对药明康德的穿透分析

**核心条款逐层解析**：

**第一层：谁是"受关注生物技术公司"(BCC)？**
- 自动纳入：所有被列入DoD 1260H清单（"中国军事企业清单"）且涉及生物技术设备/服务的实体
- 动态指定：OMB可基于国家安全评估独立指定其他实体
- **当前状态**：药明康德**不在**1260H清单上；WuXi Biologics和WuXi XDC也**不在**清单上
- **但**：2025年12月18日，参众两院多个委员会主席联名致函DoD，**建议将药明康德、药明生物及WuXi XDC加入1260H清单**
- **更麻烦的是**：据报道，五角大楼已向国会表示正在评估将药明康德纳入清单；2026年初《联邦公报》曾短暂发布含药明康德的1260H新版清单后又紧急撤回
- **关键时间节点**：OMB须在2026年12月前公布官方BCC名单

**第二层：禁令覆盖什么范围？**
- 联邦机构不得采购/获取BCC的生物技术设备或服务
- 联邦机构不得与"使用BCC设备/服务履行联邦合同"的任何实体签订/延长/续签合同
- 联邦贷款或拨款不得用于BCC的生物技术设备或服务

**第三层：禁令何时生效？**
- OMB须在2026年12月前公布BCC名单并发布实施指南
- 此后FAR（联邦采购条例）须在一年内完成修订（预计2027-2028年）
- **FAR修订后，禁令才真正具有强制执行力**
- 对生效前已签署的合同设**5年过渡期**

**第四层：对药明康德的实际影响测算**

| 收入类型 | 占比 (2025) | BIOSECURE冲击评估 | 收入影响区间 |
|:---|:---:|:---|:---:|
| 美国联邦资助项目直接相关 | ~10-15% | 一旦FAR修订后直接受限 | -100% |
| 美国商业客户（非联邦资助） | ~25-35% | 不受法案直接禁止，但客户可能主动"去风险" | -20%至-40% (客户自发转移) |
| 美国客户（有联邦合同条款波及） | ~20-30% | "合同商流转"条款可能触发间接受限 | -50%至-80% (取决于客户联邦业务占比) |
| 欧洲客户 | ~11% | 不受影响；可能受益于客户寻求中国替代 | 0%至+10% |
| 中国客户 | ~12% | 不受直接影响 | 0%至+5% |
| 其他地区 | ~4% | 不受影响 | 0% |
| **合计加权冲击** | **100%** | — | **-25%至-50% 营业收入** |

**关键判断**：BIOSECURE Act对药明康德的最大冲击窗口在**2028-2032年**。在此之前，实际禁令尚未生效，客户处于"观望但继续使用"和"逐步分拆"的混合状态。2025年美国客户收入逆势+34.3%的现象恰恰说明：制药企业的短期理性是"趁窗口期加速完成现有项目"，而长期理性才是"建立替代供应链"。

### 1.3 1260H清单风险的精算

药明康德被列入1260H清单的概率难以精确量化，但从以下信号可推断：

| 信号 | 方向 | 强度 |
|:---|:---:|:---:|
| 国会两院主席联名推荐加入 | 被加入 | ★★★★★ |
| 五角大楼内部评估中 | 被加入 | ★★★★☆ |
| 《联邦公报》短暂发布含药明康德版本后撤回 | 技术性错误 or 政策博弈 | ★★★☆☆ |
| 特朗普政府对中国科技企业的整体态度 | 被加入 | ★★★★☆ |
| 药明康德在美国建厂（Delaware）+新加坡 | 去风险 → 降低被加入概率 | ★★☆☆☆ |
| 美国制药业对药明康德的依赖度（79%企业有依赖） | 减少激进执法可能性 | ★★★☆☆ |
| 最新版法律删除直接点名 | 政策温和化信号 | ★★☆☆☆ |

**综合判断**：药明康德被列入1260H名单的概率约**40-60%**，时间节点最早在2026年下半年至2027年初。被列入后的过渡期（5年）虽然提供了缓冲，但商业信心层面的冲击将是立即的。

### 1.4 【新增重大风险】USITC 332-610调查：中国生物技术企业补贴与定价审查

2026年2月26日，美国国际贸易委员会（USITC）依据参议院拨款委员会的授权，正式启动编号**332-610**的独立事实调查，调查标题为"中国政府对生物技术行业的国家支持和定价行为对美国产业的影响"。这是BIOSECURE Act之外，美国对中国生物技术产业链施加的**第二道制度性约束框架**。

**调查核心**：

| 要素 | 内容 |
|:---|:---|
| **调查范围** | 基因组测序、合成生物学、**活性药物成分（API）制造** |
| **授权来源** | 参议院拨款委员会（《商业、司法、科学及相关机构拨款法案2026》） |
| **核心问题** | 中国政府对生物技术企业的补贴程度、市场产能过剩程度、对美国市场份额和竞争力的影响 |
| **最终报告** | 2027年1月22日发布 |
| **后续权力** | 报告可能为后续的关税、贸易限制、反补贴措施提供事实依据和法理支撑 |

**听证会就在今天（2026年5月27-28日）**：
- 5月11日：出席听证会申请截止
- 5月14日：预听证简报截止
- **5月27-28日**：公开听证会（华盛顿特区 USITC 主听证室 + WebEx 远程）
- 6月11日：听证后简报截止
- 7月17日：所有书面提交截止

**对药明康德的影响路径**：

332-610调查虽属"事实调查"（不具有直接执法权），但其潜在影响深远：
1. **为反补贴关税提供法理基础**：如果USITC认定中国政府对API/CDMO行业存在"不公平补贴"造成美国产业损害，商务部可据此启动反补贴税调查
2. **加速供应链转移**：调查报告的结论可能进一步推动美国药企的"去风险"决策，加剧从中国CDMO向印度/韩国/美国本土的订单转移
3. **与BIOSECURE形成合力**：BIOSECURE从"国家安全"角度限制联邦采购，332-610从"产业损害"角度施压关税/补贴——两者可能是"两条腿走路"的协同策略
4. **仿制API业务受影响最大**：调查明确点名API制造，药明康德小分子API业务（占化学业务约55%）中，仿制药中间体/API部分可能首当其冲

**听证会前的信号研判**：
- 美国biotech协会、学界、政策圈证人将在5月27-28日的听证会上陈词，其将中国生物技术竞争力定性为"正常产业优势"还是"补贴扭曲的不公平竞争"，将在听证会上基本清晰
- 已有媒体报道指出，MNC（跨国药企）的法务团队已开始在与中国药企的BD合同中植入BIOSECURE合规声明条款
- 药明康德管理层在3月业绩会上已表态"公司治理完全透明，与政府或军事机构无关联"

**风险评级**：★★★★★（与BIOSECURE Act同级，是2026年新增的最重要外部风险变量）

## 二、关税风险：药品领域的贸易壁垒（重大更新）

### 2.1 关税政策最新进展

**2026年4月2日**，特朗普签署Proclamation 11020号总统公告，依据232条款对进口专利药及API实施大规模关税。核心机制为**三级税率体系**：

| 条件 | 额外从价关税 | 说明 |
|:---|:---:|:---|
| 未签任何协议（中国企业默认） | **100%** | 自2026年7月31日（大型企业）或9月29日（其他）生效 |
| 仅签Onshoring协议 | **20%** | 需向商务部提交详细的产能回迁计划，2030年4月提高至100% |
| 签Onshoring + MFN定价协议 | **0%** | 需同时承诺产能回流和最惠国定价（价格对齐OECD最低价） |

**Onshoring Agreement申请——2026年6月12日截止！**

配合232条款，美国商务部对企业提出了前所未有的"量化尽调"要求。申请onshoring agreement以获取20%较低税率的企业必须在2026年6月12日前提交详细材料，包括：
- 产品组合、生产地点、合同制造商信息
- 美国投资规模、产能回流计划
- **API美国制造比例的当前值和目标值**（按数量和收入双口径）
- 2025年1月目前在美国生产的产品比例 → 2029年1月20日目标比例
- 需要由公司高级官员签署sworn statement确认内容真实准确

**药明康德的应对**：
- 公司早在2021年就在美国特拉华州米德尔顿建设含原料药与制剂的一体化生产基地（占地约77万平方米），计划2026年底投入运营
- 但需注意：药明康德本身是CDMO服务商而非专利药企，关税的直接对象是进口的**成品专利药和API**。药明康德作为中国企业出口API到美国，在无协议情况下将面临100%关税
- 药明康德的美国Delaware基地预计2026年底运营，初期产能有限，短期内难以替代中国产能的出口角色
- 公司于2026年5月15日宣布发行67.8亿元零息可转债，90%融资用于全球产能建设，本质上是用资本开支对冲关税风险

## 三、中美供应链脱钩：药物制造本土化趋势

### 3.1 美国"医药制造本土化"政策方向

BIOSECURE Act只是美国整体医药供应链本地化战略的一部分。更广泛的趋势包括：

- NSCEB报告要求美国政府在5年内投入至少$150亿以应对中国生物技术崛起
- 美国国防部推动关键药物原料药本土化生产
- 生物技术被视为"战略基础设施"，与芯片同等待遇
- JLL预测：大型跨国CDMO营收5年内翻倍（2020-2025 benchmark），受益于供应链重构

### 3.2 药明康德的应对：海外产能布局与融资战略

**最新重大举措：67.8亿元零息可转债发行（2026年5月15日公告）**

药明康德于2026年5月14日与摩根士丹利、花旗、高盛三大顶级投行签署认购协议，发行67.8亿元人民币（约10.23亿美元）的美元结算零息可转换债券，2027年到期。核心条款：

| 要素 | 内容 |
|:---|:---|
| **发行规模** | 67.8亿元（约10.23亿美元） |
| **票息** | 0%（零息）——顶级信用背书的体现 |
| **到期日** | 2027年 |
| **H股转换价** | 153.00港元（较5月14日收盘130.10港元溢价17.6%） |
| **资金用途** | 90%用于全球产能和能力发展，10%用于一般公司用途 |
| **经办人** | 摩根士丹利、花旗、高盛 |
| **上市地** | 维也纳证券交易所 + 香港联交所（转换H股） |

**战略解读——为何在手现金300亿还要发债？**

1. **"用华尔街的钱建全球的厂"**：零息可转债是极为便宜的融资工具（0%票息相当于免费借钱），转股价153港元也意味着只有当股价大幅上涨后投资者才愿意转股，当前发行实际上是一次"低息锁定未来资本"
2. **全球化产能竞赛需要弹药**：天量在手订单（597.7亿）需要匹配产能，而海外产能建设的门槛极高。药明康德2026年计划资本开支65-75亿元，为近四年最高
3. **美元负债对冲美元资产**：以美元结算的可转债提供了天然的外汇对冲——公司72%收入以美元计价，美元负债可对冲人民币波动的利润影响
4. **信号意义**：在BIOSECURE Act+关税围剿的双重压力下，华尔街三大投行依然联合承销药明康德的可转债，本身就是对"脱钩论"的有力反驳

**海外产能布局汇总**：

| 基地 | 地点 | 计划运营时间 | 产能类型 | 战略意义 |
|:---|:---|:---|:---|:---|
| 美国米德尔顿 | Delaware, US | **2026年底** | 小分子制剂/API（77万平方米） | 规避关税+1260H风险，对美onshoring要求的关键回应 |
| 新加坡基地 | Singapore | 2027年一期 | 研发+小分子生产 | 亚洲枢纽，服务全球客户 |
| 瑞士库威 | Switzerland | 已运营（2024扩产） | 口服制剂（产能翻倍） | 欧洲市场桥头堡 |
| 中国常州/泰兴/金山 | 江苏 | 已运营 | API+多肽（最大产能集群） | 成本优势核心区 |
| 常州新基地 | 江苏 | 提前启动中 | 小分子/多肽扩产 | 满足在手订单597.7亿的超预期需求 |

**评估**：海外产能布局方向正确，且美国Delaware基地在2026年底投入运营的时间节点恰好与关税生效（2026年7月31日）、OMB名单公布（2026年12月）形成对冲。但Delaware基地初期产能规模有限，短期内完全"替代中国产能"不现实。真正的产能替代可能需要5-10年。67.8亿可转债的发行表明公司正在以超常规速度推进全球化——这不是"要不要全球化"的问题，而是"是否能在这场逆全球化浪潮中跑得比政策快"的生存竞速。

## 四、宏观环境变量

### 4.1 利率与折现率

药明康德作为高成长性（但受地缘政治压制的）医药服务公司，估值对利率敏感性较高。

- **美联储降息预期**：2026年市场的核心叙事之一是Fed可能开启降息周期。降息将降低DCF模型的折现率，直接推升成长股估值，利好药明康德。
- **当前利率环境**：美联储维持较高政策利率（~4.25-4.5%），但市场隐含2026年下半年2-3次降息。若降息兑现，药明康德PE倍数可能出现5-10%的扩张。

### 4.2 汇率损益

2023-2024年人民币兑美元贬值期间，药明康德录得大额汇兑收益（财务费用为负）。2025年汇率走势趋稳，财务费用转正3.98亿元。汇率是双刃剑：

- **人民币贬值** → 中国产能更具成本优势、美元收入折算更多人民币 → 利好盈利
- **人民币升值** → 削弱成本优势、美元收入折算缩水 → 利空盈利

公司通过套期保值工具管理汇率风险，但敞口仍然较大（72%收入以美元计价）。

### 4.3 中东局势传导

药明康德管理层在业绩会上提到"中东局势可能对部分上游原材料构成潜在影响"。具体而言：
- 石化基础原料价格波动可能传导至化学试剂和溶剂成本
- 海运物流成本（红海航线受阻 → 绕行好望角）增加供应链时间和成本
- 目前影响"尚需时间传导"，未见直接量化冲击

## 五、IPO/融资环境对下游客户的影响

药明康德的终端需求高度依赖于全球biotech融资环境（尤其是美国）：

- **2023-2024年**：美联储加息周期导致全球biotech融资冻结，大量早期biotech削减R&D开支，药明康德中国/欧洲客户收入负增长
- **2025年起**：融资环境改善（利率预期回落 + XBI指数反弹），biotech客户重新活跃
- **2026年展望**：若降息落地，biotech融资将进一步回暖，支撑药明康德CRO端需求

## 六、外部边界核心结论（重大更新）

1. **BIOSECURE Act是药明康德面临的最核心外部风险，没有之一。** 该法案已于2025年12月签署生效，但实际禁令尚未触发（预计2027-2028年FAR修订后）。关键变量是OMB在2026年底前公布的BCC名单——若药明康德未被列入，股价和估值将出现显著修复；若被列入，需评估5年过渡期内的业务平滑过渡能力。

2. **【新增】USITC 332-610调查构成第二重系统性风险。** 该调查虽为"事实调查"不具直接执法权，但其聚焦API制造业、基因组学和合成生物学的范围与药明康德核心业务高度重合。5月27-28日的听证会过后，调查叙事方向将基本清晰。此调查可能为2027年后的反补贴关税提供法理基础，与BIOSECURE形成"双管齐下"的围堵格局。

3. **关税风险从"极不明朗"升级为"清晰但可对冲"**。2026年4月2日的232条款公告明确了100%/20%/0%三级税率，6月12日的onshoring agreement申请截止日在即。药明康德的美国Delaware基地（2026年底运营）提供了关键的对冲通道，但初期产能有限的制约不可忽视。

4. **药明康德67.8亿元零息可转债发行是应对逆全球化的战略性资本动员。** 0%票息+华尔街三大投行联合承销，既展示了公司信用实力，也表明其在主动加速全球化——"用华尔街的钱建全球的厂"的策略逻辑清晰，但执行风险在于产能建设速度能否跑赢政策围堵速度。

5. **宏观利率环境当前偏利空（高利率压制成长股估值），但边际改善预期明确（2026年降息）。** 这是药明康德估值修复的潜在催化剂。但地缘政治风险溢价的上升可能抵消降息带来的估值修复效应。
