## 一、投资摘要 (Investment Summary)

### 1.1 一句话结论

**药明康德Q1业绩超预期验证"双引擎"增长模式（小分子D&M +80.1% + TIDES指引+40%），但USITC 332-610调查与BIOSECURE Act构成双重地缘压力。当前估值15.2x PE处于历史极端低位（8%分位），全变量修正公允价值约103元，盈亏比0.70较前次改善。建议等待5月27-28日332-610听证会信号及Q2半年报后再做加仓决策。**

### 1.2 核心投资逻辑（5点精要——重大更新）

**1. 小分子D&M从"稳定器"升级为"双引擎之一"——本次报告最重要的增量发现**

2026Q1小分子D&M业务收入69.3亿，同比增长80.1%，远超预期。管线从3,452个分子增至3,550个（+97个QoQ），商业化项目从83个增至89个，III期从91个增至94个。这种"管线成熟度质变"——大量早期分子集中推进到后期阶段——带来的收入弹性是结构性的，而非一次性的。这意味着药明康德的增长叙事正在从"TIDES单引擎"进化为"TIDES+传统小分子双轮驱动"。确定性评分从3分大幅上调至4.5分。

**2. TIDES Q1增速骤降至+6.1%——最需要关注的风险信号**

尽管公司明确指引全年+40%（规模超150亿），且领先指标强劲（D&M客户+28%，分子数+59%），但Q1仅增6.1%的事实意味着Q2-Q4需要平均增速约+48%才能达成指引。如果Q2半年报TIDES增速仍低于30%，应警惕全年指引下调风险。

**3. USITC 332-610是BIOSECURE之外的"第二只靴子"**

2026年5月27-28日举行的USITC 332-610调查听证会，聚焦中国政府在基因组学、合成生物学和API制造领域的补贴和定价行为。虽然属于"事实调查"无直接执法权，但其2027年1月的最终报告可能为后续反补贴关税提供法理基础。这构成了独立于BIOSECURE的新的地缘风险通道。

**4. 估值安全边际改善但折价仍在扩大**

PE从15.83x进一步压缩至15.21x（8%分位），PB从3.78x降至3.64x。Q1扣非净利润TTM增长（132亿→152亿）+ 股价回调（106.80→102.63）的双重作用下，估值变得更便宜。但市场的"折价定价"也在加深——332-610调查+关税onshoring deadline（6月12日）正在推高地缘风险溢价。

**5. 67.8亿可转债既是对冲也是信号**

华尔街三大投行联合承销0%票息可转债，资金90%用于全球产能建设。这既是公司对逆全球化趋势的"武装自己"，也是资本市场对药明康德信用的"投票"。但短期内CB arb策略（做空正股+做多转债）可能压制股价。

## 三、研究结论象限图 (Final Evaluation Matrix)

```
预期收益空间
    ↑
50% │  第二象限              第一象限
    │ (Speculative Growth)   (High-Conviction Buy)
    │              ★ 药明康德(当前→修正)
25% │    ─────────────────────
    │  第四象限              第三象限
 0% │ (Avoid/Wait)           (Defensive Value)
    └──────────────────────────────→ 确定性
    低                      高
```

**定位修正：从"第三象限与第一象限的交界地带"略微向第一象限方向移动**

**定性**：
- **确定性**：中等（3.5/5，较前次3.2上调）。小分子D&M+80.1%提供了新的高确定性增长点，但332-610调查+BIOSECURE仍然压制中长期确定性。
- **预期收益空间**：中等偏高（Base +23.7%，较前次+10.5%大幅改善）。主要因股价回调(-3.9%) + EPS上调(+7.6%)的双击效应。

**投资建议（Q1后修正）**：
- **当前仓位建议**：5-8%（较前次5%小幅上调）。盈亏比从0.26改善至0.70后，风险收益结构更为合理。但鉴于5月27-28日332-610听证会+6月12日onshoring deadline两个短期催化剂的不确定性，建议在听证会信号明晰后再做仓位调整。
- **核心决策节点**：
  1. **短期（1-2周内）**：332-610听证会（5月27-28日）的叙事走向。若听证会释放温和信号，可在MA60(100.20)附近适度加仓。
  2. **中期（1-3个月）**：Q2半年报（7月底）验证小分子D&M增长持续性+TIDES增速恢复情况。
  3. **长期（6个月）**：OMB公布BCC名单（2026年12月前）。若药明康德未被列入→目标仓位可上调至组合10-15%；若被列入→清仓或减至观察仓(≤3%)。
- **适合投资者画像**：对地缘政治风险有独立判断、能接受短期20%+波动、愿意在中美科技博弈的"信息不对称"中寻找机会的逆向投资者。
