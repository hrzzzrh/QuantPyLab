## 一、价值锚定：适配性方法选择

### 1.1 方法论筛选

基于京东方"重资产+周期波动+低ROIC"的特征，筛选估值方法：

| 方法 | 适用性 | 理由 |
|:---|:---|:---|
| **PE** | ★★★★☆ | 最常用，但须注意利润周期性（PE在利润高峰时低、低谷时高） |
| **PB-ROE** | ★★★★☆ | 重资产企业天然适合PB估值，ROE是PB的锚 |
| **PCF（市现率）** | ★★★★★ | OCF远大于净利润，最能反映真实现金产出能力 |
| DCF | ★★☆☆☆ | CAPEX周期不确定、FCFF极不稳定，参数敏感度极高 |
| EV/EBITDA | ★★★☆☆ | 排除了折旧政策差异，适合重资产同行比较 |
| PEG | ★☆☆☆☆ | 利润增速低且不稳定，不适用 |
| SOTP | ★★★☆☆ | 多业务板块，但非核心板块体量太小（<20%营收） |

**选择组合**：**PB-ROE**（主）+ **PE**（辅，基于概率加权EPS）+ **PCF**（交叉验证）

**基准PE估值（Base）**：目标市值1,945亿，目标价**5.25元**（不复权）。

### 1.3 PB-ROE估值法（主力方法）

**前提**：ROE从2023年的1.89%恢复至2025年的4.39%，2026E Base为4.7%。PB的"公允值"取决于ROE能否覆盖资本成本。

```
公允PB = (ROE - g) / (COE - g)
```

参数设定：
- ROE（Base 2026E）= 4.7%
- COE（股权成本）= 无风险利率3.0% + 股权风险溢价5.5% × β(1.2) ≈ 9.6%
- g（永续增长率）= 2.0%

```
公允PB = (4.7% - 2.0%) / (9.6% - 2.0%) = 2.7% / 7.6% = 0.36x
```

**当前PB 1.14x vs 公允PB 0.36x → 当前定价包含了极高的ROE改善预期**。要使PB 1.14x成立，隐含ROE需达到：
```
1.14 = (ROE - 2%) / (9.6% - 2%)
ROE = 1.14 × 7.6% + 2% = 10.7%
```

换句话说，市场定价隐含的预期是京东方ROE将提升至10.7%——这大约是2021年峰值（23.9%）的一半，但远高于当前的4.4%。

**PB-ROE情景矩阵**：

| ROE水平 | 对应PB（公允） | 对应市值（亿） | 对应股价（元） | 对应情景 |
|:---|:---|:---|:---|:---|
| 3%（长期低迷） | 0.13x | 176 | 0.48 | 极端悲观 |
| 4.7%（2026E Base） | 0.36x | 479 | 1.29 | PB-ROE理论公允 |
| 7%（中期修复） | 0.66x | 886 | 2.39 | 中性偏保守 |
| 10%（牛市周期） | 1.05x | 1,416 | 3.82 | 接近当前价 |
| 15%（超级周期） | 1.71x | 2,300 | 6.21 | 超乐观 |

**PB-ROE基准估值**：以7% ROE（相当于行业正常化盈利水平，约为2024-2025年水平上修50%）为合理目标，对应公允PB 0.66x，目标市值886亿，目标价2.39元。

### 1.4 PCF（市现率）交叉验证

| 指标 | 当前值 | 历史低位 | 合理区间 | 
|:---|:---|:---|:---|
| PCF(TTM) | 3.26x | ~2.0x（2021年） | 4-6x（正常化） |
| EV/EBITDA | ~4.5x（估算） | ~3x | 5-8x |

京东方当前PCF 3.26x处于历史低位——从现金流角度看，公司被低估。但这是"价值陷阱"特征：如果公司持续将经营现金流再投资于低回报的CAPEX，高现金流本身并不能为股东创造价值。

假设PCF合理区间4-6x，对应OCF/股1.28元，则：
- 4x PCF → 股价5.12元，市值1,897亿
- 6x PCF → 股价7.68元，市值2,845亿

**PCF目标中枢**：5x PCF → 股价6.40元（但需公司停止毁灭价值的CAPEX，这一前提当前不成立）。

### 3.3 修正目标市值

```
修正目标市值 = 基准市值 × (1 + Σ调节幅度) = 1,550亿 × (1 - 38%) = 961亿
修正目标价（不复权） = 961亿 / 370.44亿股 = 2.59元
```

**注**：若考虑ITC风险解除的利好（原-8%折价取消），调节系数修正为Σ -30%，则：
```
乐观修正目标市值 = 1,550亿 × (1 - 30%) = 1,085亿
乐观修正目标价 = 1,085亿 / 370.44亿股 = 2.93元
```

### 3.4 锚点回测

修正后的隐含PE（Base EPS 0.17）：
```
隐含PE = 2.23 / 0.17 = 13.1x
```

13.1x PE低于5年历史中位数24.9x，但**并非不合理**：

1. **历史PE被结构性高估**：A股面板股在2019-2021年享受了"国产替代"溢价+央行放水下的整体高估值环境。当前宏观环境（利率更高、增长更慢、地缘风险更大）下，面板行业不可能回到那种估值水平。
2. **台股参照系**：友达/群创长期PE在8-15x区间波动。京东方13.1x已处于台股参照系的上沿，反映了一部分大陆市场流动性溢价。
3. **PB回测**：修正PB = 2.23 / (1344.8/370.44) = 2.23 / 3.63 = 0.61x。与PB-ROE公允价值0.66x（ROE=7%情景）基本吻合，证明修正具有逻辑一致性。

## 五、逻辑证伪与极限压力测试

### 5.1 黑天鹅场景模拟

**"一击毙命"事件**：ITC OLED禁令最终生效（14年8个月美国禁售），同时面板价格因全球经济衰退暴跌至现金成本线以下。

在此场景下：
- 成都8.6代OLED线630亿投资丧失主要目标市场（苹果MacBook/iPad的美国份额），ROI趋近于0
- LCD面板价格跌至现金成本（产能过剩+需求崩塌），毛利率归零
- 资产减值可能超过100亿（存货+固定资产）
- 长期借款1,016亿面临再融资压力

**极限支撑位审计（清算价值法）**：

| 资产 | 账面价值（亿） | 清算回收率 | 清算价值（亿） |
|:---|:---|:---|:---|
| 货币资金 | 743 | 95% | 706 |
| 固定资产（扣除受限后） | 1,863 - 964(受限) = 899 | 40% | 360 |
| 在建工程（扣除受限后） | 529 - 163(受限) = 366 | 50% | 183 |
| 存货 | 255 | 60% | 153 |
| 应收账款 | 318 | 70% | 223 |
| 其他 | — | — | 300 |
| **资产清算价值合计** | — | — | **≈1,925** |
| **减：总负债** | 2,272 | 100% | 2,272 |
| **净资产清算价值** | — | — | **≈ -347** |

**深重结论**：即使在有序清算情境下，京东方的净资产清算价值为负值（负债超过可回收资产价值）。这意味着：
1. **估值底线不在资产清算价值，而在持续经营价值**
2. 公司不能以"资产折价"概念来锚定估值——因为资产已经被负债完全覆盖
3. 唯一的估值支撑是**持续经营产生的现金流折现**

**持久经营下的极端极限价**：
假设OCF从488亿腰斩至245亿（比2023年低谷还低），PE=15x（极端悲观），每股价值 = (245×15)/370.44 = 9.9元。但这是基于OCF而非净利润的估值——如果基于净利润，极端情况下净利润可能亏20-30亿（2023年接近亏损），此时PE/股值失去意义。

**地狱价格（终极锚点）**：基于0.3x PB（A股周期股历史极端低位PB）× 每股净资产3.63元 = **1.09元**。这是假设市场对京东方给予"破产折价"定价的极端情景。

**核心判断**：
1. **考虑折旧减少和ITC风险解除后，目标价从2.23元上调至2.59-3.05元区间**，安全边际有所改善
2. 当前价4.17元仍高于概率加权目标价3.05元约**37%**，**投资评级维持"观望"，但下调风险从"高"至"中高"**
3. **关键博弈点**：2026年折旧减少带来的14.5亿净利润增量是否已被市场定价？当前PE 26x隐含的市场预期是什么？
4. **长期风险**：2027年成都B16转固后折旧压力重新加剧，与TCL华星的盈利差距可能进一步扩大，这是当前估值未充分反映的结构性劣势

投资者在当前位置买入，仍需面对"2026年改善vs 2027年恶化"的周期博弈，以及"京东方vs TCL华星"的竞争格局恶化风险。
