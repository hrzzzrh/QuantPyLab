# 研报章节二：产业周期与终端穿透

**研究日期**：2026年05月28日  
**研究对象**：神马电力 (603530.SH)  
**章节继承**：模块一识别的核心引擎为变电站复合外绝缘（占比74%，毛利率48.67%），增长高度绑定全球电网投资。

---

## 一、产业生命周期：复合材料替代的成长期，双重贝塔共振

绝缘子行业可划分为三条技术路径：
- **瓷/玻璃绝缘子**：传统路线，寿命15-40年，重量大，需定期维护
- **复合绝缘子**：高分子硅橡胶+环氧玻璃纤维，轻质、防爆、免维护，全生命周期成本仅为传统的20-40%

**行业处于渗透率驱动成长期**。截至2020年，变电站复合绝缘子渗透率约25%（公司定增预案数据），至2019年已超40%。神马在特高压领域市占率超70%，在全电压等级变电站领域也是无可争议的龙头。行业正从"技术验证期"进入"加速替代期"，驱动力三重：

1. **技术补齐短板**：传统复合绝缘子寿命短（~7年）是最大短板，神马30年+免维护产品经20+年实网验证，打消了电网公司的顾虑
2. **电网复杂度升级**：新能源并网使电网愈加复杂，对防爆、抗震、防污闪提出更高要求，复合材料的"零维护"优势凸显
3. **全生命周期成本觉醒**：全球电网进入投资扩张期，运营商从关注"采购价"转向关注"综合拥有成本（TCO）"

**全球复合绝缘子市场规模**：2025年28.2亿美元，预计2034年51.1亿美元（CAGR 6.83%，Fortune Business Insights）。高压复合绝缘子细分因特高压建设加速增速更快，其中架空输电线路占据68.4%市场份额，超/特高压等级渗透率已突破22%。

## 二、终端需求穿透：电网投资超级周期的三重驱动力

神马电力的产品最终负载于**变电站**（空心/支柱复合绝缘子、复合套管）和**输电线路**（线路复合绝缘子、复合横担）上。终端景气度取决于全球电网建设投资。

### 2.1 中国："十五五"电网投资倍增，特高压密集落地

**投资规模跨越式增长**（来源：国家发改委2026年5月25日发布会、国家电网2026年1月16日公告）：

| 时期 | 两大电网总投资 | 年均 | vs前期 |
| :--- | :--- | :--- | :--- |
| "十三五" (2016-2020) | 2.64万亿元 | 5,280亿 | - |
| "十四五" (2021-2025) | 2.85万亿元 | 5,700亿 | +8% |
| **"十五五" (2026-2030)** | **~5万亿元** | **~1万亿** | **+75%** |

国家电网计划4万亿（+40% vs "十四五"），南方电网~1万亿。这是中国电网史上最大规模的五年投资周期。

**特高压项目进入密集建设期**：
- 截至2025年底，国网累计建成42项特高压工程
- 2025年在建8条线路（4条直流+4条交流）
- 2025年新核准5个项目（蒙西—京津冀、藏东南—粤港澳、攀西特高压等）
- 2026年（"十五五"开局年）：攀西特高压（投资232亿，2028年建成）、浙江特高压交流环网开工；陕北—安徽、甘肃—浙江等线路2026年投运
- 跨区跨省输电能力较"十四五"末提升30%+

**关键推论**：每个特高压变电站/换流站需要大量绝缘子（支柱+空心+套管），而神马在特高压领域市占率超70%，供需关系高度景气。2025年-2027年是特高压设备交付的集中确认期。

### 2.2 海外：全球电网投资共振，史无前例的同步扩张

**全球层面**（来源：IEA World Energy Investment 2026、BloombergNEF 2025）：
- 2026年全球电网投资预计达$5,500亿（+20% YoY），占全球能源投资近60%
- 输电投资CAGR 16% (2024-2027)，为配电网的两倍增速
- IEA警示：全球至少3,000GW可再生能源项目在排队等待并网——**电网已成为能源转型的瓶颈**

**欧洲**（来源：EU European Grids Package）：
- 至2040年电网投资需求1.2万亿欧元（配电7,300亿+输电4,700亿）
- 2026年德国/法国/西班牙等多国电力设备采购排期延长至2028-2030年
- 神马电力通过ABB/Hitachi Energy、Siemens Energy等设备商辐射欧洲——欧美变电站**90%的复合绝缘子由神马供货**

**北美**（来源：ERCOT/PJM filings、招商证券2026年3月研报）：
- 美国765kV线路扩建计划投资~$750亿，765kV线路将从2,000英里扩至10,000英里
- 高压设备全球产能极度稀缺——海外变压器排期最长已到2030年
- 神马美国工厂（可转债募投7.1亿项目）和越南工厂（2025年设立）正在建设中

**新兴市场**：
- 2025年神马中标沙特中南/中西柔直换流站项目（中电装备），为中东首个项目，打破区域市场壁垒
- 拉丁美洲（巴西子公司2025年设立）、东南亚（越南/新加坡子公司）多点布局

### 2.3 需求动能溯源：结构性的内生增长，非短期刺激

增长驱动力的本质：

| 驱动力 | 性质 | 持续性 |
| :--- | :--- | :--- |
| 新能源并网（风光集中/分布式） | **内生** | 长周期（30年+能源转型） |
| 电网设备老化更换 | **内生** | 长周期（欧美设备超龄服役） |
| AI/数据中心用电需求 | **内生** | 新兴但确定性强 |
| 中国"十五五"电网规划 | **半内生** | 5年明确规划 |
| 短期刺激政策 | **不存在** | 本轮增长未发现政府突击补贴痕迹 |

**关键判断**：本轮电网投资超级周期是**结构性的供给侧扩张**，而非需求端刺激。全球电网过去二十年投资几乎停滞（年均$3,000亿），而电源投资（风光）翻了数倍，形成巨大的"源-网"剪刀差。当前电网投资的加速本质上是"还欠账"——弥补过去二十年的投资不足。

## 三、跨期断层审计：需求前置风险分析

### 3.1 是否存在"跨期需求移位"？

**国内视角**：存在中度的跨期集中风险。"十五五"5万亿投资若在前期（2026-2027）集中释放，可能导致2028年后设备需求回落。但缓解因素有三：
1. 特高压项目建设周期3-5年，设备交付分散在不同年份
2. "十五五"规划强调"分层分区优化"，配网升级投资比重提升，平滑了特高压脉冲式采购的波动
3. 2023-2025年已有多条特高压获批+开工，2026-2028年是密集设备交付期，2030年前需求可见度较高

**海外视角**：风险极低。欧美电网升级刚刚启动，IEA预测全球电网投资需从$3,000亿增至$8,000亿（2030年），当前只是开始。欧洲1.2万亿欧元的投资计划覆盖至2040年。

### 3.2 强制复核

**检查一：是否穿透并引用了终端产品未来两年的景气度预测？**
**是。** 全球电网投资2026年$5,500亿（+20%），中国"十五五"电网投资5万亿（年均1万亿），特高压2025-2028年密集建设期。2026-2027年全球电网设备供需仍处"供不应求"状态。

**检查二：是否审计了跨期需求移位风险？**
**是。** 国内存在中度跨期集中风险（集中在前期），海外风险极低。但特高压长建设周期（3-5年）提供了天然缓冲。

## 四、供需边际审计与压力传导

### 4.1 供需平衡：需求增速远大于供给释放，紧平衡持续

**需求端**：复合绝缘子全球需求CAGR 6.83%，高压/特高压领域增速远高于平均。神马2025年变电站复合外绝缘销量178,978支（+39.9%），远超行业增速——这是市占率提升的结果，而非行业总需求增速。

**供给端**：神马当前产能利用率已接近高位（产量187,754支，销量178,978支），库存提升至27,177支（+47.7%），表明公司已在主动备货应对订单增长。越南工厂和美国工厂尚在建设期（预计2027-2028年贡献产能），意味着**2026-2027年产能扩张速度有限**。

全球高压设备制造能力同样面临瓶颈——IEA 2025年报告指出，输电设备供应链约束（变压器、开关、绝缘子）将持续至2028年以后。神马作为极少数同时具备产品创新和产能布局的企业，受益于供需偏紧。

**供需边际判断**：至少在2026-2028年维持紧平衡——需求持续扩张而产能扩张有限。

### 4.2 压力传导特征

**从终端→神马环节的传导路径**：
- **上游原材料**：硅橡胶、环氧树脂对毛利率形成压制。2022年原材料暴涨期毛利率曾跌至24.9%，2023-2025年触底反弹至46.18%。当前原材料价格温和，但地缘冲突可能引发二次上涨
- **下游客户**：以国家电网、Hitachi Energy等巨头为主，议价能力强但注重产品品质（神马的差异化恰好建立了品质壁垒）。订单多为年度框架协议，价格调整存在1-2个季度滞后
- **传导时滞**：订单→交付→收入确认周期约3-6个月。2025年合同收入19.48亿（+27.47%）预示2026年上半年营收增速有较强确定性

**波动放大风险**：低。神马产品属于电网安全核心部件，具有刚性需求属性。电网公司不会因为短期经济波动削减绝缘子采购（涉及系统安全），需求弹性低。

**公司博弈筹码**：
1. 技术垄断：特高压领域无可替代供应商
2. 转换成本：客户已深度绑定——全球95%以上电气设备商已与神马建立合作关系，替换供应商意味着重新走认证流程（耗时12-24个月）
3. 海外布局：美国/越南工厂实现"当地生产-当地交付"，对冲关税及地缘风险

## 五、小结：产业坐标定位

| 维度 | 判断 | 确定性 |
| :--- | :--- | :--- |
| 行业生命周期 | **成长期**（复合绝缘子渗透率提升+全球电网投资扩张） | 高 |
| 终端需求趋势 | **2026-2028年加速上行**（中国5万亿+全球$5,500亿/年） | 极高 |
| 增长动能性质 | **结构性内生**（非政策刺激，非库存牛鞭效应） | 高 |
| 跨期断层风险 | **国内中度/海外极低**（2028年后需关注新增项目节奏） | 中 |
| 供需边际 | **紧平衡**（需求增速>产能扩张） | 高 |
| 压力传导 | **低弹性**（刚性需求属性+技术垄断+转换成本高） | 高 |

**核心结论**：神马电力处于"电网投资超级周期 × 复合材料替代周期"的双重贝塔共振窗口，2026-2028年是业绩弹性最大的三年。全球绝缘子供应链紧张（设备排期至2030年）不仅未威胁公司，反而是其定价权和市场份额提升的催化剂。
