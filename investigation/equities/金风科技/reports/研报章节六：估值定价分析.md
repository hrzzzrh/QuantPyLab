## 二、适配性估值方法选择

### 2.1 PE 估值（权重：40%）

**选择理由**：金风科技盈利波动大但已恢复正向增长，PE 仍是最直观的定价工具。但必须配以利润增速校验，避免高 PE 下的增长陷阱。

**可比公司 PE**（基于 2025 年一致预期）：

| 可比公司 | 2025PE | 2026PE | 备注 |
| :--- | :---: | :---: | :--- |
| 金风科技 | 29.1x | 15.6x | 全球龙头，EPS上调后前向PE显著低于可比 |
| 明阳智能 | 58.0x | 27.5x | 利润基数极低，PE失真 |
| 运达股份 | 56.1x | 21.0x | 国内第二 |
| 三一重能 | 33.9x | 20.9x | 利润下滑中 |
| 维斯塔斯 | ~18x | ~15x | 成熟市场低增长 |
| **可比均值（国内）** | **~49x** | **~23x** | |

**PE 估值区间**（基于 2026E EPS=1.37元，当前股价 21.43元）：
- 悲观(13x)：1.37 * 13 = **17.8 元**（-16.9%，低于 P10 分位）
- 保守(16x)：1.37 * 16 = **21.9 元**（+2.2%，接近 P25 分位 16.9x）
- **中性(20x)：1.37 * 20 = 27.4 元**（+27.9%，历史中位数 20.3x）
- 乐观(25x)：1.37 * 25 = **34.3 元**（+60.0%，接近可比均值）
- 激进(30x)：1.37 * 30 = **41.1 元**（+91.9%，市场情绪极度乐观）

**中性 20x PE 的逻辑（较原 22x 下调 2x）**：利润基数上调后（58 亿 vs 原 50 亿），给予较历史中位数（20.3x）一致的估值倍数。下调 2x 的审慎考量：(1) PE TTM 仍处高位（29x），利润高增（+109% YoY）尚未被市场充分定价；(2) 年末减值"黑箱"仍是压制估值倍数的结构性因素；(3) 铜价上行对毛利率修复斜率构成边际压力。以 20x PE 计算，目标价 27.4 元（+27.9% vs 现价），具备较充足的安全边际。

### 2.2 PB-ROE 模型（权重：30%）

**选择理由**：金风科技为重资产制造业，PB-ROE 框架天然适配。

- 2025A：ROE = 7.08%，PB = 2.32x，隐含永续 ROE 约 10%（r≈10% 时）
- 2026E：ROE ≈ 11%，合理 PB ≈ 1.5-2.0x（考虑资产负债率 71%）
- 2026Q1 末 BVPS = 447.08亿 / 42.24亿 = **10.58 元**（较 2025 年末 10.28 元 +2.9%）
- 2026E BVPS ≈ 10.58 + 1.18 - 0.20(分红) = **11.56 元**
- PB 2.0x → 目标价 23.1 元；PB 2.5x → 28.9 元
- **当前 PB 2.02x**，处于 PB-ROE 框架的中性偏低区间

**PB-ROE 交叉验证：中性目标价 24-25 元**。

## 三、全变量对冲审计矩阵（强制输出）

**基准市值：1,221 亿元（对应股价 28.9 元）**

| 审计维度 | 核心因子 | 审慎思考(估值影响力过滤) | 调节 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业特质** | 连续15年国内第一、4年全球第一的龙头地位 | **溢价。** "第一"的地位在制造业中具有真实的品牌溢价和采购议价权。金风的历史地位已构成进入壁垒，非短期可复制。 | **+5%** | 确定性溢价：龙头企业的利润波动性显著低于行业均值，应享有更高的估值倍数 |
| **内部：企业特质** | 全周期业务架构(制造+服务+开发)+海外高毛利(24.3%)兑现 | **溢价。** 三业务协同在行业低谷期已证明抗风险能力。海外高毛利出口业务已成为增速最快、利润最厚的增长极。 | **+3%** | 降低周期波动性的结构优势，海外盈利质量加持 |
| **内部：治理** | 无实际控制人、管理层持股、A+H 两地上市 | 中性偏正面。无单一股东控制风险，但缺乏实际控制人也可能在战略决策时效率偏低。 | **0%** | 治理结构不是当前阶段的核心定价变量 |
| **宏观：政策对齐** | "十五五"非化石能源十年倍增+海风1亿千瓦目标 | **溢价。** 这是制度性、法律级的确定性，不是"鼓励"而是"必须"。风电行业从此拥有了5年维度的确定性增量保障。 | **+5%** | 制度确定性降低了长期折现率，提升估值中枢 |
| **宏观：地缘风险** | EU FSR调查+英国否决明阳+出口暴增加大风险敞口 | **折价（上调烈度）。** 出口+215%验证了海外增长引擎，但也放大了FSR调查的利害关系——若最终裁决不利，不仅影响欧洲，更可能引发连锁效仿。 | **-5%** | 对远期自由现金流的政治风险折现。出口高增背景下维持-5%折价不变 |
| **市场：流动性与风格** | A+H双重上市、新能源赛道关注度高、国家队资金加持 | 中性偏正面。新能源主题基金配置需求持续，但2026年市场风格在红利/价值/成长间摇摆。 | **0%** | 短期风格因素不改变长期定价，不纳入修正 |
| **市场：利润释放节奏** | 2026年Q1净利9.07亿(+60%)超预期，但Q4减值是历史惯例 | 中性。利润增长趋势确定，但年末"减值黑箱"是压低估值的重要因素。 | **-3%** | 利润可预见性差→应给予盈利质量折价 |
| **综合调节** | | | **Σ=+5%** | |

**修正后目标市值** = 1,221 × (1 + 5%) = **1,282 亿元**

**修正后目标股价** = 1,282 / 42.24 = **30.4 元/股（不复权）**

## 五、逻辑证伪与极限压力测试

### 5.1 "一击毙命"的黑天鹅场景

**定义**：欧洲市场对中国风电企业**全面关闭**（非仅调查，而是实质性市场禁入），叠加国内装机因补贴退坡和电力消纳瓶颈出现断崖式下降。

**假设**：
- 海外业务收入从 180 亿萎缩至 50 亿（仅保留中东/非洲/南美非敏感市场）
- 国内风机销量降至 20GW/年（仅维持存量替换）
- 风电场因电价下跌需计提大规模减值（减值 50 亿/年）
- 净利润降至亏损边缘（约 5-10 亿/年）

**地狱价格**（基于清算价值）：
- 净资产（2025年末 BVPS）≈ 10.28 元
- 扣除商誉和无形资产后的有形净资产 ≈ 9.5 元
- **极端支撑位：8-10 元（后复权约 67-84 元）**
- 相当于 PB 0.8-1.0x，已接近 2024-2025 年低点（8.75-9.27 元）

### 5.2 逻辑证伪触发条件

| 证伪信号 | 触发阈值 | 当前状态 |
| :--- | :--- | :--- |
| 国内新增装机连续两季下滑超 20% | 季度装机 <25GW | Q1 15.8GW(+7.9%)，暂未触发 |
| 风机毛利率连续两季低于 8% | 毛利率 <8% | Q1 已超 10%，修复趋势中 |
| 在手订单连续两季下滑超 10% | 在手 <48GW | 53.9GW，创历史新高 |
| EU FSR调查做出"限制市场准入"结论 | 2027秋季前 | 调查进行中，结果未知 |
| 金风丢失去国内海风/出口第一名 | 份额<25%/出口排名跌出前二 | 市占率37.9%，出口断层第一 |

## 六、最终定价结论

**基准目标价**：**27.4 元（不复权）**（对应 20x 2026E PE，EPS 1.37 元）
**修正目标价**：**30.4 元（不复权）**（全变量审计矩阵修正+5%）
**保守情景价**：**7.0 元（不复权）**（熊市 PE 收缩 + 利润不达预期）

**当前价格 21.43 元，距修正目标价 +41.8% 上行空间。**

**投资评级**：本轮审计的核心发现是**"海外出口暴增"对盈利预测和估值锚点的双重上修效应**。Q1 六家整机商海外新增订单 +215%、金风海外毛利率 24.29% 的数据，将海外业务从"远期期权"升级为"已验证的近期增长引擎"。叠加股价 12.6% 的回调，前向 PE 从 20.7x 压缩至 15.6x——已是近 5 年 P10-P25 区间。当前价格的安全边际较 5 月 26 日初版研报已有根本性改善（盈亏比 0.085→0.62）。风险在于：(1) H1 业绩能否验证毛利率持续修复；(2) Q4 减值规模是否可控；(3) 铜价持续上行对成本端的边际侵蚀；(4) EU FSR 调查在出口高增背景下的潜在冲击。
