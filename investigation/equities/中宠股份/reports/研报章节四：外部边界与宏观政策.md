# 研报章节四：外部边界与宏观政策

**研究日期：2026年05月27日**

---

## 一、宏观环境变量

### 1.1 汇率损益：双刃剑效应

中宠股份境外收入占比约65%（2025H1数据），结算币种以美元为主。2024年下半年以来，人民币对美元经历了显著贬值周期（在岸人民币从~7.1贬至~7.35），对公司的利润表产生双重影响：

**正面效应**：
- 公司境外收入以美元计价，人民币贬值直接增厚折算后的营收和利润
- 海外工厂的本地成本（人工、原料）以当地货币计价，不受人民币汇率影响
- 2025年，人民币对美元年均贬值约2-3%，估算对境外收入端产生了约1-1.5亿的正面贡献

**负面效应**：
- 2026年以来中美贸易摩擦加剧，人民币汇率波动加剧（波动率从5%升至10%+）
- 汇率双向波动的不可预测性增加，对公司财务费用（汇兑损益）管理构成挑战
- 若人民币大幅升值（如中美贸易谈判取得突破），境外收入将受到负面影响

**综合评估**：汇率是中宠股份面临的持续性变量，但不构成系统性风险。公司通过海外工厂的本土收入（以当地货币结算）形成了一定程度的自然对冲，降低了单一汇率对的敞口。

### 1.2 利率与折现率

当前中国处于降息周期（1年期LPR 3.10%，5年期LPR 3.60%，2026年5月），低利率环境有利于：
- 降低公司有息负债的融资成本
- 提升未来现金流的DCF折现值
- 但公司短期借款高达14.82亿（2026Q1），若利率回升将直接影响财务费用

公司2022年发行的7.69亿可转债（票面利率仅0.3-2.0%），在当前利率环境下融资成本极低，属于优质负债。

### 1.3 通胀传导

公司核心原材料鸡肉价格受白羽鸡养殖周期影响较大。2026年以来，受中美关税战影响，白羽鸡价格较前期反弹了约40%，每公斤约3.62元。豆粕（饲料主要原料）同样受进口大豆关税影响存在涨价压力。

**传导效率分析**：
- 公司毛利率从2023年的26.28%升至2025年的29.54%，说明过去两年原材料价格下行已充分体现在利润中
- 若2026年鸡肉和豆粕价格持续上行，毛利率可能面临1-3个百分点的下行压力
- 但公司的战略储备制度（提前囤货）可平滑短期波动，影响存在1-2个季度的滞后

## 二、地缘政治风险：产业链重构与关税冲击

### 2.1 中美贸易摩擦：核心外部变量

2025-2026年中美关税战升级至历史最高水平：
- 美国对华加征关税叠加至**145%**
- 中国对美反制关税叠加至**125%**

这对宠物食品行业的影响是**结构性的而非周期性的**：

**（1）出口端影响：出口美国通道基本关闭（对无海外产能的企业）**

中国产宠物食品经由海运出口至美国，若叠加145%关税，终端价格将翻倍以上，完全丧失竞争力。这对纯国内产能的出口企业（如部分中小代工厂）是毁灭性打击。

**但中宠股份受到的实际冲击有限**，原因在于：
- 美国工厂（2015年投产，最大宠物零食工厂之一）：销往美国市场的产品可**直接从美国工厂出货**
- 加拿大工厂（2018年投产）：产品通过USMCA（美墨加协定）进入美国市场享受优惠关税
- 墨西哥工厂（2025年建成）：同样可通过USMCA对美出口
- 柬埔寨工厂：东南亚出口不受中美关税影响

**关税战对中宠的主要影响路径不再是"出口受阻"，而是"全球订单重新分配"**——美国客户将更多订单转向中宠的北美工厂，这实际上可能**增强中宠在北美市场的竞争地位**（因为竞争对手的中国产能被关税排除在外）。

**（2）进口端影响：原材料国产替代加速**

中国对美国鸡肉粉（宠物食品核心原料）进口依赖度极高——年消耗60-65万吨鸡肉粉中，进口超20万吨，其中98%来自美国。125%的对美关税将迫使行业加速寻找国内替代或转向新西兰等替代来源。

中宠股份的应对优势：
- 全球采购体系：美国工厂可就地采购美国产鸡肉粉（不受中国关税影响）
- 中国工厂：与国内白羽鸡屠宰企业建立长期供应关系，逐步替代进口鸡肉粉
- 短期阵痛：国产鸡肉粉在蛋白质含量和品质上可能不及美国产品，可能影响部分高端产品品质

**（3）行业格局重塑：中宠的"关税免疫"优势**

当前中美关税战的极端环境，实质上加速了宠物食品行业的**产能替代**进程：
- 无海外产能的企业 → 丢失美国市场份额 → 被迫降价在国内/东南亚/欧洲市场抢单 → 行业价格战加剧
- 有海外产能的企业（中宠、部分乖宝泰国厂、佩蒂越柬厂）→ 受益于竞争减少 → 获得更多美国订单

中宠股份凭借6国23工厂的全球网络，是中国宠物食品行业中**最具备"关税免疫"能力的企业**。这一优势在当前环境下具有极高的战略价值，且竞争对手在3-5年内难以复制。

### 2.2 供应链重构（China+1）

全球宠物食品采购正在经历"China+1"转移：
- 国际品牌商（如玛氏、雀巢）要求供应商在东南亚/美洲建立备份产能
- 泰国、越南、柬埔寨成为主要的产能转移承接地

中宠的柬埔寨工厂（2019年投产，东南亚最大宠物零食工厂之一）和第二工厂（在建）恰好处于这一转移趋势的中心。公司在东南亚的布局时机具有显著先发优势。

### 2.3 欧盟准入：潜在的增量空间

中国宠物食品进入欧盟市场需要企业通过海关推荐并获得欧盟官方注册。2025年，仅4家山东宠物食品企业申请欧盟准入，其中3家获批。虽然中宠股份尚未在公开信息中确认其欧盟准入状态，但山东作为中国宠粮出口第一大省（占全国45%+），中宠作为行业龙头企业，获得欧盟准入的可能性较高。一旦打开欧盟市场，将为公司海外业务贡献可观的增量。

## 三、监管与产业政策

### 3.1 行业规范化：利好头部，加速出清

2025-2026年，宠物食品行业迎来密集的政策规范：
- 2025年农业农村部公告第20号：一次性发布多项宠物饲料专项规范，**监管进入系统化阶段**
- 2026年5月1日起：实施《全价宠物食品处方粮通用要求》（NY/T4838）和《冻干宠物食品通用要求》（NY/T4839）
- 《宠物饲料标签规定》：2020年3月1日起执行，标签合规要求趋严

**政策影响分析**：
- 合规成本上升 → 中小品牌加速退出 → 行业集中度提升 → **利好头部企业**
- 中宠股份作为全国饲料工业标准化技术委员会宠物饲料分技术委员会副主任委员单位，**直接参与标准制定**，享有规则制定的话语权
- 公司已通过20项全球权威认证（ISO、BRCGS、HACCP、SMETA、BSCI等），政策趋严几乎不产生新增合规成本

### 3.2 产业政策契合度

宠物食品行业与国家战略方向高度契合：
- **内需消费**：宠物消费是内需消费的重要组成部分，符合"扩大内需"战略
- **乡村振兴**：宠物食品产业可带动禽类养殖、农产品深加工等上游农业价值链
- **品牌出海**："中国质造"宠物食品走向全球，符合"一带一路"和"双循环"战略

不存在显著的产业政策压制风险。

### 3.3 环保与动物福利

- 公司已通过ISO14001环境管理体系认证
- 宠物营养与健康研究院拥有300余只犬猫的实验基地，符合动物福利标准
- 暂无重大环保处罚或负面记录

## 四、核心结论

中宠股份面临的外部宏观环境呈现**"利空出尽、利好上行"**的结构：

**利空因素（已在价格中部分定价）**：
- 中美关税战已升级至极端水平（145%/125%），进一步恶化的空间有限
- 原材料价格周期性波动是行业常态，2026年的涨价压力可控
- 汇率的双向波动可通过全球产能自然对冲

**利好因素（市场可能低估）**：
- 关税战下的"产能替代"效应：中宠的北美工厂将从竞争对手手中抢得更多订单
- 监管趋严加速行业出清：中小品牌退出释放的市场份额将被头部企业瓜分
- 欧盟市场准入：一旦获批，将打开新的增长空间

**最大的外部风险（需持续监控）**：
- 关税战如何"结束"：若中美突然达成协议、关税大幅下调，中宠的"关税免疫"溢价将消失，但从竞争角度看这不改变公司的长期优势
- 全球供应链突发中断（如新的出口管制或制裁）
