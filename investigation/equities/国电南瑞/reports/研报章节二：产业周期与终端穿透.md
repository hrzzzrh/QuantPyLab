# 研报章节二：产业周期与终端穿透

**研究日期**：2026年6月14日
**标的**：国电南瑞 (600406.SH)
**前序模块红线继承**：模块一发现毛利率持续下行（26.88%→25.84%）、利润增速<营收增速（剪刀差5.7pp）、应收账款高企（312.5亿）三大隐忧，本模块将穿透终端需求验证这些隐忧的演变方向。

## 一、产业生命周期与周期定位

### 1.1 周期坐标：电网投资的"超级周期"确认

中国电网设备行业正处于**继2009-2012年（坚强智能电网）、2014-2018年（特高压第一轮）之后的第三轮大规模投资上行期**。与前两轮不同，本轮的核心驱动力并非单纯的电网扩容，而是新能源消纳需求倒逼的"源网协同"——发电侧风光装机大爆发后，输电和配电侧的瓶颈效应日益突出。

2026年1月15日，国家电网正式宣布"十五五"期间（2026-2030年）固定资产投资预计达到**4万亿元**，较"十四五"2.85万亿元增长**40%**，年均规模8,000亿元。叠加南方电网预计1万亿元，全国电网"十五五"总投资将**逼近5万亿元**（年均超1万亿元），较"十三五"2.64万亿和"十四五"2.85万亿实现量级跃升。

这一投资规模的确认，标志着电网设备行业正式进入**5年的确定性高景气周期**。

**关键边际变化**：
- 2026年Q1验证：国家电网完成固定资产投资超1,290亿元，同比+37%，配电网投资568亿元占各级电网投资比重55%，均创历史同期新高。南方电网同比+49.5%。
- 2026年国网全年预算：7,200亿元（过审版）/7,800亿元（争取版），同比+20%以上，为近年最大增幅。
- "六张网"顶层设计：2026年4月政治局会议和5月国务院常务会议明确将"新型电网"列为"六张网"之一，赋予电网投资超越传统基建的政治优先级。

**结论**：产业处于**成长期的后半程——渗透率提升+结构性升级双轮驱动**，并非简单的成熟期存量博弈。本轮上行期的特殊之处在于：电网投资增速远高于全社会用电量增速（~5%），差额完全由"新能源消纳"和"智能化改造"的投资密度上升所解释。

### 1.2 库存动力学：从"主动去库"到"被动补库"的拐点

电网设备行业的库存周期与一般制造业不同，其核心"库存"并非产成品，而是**招标-交付的时间差**。2025年是行业库存周期的一个微妙转折点：

- **2024年Q4-2025年H1：主动去库末期**。配网和电表领域因集采新标准切换（新国标电表）及价格战，供应商主动压低库存；国网2025年计量设备招标额同比-49%，配网区域联合招标中价格权重过高导致低价竞争。
- **2025年H2-2026年H1：被动补库开启**。2025年12月《关于促进电网高质量发展的指导意见》发布后，2026年Q1招标大幅放量，供应商进入交付高峰。国网明确反内卷机制——2026年起价格回升5%-30%，评分体系强化技术/安全性权重，"壳公司/代工型"供应商出清。

**牛鞭效应辨析**：存在一定程度的订单前置——2025年国网输变电设备招标额918.8亿元（+25.3%），2026年Q1新能源接网工程投资+50%，部分反映了"十五五"首年抢装效应。但差异在于：本轮订单波动的底层是**结构性物理需求**（风光装机历史高位→消纳需求真实存在），而非渠道投机。新能源大基地"发得出、送不出"的消纳瓶颈已在2025年集中暴露（如甘肃、新疆弃风弃光率回升），电网投资具有刚性补课属性。

## 二、终端需求穿透与跨期断层审计

### 2.1 最终产品画像

国电南瑞的产品/服务最终负载于以下终端形态：

| 最终产品/场景 | 对应公司业务板块 | 占营收比（估） |
|:---|:---|:---:|
| 国家级/区域级电网调度控制系统 | 电网智能 | ~20% |
| 变电站/配电站自动化与继电保护 | 电网智能 | ~30% |
| 特高压输电（换流阀/控保） | 能源低碳 | ~8% |
| 新能源场站（风/光/储）并网控制 | 能源低碳 | ~10% |
| 储能系统（构网型/电化学） | 能源低碳 | ~5% |
| 电力信息通信/网络安全/数字化 | 数能融合 | ~18% |
| 轨道交通/工业DCS/PLC | 工业互联 | ~5% |
| 充电桩/电动汽车充换电 | 能源低碳 | ~2% |
| 海外电网（亚非拉/欧洲） | 全板块 | ~9% |

**核心终端 → 电网投资（约80%收入间接或直接与两大电网资本开支挂钩）**。因此，对终端景气度的审计本质上就是对国网+南网"十五五"投资执行度的审计。

### 2.2 终端景气度审计：2026-2028年电网投资确定性

#### 结论：极高确定性（Evidence Score: 5/5）

**五重证据锚定**：

1. **政策顶层设计已就位**。2025年12月31日发布的《关于促进电网高质量发展的指导意见》明确到2030年目标：西电东送规模>4.2亿千瓦（2025年末3.4亿）、跨区输电能力+35%、接纳分布式新能源9亿千瓦、支撑充电设施4000万台。该文件是"十五五"电网投资的纲领性文件，相当于"开工令"。

2. **国网4万亿元为公开承诺（非预测）**。与2021-2022年市场对电网投资的"猜测"阶段不同，2026年初4万亿已由国网以新闻稿形式正式宣告，权威层级极高。2026年一季度已交付37%的增长，执行确定性得到初步验证。

3. **物理瓶颈倒逼投融资**。截至2026年2月，全国风光累计装机已超18.8亿千瓦，2025年"沙戈荒"大基地集中并网后，特高压外送通道成为最大瓶颈——跨区输电能力仅3.4亿千瓦 vs 潜在送出需求>5亿千瓦，输配电价空间已打开。这不是"要不要投"的问题，而是"不投就弃风弃光"的硬约束。

4. **"六张网"赋予电网超越传统基建的战略优先级**。2026年4月政治局会议首次将"新型电网"列入六张网战略，与算力网、5G网并列，意味着电网投资在宏观经济下行期将获得财政和货币政策的双重护航。

5. **特高压密集开工是先行指标**。2026年Q1开工攀西特高压交流、皖鄂直流背靠背等工程，续建甘肃-浙江、大同-怀来-天津南、闽赣直流等，项目储备极其饱满。国网计划"十五五"投运15项特高压直流工程。

### 2.3 动能溯源与透支审计

**内生vs外生动能判别**：

本轮增长的终极驱动力是**内生性的**——新能源发电成本下沉（光伏LCOE已降至0.15-0.25元/度）使风光成为最便宜电源，而与之配套的电网基础设施建设严重滞后。它不是"刺激出来的需求"，而是"被压抑多年的刚性需求"的集中释放。

**跨期需求前置审计（强制复核）**：

- **是否存在需求前置？** 是，部分存在。2026年增速37%（Q1）并非稳态增速，"十五五"首年存在追赶效应。但需要区分：首年放量并非透支，而是对过去三年"规划期"审批较慢的补偿性加速。
- **是否存在"悬崖效应"？** 不存在传统意义上的需求悬崖。电网投资的特殊性在于：项目周期长（特高压3-5年）、资金连续拨付（年度预算制）、在建规模惯性大（累计在建抽蓄超9,400万千瓦）。即便2027-2028年增速回落至10-15%，绝对投资规模仍将维持万亿级高位。
- **政策补贴透支审计**：电网设备行业不依赖财政补贴（与光伏制造端的补贴逻辑不同），不存在补贴退坡引发的需求断崖。反内卷机制带来的价格修复是结构性的，不会导致需求萎缩。

**结论**：本轮增长动能以内生刚性需求为主，跨期透支风险较低。唯一需警惕的是2028年后特高压集中投运完毕后的"投资真空期"，但考虑到配电网和数字化的投资周期更长（需要持续10年+的改造），这一风险在2026-2027年的投资视野内不构成实质威胁。

## 三、供需边际与压力传导

### 3.1 供需平衡审计

**供给侧：集中度提升，竞争态势改善**

- 国网正式推行"反内卷"机制：2026年起价格权重下调，技术/安全/商务多维评估，供应商数量控制，"壳公司/代工型"企业出清。
- 特高压换流阀领域：国电南瑞、许继电气、中国西电三足鼎立（CR3>85%），新进入者几乎没有。
- 电网调度/自动化：国电南瑞近乎垄断（市占率>60%），南瑞继保（子公司）补充。
- 储能（构网型）：技术壁垒高，当前有能力规模交付的企业<5家，南瑞位列第一梯队。
- 配网设备：相对分散，但反内卷将加速集中度提升，南瑞在一二次融合、配电自动化等高端领域优势显著。
- IGBT自主替代：南瑞4500V/3000A IGBT已实现小批量，中压IGBT年交付10万只，是重要的供给侧"卡脖子"解除信号。

**需求侧：增速斜率超前于供给侧产能扩张**

- 年均电网投资从"十四五"的~5,700亿跃升至"十五五"的~10,000亿，增幅75%。
- 在反内卷、去低端产能的背景下，有效供给（能拿到订单的合格产能）反而在收缩。
- 结论：**供需边际正由弱平衡向供不应求偏移**，优质供应商的议价权有望阶段性提升。

### 3.2 压力传导特征分析

**终端波动向公司传导的路径与弹性**：

1. **正向传导（电网投资↑→南瑞营收↑）**：
   - 传导系数高。国网投资每增加100元，电网设备采购约增加40-50元，南瑞凭借综合龙头地位可获得约8-10元份额（市占率估测）。
   - 传导时滞：招标到确认收入约6-12个月（特高压更长，约12-18个月）。
   - 2025年末在手订单520亿≈0.79x年营收，2026年Q1新签合同预期进一步增长，意味着2026-2027年营收增速的可见度极高。

2. **负向传导（光伏装机下滑→储能/新能源控制需求？）**：
   - 2026年光伏新增装机预计下滑至180-240GW（2025年315GW），表面上对南瑞新能源控制业务构成负面压力。
   - **但这是伪命题**：电网设备需求的驱动变量并非当年新增装机，而是**累计装机规模**（消纳压力与存量成正比）和**电网投资本身**。存量18.8亿千瓦风光装机的消纳改造需求和新增新能源的并网需求是叠加而非替代关系。
   - 换句话说：装机增速放缓（今年325GW→明年250GW）并不等于消纳改造需求放缓，后者是由存量的"积压效应"决定的。

3. **原材料成本传导**：
   - 直接材料占成本71%（2025年），铜、钢、IGBT芯片等价格波动对毛利率的边际影响显著。
   - 传导滞后：集采合同中价格锁定期通常3-6个月，原材料短期波动无法及时转嫁。
   - 2026年铜价受地缘因素扰动上行，对毛利率构成向下拉力；但反内卷带来的售价回升形成向上推力，两者博弈结果需在模块五量化测算。

4. **海外传导（高增长+低毛利率的拉扯）**：
   - 海外业务增速84%但毛利率仅19.2%，与国内26.5%的差距约7个百分点。
   - 当前海外收入占比9.1%，对综合毛利率的拖累约0.6个百分点。
   - 若海外占比提升至15%（可能发生于2027-2028），拖累将放大至1个百分点。
   - 好消息是：海外毛利率目前处于"市场开拓期"的历史低位，随着规模效应和本地化服务能力提升，中期有望收敛至22-24%。

## 四、强制复核项确认

在进入模块三（竞争格局）之前，以下三项强制复核已完成：

1. **终端产品景气度预测已引用**：已穿透至电网投资（4万亿/5年）和新能源装机（年均200GW+）两大终端指标，2026-2028年景气度极高且确定性高。

2. **跨期需求移位已审计**：不存在补贴断崖风险；2026年增速部分含首年追赶效应，但后续绝对规模维持高位，不是"透支-断崖"模式。光伏装机2026年小幅下降不会对电网设备需求构成实质冲击。

3. **供需边际与压力传导已分析**：供给侧受反内卷出清，有效供给收缩；需求侧年均万亿，供不应求格局初现。毛利率的正向（价格修复）与负向（原材料、海外稀释）力量并存，需在模块五定量分解。
