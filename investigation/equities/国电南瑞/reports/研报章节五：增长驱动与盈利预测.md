# 研报章节五：增长驱动与盈利预测

**研究日期**：2026年6月14日
**标的**：国电南瑞 (600406.SH)
**目标年份**：2026年（当年锚定，Q2适用）
**前序模块红线继承**：
- 模块一：毛利率下行→需在本模块验证修复拐点；AR高企→需量化对现金流的影响
- 模块二：电网投资5万亿超级周期→营收增长的底线确定性极高
- 模块三：反内卷价格修复→毛利率向上驱动力；多领域技术壁垒→盈利可持续性
- 模块四：外部环境"五利一空"格局→收入侧极强、成本侧温和

## 一、分业务 P×Q 模型与证据锚定

### 1.1 销量（Q）确定性溯源

#### 核心驱动力一：国网2026年Q1固定资产投资同比+37%（证据强度：5/5）

国网一季度完成投资超1,290亿元，配网投资568亿元占比55%，新能源接网工程投资+50%。这一增速是南瑞营收的直接领先指标（滞后1-3个季度）。

**传导逻辑**：国网投资↑→设备招标↑（2025年输变电招标918.8亿，+25.3%）→南瑞中标→收入确认（6-12个月后）。2026Q1的37%投资增速，对应2026H2-2027H1的营收加速确认。

**量化锚定**：
- 国网2026年全年投资预算：7,200亿元（过审版，+11%）/ 7,800亿元（争取版，+20%），取中值7,500亿元（+15%）。
- 南瑞在国网设备采购中的份额约8-10%（基于网内营收420亿/国网设备总投资约4,500亿）。
- 网内收入增速 ≈ 国网投资增速 × 传导系数（0.9-1.1，因产品结构偏高景气领域而>1）。

#### 核心驱动力二：在手订单520亿提供极高可见度（证据强度：5/5）

2025年末在手订单520.31亿元，其中本年新签331.92亿元。520亿在手订单相当于2025年营收的78.6%，意味着2026年Q1-Q3的收入有极强的订单保障。全年新签合同758.68亿（+14.4%），订单/收入比（Book-to-Bill）= 1.15，处于扩张区间。

**量化锚定**：
- 520亿在手订单中约70%将在2026年内转化为收入 → ~365亿确定性营收
- 剩余~385亿来自2026年新签（预计全年新签850-900亿）→ 6-12个月转化

#### 核心驱动力三：海外业务"从1到10"的爆发（证据强度：4/5）

2025年海外营收60.38亿（+84.1%），2026年有望继续高速增长。公司在沙特（储能+STATCOM）、巴基斯坦（默拉直流）、巴西（控保）、智利（保护及自动化）等市场持续突破，坦桑尼亚、马达加斯加等5国为2025年新开拓市场，贡献将主要在2026-2027年体现。

**量化锚定**：
- 全球电网投资预计2030年达6,000亿美元（IEA），全球变压器/开关设备缺口30%，交付周期拉长至36-60个月→中国设备商处于"卖方市场"
- 南瑞海外增速预测：2026E +50%（~90亿），2027E +35%（~122亿），2028E +30%（~158亿）
- 证据支撑：公司2025H2新增海外认证50+产品、新开拓5国、BNEF Tier1储能认证

#### 核心驱动力四：新能源+储能+特高压三箭齐发（证据强度：4/5）

- 特高压：国网2026年Q1开工攀西特高压交流、皖鄂直流背靠背，续建甘肃-浙江、大同-怀来-天津南、闽赣直流等，"十五五"投运15条直流
- 储能：构网型储能渗透率从12.8%跃升至19.2%（2026Q1），南瑞继保Q1储能中标1,300MWh行业前三
- 新能源：风光年新增超3亿kW（2026E），虽较2025年4.3亿kW下降，但累计装机18.8亿kW的消纳改造需求持续释放

### 1.2 价格（P）趋势逻辑推演

**正面驱动力**：
1. **国网反内卷**：招标价格权重下降、技术权重提升 → 预估2026年综合售价回升3-8%（电网智能/数能融合受益最显著）
2. **构网型储能技术溢价**：南瑞继保构网型PCS成本较跟网型高约1倍（"三倍10秒"过载能力），整体系统造价提升10%，享有明确的品牌和技术溢价
3. **海外高毛利订单**：海外项目报价灵活性大，且国内过剩的激烈比价在海外不适用

**负面压力**：
1. 原材料：铜价中高位、IGBT芯片涨价10-20%
2. 能源低碳板块毛利率结构性偏低（20.2%），随着该板块占比扩大（25%→30%），综合毛利中枢提升难度增大

**净预判**：毛利率在2025年25.84%触底，2026年回升至26.5%左右，2027-2028年逐步回升至27-27.5%。

## 四、分部估值（SOTP）利润模拟

国电南瑞具备多个不同属性的业务板块，适用分部估值。根据框架算法规范：

`板块模拟净利_i = 板块营收_i × [板块毛利率_i - (总期间费用率 × K_i)] × (1 - 有效税率)`

### 2026E SOTP 模拟

| 板块 | 营收 | 毛利率 | K系数 | 费用率11%×K | 模拟净利 | 说明 |
|:---|---:|---:|---:|---:|---:|:---|
| 电网智能 | 384亿 | 30.5% | 1.0 | 11.0% | 67.4亿 | 成熟业务，标准K=1.0 |
| 能源低碳 | 218亿 | 21.0% | 1.3 | 14.3% | 13.1亿 | 高研发/出海投入，K=1.3 |
| 数能融合 | 133亿 | 23.0% | 1.2 | 13.2% | 11.7亿 | 研发驱动数字业务，K=1.2 |
| 工业互联 | 33亿 | 22.0% | 1.4 | 15.4% | 2.0亿 | 孵化期业务，K=1.4 |
| 集成及其他 | 4亿 | 26.0% | 0.6 | 6.6% | 0.7亿 | 轻资产，K=0.6 |
| **合计** | **772亿** | | | | **90.1亿** | |
| 加：财务收入 | | | | | 5.9亿 | 利息/汇兑净收益 |
| **模拟净利** | | | | | **96.0亿** | |

**勾稽校验**：∑(板块模拟净利) = 90.1亿 + 5.9亿 = 96.0亿 ≈ 整体预测96.0亿 ✓

SOTP分析表明，**电网智能**贡献了约67%的利润，是公司盈利的"压舱石"；**能源低碳**尽管营收增速最快（+30%），但利润贡献仅13%，印证了模块一"增收不增利"的结构性隐忧——该板块的利润弹性将在规模跨越300亿营收（预计2027-2028年）之后才能显著释放。

## 五、证据链强度与确定性评分

| 增长驱动力 | 评分 | 证据支撑 |
|:---|:---:|:---|
| 国网投资高增长（2026E+15%） | **5/5** | Q1已实现+37%，国网年度目标7,200-7,800亿公开文件，国务院"六张网" |
| 在手订单520亿转化 | **5/5** | 2025年报披露合同负债78亿（+21%），在手订单为具体金额 |
| 特高压密集开工→换流阀放量 | **4/5** | 甘肃-浙江、攀西等已开工采购，南瑞中标50%+份额（白鹤滩、甘肃-浙江实际数据） |
| 海外50%+增长 | **3/5** | 2025年+84%已兑现，但海外增速基数抬高后能否维持50%存不确定性，受地缘影响 |
| 毛利率回升至26.5%+ | **4/5** | 国网反内卷机制明确（价格回升5-30%），但原材料涨价形成对冲，净效果需逐季验证 |
| 费用率下降（规模效应） | **3/5** | 趋势确定但程度温和，研发费用率大概率维持5-5.5%不下行 |
| 构网型储能爆发 | **4/5** | 渗透率19.2%（Q1），南瑞已入选BNEF Tier1，但行业竞争窗口在收窄 |
| IGBT自主替代加速 | **2/5** | 晶圆制造已终止，设计+封测模式需要验证长期良率与成本竞争力 |

**综合确定性评分：3.75/5（中高确定性）**

核心逻辑（国网投资增长→订单→收入）的确定性极高（4-5分），但利润率的弹性（毛利率回升幅度、费用率下降速度）存在一定不确定性（3分），综合给出中高确定性评级。
