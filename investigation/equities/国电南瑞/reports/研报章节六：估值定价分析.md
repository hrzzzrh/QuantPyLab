## 一、价值锚定：适配性方法选择

### 1.1 方法论选择理由

国电南瑞的商业模式特征是：**高ROE（16%）+ 稳定增长（10-20%）+ 强现金流 + 低资本开支**。适用以下估值方法：

| 方法 | 适用性 | 权重 | 选择理由 |
|:---|:---:|:---:|:---|
| **PE估值**（主力） | ★★★★★ | 50% | 盈利稳定、扣非占比>96%，PE是可比的行业通用语言 |
| **PEG修正** | ★★★★ | 15% | 增长差异大，PEG可修正单纯PE的静态缺陷 |
| **PB-ROE** | ★★★★ | 20% | 净资产定价有锚，ROE>15%应享有PB溢价 |
| **DCF** | ★★★ | 15% | 现金流优秀（OCF/NP=1.54x），但电网投资周期性强，远期假设敏感 |

**不适用方法**：PS（净利率稳定，PS冗余）、EV/EBITDA（无显著带息负债，杠杆中性）

### 1.2 历史分位审计

基于近10年（2016-2026年）交易数据：

| 指标 | P5 | P10 | P25 | **P50** | P75 | P90 | P95 | **当前(6/14)** | **前值(5/26)** | 当前分位 |
|:---|---:|---:|---:|---:|---:|---:|---:|---:|---:|:---:|
| PE TTM | 13.27 | 19.91 | 22.46 | **25.35** | 28.36 | 32.84 | 35.79 | **~22.5** | 24.72 | ~P26 |
| PB | 3.05 | 3.17 | 3.64 | **4.16** | 4.64 | 5.16 | 6.05 | **~3.55** | 3.89 | ~P23 |

**近年PE中位趋势**：2021: 30.9x → 2022: 31.3x → 2023: 28.0x → 2024: 26.9x → 2025: 23.5x。PE中枢逐年下移反映了市场对电网投资增速放缓的持续担忧——但2026年"十五五"4万亿的官宣完全打破了这一叙事。

**6月14日更新**：股价进一步下挫（25.48→23.24）使PE TTM从P44降至P26（低于历史中位一个标准差），PB从P35降至P23。即便考虑到铜价超预期上涨对毛利率的压制，当前估值已price-in相当程度的风险。**核心论点不变**：当前南瑞面临的环境已发生质变——1）"十五五"电网投资从2.85万亿跃升至5万亿（+75%），增长率而非存量规模决定了PE；2）反内卷→利润率修复，盈利质量改善；3）海外+储能打开全新市场空间。**合理PE应略高于历史中位，但需保守取值（不高于P75的28.36x）。当前22.5x处于历史底部分位，提供了显著的安全边际。**

## 二、多方法交叉估值

### 2.1 PE估值（权重50%）

```
目标市值 = 2026E归母净利润 × 合理PE
```

**合理PE确定**：
- 近5年PE中位：27x（各年波动中枢的均值）
- 近10年PE中位：25.4x
- 行业可比（电力二次设备龙头+15%增速）：25-30x
- 综合取：**26x**（处于近10年P50-P60之间，保守但不过度保守）

**2026E PE估值**：96.0亿 × 26 = **2,496亿** → 目标价 **31.08元**（+22.0%）

**2027E前瞻PE估值**（Q4滚算时适用）：118.0亿 × 24 = 2,832亿 → 35.26元

### 2.2 PEG修正（权重15%）

```
2026E增速 = 16.0%（归母净利）
合理PEG = 1.0（稳定增长行业中性值，考虑南瑞确定性溢价可给到1.1-1.2）
合理PE = PEG × G = 1.0~1.2 × 21.4 = 21.4~25.7x
```

**PEG估值**：取22-24x → 100.5亿 × 23 = **2,312亿** → 目标价 **28.78元**（+12.9%）

PEG估值偏低的原因：南瑞2026E的21.4%增速部分受益于低基数（2025年仅+8.8%），持续维持20%+增速的置信度不足。因此PEG法给出的估值偏保守，作为安全边际参考。

### 2.3 PB-ROE模型（权重20%）

```
合理PB = (ROE - g) / (COE - g)
       = (16.5% - 5%) / (9% - 5%)
       = 11.5% / 4%
       = 2.875x

但在A股，ROE>15%的优质龙头通常享受PB溢价。历史PB中位4.16x。
取3.5-4.0x（低于历史中位，保守起见）。
```

2025年末BVPS = 6.55元。2026年末BVPS ≈ 6.55 + 1.25 - 0.62（分红）= 7.18元。

**PB-ROE估值**：7.18元 × 3.8 = 27.28元/股 → 市值 **2,191亿**

### 2.4 DCF估值（权重15%）

核心假设：
- 自由现金流 = OCF - CAPEX。2025年OCF 127.7亿，CAPEX约8亿（购建固定资产），FCF ≈ 120亿
- 2026-2028年FCF增速：20%/18%/15%（与利润增速匹配）
- 2029-2030年FCF增速：12%/10%
- 永续增长率g=3%（通胀+电网自然增长）
- WACC=8.5%（无风险利率1.7%+股权风险溢价6.8%，β=1.0）

经DCF模型计算（简化版）：

| 年份 | FCF（亿） | PV（亿，折现率8.5%） |
|:---|---:|---:|
| 2026E | 144 | 132.7 |
| 2027E | 170 | 144.4 |
| 2028E | 196 | 153.5 |
| 2029E | 219 | 158.0 |
| 2030E | 241 | 160.2 |
| 终值 | 4,517 | 3,001.5 |
| **企业价值** | | **3,750.3** |
| 加：现金 | | +73 |
| 减：带息债务 | | -10 |
| **股权价值** | | **3,813亿** |

**DCF估值**：3,813亿 → **47.48元/股**

DCF显著高于PE/PB估值，原因是永续增长部分（3%）对长生命周期电网龙头给予了过高权重。鉴于电网投资周期性强、2030年后增长能见度低，**DCF仅作上限参考，不参与目标价合成**。

## 三、核心：估值全变量对冲审计矩阵

### 3.1 估值影响力过滤（审慎思考）

在众多利好利空中，经由审慎辨析，**只有以下因子真正具备改变南瑞估值中枢的能力**：

| 审计维度 | 核心因子 | **审慎思考（估值影响力过滤）** | 调节 | 深度理由 |
|:---|:---|:---|:---:|:---|
| **DNA：战略定力** | "设计+封测"不搞晶圆制造的克制 | **留存**。这不是平庸的战略选择——在MATCH法案落地前精准预判产业链博弈格局，体现了穿越周期的战略定力。具备改变估值中枢的稀缺性。 | **+5%** | 战略克制→资本配置效率↑→长期ROE韧性→PB估值溢价。 |
| **DNA：内生孵化能力** | 南瑞继保（构网储能全球领先）+ 科东（调度系统）均为内部孵化 | **留存**。持续孵化出行业第一的能力是极少数公司具备的"估值溢价基因"，如同腾讯的"赛马机制"。 | **+3%** | 内生孵化→增长可持续性↑→长期CAGR预期↑→PE估值溢价。 |
| **宏观：政策与战略对齐** | "六张网"政策优先级 | **过滤**。政策红利已充分体现在利润预测（E）中——国网投资15%的增长已经计入。对PE而言不产生额外溢价，因为市场本就知晓南瑞的政策属性。 | **0%** | E已体现。政策加持不构成对PE倍数的额外提升，除非政策力度持续超预期。 |
| **宏观：地缘与外部约束** | 中美芯片出口管制长期风险 | **留存但折价温和**。MATCH法案和EDA管制锁定了中国半导体产业进阶先进制程的路径，南瑞的IGBT自研（设计+封测）虽然规避了"造不出"的风险，但"DSP/FPGA高端芯片进口受限"是真实存在的长期约束。 | **-3%** | 地缘约束→技术天花板不确定性→长期PE折价，但幅度有限（南瑞的核心能力不在芯片制造）。 |
| **市场：流动性溢价** | 机构重仓+高分红+市值约1,900亿 | **过滤**。南瑞已是MSCI/沪深300/中证红利核心成分股，风格属性已被充分定价。当前市值约1,867亿不存在显著的流动性折价或溢价空间。 | **0%** | 现有估值已反映其大盘蓝筹的流动性属性。 |
| **市场：电力设备板块轮动** | 2026年"特高压+储能"热点的风格契合度 | **留存**。"十五五"电网投资的故事才刚刚开始展开（2026年为第一年），市场认知正在从"规划期"向"兑现期"切换。板块β上行的窗口期约在2026-2027年。 | **+2%** | "十五五"首年执行验证→行业β提升→阶段性估值扩张，2%为温和调节。 |
| **综合调节系数** | | | **+7%** | |

### 3.2 修正目标市值计算

```
修正目标市值 = 2,663亿 × (1 + 7%) = 2,850亿
修正目标价 = 2,850亿 / 80.32亿股 = 35.48元
```

**修正后2026年目标价：35.48元（不复权），较现价23.24元，潜在升幅 52.7%。**

### 3.3 逻辑纯净性校验

| 利好/利空 | 体现在E | 体现在PE | 理由 |
|:---|:---:|:---:|:---|
| 国网投资+20-37% | ✓ | — | 直接转化为订单和收入，可计算的增量 |
| 反内卷→毛利率回升 | ✓ | — | 具体化为毛利额增加，可计算 |
| 海外业务爆发 | ✓ | — | 具体化为海外分部营收和利润 |
| 战略定力（不搞晶圆制造） | — | ✓ | 不可直接计算为利润，但提升长期确定性 |
| 内生孵化能力（南瑞继保模式） | — | ✓ | 增长的可复制性提升，不可计算为单年利润 |
| 芯片地缘风险 | — | ✓ | 远期不确定性，体现为PE折价 |
| 行业β上行窗口 | — | ✓ | 市场情绪驱动的估值扩张，非基本面 |

**校验通过**：所有直接可计算的利好已纳入EPS预测，不可量化但改变长期确定性的因子纳入了PE调节。

## 四、确定性与胜率矩阵

### 4.1 三态逻辑加权

基于模块五的情景分析：

| 情景 | 概率 | 2026E EPS | 简化PE | 目标价 | 备考 |
|:---|---:|---:|---:|---:|:---|
| **保守（Bear）** | 20% | 1.10元 | 22x | **24.20元** | 国网执行不力+原材料大涨 |
| **基准（Base）** | 60% | 1.25元 | 26x | **32.50元** | 核心假设 |
| **乐观（Bull）** | 20% | 1.39元 | 28x | **38.92元** | 超预期兑现 |

**加权目标价** = 0.2×24.20 + 0.6×32.50 + 0.2×38.92 = **32.12元**（较现价23.24元+38.2%）

**6月14日重新计算盈亏比**：当前价23.24元已低于保守情景目标24.20元（低4.0%），意味着市场定价已隐含比我们此前的Bear case更为悲观的预期。以极限支撑位~22元（模块五压力测试的"软下限"）作为下行锚点：
**盈亏比** = (32.12 - 23.24) / (23.24 - 22.00) = 8.88 / 1.24 = **7.16:1**，不对称性进一步增强。

### 4.2 核心结论

以当前价23.24元：
- **乐观情景**（+67.5%上行至38.92元）
- **保守情景目前已被市场价格向下击穿**（23.24 < 24.20），市场定价隐含的悲观程度已超出我们Bear case的假设。这要么意味着市场非理性低估（买入机会），要么是我们的Bear case对铜价冲击的估计不足（需重新审视毛利率假设）
- **不对称性进一步增强**：加权目标价32.12元，上行空间8.88元（+38.2%），而以极限支撑22元计算的下行风险仅1.24元（-5.3%）

## 五、逻辑证伪与极限压力测试

### 5.1 "一击毙命"黑天鹅定义

最能推翻当前所有看涨逻辑的事件：**国家电网在未来3年内实质性削减设备采购预算，或因电力体制改革被迫将核心调度/控保系统开放竞标**。

### 5.2 极端场景审计

| 压力维度 | 极端假设 | 对南瑞影响 |
|:---|:---|:---|
| 国网投资削减 | 年投资从7,500亿降至5,000亿（-33%） | 网内营收下降至~280亿，总营收~550亿，净利~60亿，EPS=0.75元 |
| 调度/控保系统开放竞争 | 市占率从70%降至40% | 电网智能毛利率从30%降至22%，营收降至~250亿 |
| 重大产品事故 | 核心产品召回/赔偿 | 一次性损失50-100亿，品牌信誉永久损伤 |
| 中美全面脱钩 | IGBT/DSP/FPGA对华全面禁运 | 电网智能和能源低碳业务产能受限，毛利率断崖式下跌 |

### 5.3 "地狱价格"估算

最坏情况下（国网投资削减+竞争开放+中美脱钩同时发生）：
- 营收降至400亿，净利润30亿，EPS=0.37元
- PB定价：BVPS 7.18元 × 1.5x（清算折价50%）= **10.77元**
- PE定价：0.37元 × 15x = 5.55元

**综合极限支撑位**：取PB定价，约 **10-12元**。较当前价存在约55%的理论下行空间——这一定价仅在"电网行业基本面崩盘"的极端事件中出现，在当前"六张网"政策加持和全球电网设备短缺的背景下，概率趋近于零。

更现实的"软下限"（温和熊市+增速放缓至5%）：EPS 1.10元 × 20x = **22元**，下行空间13.7%，为可承受的风险区间。
