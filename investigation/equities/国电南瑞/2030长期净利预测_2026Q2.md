## 六、操作建议框架

```
当前价位: 23.45元（2026-06-05）
┌──────────────────────────────────────────────────────┐
│  19.60元(MA250)  ←── 强制止损（投资逻辑重审）         │
│  22.63元(MA120)  ←── 恐慌性加仓（重仓建仓）           │
│  24.00-24.57元   ←── 加仓区（分批配置）               │
│  25.00-26.50元   ←── 当前区间（持有/观望）             │
│  27.21元(MA60)   ←── 短期转强信号（突破后加仓）       │
│  32.50元         ←── 2026E基准目标（部分兑现）         │
│  35.48元         ←── 2026E修正目标（主仓兑现）         │
│  51.20元(2030E)  ←── 5年长期目标（需5年持有）          │
└──────────────────────────────────────────────────────┘
```

**基本原则**：
1. 22.63元以下：每下跌1元增加配置（最理想建仓区）
2. 持有周期：至少3-5年（覆盖"十五五"全周期）
3. 短期回撤容忍：20%以内不触发止损，对应22.63元
4. 关键兑现：2027年8月半年报+12月国网年度预算（确认2028年增长方向）
5. 退出时机：
   - 2027年：若Q2毛利率未回升至25.5%+ → 提前减仓
   - 2028年：若海外占比未达12% → 减仓
   - 2030年：达到51.20元（5年CAGR 17%目标）→ 兑现主仓

## 九、最终投资判断

**评级（2026年6月）**：**核心重仓配置（适度）**

| 情景 | 价格区间 | 操作 |
|:---|:---|:---|
| 22-23元 | 极度低估 | **重仓建仓** |
| 23-25元 | 估值合理 | 持有+适度加仓 |
| 25-28元 | 估值修复中 | 持有+观望 |
| 28-32元 | 接近短期目标 | 减仓兑现部分 |
| >32元 | 短期超涨 | 兑现主仓 |

**5年视角（2030E）**：
- 加权目标净利193亿，对应5年CAGR 18.5%
- 理论目标价51.20元（PE 22x × EPS 2.40）
- 现价23.45元 → 5年潜在回报+118%
- 加2.4%股息率 = 5年总回报率约+132%

**风险提示**：
- 短期（6-12个月）：财务费用持续恶化+毛利率弱反弹
- 中期（1-3年）：海外业务能否兑现是核心变量
- 长期（3-5年）：地缘政治+技术封锁+电网投资节奏
