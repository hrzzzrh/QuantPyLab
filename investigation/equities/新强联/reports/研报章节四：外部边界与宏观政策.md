# 研报章节四：外部边界与宏观政策

**研究日期**：2026年05月26日
**股票代码**：300850（新强联）
**前序模块核心"红线"继承**：①2025年抢装脉冲后2026年面临需求回落（模块二）；②产业链弱势地位——合同负债趋零、应收高企（模块三）；③2026Q1已现负增长。**本章核心使命**：评估宏观政策与地缘变量对新强联估值中枢和经营底线的边际影响。

---

## 一、宏观环境变量

### 1.1 利率与折现率：中性偏正面

2026年中国货币政策延续"适度宽松"基调，LPR处于历史低位区间（1年期约3.1%、5年期约3.6%）。对于DCF估值而言，低利率环境意味着较低的折现率（WACC），理论上支撑更高的估值中枢。但需要注意：新强联的实际借款成本（财务费用率约2.0%、短期借款利率估计约3%-4%），在低利率环境下融资成本可控，有助于缓解其高资本开支带来的财务压力。

**综合判断**：宏观利率环境对新强联估值**中性偏正面**（+3%~5%），但影响有限。

### 1.2 汇率：基本无关

公司98.34%的营收来自境内市场，境外收入仅0.77亿元（1.66%），且主要原材料均在国内采购。人民币汇率波动对公司经营的直接影响几乎为零。即使未来加大出口力度，风电轴承作为非标定制化产品，定价权主要取决于技术和认证壁垒，而非汇率短期波动。

**综合判断**：汇率影响**可忽略**。

### 1.3 通胀与原材料成本传导：关键变量

钢材（连铸圆坯、钢锭、轴承钢）是新强联最主要的生产成本。根据Mysteel数据：

| 品种 | 2026年4月价格 | 趋势判断 |
| :--- | :--- | :--- |
| 轴承钢GCr15 | 约4,840元/吨 | "震荡筑底、温和修复" |
| 铁矿石（62%澳粉） | 约109.5美元/吨 | 供需相对宽松，窄幅震荡 |
| 废钢 | 约2,481元/吨 | 成本支撑但上行动力不足 |
| 结构钢45# | 约3,615元/吨 | 成本刚性支撑，高位震荡 |

整体而言，2026年钢材市场呈现"供需弱平衡、成本有支撑"的格局。Mysteel预计轴承钢价格全年呈"震荡筑底、温和修复"态势，不存在大幅上涨的基础。

**对新强联的影响**：
- **有利面**：原材料价格平稳甚至偏弱，有利于维持或改善毛利率。2025年公司毛利率从18.53%恢复至28.00%，部分得益于原材料价格下行。
- **不利面**：若下游整机价格持续下压，"136号文"后风电场IRR压力传导至整机→零部件，轴承价格降幅可能超过原材料成本降幅。
- **关键量化参考**：2025年公司营业成本从24.00亿增至33.32亿（+38.85%），而营收增速（+57.11%）快于成本增速，说明"量价齐升"而非单纯的"成本红利"是毛利率改善的主因。2026年若营收增速放缓甚至转负，即便原材料价格稳定，毛利率也可能因固定成本摊薄减弱而回落。

**综合判断**：原材料价格可控，但**终端价格下行压力是更大威胁**。通胀传导效率总体**中性**。

## 二、地缘政治风险

### 2.1 中美贸易摩擦：短期影响有限，中长期需警惕

**直接关税风险**：2025年以来，中美贸易战经历剧烈升级-缓和的过山车行情。当前（2026年5月），美国对中国商品综合关税水平约30%（10%对等关税+20%芬太尼关税），双方在日内瓦会议（2025年5月）后同意暂停90天加征期。但关税政策的高度不确定性仍是中长期风险。

对新强联的直接影响**极为有限**：
1. 公司境外收入仅1.66%，且主要来自东南亚等非美市场
2. 风电轴承属于非标定制化工业品，不属于美方针对中国加征关税的重点品类（光伏、锂电池、电动汽车）

**间接风险值得关注**：
1. 美国于2025年8月启动对进口风力涡轮机及零部件的232条款国家安全调查，可能对风电机组及零部件加征关税。虽然新强联不直接出口美国，但若全球风电供应链因贸易壁垒而碎片化，中国整机商的出口受阻将间接影响轴承订单。
2. 美国对中国产轴承钢、特种钢的关税（25%），影响有限——新强联不出口钢材。
3. 更大的风险来自**技术封锁**：风电轴承的高端检测设备、精密磨床仍部分依赖进口（日本、德国），若中美技术脱钩扩大化，可能影响公司设备采购和技术升级。

**综合判断**：地缘政治风险**短期低、中长期中等**。不会构成"一击毙命"的威胁，但需纳入风险折价考量。

### 2.2 供应链重构（China+1）：影响可控

全球制造业的"中国+1"策略对风电轴承行业影响有限，原因在于：
- 风电轴承技术壁垒高、认证周期长（5-6年），供应链迁移难度极大
- 中国占据了全球70%以上的风电新增装机，是全球最大的终端市场，"中国+1"逻辑在风电领域缺乏替代市场
- 新强联固定资产39.24亿全在中国（洛阳+张家港），无法转移

### 2.3 能源地缘与大宗商品

2025-2026年全球地缘冲突（中东局势、俄乌冲突延续）对能源和钢铁原材料价格有间歇性扰动。但从实际走势看，铁矿石、焦煤等大宗品价格保持相对稳定，输入性通胀风险可控。

## 三、监管与产业政策——最大变量

### 3.1 "136号文"：重塑风电商业逻辑的制度性变革

2025年2月发布的《关于深化新能源上网电价市场化改革促进新能源高质量发展的通知》（发改价格〔2025〕136号），是新强联面临的**最重大外部政策变量**。

**政策核心**：
- 风电/光伏上网电量**原则上全部进入电力市场**，电价由市场交易形成
- 以2025年6月1日为界区分"存量"和"增量"项目，存量保持煤电基准价托底，增量需通过年度竞价获保障性电价
- 建立"差价结算"机制，市场交易均价与机制电价的差额由电网结算

**对整条风电产业链的影响链**：

```
136号文 → 增量项目电价竞价 → 部分区域机制电价接近下限（0.11-0.20元/度）
       → 风电场IRR承压 → 开发商决策更审慎 → 装机节奏放缓
       → 整机招标价继续下压 → 轴承等零部件价格承压
```

**2026年是136号文全面实施元年**，全国30个省市已发布实施细则。从已公布的竞价结果来看：
- 存量项目机制电价以煤电基准价（0.23-0.45元/度）为托底，过渡相对平稳
- 增量项目竞价上限多为当地煤电基准价，下限不设或很低
- 山东等省份已明确"后期将适时取消竞价下限"
- **机制电量比例**：存量项目多数100%纳入，增量项目最低仅30%

这意味着什么？2025年6月后新投产的风电项目，其**保障性收益比例可能低至30%**，剩余70%的电量需在现货市场以不确定的价格出售。风电场运营商面临的是"**收益高度不确定**"的新世界。

**对新强联的具体影响**：
1. **短期（2026年）**：2025年抢装透支了存量政策红利，2026年增量项目IRR不确定性上升→装机节奏可能低于预期→轴承订单环比回落。
2. **中期（2027-2028年）**：市场化电价机制将加速行业洗牌。只有技术领先、成本最优的风电项目才能存活→优质的整机商和零部件供应商反而可能受益于行业集中度提升。
3. **长期**：若风电无法在市场化电价下实现合理IRR，整个产业将面临"增速断崖"——新强联作为纯风电零部件供应商，对此缺乏对冲手段。

### 3.2 "十五五"规划与《风能北京宣言2.0》：远水难解近渴

| 政策 | 核心目标 | 落地约束 |
| :--- | :--- | :--- |
| "十五五"规划纲要 | 非化石能源消费比重达25%，海上风电累计1亿千瓦+ | 需配套电网消纳和电力市场化改革 |
| 《风能北京宣言2.0》 | "十五五"风电年均新增≥1.2亿千瓦（120GW） | **产业界宣言，非政府强制目标** |
| 非化石能源十年倍增行动 | 2035年风光总装机达36亿千瓦 | 长期方向明确，但实现路径不确定 |
| 国家能源局"十五五"规划 | 风电光伏保持年均2亿千瓦增长节奏 | 需"十五五"能源规划正式出台 |

**核心矛盾**：政策目标极具野心（2035年36亿千瓦风光），但实现路径面临**消纳瓶颈、电价市场化冲击、电网投资滞后**三大硬约束。对2026年而言，"十五五"的宏伟蓝图难以转化为对轴承厂的实质性订单。

### 3.3 产业政策与国产替代红利

**利好面**：
- 《风电装备制造行业规范条件》明确将轴承列为风电关键部件，要求符合或优于国标
- 关税摩擦加剧了"自主可控"的政策诉求，国产高端轴承获得更多政策资源倾斜
- 风电轴承被列入"卡脖子"技术攻关清单
- 轴承行业可享受高新技术企业15%所得税优惠

**利空面**：
- 环保政策趋严增加合规成本
- "反内卷"政策在钢铁/制造业推进，可能通过原材料端传导成本
- 地方政府对风电项目的审批节奏存在不确定性

---

## 四、模块四核心结论

1. **最大的外部变量是"136号文"主导的电价市场化改革**。这不是简单的利好或利空——它在短期内（2026年）因政策不确定性压制装机节奏（偏负面），中长期因行业洗牌出清低效产能反而有利于龙头企业（偏正面）。但对于一家全年营收几乎all-in风电的企业而言，**政策过渡期的阵痛不可避免**。

2. **原材料成本环境相对友好**。2026年钢材/轴承钢价格呈"震荡筑底"态势，不存在大幅上行的基础，有利于公司维持毛利率水平。

3. **地缘政治风险短期可控**。新强联的纯内需业务属性使其在中美贸易摩擦中几乎免疫。但需警惕高端设备进口受限对技术升级的潜在制约。

4. **政策面"远景美好、近景承压"的格局清晰**。"十五五"规划、《风能北京宣言2.0》勾勒了风电行业的宏大蓝图，但2026年作为政策过渡期的"青黄不接之年"，外部政策环境对公司的短期经营偏负面。

5. **利率环境温和利好估值**，但影响权重要低于公司自身的盈利周期。

---

*模块四完结。外部政策变量（尤其是"136号文"）的判断将直接影响模块五中2026-2027年的营收预测假设和模块六中估值折价因子的选择。*
