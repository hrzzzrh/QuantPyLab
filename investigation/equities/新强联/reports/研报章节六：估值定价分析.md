## 一、价值锚定：适配性方法选择

### 1.1 方法论筛选

新强联是**重资产、高周期、政策敏感性制造业**企业，资产属性偏向固定资产驱动，盈利属性高度波动，现金流质量极差。基于其特征，排除以下方法：
- **DCF**：不适用。经营现金流极度不稳定（2025年OCF仅0.18亿），对未来10年自由现金流的预测可信度几乎为零。
- **PS（市销率）**：参考价值有限。公司有利润，毛利率尚可，PS无法反映盈利能力的阶跃变化。
- **SOTP（分部估值）**：部分适用但难以精确。公司多业务板块，但85%利润来自风电，分部间缺乏可比估值锚。

**选用方法**：
1. **PE（市盈率）——主方法**：作为最直观的盈利倍数估值工具，适合有稳定盈利（虽然波动大）的制造业企业。
2. **PB-ROE（市净率-ROE模型）——辅助验证**：适合固定资产驱动的重资产企业，通过ROE判断PB的合理性。
3. **PEG（成长性估值）——参考锚**：2026年EPS大概率下滑，PEG为负值，参考意义有限。

### 1.2 历史分位审计（近3年PE/PB）

| 分位 | PE TTM | PB |
| :--- | :--- | :--- |
| P10（10%分位） | 21.28 | 1.21 |
| P25 | 24.78 | 1.85 |
| P50（中位） | 28.89 | 2.48 |
| P75 | 41.41 | 2.77 |
| P90 | 48.14 | 3.04 |
| **当前值** | **19.76** | **2.20** |
| 当前所处位置 | **低于P10** | **P25-P50之间** |

**表面解读**：当前PE_TTM 19.76x处于近3年绝对底部区（低于P10），PB 2.20x处于中低区域——**看似"极度低估"**。

**深层辨析——历史分位不可机械套用**（框架第158条规则明确警示）：

1. **PE_TTM的"幻觉"**：当前19.76倍PE是基于2025年全年（抢装高峰年）的归母净利润8.18亿计算的。而模块五预测2026年盈利大概率下滑23.3%（中性）至6.27亿。若以Forward PE计算：157.56÷6.27＝**25.1倍**，较表面PE高出27%。

2. **市场已经在定价盈利下行**。2025年PE中位数为29.94x、而2026年YTD PE中位仅23.84x——市场已经将PE估值压缩了约20%，这正是对盈利拐点的提前反映。19.76x的"低估"表象背后，可能是市场的**精准定价**而非错误定价。

3. **永续"低PE"陷阱**。A股市场上，周期股在盈利峰值期PE最低——贵州茅台在2013年PE仅8倍、三一重工在2021年PE不足10倍。用PE选时经常会"抄在半山腰"，历史上2024年PE曾触及17.79x的更低点（对应股价29.3元，比当前低23%）。

4. **行业可比估值参考**：风电零部件行业（轴承/铸件/法兰/主轴）可比公司当前PE普遍在15-25倍区间（考虑盈利下滑预期）。新强联25.1倍的Forward PE处于行业合理偏上水平，不存在明显低估。

### 1.3 PB-ROE 模型验证

| 指标 | 数值 | 解读 |
| :--- | :--- | :--- |
| 每股净资产（2026Q1） | 17.27元 | BVPS因可转债转股持续增厚 |
| 当前PB | 2.20倍 | 高于1倍，市场对ROE有正期望 |
| 预测ROE（2026E） | 9.0%（中性） | 较2025年13.78%显著下降 |
| 隐含PB/ROE比 | 0.24 | — |
| 理论合理PB（ROE=9%/要求回报10%） | ~0.9倍 | 若ROE低于资本成本，PB应<1 |
| 理论合理PB（ROE=12%/要求回报10%） | ~1.2倍 | 历史平均ROE情景 |

**结论**：以PB-ROE模型衡量，当前PB 2.20倍**偏贵**。9%的ROE无法支撑2倍以上的PB，除非市场预期ROE将在2027年后显著回升至15%+。这反映了市场对新强联中长期盈利能力的"信念溢价"，而非当前基本面的支撑。

## 二、基准市值与目标价计算

### 2.1 PE方法（主方法）

**合理PE倍数的锚定逻辑**：新强联作为风电轴承龙头，中长期受益于国产替代深化+风机大型化趋势，但短期面临盈利下滑+产业链弱势地位+现金流质量差三大瑕疵。给予如下PE倍数：

| 估值参数 | 保守 | 中性 | 乐观 |
| :--- | :--- | :--- | :--- |
| 2026E EPS（元） | 1.13 | 1.76 | 2.76 |
| 合理PE倍数 | 22x | 22x | 25x |
| **对应股价（元）** | **24.86** | **38.72** | **69.00** |
| 隐含市值（亿） | 102.9 | 160.4 | 285.8 |

PE 22x在中性情景中对应：
- 较当前19.76x（TTM）溢价11%（合理，Forward PE在盈利下滑年应高于TTM PE）
- 较近3年PE中位数28.89x折价24%（合理，因盈利增速为负+现金流质量差+客户集中度高风险）
- 较风电零部件行业可比中位20-25x处中间位置

**基准市值（中性PE×中性EPS）**：**38.72元 × 4.14亿 = 160.4亿**

### 2.2 PB-ROE 交叉验证

若给予PB 2.0倍（匹配约10% ROE的中性预期）：2.0 × 17.27 = **34.54元**，略低于PE方法的38.72元，反映PB-ROE模型对公司高盈利能力可持续性的谨慎态度。

## 四、逻辑纯净性校验

### 4.1 "E vs PE" 区分

| 因素 | 归属 | 理由 |
| :--- | :--- | :--- |
| 2026年风电装机回落→营收-8.7% | **E（利润）** | 具体的、可量化的量价变化 |
| "136号文"政策不确定性 | **PE（估值倍数）** | 不可精确量化为利润，关乎远期信心 |
| 经营现金流极度恶化 | **PE（估值倍数）** | 影响的是利润的"折现价值"而非利润本身 |
| 客户集中度风险 | **PE（估值倍数）** | 尚未发生的或有风险，不可计入当期E |
| 原材料价格稳定 | **E（利润）** | 已在毛利率预测中量化体现 |
| TRB技术领先 | **PE（估值倍数）** | 技术优势影响的是长期成长性和行业地位，非当期利润 |

### 4.2 锚点回测

修正隐含PE = 32.54 / 1.76 = **18.5倍**（基于中性EPS）

该PE水平：
- 低于近3年PE中位数28.89x（折价36%），合理——因为盈利在回落而非增长
- 接近2024年PE低点17.79x，具有历史参考意义
- 高于行业周期底部的绝对极限（12-15倍），赋予了一定的技术壁垒+齿轮箱成长溢价
- 给予的18.5x PE在"盈利小幅下滑+齿轮箱增长接力"的中性情景下属于合理偏保守定价

## 五、确定性与胜率矩阵

### 5.1 三态逻辑加权与盈亏比

| 情景 | 概率 | 目标股价（元） | 盈亏计算 |
| :--- | :--- | :--- | :--- |
| **保守** | 25% | 24.86 | 亏损 = 38.05 - 24.86 = -13.19元（-34.7%） |
| **中性** | 50% | 32.54（修正后） | 亏损 = 38.05 - 32.54 = -5.51元（-14.5%） |
| **乐观** | 25% | 69.00 | 盈利 = 69.00 - 38.05 = +30.95元（+81.3%） |
| **概率加权目标价** | | **44.60元** | 较现价溢价 **+17.2%** |

**盈亏比计算**：

(目标价 - 现价) / (现价 - 保守价格) = (44.60 - 38.05) / (38.05 - 24.86) = 6.55 / 13.19 = **+0.50**

盈亏比为正！——**在概率加权视角下，齿轮箱轴承修正后当前股价38.05元开始呈现正期望收益（盈亏比+0.50）**。

### 5.2 隐含胜率分析

在现价38.05元下：
- 保守情景（25%）：亏损34.7%
- 中性情景（50%）：亏损14.5%
- 乐观情景（25%）：盈利81.3%

加权期望收益率 = 25%×(-34.7%) + 50%×(-14.5%) + 25%×(+81.3%) = -8.68% - 7.25% + 20.33% = **+4.4%**

**盈亏比转为正值（+0.50），加权期望收益率为+4.4%——齿轮箱轴承修正后，当前股价开始呈现弱正期望。**

## 六、逻辑证伪与极限压力测试

### 6.1 黑天鹅场景模拟——"一击毙命"事件

**场景定义**：新强联最大客户（占营收约22%，推测为明阳智能或金风科技）宣布全面自研主轴轴承，并在2年内完成供应商切换，新强联失去该客户全部订单。

**地狱价格计算**：
- 失去22%营收→营收降至约33亿（-28%）
- 固定成本不变→毛利率坠至约18%
- 利润降至约2-3亿（EPS约0.6-0.7元）
- 市场情绪崩溃→PE压缩至15倍（周期底部极限估值）
- **地狱股价**：15 × 0.65 = **9.75元**（较现价-74.4%）

**资产清算底价**：
- 归母净资产70.0亿÷4.14亿 = 16.91元/股（账面值）
- 扣除商誉0.51亿、存货跌价准备（可变现价值打折）→**资产底约12-14元/股**
- 该价格水平对应PB约0.8倍，属于制造业重资产的极限清算估值

**极限支撑位**：**12-14元/股**。这是"企业价值毁灭但不至于归零"的终极锚点。

## 七、模块六核心结论

1. **齿轮箱轴承批量出货修正后，估值画像显著改善**。当前股价38.05元对应Forward PE 21.6倍（基于中性EPS 1.76元），较原25.1倍明显降低。全变量对冲修正目标价32.54元，较现价存在-14.5%的下行空间（原-26.6%）。盈亏比从-0.25转为**+0.50**，加权期望收益率为**+4.4%**。

2. **但仍不够便宜**。修正目标价32.54元 vs 当前价38.05元仍有显著差距。虽然盈亏比转为正值，但仅+4.4%的加权期望收益率在周期股中不具备足够的吸引力（通常要求+15%以上才值得承担周期风险）。

3. **乐观情景提供了诱人的上行空间（+81.3%至69.00元）**，但这需要风电装机超预期+齿轮箱获头部定点双条件同时成立。在当前信息状态下，25%的乐观概率属于"偏向慷慨"的估计。

4. **经营现金流质量差（OCF/NP=0.02）、客户过度集中（CR5=74.80%）、"136号文"政策不确定性——三项结构性瑕疵合计给予16%的估值折价**维持不变。齿轮箱轴承放量改善了盈利预期（E端），但未改变这些结构性风险（PE端）。

5. **安全边际仍不足**。建议股价回落至**32-35元区间**（对应Forward PE 18-20倍，接近全变量修正目标价）后考虑试探性建仓。当前38.05元的位置，风险收益比从"显著不利"改善为"弱正期望但安全边际不足"。
