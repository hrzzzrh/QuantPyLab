## 一、价值锚定：适配性方法选择

### 1.1 方法论筛选

新强联是**重资产、高周期、政策敏感性制造业**企业，资产属性偏向固定资产驱动，盈利属性高度波动，现金流质量极差。基于其特征，排除以下方法：
- **DCF**：不适用。经营现金流极度不稳定（2025年OCF仅0.18亿），对未来10年自由现金流的预测可信度几乎为零。
- **PS（市销率）**：参考价值有限。公司有利润，毛利率尚可，PS无法反映盈利能力的阶跃变化。
- **SOTP（分部估值）**：部分适用但难以精确。公司多业务板块，但85%利润来自风电，分部间缺乏可比估值锚。

**选用方法**：
1. **PE（市盈率）——主方法**：作为最直观的盈利倍数估值工具，适合有稳定盈利（虽然波动大）的制造业企业。
2. **PB-ROE（市净率-ROE模型）——辅助验证**：适合固定资产驱动的重资产企业，通过ROE判断PB的合理性。
3. **PEG（成长性估值）——参考锚**：2026年EPS大概率下滑，PEG为负值，参考意义有限。

**表面解读**：当前PE_TTM 17.28x已跌至近3年历史性低位——甚至低于2024年盈利崩塌期的最低PE（17.79x，对应股价29.3元）。PB 1.93x处于P25-P50之间，也属偏低。

**深层辨析——估值压力测试**（延续5.15日框架分析逻辑）：

1. **PE_TTM的"幻觉"已大幅消退**：5.15日PE_TTM 19.76x基于2025年利润8.18亿。当前17.28x意味着Forward PE（中性1.76元）= 18.9x，Forward PE（概率加权1.89元）= **17.6x**。估值压缩已达极值——即使2026年盈利如期下滑10.9%，当前估值也比表面TTM PE更加"诚实"。

2. **市场已过度定价盈利下行**。2025年PE中位数29.94x→2026年YTD PE中位23.84x→当前17.28x，PE已压缩42.3%。而模块五预测盈利仅下滑10.9%（中性）——**股价的"恐慌深度"已远超盈利下滑的"基本面深度"**。

3. **"周期股PE陷阱"的警示依然有效但需重新思考语境**：5.15日我们警告19.76x可能是"景气峰值PE的幻觉"。但当PE已压缩至17.28x（Forward 17.6x），历史极低分位的估值已经开始提供保护。2024年盈利崩塌期PE最低17.79x时，EPS仅0.18元（Forward PE = 17.79/0.18 = ！那是盈利地板期）。而当前Forward PE 17.6x对应的EPS为1.76元（盈利仍在高水平），两者不可同日而语。

4. **行业可比估值重新校准**：风电零部件行业可比公司当前PE普遍在15-25倍区间。新强联17.6x的Forward PE已处于行业**偏下限**而非偏上限，较可比中位20x折价12%。

5. **重大修正**：5.15日我们判断"25.1倍的Forward PE处于行业合理偏上水平，不存在明显低估"。11天后，Forward PE已跌至17.6x，**估值从"合理偏贵"切换为"显著低估"**。

### 1.3 PB-ROE 模型验证

| 指标 | 数值 | 解读 |
| :--- | :--- | :--- |
| 每股净资产（2026Q1） | 17.27元 | BVPS因可转债转股持续增厚 |
| 当前PB | 2.20倍 | 高于1倍，市场对ROE有正期望 |
| 预测ROE（2026E） | 9.0%（中性） | 较2025年13.78%显著下降 |
| 隐含PB/ROE比 | 0.24 | — |

**结论**：以PB-ROE模型衡量，当前PB 2.20倍**偏贵**。9%的ROE无法支撑2倍以上的PB，除非市场预期ROE将在2027年后显著回升至15%+。这反映了市场对新强联中长期盈利能力的"信念溢价"，而非当前基本面的支撑。

## 二、基准市值与目标价计算

### 2.1 PE方法（主方法）

**合理PE倍数的锚定逻辑**：新强联作为风电轴承龙头，中长期受益于国产替代深化+风机大型化趋势，但短期面临盈利下滑+产业链弱势地位+现金流质量差三大瑕疵。给予如下PE倍数：

| 估值参数 | 保守 | 中性 | 乐观 |
| :--- | :--- | :--- | :--- |
| 2026E EPS（元） | 1.13 | 1.76 | 2.76 |
| 合理PE倍数 | 22x | 22x | 25x |
| **对应股价（元）** | **24.86** | **38.72** | **69.00** |
| 隐含市值（亿） | 102.9 | 160.4 | 285.8 |

PE 22x在中性情景中对应：
- 较当前19.76x（TTM）溢价11%（合理，Forward PE在盈利下滑年应高于TTM PE）
- 较近3年PE中位数28.89x折价24%（合理，因盈利增速为负+现金流质量差+客户集中度高风险）
- 较风电零部件行业可比中位20-25x处中间位置

**基准市值（中性PE×中性EPS）**：**38.72元 × 4.14亿 = 160.4亿**

### 2.2 PB-ROE 交叉验证

若给予PB 2.0倍（匹配约10% ROE的中性预期）：2.0 × 17.27 = **34.54元**，略低于PE方法的38.72元，反映PB-ROE模型对公司高盈利能力可持续性的谨慎态度。

## 四、逻辑纯净性校验

### 4.1 "E vs PE" 区分

| 因素 | 归属 | 理由 |
| :--- | :--- | :--- |
| 2026年风电装机回落→营收-8.7% | **E（利润）** | 具体的、可量化的量价变化 |
| "136号文"政策不确定性 | **PE（估值倍数）** | 不可精确量化为利润，关乎远期信心 |
| 经营现金流极度恶化 | **PE（估值倍数）** | 影响的是利润的"折现价值"而非利润本身 |
| 客户集中度风险 | **PE（估值倍数）** | 尚未发生的或有风险，不可计入当期E |
| 原材料价格稳定 | **E（利润）** | 已在毛利率预测中量化体现 |
| TRB技术领先 | **PE（估值倍数）** | 技术优势影响的是长期成长性和行业地位，非当期利润 |

### 4.2 锚点回测

修正隐含PE = 32.54 / 1.76 = **18.5倍**（基于中性EPS）

该PE水平：
- 低于近3年PE中位数28.89x（折价36%），合理——因为盈利在回落而非增长
- 接近2024年PE低点17.79x，具有历史参考意义
- 高于行业周期底部的绝对极限（12-15倍），赋予了一定的技术壁垒+齿轮箱成长溢价
- 给予的18.5x PE在"盈利小幅下滑+齿轮箱增长接力"的中性情景下属于合理偏保守定价

**极限支撑位**：**12-14元/股**。这是"企业价值毁灭但不至于归零"的终极锚点。

## 七、模块六核心结论（2026.05.26重大修订）

1. **估值画像已发生根本性逆转**。5月15日我们判断38.05元"估值合理偏高、安全边际不足"。11天后，33.28元的价格已完全改变估值结构：Forward PE（中性）18.9x→**17.6x**（概率加权），较历史P10折价17%，处于近3年绝对极低分位。**估值从"合理偏贵"切换为"显著低估"**。

2. **盈亏比从+0.50跃升至+1.28**（提升156%），加权期望收益率从+4.4%跃升至**+20.6%**（提升368%）。这两个数字的核心变化并非因为盈利预测大幅上调（EPS仅+2.2%），而是因为**价格下跌了-12.5%**——市场用更便宜的价格提供了同样的基本面。这是价值投资者的"礼物窗口期"。

3. **当前价33.28元较全变量修正目标价32.54元仅溢价2.3%**——这意味着即使计入16%的结构性折价，当前价格已具备充分的安全边际。若给予长期投资者5-10%的买入误差容忍，33.28元已处于"可建仓价格带"。

4. **乐观情景提供了107.3%的上行空间（至69.00元）**。盈亏比+1.28意味着即使乐观情景概率仅25%，其潜在收益已足以覆盖保守情景的整体风险——这是"非对称收益"的典型特征。

5. **经营现金流质量差（OCF/NP=0.02）、客户过度集中（CR5=74.80%）——两项结构性瑕疵仍需警惕**。它们解释而非改变当前的低估值。低估价投机会往往存在于"有问题的好公司"而非"完美的平庸公司"中。

6. **入场建议（重大修正）**：33.28元已进入可建仓区间。建议：
   - **当前价试探性建仓5-10%**（33-34元，Forward PE 17-19x）
   - **28-30元加仓至15-20%**（Forward PE 15-17x，极度低估区域）
   - **<28元可重仓至25-30%**（PE<15x，仅2024年盈利崩塌期出现过）
   - 每批建仓间隔至少2周或5%价格间距，严格止损于25元以下（符合跟踪手册仓位纪律）
