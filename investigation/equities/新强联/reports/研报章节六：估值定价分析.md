## 一、价值锚定：适配性方法选择

### 1.1 方法论筛选

新强联是**重资产、高周期、政策敏感性制造业**企业，资产属性偏向固定资产驱动，盈利属性高度波动，现金流质量极差。基于其特征，排除以下方法：
- **DCF**：不适用。经营现金流极度不稳定（2025年OCF仅0.18亿），对未来10年自由现金流的预测可信度几乎为零。
- **PS（市销率）**：参考价值有限。公司有利润，毛利率尚可，PS无法反映盈利能力的阶跃变化。
- **SOTP（分部估值）**：部分适用但难以精确。公司多业务板块，但85%利润来自风电，分部间缺乏可比估值锚。

**选用方法**：
1. **PE（市盈率）——主方法**：作为最直观的盈利倍数估值工具，适合有稳定盈利（虽然波动大）的制造业企业。
2. **PB-ROE（市净率-ROE模型）——辅助验证**：适合固定资产驱动的重资产企业，通过ROE判断PB的合理性。
3. **PEG（成长性估值）——参考锚**：2026年EPS大概率下滑，PEG为负值，参考意义有限。

**表面解读**：当前PE_TTM 14.71x已跌至近3年从未触及的历史性极端低位——5月26日的17.28x已远低于P10（21.28x），而今日14.71x较P10再折价31%，较P50折价49%。近3年PE仅在2024年盈利崩塌期短暂触及过17.79x（彼时EPS仅0.18元），当前14.71x对应的是盈利水平仍在高位的7.97亿TTM净利——两者性质完全不同。PB 1.64x处于P10-P25之间，即将触及P10。

**深层辨析——估值压力测试**（延续5.15日框架分析逻辑，2026.06.08重大修订）：

1. **PE_TTM的"幻觉"已完全消散**。5月15日PE_TTM 19.76x基于年利润8.18亿，存在"景气峰值PE"的误导。6月8日PE_TTM 14.71x对应Forward PE（中性1.76元）= **16.1x**，Forward PE（概率加权1.89元）= **15.0x**。即使2026年盈利如期下滑10.9%，估值也比TTM数字更加"诚实"——而当前14.71x的TTM PE本身已接近周期底部。

2. **市场已严重过度定价盈利下行**。2025年PE中位数29.94x→2026年YTD PE中位23.84x→5月26日17.28x→6月8日14.71x。自3月高点以来PE已压缩**50.8%**，而模块五预测盈利仅下滑10.9%（中性）——**股价的"恐慌深度"已3倍于盈利下滑的"基本面深度"**。这是典型的过度抛售特征。

3. **"周期股PE陷阱"的警示已从"防范高估"逆转为"警惕错失低估"**。5月15日我们警告19.76x可能是景气峰值PE的幻觉；6月8日14.71x已跌至2024年盈利崩塌期（EPS仅0.18元，PE却高达17.79x）都未曾触及的极端低位——但当前EPS是1.89元（概率加权），盈利质量与2024年有天壤之别。**当前估值不是"景气幻觉"，而是"恐慌错杀"**。

4. **行业可比估值已进入极端折价区间**。风电零部件行业可比公司当前PE普遍在15-25倍区间。新强联15.0x的Forward PE（概率加权）已处行业**绝对下限**，较可比中位20x折价25%，较行业龙头（如金雷股份、日月股份等，PE约18-22x）折价15-30%。

5. **极值回测——近5年PE底部实证**：新强联上市以来（2020.07-2026.06）的PE_TTM最低值为以下几处：①2024年盈利崩塌期低点约17.79x（对应股价29.30元，EPS仅0.18元）；②2022年10月约18-19x。**当前14.71x已创近5年PE最低纪录**。若以Forward PE（1.89元）衡量，15.0x同样是史无前例的低位。

6. **重大修正（连续第二次）**：5月15日判断"估值合理偏高"→5月26日修正为"显著低估"→6月8日进一步修正为**"极端低估/恐慌定价"**。Forward PE从25.1x→17.6x→15.0x，不到一个月内估值压缩40%。这已不能用"基本面恶化"来解释，而是**流动性冲击+情绪崩溃+技术面强制平仓**的多重叠加。

## 二、基准市值与目标价计算

### 2.1 PE方法（主方法）

**合理PE倍数的锚定逻辑**：新强联作为风电轴承龙头，中长期受益于国产替代深化+风机大型化趋势，但短期面临盈利下滑+产业链弱势地位+现金流质量差三大瑕疵。给予如下PE倍数：

| 估值参数 | 保守 | 中性 | 乐观 |
| :--- | :--- | :--- | :--- |
| 2026E EPS（元） | 1.13 | 1.76 | 2.76 |
| 合理PE倍数 | 22x | 22x | 25x |
| **对应股价（元）** | **24.86** | **38.72** | **69.00** |
| 隐含市值（亿） | 102.9 | 160.4 | 285.8 |

PE 22x在中性情景中对应：
- 较当前14.71x（TTM）溢价50%（合理，Forward PE在盈利下滑年应高于TTM PE，且当前TTM PE已处于近5年最低位）
- 较近3年PE中位数28.89x折价24%（合理，因盈利增速为负+现金流质量差+客户集中度高风险）
- 较风电零部件行业可比中位20-25x处中间位置

**基准市值（中性PE×中性EPS）**：**38.72元 × 4.14亿 = 160.4亿**

### 2.2 PB-ROE 交叉验证

若给予PB 2.0倍（匹配约10% ROE的中性预期）：2.0 × 17.27 = **34.54元**，略低于PE方法的38.72元，反映PB-ROE模型对公司高盈利能力可持续性的谨慎态度。

## 四、逻辑纯净性校验

### 4.1 "E vs PE" 区分

| 因素 | 归属 | 理由 |
| :--- | :--- | :--- |
| 2026年风电装机回落→营收-8.7% | **E（利润）** | 具体的、可量化的量价变化 |
| "136号文"政策不确定性 | **PE（估值倍数）** | 不可精确量化为利润，关乎远期信心 |
| 经营现金流极度恶化 | **PE（估值倍数）** | 影响的是利润的"折现价值"而非利润本身 |
| 客户集中度风险 | **PE（估值倍数）** | 尚未发生的或有风险，不可计入当期E |
| 原材料价格稳定 | **E（利润）** | 已在毛利率预测中量化体现 |
| TRB技术领先 | **PE（估值倍数）** | 技术优势影响的是长期成长性和行业地位，非当期利润 |

### 4.2 锚点回测

修正隐含PE = 32.54 / 1.76 = **18.5倍**（基于中性EPS）

该PE水平：
- 低于近3年PE中位数28.89x（折价36%），合理——因为盈利在回落而非增长
- 接近2024年PE低点17.79x，具有历史参考意义
- 高于行业周期底部的绝对极限（12-15倍），赋予了一定的技术壁垒+齿轮箱成长溢价
- 给予的18.5x PE在"盈利小幅下滑+齿轮箱增长接力"的中性情景下属于合理偏保守定价

## 五、确定性与胜率矩阵

### 5.1 三态逻辑加权与盈亏比（2026.06.08重大修订）

**盈亏比计算（重大修正）**：

(目标价 - 现价) / (现价 - 保守价格) = (44.03 - 28.32) / (28.32 - 24.86) = 15.71 / 3.46 = **+4.54**

**盈亏比从5月26日的+1.28跃升至+4.54**——在不到两周内再提升255%。盈亏比>3.0已进入"极高赔率"区间，意味着概率加权后的潜在上行收益是潜在下行风险的4.54倍。这是典型的非对称收益结构——"向下有限（-12.2%至保守情景），向上巨大（+55.5%至概率加权目标，+143.6%至乐观情景）"。

- 利润降至约2-3亿（EPS约0.6-0.7元）

- 归母净资产70.0亿÷4.14亿 = 16.91元/股（账面值）

**极限支撑位**：**12-14元/股**。这是"企业价值毁灭但不至于归零"的终极锚点。

## 七、模块六核心结论（2026.06.08连续第三次重大修订）
