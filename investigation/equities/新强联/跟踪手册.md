## 三、逻辑证伪警报（Invalidation Triggers）

以下信号出现时，必须承认投资逻辑已死，直接触发止损或下调评级：

| 序号 | 证伪信号 | 证伪逻辑 | 行动触发 |
| :--- | :--- | :--- | :--- |
| **A** | 风电类产品营收连续两季同比下滑超15% | "十五五"120GW年均目标实质性落空，行业已进入结构性下行 | **清仓，无论价格** |
| **B** | 第一大客户（占22%+）宣布自研主轴轴承或切换供应商 | 核心大客户流失，商业模式逻辑崩塌 | **清仓** |
| **C** | 毛利率连续两季低于22% | 成本优势丧失+价格战失控，盈利中枢系统性下移 | **减仓50%以上** |
| **D** | 经营现金流连续四个季度为负且无改善趋势 | 盈利彻底沦为"纸面利润"，企业经营性流动性风险爆发 | **清仓** |
| **E** | **齿轮箱轴承2026年全年营收<2亿，且2027年中仍无头部齿轮箱厂定点公告** | 第二增长曲线证伪，公司成长性逻辑坍塌 | **下调至"永久回避"评级** |
| **F** | 滑动轴承在齿轮箱中渗透率超20%（替代风险） | 技术路线被颠覆，滚动轴承产线面临资产减值 | **减仓50%，重新评估** |
| **F** | 公司出现大额财务造假或审计意见非标 | 财务可信度崩塌 | **无条件清仓** |
| **G** | "十五五"能源规划正式版中风电目标低于80GW/年 | 政策顶层设计转向，行业天花板被实质性压低 | **清仓** |

## 五、仓位管理建议（如未来决定建仓）

| 价格区间（不复权） | 对应PE（2026E中性EPS 1.51） | 建议仓位 | 逻辑 |
| :--- | :--- | :--- | :--- |
| **38-45元** | 25-30x | **0%——观察区** | 估值偏贵，无安全边际 |
| **33-38元** | 22-25x | **0%——观察区** | 概率加权合理价附近，但无显著折扣 |
| **28-33元** | 18.5-22x | **试探性建仓5-10%** | 接近全变量修正目标价，开始具备安全边际 |
| **25-28元** | 16.5-18.5x | **加仓至15-20%** | 显著低于合理价值，估值保护充分 |
| **22-25元** | 14.5-16.5x | **加仓至25-30%** | 极度悲观定价，技术+资产支撑 |
| **<22元** | <14.5x | **重仓至40%** | 极端恐慌定价（2024年熊市底部），高赔率 |

**仓位纪律**：
- 单一股票仓位上限：**不超过总组合的15%**（周期股高波动特性）
- 止损纪律：若买入后公司出现逻辑证伪信号（见第三节），无条件执行止损
- 分批建仓：至少分3批建仓，每批间隔至少1个月或10%的价格间距
