## 一、核心逻辑验证点（Thesis Validators）

随财报发布需逐一验证的定量指标：

### 2026年中报（预计 2026年8月）重点验证

| 验证指标 | 阈值标准 | 若未达标→行动 |
|:---|:---|:---|
| 营收增速 | ≥16% YoY | 若＜12%，下调至谨慎 |
| 毛利率 | ≥69% | 若＜67%，检查 VBP 扩围是否超预期 |
| 归母净利增速 | ≥28% YoY（修订） | 若＜20%，下调至谨慎 |
| 销售费用率 | ≤26% | 若＞28%，经营杠杆逻辑受损 |
| 创伤类产品增速 | ≥12% | 续标未落地前大幅放缓需警惕 |
| 关节类产品增速 | ≥25% | 存量红利是否持续的关键信号 |
| 经营性现金流/净利润 | ≥1.0 | 盈利质量验证 |
| 存货周转天数 | ≤550天 | 继续下行则确认去库存顺利 |

## 三、逻辑证伪警报（Invalidation Triggers）

**重点！** 以下信号出现时，必须承认投资逻辑已死/受损，触发止损或下调评级：

### 🔴 一级警报（立即减仓/清仓）

| 信号 | 定量阈值 | 逻辑意义 |
|:---|:---|:---|
| **脊柱续标降价 >20%** | 平均降幅超 20% | "温和续标"逻辑彻底崩坏，集采重回激进模式 |
| **公司卷入重大质量安全事件** | 产品召回/NMPA通报 | 骨科植入物的质量信誉是生存底线 |
| **毛利率跌破 65% 并连续两季** | 毛利率 <65% 持续 2Q | 定价权丧失或成本失控的强信号 |
| **核心管理层（林志雄兄弟）大幅减持** | 半年内减持超 5% | 实际控制人信心动摇的最直接信号 |

### 🟡 二级警报（下调至观望/中性）

| 信号 | 定量阈值 |
|:---|:---|:---|
| **归母净利连续两个季度 YoY <15%** | 连续 2Q 利润增速 <15% |
| **脊柱市占率从 7% 下降至 <5%** | 脊柱续标后市占率下滑 |
| **关节市占率停滞（<4%）连续两季** | 关节增长逻辑核心受损 |
| **销售费用率回升至 >28% 并持续** | 经营杠杆反转 |
| **存货跌价准备年化 >1.5 亿** | 存货问题恶化 |
| **经营性现金流/净利润 <0.8** | 盈利质量恶化 |

## 五、跟踪操作清单

### 每季度必查数据库查询

```sql
-- 1. TTM 滚动数据趋势
SELECT report_date, revenue_ttm/1e8, net_profit_ttm/1e8, ocf_ttm/1e8
FROM fin_ttm WHERE symbol='002901' ORDER BY report_date DESC LIMIT 6;

-- 2. 产品线增速（从利润表推算毛利率变化）
SELECT report_date, "营业收入"/1e8, "营业成本"/1e8,
       "销售费用"/1e8, "研发费用"/1e8
FROM fin_income_statement WHERE symbol='002901' ORDER BY report_date DESC LIMIT 4;

-- 3. 当前估值
SELECT date, raw_close, pe_ttm, pb, market_cap/1e8
FROM v_daily_valuation WHERE symbol='002901' ORDER BY date DESC LIMIT 1;
```

### 每月必关注

- 国家医保局/药监局政策动态
- 骨科耗材集采相关公告
- 竞品（威高骨科、春立医疗）季报
