# 研报章节二：产业周期与终端穿透

**研究日期**：2026年06月03日

**前序模块核心结论回顾**：模块一发现兆易创新2023年遭遇深度周期低谷（净利1.61亿，ROE 1.06%），2025年修复至历史新高营收（92.03亿），2026Q1出现爆发式增长（单季净利14.61亿），TTM净利28.75亿已超上轮周期峰值。公司资产负债极保守（负债率10.16%），但盈利波动幅度高达17倍，周期属性极强。本模块将穿透至产业层，验证此轮高增长的可持续性。

---

## 一、产业生命周期与库存节奏

### 1.1 周期定位：结构性超级牛市，非传统周期性复苏

当前全球半导体存储行业正处于一轮**非同寻常的超级上行周期**。与传统周期由需求爆发驱动的模式不同，本轮周期的核心驱动力是**供给侧的结构性收缩**——全球存储巨头（三星、SK海力士、美光）正在系统性地退出成熟制程产品线（DDR3/DDR4、2D NAND、MLC NAND），将晶圆产能大规模转向HBM和DDR5等AI高利润产品。

高盛将当前存储供需失衡定性为**15年来最严重**：2026年全球DRAM缺口4.9%、NAND缺口4.2%、HBM缺口5.1%，均为2011年以来最大。摩根士丹利预计2026年下半年DDR4供需缺口可能扩大至19%-20%——在半导体行业，缺口超过5%即触发显著涨价，接近20%意味着持续性强势涨价周期。

Counterpoint Research明确将当前行情定性为"超级牛市"，热度已超过2018年历史峰值。TrendForce数据显示，2026Q1 DRAM合约价环比上涨90%-95%，NAND合约价上涨55%-60%，Q2 DRAM预计再涨58%-63%、NAND再涨70%-75%——NAND首次在本轮周期中超越DRAM涨价速度。

### 1.2 库存动力学：供给真空下的防御性备货

与典型周期的"主动补库→被动补库→主动去库→被动去库"四阶段模型不同，当前产业链面临的是**供给侧的"结构性真空"**：

- **上游（原厂）**：三星2025Q1关闭最后一条2D NAND产线（华城Line 12，月产能约8-10万片），美光削减利基型产出、全面聚焦数据中心客户，SK海力士无锡工厂从DDR4向DDR5后端产能切换。
- **中游（分销商/代理商）**：TrendForce观察到，医疗器械、工业机器人、汽车电子等行业正进行"集中性采购"，部分产品采购周期从传统的3个月延长至半年以上。宏旺（Macronix）已将客户定价从季度调整改为月度调整——管理层不相信今天的价格就是明天的价格。
- **下游（终端厂商）**：出现明显的"防御性备货"行为。但需要辨明：当前备货并非传统意义上的"牛鞭效应"投机囤货，而是对**供应可得性（availability）**而非仅对价格的担忧驱动。因为部分2D NAND型号的交期已延长至12个月以上，且原厂已明确宣布不再复产。

**核心研判**：当前库存行为是对**结构性供应退出**的理性响应，而非单纯的渠道投机放大。但随着价格持续上行，若2026H2下游客户开始基于"涨价预期"而非"供应可得性"加大备货，则存在需求前置风险。

## 二、终端需求穿透与跨期断层审计

### 2.1 最终产品画像

兆易创新的产品最终负载于以下终端形态：

| 产品线 | 主要终端产品 | 2025年收入占比 |
|:---|:---|:---:|
| **NOR Flash** | 汽车ECU/智能座舱、IoT传感器、TWS耳机、PC BIOS、5G基站、工业PLC | ~50%（估算） |
| **SLC NAND Flash** | 网络通信设备、机顶盒、工业控制、汽车电子、医疗设备 | ~15%（估算） |
| **利基型DRAM** | 电视、机顶盒、网络通信、智能家居、车载信息娱乐 | ~6%（估算） |
| **MCU（32位）** | 工业自动化、家电、汽车电子、消费电子、能源电力 | ~21% |
| **传感器** | 手机、平板、智能门锁 | ~4% |
| **模拟芯片** | 移动电源、智能穿戴、TWS充电仓、扫地机器人 | ~4% |

### 2.2 终端景气度审计（2026-2028）

#### A. AI驱动的数据中心存储需求（2026年最核心变量）

2026年全球云服务商（CSP）资本开支约7,550亿美元，其中存储采购占比约30%。TrendForce数据显示，2026Q1全球前五大NAND Flash供应商合计营收环比暴增83.7%，超越389亿美元，核心驱动力为企业级SSD的指数级需求增长。企业级SSD预计在2026年首次超越消费级存储，成为NAND最大应用领域。

**对于兆易创新的传导逻辑**：AI并非直接拉动兆易创新的产品需求（公司不做HBM/企业级SSD），而是通过**间接的产能挤出效应**产生巨大利好——当三星/海力士/美光将所有产能和资本开支投入HBM和3D NAND时，利基型存储（NOR Flash、SLC NAND、DDR3/DDR4）的供给被系统性压缩，形成结构性涨价。

#### B. 汽车电子（长期结构性增量）

2025年全球NOR Flash市场中，汽车电子占比22.7%（较2022年提升8.1pp），是增长最快的应用场景。单车NOR Flash需求量持续攀升：
- 智能座舱域控制器：1-2颗高密度NOR（256Mb+）
- ADAS处理器：1-2颗
- 车载以太网PHY：1颗SPI NOR
- 合计：中高端智能汽车搭载10-15颗NOR Flash，容量从64Mb向256Mb以上升级

兆易创新车规Flash累计出货量已超3亿颗，车规级2Mb~2Gb容量全线铺齐，并获得"2025汽车电子·金芯奖卓越产品奖"。在国产替代政策（工信部强制关键基础设施采购国产半导体）推动下，车规存储的国产化率将从当前不到15%持续提升。

#### C. 工业控制与边缘AI

工业PLC、机器人控制器、智能电网终端等场景对NOR Flash和SLC NAND的可靠性和长寿命有刚性需求。边缘AI的兴起（AI手机、AI PC、智能工厂）增加了终端设备的代码量和固件复杂性，推动存储密度升级（128Mb→256Mb→512Mb）。Omdia指出，"更大的固件容量、无线升级（OTA）需求以及边缘AI的集成"共同推动了高密度NOR Flash的采用。

#### D. 消费电子：结构分化

消费电子（手机/PC）仍是最大的出货量场景但增速放缓（IoT增速从2023年27.6%降至12.1%）。但2026年出现重要边际变化：
- **高价格抑制需求**：DRAM价格较2024年累计上涨200%-400%，NAND上涨100%+，PC OEM已面临严重的BOM成本压力。TrendForce注意到，Q2 2026消费者NAND现货价格已出现30%-40%的下跌（与合同价上涨形成背离），反映终端消费者对高价的抵制。
- **但利基产品需求刚性**：SLC NAND/MLC NAND在工业、汽车、医疗等场景难以被3D NAND替代——因为重新设计、重新认证的周期长达12-24个月，且安全认证成本极高。这意味着即使消费端需求走弱，利基型产品的需求仍有坚实的"地板"。

### 2.3 动能溯源与透支审计

**增长动能来源辨析**：

| 增长动力 | 性质 | 持续性评估 |
|:---|:---|:---|
| 海外大厂退出利基市场（供给侧） | 内生结构变化，非周期波动 | **高确定性**（三星/美光/海力士已公开确认且不可逆） |
| AI服务器建设（需求侧间接溢出） | 外生技术变革 | **高确定性**（$755B CSP资本开支已锁定） |
| 汽车智能化渗透率提升 | 内生长周期趋势 | **高确定性**（单车芯片含量持续增长） |
| 中国国产替代政策 | 外生政策驱动 | **中高确定性**（政策方向明确，但执行节奏存变数） |
| 下游防御性备货（库存周期） | 短期行为 | **低确定性**（可能在2026H2出现需求透支） |

**跨期断层风险审计**：

1. **供给侧退出不可逆**：与过往的"周期减产"不同，本次三星等巨头的退出是**物理层面的产线拆除/转换**（将2D NAND产线改造为DRAM后端工厂），即使未来价格上涨，重新复产的成本和时间窗口都极高。因此，供给侧的逻辑具有**3-5年的持续性**，而非12-18个月的周期波动。

2. **需求前置风险需警惕**：当前产业链下游的防御性备货是理性的（基于供应可得性），但需要注意的是——一旦供应恐慌缓解（如2027年Winbond/兆易创新等的新产能陆续释放），前期累积的超额库存可能触发一轮去库存周期。这可能在2027-2028年形成阶段性压力。

3. **价格承受力天花板**：NAND消费端现货价已开始下跌，说明消费者对涨价的承受力并非无限。如果利基型存储涨价过快，可能加速下游客户的替代方案（如用3D TLC替代MLC、用eMMC替代SLC NAND）——尽管短期替代难度大，但"贵到一定程度"会显著加速替代进程。

## 三、供需边际与压力传导

### 3.1 供需平衡审计

**利基型DRAM**：
- 供给：三星/海力士/美光持续退出DDR3/DDR4产能 → 2026年全球供给预计收缩15%-20%
- 需求：工业控制、网络通信、AI服务器配套（非HBM部分）需求稳定增长5%-8%
- 缺口：2026H2预计达到19%-20%，为历史最高水平
- 竞争格局：留存供应商（南亚科、华邦电、兆易创新、北京矽成）享受卖方市场定价权
- **供需边际方向：正在从紧平衡向严重短缺滑落**

**SLC NAND Flash**：
- 供给：三星退出2D NAND → 全球MLC产能2026年同比降41.7%，SLC供给同步收缩；Kioxia宣布2029年全面退出，最终订单日2026年9月
- 需求：边缘AI、工业控制、通信设备持续增长，且替代难度极高
- 缺口：TrendForce预计SLC NAND 2026年Q2-Q4涨价约120%
- 留存供应商：华邦电、旺宏、兆易创新、东芯股份
- **供需边际方向：正在从紧平衡向严重短缺滑落**

**NOR Flash**：
- 供给：主要厂商产能向高密度（256Mb+）和车规产品倾斜，低密度供应受到挤压；但NOR Flash产能并未像DRAM/NAND那样被系统性退出
- 需求：高密度NOR Flash（128Mb→256Mb+升级）需求旺盛（汽车/工业/边缘AI），低密度消费级需求平稳
- 价格：Omdia认为"NOR Flash定价保持稳定，基本不受存储周期性波动影响"——这与兆易创新年报中提到的"温和上涨"一致
- **供需边际方向：结构性偏紧但非短缺，高密度产品溢价持续扩大**

### 3.2 压力传导特征分析

**从AI服务器到利基型存储的传导路径**：

AI需求爆发 → 三星/海力士/美光将所有产能转向HBM/DDR5/3D NAND → 物理性退出DDR3/DDR4/2D NAND/部分NOR Flash产能 → 利基市场供给真空 → 留存供应商（兆易创新等）享受量价齐升

**独特的"反向传导"特征**：传统周期中，行业下游（消费电子）波动向上游（芯片设计→晶圆制造）传导，牛鞭效应放大波动。但在本轮周期中：

1. **波动源在供给端而非需求端**：不是终端需求突然暴增，而是供给端被"抽真空"
2. **需求端有底部支撑**：汽车/工业/医疗等场景的认证周期长（12-24个月）、替代成本高，形成了坚实的需求"地板"
3. **兆易创新处于传导链的有利位置**：作为Fabless设计公司，公司不直接受晶圆产能分配约束（通过与代工厂的长期协议锁定产能），同时受益于终端产品涨价

**压力时滞评估**：如果消费电子需求因高价而走弱（TWS、手机等），传导到兆易创新需要1-2个季度。但公司在消费电子NOR Flash之外的业务占比（汽车+工业+i-SERVer等）正在快速提升，消费电子风险的暴露面在缩小。

## 四、强制复核项

### 复核1：是否穿透并引用了最终终端产品的未来两年景气度预测？

**是**。已对四大终端场景进行了2026-2028景气度审计：
- AI数据中心：$755B资本开支锁定，2026-2027持续高景气
- 汽车电子：单车芯片含量持续升级，国产替代政策加持
- 工业/IoT/边缘AI：结构性增长，但增速从高位放缓
- 消费电子：出现价格承受力天花板，消费端NAND现货价已跌30%-40%

### 复核2：是否审计了当前增长逻辑中是否存在由于刺激政策导致的"跨期需求移位"？

**是**。中国国产替代政策（工信部强制关键基础设施采购国产半导体）为兆易创新的车规/工控存储提供了额外的需求增量。但该政策属于**长期结构性变量**（国产化率从15%持续提升至40-50%以上），而非短期的补贴刺激，因此跨期透支风险有限。真正的风险在于短期防御性备货可能在2027年新产能释放后形成库存去化压力。

### 复核3：是否基于行业产能与终端需求，分析了供需边际的演变方向及其对公司的压力传导特征？

**是**。详见第三节。核心结论：
- 供需边际正在从紧平衡向严重短缺滑落（利基型DRAM和SLC NAND）
- 压力传导对兆易创新整体有利：供给退出驱动涨价，Fabless模式灵活应对产能约束
- 主要风险监测指标：消费端价格承受力天花板（已观察到的NAND消费端现货跌价）、替代方案加速出现
