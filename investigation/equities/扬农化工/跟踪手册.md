## 三、逻辑证伪警报 (Invalidation Triggers)

**以下信号出现时，必须承认投资逻辑已死，直接触发止损或下调评级：**

### 3.1 一级警报（核心逻辑被颠覆 → 无条件止损）

| 警报编号 | 触发条件 | 逻辑死亡原因 |
| :---: | :--- | :--- |
| 🔴 **A1** | 公司连续两个季度毛利率<21%（2026年Q2+Q3均未改善） | 毛利率反转逻辑被证伪，行业供需改善不及预期 |
| 🔴 **A2** | 菊酯类产品(功夫菊酯/联苯菊酯)价格跌破2025年低点且无反弹迹象 | 核心品类竞争壁垒被侵蚀，可能面临技术替代或印度大规模替代 |
| 🔴 **A3** | 公司核心工厂（优嘉/优创）发生重大安全/环保事故导致停产>3个月 | 生产中断+客户流失+品牌受损，短期不可逆 |
| 🔴 **A4** | 中国农药出口被主要市场（巴西/美国/欧盟）加征惩罚性关税(>50%) | 58%出口收入面临断崖风险 |
| 🔴 **A5** | 辽宁优创项目出现重大工程质量问题，产能无法按期释放 | 最大增量引擎熄火，2026E量增假设破灭 |

### 3.2 二级警报（部分逻辑受损 → 下调至"持有/观望"）

| 警报编号 | 触发条件 | 应对措施 |
| :---: | :--- | :--- |
| 🟡 **B1** | 2026年全年净利<14亿(增速<9%)，显著低于Base情景(16.3亿) | 下调目标价至70-75元，评级降至"持有" |
| 🟡 **B2** | 应收账款/营收比率季度环比持续恶化(>30%) | 警示回款风险和潜在坏账，下调盈利质量评分 |
| 🟡 **B3** | 行业"一证一品"政策执行明显打折扣，灰色产能未如期退出 | 政策红利逻辑受损，估值中需剔除政策溢价(+3%) |
| 🟡 **B4** | 先正达集团大幅削减关联采购额(>20%) | 核心客户关系松动，需评估替代客户获取能力 |
| 🟡 **B5** | 全球农药市场再次进入去库周期(美国库存同比转负) | 行业周期反转逻辑推迟，下调盈利预测 |
