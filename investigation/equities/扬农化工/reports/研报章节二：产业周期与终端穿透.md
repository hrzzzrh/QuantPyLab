## 二、产业生命周期与周期定位

### 2.1 全球农药行业生命周期

农药行业处于**成熟期中段**，具有以下特征：

| 维度 | 特征 |
| :--- | :--- |
| 市场增速 | 全球CAGR 4-7%（2026-2030），从"量增"转向"质升" |
| 竞争格局 | 国际巨头（先正达/拜耳/科迪华/巴斯夫）垄断创制药，中国企业主导非专利药 |
| 技术演进 | 从化学农药向生物农药（CAGR 12-15%）、精准农业迁移 |
| 监管趋势 | 全球趋严，中国"减量增效"、欧盟2030年减半目标 |
| 增长动力 | 人口增长（2050年97亿）、粮食增产（+70%需求）、气候变化驱动病虫害压力 |

**关键结论**：农药市场总量仍在增长，但增速从过去的"量价齐升"转向"结构升级"。扬农化工所处的非专利原药赛道面临"量增价跌"困境——2025年中国化学农药产量首次突破400万吨（+12%），但行业营收仅增长5%，利润被严重压缩。

### 2.2 周期位置判定：底部确认，复苏启动

中国农药行业呈现典型的**5-6年一轮大周期**，上行和下行各持续2-3年。本轮周期：

```
2021-2022H1：超级景气（俄乌冲突→粮价暴涨→农药需求激增→草甘膦8万元/吨）
2022Q3-2025Q2：深度下行（产能释放+渠道去库+需求走弱→草甘膦跌至2.34万元/吨）
2025Q2-现在 ：底部确认+复苏启动
```

**底部确认的八大信号**：

1. **价格深度超跌**：草甘膦从2022年高点8.03万元/吨跌至2025年5月低点2.34万元/吨，跌幅70.9%。2025年12月行业原药价格指数71.44点，同比跌2.03%。
2. **下行时间充分**：自2022Q3起已下行3年，与历史周期（2-3年下行）完全匹配。
3. **渠道库存见底**：美国农化品库存同比2025Q2首次转正（4月+0.9%，5月+0.7%），标志着历时3年的去库周期终结。草甘膦库存从2025年4月8.26万吨骤降至8月中旬2.85万吨（-67%），与2020年周期启动初期高度相似。
4. **出口数据反转**：中国农药制剂出口金额历经10个季度大幅调整后，2025Q2同比+22%、环比+25%，接近2022Q2水平。
5. **行业集体提价**：2026年3月，印度UPL、中国利民股份、中天邦正等多家企业同步提价5-15%，为2022年以来首次系统性涨价。
6. **供给侧出清加速**："一证一品"新政（2026年1月实施）+ 中国农药工业协会"反内卷"三年行动 + 出口退税取消（草铵膦等品种），三管齐下淘汰落后产能。
7. **成本推动底部上移**：原油价格上涨 + 菊酯核心中间体贲亭酸甲酯涨至6万元/吨 + 功夫酸涨至11.5万元/吨，成本端形成硬支撑。
8. **企业盈利触底**：行业普遍微利甚至亏损，草甘膦行业毛利率2025年5月一度为负，后在7月转正至3.82%。

**周期阶段判定**：**底部右侧，复苏早期。** 本轮涨价周期预计持续6-12个月（多家机构共识），但需注意：本轮为成本驱动型上涨，与2021-2022年需求驱动型超级周期有本质区别——涨幅可能更温和、更渐进。

## 三、终端需求穿透（End-Demand Penetration）

### 3.1 最终产品画像

扬农化工产品的最终终端是**全球粮食与农产品生产体系**：

```
原油/矿产 → 化工中间体 → 农药原药（扬农化工核心环节）→ 农药制剂 → 农作物种植 → 粮食/果蔬消费
```

公司产品最终负载于以下终端形态（按应用领域）：

| 产品类别 | 代表品种 | 终端作物 | 终端消费场景 |
| :--- | :--- | :--- | :--- |
| 杀虫剂（菊酯类） | 高效氯氟氰菊酯、联苯菊酯 | 蔬菜、茶叶、果树、棉花 | 日常食品消费 |
| 卫生用药（菊酯） | 拟除虫菊酯系列 | 家用/公共卫生消杀 | 居民健康防护 |
| 除草剂 | 草甘膦、麦草畏、烯草酮 | 大豆、玉米、水稻、小麦 | 主粮与饲料消费 |
| 杀菌剂 | 氟啶胺、吡唑醚菌酯 | 果蔬、大田作物 | 食品品质保障 |

### 3.2 终端景气度审计（2026-2028年）

**全球粮食需求端**：刚性上行。

| 指标 | 数据 | 来源 |
| :--- | :--- | :--- |
| 全球人口 | 2026年约82亿，2050年预计97亿 | UN Population Division |
| 粮食增产需求 | 2050年前需增产70% | FAO |
| 全球谷物产量 | 2025/2026年度约28.5亿吨，持平微增 | FAO/IGC |
| 全球主要农作物种植面积 | 2025年持平微增（+0.5-1%） | USDA |
| 全球农药市场规模 | 2026年$800-910亿，CAGR 4-7%至2030年 | 多源综合 |
| 中国农药市场规模 | 2026年超4000亿元人民币 | 中研普华 |

**关键驱动力**：
1. **粮食安全刚需**：全球每年约1/4农作物遭受病虫草害，农药是粮食生产的"保险"。联合国粮农组织持续强调粮食安全，各国均将农药作为战略物资。
2. **气候变化放大病虫害压力**：研究表明气温每升高1℃，病虫害导致的作物损失增加10-25%。全球变暖导致病虫害向北扩张、繁殖季延长。
3. **饲用农产品需求增长**：新兴市场膳食结构升级（更多肉类消费）→ 饲料粮（大豆、玉米）需求增加 → 种植面积扩大 → 农药需求上升。
4. **生物农药增量市场**：全球生物农药市场2025年约$75亿，CAGR 12%+，但占比仍仅15-22%，化学农药仍为绝对主体。

**扬农化工特有终端——卫生用药**：
- 全球卫生杀虫剂市场约$60-80亿，受益于热带地区城市化、登革热/基孔肯雅热等蚊媒病防控
- 公司国内+东南亚市占率约70%，属于"类消费"的刚性需求，周期性弱于农用农药

### 3.3 动能溯源与透支审计

**增长动能来源判定**：

| 驱动力 | 性质 | 持续性判断 |
| :--- | :--- | :--- |
| 全球人口增长+膳食升级 | 内生（结构性） | 长期确定，10年+ |
| 气候变化病虫害压力 | 内生（环境性） | 中期确定，持续强化 |
| 生物农药替代化学农药 | 内生（技术迭代） | 长期渐进，非颠覆性 |
| 中国"一证一品"政策 | 外生（监管） | 中期确定，一次性效力 |
| 全球渠道补库 | 外生（周期性） | 短期确定，6-12个月 |
| 转基因作物推广 | 外生（技术政策） | 中期渐进 |
| 俄乌冲突/地缘波动 | 外生（偶然） | 不可预测 |

**跨期需求透支审计（强制复核）**：

> **审计结论：不存在显著的"需求前置"导致的跨期断层风险。**

理由：
1. 当前处于**去库终结→补库启动**的底部反转阶段，而非需求透支后的回落阶段。
2. 2021-2022年的超级景气透支了需求，但经过3年的深度调整（渠道持续去库），透支效应已完全消化。
3. 2025年"冬储备货推迟"（见年报），反而意味着2026年春耕需求未被提前释放。
4. 农药作为消耗性农资，每年需重复施用，无法跨年"囤积需求"。唯一可能透支的是渠道库存，而渠道库存已降至历史低位。

**需警惕的跨期扰动**：
- 2026年春季的集中补库可能造成Q1-Q2数据"虚高"，下半年面临增速回落
- "一证一品"新政可能导致部分企业在过渡期（2026H1）集中处理非合规库存，低价甩货扰动价格

## 四、供需边际与压力传导

### 4.1 供需平衡审计

**供给侧：总量过剩，边际改善。**

| 供给因素 | 方向 | 强度 | 时间窗口 |
| :--- | :---: | :---: | :--- |
| 中国化学农药产量突破400万吨（+12%YoY） | 利空 | ★★★★★ | 2025-2026 |
| 印度等新兴国家产能扩张 | 利空 | ★★★ | 2026-2028 |
| "一证一品"淘汰非合规产能 | 利多 | ★★★★ | 2026Q1起 |
| 草甘膦无新增产能（2018年起零增长） | 利多 | ★★★ | 持续性 |
| 环保监管趋严（江苏限制目录2025年本） | 利多 | ★★★ | 持续性 |
| 出口退税取消（草铵膦等品种）倒逼出清 | 利多 | ★★★ | 2026年4月起 |
| 行业"反内卷"三年行动 | 利多 | ★★★ | 2026-2028 |
| 菊酯中间体贲亭酸甲酯供应缩减 | 利多 | ★★★ | 2025Q4起 |

**需求侧：边际回暖。**

| 需求因素 | 方向 | 强度 | 时间窗口 |
| :--- | :---: | :---: | :--- |
| 全球渠道去库结束→补库启动 | 利多 | ★★★★★ | 2026年全年 |
| 南美春季种植需求旺盛 | 利多 | ★★★★ | 2026H1 |
| 全球粮食种植面积持平微增 | 利多 | ★★ | 持续性 |
| 中国国内春耕需求 | 利多 | ★★★ | 2026Q1-Q2 |
| 全球贸易摩擦加剧出口不确定性 | 利空 | ★★★ | 持续性 |

**供需边际预判**：

> **供需边际正在从"严重过剩"向"紧平衡"方向演进，但尚未到达"供不应求"。**
> 
> 2026年是供需格局改善的转折年。渠道补库需求将阶段性缓解过剩压力，但庞大产能基数意味着：一旦价格回升至全行业盈利，闲置产能将迅速释放，限制价格上涨空间。预计2026年行业将从"全行业亏损/微利"恢复至"龙头合理盈利、边际产能盈亏平衡"，全面进入"供不应求"尚需产能进一步出清。

### 4.2 扬农化工特有竞争优势的供需缓冲

在行业整体过剩的背景下，扬农化工处于相对优势地位：

1. **菊酯品类壁垒高**：拟除虫菊酯合成工艺复杂（涉及贲亭酸甲酯、功夫酸等关键中间体），进入壁垒远高于草甘膦等大宗品种。公司菊酯品种从2个扩展到50+个，中间体自成体系，品类齐全。
2. **卫生用药近乎垄断**：全球第二大卫药供应商，国内+东南亚市占率约70%，需求偏刚性，价格相对稳定。
3. **一体化成本优势**：关键中间体自供率高，"研产销"一体化运营，即使在全行业亏损时仍保持盈利。
4. **重资产+环保门槛**：四大原药基地均通过绿色工厂认证，环保合规成本构成对新进入者的天然壁垒。

### 4.3 压力传导特征分析

**传导路径**：全球粮食价格波动 → 农户种植意愿 → 渠道库存调整 → 农药需求 → 原药价格 → 公司毛利率

**波动放大效应（Bullwhip Effect）**：

```
终端农药需求 ±5%
    ↓（渠道安全库存调整）
渠道订购量 ±15-25%
    ↓（原药企业产能响应滞后）
原药价格波动 ±30-50%
```

2021-2024年完美演绎了这一放大机制：
- 2021年粮价上涨10% → 渠道恐慌性囤货 → 农药需求暴增 → 草甘膦暴涨至8万/吨（+300%）
- 2022-2024年渠道去库 → 需求"消失" → 草甘膦暴跌至2.34万/吨（-70%）
- 实际终端农药施用量波动远小于此

**传导时滞与弹性**：

| 传导环节 | 典型时滞 | 扬农化工的博弈筹码 |
| :--- | :---: | :--- |
| 原油/原料价格 → 原药成本 | 1-3个月 | 中间体自供，缓冲部分冲击 |
| 行业原药价格 → 公司出厂价 | 1-2个月 | 品类齐全可调结构，菊酯价格相对坚挺 |
| 出厂价变化 → 财报毛利率 | 1个季度 | 库存周期平滑短期波动 |
| 渠道补库 → 公司订单 | 即时（春耕旺季） | 刚需属性，农时不等人 |

**扬农化工的议价博弈能力**：
- **对上游**：公司通过集采、长期合约、中间体自供（关键中间体自成体系）实现对原料成本的较强控制。应付账款49.73亿（2026Q1），对上游占款能力突出。
- **对下游**：菊酯类产品（尤其是卫生用药）因品类壁垒和品牌效应，议价能力较强。但草甘膦等大宗品种面临充分竞争，议价能力有限。先正达集团作为第一大客户（占销售31.2%）构成较高客户集中度风险，但关联交易关系也提供订单稳定性。

### 4.4 毛利率反转的触发条件与时间判断

基于以上分析，毛利率恢复需要三个条件逐次满足：

| 条件 | 当前状态 | 预计触发时点 |
| :--- | :---: | :---: |
| ① 行业价格止跌企稳 | ✅ 已实现（2025Q2） | 2025年Q2 |
| ② 渠道补库启动，价格温和回升 | 🔄 进行中（2026Q1） | 2026年Q1-Q2 |
| ③ 过剩产能持续出清，供需回归平衡 | ⏳ 早期 | 2027-2028年 |

**扬农化工毛利率预测路径**：
- 2025年：21.99%（实际值，行业底部）
- 2026年：22.5-24.0%（温和恢复，受益于补库周期+成本推动提价）
- 2027年：23.5-25.5%（若产能出清持续+菊酯中间体短缺持续）

毛利率完全恢复到2022年25.7%水平需要产能进一步出清，短期内（2026年）概率较低，但方向确定向上。

## 五、强制复核确认

> **复核项1**：是否穿透并引用了终端产品的未来两年景气度预测？
> ✅ 已穿透至全球粮食生产体系。2026-2028年全球农药终端需求CAGR 4-7%，粮食种植面积持平微增，无需求崩塌风险。气候变化推升病虫害压力，结构性利好。

> **复核项2**：是否审计了跨期需求透支？
> ✅ 已审计。当前为去库终结后的正常补库启动，不存在需求透支。2021-2022年超级景气透支效应经3年消化已消除。

> **复核项3**：是否基于产能与终端需求分析了供需边际演变及对公司压力传导？
> ✅ 已完成。供需边际从"严重过剩"向"紧平衡"演进，扬农化工凭借菊酯品类壁垒和一体化优势处于相对有利位置。毛利率2026年有望温和恢复至22.5-24%。

## 六、模块二小结

中国农药行业经历了自2022Q3至2025Q2总计3年的深度下行周期，当前已确认**周期底部并于2026年进入复苏早期**。八大底部信号共振（价格超跌、库存见底、出口反转、行业提价、供给出清、成本支撑、盈利触底、时间充分），反转概率较高。

扬农化工的终端产品——农作物保护——具备不可替代的刚性需求属性。全球人口增长、气候变化、粮食安全战略共同构成长期需求支撑。但本轮复苏为成本驱动型而非需求爆发型，涨幅和斜率将比2021-2022年超级周期温和。

对扬农化工而言，毛利率已处于周期底部（21.99%），2026年有望恢复至22.5-24%区间。菊酯品类壁垒和卫生用药垄断地位提供了行业下行期的超额韧性，也将在上行期提供弹性。下一模块将分析公司在竞争格局中的地位与护城河宽度。
