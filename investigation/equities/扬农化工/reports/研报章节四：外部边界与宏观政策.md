> **红线一（模块一）**：出口占比58.4%，财务费用受汇率影响显著。资产负债率36.9%，财务结构极为保守，抗风险能力强。

## 二、宏观环境变量

### 2.1 利率与折现率

| 指标 | 现状 | 对扬农化工估值的影响 |
| :--- | :--- | :--- |
| 美联储利率 | 降息周期中（2025年已多次降息） | 全球资金成本下行→提升成长股估值中枢 |
| 中国LPR（5年期） | 3.6%（持续下行通道） | 降低DCF折现率→提高合理PE |
| 中国10年期国债收益率 | ~1.7%（历史低位） | 类债券资产（高分红）吸引力增强 |
| 实际利率环境 | 宽松 | 利好权益资产整体，尤其是现金流稳定的龙头企业 |

**估值含义**：当前低利率环境有利于提升农药龙头股的"合理PE"中枢。扬农化工派息募资比227%（累计），具备"现金牛+分红"属性，在低利率时代具备类债券的配置价值。

### 2.2 汇率损益（高优先级风险）

**最新汇率数据（2026年5月12日）**：
- 人民币兑美元中间价：**6.8426**（100美元兑684.26元人民币）
- 近期趋势：自4月底以来**持续升值**（从6.86升至6.84）

| 指标 | 数据 | 对公司的影响 |
| :--- | :--- | :--- |
| 境外收入占比 | 58.4%（68.08亿/2025年） | **人民币升值→利空出口**（每升值1%影响营收约0.5-0.6%） |
| 2025年财务费用 | -5,487万元（净收益） | 财务收益**同比下降**（汇兑损失增加） |
| 2026年Q1财务费用 | 约-2,800万元 | 延续汇兑损失压力 |
| 公司应对 | 开展外汇远期结汇业务 | 部分对冲，但无法完全消除 |

**汇率敏感性分析（2026年预测）**：

| 汇率情景 | 假设区间 | 对营收影响 | 对净利润影响 | 发生概率 |
| :--- | :---: | :---: | :---: | :---: |
| **升值情景** | 6.5-6.7 | -3~5% | **-10~15%** | 30% |
| **基准情景** | 6.8-6.9（当前） | 0 | 0 | 50% |
| **贬值情景** | 7.0-7.2 | +3~5% | **+5~10%** | 20% |

**核心判断修正**：
- 原判断：汇率是"次要扰动项" ❌ 
- **修正判断：人民币升值是实质性利空，需纳入盈利预测的核心变量** ✅
- 公司58%收入来自出口，境外业务毛利率（18%）显著低于境内（28%），人民币升值将直接压缩出口利润和汇兑收益
- 2025年年报已显示财务收益同比下降，2026年Q1延续此趋势

**应对策略评估**：
- 公司虽已开展远期结汇，但套保比例和效果有限
- 若人民币持续升值至6.7以下，公司盈利将面临实质性下修压力

### 2.3 通胀传导效率

| :--- | :---: | :--- | :--- |

## 三、地缘政治与贸易摩擦

### 3.1 中美关税战的量化影响

| 时间节点 | 关税措施 | 涉及产品 | 对扬农化工的影响 |
| :--- | :--- | :--- | :--- |
| 2018年起 | 113种原药加征25%，18种加征7.5% | 覆盖主要出口品种 | 已有6年适应期 |
| 2025年2月 | 额外加征10% | 全部中国商品 | 边际影响 |
| 2025年4月 | 额外加征34%+50% | 部分产品综合税率达129% | 影响有限（见下文） |

**量化评估**：
1. **对美出口占比低**：中国农药制剂对美出口仅占总额的6%（5.25亿美元/2024年），扬农化工出口以原药为主，但对美直接贸易额有限。
2. **全球替代难度高**：中国掌握全球70%原药产能，美国本土无原药生产能力，印度产能利用率仅66%且中间体依赖中国进口——"中国制造"在短期内不可替代。
3. **关税差异创造机会**：美国2025年8月对印度加征50%关税（涵盖联苯菊酯、高效氯氰菊酯等），而中国同类产品关税仅26.5-44%。在麦草畏、磺胺草唑、丙环唑等品种上，中国产品的关税优势分别为6、23.5、23.5个百分点。扬农化工的麦草畏（全球产能第一，2.5万吨）和菊酯类品种直接受益于这一关税差异。
4. **应对策略有效性**：公司年报明确表示"全力打造多层次市场、多元化客户，开拓国际新兴市场，努力促使全球贸易的地区平衡"——并非被动承受，而是主动分散风险。

**结论：中美贸易摩擦对扬农化工的影响为"中性偏正面"。** 高关税打击了中国低端出口商（加速出清），但对拥有品类壁垒和全球多元化布局的龙头企业，反而可能因竞争对手退出而获益。

### 3.2 印度竞争态势

| 维度 | 中国（扬农化工） | 印度（UPL/Tagros等） |
| :--- | :--- | :--- |
| 原药产能利用率 | ~70-80% | ~66% |
| 关键中间体自给率 | 高（全链条） | 低（依赖中国进口） |
| 菊酯类技术 | 全球领先 | 中低端 |
| 美国市场关税 | 26.5-44% | 50%（2025年8月起） |
| 2026年3月提价 | 率先提价5-15% | 同步跟涨 |

印度是扬农化工在菊酯领域最主要的国际竞争对手（Tagros、Meghmani、Heranba等），但受制于中间体依赖中国进口和更高的美国关税，其竞争优势正在削弱。扬农化工年报显示卫生用药东南亚市占率保持在70%左右，未受印度挑战，验证了其先发优势和品牌壁垒的有效性。

### 3.3 全球供应链重构下的机遇

"China+1"策略驱动跨国公司寻求供应链多元化，但农药行业的特殊性在于：
- 产能转移周期极长（建厂+审批+环保验收通常3-5年）
- 中国完整配套的化工基础设施短期内无法复制
- 环保合规成本在许多发展中国家不具备可行性

因此，供应链重构对扬农化工的威胁被高估。相反，公司通过自建海外登记（推进全球登记布局）+收购中化作物海外渠道，正主动向"中国品牌出海"转型。

## 四、监管与产业政策

### 4.1 "一证一品"——行业格局重塑的加速器

**政策内容**（2026年1月1日起实施）：
- 同一登记证号的农药产品必须标注同样的商标
- 委托加工产品不得标注受托方商标
- 互联网经营农药须取得许可证并备案

**产业影响量化**：

| 影响维度 | 估算量级 | 受益/受损方 |
| :--- | :---: | :--- |
| 市场上产品SKU数量 | 减少40-50% | 品牌制剂企业受益 |
| 依赖借证/套证的中小企业 | 15-20%面临退出 | 龙头市占率提升 |
| 农药登记证"含金量" | 大幅提升 | 拥有自主登记证的企业 |
| 行业品牌化进程 | 加速 | 扬农化工、诺普信等品牌企业 |
| 灰色产能退出 | 系统性清理 | 合规龙头企业 |

**扬农化工的受益路径**：
- 公司拥有数百个自主农药登记证（含海外登记），属于"一证一品"的最大受益群体
- 依赖借证的代工企业退出→菊酯市场份额进一步向扬农集中
- 品牌制剂业务（2025年收入14.98亿，毛利率29.66%）有望在清理灰色产品后获得量价双击

### 4.2 出口退税取消——"堵与疏"的产业升级

**政策内容**（2026年4月1日起实施）：
- 取消草铵膦、精草铵膦、敌敌畏等20余种农药原药/中间体的增值税出口退税（原9-13%）
- **制剂出口退税保留**

**扬农化工的差异化影响**：

| 品种 | 是否在取消清单中 | 扬农化工产能 | 影响评估 |
| :--- | :---: | :---: | :--- |
| 草甘膦 | 否（此前已取消） | 3万吨 | 已消化，无新增影响 |
| 草铵膦/精草铵膦 | 是 | 无（非主营业务） | 几乎无直接影响 |
| 菊酯类（功夫菊酯、联苯菊酯等） | **否** | 核心品种 | **不受影响** |
| 敌敌畏/敌百虫 | 是 | 极小或无 | 几乎无影响 |
| 杀菌剂（氟啶胺等） | 部分可能涉及 | 较小产能 | 影响可控 |

**核心判断：扬农化工在此次退税调整中几乎不受直接冲击，属于"政策避风港"。** 公司的菊酯核心品种未被列入取消清单，草甘膦退税此前已取消（已消化）。而竞争对手（利尔化学的草铵膦）面临退税取消冲击，间接强化了扬农化工的相对竞争优势。

长期来看，"保留制剂退税+取消原药退税"的政策导向与扬农化工"原药+制剂一体化"战略高度契合，有利于公司向下游高附加值环节延伸。

### 4.3 "十五五"规划中的农药产业定调

| 政策方向 | 具体内容 | 扬农化工的契合度 |
| :--- | :--- | :---: |
| 控总量 | 不新增农药生产企业，园区化聚集 | ★★★★★ 已入园 |
| 优结构 | 高毒农药退出，生物农药鼓励 | ★★★★ 菊酯为高效低毒品种 |
| 绿色化 | 新排放标准、碳足迹核算、零碳工厂 | ★★★★★ 辽宁优创已为零碳工厂 |
| 集约化 | CR10提升至45%+ | ★★★★★ 核心受益 |
| 数智化 | 5G工厂、智能制造 | ★★★★★ 优士+优嘉入选5G工厂名录 |

**制度红利量化**：
- 江苏优科创成公司首家"零碳工厂"→出口欧盟产品可获绿色溢价
- 入选国资委"科改"企业→政策资源、科研经费倾斜
- 农业农村部杀虫剂智能创制重点实验室→行业标准制定话语权
- FORUS体系6级（行业最高）→安全生产合规无忧

### 4.4 环保监管的差异化影响

作为4家工厂全部获评工信部"绿色工厂"的企业，扬农化工在环保趋严时代享有三重优势：
1. **不受停产限产影响**：辽宁优创、江苏优嘉获评重污染天气重点行业绩效A级企业，橙色/红色预警期间可自主减排
2. **合规成本已内化**：竞争对手面临新一轮环保改造投入时，公司已完成
3. **融资成本优势**：绿色工厂认证可享受绿色信贷利率优惠

- 次要风险是能源成本持续上涨（占原药成本17%）和全球贸易环境极端恶化
- 在中小农药企业面临三重政策压力（"一证一品"+"退税取消"+"排放新标"）的背景下，扬农化工作为合规龙头企业，相对竞争地位正在强化
- **关键监控**：人民币汇率走势（若升破6.8需重新评估盈利预测）

## 六、小结

扬农化工所处的外部环境呈现"压力与机遇并存，总体上利大于弊"的格局。

**压力面**：全球经济增长动能不足、贸易摩擦持续、能源成本上行、汇率波动加大。

**机遇面**（权重更高）：
1. 行业政策（"一证一品"、退税取消、"反内卷"）正在系统性淘汰竞争对手的低效产能
2. 美国对印度加征50%关税，为中国菊酯/麦草畏品种创造了关税套利空间
3. "十五五"规划的绿色化、集约化方向与公司既有优势高度契合
4. 低利率环境提升"现金牛+高分红"龙头的估值中枢
5. 扬农化工的4家绿色工厂+全链条合规在环保趋严时代构成排他性竞争优势

**核心投资含义**：外部边界变量并非扬农化工投资价值的减分项——相反，行业政策的系统性收紧正在加速"良币驱逐劣币"，公司作为行业"良币"的代表，将在这场淘汰赛中获得份额与定价权的双重提升。政策红利将在未来2-3年逐步兑现，构成估值上调的重要催化剂。
