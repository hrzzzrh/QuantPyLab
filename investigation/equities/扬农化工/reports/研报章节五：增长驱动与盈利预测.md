## 二、分业务 P×Q 模型

### 2.1 原药业务（收入占比62.6%，利润核心）

**2025年基线**：收入73.15亿，销量113,475吨，均价~6.45万元/吨，毛利率26.46%

**核心增长驱动力的证据锚定**：

| 驱动因素 | 来源 | 量化判断 |
| :--- | :--- | :--- |
| 辽宁优创一期二阶段达产 | 年报："高质量完成一期二阶段产品调试，尽快达产见效" | +5-8%产量（新增产能爬坡） |
| 优嘉新项目生产调试 | 年报："抓好优嘉新产品生产调试" | +2-3%产量 |
| 行业补库周期启动 | 模块二：美国库存同比转正，中国制剂出口+22%QoQ | +3-5%销量（渠道回补） |
| 行业价格温和回升 | 模块二：2026年3月行业集体提价5-15% | +3-5%均价 |
| 菊酯中间体供应偏紧 | 模块二/三：贲亭酸甲酯6万元/吨，功夫酸11.5万元/吨 | 支撑菊酯价格，但对扬农（自供）成本影响有限 |
| "一证一品"淘汰灰色菊酯产能 | 模块四：15-20%产能出清 | 菊酯市占率进一步提升 |

**原药业务 P×Q 情景推演**：

| 假设项 | 保守(Bear) | 中性(Base) | 乐观(Bull) | 证据来源 |

**证据索引**：
- [E1] 年报：2025年原药产量11.38万吨(+17.4%)，辽宁优创一期已投产即盈利，二阶段建设中
- [E2] 2026Q1财报：Q1营收34.67亿(+6.99%YoY)，公司Q1通常为淡季，全年增速通常高于Q1
- [E3] 模块二：2026年3月行业集体提价5-15%，草甘膦从2.34→2.66万元/吨(+13.7%)，菊酯类（高效氯氟氰菊酯11.5万/吨，联苯菊酯13.1万/吨）价格稳定偏强
- [E4] 年报：2025年产品价格损失2.78亿元，低基数效应下2026年价格同比改善确定性高
- [E5] 模块一：原药成本中原材料占56.7%，能源17.0%，折旧10.7%。中间体自供率高，能源成本上涨构成压力，但产能利用率提升摊薄单位折旧
- [E6] 模块三：菊酯品类壁垒高、竞争格局优，议价能力优于大宗品种

### 2.2 制剂业务（收入占比12.8%，品牌溢价）

- 创制品种商业化（氟螨双醚预计2026年上市）→制剂端增长新动能

## 六、核心假设的确定性评分

| 核心假设 | 确定性(1-5) | 理由 |
| :--- | :---: | :--- |
| 辽宁优创产能爬坡贡献增量 | **4** | 一期已投产即盈利，二期完成中交，产能释放为确定性事件 |
| 行业原药价格温和回升 | **3** | 多家企业已提价5-15%，但持续性取决于供需再平衡进度 |
| "一证一品"释放制剂增长 | **4** | 政策已落地实施，灰色产能退出正在进行 |
| 全球补库周期持续 | **3** | 美国库存数据已转正，出口数据改善，但关税政策存变数 |
| 公司菊酯市占率继续提升 | **4** | 中小对手出清+技术壁垒+印度关税劣势，趋势确定 |
| 毛利率恢复至22.9% | **3** | 价格回升确定，但能源成本上行构成对冲 |
| 创制药氟螨双醚2026年上市 | **3** | 公司表示"力争2026年上市"，新药上市节奏有不确定性 |
| EPS达4.01元 | **3** | 基于量+价+费用率综合判断，中等确定性 |

**综合确定性评分：3.4/5**——核心增长逻辑清晰且证据链较充分，但外部变量（行业价格恢复速度、能源成本）构成中等不确定性。

## 七、核心分歧与风险提示

1. **上行风险（超预期因素）**：
   - 行业价格恢复速度快于预期（若印度产能持续收缩+南美需求超预期）
   - 创制药氟螨双醚上市后快速放量
   - 中美关税谈判取得突破，贸易环境改善

2. **下行风险（低于预期的因素）**：
   - 2026H2补库结束后需求回落（牛鞭效应反向）
   - 能源成本超预期上涨侵蚀毛利率
   - 全球贸易摩擦进一步升级影响出口
