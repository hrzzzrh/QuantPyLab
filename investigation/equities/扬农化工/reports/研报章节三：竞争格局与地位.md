# 研报章节三：竞争格局与地位

**研究日期**：2026年05月24日

---

## 一、核心逻辑继承（来自前序模块）

> **红线一（模块一）**：毛利率2022年25.68%→2025年21.99%，ROE从23.18%降至11.69%。公司原药品种近70个，菊酯为核心，产能超10万吨/年。辽宁优创一期2025年投产即盈利。
>
> **红线二（模块二）**：行业处于周期底部→复苏早期转换阶段，供需边际改善，但全面复苏仍需产能进一步出清。菊酯因品类壁垒和卫生用药刚需，价格下行压力小于草甘膦等大宗品种。
>
> **本模块核心问题**：扬农化工的竞争壁垒能否在行业复苏中转化为超额收益？

## 二、竞争格局扫描

### 2.1 行业金字塔与扬农化工的位置

全球农药行业呈清晰的三梯队格局：

| 梯队 | 代表企业 | 2025年植保营收 | 核心优势 |
| :--- | :--- | :---: | :--- |
| 第一梯队（国际巨头） | 先正达、拜耳、科迪华、巴斯夫、UPL | $80-170亿 | 创制药专利+种子+数字农业生态 |
| 第二梯队（中国龙头） | 安道麦、润丰股份、扬农化工、利尔化学 | ¥100-330亿 | 非专利药规模化+全品类+全球化渠道 |
| 第三梯队（区域/单品类） | 江山股份、利民股份、泰禾股份等 | ¥10-70亿 | 特色品种+成本优势 |

**关键发现**：扬农化工2025年营收118.70亿（+13.8%），增速在百亿级企业中位列第一，正在拉近与润丰股份的差距。

### 2.2 行业集中度演变

2025年农药行业CR10从34%快速提升至45%，规模以上企业数量持续缩减但平均产能规模扩大。核心驱动力：

1. **"一证一品"政策**（2026年1月实施）：同一登记证产品须标注同一商标，直接打击"借证"、"套证"的灰色产能，中小代工企业加速出清。
2. **"反内卷"三年行动**：中国农药工业协会主导，推动行业从"价格战"转向"有序竞争"。
3. **环保合规成本攀升**：江苏省2025年本《化工产业结构调整限制和淘汰目录》将草甘膦等27种农药列为限制类，地方政府严控新产能审批。
4. **并购整合提速**：利尔化学收购山东汇盟、泰禾股份收购瑞德丰、利民股份收购巴西登记证——头部企业通过并购加速做强。

**对扬农化工的影响**：行业集中度提升有利于龙头企业。扬农化工作为CR10核心成员，有望在"剩者为王"的淘汰赛中获取份额红利。

### 3.2 分品类竞争格局

**① 杀虫剂（菊酯类）——扬农化工的"护城河"**

| 企业 | 菊酯产能 | 市占率（中国） | 技术特征 |
| :--- | :---: | :---: | :--- |
| **扬农化工** | 农用菊酯产能占全国23%（功夫菊酯）+18%（联苯菊酯），卫药2600吨 | 卫药~70%，农用~25% | 从基础化工到原药全链条，唯一自主中间体合成 |
| 印度企业（Tagros, Meghmani, Heranba） | 合计产能较大 | 全球~30% | 成本优势（劳动力），但中间体依赖中国进口 |
| 跨国公司（FMC, Syngenta, BASF, Bayer） | 专利菊酯品种 | 高端市场 | 品牌+登记壁垒 |
| 国内其他（北京颖泰等） | 产能较小 | <5% | 非全链条，成本劣势 |

**菊酯竞争壁垒的本质**：
- **工艺壁垒**：拟除虫菊酯合成路径长（10-15步反应），涉及贲亭酸甲酯、功夫酸、醚醛等关键中间体。2025年底贲亭酸甲酯因上游供应缩减涨至6万元/吨，功夫酸涨至11.5万元/吨，中间体自主供应成为核心竞争分水岭。
- **登记壁垒**：卫生用药需各国卫生部/农药管理部门的严苛审批，通常耗时3-5年，先发优势极强。
- **品牌壁垒**：扬农化工卫生用药"是国内外众多下游企业的首选品牌"——年报原话，品牌黏性高。

**② 除草剂——"被动参与者"，利润贡献有限**

| 品种 | 扬农化工产能 | 全球/中国排名 | 竞争烈度 |
| :--- | :---: | :---: | :---: |
| 草甘膦 | 3万吨 | 中国第6-8位 | ★★★★★（极度激烈） |
| 麦草畏 | 2.5万吨 | 全球第一 | ★★★（供给集中） |
| 烯草酮 | 产能不详 | — | ★★★★ |
| 硝磺草酮 | 收购中化作物获得 | — | ★★★ |

除草剂是扬农化工的"规模补充器"而非"利润发动机"。草甘膦面临兴发集团（磷矿一体化）、新安股份（有机硅协同）、福华通达等低成本竞争对手的挤压。麦草畏和硝磺草酮等差异化品种提供了一定的缓冲。

**③ 杀菌剂——"战略增长极"**

扬农化工通过自建优嘉项目+收购宝叶化工，形成了代森锰锌+吡唑醚菌酯+丙环唑三大全球前15品种组合，并在SDHI类杀菌剂（琥珀酸脱氢酶抑制剂）等新型方向布局。杀菌剂毛利率通常高于除草剂，是公司品类升级的重要方向。

### 3.3 竞品资本开支周期对比

| 企业 | 近年主要资本开支 | 在建工程（2024/2025） | 达产节奏 |
| :--- | :--- | :---: | :--- |
| 扬农化工 | 辽宁优创一期（葫芦岛） | 16.15亿→12.45亿 | 2025年投产即盈利，一期二阶段2026年调试 |
| 利尔化学 | 荆州三才堂1万吨精草铵膦 | 9.97亿 | 建设中 |
| 润丰股份 | 全球化制剂产能分散布局 | 3.81亿 | 渐进式 |
| 新安股份 | 多项目群（磷系+硅基+草铵膦） | 45.04亿 | 分阶段 |

**核心洞察**：扬农化工的资本开支高峰已过（辽宁优创一期一期阶段完成），2026年进入"产能爬坡→利润释放"阶段。而竞争对手如新安股份、和邦生物仍在大规模建设中，面临更大的资本开支压力和转固后的折旧压力。

## 四、护城河识别（Moat Analysis）

依据晨星护城河框架，扬农化工的五大竞争壁垒：

### 4.1 无形资产（品牌+专利+监管许可）——★★★★☆（宽护城河）

1. **卫生用药品牌**：国内+东南亚市占率~70%，全球第二大卫药供应商，客户黏性极强。该品类属于"类消费品"，下游对价格敏感度低，品牌溢价显著。
2. **农药登记证壁垒**：拥有数百个国内外农药登记证，海外登记（尤其是巴西5-8年审批周期）构成极高的时间壁垒。
3. **创制专利**：两个创制品种获ISO通用名（四氯虫酰胺、乙唑螨腈），全新机理创制药氟螨双醚预计2026年上市。氯氟醚菊酯等自主创制品种市占率稳中有升。
4. **国家级平台**：创成农业农村部杀虫剂智能创制重点实验室，入选国资委"科改"企业——这些机构的隐性价值在于优先获取政策资源、行业标准制定话语权。

### 4.2 成本优势——★★★★☆（宽护城河）

1. **中间体一体化**：国内唯一从基础化工原料到菊酯原药全链条企业，关键中间体（贲亭酸甲酯、功夫酸）自供比例高。2025年底中间体价格上涨时，竞争对手被迫高价采购，而扬农化工享受成本锁定优势。
2. **规模效应**：原药产能>10万吨/年，品类近70个，劳动生产率国际领先。
3. **制造工艺Know-How**：连续11种高危工艺工程化能力，本质安全水平行业领先；2025年降本9105万元，原药工厂大部分消耗指标同比持平或下降。
4. **绿色工厂集群**：4家原药工厂均为工信部绿色工厂（江苏优士、江苏优嘉、沈阳科创、辽宁优创），在环保监管趋严时代构成合规成本优势。

### 4.3 转换成本——★★★☆☆（中等护城河）

1. 农药原药客户切换需重新登记、做药效试验，时间成本1-3年。跨国农化公司对供应商稳定性要求极高，合作关系一旦建立不易更换。
2. 制剂端品牌效应：农户对"扬农"品牌认知度高，渠道和终端黏性较好。
3. 但大宗品种（如草甘膦）属于标准化产品，转换成本较低。

### 4.4 网络效应——★★☆☆☆（不适用）

农药行业不具备典型的网络效应（用户越多价值越大），但卫生用药在东南亚的渠道密度构成一定的"覆盖壁垒"。

### 4.5 有效规模——★★★☆☆（中等护城河）

中国农药行业严格的环保和安全生产监管形成了新进入者壁垒。江苏等核心省份限制新产能审批，存量优质产能具备稀缺性价值。

**综合护城河评级：宽护城河（Wide Moat）**。菊酯品类的技术壁垒+卫生用药的品牌垄断性+全链条成本优势，构成了竞争对手难以在短期内复制的综合竞争壁垒。

## 五、产业链定价权分析

### 5.1 对上游：议价能力强

| 指标 | 数值 | 解读 |
| :--- | :---: | :--- |
| 应付账款+应付票据 | 49.73亿（2026Q1） | 对上游占款能力突出 |
| 应付 vs 应收 | 49.73亿 vs 39.83亿（+9.90亿） | 净占用上游资金，OPM策略成功 |
| 前五大供应商集中度 | 19.4%（含关联方） | 供应链分散，无单一依赖风险 |
| 中间体自供率 | 菊酯关键中间体自成体系 | 降低对外部供应商的依赖 |

关联方（先正达、中化国际等）采购占总采购额的12.35%，构成稳定的战略供应关系。

### 5.2 对下游：品类分化明显

| 品类 | 议价能力 | 证据 |
| :--- | :---: | :--- |
| 卫生用药（菊酯） | ★★★★★ 极强 | 市占率70%，客户首选品牌，定价权显著 |
| 农用菊酯 | ★★★★ 较强 | 品种齐全（50+种），技术壁垒高 |
| 草甘膦等大宗除草剂 | ★★ 较弱 | 标准化产品，跟随市场价格 |
| 制剂（品牌） | ★★★★ 较强 | 品牌溢价，渠道掌控力较强 |
| 贸易业务 | ★ 极弱 | 通道业务，毛利率仅7.27% |

**客户集中度风险与对冲**：
- 先正达集团为第一大客户（占销售31.2%，关联交易），存在较高的单一客户依赖度。但由于双方同属中国中化体系，合作关系具有战略性质，集中度风险被关联属性部分对冲。
- 前五大客户销售额37.47亿（占31.57%），集中度较高但尚在可控范围（<50%红线）。

## 六、行业供给侧出清与扬农化工的受益路径

### 6.1 中小产能出清加速

"一证一品"+"反内卷"三年行动+出口退税取消三重政策叠加，正在系统性清除行业"劣币"：

- 依赖出租/借用登记证的中小企业面临合规困境，过渡期后将被清退
- 环保不达标产能加速关闭（江苏限制目录2025年本）
- 出口退税取消（草铵膦等）挤压低附加值出口商利润空间

### 6.2 扬农化工的受益逻辑

1. **市场份额提升**：落后产能退出→市场供给缺口→龙头填补。2025年扬农化工原药产量11.38万吨（+17.4%），远超行业产量增速（+12%），已在收获份额红利。
2. **价格环境改善**：低效产能出清→价格竞争烈度下降→毛利率中枢上移。2026年3月行业集体提价5-15%即为早期信号。
3. **政策资源倾斜**：作为"科改"企业+农业农村部重点实验室+中化央企体系，在行业标准制定、项目审批、环保豁免等方面享有"制度红利"。

## 七、竞争地位小结

| 维度 | 评级 | 说明 |
| :--- | :---: | :--- |
| 市场份额 | ★★★★ | 中国农药TOP4，菊酯品类绝对第一 |
| 技术壁垒 | ★★★★★ | 菊酯全链条自主合成，两个ISO创制品种 |
| 品牌壁垒 | ★★★★★ | 卫药市占率70%，全球第二 |
| 成本优势 | ★★★★ | 中间体自供+10万吨规模+绿色工厂 |
| 规模效应 | ★★★★ | 近70个原药品种，产能超10万吨 |
| 客户黏性 | ★★★★ | 跨国农化公司长期合作+品牌制剂 |
| 对上游议价 | ★★★★ | 净占用上游资金9.9亿元 |
| 对下游议价 | ★★★☆ | 菊酯强、大宗弱，整体偏强 |
| 行业出清受益弹性 | ★★★★★ | 龙头中最受益于中小出清 |

**综合评分：在A股农药板块中，扬农化工的竞争壁垒位列第一梯队，仅次于（或不逊于）润丰股份（全球化渠道）、利尔化学（草铵膦龙头），但品类宽度和抗周期能力更优。**

在"剩者为王"的行业淘汰赛中，扬农化工凭借菊酯技术垄断+全品类布局+中化体系资源，具备穿越周期的综合竞争优势。其竞争地位将在行业复苏中转化为利润弹性——这也是下一模块（盈利预测）的关键嵌入变量。
