## 二、趋势架构分析

### 2.1 多周期趋势定位

| 周期 | 均线状态 | 当前趋势 |
| :--- | :--- | :--- |
| 短期（MA5/10/20） | 价格在MA5(70.98)/MA10(71.48)/MA20(74.39)下方 | **下降** |
| 中期（MA60/120） | 价格在MA60(75.70)/MA120(72.01)下方 | **下降** |
| 长期（MA250） | 价格在MA250(67.49)上方，**仅存一根长期均线支撑** | **上升（但考验中）** |

**趋势诊断：短中期空头排列，长期趋势临界。**

过去60个交易日，股价从85.54元（2026年3月高位）持续回调至68.21元，累计跌幅-20.3%。当前价格跌穿所有中短期均线，仅依靠年线（MA250=67.49元）勉强维持长期上升趋势。年线位置67.49元与当前价格68.21元仅差1.1%，**这是多空分水岭级别的关键支撑位。**

### 2.2 结构定位

将2024年以来的价格走势分为三个阶段：

```
阶段一（2024.01-2024.09）：探底阶段
  价格从60元区域持续下跌，2024年9月触及近5年最低46.81元
  特征：行业深度衰退+大盘走弱共振，成交量萎缩，市场极度悲观

阶段二（2024.09-2025.03）：第一波估值修复
  价格从46.81元反弹至约77元（+64.5%）
  特征：行业去库尾声+市场预期反转，量能温和放大

阶段三（2025.03-2025.09）：区间整理
  价格在48-77元区间震荡，构建中期底部平台
  特征：等待基本面确认，波动收窄

阶段四（2025.09-2026.03）：第二波上攻
  价格从65元区域拉升至85.54元（+31.6%），创2023年以来新高
  特征：2025年三季报验证业绩反转+行业提价预期，放量突破

阶段五（2026.03-至今）：回调修正
  价格从85.54元回落至68.21元（-20.3%）
  特征：获利回吐+Q1财报确认温和增长(非爆发性)，成交量恢复至均值
```

**结构判断**：当前处于"牛市回调"形态——长期趋势（MA250）仍向上，短中期调整属于对前期涨幅（46.81→85.54，+82.7%）的正常修正。回调幅度（-20.3%）在技术性调整的合理范围内（通常牛市回调幅度为前期涨幅的1/3至1/2，即-21%至-33%）。

## 五、未来走势推演

### 5.1 短期走势（1-3个月）

**核心博弈**：年线（67.49元）的得失。

**情景A（基准，概率50%）**：年线获得支撑，震荡筑底
- 价格在67-72元区间反复测试MA250
- 磨底时间2-4周
- 随后在基本面催化（2026年半年报预告、行业价格数据改善）下，放量突破MA20/60
- 反弹目标：75-80元（前期平台区）

**情景B（乐观，概率25%）**：基本面催化驱动V型反转
- 2026Q2财报超预期（行业提价效果显现）
- 价格快速收复72元以上，挑战前高85.54元
- 反弹目标：85-95元

**情景C（悲观，概率25%）**：年线失守，二次探底
- 年线67.49元被有效跌破（连续3日收盘低于）
- 下一支撑位在65元（2025年9月平台）和58元（2024年10月低点）
- 若年线失守，短期应转为防御，等待重新企稳信号

**短期关键点位**：
- 止损参考位：66.50元（年线下方1.5%，跌破意味着长期趋势转弱）
- 加仓位：站稳MA20（74.39元以上）且放量确认

### 5.2 长期走势（6-12个月以上）

**大级别形态判断：上升通道维持，回调是"黄金坑"。**

支撑长期看好的技术逻辑：
1. **后复权MA250稳步上行**：当前后复权年线255.16元，且斜率向上，长期趋势未被破坏
2. **底部逐级抬高**：2024年9月46.81元→2025年3月48.26元→当前若67-68元区域企稳，底部抬高趋势延续
3. **前期涨幅已充分消化获利盘**：从46.81元到85.54元累计涨幅82.7%，当前回调20.3%（至68.21元），回调比例约24.6%（20.3/82.7），符合牛市正常回调特征
4. **成交量在下跌中萎缩**：未出现恐慌性放量杀跌，说明持仓者惜售

**12个月目标推演**：

| 情景 | 目标价（不复权） | 后复权对应价 | 催化剂 |
| :--- | :---: | :---: | :--- |
| 保守 | 72-75元 | ~377-392元 | 行业企稳，温和修复 |
| 中性 | 85-91元 | ~445-476元 | 盈利恢复+政策红利兑现 |
| 乐观 | 100-110元 | ~523-575元 | 行业反转超预期+牛市共振 |

中性目标85-91元与技术面目标不矛盾——前高85.54元是第一个重要阻力位，突破后将打开新的上行空间（下一个阻力为2023年高点111.09元）。

**技术面结论**：短期承压但长期趋势未破坏。68元附近的年线支撑是关键的"试金石"——若企稳，是中线布局的有利位置（对应模块六目标价86.09元有26%上行空间）；若跌破，需等待65元以下出现企稳信号。当前缩量回调至年线的技术形态，在基本面改善的背景下，更倾向于解读为"牛市回调"中的布局机会，而非趋势反转。
