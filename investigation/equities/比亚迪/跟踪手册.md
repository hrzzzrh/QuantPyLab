## 三、逻辑证伪警报（Invalidation Triggers）

> **以下信号出现时，必须严肃评估是否应触发止损或下调评级。这些信号比普通风险更具判别性。**

### 🛑 一级警报（触发即应大幅减仓/清仓）

| # | 证伪信号 | 失效的核心逻辑 |
| :---: | :--- | :--- |
| 1 | **比亚迪国内新能源市场份额连续两个季度 < 17%** | 「龙头地位稳固」假设被证伪 |
| 2 | **海外季度销量同比增速 < 10% 且环比下降** | 「海外是第二增长曲线」假设被证伪 |
| 3 | **欧盟对所有中国进口电动车（含插混）加征 50%+ 关税** | 「国际化战略」的盈利基础被摧毁 |
| 4 | **单季度出现净亏损（归母净利润 < 0）** | 「垂直整合的成本韧性」假设被证伪 |
| 5 | **发生因车辆安全/质量导致的重大召回（>50 万辆）或致命事故（涉及智驾系统）** | 「技术领先/安全可靠」的品牌叙事崩塌 |

### ⚡ 二级警报（触发应重新审视仓位，考虑减仓）

| # | 证伪信号 | 失效的核心逻辑 |
| :---: | :--- | :--- |
| 6 | **毛利率连续两个季度 < 16%** | 「成本护城河抵御价格战」假设被削弱 |
| 7 | **海外建厂项目出现重大延误或取消**（如匈牙利工厂投产延至 2027） | 「全球化扩张执行能力」受到质疑 |
| 8 | **连续两次财报中研发费用率 > 8.5% 且净利润率 < 3%** | 「研发投入能转化为利润」的逻辑未被数据支持 |
| 9 | **核心高管（王传福）意外变动或大规模研发人才流失** | 「战略定力和技术传承」的基础动摇 |
| 10 | **信用评级被下调或出现重大融资困难** | 「财务稳健性」假设被破坏 |

### 📊 三级警报（关注信号，暂不需行动但需密切监测）

| # | 证伪信号 | 说明 |
| :---: | :--- | :--- |
| 11 | 存货持续攀升（增长 >30% YoY）且存货周转天数 > 120 天 | 渠道库存过度积压 |
| 12 | 应付账款连续两个季度下降 >15% | 供应商信心动摇 |
| 13 | 吉利/零跑/小米三家中任意两家市占率超过比亚迪的 60% | 竞争格局根本性改变 |
| 14 | 碳酸锂价格突破 25 万/吨且无回落迹象 | 成本端结构性恶化 |
