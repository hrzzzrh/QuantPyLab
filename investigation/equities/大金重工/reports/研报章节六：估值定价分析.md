# 研报章节六：估值定价分析

**研究日期**：2026年05月09日
**研究对象**：大金重工 (002487.SZ)
**当前股价**：82.98元（不复权市价）| 市值：529.2亿 | PE(TTM)：40.5x | PB：6.08x
**目标年份**：2026年 | 总股本：6.377亿股

---

## 一、价值锚定：适配性估值方法选择

### 1.1 方法论筛选逻辑

公司的核心属性决定了估值方法的选择：

| 公司属性 | 对估值方法选择的影响 |
|:---|:---|
| 高成长（营收+70%，净利+50%预期） | PE、PEG适用 |
| 重资产制造（固定资产+在建>50亿） | PB-ROE适用、EV/EBITDA适用 |
| 盈利质量高（OCF>NP）、产能爬坡中 | DCF可作为校验工具 |
| 海外出口为主，汇率影响显著 | 需在盈利预测中体现，不单独估值 |
| 含发电/航运等不同属性业务 | SOTP可作为交叉验证 |
| 业务转型期（国内→海外、FOB→DAP） | **历史PE分位参考价值有限** |

### 1.2 估值锚点规则

当前为2026年5月（Q2），根据框架"当年Q1-Q3：锚定当年EPS"规则，估值锚定为**2026年全年预测EPS**。

### 1.3 方法论组合

**主方法：PE估值**（成长型制造业最通用的定价语言，市场参与者广泛使用）

**交叉验证：PB-ROE模型**（重资产制造业，揭示PB与ROE的隐含一致性）

**辅助校验：PEG**（确保增长与估值的匹配度）

**不适用方法及理由：**
- **DCF**：公司正处于重资本投入期（FCF为负），DCF需要预测到2029+的稳态期，变量过多导致精确性丧失
- **PS（市销率）**：毛利率从17%升至39%，不同时期的PS不可比

## 二、PE估值法（主方法）

### 2.1 历史PE分位的参考与扬弃

从系统数据提取近5年（2021.05-2026.05）PE分位：

| 分位数 | PE值 | 对应历史阶段 |
|:---|:---:|:---|
| P5 | 10.4x | 2021年6月：高杠杆+国内为主+钢材涨价压利润 |
| P25 | 29.4x | 2022年Q1-2：海外转型初期 |
| P50（中位数）| 35.9x | — |
| P75 | 44.1x | 2023年初估值峰值 |
| P95 | 51.8x | 极值期 |
| **当前** | **40.5x（TTM）** | **约P70分位** |

**历史分位的缺陷**：过去5年的PE样本中，前3年（2021-2023）的公司处于完全不同的经营状态——高杠杆（54.9%→32.4%）、低毛利率（16.7%→23.2%）、内需主导。直接套用历史分位均值会低估公司在当前高质量增长期应享有的估值溢价。

**更合理的对标**：公司当前的毛利率（31-39%）、ROE（14-18%）、海外收入占比（74.5%）与Sif Group（欧洲同业）更为接近。Sif的历史PE（2021-2024）在**25-50x**区间波动，当前Forward PE约30-35倍（基于Sif 2026E EBITDA €135M指引估算）。

### 2.2 基准PE的确定

采用"三维交叉法"确定大金重工的合理PE中枢：

| 定价维度 | 参考值 | 对大金的适用性 |
|:---|:---:|:---|
| A股风电设备行业中位数 | ~25x（含大量纯内需低毛利企业） | 偏低——大金的海外高毛利结构显著优于行业 |
| 大金自身历史中位数 | ~36x | 合理起锚点——反映市场对公司转型的认可 |
| 可比海外同业（Sif/EEW） | 30-35x Forward PE | 上限参考——欧洲利率环境不同 |
| 成长性校准（PEG=1） | 50%增长→50x | 上限过于激进 |

**基准PE设定：32-38x 2026E EPS**

- 下限32x：假设市场对"中国企业在欧洲的可持续性"给予一定折价
- 中枢35x：假设公司维持当前估值水位（与历史中位数一致）
- 上限38x：假设市场给予"全球化龙头"的估值溢价

### 2.3 基准市值区间

| 情景 | 2026E EPS | PE区间 | 目标市值（亿） | 目标股价（元） |
|:---|:---:|:---:|:---:|:---:|
| Bear | 1.96元 | 28-32x | 353-401 | 55.3-62.9 |
| **Base** | **2.59元** | **32-38x** | **528-627** | **82.8-98.3** |
| Bull | 3.45元 | 30-36x | 658-789 | 103.2-123.8 |

**关键发现**：当前市值529亿恰好位于Base情景的PE 32x位置——即市场定价基本反映了本报告的Base预测，但尚未计入任何估值溢价。若以中枢35x计算，合理市值为579亿（对应股价90.8元），较当前有**9.4%的上行空间**。

## 三、PB-ROE模型（交叉验证）

### 3.1 模型逻辑

PB-ROE模型的隐含关系：`合理PB = (ROE - g) / (r - g)` 或更直观的：**高且可持续的ROE应享有更高的PB**。

### 3.2 大金重工ROE预测

| 指标 | 2024A | 2025A | 2026E Base | 2027E（外推）|
|:---|:---:|:---:|:---:|:---:|
| ROE（加权）| 6.68% | 14.18% | **~18.5%** | ~22% |
| 每股净资产（元）| 11.40 | 12.98 | **~15.0** | ~18.0 |
| 净利润（亿）| 4.74 | 11.03 | **16.5** | ~24 |
| 净资产（亿，年末）| 72.72 | 82.80 | **~96** | ~115 |

注：2026E净资产 = 82.80 + 16.5 - 分红(约2亿) + 其他变动 ≈ 96亿

### 3.3 PB-ROE合理性检验

以18.5%的ROE和10%的权益成本（r），在零增长假设下：`合理PB = ROE/r = 18.5%/10% = 1.85x`

在5%永续增长假设下：`合理PB = (ROE-g)/(r-g) = (18.5%-5%)/(10%-5%) = 2.70x`

在8%永续增长假设下（乐观）：`合理PB = (18.5%-8%)/(10%-8%) = 5.25x`

**当前实际PB = 6.08x**，显著高于PB-ROE模型的静态合理值。这意味着：
- 市场定价已隐含ROE持续提升至22%+（超越2026E的18.5%）
- 或者市场对公司的长期增长率给予了8%+的永续假设
- PB的相对高估提示我们：**PE估值可能比PB估值更适合当前阶段，因为PE更能捕获中短期的成长性，而PB受重资产属性拖累会高估合理值**

以2026E每股净资产15.0元、PB 5.5-6.5x（当前水平）计算，对应股价82.5-97.5元，与PE法的中枢价位（90.8元）大致吻合。

## 四、PEG校验

| 情景 | 2026E净利增速 | 2026E PE（以当前市值计算） | PEG |
|:---|:---:|:---:|:---:|
| Bear | +13% | 42.3x | 3.3 — **偏高** |
| Base | +50% | 32.0x | **0.64** — 合理偏低 |
| Bull | +100% | 24.0x | **0.24** — 极低 |

**PEG校验结论**：以PEG=1为合理基准，Base情景下的PEG=0.64意味着当前估值对50%的利润增速来说**具有吸引力**。Bear情景下PEG=3.3，说明若增长不及预期，估值将承受较大压力。

但需注意PEG的局限性：PEG=0.64可能是市场对公司"增长可持续性"的折价（即55%增速不可持续，而非市场不理性）。

## 六、全变量对冲审计矩阵（强制性输出）

基于Base情景基准市值535亿（PE 32x * 16.5亿）为"底色"，进行估值影响力过滤和调节。

### 6.1 估值影响力过滤——审慎思考

**剔除的"伪因子"（影响估值=0）：**
- 曹妃甸基地规模大、技术先进 → 已在EPS中体现（出货量），不影响PE
- 自有码头降低运输成本 → 已在毛利率中体现，不影响PE
- 公司在手订单100亿 → 已在营收预测中体现，不影响PE
- 欧洲海上风电装机加速 → 已在行业增速中体现（Q的驱动），不影响PE

**真正能改变估值中枢的因子（需进入矩阵）：**

| 因子类别 | 能否改变PE？| 审理逻辑 |
|:---|:---|:---|
| "唯一具备XXL出口能力的亚太企业"→稀缺性溢价 | **可以** | 不可替代性→确定性溢价 |
| 欧盟对单桩加征关税的长期风险 | **可以** | 地缘折价→"中国企业在欧"的持续性质疑 |
| DAP"一站式"模式的黏性 | **可以** | 客户锁定→收入可预见性溢价 |
| 大客户集中风险（RWE+Nordseecluster占大头）| **可以** | 集中度折价→风险溢价要求更高 |
| 公司管理层战略定力（十年磨一剑转型）| **可以** | 治理溢价→长期主义文化 |

### 6.2 审计矩阵

**审计公式**：修正目标市值 = 基准市值 × (1 + Σ调节幅度)

| 审计维度 | 核心审计因子 | 估值影响力过滤 | 调节建议 | 深度理由 |
|:---|:---|:---|:---:|:---|
| **内部：企业底层特质** | **战略定力与转型执行力**：公司用3年时间完成从"国内低端制造"到"欧洲高端供应链核心"的断层式转型，期间主动放弃短期收入增长、忍受利润下滑——这种战略纪律在A股制造业中极为稀缺 | 通过。这种"内生蜕变"能力意味着公司对"第二曲线下降/第三曲线启动"具备复制能力，降低了长期不确定性——这是估值溢价的真实来源 | **+5%** | 长期确定性提高→PE应从32x向上修正；类似逻辑可参照Sif从石油钢管企业转型为风电单桩龙头的估值重塑过程 |

| **宏观：地缘与外部约束** | **欧盟贸易保护的"达摩克利斯之剑"**：OWFA游说可能导致未来1-3年内对单桩加征7-20%关税（模块四）。若落地，将大幅压缩盈利能力甚至改变竞争格局 | **通过。这是影响估值中枢的极少数宏观因子之一。** 它不是一次性的EPS冲击（那应进E），而是关乎"公司能否长期在欧洲市场存在的根本性质疑"——这会系统性压低市场愿意赋予的PE倍数 | **-8%** | 这种风险类似于中概股面临的"VIE结构风险"——只有当它爆发时才影响EPS，但不确定性本身就已经折现在估值倍数中。大金当前的"中国企业在欧"标签无法完全消除这一折价 |

| **综合调节系数** | — | — | **-3%** | 战略稀缺性(+5%) 与地缘折价(-8%)对冲后净调节-3% |

### 6.3 修正后的目标市值

**修正目标市值 = 535亿 × (1 - 3%) = 519亿**

## 七、确定性与胜率矩阵

### 7.1 三态逻辑加权

| 情景 | 2026E EPS | 发生概率 | 目标市值（修正PE 31.5x）| 目标股价（元）|
|:---|:---:|:---:|:---:|:---:|
| 保守（Bear）| 1.96元 | **25%** | 394亿 | 61.7 |
| 中性（Base）| 2.59元 | **55%** | **519亿** | **81.4** |
| 乐观（Bull）| 3.45元 | **20%** | 693亿 | 108.7 |

**概率加权综合目标价** = 61.7×0.25 + 81.4×0.55 + 108.7×0.20 = **81.9元**

### 7.2 盈亏比计算

现值：82.98元 | 目标价（概率加权）：81.9元 | 保守价：61.7元

`盈亏比 = (81.9 - 83.0) / (83.0 - 61.7) = -1.1 / 21.3 = -0.05`

**盈亏比为负**——当前价格已略高于概率加权目标价，短期内上涨空间有限，但下行保护在61.7元（约25%下行空间）。

若以Base情景直接计算：`盈亏比 = (81.4 - 83.0) / (83.0 - 61.7) = -1.6 / 21.3 = -0.08`

### 7.3 乐观视角的盈亏比

若市场给予更高的确定性溢价（地缘折价收窄至-3%），修正PE升至34x：

| 情景 | 目标价 |
|:---|:---:|
| Bear | 66.6元 |
| Base | **88.1元** |
| Bull | 117.3元 |
| 概率加权 | **87.0元** |

此时盈亏比 = (87.0 - 83.0) / (83.0 - 66.6) = 4.0 / 16.4 = **0.24**（正值，具备吸引力）

## 八、逻辑证伪与极限压力测试

### 8.1 黑天鹅场景——"一击毙命"事件

**场景定义**：欧盟委员会应OWFA请求，于2027年对来自中国的海上风电单桩基础加征**30%+的反倾销关税**（远高于塔筒的7.2-19.2%）。同时，主要欧洲客户（RWE、Ocean Winds）应政治压力终止与中国供应商的合作。

**冲击量化**：

| 冲击维度 | 量化影响 |
|:---|:---|
| 出口量 | 从56万吨骤降至0-10万吨（仅保留西班牙工厂本地化生产）|
| 出口ASP | 无关（关税后无竞争力）|
| 2027年净利润 | **从此前预期的~24亿骤降至~3-5亿**（仅剩国内+发电+航运）|
| PE（市场恐慌定价）| **降至15-20x** |
| **地狱股价** | **约10-15元/股**（对应市值64-96亿） |

### 8.2 极限支撑位审计——基于重资产清算价值

在"欧洲市场完全关闭"的极端情景下，使用清算价值法估算底部：

### 9.2 投资含义

- **82-83元**：合理价。盈亏比不具吸引力，建议观望或仅建立观察仓
