## 一、价值锚定：适配性估值方法选择

### 1.1 方法论筛选逻辑

公司的核心属性决定了估值方法的选择：

| 公司属性 | 对估值方法选择的影响 |
|:---|:---|
| 高成长（营收+60%，净利+33%预期） | PE、PEG适用 |
| 重资产制造（固定资产+在建>50亿） | PB-ROE适用、EV/EBITDA适用 |
| 盈利质量高（OCF>NP）、产能爬坡中 | DCF可作为校验工具 |
| 海外出口为主，汇率影响显著 | 已在盈利预测中体现 |
| 含发电/航运/造船等不同属性业务 | SOTP可作为交叉验证 |
| 业务转型期（国内→海外、FOB→DAP、制造→EPCI） | **历史PE分位参考价值有限** |
| **H股已发行（约1亿股，占总股本约13.6%）** | **稀释效应已发生，A股PE计算须纳入摊薄** |

### 1.2 估值锚点规则

当前为2026年5月（Q2），根据框架"当年Q1-Q3：锚定当年EPS"规则，估值锚定为**2026年全年预测EPS**。

## 二、PE估值法（主方法）

### 2.1 历史PE分位的参考与扬弃

从系统数据提取近5年（2021.05-2026.05）PE分位：

| 分位数 | PE值 | 对应历史阶段 |
|:---|:---:|:---|
| P5 | 10.4x | 2021年6月：高杠杆+国内为主+钢材涨价压利润 |
| P25 | 29.4x | 2022年Q1-2：海外转型初期 |
| P50（中位数）| 35.9x | — |
| P75 | 44.1x | 2023年初估值峰值 |
| P95 | 51.8x | 极值期 |
| **当前** | **35.9x（TTM）** | **接近P50分位（H股上市后首周数据）** |

**近期PE走势**（30个交易日，截至6月5日）：
- P5: 34.5x | P25: 36.5x | P50: 38.4x | P75: 41.5x | P95: 51.0x
- 当前35.9x处于近期**约P5位置**——估值已大幅压缩至近期极端低位

**历史分位的缺陷**：过去5年的PE样本中，前3年（2021-2023）的公司处于完全不同的经营状态。直接套用历史分位均值会低估公司在当前高质量增长期应享有的估值溢价。但H股摊薄后实际股本增加约13.6%，需在计算中纳入。

### 2.2 基准PE的确定（维持但调整摊薄影响）

采用"三维交叉法"确定大金重工的合理PE中枢：

| 定价维度 | 参考值 | 对大金的适用性 |
|:---|:---:|:---|
| A股风电设备行业中位数 | ~25x（含大量纯内需低毛利企业） | 偏低——大金的海外高毛利结构显著优于行业 |
| 大金自身历史中位数 | ~36x | 合理起锚点——反映市场对公司转型的认可 |
| 可比海外同业（Sif/EEW） | 30-35x Forward PE | 上限参考——欧洲利率环境不同 |
| **当前股价隐含（63.67元，2026E Base EPS~1.99元摊薄后）** | **~32.0x** | 较5月31日的31.2x（摊薄前）基本持平，但价格已大跌 |
| 东吴证券2026E PE | 26.6x（基于20.1亿预测） | 机构乐观预测下的估值 |

**基准PE设定：26-32x 2026E摊薄EPS**（较上次报告28-34x下移2x，以反映H股上市后摊薄效应+MA120破位后市场风险偏好下降）

- 下限26x：假设地缘折价加深+技术面破位引发的估值收缩
- 中枢29x：合理估值区间中值（较上次31x下移，反映摊薄+风险偏好变化）
- 上限32x：假设市场重新给予"全球化龙头+造船+A+H双平台"估值溢价

**H股上市后的实际情况（更新）**：
1. **H股首日表现不佳**：盘中最低59.05港元（-11.1%），收平66.40港元。A股同日暴跌5.79%至63.67元
2. **H股成为A股的"影子压力"**：H股66.40港元（约60.8元人民币）的定价低于A股——若H股持续在发行价下方运行，将不断通过A-H溢价心理向A股施压
3. **"A+H"双平台的积极效应需时间兑现**：GIC等13家基石投资者的国际背书价值是结构性的，但短期市场并未给予定价
4. **综合判断**：H股上市从中长期看仍是净利好，但短期已成为压制A股的技术面因素

## 三、PB-ROE模型（交叉验证）

### 3.1 大金重工ROE预测

| 指标 | 2024A | 2025A | 2026E Base | 2027E（外推）|
|:---|:---:|:---:|:---:|:---:|
| ROE（加权）| 6.68% | 14.18% | **~16.5%** | ~20% |
| 每股净资产（元）| 11.40 | 12.98 | **~14.5** | ~17.5 |
| 净利润（亿）| 4.74 | 11.03 | **14.7** | ~21 |
| 净资产（亿，年末）| 72.72 | 82.80 | **~92** | ~112 |

### 3.2 PB-ROE合理性检验

以16.5%的ROE和10%的权益成本（r），在零增长假设下：`合理PB = ROE/r = 1.65x`

在5%永续增长假设下：`合理PB = (16.5%-5%)/(10%-5%) = 2.30x`

**当前实际PB = 5.40x（系统数据）**，显著高于PB-ROE模型的静态合理值。H股上市后净资产增加约56亿港元（约52亿人民币募资净额），每股净资产提升至约18.3元（含H股增厚）。以2026E A股每股净资产约18元和PB 3.5-4.0x（因H股摊薄ROE）计算，对应股价63-72元。当前63.67元处于该区间下沿。

**较上次报告（5月31日）**：上次概率加权目标75.8元 → 现54.5元（-28.1%）。变化来源：(1) H股摊薄使A股EPS降低约14%；(2) 全变量审计矩阵净调节从+8%→0%；(3) Bear概率从25%上调至30%，Bull从25%下调至20%，反映地缘+汇率+技术面三重风险累积。

### 7.2 盈亏比计算（更新）

现值：63.67元 | 目标价（概率加权）：54.5元 | 保守价：39.5元

`盈亏比 = (54.5 - 63.67) / (63.67 - 39.5) = -9.17 / 24.17 = -0.38`

**盈亏比转负**——当前价位63.67元高于概率加权目标价54.5元。尽管价格已大幅下跌，但修正后的估值中枢下移幅度更大（-28.1% vs -12.8%）。投资的风险-收益比在当前价位**不具备吸引力**。

### 7.3 需要什么价格才有吸引力？

以盈亏比≥1.5为目标（具备充足安全边际的建仓价位）：

- 若概率加权目标维持54.5元、保守价维持39.5元：
  - `(54.5 - X) / (X - 39.5) ≥ 1.5`
  - `54.5 - X ≥ 1.5X - 59.25`
  - `113.75 ≥ 2.5X`
  - `X ≤ 45.5元`

**即A股价格需回落至45-46元区间（约PE 22-23x 2026E Base），才具备足够的安全边际吸引理性投资者建仓。** 较当前63.67元仍有约28%的下行空间——这与Bearer情景目标价39.5元配合，构成了完整的风险-收益框架。
