## 一、价值锚定：适配性估值方法选择

### 1.1 方法论筛选逻辑

公司的核心属性决定了估值方法的选择：

| 公司属性 | 对估值方法选择的影响 |
|:---|:---|
| 高成长（营收+60%，净利+33%预期） | PE、PEG适用 |
| 重资产制造（固定资产+在建>50亿） | PB-ROE适用、EV/EBITDA适用 |
| 盈利质量高（OCF>NP）、产能爬坡中 | DCF可作为校验工具 |
| 海外出口为主，汇率影响显著 | 已在盈利预测中体现 |
| 含发电/航运/造船等不同属性业务 | SOTP可作为交叉验证 |
| 业务转型期（国内→海外、FOB→DAP、制造→EPCI） | **历史PE分位参考价值有限** |
| 即将发行H股（≤15%总股本） | 稀释效应约2-3%，但融资增强资本实力 |

### 1.2 估值锚点规则

当前为2026年5月（Q2），根据框架"当年Q1-Q3：锚定当年EPS"规则，估值锚定为**2026年全年预测EPS**。

### 1.3 方法论组合

## 二、PE估值法（主方法）

### 2.1 历史PE分位的参考与扬弃

从系统数据提取近5年（2021.05-2026.05）PE分位：

| 分位数 | PE值 | 对应历史阶段 |
|:---|:---:|:---|
| P5 | 10.4x | 2021年6月：高杠杆+国内为主+钢材涨价压利润 |
| P25 | 29.4x | 2022年Q1-2：海外转型初期 |
| P50（中位数）| 35.9x | — |
| P75 | 44.1x | 2023年初估值峰值 |
| P95 | 51.8x | 极值期 |
| **当前** | **36.6x（TTM）** | **约P55分位** |

**近期PE走势**（130个交易日）：
- P5: 27.3x | P25: 31.9x | P50: 37.0x | P75: 41.5x | P95: 48.1x
- 当前36.6x处于近期**约P40位置**，已从5月初的40.5x显著回落

**历史分位的缺陷**：过去5年的PE样本中，前3年（2021-2023）的公司处于完全不同的经营状态。直接套用历史分位均值会低估公司在当前高质量增长期应享有的估值溢价。

### 2.2 基准PE的确定（更新）

采用"三维交叉法"确定大金重工的合理PE中枢：

| 定价维度 | 参考值 | 对大金的适用性 |
|:---|:---:|:---|
| A股风电设备行业中位数 | ~25x（含大量纯内需低毛利企业） | 偏低——大金的海外高毛利结构显著优于行业 |
| 大金自身历史中位数 | ~36x | 合理起锚点——反映市场对公司转型的认可 |
| 可比海外同业（Sif/EEW） | 30-35x Forward PE | 上限参考——欧洲利率环境不同 |
| 当前股价隐含（TTM 13.07亿） | 36.6x | 已从5月初40.5x回落 |
| 东吴证券2026E PE | 26.6x（基于20.1亿预测） | 机构乐观预测下的估值 |

**基准PE设定：28-34x 2026E EPS**（较原报告32-38x下调，因汇率风险兑现+地缘折价加深）

- 下限28x：假设市场对"中国企业在欧"的地缘折价加深
- 中枢31x：合理估值区间中值
- 上限34x：假设市场给予"全球化龙头+造船"的估值溢价

**下调理由**：
1. 汇率风险已从理论转为现实（Q1已确认汇兑损失），盈利确定性下降
2. OWFA直接点名攻击，地缘折价应扩大
3. 股价从83元快速下跌至75元的市场行为本身反映了估值中枢下移

### 2.3 基准市值区间

| 情景 | 2026E EPS | PE区间 | 目标市值（亿） | 目标股价（元） |
|:---|:---:|:---:|:---:|:---:|

**关键发现**：当前市值478亿恰好位于Base PE 32.4x位置——这是市场的中性定价。以中枢31x计算，合理市值为459亿（对应股价72.0元），较当前74.92元有**3.9%的下行空间**。以PE 30x计算，目标价69.3元。

## 三、PB-ROE模型（交叉验证）

### 3.1 大金重工ROE预测

| 指标 | 2024A | 2025A | 2026E Base | 2027E（外推）|
|:---|:---:|:---:|:---:|:---:|
| ROE（加权）| 6.68% | 14.18% | **~16.5%** | ~20% |
| 每股净资产（元）| 11.40 | 12.98 | **~14.5** | ~17.5 |
| 净利润（亿）| 4.74 | 11.03 | **14.7** | ~21 |
| 净资产（亿，年末）| 72.72 | 82.80 | **~92** | ~112 |

### 3.2 PB-ROE合理性检验

以16.5%的ROE和10%的权益成本（r），在零增长假设下：`合理PB = ROE/r = 1.65x`

在5%永续增长假设下：`合理PB = (16.5%-5%)/(10%-5%) = 2.30x`

**当前实际PB = 5.49x**，显著高于PB-ROE模型的静态合理值。这意味着市场给定了很高的增长隐含假设。

以2026E每股净资产14.5元、PB 5.0-5.5x计算，对应股价72.5-79.8元。当前74.92元处于该区间中偏下位置。

## 七、确定性与胜率矩阵

### 7.1 三态逻辑加权

| 情景 | 2026E EPS | 发生概率 | 目标市值（修正PE 30.3x）| 目标股价（元）|
|:---|:---:|:---:|:---:|:---:|
| 保守（Bear）| 1.57元 | **25%** | 303亿 | 47.5 |
| 中性（Base）| 2.31元 | **50%** | **446亿** | **69.9** |
| 乐观（Bull）| 3.06元 | **25%** | 591亿 | 92.7 |

**概率加权综合目标价** = 47.5×0.25 + 69.9×0.50 + 92.7×0.25 = **70.0元**

**较原报告（5月9日）**：原概率加权目标价81.9元 → 现70.0元（-14.5%）。变化主要源于EPS下修。

### 7.2 盈亏比计算

现值：74.92元 | 目标价（概率加权）：70.0元 | 保守价：47.5元

`盈亏比 = (70.0 - 74.92) / (74.92 - 47.5) = -4.9 / 27.4 = -0.18`

**盈亏比仍为负**——当前价格仍高于概率加权目标价，短期内缺乏上行空间，但下行保护在47.5元（约37%下行空间）。

### 7.3 乐观视角的盈亏比

若地缘风险缓解（OWFA游说受挫、欧盟维持单桩零关税），修正PE可回升至33x：
- 目标价(概率加权) = 76.2元
- 盈亏比 = (76.2 - 74.92) / (74.92 - 47.5) = 1.28 / 27.4 = **0.05**（微正）

**这意味着在当前价位，投资的核心变量是对地缘/政策风险的判断，而非对业绩的判断。**
