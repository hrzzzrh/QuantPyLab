## 二、PE估值法（主方法）

### 2.1 历史PE分位的参考与扬弃

从系统数据提取近5年（2021.05-2026.05）PE分位：

| 分位数 | PE值 | 对应历史阶段 |
|:---|:---:|:---|
| P5 | **20.5x** | 2021年6月：高杠杆+国内为主+钢材涨价压利润 |
| P10 | **25.6x** | — |
| P25 | **30.1x** | 2022年Q1-2：海外转型初期 |
| P50（中位数）| **36.3x** | — |
| P75 | **44.1x** | 2023年初估值峰值 |
| P95 | **51.8x** | 极值期 |
| **当前** | **32.3x（TTM，6月29日）** | **P10-P25分位——估值已压缩至历史偏低水平，接近P10（25.6x）** |

**近期PE走势**（30个交易日，截至6月29日）：
- P5: **32.1x** | P25: 33.4x | P50: 34.3x | P75: 36.3x | P95: 38.1x
- 当前32.3x处于近期**P5位置**——估值已被压缩至近期极端低位

**历史分位的缺陷**：过去5年的PE样本中，前3年（2021-2023）的公司处于完全不同的经营状态。直接套用历史分位均值会低估公司在当前高质量增长期应享有的估值溢价。但H股摊薄后实际股本增加约15.7%，需在计算中纳入。

### 2.2 基准PE的确定（维持但调整摊薄影响）

采用"三维交叉法"确定大金重工的合理PE中枢：

| 定价维度 | 参考值 | 对大金的适用性 |
|:---|:---:|:---|
| A股风电设备行业中位数 | ~25x（含大量纯内需低毛利企业） | 偏低——大金的海外高毛利结构显著优于行业 |
| 大金自身历史中位数 | ~36x | 合理起锚点——反映市场对公司转型的认可 |
| 可比海外同业（Sif/EEW） | 30-35x Forward PE | 上限参考——欧洲利率环境不同 |
| **当前股价隐含（57.17元，2026E Base EPS~1.81元摊薄后，A股归属净利润按86.4%折算）** | **~31.6x** | 较6月14日的29.9x（59.55元）有所提升——因盈利预测上调幅度（+4.8%）超过了股价跌幅（-4.0%）。估值压缩幅度反而有所缓和 |

- 下限25x：假设地缘折价加深+技术面深度破位引发的估值持续收缩
- 中枢28x：合理估值区间中值
- 上限31x：假设市场重新给予"全球化龙头+造船+A+H双平台"估值溢价（但短期内概率较低）

## 三、PB-ROE模型（交叉验证）

### 3.1 大金重工ROE预测

| 指标 | 2024A | 2025A | 2026E Base | 2027E（外推）|
|:---|:---:|:---:|:---:|:---:|
| ROE（加权）| 6.68% | 14.18% | **~16.5%** | ~20% |
| 每股净资产（元）| 11.40 | 12.98 | **~14.5** | ~17.5 |
| 净利润（亿）| 4.74 | 11.03 | **14.7** | ~21 |
| 净资产（亿，年末）| 72.72 | 82.80 | **~92** | ~112 |

### 3.2 PB-ROE合理性检验

以16.5%的ROE和10%的权益成本（r），在零增长假设下：`合理PB = ROE/r = 1.65x`

在5%永续增长假设下：`合理PB = (16.5%-5%)/(10%-5%) = 2.30x`

**当前实际PB = 4.84x（6月29日，系统数据）**，显著高于PB-ROE模型的静态合理值。H股上市后净资产增加约59亿人民币（募资净额64.65亿港元），每股净资产提升至约18.3元（含H股增厚）。以57.17元计算，PB约3.12x（A股口径净资产）。

### 6.3 修正后的目标市值

**基准市值** = PE 28x（基准中枢）× 13.3亿（A股归母净利）= **372亿**（A股口径）

**修正目标市值 = 372亿 × (1 + 0%) = 372亿**（A股口径）

对应A股目标股价 = 372亿 ÷ 6.377亿股 = **58.3元**（不复权市价）

**修正后的PE（A股）** = 372 ÷ 13.3 = **28.0x**

**较上次报告（6月14日）的变化**：上次修正目标57.9元 → 现58.3元（**+0.7%**，基本持平）。变化来源：(1) 盈利预测上调+4.8%（净利14.7→15.4亿）推高目标价至60.8元；(2) 基准PE中枢从29x下调至28x（因汇率/地缘风险溢价上升）→ 拉低至58.3元；(3) 全变量审计净调节维持0%。**盈利改善基本抵消PE中枢下移——目标价实际未发生显著变化。这与公司基本面改善的趋势一致。**

**核心逻辑演进**：A股从59.55→57.17（-4.0%）的下跌并非基本面恶化驱动。估值压缩的成果是PEG降至0.69，2026E A股PE降至27.5x。公司基本面的改善（Caofeidian爬坡完成、盈利上调）尚未得到市场定价——这正是均值回归的潜在驱动力。

### 6.4 估值与基本面的背离分析（新增）

本轮下跌的核心矛盾：**基本面持续改善（Q1+88%、曹妃甸爬坡完成、造船88亿），但股价从ATH 94.18元跌至57.17元（-39.3%）。** 这种背离的幅度和时间长度已超出正常"估值消化"的范畴。

三个可能的解释（按概率排序）：
1. **市场在系统性减配"出口主题"**（概率50%）：EUR/CNY持续贬值+中欧贸易摩擦担忧+全球风险偏好下行 → 资金流向防御性资产
2. **H股上市引发的流动性虹吸效应**（概率30%）：H股破发→A-H联动→趋势资金+量化基金系统性降仓
3. **基本面被错误定价**（概率20%）：市场过度关注短期汇率波动而忽略了中长期产能释放和第三曲线成长
