# 研报章节二：产业周期与终端穿透

**研究日期**：2026年06月06日

---

## 一、产业生命周期与库存节奏

### 1.1 光伏设备行业的周期定位

捷佳伟创所处的光伏电池设备行业，其需求本质上由下游电池片制造商的**产能扩张（Greenfield Capex）+ 技术迭代升级（Brownfield Capex）**共同驱动。该行业具有典型的"二阶导数"特征——设备需求取决于下游新增装机需求的变化率，而非绝对水平。我们可以通过以下周期框架定位当前行业坐标：

| 阶段 | 时间段 | 下游特征 | 设备商特征 |
| :--- | :--- | :--- | :--- |
| **上行期（扩产潮）** | 2022-2024H1 | TOPCon对PERC的全面替代，产能从80GW暴增至800GW+ | 订单暴增，合同负债攀升，以产定销 |
| **峰值拐点** | 2024H2 | 产能突破1000GW，供需失衡开始暴露 | 在手订单见顶，交付节奏维持高位 |
| **下行期（订单萎缩）** | 2025-2026H1 | 下游全线亏损，扩产停止，仅剩零星技改 | 新签订单断崖，存量订单消耗殆尽 |
| **筑底期（预期中）** | 2026H2-2027 | 落后产能出清，行业盈利修复 | 设备订单进入修复通道 |
| **复苏期（预期中）** | 2027-2028 | 新技术路线（HJT/钙钛矿）开启新一轮扩产 | 新一轮设备订单周期启动 |

**当前定位**：行业正处于**下行期末段向筑底期过渡**的关键窗口。核心判断依据如下：

（1）**产能扩张已实质性停止**。据中国光伏行业协会数据，2025年前10月国内产能投产/开工/规划项目数量同比下降超75%，电池片环节降幅最为显著。TrendForce数据显示，2025年底N型TOPCon电池产能约967GW，已接近阶段性天花板。2026年Q1全国新增光伏装机41.39GW，同比下滑约31.3%。

（2）**主产业链全面亏损，无力扩张**。2025年光伏主产业链产品价格跌破成本线，全行业集体陷入巨额亏损。晶科能源+天合光能两家合计亏损139亿元。2025年电池片均价0.289元/W、组件0.683元/W，均为历史最低水平。此背景下，下游客户已连续两年无新增设备订单。

（3）**产能过剩程度触目惊心**。截至2025年底，中国硅片产能超1500GW、电池产能超1400GW、组件产能超1100GW，而全球年均装机需求仅580-870GW。产能供需比超过2:1，超三分之一存量产能面临空置。2026年4月太阳能电池产量同比下降25.6%，1-4月累计同比下降15.3%。

### 1.2 库存动力学："牛鞭效应"的放大与回撤

光伏设备行业存在显著的"牛鞭效应"——终端需求的小幅波动，通过产业链层层放大，在设备端呈几何级数膨胀或收缩。

**上行放大器（2022-2024）**：

下游对TOPCon技术路线的"非抢不可"心态，叠加地方政府的招商引资竞争，推动电池片产能从一个合理水平（满足全球需求约需800GW）飙升至1400GW。这导致设备需求不仅覆盖了"为满足终端装机所需的合理增量"，还包含了大量"为抢占市场份额而超越实际需求的过度扩张"。捷佳伟创在此期间订单规模远超终端装机增速——2022年收入60亿，2023年87亿，2024年飙至189亿，三年增长215%。

**下行放大器（2025-2026）——正在剧烈释放**：

当下行周期来临，设备端承受的是**三重叠加**的负面冲击：
1. **新增扩产需求归零**：下游企业停止所有新建项目（如阿特斯终止14GW硅片扩产、亿晶光电12.5GW电池停产）
2. **存量订单延迟验收**：客户资金链紧张，刻意拖延设备验收节奏，导致设备商的发出商品无法转化为收入（捷佳伟创存货中发出商品占比79.57%）
3. **已交付设备遭遇退货**：极端案例中，琏升科技已将安装调试完毕的1GW HJT设备办理退货，设备供应商金石能源拆机运回

捷佳伟创的数据充分体现了这一牛鞭效应的残酷性：
- 合同负债从181亿（2023末）→ 131亿（2024末）→ 46亿（2025末）→ 46亿（2026Q1），降幅74.5%
- 2026年Q1收入仅14.95亿，同比骤降63.5%

**核心辨析**：当前设备订单的"真空期"主要反映了**牛鞭效应的回撤而非终端需求的毁灭性崩塌**。全球光伏装机2026年虽降至500-667GW，但并非崩盘——而是从2025年580GW的历史峰值回落至平台期。问题在于，过去三年的产能扩张已经把未来3-5年的设备需求提前透支殆尽。

## 二、终端需求穿透与跨期断层审计

### 2.1 最终产品画像

捷佳伟创的产品（光伏电池制造设备）的终端需求链路为：

```
光伏电池设备 → 电池片制造商 → 光伏组件 → 光伏电站 → 电力消费终端
```

设备需求的核心驱动来自下游新增装机需求所引致的电池片产能缺口。因此，对全球/中国光伏新增装机的预判是设备需求分析的核心基础。

### 2.2 终端景气度审计：2026-2028年全球光伏需求预测

| 预测来源 | 2025年（实际/预估值） | 2026年预测 | 2027年展望 | 2028年展望 |
| :--- | :---: | :---: | :---: | :---: |
| 中国光伏行业协会 | 315GW（中国）/ 580GW（全球） | 180-240GW / 500-667GW | 重回上升 | 270-320GW |
| SolarPower Europe | 655GW（全球） | 665GW（中等） | 755GW | 847GW |
| 彭博新能源财经 | 280GW+（中国） | 264GW（交流侧） | - | - |
| IEA | - | 负增长/增速放缓 | 回调至增长 | 持续增长 |
| 国金证券 | 285GW | 185-275GW | - | - |

**趋势共识**：
1. 2026年全球新增装机首次出现增速下降甚至负增长，中国降幅尤为显著（-24%至-43%）
2. 回调主因：分布式管理办法落地后观望情绪、上网电价市场化改革、前期抢装透支
3. 2027年起重回增长通道，但增速将较"十四五"明显放缓
4. "十五五"期间全球年均725-870GW，中国年均238-287GW
5. 新兴市场（印度、中东、非洲）将成为下一阶段增长核心引擎

### 2.3 动能溯源与透支审计（强制审计）

**审计问题1：增长动力是内生还是外生？**

本轮（2023-2025年）中国光伏装机的强劲增长，呈现"内生+外生"混合动力特征：
- **内生因素**（可持续）：光伏度电成本已低于火电，经济性驱动长期增长；N型技术迭代提升转化效率；全球碳中和目标形成的长期政策支撑
- **外生因素**（不可持续/存在透支）：2025年上半年的抢装潮（为赶在136号文政策节点前并网）、地方政府招商引资补贴驱动的产能过度扩张、"十四五"规划期末的节点性需求

**审计问题2：是否存在"跨期断层"风险？**

**存在显著的跨期需求透支风险**。证据如下：

1. **2025年装机透支2026年需求**：2025年中国新增装机315GW，是在分布式市场化改革前的抢装驱动下达成的。2026年政策真空期叠加前期透支，新增装机预计跌落至180-240GW区间。

2. **TOPCon设备需求的跨期透支更为严重**：2023-2024年，TOPCon产能从80GW暴增至800GW+，仅两年完成了"十年"的替代进程。这种超常规的加度替代，意味着未来3-5年的技改和新建需求已高度前置。

3. **行业明确认知到透支风险**：中国光伏行业协会在2026年2月的年度研讨会上首次官方预测装机量"先回落后回归"，工信部将2026年定义为光伏行业治理的"攻坚期"。

**跨期断层对设备商的量化影响**：
- 若中国新增装机从315GW降至200GW（中值），则对应电池片产能需求约260GW
- 而现有有效TOPCon产能约600GW，全行业产能利用率仅43%
- 这意味着：在下游产能利用率恢复到70%+之前，设备商几乎不会获得大规模新建订单

## 三、供需边际与压力传导

### 3.1 供需平衡审计

| 指标 | 2023年 | 2024年 | 2025年 | 2026年（预测） |
| :--- | :---: | :---: | :---: | :---: |
| 全球光伏新增装机（GW） | ~390 | ~530 | 580 | 500-667 |
| 对应电池片需求（GW） | ~507 | ~689 | 754 | 650-867 |
| 中国电池片有效产能（GW） | ~670 | ~1000 | ~1400 | ~1400 |
| 产能供需比 | 1.32 | 1.45 | 1.86 | 1.62-2.15 |
| 全行业产能利用率（估算） | ~75% | ~69% | ~54% | ~46-62% |
| 国内电池扩产规模（GW） | 545(产量) | 大幅放缓 | 几乎停滞 | 仅剩技改 |
| 捷佳伟创新签订单趋势 | 高增 | 峰值 | 断崖 | 低位徘徊 |

**核心判断**：供需边际正在从"紧平衡"向"严重过剩"滑落，且过剩状态将至少持续至2027年。电池片环节的供需比已达1.86-2.15，远超合理水平（1.1-1.3）。

### 3.2 库存周期的三个维度

光伏设备行业的库存问题需要从三个维度理解：

**维度1：终端产品库存（组件/电池片成品）**
2026年一季度，行业整体库存接近50万吨（硅料口径），隐性库存高企。组件价格从2024年的0.84元/W进一步降至2025年的0.68元/W。2026年虽有短暂反弹（受产能出清和上下游博弈影响），但整体仍在成本线附近，说明终端库存消化仍在进行中。

**维度2：设备商发出商品库存（捷佳伟创的未验收设备）**
这是设备商特有的库存形式——设备已交付客户现场但尚未完成验收（从而无法确认收入）。捷佳伟创报告期发出商品占存货79.57%，约44.2亿元。在行业下行期，客户倾向于延迟验收以缓解资金压力，这部分"库存"转化为收入的周期被显著拉长。

**维度3：产能库存（闲置产能）**
这是最具破坏性的库存。1400GW电池产能对应754GW需求，意味着约646GW的产能处于闲置或低利用率状态。在供给严重过剩的环境下，价格竞争恶性循环——低价→亏损→不愿或无力采购新设备→设备订单持续萎缩。

### 3.3 压力传导特征分析

**波动放大效应（已充分显现）**：

终端装机从315GW降至200GW（降幅36%），对应电池片产量从约850GW降至500GW（降幅41%），而设备订单的降幅则高达70-90%（新建产能需求几乎归零，仅剩零星技改订单）。这种逐级放大的效应正是光伏设备行业周期属性的核心特征。

**对捷佳伟创的具体传导路径**：

```
终端装机下降36% 
→ 电池片厂商开工率降至50%以下 
→ 停止一切新建产能投资 
→ 存量订单验收被无限拖延 
→ 设备商新签订单断崖式下跌（2026Q1收入-63.5%）
→ 经营现金流急剧恶化（2025年经营现金流-12.04亿）
→ 应收账款坏账风险大幅上升（2025年信用减值损失8.04亿）
```

**传导时滞与博弈筹码**：

光伏设备的传导存在6-12个月的滞后效应，这意味着：
1. 2024年下半年的需求放缓信号，在2025年下半年才完全体现在设备商的报表中
2. 2025-2026年的订单真空，将在2026-2027年持续拖累设备商业绩
3. 即使2026年下半年下游出现盈利修复信号，设备商的新签订单也要至2027年才可能实质回暖

**捷佳伟创是否有博弈筹码？**

公司在产业链中的博弈筹码相对有限：
- **正面**：整线交付能力构成一定客户粘性，存量客户的技术升级仍需依赖公司设备
- **负面**：设备属于资本品而非耗材，客户在亏损状态下完全可以选择"不买"，而设备商毫无反制能力
- **中性**：海外市场正在形成新的需求支撑，但规模尚不足以完全对冲国内下滑

### 3.4 需求结构的空间位移：海外市场正在补位

捷佳伟创2025年海外收入31.16亿元，同比增长79.13%，占比从9.21%跃升至20.14%。海外毛利率36.55%显著高于境内的26.16%，是盈利能力的重要支撑。

海外需求的核心驱动力来自：
1. **印度**：本土光伏制造产能快速扩张（组件产能约210GW，电池产能约27GW），在ALMM、PLI、进口关税等政策推动下，电池片设备需求将持续释放
2. **美国**：IRA法案刺激本土制造，40GW+本土产能规划，捷佳伟创美国TOPCon设备市占率约60%
3. **中东**：沙特、阿联酋依托光照资源+能源转型政策，光伏终端需求将成为中国设备海外布局的核心阵地
4. **欧洲**：电网改造+消纳改善驱动装机增速触底回升，对高效设备存在持续性需求

2026年，捷佳伟创全球在手订单≥280亿元，其中海外≥120亿元。但需注意：海外订单同样面临政策不确定性（美国对华设备出口限制的潜在风险）和竞争加剧的挑战。

## 【强制复核项：结果确认】

1. **终端产品景气度预测已引用**：已明确引用CPIA、SolarPower Europe、IEA、彭博新能源财经等权威机构对2026-2028年全球/中国光伏新增装机的预测数据，核心判断为"2026年回调、2027年起重回增长"

2. **跨期需求移位已审计**：已识别并分析了2025年抢装潮对2026年装机需求的透支效应，以及2023-2024年TOPCon产能暴增对未来3-5年设备需求的超前透支

3. **供需边际与压力传导已分析**：已基于有效产能与终端需求的对比，判断供需边际正从紧平衡滑向严重过剩；已详细刻画了压力从终端装机→电池片开工→设备订单→经营现金流的逐级放大传导路径，并评估了捷佳伟创在产业链中的博弈筹码
