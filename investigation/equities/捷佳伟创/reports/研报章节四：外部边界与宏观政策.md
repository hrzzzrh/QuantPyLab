# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月06日

---

## 一、宏观环境变量

### 1.1 利率与折现率：估值环境偏友好，但对实体传导有限

当前全球处于高利率逐步下行的过渡期，国内货币政策维持适度宽松。对捷佳伟创而言，利率环境的影响通过两条路径传导：

**路径一：DCF估值分母端**——偏正面。无风险利率中枢下行有利于成长性设备股的估值中枢抬升。但考虑到公司当前仅约10倍PE，估值折价主因在于盈利端的确定性崩塌而非利率环境。

**路径二：下游客户融资成本**——偏负面。光伏下游电站/制造业属于资本密集型行业。虽然国内利率下行，但海外（尤其是新兴市场）融资成本仍高企，这对海外光伏产能扩张的节奏形成抑制。此外，国内对光伏制造业的融资趋严——银行对亏损电池厂的信贷明显收紧，间接传导至设备采购需求。

### 1.2 汇率损益：双刃剑效应

2025年，捷佳伟创财务费用从-1.38亿（净收益）恶化至-0.36亿（净收益收窄），主因利息收入减少和外币货币性项目汇率波动形成汇兑损失。公司以外币计价的资产负债余额较大。

| 汇率变动方向 | 对捷佳伟创的影响 |
| :--- | :--- |
| 人民币升值 | 负面影响：境外收入折算后下降，汇兑损失增加 |
| 人民币贬值 | 正面影响：境外收入折算后上升，汇兑收益增加 |
| 央行汇率市场化推进 | 增加波动性，需要持续的外汇对冲管理 |

2025年，公司积极运用外汇掉期、远期锁汇等衍生品管理汇率风险。海外收入占比从9.21%升至20.14%且仍在提升，汇率影响权重持续增大。

### 1.3 通胀传导与原材料成本

捷佳伟创的直接材料在营业成本中占81.26%（2025年），原材料以电子元器件、炉机柜、结构件等为主，对大宗商品价格的直接敏感度低于下游电池制造商。但近期银价高位震荡（影响TOPCon银浆成本→压缩客户利润空间）、铜铝等大宗商品价格上涨仍会通过以下路径间接传导：

```
大宗商品涨价 → 客户制造成本上升 → 利润空间收窄 → 
设备采购意愿下降 → 捷佳伟创订单增长受限
```

## 二、地缘政治风险

### 2.1 美国贸易壁垒：从"堵产品"升级为"堵体系"

美国对华光伏贸易政策已经从针对终端产品（电池、组件）的双反关税，全面升级为对全产业链的系统性围堵。以下是核心政策的演变轨迹：

| 时间 | 事件 | 实质影响 |
| :--- | :--- | :--- |
| 2012年起 | 对中国光伏电池/组件征收双反关税 | 中国组件出口美国受阻，产能向东南亚转移 |
| 2022年 | 启动对东南亚四国（越/泰/马/柬）反规避调查 | 东南亚代工路径被封堵 |
| 2024-2025年 | 对东南亚四国征收惩罚性关税（柬埔寨3404%等） | 东南亚产能全线沦陷 |
| 2025年7月 | 针对印度、印尼、老挝发起反规避调查 | 第二波产能转移再次被堵 |
| 2026年3月 | 启动对TOPCon太阳能电池/组件的337调查 | 首次针对特定技术路线进行调查 |
| 2026年5月 | ITC对华光伏双反日落复审终裁，措施再延5年 | 对华光伏产品征税长期化 |
| 2026年5月 | 对埃塞俄比亚发起反规避调查 | 首次针对非洲大陆，追堵"中国供应链" |

**对捷佳伟创的具体影响**：

1. **正面：设备出口美国目前享有关税豁免**。美国对中国光伏设备的301条款关税豁免已延长至2026年11月。这为捷佳伟创向美国出口设备创造了时间窗口。

2. **负面：中国可能限制先进光伏设备对美出口**。2026年4月路透社报道，中国已与设备商初步磋商，考虑限制最先进光伏设备（包括HJT）向美国出口。虽然尚未形成正式规则，但这一政策不确定性对捷佳伟创的美国业务构成重大风险——特别是特斯拉订单若涉及HJT技术，可能面临出口管制。

3. **负面：海外产能扩张路径被封堵**。美国正在将贸易围堵从"中国制造"延伸到"中国供应链控制"——即使企业在非洲、拉美建厂，只要核心价值链仍由中国控制，就可能被征税。这对捷佳伟创的海外设备销售构成中长期隐忧。

4. **商务部反制**：中国商务部已于2026年3月27日对美国阻碍绿色产品贸易的做法启动贸易壁垒调查，贸易摩擦仍在升级中。

### 2.2 印度ALMM：本土化的加速与机遇

印度光伏市场的政策取向对捷佳伟创喜忧参半：

- **ALMM List-I（组件清单）已生效**：政府项目只能用列表内的本土组件，已大幅限制中国组件出口印度
- **ALMM List-II（电池清单）2026年6月1日生效**：政府项目必须使用本土电池，电池片出口空间进一步收窄
- **ALMM List-III（硅锭/硅片清单）计划2028年6月执行**：倒逼硅片本土化

**但**，印度严重缺乏电池片制造设备能力。目前印度电池产能约27GW，而组件产能已达210GW，未来需要大规模补齐电池片制造能力。正如中国电池设备在2022-2024年受益于TOPCon扩产，印度未来3-5年的电池产能建设将为捷佳伟创创造大量设备出口机会。这与美国对中国电池/组件的贸易壁垒形成反差——设备出口印度目前未受明显限制。

### 2.3 欧盟NZIA：本土制造崛起，设备出海窗口收窄

欧盟《净零工业法案》（NZIA）要求2030年本土制造需满足40%的年度部署需求。欧盟基金补贴86亿欧元，成员国国家援助236亿欧元。在补贴驱动下，欧洲本土组件产能从5GW（2022年）跃升至15GW（2024年）。但电池片制造能力仍严重不足——这为捷佳伟创创造了设备出口欧洲的机会。

### 2.4 出口退税取消：光伏产品端的冲击与设备端的相对免疫

2026年4月1日起，光伏产品（硅料、硅片、电池、组件、逆变器等249项）的增值税出口退税从9%降至0%。这一政策主要影响的是光伏产品（终端输出）出口，对光伏设备（捷佳伟创的主营产品）影响相对有限。但需关注两个间接效应：

1. **客户成本上升**→利润空间被压缩→设备采购能力下降
2. **行业洗牌加速**→中小落后产能出清→短期加剧订单萎缩，长期利好龙头设备商

## 三、监管与产业政策

### 3.1 136号文：新能源电价市场化改革的重磅冲击

2025年2月，国家发改委发布136号文，推动新能源上网电价全面市场化。核心变化：

| 维度 | 改革前 | 改革后 |
| :--- | :--- | :--- |
| 定价机制 | 固定电价（燃煤基准价保障性收购） | 全电量市场交易形成电价 |
| 收益结构 | 保障性收入为主 | 市场交易收入 + 价差补偿 - 辅助服务分摊 |
| 投资确定性 | 高（固定电价保护） | 低（市场化波动） |
| 增量项目 | 享受同等保障 | 机制电价通过市场化竞价确定 |
| 政策节点 | 无 | 2025年6月1日为存量/增量分界线 |

**136号文对光伏设备需求链的传导**：

```
新能源电价市场化 
→ 投资收益率不确定性大幅上升 
→ 增量光伏项目开发趋于谨慎 
→ 2026年国内新增装机从315GW降至180-240GW 
→ 电池片制造商的扩产意愿进一步下降 
→ 光伏设备需求持续低迷
```

各地竞价结果已显示出极端分化：上海机制电价0.4155元/kWh（接近上限）、山东光伏仅0.225元/kWh（接近下限）。西部"源多荷少"地区的低电价将显著抑制当地光伏投资，进而压缩设备采购空间。

**对捷佳伟创的直接影响**：国内下游客户的设备采购意愿在136号文后进一步恶化。即便行业盈利有所恢复，客户在面对不确定的电价前景时，仍将优先保护现金流而非扩大资本开支。

### 3.2 "反内卷"治理：从无序竞争到有序出清

2025年以来，国家层面持续推进光伏行业"反内卷"治理：

- **2024年底**：中国光伏协会设定组件中标价不低于0.68元/W的"地板价"
- **2025年7月**：中央财经委会议强调"依法依规治理企业低价无序竞争，推动落后产能有序退出"
- **2026年**：工信部将2026年定义为光伏行业治理的"攻坚期"，综合运用产能调控、标准引领、质量监督、价格执法等手段

**政策效果评估**：
- 硅料环节"反内卷"成效较明显，产能整合收购开始落地
- 电池/组件环节仍处于血拼阶段，2025年电池片均价0.289元/W、组件0.683元/W
- 2026年预计从"全行业亏损"向"头部盈利、尾部出清"转变

对设备商的含义：**短期利空（出清过程中需求更弱），中长期利好（出清完成后龙头企业恢复采购能力）**。

### 3.3 光伏设备出口管制风险：尚未落地但需高度警惕

2026年2月，中国相关部委已就先进光伏设备出口限制征求行业意见。核心关切在于：
- 中国光伏设备占全球超80%市场份额
- 限制先进设备出口可间接维护中国光伏产品的全球市场地位
- 保持光伏制造技术的"代际差"

潜在的管控方向可能是"建立设备分级目录"：≤26%效率的成熟技术设备鼓励出口，>26%的高端设备实施管控。若实施，对捷佳伟创的影响取决于其出口设备的技术代际。TOPCon设备（效率~25-26%）可能在豁免范围内，但HJT/钙钛矿等前沿设备可能面临限制。

## 四、外部变量的综合评估矩阵

| 变量类别 | 具体因子 | 影响方向 | 烈度评级 | 核心逻辑 |
| :--- | :--- | :---: | :---: | :--- |
| 宏观利率 | 国内适度宽松 | 轻微正面 | ★★ | 降低融资成本，但下游扩产决策更依赖行业景气度 |
| 汇率波动 | 人民币汇率双向波动加大 | 负面 | ★★☆ | 海外收入占比增至20%，汇兑风险上升 |
| 美国关税 | 光伏双反、337调查 | 正面+负面 | ★★★ | 设备进口豁免（利好），但出口管制风险上升（利空） |
| 出口退税取消 | 光伏产品退税归零 | 轻微负面 | ★★ | 直接影响终端产品，间接传导至设备需求 |
| 中国设备出口管制 | 先进光伏设备可能受限 | 严重负面 | ★★★★☆ | 若落地将直接冲击海外订单（尤其是美国） |
| 136号文 | 电价全面市场化 | 严重负面 | ★★★★ | 系统性削弱下游投资意愿，短期冲击严重 |
| 反内卷治理 | 产能出清加速 | 短期负面/中长期正面 | ★★★ | 加速出清→短期压制需求，长期优化行业格局 |
| 印度ALMM | 本土化政策推进 | 短期负面/中长期正面 | ★★★ | 堵住组件/电池出口，但创造设备需求 |
| 美国IRA | 设备进口豁免至2026年11月 | 正面 | ★★★☆ | 创造设备出口窗口期，但窗口正在收窄 |

**综合判断**：外部变量此前对于捷佳伟创以**利好为主**（TOPCon扩产受益于政策驱动、美国对设备进口的关税豁免），但当前已系统性转向**利空为主**——136号文压制国内需求、出口管制压制海外需求、反内卷压制短期订单。公司面临的政策环境正处于"从顺风到逆风"的关键转换期。

## 【逻辑继承：前序模块结论与模块四的交叉验证】

1. **M1发现的现金流恶化（经营现金流-12.04亿）在M4中得到政策层面的解释**：136号文+反内卷导致下游客户资金链紧张，回款出现系统性延迟

2. **M2判断的"2026年装机回调"与M4的136号文分析高度吻合**：新能源电价市场化改革是2026年装机下降的核心政策驱动因素

3. **M3发现的"海外收入高增但占比仍低"在M4中面临政策不确定性**：中国设备出口管制若落地，将削弱海外增长的最大驱动力

4. **美国进口设备关税豁免延长至2026年11月**：为捷佳伟创创造了关键的时间窗口，特斯拉订单（若存在）必须在2026年秋季前完成交付
