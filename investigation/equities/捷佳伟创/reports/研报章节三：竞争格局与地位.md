# 研报章节三：竞争格局与地位

**研究日期**：2026年06月06日

---

## 一、竞争格局全景扫描

### 1.1 光伏电池设备行业竞争地图

光伏电池设备行业围绕不同技术路线和工艺环节形成了高度分化的竞争格局。下表梳理了核心赛道与竞争参与方：

| 赛道 | 核心工艺环节 | 全球龙头 | 国内主要竞争者 | 进入壁垒 |
| :--- | :--- | :--- | :--- | :--- |
| TOPCon整线 | 全工序成套 | **捷佳伟创** | 拉普拉斯、北方华创、先导智能 | 极高（整线集成+工艺参数包） |
| TOPCon镀膜 | PECVD/LPCVD | **捷佳伟创/拉普拉斯** | 北方华创、微导纳米 | 高（工艺Know-how+客户验证周期） |
| TOPCon扩散 | 硼扩散 | **拉普拉斯** | 捷佳伟创、北方华创 | 高（热场设计+大产能） |
| HJT整线 | 全工序成套 | **迈为股份** | 捷佳伟创、金石能源 | 极高（70%+份额+学习曲线） |
| 钙钛矿设备 | RPD/PVD/涂布 | **捷佳伟创/迈为股份** | 德沪涂膜、众能光电 | 中高（多数处于中试/实验阶段） |
| 激光设备 | SE/消融/转印 | **帝尔激光** | 大族激光、海目星 | 高（激光+工艺专利壁垒） |
| 组件设备 | 串焊/层压 | **奥特维** | 先导智能、金辰股份 | 中（标准化程度较高） |

### 1.2 TOPCon设备：捷佳伟创的绝对根据地

捷佳伟创在TOPCon设备领域的市场地位难以撼动：

- **整线市占率**：58%-70%（根据行业调研），在TOPCon产线价值量占比70%-80%的核心品类中，除LPCVD外，公司在其余几大品类的市占率均超过50%
- **核心设备PE-Poly**：市占率超50%，其"三合一"工艺（隧穿层+Poly层+原位掺杂层）已成为TOPCon行业主流技术路线
- **客户覆盖**：直接供应隆基、通威、晶澳、晶科、天合等所有头部电池/组件厂商，参与全球超4000条电池产线建设
- **市场地位演变**：市占率从2023年的23%飙升至2024年的71.1%，核心驱动力是PE-Poly技术路线的行业主导地位确立

**但是**：TOPCon的"甜蜜陷阱"已经显现。当TOPCon产能从80GW暴增至800GW+后，扩张空间已近封顶。公司对TOPCon的高度依赖（工艺设备收入占比81.85%）正从"护城河"变成"风险敞口"——当整个赛道面临下行压力时，龙头地位无法提供下行保护，反而放大了业绩冲击。

### 1.3 与主要竞争对手的对比分析

#### **迈为股份（300751）——HJT领域的捷佳伟创镜像**

| 对比维度 | 捷佳伟创 | 迈为股份 |
| :--- | :--- | :--- |
| 核心技术路线 | TOPCon（市占58-70%） | HJT（市占70%+） |
| 2025年营收 | 154.72亿（-18.1%） | 81.52亿（-17.1%） |
| 2025年归母净利润 | 26.17亿（-5.3%） | 7.22亿（-22.1%） |
| 2026Q1营收 | 14.95亿（-63.5%） | 13.37亿（-40.0%） |
| 2026Q1毛利率 | 34.59% | 51.39% |
| 合同负债趋势 | 46亿（下降中） | 59.4亿（Q1逆势回升+16.3亿） |
| 2025年经营现金流 | -12.04亿 | -10.42亿 |
| 战略定位 | TOPCon平台型整线 | HJT技术专家型整线 |
| 第二曲线进展 | 半导体湿法+锂电+PCB | 半导体/显示设备（6.62亿，+887%） |
| 海外收入占比 | 20.14% | 35.32% |
| 2025年研发费用率 | 3.93% | 14.20% |

**核心分歧点**：迈为股份在HJT领域的技术壁垒更高（70%+细分市占率且短期内无强力挑战者），且合同负债在2026Q1逆势增长+16.3亿，显示出新签订单正在恢复。相比之下，捷佳伟创作为"平台型"企业的优势在行业上行期能够发挥最大效应（规模+品类），但在下行期反而面临"量价齐跌"的困境。

**迈为的挑战**：研发费用率高达14.2%（2026Q1更升至18.0%），在收入下降期构成利润拖累；有息负债率27.7%，资金压力大于捷佳伟创；HJT的渗透率目前仅约2.6%，商业化拐点何时到来存在不确定性。

#### **拉普拉斯（688726）——隐形冠军的工艺卡位**

拉普拉斯通过LPCVD+硼扩散设备，精准卡位了TOPCon/XBC电池的两道核心工艺瓶颈：
- LPCVD产能（550MW/年）显著高于捷佳伟创的同类产品（370MW/年）
- 深度绑定隆基绿能、晶科能源等行业龙头客户
- 2020-2024年营收CAGR达245%，2025年预计营收54.59亿
- 半导体SiC设备已导入比亚迪，毛利率高于光伏设备

拉普拉斯是捷佳伟创在TOPCon镀膜/扩散环节最直接的竞争对手，但其体量远小于捷佳伟创（约为1/3），且缺乏整线能力。

#### **北方华创（002371）——光伏设备营收规模第一，但非核心竞争者**

北方华创2024年光伏设备营收277.1亿（市占21%+），涵盖硅片炉、PECVD、扩散炉等产品。但其光伏设备毛利率持续承压，2025年出现净利润近十年来首次下滑。北方华创是硅片设备领域的王者，但在电池片设备环节，其PECVD产品面临捷佳伟创的强力竞争，市占率远不及后者。

## 二、护城河识别

### 2.1 捷佳伟创的护城河结构分析

| 护城河类型 | 强度评级 | 关键证据 | 可持续性评估 |
| :--- | :---: | :--- | :--- |
| **规模/成本壁垒** | ★★★★ | TOPCon整线交付能力，年产19,411管/台设备，中国电子专用设备行业前十名（连续8年）。海外马来西亚工厂投产。研发投入累计>17亿。 | 短期稳固，但产能扩张停止后规模优势消退 |
| **技术/专利壁垒** | ★★★★ | PE-Poly三合一工艺，1,038项专利（135项发明专利），唯一具备多技术路线整线解决方案。钙钛矿GW级设备出货。 | 在TOPCon路线具有持久性，但技术路线切换（如向HJT/BC）可能削弱 |
| **客户转换成本** | ★★★ | 整线交付+工艺参数包绑定，客户若更换设备需重新调试整条产线。覆盖所有头部电池厂。 | 中等——设备属于资本品，下行期客户可完全停止采购 |
| **品牌/声誉壁垒** | ★★★ | 连续8年行业前十，全球参与超4000条产线建设。 | 中等——品牌在行业下行期不足以驱动订单 |
| **网络效应** | ★★ | 通过多品类设备形成交叉销售能力。 | 弱——光伏设备行业不具备传统网络效应 |

**综合评价**：捷佳伟创的护城河主要建立在"技术+规模"层面，在TOPCon赛道具备深厚的竞争优势。但**光伏设备行业的护城河本质上是周期性的**——上行期的壁垒在下行期会迅速弱化。当客户无力扩产时，再强的技术也卖不出去。

### 2.2 护城河的相对减损

与迈为股份相比，捷佳伟创在技术壁垒的"锐度"上有所不足：
- **HJT领域**：迈为70%+市占率，捷佳伟创有布局但未形成主导地位
- **研发强度**：迈为14.2% vs 捷佳3.93%，迈为在下一代技术的投入更为激进
- **合同负债趋势**：迈为Q1回升+16.3亿，捷佳几乎持平，暗示新签订单动能分化

**核心判断**：捷佳伟创在TOPCon赛道的护城河正在"老化"——技术壁垒仍在，但赛道本身正在萎缩。公司能否在新一代技术路线（HJT、钙钛矿）中复制TOPCon时代的成功，存在高度不确定性。

## 三、产业链定价权评估

### 3.1 对上游的议价能力：较强

- 前五名供应商采购占比仅13.28%（2025年），分散度高
- 最大单一供应商采购占比3.37%
- 原材料以标准件（电子元器件、炉机柜等）+外协加工为主，供应链不依赖单一来源
- 应付账款周转天数184.85天（2025H1数据），说明对供应商存在明显占款

### 3.2 对下游的议价能力：弱化中

- **正面信号**：预收款模式（"3331"——签约30%+发货30%+验收30%+质保10%），合同负债仍近46亿
- **负面信号**：
  1. 应收账款+合同资产70亿（占总资产28%），且2025年信用减值损失8.04亿
  2. 应收账款周转天数从62天（2024）延长至90天（2025），回款持续恶化
  3. 客户集中度高——前五大客户占销售42.52%，单一客户可达12.13%
  4. 下游客户全线亏损，设备采购属"可选消费"性质，设备商议价空间急剧收窄

### 3.3 定价权的动态演变

在下行周期中，捷佳伟创的定价权呈明显的"剪刀差"效应：

```
上游（供应商）：议价权保持 → 采购成本可控
下游（客户）：议价权急剧增强 → 压价、延迟验收、延长付款周期
自身：夹在中间 → 毛利率承压、现金流恶化、信用减值大幅上升
```

**量化验证**：2025年经营现金流-12.04亿、应收账款+41.96亿、信用减值损失8.04亿，三大指标同时恶化，充分说明公司在产业链中的定价权正在被系统性侵蚀。

## 四、非财务竞争信号的交叉验证

### 4.1 订单层面：海外正在成为新引擎

截至2026年2月，捷佳伟创全球在手订单≥280亿元，其中海外≥120亿元（占比≥43%）。公司2025年境外收入31.16亿，同比增长79.13%，海外毛利率36.55%显著高于境内的26.16%。

**特斯拉订单传闻**：2026年3月，路透社报道特斯拉计划采购约29亿美元中国光伏设备，捷佳伟创被视为TOPCon设备的核心供应商。若该传闻落地，将为公司带来额外的大规模订单。但需要理性看待：首批预计仅2-3GW，完全兑现需较长时间。

### 4.2 招投标与产能信号

- 2025年光伏行业组件招标规模198GW，同比大幅收缩。多家央国企（中广核、国家电投等）招标规模缩减10GW以上。
- 电池片环节：2025年前10月产能投产/开工/规划项目数量同比下降超75%
- 设备商产能利用：捷佳伟创2025年生产量同比-72.41%，出货量同比-75.89%，反衬新订单的稀缺程度

### 4.3 技术路线竞争的风险敞口

TOPCon仍是当前主流（87.6%市占率），但XBC增长至6.7%、钙钛矿技术加速推进。如果未来2-3年XBC或钙钛矿路线取得重大突破，捷佳伟创在TOPCon领域的产能和技术优势可能面临"价值重置"。公司虽对HJT和钙钛矿有所布局，但在HJT领域远不如迈为股份、在XBC领域不如拉普拉斯深耕。

### 4.4 竞争对手的跨越式演进

迈为股份在行业下行期实现了三个跨越：
1. **毛利率跳升**：从2025Q1的29.09%升至2026Q1的51.39%（+22.3pct）
2. **合同负债逆增**：Q1末59.4亿，较上年末增加16.3亿
3. **半导体/显示业务爆发**：收入6.62亿，同比增长887%

这表明在光伏设备行业的结构性分化中，HJT+半导体双轮驱动的迈为正在实现"弯道超越"，而捷佳伟创现阶段仍高度依赖TOPCon的存量消化。

## 【逻辑继承：模块一/二的结论与模块三的交叉验证】

1. **M1发现的财务隐忧（现金流恶化、信用减值、合同负债坍塌）在M3的定价权分析中得到明确解释**：核心原因是下游客户全线亏损→设备商议价权丧失→回款周期拉长+坏账风险上升

2. **M2判断的"行业下行期末段"在M3的竞争格局分析中得到印证**：迈为股份合同负债逆势回升表明分化已经开始，HJT/海外赛道正在率先复苏，但捷佳伟创主场（TOPCon）尚未出现企稳信号

3. **M1发现的风险（存货规模大、应收质量差）需要与M3的竞争地位结合看待**：公司目前仍有46亿合同负债和约280亿在手订单作为缓冲，非单纯的"裸奔下行"，但缓冲垫正在快速消耗
