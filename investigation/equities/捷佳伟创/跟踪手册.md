## 五、跟踪操作指南

### 季度财报后的标准核查清单

每次财报发布后，完成以下核查：

- [ ] 单季收入是否达到或超过中性预期（Q2≥25亿、Q3≥35亿、Q4≥45亿）？
- [ ] 合同负债环比是否企稳或回升（底线30亿，目标企稳在40亿+）？
- [ ] 毛利率是否在26%-30%合理区间（<25%视为恶化，>32%视为异常）？
- [ ] 经营现金流是否出现季度转正迹象？
- [ ] 信用减值损失/利润总额比率是否收窄（<20%为健康）？
- [ ] 应收账款占总资产比例是否下降（<25%为健康）？
- [ ] 海外收入占比是否继续提升（>25%为达标）？
- [ ] 公司管理层对未来展望的措辞是趋于乐观还是悲观？
- [ ] 是否有新签大额订单（>10亿）需要单独评估？
- [ ] 是否有竞争对手/客户/供应商的重大公告需要关联分析？

### 评级调整规则

| 触发条件 | 调整动作 |
| :--- | :--- |
| 连续两个季度业绩超中性预期20%+，且合同负债环比回升 | 上调至"观望偏多"（第三象限→第二象限过渡） |
| 特斯拉正式订单公告金额>100亿，且出口管制风险解除 | 上调至"增持"（第二象限） |
| 单季收入连续低于保守预期（<20亿），且合同负债<30亿 | 下调至"强烈回避"（维持第四象限，但更悲观） |
| 任何一级警报触发 | 重新召开投资决策会，重新定价 |
