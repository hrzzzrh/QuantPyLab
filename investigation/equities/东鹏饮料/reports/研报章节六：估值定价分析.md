## 二、价值锚定：适配性估值方法选择及理由

### 2.1 方法选择与理由

东鹏饮料具备以下特点：稳定盈利（净利率20%+）、高速增长（3年CAGR 22%）、轻资产（非资本密集型）。基于此，选择以下三种方法进行交叉验证：

| 方法 | 适用性 | 权重 | 理由 |
|:---|:---|:---|:---|
| **PE估值** | 核心方法 | 50% | 盈利稳定可持续，成长性可量化，消费品行业首选 |
| **PEG估值** | 辅助验证 | 25% | 高增长公司需用PEG校准PE，防止高PE陷阱 |
| **DCF估值** | 参考锚点 | 25% | 现金流可预测，轻资产模式下FCF质量高 |

**不适用方法说明**：
- **PB-ROE**：H股IPO导致净资产翻倍（76亿→205亿），PB被动大幅稀释，历史分位已严重失真，不适用
- **SOTP**：海外业务几乎为零，各板块虽然属性不同但共用渠道/供应链，难以精确拆分估值
- **PS**：毛利率差异大（能量饮料51% vs 其他饮料16%），PS会严重高估低毛利业务价值

### 2.2 历史分位审计与谨慎评估

| PE分位 | 数值 | 位置解读 |
|:---|:---|:---|
| 5年最低 | 17.35 | 2026年6月26日触及 |
| 5%分位 | 26.78 | 当前20.11远低于此 |
| 25%分位 | ~31 | |
| 50%分位 | ~38 | |
| 75%分位 | ~45 | |
| 5年最高 | 50+ | 2021年IPO初期 |

**谨慎评估**：当前PE-TTM 20.11倍处于5年历史最低区间以内。但需注意以下历史分位的参考意义修正：

1. **2021年IPO初期的高PE（50+）包含新股溢价，不可作为合理估值锚点**
2. **2022-2023年的PE中枢（35-45倍）对应当时40%+的营收增速，现在已经放缓至25-32%**
3. **H股IPO导致的ROE稀释（47%→22%）尚未完全反映在历史分位中**
4. **整体A股消费板块估值中枢在过去两年持续下移**

因此，合理PE区间的判断需结合PEG方法和可比公司分析，不能简单套用历史中位数。

## 三、核心估值计算

### 3.1 PE估值法（权重50%）

**可比公司对标**：

| 公司 | 2026E PE | 3年CAGR | PEG | 备注 |
|:---|:---|:---|:---|:---|
| 农夫山泉 | ~35x | ~20% | ~1.75 | 软饮龙头，品类多元 |
| 康师傅 | ~18x | ~5% | ~3.6 | 方便面+饮料，增速低 |
| 统一企业 | ~15x | ~5% | ~3.0 | 增速低，估值折扣 |
| 贵州茅台 | ~20x | ~10% | ~2.0 | 白酒龙头，确定性溢价 |
| **东鹏饮料** | **~20x** | **22%** | **~0.91** | 高增长低估值 |

**合理PE区间判断**：

- 东鹏净利润3年CAGR约22%，高于农夫山泉（20%）和茅台（10%）
- PEG低于1.0，与农夫山泉的PEG 1.75相比存在明显折价
- 折价原因：品类单一风险（能量饮料占75%）、PET成本不确定性、H股上市后的ROE稀释
- 给予10-15%的PEG折价后，合理PEG约1.0-1.1
- **合理Forward PE = 1.0~1.1 × 22% = 22-24倍**

**基准市值计算**：
- 2026E加权EPS: 6.3元
- 合理PE: 22-24倍
- **目标价（不复权市价）: 138.6 ~ 151.2元**
- **基准市值: 1012 ~ 1104亿**（基于约7.3亿总股本）

### 3.2 PEG估值法（权重25%）

PEG = Forward PE / 3年净利润CAGR（22%）

- 合理PEG: 1.0（消费品龙头正常定价）
- 隐含合理PE = 1.0 × 22 = **22倍**
- 目标价 = 22 × 6.3 = **138.6元**

若给予PEG=1.2（考虑品牌护城河+渠道壁垒溢价）：
- 合理PE = 26.4倍
- 目标价 = **166.3元**

### 3.3 DCF估值法（权重25%）

**关键假设**：

| 参数 | 假设值 | 依据 |
|:---|:---|:---|
| 预测期 | 10年（2026-2035） | 涵盖一个完整资本开支周期 |
| 永续增长率 | 3.0% | 饮料行业长期与GDP同步 |
| WACC | 9.0% | 无风险利率~2.8%+ERP~6.2% |
| 初始FCF（2026E） | 0亿 | 资本开支高峰 |
| 稳态FCF利润率 | 15% | 成熟消费品公司水平 |
| 终值占比 | ~55% | 合理的终值依赖度 |

**简化DCF估算**：

| 阶段 | 期间 | 净利润CAGR | FCF转化率 | 累计折现FCF（亿） |
|:---|:---|:---|:---|:---|
| 高增长期 | 2026-2028 | 22% | 30%（扩产期） | ~65 |
| 稳定增长期 | 2029-2032 | 12% | 60% | ~250 |
| 成熟期 | 2033-2035 | 5% | 75% | ~200 |
| 终值（永续） | 2035+ | 3% | 75% | ~650 |
| **企业价值** | | | | **~1165** |
| 减：净债务 | | | | +80（净现金） |
| **股权价值** | | | | **~1245亿** |

**DCF隐含目标价 = 1245 / 7.3 ≈ 170.5元**

### 3.4 三种方法综合

| 方法 | 权重 | 目标价区间 | 隐含PE |
|:---|:---|:---|:---|
| PE估值 | 50% | 138.6-151.2 | 22-24x |
| PEG估值 | 25% | 138.6-166.3 | 22-26.4x |
| DCF估值 | 25% | ~170.5 | ~27x |
| **综合目标价** | | **145-155元** | **23-25x** |
| **基准市值** | | **1058-1132亿** | |

## 四、全变量对冲审计矩阵（核心）

| 审计维度 | 核心审计因子 | 审慎思考（估值影响力过滤） | 调节幅度 | 深度理由 |
|:---|:---|:---|:---|:---|
| **内部：企业底层特质** | 渠道壁垒+数字化护城河 | 450万终端和五码合一体系是真实的稀缺资产，具有改变估值中枢的能力 | **+5%** | 快消行业中唯一实现全链路数字化的企业，数据资产和渠道掌控力超越了传统饮料公司的估值框架 |
| **内部：战略定力** | 品类扩张执行力 | 补水啦从0到33亿的爆品孵化能力已验证，果之茶复刻逻辑清晰，应享有平台型公司溢价 | **+3%** | 证明公司具备跨品类孵化爆品的系统能力，从单一能量饮料公司升级为综合性饮料平台 |
| **宏观：成本逆风** | PET涨价40%+ | 这是真实的短期利润压缩，但不改变长期竞争力，定性为阶段性扰动而非结构性折价 | **-3%** | 油价上行周期通常2-3年，PET在总成本中占比28%，影响时间有限 |
| **宏观：消费降级** | 性价比趋势 | 宏观逆风实际利好东鹏：消费者转向高质价比产品正是东鹏的核心定位 | **0%** | 这是商业模式优势，已在盈利预测中体现为收入增长，不应重复计入估值 |
| **市场：ROE稀释** | H股IPO权益翻倍 | ROE从47%降至22%是账面上的会计变化，不是经营恶化。但市场可能给予ROE下降的折价反应 | **-2%** | 这是结构性的估值下调压力。虽然会计ROE下降不代表盈利质量变差，但市场情绪需要时间适应 |
| **市场：流动性与风格** | A+H双上市 | 港股上市增加了外资配置渠道，扩大了投资者基础，长期有利于估值中枢提升 | **+2%** | 农夫山泉等港股消费品龙头的估值高于A股同业，双市场定价可能带来估值提升 |
| **宏观：地缘风险** | 中东冲突→油价 | 地缘不确定性增加了成本端的不可预测性，应给予适度折价 | **-2%** | 公司对石油价格零对冲能力，油价高位波动增加盈利波动性 |
| **产业：价格战风险** | 电解质饮料价格战 | 品类利润率可能低于预期，但东鹏的渠道优势使其在价格战中更可能胜出 | **-1%** | 短期利润可能承压，但长期竞争格局改善后利润率有望恢复 |
| **综合调节系数** | | | **+2%** | **修正目标市值 = 基准市值 × 1.02** |

### 4.1 调节说明

基准市值区间1058-1132亿，综合调节+2%，修正后：

**修正目标市值 = 1079-1155亿**
**修正目标价（不复权市价） = 148-158元**

### 5.2 盈亏比计算

- 当前股价（不复权市价）：**128.53元**
- 中性目标价：159元
- 保守目标价：104元
- 乐观目标价：216元

## 六、逻辑证伪与极限压力测试

### 6.1 黑天鹅场景定义

**场景：能量饮料消费习惯被替代性颠覆**

触发条件（需同时发生）：
1. 权威医学研究证实能量饮料与严重健康风险强相关（类似烟草的致癌定论）
2. 国家出台能量饮料广告禁令和消费年龄限制（类似烟酒管控）
3. 消费者恐慌性转向替代品（咖啡、茶饮）

### 6.2 极限支撑位审计

在此极端场景下，计算东鹏的地狱价格：

| 资产/指标 | 数值 | 清算价值（折价后） |
|:---|:---|:---|
| 货币资金 | 149.54亿 | 149.54亿（100%） |

| 土地/在建工程 | ~80亿 | 60亿（75%） |
| 存货 | 5亿 | 3亿（60%） |
| 减：负债合计 | 164亿 | 150亿（部分可清偿） |
| **净清算价值** | | **约162亿** |

**极限价格判断**：
- 净现金价值：货币资金149.54亿 - 有息负债（几乎为零）≈ 150亿，对应每股约20.5元
- **综合极限支撑位：20-22元/股（不复权市价）**
- 这意味着在当前128.53元下方，极端情况下仍有约85%的下跌空间（虽然概率极低）

### 6.3 非极端但现实的下行场景

如果以下条件同时触发（概率约15%）：
1. 补水啦增速骤降至20%（价格战+需求饱和）
2. PET维持9000元/吨以上
3. 能量饮料增速降至5%

则2026E净利润可能降至30-35亿，对应20倍PE的目标价约**82-96元**。

## 七、模块六核心结论

1. **综合目标价148-158元（不复权市价）**，较当前128.53元有15-23%的上行空间。

2. **当前PE 20.11倍处于历史极低分位**（低于5%分位26.78倍），PEG约0.91（低于1.0），估值具有一定吸引力但不是极度低估。

3. **全变量对冲审计后调节幅度仅+2%**，说明东鹏的护城河溢价（渠道+数字化+平台化）基本上被成本逆风和ROE稀释的折价所抵消。当前价位大体反映合理价值，缺乏显著的安全边际。

4. **盈亏比1.24:1**，在中性情景下上行空间有限（+24%），建议等待更好买点（如PET成本预期明朗化后的回调机会）。

5. **极限支撑位22元（净现金价值）**提供了极端的底线参考，但正常情景下若2026E业绩不及预期，股价可能下探至80-100元区间。

6. **催化剂关注**：2026Q2季报（验证补水啦旺季动销+PET成本影响）、补水中期份额数据、果之茶放量节奏。
