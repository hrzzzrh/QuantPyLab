# 研报章节三：外部边界与宏观政策

**研究日期**：2026年05月08日

**前序模块核心结论继承**：
- 章节一：云天化为磷化工全产业链龙头，100%磷矿自给率（约60-70亿/年成本优势），毛利率/净利率持续改善，有息负债四年减半至135亿，OCF持续2x净利润。PE 12.18x，股息率~4%。
- 章节二：行业处于"化肥稳量+新能源增量+资源涨价"三重属性交叠期。库存处于被动补库→主动去库过渡。磷矿石2025年紧平衡→2026年趋于宽松。硫磺是利润表最敏感单变量。核心护城河为"磷矿资源+一体化"，竞争对手无法复制。

---

## 一、宏观环境变量：温和再通胀与降息周期

### 1.1 利率/折现率环境

| 经济区 | 2026年政策方向 | 十年期国债利率参考 | 对估值的影响 |
| :--- | :--- | :---: | :--- |
| 中国 | "适度宽松"货币政策，降息降准有空间 | 1.70%-2.0% | 低利率环境支撑权益资产估值 |
| 美国 | 处于降息周期，预计降息2-4次(50-75bp) | ~4.0% | 全球流动性宽松利好新兴市场 |

**对云天化估值定价的传导**：
- 国内无风险利率（~1.8%）处于历史低位，以DCF框架计算，折现率每下降0.5个百分点，云天化这样的稳定现金流资产（年OCF 100亿+），理论估值可提升约8-12%。
- 但云天化作为周期属性较强的化工股，市场给予的折现率通常高于消费/科技股。利率下行对云天化的估值提振效应弱于纯成长股，但强于银行/地产等高杠杆板块。

### 1.2 人民币汇率与损益传导

2026年人民币主流预期为**渐进升值**，年底或升至6.80（vs 当前约7.0），升值幅度约3%。对云天化形成"此消彼长"的对冲效应：

| 通道 | 影响方向 | 量级估算 |
| :--- | :---: | :--- |
| **出口端**：磷肥出口（年约100-150万吨） | 负面影响 | 人民币每升值1%→出口收入减少约1.5-2亿人民币 |
| **进口端**：硫磺进口（年约204万吨） | 正面影响 | 人民币每升值1%→硫磺进口成本节省约2-3亿人民币 |
| **净效应** | **大致中性偏正** | 进口（硫磺）敞口略大于出口（磷肥）敞口 |

此外，人民币升值有助于降低美元计价债务的偿还成本（公司目前外币负债规模有限），但对以美元计价的海外磷肥销售价格形成压力。

### 1.3 通胀传导效率

2026年中国预计处于**"温和再通胀"**环境（CPI+0.5%，PPI-0.9%），而非高通胀。对云天化的影响：

- **成本端**：PPI虽仍为负但降幅收窄，工业品通缩压力减轻。但硫磺、硫酸等上游原料价格远超CPI/PPI平均水平（硫磺同比涨幅超100%），公司面临的是"结构性成本通胀"而非"系统性通胀"。
- **产品端**：磷肥价格受制于"保供稳价"政策天花板，难以将硫磺成本上涨完全传导至终端农户——这意味着云天化在宏观层面表现出"**上游通胀、下游通缩**"的利润挤压特征。

## 二、地缘政治风险：霍尔木兹海峡——硫磺供应链的"黑天鹅"

### 2.1 事件背景与现状

这是当前对云天化利润表**影响烈度最高的外部变量**。

| 时间节点 | 事件 |
| :--- | :--- |
| 2026年2月28日 | 美以对伊朗发动军事打击，伊朗封锁霍尔木兹海峡 |
| 2026年3月-5月 | 海峡通行效率极低，全球约56%的海运硫磺贸易受阻 |
| 当前状况（2026年5月） | 海峡封锁已持续69天+，美伊谈判僵局 |

### 2.2 硫磺供应链冲击量化

| 指标 | 2024年 | 2025年 | 2026年（YTD） |
| :--- | :---: | :---: | :---: |
| 镇江港硫磺均价（元/吨） | ~1,091 | 2,607 | 4,328 |
| 同比涨幅 | — | +139% | +66%（较2025年） |
| 中国进口硫磺均价（美元/吨） | ~116 | 265 | 1,000+（现货） |
| 国际硫磺CIF成交（美元/吨） | — | — | 1,000-1,200 |

**对云天化的利润冲击测算**：

- 年硫磺外购量：约204万吨（自产目标75万吨预计2026年尚未完全达产）
- 价格假设：2026年均价4,000元/吨（保守）vs 2024年均价1,091元/吨
- 增量成本：(4,000-1,091) × 204万 ≈ **59.3亿元**（vs 2024基准）
- 即使考虑长协锁量100万吨（~150美元/吨≈1,050元/吨），额外100万吨现货采购的增量成本仍达约29亿元

**这是2025Q4业绩骤降（归母净利仅4.27亿元，同比-53.2%）的核心原因。**

### 2.3 云天化的防御工事与脆弱性

| 防御手段 | 覆盖量 | 有效性评估 |
| :--- | :---: | :--- |
| 中东长协锁价（150美元/吨） | 100万吨 | ★★★★☆ 覆盖约50%需求 |
| 推进自产硫磺 | 目标75万吨/年 | ★★★☆☆ 2026年预计部分达产 |
| 磷石膏制硫酸 | 中试/示范阶段 | ★★☆☆☆ 远期替代，短期贡献有限 |
| 哈萨克斯坦/土库曼斯坦替代进口 | 运力有限 | ★★☆☆☆ 非主流来源 |

**核心脆弱性**：仍有约100万吨/年的硫磺需从现货市场采购，完全暴露于地缘溢价。若海峡封锁持续至2026年下半年，公司2026年全年利润可能被压缩至40亿以下（vs 2025年51.56亿）。

### 2.4 地缘政治的"反身性"机会

如果中东局势意外缓和（如美伊达成协议、海峡恢复通航），硫磺价格可能从当前5,000+元/吨快速回落至2,000元/吨以下——这将释放可观的利润弹性：硫磺每跌1,000元/吨，云天化年化税前利润增厚约10亿元（按100万吨现货采购计）。

**此即为云天化当前最大的"困境反转"期权价值。**

## 三、贸易摩擦与供应链重构

### 3.1 美国磷肥反补贴税（CVD）

- 美国对摩洛哥（19.97%）和俄罗斯（9.19%-47.05%）磷肥持续征收反补贴税，2021年3月起执行。
- 2026年3月启动"日落复审"，ITC预计2027年春作出最终裁定。
- **对中国的影响**：中国磷肥因自身出口限制已基本退出美国市场，CVD存废对中国磷肥企业的直接影响极为有限。但若CVD取消→摩洛哥磷肥重返美国→挤压其他出口市场→间接增加中国在东南亚/南美等市场的竞争压力。

### 3.2 全球硫磺/硫酸出口管制升级

- 俄罗斯多次延长硫磺/硫酸出口禁令（最近延长至2026年6月）。
- 中国自2026年5月起全面限制硫酸出口（仅电子级硫酸除外），优先保障国内化肥生产。
- 全球硫磺贸易已从"市场化定价"切换至"安全优先+政策干预"模式。

### 3.3 中国磷肥出口的"结构性退场"趋势

中国磷肥出口从2015年约1,100万吨（实物）降至2025年仅约450万吨（实物），十年降幅近60%。这不仅是周期性波动，更是**政策驱动的结构性转变**——磷矿被列为战略性矿产、保供稳价优先于出口创汇、环保约束趋严三者共振的结果。

对云天化而言：出口量下降（约占总销量20-25%）意味着更依赖国内市场，而国内市场受政策价格天花板约束，利润空间相对有限。**出口溢价的阶段性消失是中期利润压制因素。**

## 四、监管与产业政策：政策红利与隐性成本

### 4.1 西部大开发税收优惠（核心政策红利）

云天化主营业务完全落入《西部地区鼓励类产业目录（2025年本）》云南省鼓励类条目（精细磷化工、电子级磷酸、六氟磷酸锂、磷石膏建材等），享受**15%企业所得税优惠税率**，政策有效期明确延续至**2030年12月31日**。

| 优惠类型 | 内容 | 对净利润的影响 |
| :--- | :--- | :--- |
| 企业所得税 | 减按15%征收（标准25%） | 年节省约5-6亿元 |
| 增值税（磷石膏） | 即征即退70% | 相对较小 |
| 西部大开发 | 鼓励类产业确认 | 政策确定性高（至2030年） |

以2024年利润总额70.57亿为基数，15% vs 25%税率的税差约7.06亿元——相当于归母净利润的13.2%。**这是云天化相比东部非鼓励类化工企业的结构性政策红利。**

### 4.2 环保成本：磷石膏是"房间里的大象"

每生产1吨磷酸产生4.5-5吨磷石膏，全国累计堆存量超8亿吨，云天化年产生量估计在1,500-2,000万吨级别。

政策时间表：
- **2026年**：综合利用率达65%，产消动态平衡
- **2027年**：云南等6省市完成存量磷石膏库整治
- **2030年**：全国60%以上历史遗留堆场治理完成

云天化在磷石膏综合利用（制建材、制硫酸、矿井充填）方面的投资进度，直接影响未来的环保合规成本。若2027年整治期限无法达标，可能面临限产或罚款风险。但目前公司在磷石膏制硫酸联产水泥方面已有技术储备，且享受增值税即征即退70%的税收优惠。

### 4.3 "保供稳价"——悬在头顶的双刃剑

| 维度 | 利好 | 利空 |
| :--- | :--- | :--- |
| 出口限制 | 保障国内供给，稳固市场份额 | 失去出口溢价（内外价差1,400-1,900元/吨） |
| 国储投放 | 平抑旺季价格波动 | 压制磷肥价格上行空间 |
| 配额分配 | 头部企业获更大出口份额 | 政策不确定性高，难以做长期出口规划 |

总体来看，中国化肥产业政策的核心导向是"**以国内保供稳价为绝对优先**"，云天化作为行业龙头在配额分配中处于有利地位，但政策的不可预测性构成持续的经营不确定性。

### 4.4 国家战略契合度评分

| 国家战略方向 | 云天化契合度 | 理由 |
| :--- | :---: | :--- |
| 粮食安全（化肥保供） | ★★★★★ | 国内磷肥龙头，保供核心力量 |
| 新能源转型（磷酸铁锂） | ★★★★☆ | 10万吨磷酸铁产能+六氟磷酸锂布局 |
| 战略性矿产安全（磷矿） | ★★★★★ | 8亿吨+24亿吨远景储量 |
| 绿色制造（磷石膏利用） | ★★★☆☆ | 有技术储备但进度需观察 |
| 西部大开发 | ★★★★★ | 15%税率至2030年 |
