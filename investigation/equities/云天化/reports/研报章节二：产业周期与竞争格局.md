# 研报章节二：产业周期与竞争格局

**研究日期**：2026年05月08日

**前序模块核心结论继承**：云天化是磷化工全产业链龙头，100%磷矿自给率构成结构性成本优势（约60-70亿元/年）。盈利质量持续改善（毛利率从13.68%→20.21%）但营收因商贸剥离连续三年下滑。硫磺成本是利润表最敏感变量。资产负债表质变：有息负债四年减半至135亿。当前PE 12.18x，PB 2.43x，分红率约50%，股息率~4%。

---

## 一、产业生命周期：从"成熟周期"到"战略资源+新能源成长"的三重属性切换

磷化工行业正经历历史性的属性重塑，从传统"化肥周期品"的单一标签，转向"战略资源+新能源成长+农化刚需"的三重属性复合体。

### 1.1 传统化肥维度：成熟期后半段

- 中国磷肥总产能约2,200万吨（折纯），2025年产量1,869万吨，产能利用率约85%，供需基本平衡但缺乏总量增长空间。
- 农业部"化肥零增长"政策长期约束，国内磷肥消费量已见顶并缓慢下行（年降幅约1-2%）。
- 行业CR10磷肥产量集中度63.4%，磷酸二铵CR10更高达94.6%——**寡头格局已定，增量博弈转为存量博弈**。

### 1.2 新能源材料维度：成长期前半段

- 磷酸铁锂（LFP）正极材料对磷源的需求处于爆发性增长阶段。2025年中国磷酸铁锂产量约390万吨，预计2026年达600万吨（+54%），对应磷矿石需求约1,650万吨（占国内产量~12%）。
- 储能电池出货量从2025年600GWh增至2026年800GWh（+33%），为磷需求提供第二增长极。
- 精制磷酸/工业磷酸一铵作为LFP前驱体，正在从"副业"升级为磷化工企业的核心增长引擎。

### 1.3 战略资源维度：供给刚性定价

- 中国磷矿储量仅占全球约5%，储采比仅34年（全球平均~308年）。磷矿已被列入中国战略性矿产目录。
- 30%以上高品位富矿占比不足10%，平均品位仅16.9%——低品位矿选矿成本高，实质上构成产能释放的隐性约束。

**综合判断**：云天化所处的磷化工行业正处于"化肥稳量+新能源增量+资源涨价"的黄金交叉窗口期。传统化肥提供稳定现金流（年OCF 100亿+），新能源材料贡献成长期权，磷矿资源赋予定价权护城河。

## 二、库存周期定位：被动补库→主动去库过渡期

### 2.1 当前库存状态

| 指标 | 2025年Q3状态 | 方向判断 |
| :--- | :--- | :--- |
| 磷酸一铵样本企业库存 | >23万吨，单周增超5万吨 | ↑ 被动累积 |
| 磷肥企业产能利用率 | 66.14%（近4年最高） | ↑ 高开工 |
| 下游复合肥产能利用率 | 39.22%（环比-1.62pct） | ↓ 旺季不旺 |
| 复合肥样本企业库存 | 82.62-86.72万吨（同比+5-9万吨） | ↑ 去库压力大 |

**核心矛盾**：上游磷肥企业维持高开工率，但下游复合肥需求疲弱且自身库存高企，导致磷肥库存在生产端被动累积——这是典型的**"被动补库存"**特征。

### 2.2 2026年库存周期推演

| 时间段 | 库存阶段 | 核心驱动 |
| :--- | :--- | :--- |
| 2026Q1（春耕备肥） | 主动去库→局部补库 | 春耕旺季消耗渠道库存，磷肥企业可能降价促销 |
| 2026Q2（春耕收尾） | 被动去库/弱平衡 | 需求回落，出口仍暂停（至8月底） |
| 2026Q3（秋销备肥） | 主动补库启动？ | 取决于渠道库存去化程度和出口政策是否调整 |
| 2026Q4 | 取决于8月后出口政策 | 若出口放开→主动补库加速；若延续限制→持续弱平衡 |

**辨析：订单波动是真实需求还是牛鞭效应？**

当前磷酸一铵库存累积，很大程度上是**牛鞭效应的逆向传导**：复合肥企业因2025年秋销"旺季不旺"遭遇销售受阻→复合肥库存积压→减少对上游磷肥的采购→磷肥企业库存被动累积。并非终端农户施肥量骤降（农业用肥为刚性需求），而是渠道环节的库存波动被人为放大。这意味着一旦渠道库存消化完毕（预计2026年Q1-Q2春耕），补库需求将具有一定弹性。

## 三、终端需求穿透与跨期断层审计（Mandatory）

### 3.1 最终产品画像

云天化的核心产品最终负载于以下终端形态：

| 公司产品 | 最终用途 | 终端驱动因素 |
| :--- | :--- | :--- |
| 磷肥（DAP/MAP） | 粮食种植（玉米/大豆/水稻/小麦） | 全球人口增长、粮食安全、种植面积 |
| 尿素 | 粮食种植（追肥） | 同上 + 能源价格（煤/气头成本） |
| 复合肥 | 经济作物/大田作物 | 农业结构升级、测土配肥 |
| 饲料级磷酸钙盐 | 畜禽饲料添加剂 | 畜牧业规模、肉类消费 |
| 聚甲醛 | 汽车/电子/家电零部件 | 制造业景气度、工程塑料替代 |
| 磷酸铁 | 动力电池/储能电池正极 | 新能源车渗透率、储能装机量 |
| 黄磷 | 除草剂/阻燃剂/电子级磷酸 | 农药需求、防火安全、半导体 |

### 3.2 终端景气度审计：未来1-3年展望

**农业需求（占磷消费约60%）——刚性但受"可负担性"约束**：

- 2026年全球化肥（磷肥）可负担性指数处于2008年以来最差水平。美国玉米种植者化肥成本预计$166/英亩（+5.3%），大豆$57/英亩（+5.2%）。
- Rabobank警告可能出现"需求破坏"——农户减少施用量或改种低肥耗作物。但中国化肥市场具有特殊性：国家"保供稳价"政策压制国内磷肥价格低于国际20-30%，国内农户可负担性相对优于海外。
- 全球粮食库存仍处低位，农产品价格维持中高水平，对化肥需求构成支撑。2026年春耕中国化肥需求量预计约5,600万吨（+3%）。

**新能源需求（占磷消费约12%，增速最快）——结构性高景气**：

- 2026年中国磷酸铁锂产量预计达600万吨（+54%），全球储能电池出货量800GWh（+33%）。
- 磷酸铁对磷矿石的年增量需求约400-500万吨/年，是磷矿石需求端增长最快的细分领域。
- 六氟磷酸锂价格从2025年低点约4.7万元/吨涨至2026年初约18万元/吨（涨幅近300%），氟化工产业链景气度极高。

### 3.3 动能溯源与透支审计

**增长动力溯源**：

| 增长来源 | 内生/外生 | 可持续性评估 |
| :--- | :--- | :--- |
| 磷矿资源自给（成本优势） | 内生 | ★★★★★ 持续性极强 |
| 全球化肥价格高位 | 外生（地缘冲突驱动） | ★★★☆☆ 地缘溢价可能消退 |
| 磷酸铁锂需求爆发 | 内生（技术趋势）+ 外生（政策补贴） | ★★★★☆ 技术趋势确定性高 |
| 中国磷肥出口限制 | 外生（政策干预） | ★★☆☆☆ 政策随时可能调整 |
| 硫磺价格飙升（挤压利润） | 外生（中东地缘） | ★★★☆☆ 中期维持高位 |

**跨期需求透支审计（强制项）**：

1. **"出口限制→国内供应过剩→价格承压"的跨期错配风险**：2026年3-8月磷肥出口原则上"零出口"，约350-400万吨磷肥回流国内。若8月后出口集中放量，可能造成"先过剩→后抢出口"的跨期波动。对云天化的影响：出口量下降约20-25%，但国内销售占比提升，总体销量受影响有限。

2. **新能源补贴政策的"需求前置"风险**：锂电池产业链过去曾出现"补贴退坡→抢装→透支后续需求"的模式。当前储能需求高增部分受国内强制配储政策驱动，若政策力度减弱，2027年磷酸铁锂需求增速可能放缓至20-30%（vs 2026年的54%），但绝对增量仍为正。

3. **硫磺"恐慌性备货"风险**：中东地缘冲突下，国内磷肥企业可能在2025Q4-2026Q1大量采购硫磺以锁定成本，形成短期"恐慌性补库"，但这更多影响成本节奏而非需求透支。

**审计结论**：未发现严重的跨期需求透支风险。农业需求为绝对刚需，新能源需求虽然存在政策驱动成分，但磷酸铁锂的技术路线优势（低成本+安全）已是市场自发选择，补贴退坡不会改变长期趋势。

## 四、供需边际与压力传导

### 4.1 磷矿石：2025年紧平衡→2026年趋于平衡

| 年份 | 预计产量(万吨) | 预计需求量(万吨) | 供需比 | 价格预判 |
| :--- | :---: | :---: | :---: | :--- |
| 2025 | 12,038 | 11,833 | 98.3% | 1,000-1,100元/吨（高位） |
| 2026 | 13,633 | 12,307 | 90.3% | 900-1,000元/吨（温和下行） |
| 2027 | 15,173 | 12,504 | 82.4% | 700-900元/吨（压力增大） |

**对云天化的影响**：公司磷矿100%自给且可外售部分精矿，磷矿石价格温和下行对公司自用成本无影响，但外售收入可能小幅减少。更重要的是：磷矿石价格下行将削弱无矿竞争对手的成本劣势，行业竞争可能加剧。但云天化8亿吨储量的碾压级资源优势在5-10年内仍不可撼动。

### 4.2 磷肥：出口限制下的"政策顶"与"成本底"博弈

- **成本底**：硫磺价格高位（2026年3月镇江港约4,800元/吨）构筑磷肥生产的刚性成本底线。
- **政策顶**：出口暂停+1,000万吨国储投放构成供应压力，磷肥价格难以大涨。
- **结论**：磷酸一铵维持4,000元/吨以上，磷酸二铵4,750-4,850元/吨区间窄幅震荡，"大涨大跌均难现"。

### 4.3 压力传导特征分析

**传导路径：国际地缘→硫磺成本→磷肥利润→企业开工意愿**

1. **波动放大效应**：中东地缘冲突→硫磺价格暴涨（从2024年均价~1,200元→2026年3月4,800元/吨，涨幅300%）→云天化磷肥成本端被急剧放大。这是在需求端（化肥价格）相对稳定情况下的单向成本挤压，属于"供给冲击→利润压缩"的经典传导。

2. **传导时滞与博弈筹码**：
   - 云天化通过**长期合同锁价**（中东长协150美元/吨锁定100万吨）和**推进硫酸/硫磺自产**（磷石膏制硫酸项目、目标自产硫磺75万吨/年）来延缓和对冲传导。
   - 但硫磺外购量仍约100万吨/年，2026年硫磺成本同比2024年可能增加约20-30亿元，是不可忽视的利润拖累。
   - 磷矿100%自给是公司最大的博弈筹码——在行业普遍受困于"磷矿涨价+硫磺涨价"的双重挤压时，云天化只需面对硫磺的单一变量，竞争格局中处于相对优势。

## 五、竞争格局与护城河

### 5.1 行业集中度与竞争版图

| 细分领域 | CR3 | CR10 | 云天化排名 |
| :--- | :---: | :---: | :---: |
| 磷肥总产量(折纯) | ~35% | 63.4% | **第1名** |
| 磷酸二铵产量 | ~65% | 94.6% | **第1名**（445万吨/年） |
| 磷酸一铵产量 | ~30% | 59.4% | 第3名 |
| 复合肥产量 | — | 54.1%(CR15) | 第7名 |

### 5.2 核心竞争对手对比

| 维度 | **云天化(600096)** | 兴发集团(600141) | 新洋丰(000902) |
| :--- | :--- | :--- | :--- |
| 磷矿储量 | **近8亿吨** | 3.95亿吨 | 相对有限 |
| 磷矿自给率 | **100%** | ~70% | <30% |
| 核心优势 | 资源规模+全产业链 | 电子级磷酸+草甘膦 | 复合肥渠道+一铵 |
| 成长逻辑 | 磷矿资源壁垒+高分红 | 高端化+进口替代 | 渠道壁垒+农服 |
| 2025年ROE | 21.87% | ~18% | ~15% |
| PE(TTM) | 12.2x | ~25x | ~15x |

云天化在资源端对竞争对手形成**代差级压制**——8亿吨储量 vs 兴发3.95亿吨 vs 新洋丰主要外购。更关键的是，参股公司聚磷新材已取得镇雄碗厂磷矿采矿权（24.38亿吨，品位22.54%），集团承诺3年注入，届时公司远景磷矿储量将跃升至30亿+吨级别，超越所有A股同行。

### 5.3 护城河识别

| 护城河类型 | 强度 | 证据 |
| :--- | :---: | :--- |
| **成本规模壁垒** | ★★★★★ | 磷矿100%自给，成本优势60-70亿/年 |
| **资源独占性** | ★★★★★ | 8亿吨储量+24亿吨远期，磷矿为战略性稀缺资源 |
| **产业链一体化** | ★★★★☆ | 矿-酸-肥-化工-新材料全链条，协同降本 |
| **转换成本** | ★★☆☆☆ | 化肥产品标准化，客户转换成本低 |
| **品牌溢价** | ★★☆☆☆ | 化肥标准化程度高，品牌溢价有限 |
| **特许经营权** | ★★★☆☆ | 磷矿采矿权、出口配额均具有政策壁垒 |

**核心护城河是"磷矿资源+一体化"组合**：竞争对手无法在合理时间内（新建大型磷矿需5-8年审批+建设周期）复制公司的资源禀赋。

### 5.4 产业链定价权验证

| 议价维度 | 数据证据 | 判断 |
| :--- | :--- | :--- |
| 对上游(磷矿) | 100%自给，无需外购 | ★★★★★ 完全掌控 |
| 对上游(硫磺) | 50%依赖进口，长协锁量100万吨 | ★★☆☆☆ 受制国际市场 |
| 对下游(经销商) | 合同负债20.51亿(2024)，预收模式 | ★★★★☆ 议价能力强 |
| 对下游(磷酸铁客户) | 新能源材料高速增长，产品紧俏 | ★★★★☆ 供不应求 |
| 应付账款 | 50.73亿(2024)，压款能力增强 | ★★★☆☆ 中等偏强 |

**应收账款从9.03亿(2023)升至16.71亿(2024)，需关注是否因销售压力而放宽信用政策。但合同负债（预收款）20.51亿元的存在，说明核心化肥业务仍维持预收模式。**

## 六、强制复核确认

1. ✅ **终端产品景气度**：已穿透并引用了农业种植需求（全球化肥可负担性、春耕需求量5,600万吨）和新能源需求（LFP产量600万吨、储能800GWh）的未来两年预测。
2. ✅ **跨期需求审计**：已审计出口限制→国内供应过剩的错配风险、新能源补贴退坡的需求前置风险。结论：未发现严重的跨期透支。
3. ✅ **供需边际与压力传导**：已基于磷矿石供需平衡表（98.3%→90.3%→82.4%）分析了紧平衡→宽松的演变方向，并对硫磺成本→磷肥利润→企业开工的传导路径进行了特征分析。
