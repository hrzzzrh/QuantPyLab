## 一、价值锚定：适配性方法选择

### 1.1 方法论工具箱与选择理由

| 方法 | 适用性 | 选择 | 理由 |
| :--- | :---: | :---: | :--- |
| **PE（市盈率）** | ★★★★★ | ✅ 核心锚 | 盈利稳定（50亿级别）、分红清晰，周期股主流定价方法 |
| **PB-ROE** | ★★★★★ | ✅ 交叉验证 | 重资产属性（固定资产占比40%），ROE与PB存在理论映射 |
| **DCF（现金流折现）** | ★★★★☆ | ✅ 辅助验证 | OCF持续2x净利润，"隐藏"的现金创造能力需DCF揭示 |
| **股息率模型** | ★★★☆☆ | 参考 | 分红比例~50%、股息率~4%，适合防御型投资者估值 |
| **EV/EBITDA** | ★★★☆☆ | 参考 | 重资产+高折旧，EBITDA更能反映经营实力 |

### 1.2 PE 估值：历史分位审计与合理中枢判定

**历史PE分位（2020年以来，月度均值）**：

| 分位 | PE值 | 对应时期 |
| :--- | :---: | :--- |
| 最低 | 5.06x | 2021年利润爆发期（分母失真） |
| 25分位 | 7.59x | 极端悲观区域 |
| 中位数 | 9.55x | 2022-2024年大部分时期 |
| 75分位 | 25.32x | 2021年及2024年阶段性高位 |
| **当前** | **12.18x** | 2026年5月8日 |

**历史分位的参考局限性**：
1. 2020-2021年PE极大值（>25x甚至>100x）对应的是微利时期（2020年净利润仅2.72亿），分母失真，参考意义为零。
2. 2022-2024年PE中位数9.55x对应的是：高杠杆（63-76%负债率）+高利润波动（大宗商品价格震荡期）的市场定价。
3. **当前估值环境已发生质变**：负债率降至47%→财务更稳健、分红率提升至~50%→类"现金牛"属性增强、磷矿资源战略价值重估。

**合理PE中枢判定**：

| 参照系 | 当前PE | 合理区间 | 逻辑 |
| :--- | :---: | :---: | :--- |
| 历史中位数 | 9.55x | 8-12x | 过于保守，未反映资产负债表质变 |
| 同行兴发集团 | ~25x | — | 不具可比性（兴发含电子化学品溢价） |
| 同行新洋丰 | ~15x | — | 较可比，云天化资源更优 |
| 国际化肥巨头（Nutrien/Mosaic） | 12-16x | — | 云天化成长性更好 |
| **综合判定** | **12-14x** | **11-15x** | **考虑财务改善+资源稀缺溢价+分红价值** |

**基准市值（PE法）**：
- 11x PE（保守）：50.6亿 × 11 = 556.6亿 → 股价 30.54元
- 13x PE（基准）：50.6亿 × 13 = 657.8亿 → 股价 36.09元
- 15x PE（乐观）：50.6亿 × 15 = 759.0亿 → 股价 41.64元

### 1.3 PB-ROE 交叉验证

PB-ROE理论公式：PB = (ROE - g) / (r - g)

| 参数 | 假设值 | 说明 |
| :--- | :---: | :--- |
| ROE | 19% | 2026E加权ROE（权益增厚→ROE自然回落） |
| 永续增长率g | 3% | 保守假设，略高于通胀 |
| 折现率r | 10% | 化工周期股通常的权益成本 |
| **理论PB** | **2.29x** | (19%-3%)/(10%-3%) |

当前实际PB 2.43x vs 理论PB 2.29x——略高于理论值约6%，处于合理略偏高区间。以2026E BVPS约15.5元计算：
- 理论价值 = 15.5 × 2.29 = **35.50元**

### 1.4 DCF 辅助验证

| 假设 | 值 | 说明 |
| :--- | :---: | :--- |
| 基准FCF | 75亿/年 | OCF 100亿 - 维持性Capex 25亿 |
| 增长阶段 | 前5年增长3% | 磷肥稳定+新能源补充 |
| 永续增长率 | 2% | 保守 |
| WACC | 8.5% | 低杠杆下综合资本成本 |

简化DCF估算（两阶段）：
- 前5年FCF现值：75×1.03/(1.085) + ... ≈ 320亿
- 终值：75×1.03^5×(1.02)/(0.085-0.02)/(1.085^5) ≈ 780亿
- 企业价值 ≈ 1,100亿
- 减：净负债（有息负债135亿-现金69亿）= 66亿
- 股权价值 ≈ 1,034亿 → 股价 **56.7元**（明显高于PE/PB估值）

**DCF高估原因**：云天化作为周期股，FCF可能被近年高利润期夸大。若将FCF标准化为55亿（近5年均值），则股权价值约760亿→股价**41.7元**——与PE 15x的估值接近。

## 二、全变量对冲审计矩阵

### 2.1 审计原则声明

以下审计严格区分：
- **进"E"的因子**：具体、可计算的钱（增减）→已在章节四的盈利预测中体现
- **进"PE"的因子**：关乎信心、不确定性、稀缺性的因素→在本矩阵中进行估值倍数调节

### 2.2 估值影响力过滤与审计调节

| 审计维度 | 核心因子 | 估值影响力过滤 | 调节幅度 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部DNA：资源壁垒** | 磷矿100%自给（8亿吨+24亿吨远景） | ✅ **真正稀缺**。磷矿为战略性矿产，全球富矿短缺。云天化的资源体量在A股独一无二，享资源稀缺溢价 | **+8%** | 类似茅台之赤水河、神华之神东煤矿——资源独占性应享受结构性的PE溢价 |
| **内部DNA：财务质量** | 有息负债从273亿→135亿，OCF 2x NP | ✅ **真正改善**。资产负债表质变意味着抗周期能力飞跃，降低破产风险→降低折现率 | **+5%** | 去杠杆降低尾部风险，DCF模型中的股权成本可从10%降至9%——这直接体现为PE抬升 |
| **内部DNA：分红文化** | 分红比例~50%，股息率~4%，连续3年提升 | ⚠️ **边际影响**。高分红对化工股PE的提升有限（市场不会按"类债"给化工股定价），但能吸引红利资金 | **+2%** | "红利+"标签引入新边际买家（险资、养老金），小幅提升估值中枢 |
| **内部DNA：第二曲线** | 新能源材料（磷酸铁+氟化工） | ⚠️ **尚待证明**。营收增速快（+75%）但利润贡献极小（模拟净利0.62亿），市场不会给予估值溢价除非利润占比超10% | **+0%** | 属于"期权价值"而非"确定性溢价"。当前阶段不进PE调节，作为免费附赠的成长期权看待 |
| **宏观：政策对齐** | 西部大开发15%税率、保供稳价核心企业、出口配额优先 | ✅ **真正加分**。15%税率为确定性红利（至2030年），保供核心地位意味着政策不会"伤害"头部企业 | **+3%** | 15% vs 25%税差→净利润增厚13%。这不是一次性的，而是制度性优势→应反映在估值中 |
| **宏观：地缘风险** | 霍尔木兹海峡→硫磺暴涨 | ✅ **实质性折价**。海峡封锁已持续69天，硫磺现货5,000+/吨。这是可量化的利润冲击+不可量化的悲观情绪 | **-5%** | 地缘不确定性是周期股最大的折价来源。即使利润冲击已部分反映在EPS中，情绪面的PE压抑仍需体现 |
| **宏观：出口政策** | 磷肥出口"零出口"至8月，政策反复 | ⚠️ **已在EPS中体现**。出口政策影响的是"量"和"价差"（进E），不应重复计入PE | **+0%** | 出口限制对利润的影响已通过销量假设和价格假设反映在EPS预测中 |
| **市场：风格契合** | 周期+红利+磷矿资源主题 | ⚠️ **短期风格**。2026年市场风格偏科技成长，周期/红利非当前主线，但资源主题偶有风口 | **-2%** | 市场风格偏向科技成长，化工周期股的估值被阶段性压制。但这是短暂的，不应给予过大权重 |
| **市场：流动性** | 自由流通市值约400亿，日均成交额约3-5亿 | 中性。流动性充裕，不存在折价 | **+0%** | 大盘股，沪股通标的，流动性充足 |
| **综合调节系数** | — | — | **+11%** | — |

### 2.3 修正目标市值与股价

修正目标市值 = 基准市值 × (1 + 11%) = 665亿 × 1.11 = **738亿**

**修正目标股价（不复权）= 738亿 / 18.23亿股 = 40.48元**

检验该市值隐含的估值倍数：
- 2026E PE = 738 / 50.6 = **14.6x**（在合理区间上沿，有资源溢价支撑）
- 2026E PB = 738 / (18.23 × 15.5) = **2.61x**（PB-ROE框架下可接受）
- 2026E股息率 = (738 × 50% / 18.23) / 40.48 ≈ **约3.1%**

## 三、逻辑纯净性校验

### 3.1 E vs PE 分配审计

| 因子 | 进E还是进PE？ | 处理方式 | 校验 |
| :--- | :--- | :--- | :--- |
| 硫磺成本上升 | **进E** ✅ | 已在Base Case中通过降低磷肥毛利率（33%）体现 | 不进PE是对的——这是具体可计量的成本 |
| 磷矿自给优势 | **进PE** ✅ | 通过+8%的资源稀缺溢价体现 | 进PE是对的——这是结构性优势，不随年度利润波动 |
| 出口限制 | **进E** ✅ | 已在Base Case中通过销量和内外价差假设体现 | 不进PE是对的——影响的是"量×价" |
| 地缘不确定性 | **进PE** ✅ | 通过-5%的地缘折价体现 | 进PE是对的——"不知道海峡什么时候开"是信心问题 |
| 财务改善 | **进PE** ✅ | 通过+5%的财务质量溢价体现 | 进PE是对的——更稳健的资产负债表降低的是折现率 |

**校验结论**：E与PE的分配无重复计算，各因子均归入正确的定价层级。

### 3.2 锚点回测

修正后隐含PE 14.6x vs 历史中位数 9.55x——溢价53%。这一溢价是否合理？

参考系验证：
- A股化工龙头（万华化学）：历史PE均值约18-20x → 云天化14.6x低于此
- 国际磷肥巨头（Mosaic）：PE约12-14x → 云天化14.6x与之相当，但云天化资源+成长性更优
- A股煤炭龙头（中国神华）：PE约10-12x → 云天化估值略高，但云天化多了新能源成长期权

**结论**：14.6x PE在磷化工行业具备合理性，主要由"磷矿资源壁垒溢价"支撑，不存在明显泡沫。

> 注：乐观情景目标PE低于Base是因为"利好兑现"后部分资源/政策溢价收窄；悲观情景目标PE为12x（更接近历史中位数）是因为"危机模式下溢价逻辑瓦解"。

### 4.2 盈亏比计算

- 当前股价（不复权）：34.77元
- 概率加权目标价：38.39元
- 悲观情景目标价：26.28元
- **盈亏比 = (38.39 - 34.77) / (34.77 - 26.28) = 3.62 / 8.49 = 0.43**

盈亏比<1，在传统意义上并不吸引人——这意味着上行空间（+10.4%）远小于下行风险（-24.4%）。但这一计算的局限性在于：悲观情景（硫磺$5,000+全年）的概率仅25%，且目标价为12个月后的预期。

若以Base Case为锚：盈亏比 = (40.59-34.77)/(34.77-26.28) = 5.82/8.49 = 0.69，仍低于1。

**这是云天化当前投资的核心矛盾**：给定硫磺的地缘溢价高度不确定，盈亏比并不吸引人；但公司的资源壁垒和财务质量提供了厚实的"地板"，而硫磺价格的任何回落都可能触发显著的利润弹性。

## 五、逻辑证伪与极限压力测试

### 5.1 黑天鹅场景定义

**"一击毙命"事件**：**全球磷肥需求出现结构性崩塌**。

触发条件组合：
1. CRISPR基因编辑技术突破→全球粮食产量暴增30%→化肥需求断崖式下降
2. 或：磷酸铁锂电池被钠离子/固态电池全面替代→新能源磷需求归零
3. 叠加：磷矿被移出战略性矿产目录→资源溢价逻辑瓦解

在此场景下，公司年净利润可能从50亿级别降至15-20亿，市场给予8x PE，市值约120-160亿。

**现实性判断**：上述场景在5-10年内发生概率极低（<5%）。化肥为绝对刚需，钠离子/固态电池短期内无法全面替代LFP。更现实的"半黑天鹅"是硫磺价格长期维持5,000+元/吨+磷矿新增产能超预期释放→双杀利润和估值。
