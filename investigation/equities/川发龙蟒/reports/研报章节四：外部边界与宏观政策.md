# 研报章节四：外部边界与宏观政策

> **研究日期**：2026年05月18日
> **研究对象**：川发龙蟒（002312.SZ）
> **逻辑继承**：模块二确认了磷化工主业（工铵+DCP）处于景气上行周期，但原材料硫磺暴涨侵蚀利润；模块三确认了公司在这两个领域是行业龙头但新能源赛道处于边缘地位。本模块分析影响这一格局的外部宏观政策变量——哪些是在加速行业整合的"推力"，哪些是可能颠覆现有逻辑的"黑天鹅"。

---

## 一、地缘政治：硫磺的"霍尔木兹时刻"——公司面临的最大成本变量

### 1.1 事件背景

2026年2月28日，美伊冲突升级，伊朗宣布关闭霍尔木兹海峡。这是该全球能源与化工运输核心通道首次遭遇实质性停航，至今（截至5月18日）仍未恢复。

### 1.2 硫磺供给冲击的传导链条

**全球维度**：
- 全球海运硫磺贸易中，**50%的货物（约2000万吨/年）**源自中东波斯湾地区，必须经霍尔木兹海峡
- 中东硫磺核心出口港——阿联酋Ruwais、沙特Jubail/Ras al-Khair、卡塔尔Ras Laffan、科威特Al Zour、伊朗Bandar Imam Khomeini——**全部在封锁区内**
- 绕行替代路线（沙特红海港口、阿联酋富查伊拉港）面临陆运成本高昂和运力有限问题

**中国维度**：
- 中国硫磺对外依存度长期维持在50-53%
- 2025年中国硫磺进口中，**中东地区占比56.2%**，面临直接断供风险
- 国内主要港口（镇江港、南京港、防城港）库存持续下降

**价格冲击**：

| 时间节点 | 硫磺价格（镇江港） | 变化 |
| :--- | :--- | :--- |
| 2025年6月 | ~2010元/吨 | 基准 |
| 2026年1月 | ~4050元/吨 | +101% |
| 2026年3月 | ~5000元/吨 | +149% |
| 2026年5月6日 | **7300元/吨（历史新高）** | **+263%** |
| 2026年5月11日 | 7250元/吨 | 高位震荡 |

### 1.3 对川发龙蟒的量化影响

硫磺是磷化工生产的核心原料（用于制硫酸，进而生产湿法磷酸）。根据行业测算：
- 硫磺价格每上涨100元/吨，磷肥生产成本增加约45元/吨
- 以川发龙蟒2025年硫磺消耗量估算（工铵38.67万吨 + 肥料199万吨 + DCP 57.45万吨等），年硫磺消耗量约80-100万吨
- 硫磺价格从2000元涨至7300元（+5300元/吨），公司年度硫磺成本增量约**42-53亿元**（按80-100万吨消耗量×5300元/吨增幅估算）

⚠️ **但需注意**：川发龙蟒正在建设自有的硫磺制酸装置（德阿50万吨/年+攀枝花30万吨/年=80万吨/年硫酸产能），这部分产能可以使用自产和外购硫磺。实际影响取决于：
1. 公司硫磺自给率（进口 vs 国产 vs 自产比例）
2. 硫磺成本向产品价格的传导能力（工铵已涨价26%，DCP已涨价60%，说明传导部分有效）
3. 公司硫磺库存水平和长协采购价格

**核心结论：硫磺的地缘政治风险是川发龙蟒当前面临的最大外部变量。如果霍尔木兹海峡封锁持续至2027年，全球硫磺供应链将迎来深刻重构，拥有硫资源自给能力（如硫铁矿制酸、硫精砂制酸）的企业将获得巨大的相对成本优势。**

川发龙蟒在攀枝花和德阿配套了硫磺制酸装置，但硫磺原料仍需外部采购。公司是否具备硫铁矿或硫精砂制酸的替代路径，是需要进一步调研的关键问题。天宝公司具备50万吨/年硫精砂制酸产能——这是一个重要亮点，意味着天宝基地在硫磺断供时具备替代方案。

## 二、产业政策：加速行业分化的制度力量

### 2.1 磷矿资源管控——利好龙头企业

**政策框架**：
- 商务部2025年继续暂停磷矿石出口配额管理，实行出口许可证管理（2024年10月公告）
- 《推进磷资源高效高值利用实施方案》：严格有序投放磷矿矿业权，支持优强企业整合中小磷矿
- 到2026年目标：形成3家具有全球竞争力的一流磷化工企业

**影响**：磷矿开发门槛持续提高，无自有矿的中小企业面临原料成本上升和供应不稳定。川发龙蟒作为拥有231万吨/年自有磷矿的龙头，将直接受益于这一政策方向。

### 2.2 "以渣定产"与绿色转型——加速低效产能出清

**政策框架**：
- 到2026年，磷石膏综合利用率达到65%，综合消纳量与产生量实现动态平衡
- 《固体废弃物综合治理行动计划》（国发〔2025〕14号）：到2027年，云贵川鄂皖等地完成存量磷石膏库整治
- 鼓励有条件地区推行"以渣定产"，产生磷石膏的重点企业"一企一策"

**影响**：
- 磷石膏处理成本将成为企业生产成本的重要组成部分
- 中小企业因无力承担治废成本而加速退出
- 川发龙蟒拥有1座国家级绿色矿山、1座省级绿色矿山、3家绿色工厂，在磷石膏综合利用方面已有技术储备
- **利好头部企业，但增加了全行业的合规成本**

### 2.3 化肥保供稳价政策——限制价格弹性

**政策框架**：
- 发改委《关于做好2026年春耕及全年化肥保供稳价工作的通知》：
  - 督促企业落实最低生产计划
  - 磷肥行业倡议2026年8月前不安排出口（非强制禁令，但反映了政策导向）
  - 出口配额：磷酸二铵500万吨、磷酸一铵200万吨（总量控制）
  - 鼓励国产硫磺优先直接供应国内磷肥生产企业

**影响**：
- 磷肥出口受限，国内实施"保供稳价" → 肥料系列产品价格难以大幅上涨
- 川发龙蟒的肥料业务（41.67亿营收，11.83%毛利率）价格弹性被制度性压制
- **但工铵（新能源用途）和DCP（饲料添加）不在保供稳价范围内，不受此限制**

### 2.4 新能源材料产业政策——出口管制与技术壁垒

**锂电出口退税取消**：
- 2026年4月1日起：电池产品出口退税率从9%→6%
- 2027年1月1日起：完全取消电池产品出口退税
- 加速低端产能出清，倒逼企业从价格战转向技术竞争

**磷酸铁锂出口管制（2025年58号公告）**：
- 压实密度≥2.5g/cm³且克容量≥156mAh/g的磷酸铁锂正极材料纳入两用物项出口管制
- **这反而构成对已掌握高压实技术企业的保护壁垒**

**对川发龙蟒的影响**：公司的磷酸铁锂产品（与富临精工合作的高压密路线）若达到出口管制标准，反而因管制壁垒减少了国际竞争者。但若产品未达标准，则享受不到这一政策红利。

### 2.5 新能源汽车补贴退坡风险

- 2025年"以旧换新"补贴申请突破1000万份，新能源汽车占比57.2%
- 2026年补贴存在退坡可能 → 新能源汽车销量增速可能从28%放缓至15%
- 但储能需求爆发（+75%）可对冲动力电池增速放缓

## 三、宏观环境变量

### 3.1 利率环境对估值的影响

当前中国处于低利率环境（10年期国债收益率约2.5-2.8%），理论上有利于高PE股票估值。但川发龙蟒PE TTM约46x，属于中高估值区间，对利率变化敏感。

若美联储维持高利率（应对中东冲突推升的通胀），中国央行可能跟随收紧货币 → 高PE股票首当其冲面临估值压缩。

### 3.2 汇率损益

公司2025年海外收入15.85亿（占比15.92%），同比增长63.16%。人民币若因中美贸易谈判缓和而升值，将对出口竞争力产生一定压力。但考虑DCP出口以东南亚、非洲等非美元区为主，影响相对有限。

### 3.3 通胀传导与成本压力

硫磺价格暴涨已引发全行业成本冲击。磷肥行业因保供稳价政策，价格波幅有限 → 成本压力无法完全传导。但工铵和DCP价格弹性较好，部分传导了成本。

从中长期看，全球能源转型导致的"化石燃料副产品（硫磺）供应下降"与"新能源需求增长"之间的结构性矛盾将长期存在——**硫磺可能成为一种新的"稀缺资源"**。

## 五、模块四小结：外部环境的核心矛盾

川发龙蟒面临的外部环境呈现出"**利好长期逻辑，但加剧短期痛苦**"的特征：

**利好长期逻辑的变量**：
1. 磷矿资源管控强化 → 存量矿权价值重估 → 公司磷矿自给的护城河持续加深
2. 环保政策收紧 → 中小竞争者退出 → 工铵和DCP行业集中度提升 → 龙头享有更大定价权
3. LFP出口管制 → 高端产品产生壁垒 → 若公司能突破高压实技术，将受益

**加剧短期痛苦的变量**：
1. **硫磺价格因中东冲突飙升至历史新高（7300元/吨）** → 这是当前公司面临的最大和最紧迫的成本冲击
2. 化肥保供稳价政策限制价格传导 → 肥料业务毛利率可能进一步压缩
3. 新能源补贴退坡 → 终端需求增速放缓

**最核心的外部变量判断：硫磺价格的走向将决定公司2026-2027年的利润底色。**

霍尔木兹海峡封锁何时解除，是全球硫磺价格走向的决定性变量。当前市场博弈的是：封锁持续至年底 vs 短期谈判缓和。如果封锁持续超过6个月，部分磷化工企业的硫磺库存将耗尽，装置可能被迫降负甚至停产——届时，自有硫资源的企业将获得巨大的成本优势和市场份额扩大的机遇。
