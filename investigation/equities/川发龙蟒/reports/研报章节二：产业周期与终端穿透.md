# 研报章节二：产业周期与终端穿透

> **研究日期**：2026年05月18日
> **研究对象**：川发龙蟒（002312.SZ）
> **逻辑继承**：模块一揭示公司面临"营收扩张、利润坍塌"的结构性撕裂——毛利率从20%+坠落至12-13%，新增业务严重稀释盈利能力。本模块追溯这一撕裂的行业根源：是周期性的（等待反转）还是结构性的（永久损害）？

---

## 一、前序模块核心结论回顾（逻辑继承）

在开启模块二前，显式确认模块一的关键"红线"：

1. **毛利率持续下行**：2021年21.53% → 2026Q1 12.24%，创多年新低
2. **低毛利业务主导营收结构**：肥料系列（11.83%）+ 饲料钙盐（11.79%）合计占营收61%，高毛利工业级磷酸一铵（24.33%）仅占21%
3. **新能源材料板块大概率亏损**：2.76亿营收，毛利率未披露，从产品结构推断拉低整体盈利
4. **经营现金流2026Q1首现负值（-1.62亿）**：极度危险的信号
5. **巨额债务投入新能源产能**：三年投资现金流出45亿，但利润贡献近乎为零

**模块二的使命**：判断这些问题是"黎明前的黑暗"还是"结构性的黄昏"。

## 二、公司终端产品全景画像

公司产品最终流向下游终端形态：

| 公司产品 | 下游环节 | 最终终端 | 需求属性 |
| :--- | :--- | :--- | :--- |
| 工业级磷酸一铵 | 磷酸铁 → 磷酸铁锂正极 | **新能源汽车 + 储能电站** | 高增长弹性 |
| 工业级磷酸一铵 | 直接施用/水溶肥 | **农业种植** | 刚性稳定 |
| 肥料系列产品 | 复合肥 → 农户施用 | **农业种植** | 刚性稳定 |
| 饲料级磷酸氢钙 | 饲料添加剂 → 养殖业 | **猪肉/禽肉/水产品消费** | 温和增长 |
| 磷酸铁/磷酸铁锂 | 电池正极 → 动力电池/储能电池 | **新能源汽车 + 储能电站** | 高增长弹性 |
| 副产衍生品（铁精粉/石膏等） | 建材/冶金 | **基建/房地产** | 周期波动 |

**核心结论**：公司营收约60%锚定农业（刚性需求），约30%锚定新能源（高增长弹性），约10%为贸易与其他。

## 三、磷化工主业的周期定位

### 3.1 工业级磷酸一铵（工铵）：供给约束下的紧平衡

**产能格局**：
- 全国工铵产能超380万吨/年（2025H1），较2020年增长43.5%
- CR10达72.49%（远高于农业级磷酸一铵的47.29%），属于中高集中度行业
- 川发龙蟒工铵产能约38.67万吨/年（产量口径），全球产销量领先，产能占比约10%

**价格走势**：
- 2025年10月：73%工铵湖北出厂5736元/吨，四川5839元/吨
- 2026年5月：湖北73%工铵7200-7300元/吨，电池级报价7500元/吨
- **6个月内涨幅约26%**，驱动因素是硫磺暴涨推动成本上行

**需求拆分**：
- 2025年上半年工铵市场规模83.58亿元（含新能源+水溶肥）
- 电池领域（磷酸铁前驱体）：2025H1规模约39.32亿元，占比47%
- 水溶肥领域：2025H1规模约36.70亿元，占比44%
- 2024年全年工铵需求量153万吨，2025H1已达134万吨（全年估算约280万吨）

**供需平衡判断**：
- 需求增速：新能源拉动约+30-40% YoY（磷酸铁产量增速），农业基本盘稳定
- 供给增速：产能增长但受制于磷矿审批收紧和环保约束，实际产量增长慢于产能
- **结论**：工铵处于**紧平衡偏紧缺**状态，供给约束强于需求增长，价格中枢持续上移

### 3.2 磷矿石：战略性稀缺资源，持续强化定价权

- 湖北28%品位磷矿船板价980-1000元/吨（2025-2026年基本稳定），处于历史高位
- 磷矿审批被严格限制（"三磷"整治），新矿权获取困难
- 川发龙蟒自有磷矿231万吨/年，另参股天盛矿业（在建550万吨/年），资源自给率高
- **磷矿的稀缺性决定了产业链利润向上游矿端集中**，自有矿企业享有持续成本优势

### 3.3 肥料系列：存量博弈，价格受困于成本

- 55%粉状磷酸一铵（农铵）：湖北4200-4250元/吨（2026年5月），保供企业3850元/吨
- 磷肥行业开工率已低于50%，企业受困于高硫磺成本与弱需求之间的挤压
- 国内磷肥产能约1800万吨/年（农铵），显著过剩
- **结论**：肥料业务处于**成熟期存量博弈**，利润空间受上游成本（硫磺）和下游需求（粮价/农户购买力）双向挤压，毛利率11.8%基本反映行业平均利润水平，难以大幅改善

## 四、饲料级磷酸氢钙（DCP）的周期定位

### 4.1 供给端：环保约束下的持续收缩

- 行业有效产能以每年低个位数递减（"两高"项目审批收紧+长江经济带限排）
- 2025年川发龙蟒通过收购天宝公司跃居行业第一（南漳32.15万吨+天宝25.30万吨=57.45万吨/年）
- 中小厂商被环保成本淘汰，行业集中度快速提升（前10大企业>45%份额）

### 4.2 需求端：养殖业温和增长提供稳定支撑

- 生猪养殖规模化率超65%，对DCP需求刚性
- 水产饲料增长（全球年均+5.8%）拉动MCP/DCP需求
- 2025年DCP出口26.14万吨，同比+35.66%
- 东南亚、非洲新兴市场需求旺盛（进口依赖度83%）

### 4.3 价格与盈利判断

- 2025年6月DCP报价2980-3250元/吨 → 2026年3月4950-5250元/吨，**涨幅达60%**
- 驱动因素：硫磺暴涨（原料成本）+供给收缩+出口增长
- 川发龙蟒DCP毛利率11.79%（2025年全年），但价格涨幅在2025H2/Q4才开始体现，2026年有望受益

**结论**：DCP行业处于**供给收缩叠加需求温和增长的"紧平衡"阶段**，龙头企业凭借规模+矿化一体优势享有超额利润。但需警惕硫磺成本上涨对毛利率的侵蚀。

## 五、新能源材料板块的周期定位

### 5.1 磷酸铁锂（LFP）行业全景

这是对川发龙蟒最关键的板块——公司过去三年投资45亿+的核心去向。

**产能与产量（极致分化）**：

| 指标 | 2025年 | 2026年E |
| :--- | :--- | :--- |
| 中国名义产能 | 647.3万吨 | 950万吨 |
| 中国实际产量 | 350万吨 | 560-580万吨 |
| 产能利用率 | ~54% | ~60% |
| 全球出货量 | 394.4万吨 | 560-580万吨 |
| 出货增速 | +62.5% | +~55% |

**核心矛盾：高端紧缺、低端过剩**

- **高端（四代高压实密度 ≥2.6g/cm³）**：仅5-6家企业能量产，2026年需求150万吨 vs 供应100-120万吨，**供需偏紧**
- **低端（三代及以下）**：产能严重过剩，价格战惨烈，大量企业亏损
- 行业供应格局：CR5 >70%，头部企业产能利用率90%+且满产超销，中小企业开工率低于60%

**价格与利润**：
- 磷酸铁锂均价：2025年6月最低3.0万元/吨 → 2026年1月6万元/吨（高端），反弹明显
- 但行业全产业链已连续亏损超36个月（截至2025年底）
- 头部企业负债率攀升至70%左右
- 毛利率从2022年的39.57%降至2025H1的26.92%（新增订单口径，中小企业更低）

**2026年展望**：
- 新增产能超410万吨规划，但实际落地预计150万吨
- 储能市场需求爆发（+75%增速）成为最强增长引擎
- 磷酸铁锂电池装机占比81.2%（2025年1-11月），渗透率持续提升
- 行业正从"以价换量"向"量价齐升"过渡，但仅限于高端产品

### 5.2 川发龙蟒在LFP产业中的位置评估

| 维度 | 川发龙蟒现状 | 行业头部对比 |
| :--- | :--- | :--- |
| 磷酸铁锂产能 | 6万吨/年（德阿） | 湖南裕能150万吨、容百科技规划100万吨 |
| 磷酸铁产能 | 15万吨/年（攀枝花5+德阿10） | 行业总有效产能超470万吨 |
| 产能排名 | **不在行业TOP10** | 头部企业出货量均超30万吨 |
| 技术代次 | 未披露，推断以三代为主 | 头部已量产四代（≥2.6g/cm³） |
| 客户结构 | 未披露核心客户长单 | 湖南裕能深度绑定宁德时代，富临精工获预付款 |
| 2025年实际收入 | 新能源产品仅2.76亿 | 湖南裕能出货>100万吨，营收>300亿 |
| 一体化程度 | 矿-酸-铁-锂全链，但量小 | 头部同样布局一体化，且规模碾压 |

**核心判断：川发龙蟒在LFP行业处于边缘位置，规模差距悬殊。**

6万吨磷酸铁锂 vs 头部企业100-150万吨——这不是同一重量级的竞争。如果公司生产的是低端/普通产品，在当前行业环境下大概率处于亏损状态。

10万吨/年磷酸二氢锂在建项目（预计2026下半年建成）将进一步增加产能，但若无法突破高端技术认证或锁定大客户长单，新增产能可能只是"投产即亏损"。

### 5.3 磷酸铁板块

- 攀枝花5万吨/年磷酸铁已量产量销
- 德阿10万吨/年磷酸铁顺利试生产
- 合计15万吨/年，在全国470+万吨总产能中占比仅3.2%
- 磷酸铁行业同样处于低端过剩状态，毛利率极低

## 六、终端需求穿透审计

### 6.1 新能源汽车终端（工铵/LFP的最终出口）

| 指标 | 2024 | 2025 | 2026E |
| :--- | :--- | :--- | :--- |
| 中国新能源车销量 | 1287万辆 | 1649万辆 | 1900万辆 |
| 增速 | — | +28.2% | +15.2% |
| 新能源渗透率 | ~45% | ~52% | ~58% |
| 磷酸铁锂装机占比 | 74.6% | 81.2% | ~83% |

**趋势判断**：2026年中国新能源车销量仍保持15%+增速，磷酸铁锂路线占比持续提升。终端需求增长确定性高，但增速从30%档放缓至15%档，边际斜率在降低。

### 6.2 储能终端（磷酸铁锂的第二增长极）

| 指标 | 2024 | 2025 | 2026E |
| :--- | :--- | :--- | :--- |
| 中国储能锂电池出货量 | ~330GWh | ~580GWh | >800GWh |
| 增速 | — | +75% | +40%+ |
| 磷酸铁锂占比 | ~99% | ~99% | ~99% |

**储能是磷酸铁锂需求增长的最强引擎**，增速远超动力电池。受益于：
- 国内新型电力系统建设政策（136号文推动市场化配储）
- 海外AI数据中心配储刚性需求
- 全球可再生能源装机快速增长

### 6.3 农业终端（肥料 + 饲料钙盐）

**化肥需求**：国内化肥需求整体呈减量趋势（"减肥增效"政策），但水溶肥、特种肥等高端品类需求增长。工业级磷酸一铵在水溶肥领域的应用持续渗透。

**饲料需求**：
- 中国生猪存栏稳定在较高水平（规模化率提升）
- 水产养殖持续增长（全球+5.8%/年）
- 禽类养殖稳定
- 磷酸氢钙出口市场强劲（+35.66% YoY）

## 七、强制性复核项确认（Mandatory Audit Checks）

### 复核一：是否穿透并引用了最终终端产品的未来两年景气度预测？

**确认**：
- ✅ 新能源汽车2026年销量1900万辆（+15.2%），增速放缓但绝对值仍大
- ✅ 储能2025年锂电出货580GWh（+75%），2026年预计800GWh+，超级景气
- ✅ 农业刚性需求稳定，饲料钙盐出口强劲
- ✅ 磷矿资源战略性稀缺，价格高位持续

### 复核二：是否审计了跨期需求移位风险？

**确认**：
- ⚠️ 新能源汽车存在"以旧换新补贴"刺激效应——2025年补贴申请突破1000万份，新能源车占比57.2%。2026年补贴可能退坡，存在部分需求前置风险
- ⚠️ 储能市场从"强制配储"转向"市场化驱动"（136号文），短期装机可能因政策切换而波动，但长期逻辑更强
- ✅ 饲料钙盐不存在明显政策刺激，需求由养殖基本面驱动

### 复核三：是否分析了供需边际演变及压力传导特征？

**确认**：
- ✅ 工铵：供给约束（磷矿审批+环保）>需求增长 → 紧平衡偏紧缺，价格向上
- ✅ DCP：供给收缩（环保淘汰）>需求增长 → 供需缺口逐年拉大，利好龙头
- ✅ LFP：供给严重过剩（低端）但高端紧缺 → 结构性分化，川发龙蟒处于不利位置
- ✅ 压力传导：硫磺成本暴涨正在通过硫酸→磷酸→磷酸盐链条向全行业传导，自有硫资源的企业受益，川发龙蟒对外购硫磺的依赖度需进一步调研

## 八、模块二小结：周期定性与核心矛盾升级

### 周期定位矩阵

| 业务板块 | 周期位置 | 供需方向 | 价格趋势 | 对川发龙蟒的影响 |
| :--- | :--- | :--- | :--- | :--- |
| 工业级磷酸一铵 | 成长后期/成熟期 | 紧平衡偏紧 | ↑ 上行 | **利好**，自有磷矿享成本优势 |
| 肥料系列 | 成熟期 | 过剩 | → 承压 | 中性偏负，成本传导不畅 |
| 饲料级磷酸氢钙 | 成熟期（供给收缩） | 供需缺口拉大 | ↑ 强势上行 | **利好**，行业第一享定价权 |
| 新能源材料（LFP/FP） | 成长期（野蛮扩张→洗牌） | 严重过剩（低端） | ↓ 承压（低端） | **显著利空**，规模太小难以立足 |

### 核心矛盾升级

模块一发现的"营收扩张、利润坍塌"矛盾，在模块二中获得了更深层的解释：

1. **高毛利业务（工铵）享受行业红利**：工铵价格6个月涨26%，但该业务仅占总营收21%，不足以扭转整体颓势
2. **低毛利业务（肥料61%占比）困于成本挤压**：硫磺暴涨推高成本但无法完全传导至下游
3. **新能源材料业务遭遇行业逆风**：15万吨磷酸铁+6万吨磷酸铁锂在行业470+万吨/650万吨的产能面前微不足道，且大概率产出的是低端产品，在"高端紧缺、低端过剩"的格局下，**这部分业务可能正在持续亏损，并消耗传统磷化工的现金流**
4. **"第二增长曲线"变成了"第二出血点"**：公司过去三年投入45亿+建设新能源产能，但该板块2025年仅贡献2.76亿营收，利润贡献大概率负值。在建的10万吨磷酸二氢锂若无法突破技术壁垒，可能加剧这一困局

### 关键待验证问题（带入模块三）

1. 川发龙蟒的磷酸铁/磷酸铁锂产品在行业中属于哪个技术代次？是否进入了头部客户的供应链？
2. 公司硫磺采购成本与行业竞争对手的差距？是否具有硫酸自给能力？
3. 新能源材料业务的毛利率/净利润究竟是多少？能否在2026-2027年实现盈亏平衡？
