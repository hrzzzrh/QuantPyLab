# 湖南裕能 (301358.SZ) 投资逻辑跟踪手册

**版本**：v1.0 (2026-04-29)
**研究锚点**：2026年核心盈利预测 57.2 亿元（剔除库存收益），目标价 122.1-134.3 元。

## 1. 核心逻辑验证点 (Thesis Validators)
*建议在每季度财报发布后第一时间核对：*

1.  **经营性现金流 (OCF) 是否转正/改善 (优先级：最高)**：监控 OCF 与净利润的背离情况。若利润持续高增而现金流持续为负，估值将面临系统性下修。
2.  **核心单吨净利是否维持在 0.3 万元以上**：通过 `(扣非净利 - 测算库存收益) / 销量` 计算，验证一体化降本的真实持续性。
3.  **2027 年新增产能排产调研**：重点关注二线厂商在 2026 下半年的扩产意愿，若扩产超预期，2027 产能悬崖风险将提前爆发。
4.  **海外出口订单的利润率变化**：监控出口退税下调 6% 后，公司能否通过提价转嫁成本。

## 2. 关键外部变量监控 (Exogenous Watchlist)
1.  **碳酸锂价格斜率**：关注价格从“单边反弹”向“高位震荡”切换的节点，此时库存收益将迅速衰减。
2.  **行业平均加工费**：若行业平均加工费跌破 1.5 万元/吨，即使是一体化龙头，盈利空间也将被压缩至极致。
3.  **欧盟电池护照与碳关税执行细则**：这是海外逻辑能否兑现的关键。

## 3. 逻辑证伪警报 (Invalidation Triggers)
**若出现以下信号，必须承认投资逻辑已死，直接触发减仓或止损：**

*   **警报 A**：经营性现金流净额连续四季度为负且缺口持续扩大。
*   **警报 B**：核心单吨净利跌破 0.2 万元，意味着一体化成本优势已被行业价格战完全对冲。
*   **警报 C**：2027 年行业总产能利用率预测跌破 50%，预示着毁灭性的产能出清即将来临。
*   **警报 D**：海外工厂因碳排放审计不达标被拒绝进入欧洲主流车企供应链。

## 4. 关键时间节点 (2026 Calendar)
*   **2026年08月下旬**：2026年半年度报告发布（关注单吨净利稳定性）。
*   **2026年10月下旬**：2026年三季度报告发布（关注储能旺季交付量）。
*   **2027年03-04月**：2026年年度报告发布（逻辑最终兑现期）。
