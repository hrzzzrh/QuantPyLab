## 一、美国市场：FEOC限制构成"事实禁入"，但储能侧留有缝隙

### 1.1 从IRA到OBBBA——政策环境的急转直下

2025年7月4日，《One Big Beautiful Bill Act》(OBBBA)签署生效，彻底重塑了美国清洁能源税收优惠格局：

| 维度 | IRA时期 (2022-2025) | OBBBA新规 (2025.7.4起) |
| :--- | :--- | :--- |
| EV购买税收抵免 | $7,500/辆 (含电池组件限制) | **直接取消** |
| 储能ITC | 30%投资税收抵免 | 保留（**利好储能**），但新增PFE限制 |
| 45X先进制造补贴 | $35/kWh电芯+$10/kWh模组 | 保留但FEOC限制大幅收紧 |
| 外国关注实体(FEOC) | 中国公司电池/矿物限制 | 升级为"禁止外国实体"(PFE)六个类别 |
| 关键矿物关税 | 第301条款25% | 第232条款调查进行中(锂/钴/石墨) |

### 1.2 FEOC/PFE六类限制对CATL的影响评估

| 限制类别 | 触发条件 | 对CATL的影响 | 严重度 |
| :--- | :--- | :--- | :--- |
| 所有权限制 | 被覆盖国(中/俄/朝/伊)政府持股≥25% | 可能触发——CATL有一定国有持股背景 | ★★★☆☆ |
| 董事会控制 | 覆盖国公民在董事会中占主导 | 低风险——CATL为民营企业 | ★☆☆☆☆ |
| 债务控制 | 覆盖国实体持有≥15%债务 | 低风险 | ★☆☆☆☆ |
| 许可证/合资控制 | 覆盖国实体通过协议实施控制 | CATL技术授权模式需谨慎设计 | ★★★☆☆ |
| **有效控制** | 实质性运营和功能控制 | **核心关切**——IRS解释仍在演进中 | ★★★★☆ |
| **物质援助** | 电池PFE组件成本占比超阈值 | **储能市场的主要风险** | ★★★★★ |

### 1.3 福特密歇根工厂投产：技术授权模式的里程碑验证（重大新增，06.30）

2026年6月，CATL与福特汽车以技术授权服务模式合作建设的密歇根州Marshall电池工厂（BlueOval Battery Park Michigan）正式建成投产。该工厂具有里程碑意义：

| 维度 | 详情 |
| :--- | :--- |
| **合作模式** | **技术授权+服务**：福特全资持有工厂（$2B投资），CATL提供LFP电池技术授权及工厂建设运营服务。非合资，CATL无股权。 |
| **产能与产品** | 初始20GWh（受政治压力从35GWh缩减），部分产能转向储能电池产品。2026年计划交付首批车用动力电池。 |
| **政治历程** | 2023年2月宣布（$3.5B/35GWh）→遭共和党议员质疑→2023年11月重启（缩减至$2B/20GWh）→2026年1月再次遭储能业务质疑→依然推进投产。历经三年政治拉锯后最终落地。 |
| **客户车型** | 首批搭载于福特经济型及中型电动皮卡（基于新一代UEV平台，2027年量产） |
| **就业规模** | 当前500+人，年底800人，最终1,700人。已收到11,500+份申请。 |
| **战略意义** | CATL成为首家通过技术授权模式将电池制造落地美国的中资企业，验证了"技术输出+本地生产"模式的可行性 |

**对CATL的地缘政治评估的重大修正**：

1. **技术授权模式可行性已获实证**：此前市场（包括本报告）对"技术授权能否真正落地"持怀疑态度。福特工厂从宣布到投产历时三年有余，期间经历投资缩减、共和党质疑、FEOC审查等多重障碍，但最终成功投产。这意味着：
   - CATL在美国市场的**技术入口**已被打开——LFP电池技术是美国本土供应链的刚需短板
   - 该模式可复制——CATL已与通用汽车（GM）就类似技术授权展开谈判（2024年3月报道），未来可能扩展到更多美国车企
   - 储能电池生产落地美国更具政策优势——OBBBA保留了储能ITC（30%），福特已将部分产能转向储能

2. **"美国市场实质关闭"的判断需要修正**：此前本报告判断"美国动力电池市场对CATL已实质关闭"。福特工厂的投产表明——**通过技术授权模式，CATL可以进入美国市场**。虽然CATL不直接持有工厂股权、不分走制造利润，但通过技术授权费+运营服务费，仍能获得持续性收入。更重要的是，这为CATL在北美电池供应链中确立了不可替代的技术地位。

3. **风险提示**：
   - 技术授权模式仍面临政治不确定性——若中美关系进一步恶化，FEOC/PFE规则可能收紧至涵盖技术授权
   - 该模式的经济效益远低于自有工厂——CATL仅收取技术授权费（通常为营收的2-5%），而非全链条利润
   - 福特自身财务状况不佳（2025-2026年持续重组），可能影响产能爬坡节奏

### 1.4 美国储能市场的"缝隙"

尽管动力电池通过传统出口进入美国市场近乎不可能，但**储能侧出现了结构性机遇**：

- **政策不对称**：OBBBA取消了EV税收抵免但**保留了储能ITC**（30%），使得储能成为政策避风港
- **本土产能缺位**：美国2026年FEOC合规的电芯产能由韩国供应商主导（LG/SK/Samsung），但实际产能利用率仅70-80%。Wood Mackenzie估计2026年美国本土合规产能约49GWh，存在约10%的盈余
- **制造转移效应**：福特已将部分产能转向储能电池生产（基于CATL技术），此外AESC等也正在转向储能
- **CATL的应对路径**：
  - **技术授权模式**（已实证可行）——与福特合作模式可复制到储能领域
  - **通过海外子公司供货**——如果CATL能证明其匈牙利工厂不使用中国受控供应链，可能获得PFE豁免
  - **间接供货**——通过系统集成商(如Fluence等)曲线进入

**核心判断（修正）**：美国动力电池市场通过传统出口模式已实质关闭，但**技术授权模式提供了新的进入路径**。美国储能市场（2025年装机约49GWh，2026年预计60GWh+）仍有一定可触及性，且福特工厂的储能产能转向进一步验证了这一路径。

## 二、欧洲市场：关税可控，但"本地化"要求日益紧迫

### 2.1 反补贴关税——短期可消化

2024年10月，欧盟对中国产纯电动汽车征收7.8%-35.3%的反补贴税。但该关税主要针对**整车**而非电池。对CATL的直接影响有限：

- CATL在欧洲的主要客户（VW、BMW、Stellantis、Volvo）大多在欧洲本地设厂，电池由CATL的德国/匈牙利工厂供应
- 2026年1月，欧盟发布价格承诺指南，允许中国车企通过提高出口价格替代关税——这一"谈判换关税"的框架为CATL的欧洲业务保留了政策空间

### 2.2 《净零工业法案》(NZIA) + 欧盟电池法规——长期约束

**关键技术要求**：

| 政策 | 生效时间 | 核心条款 | 对CATL的影响 |
| :--- | :--- | :--- | :--- |
| NZIA第28条 | 2026年1月起 | EV补贴方案须要求电池包来自**多元化供应源**，减少对中国依赖 | 匈牙利工厂成为合规关键 |
| 欧盟电池法规 | 2024年2月起分阶段 | 碳足迹声明(2025)、碳足迹上限(2028)、回收含量要求(2031) | CATL需建立完整碳足迹追踪体系 |
| **电池助推器计划**(2025.12) | 2026年推进 | 六大支柱：投资/上游/外资对齐/需求激励/研发/协调 | 可能新增EU电池含量要求 |
| 外国补贴条例(FSR) | 2023年起 | 对获得非欧盟政府补贴的企业在欧盟投资进行审查 | CATL匈牙利项目可能受审查 |

**CATL的应对策略**：
- **匈牙利德布勒森工厂（投资73亿欧元，规划100GWh）**：模块组装产线于2026年5月6日正式启动（5GWh年产能，理论可配套5万辆大型或12.5万辆小型电动车用电芯）。但需注意：①目前生产的模组所用电池单体（cell）仍从CATL其他工厂调运，**德布勒森本地的电芯生产尚未启动**；②电芯工厂厂房已建设完毕，设备也已安装，但仍在等待最终审批许可，试产延后（此前预期2026年3-4月启动）。电芯本地化生产的时间窗口从此前预期推迟了3-6个月。③相较此前预期的"2026年初投产"的乐观叙事，当前进度面临客观的审批延迟，对2026年欧洲业务的"本地化生产"叙事构成轻微负面影响——但模块组装（即使电芯外部调运）仍意味着部分价值在欧洲本地转化。④CATL自2024年秋季起已利用德布勒森租赁厂房生产约24万个电池模块（可配套约6万辆电动车），说明本地化并非从零开始。
- 与Stellantis合资西班牙工厂（50GWh）筹建中，预计2026年底试生产
- 德国图林根工厂（14GWh）已于2024年实现盈利，运营稳定
- H股上市募集的410亿港元中约90%用于匈牙利项目建设
- 在欧洲本地建厂既能规避关税、又能满足NZIA的"多元化供应"要求

**2026年新增重大政策合规事项**：
- **EU电池法规（2023/1542）2026年4月24日全面生效**：所有工业电池（≥2kWh）和动力电池须附带数字电池护照，包含经核查的碳足迹声明、回收成分和供应链尽职调查。不合规产品将被欧盟海关拒绝清关
- **2026年8月18日起**：所有电池须在产品和包装上标注化学成分、重量、容量、制造商、生产日期和有害物质（>0.1%重量的须披露）
- **2027年2月18日起**：EV电池和工业电池须通过QR码提供Digital Battery Passport
- CATL匈牙利工厂的本地化生产可有效规避上述法规的跨境合规成本（本土生产→本土销售）

### 2.3 匈牙利工厂排污事件：本地化叙事的首次合规危机（持续跟踪，06.30更新）

2026年6月5日，匈牙利哈伊杜-比豪尔州政府对CATL德布勒森工厂启动诉讼程序。此后事件持续发酵：

| 时间节点 | 事件 | 影响 |
| :--- | :--- | :--- |
| 2026.05.05 | 分包商违规将绿色液体排入市政下水道 | 污染事件发生 |
| 2026.06.05 | 州政府启动诉讼程序，水务部门**即时吊销工业废水预处理许可证** | CATL工厂丧失废水排放资质 |
| 2026.06.08 | 匈牙利今日报报道，蒂萨党政府区域与农村发展部长下令展开特别调查 | 事件上升至中央政治层面 |
| 2026.06.10 | CATL发表致歉声明，称染料无毒，已完成污染补救 | 公关危机管理启动 |
| **2026.06.15前后（新增）** | **独立检测声称发现NMP溶剂残留**，蒂萨党议员要求"零容忍"，德布勒森市长要求澄清检测矛盾 | **事件性质升级——从"操作违规"向"有害物质泄漏"叙事演变** |
| **2026.06中旬（新增）** | **CATL官方否认政府检测发现NMP**，称政府办公室未检出NMP，公司认为官方测量可信 | 形成"独立检测vs官方检测"的舆论拉锯 |
| **2026.06中旬（新增）** | **水务部门已处以100万福林（约2万人民币）罚款** | 罚款金额极小，但许可证恢复尚无时间表 |
| **截至06.30** | **特别调查仍在进行，废水许可证尚未恢复** | 事件尚未完全平息 |

**影响评估（06.30更新）**：

1. **合规风险持续但烈度已降低**：罚款仅100万福林（约2万人民币），象征性惩戒。CATL已完成污染补救。但废水许可证尚未恢复（已逾50天），这本身构成了运营限制——虽然模块组装产线（当前5GWh）对工业废水依赖有限，但电芯产线（建成待许可）若长期无废水许可证将无法启动。

2. **政治风险阶段性稳定**：事件初期（6月5-10日）的政治化冲击波已有减弱迹象——匈牙利中央政府未采取进一步升级措施（如停产令），蒂萨党政府的关注焦点已转向更广泛的"中资审查"议程（同时审查比亚迪）。这表明该事件并未演变为针对CATL的定向政治打击。

3. **NMP争议的两面性**：独立检测声称发现NMP溶剂残留是事件最敏感的转折点。但CATL坚称官方检测未发现NMP，德布勒森市政府正在组织比对验证。若官方检测最终确认无NMP，CATL的舆论被动有望反转；若确认有NMP，则罚款金额和合规成本将显著上升。

4. **对欧洲业务叙事的综合影响**：
   - **短期（1-3个月）**：ESG形象受损已确定，但罚款金额极低（100万福林）说明违规程度尚属轻微。许可证恢复时间表是短期核心变量。
   - **中期（3-12个月）**：取决于最终调查结论和许可证恢复进度。若能在Q3内恢复许可证且罚款维持小额，影响将局限于短期声誉冲击；若许可证延迟至年底+罚款加码，可能影响2027年电芯本地化生产时间表。
   - **长期**：此事件是CATL从"中国冠军"向"全球冠军"跨越过程中的**必要学费**。跨国运营中的本地合规管理（分包商监管、环境合规、政府关系）是企业出海必须补足的短板。从积极角度看，提前暴露问题有助于在更大规模扩产前完善管理体系。

5. **与福特工厂进展的对比启示**：福特密歇根工厂历经三年政治拉锯（投资缩减35GWh→20GWh，多次国会质疑）后最终投产，证明CATL在应对海外合规挑战方面具备韧性和变通能力。匈牙利事件应被视作类似性质的"成长痛"而非系统性危机。

欧洲生产成本比中国高约50%（McKinsey数据）。CATL匈牙利工厂"成本虽较国内稍高，但相比德国工厂有明显优势"（2026Q1业绩会）。随着产线优化和规模效应释放，欧洲工厂盈利能力有望逐步改善。

## 三、汇率与利率：对冲工具下的可控变量

### 3.1 汇率风险

CATL 2025年海外收入占30.6%（1,296亿元），主要结算货币为欧元和美元。2026Q1财务费用从-22.9亿转为+0.62亿，主要是"外币货币性项目受汇率变动影响产生汇兑损失"——人民币升值对CATL的海外收入构成一定压力。

**对冲工具**：
- H股上市募集港元可天然对冲部分汇率风险
- 海外工厂本地化生产→本地化销售，降低汇率错配
- 公司拥有成熟的套期保值操作（2025年公允价值变动收益9.7亿）
- 2026Q1汇率损失约2.3亿（估算），占总营收不到0.2%，影响可控

### 3.2 利率环境

2026年全球利率环境偏宽松：美联储可能在2026年下半年降息，欧央行持续宽松。对CATL的影响渠道：
- **利好**：低利率降低海外项目融资成本（匈牙利工厂部分资金来自H股IPO，无利息负担）
- **利空(间接)**：低利率可能推高资产价格和原材料价格
- **估值影响**：DCF估值中折现率每降低1个百分点，CATL的理论估值可提升约10-15%

**2026年锂价路径追踪（第三次修正，06.30）**：

自06.14上次报告以来，锂价经历了加速下行的走势：6月12日17.05万元/吨→6月26日盘中最低触及14.5万元/吨（碳酸锂主力合约单日暴跌超6%，15万整数关口被击穿），随后6月29日现货反弹至16.3-17.0万元/吨区间。整个6月下半月，锂价从月初17.8万元/吨降至月末约15.4万元/吨（生意社基准价），月内跌幅约13.5%。

驱动本轮加速下行的核心因素出现重大变化：

1. **枧下窝复产预期升温（重大变量）**：6月25日前后，枧下窝矿区用地预审意见书获批的消息在市场中发酵。虽然从用地预审到实际产出碳酸锂还需经历环评、安评、矿区基建、设备安装等流程（预期Q4甚至更晚），但市场已开始提前定价远期供给释放预期。与此同时，湖南临武鸡脚山锂矿（年产4万吨LCE）已于6月18日点火，澳洲泰利森三期、皮尔巴拉工厂等均有复产或扩产计划。

2. **仓单高压持续压制**：广期所碳酸锂仓单从6月10日的5.56万吨降至6月24日的4.91万吨（小幅去化），但仍处于历史极高水平。7月面临仓单集中注销压力，仓单如何消化将直接影响近月价格。

3. **津巴布韦矿7月到港预期**：此前二季度国内锂精矿到港量阶段性缩减，但津巴布韦已恢复配额式发运，海运货物将于7月集中抵达国内口岸。市场提前计价原料供给修复预期。

4. **6月供需基本面实际偏紧但市场定价远期宽松**：SMM数据显示碳酸锂已连续8周去库（截至6月25日当周库存9.64万吨，环减564吨），7月锂电排产环比+5%以上（连续环比增长）。当前基本面实际偏紧，但远期供给释放预期压制了现货价格反弹空间——这是典型的"强现实、弱预期"格局。

5. **6月26日单日暴跌6%的特殊背景**：除枧下窝复产传闻外，当日为A股季度末（630）机构调仓高峰，新能源板块整体大幅调整（创业板新能源ETF跌3.83%），锂矿股批量下跌带动期货恐慌性抛售。

### 4.2 通胀环境

全球通胀已从2022年高点回落，但关税战可能带来新的输入性通胀压力。CATL作为制造业龙头，在成本推动型通胀中通常具有一定的转嫁能力，但程度取决于下游需求弹性。

## 五、中国国内政策：从"扶持"到"规范"，格局利好龙头

### 5.1 产业政策基调转变

中国对锂电池产业的政策正从"大力扶持"转向"规范引导"：
- 2026年1月，工信部召见电池企业讨论**优化产能管理**，标志着官方开始关注行业过剩问题
- 取消"强制配储"政策（2025年1月）旨在推动储能从行政驱动转向市场驱动
- 2026-2027年新能源车购置税减半征收（从免征0%→征5%），补贴力度渐进退坡
- 以旧换新补贴从"固定金额"改为"比例补贴+上限"，低价车型实际补贴下降60-70%

### 5.2 政策调整对CATL的实际影响

**利好方面**：
- **行业出清加速**：规范产能管理将加速落后产能淘汰，CATL作为行业龙头将受益于集中度提升
- **"反内卷"效应**：2025年价格战车型减少42款，无序降价得到遏制，有利于维持合理利润
- **新型储能国家战略**：2025-2027年规划新增装机180GW，带动投资2,500亿元
- **零碳战略协同**：CATL的零碳园区建设与国家"双碳"目标高度契合，获得政府支持

**利空方面**：
- 补贴退坡短期压制汽车终端需求（2026年1月销量-28%）
- 但结构上，政策更利好高价值车型（补贴与车价挂钩，高价车补贴不减反增），CATL的高端电池配套反而受益

## 七、核心结论

1. **美国市场通过技术授权模式已被打开（重大修正）**：福特密歇根工厂6月投产标志着CATL技术授权模式在美国的首次成功落地。虽然经济效益远不及自有工厂，但战略意义深远——证明了CATL能够以"轻资产"方式进入全球最大的电池消费市场之一。后续关注GM等潜在技术授权合作对象。

2. **匈牙利工厂排污事件烈度趋于缓和但尚未结束**：罚款仅100万福林（约2万人民币），事件未导致生产暂停。但废水许可证恢复时间和NMP争议结论仍是短期核心变量。与福特工厂应对三年政治冲击的韧性对比，匈牙利事件应被视作"成长痛"而非系统性危机。

3. **碳酸锂价格在15-16万元/吨区间震荡下行**：自6月12日17.05万持续下跌至6月26日低点14.5万（-15%），月内跌幅约13.5%。枧下窝用地预审获批+7月津巴布韦矿到港+仓单4.9万吨高位构成三重压制。全年均价从此前15-17万元/吨下修至15-16万元/吨。这对CATL成本端构成**确定性正面推动**。

4. **钠电储能全球首发提供新的增长极验证**：6月22日天恒钠电储能系统发布，9月中国交付+2026年1GWh出货→2027年全球交付。钠电产业化时间表进一步清晰，对CATL估值中的"第二增长曲线"叙事有支撑。

5. **EU电池法规2026年4月全面生效**，数字护照+碳足迹声明成为对欧出口硬门槛。CATL匈牙利本地化生产（即使当前仅模块阶段）天然满足合规要求，但排污事件若被EU层面引用，可能影响合规认定。

6. **汇率和利率是可控变量**，不会成为利润的决定性因素。

7. **最大的无法量化的风险**：地缘政治黑天鹅(台海/中美全面脱钩)一旦发生，所有基本面分析都将失效。但福特工厂的实证表明，即使在极端政治压力下，技术授权模式仍具备生存韧性。
