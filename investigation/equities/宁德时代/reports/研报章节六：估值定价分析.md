**研究日期**：2026年05月24日  
**研究标的**：宁德时代（300750.SZ）  
**当前股价**：不复权市价 **411.16元** | 后复权 801.67元  
**较上次研究（05.11）**：-7.8%（446.11→411.16）  
**当前市值**：约**19,022亿元**（总股本45.64亿股）  
**PE TTM**：24.09倍 | PB：5.32倍  
**EPS 2025A**：16.14元/股 | **EPS 2026E(中性·修正)**：18.14元/股（↓原19.61，-7.5%）  
**修正触发因素**：碳酸锂价格从约8万元/吨飙升至约20万元/吨，触发毛利率假设从25.5%下修至24.0%。

## 一、价值锚定：适配性方法选择

### 1.1 方法论适配分析

| 方法 | 适配度 | 理由 |
| :--- | :--- | :--- |
| **PE(市盈率)** | ★★★★★ 首选 | 公司盈利稳定、可预测(ROE 24-25%)，全球对标清晰，是市场最认可的方法 |
| **PB-ROE** | ★★★★☆ 辅助 | ROE超20%支撑PB溢价，适合与PE交叉验证 |
| **PS(市销率)** | ★★★☆☆ 参考 | 适用于利润率波动较大时作为下限锚定 |
| **DCF(现金流折现)** | ★★★☆☆ 参考 | 公司FCF强劲(OCF TTM 1,340亿)，但重资产周期性导致终端价值假设敏感 |
| **PEG** | ★★★☆☆ 参考 | 适合成长型公司，但需要通过增速预估来判断估值是否合理 |
| **SOTP(分部估值)** | ★★★☆☆ 辅助 | 多业务板块属性差异大，但分部数据颗粒度不足 |
| **EV/EBITDA** | ★★★☆☆ 参考 | 可排除折旧政策和资本结构影响，适合重资产行业 |

**选择组合**：PE(主) + PB-ROE(交叉验证) + DCF(底线验证) + PEG(增速调节)

**分年度PE趋势**（更具参考价值）：

| 年份 | PE中位数 | PE范围(P10-P90) | 背景 |
| :--- | :--- | :--- | :--- |
| 2019 | 40.8x | 34.5-51.4 | 上市初期，快速增长预期 |
| 2020 | 97.7x | 59.4-133.9 | 新能源泡沫期 |
| 2021 | 163.0x | 139.7-196.0 | 极端投机泡沫（不可持续，不宜参考） |

| 2023 | 25.0x | 17.1-39.8 | 利润大增+估值回归 |
| 2024 | 18.6x | 15.6-23.9 | 营收下滑+继续消化 |
| 2025 | 23.1x | 20.4-28.0 | 业绩修复带动PE温和回升 |

**关键判断**：

1. **2021年的163倍PE是极端异常值**——新能源赛道狂热+流动性泛滥+散户化炒作。以此为"历史均值"参考是严重误导。
2. **2023-2026年的18-26倍PE区间**更有参考价值——这一时期公司营收/利润趋于稳定、行业回归理性、估值体系成熟。
3. **当前26.1倍PE**处于2023年以来估值区间的**中上位置**（高于2023-2024年均值，低于2025年高位）。
4. **PB中位数5.67x vs 当前5.78x**——PB基本处于历史中位水平，反映资产端估值合理。

## 二、全变量对冲审计矩阵

### 2.1 基准市值确定

**基准情景**（中性2026E净利828亿，PE 25倍）**←修正**：

**基准市值 = 828亿 × 25 = 20,700亿元** (对应股价约 **454元**)

*选择25倍PE的理由不变*：(1)全球龙头溢价(40.7%份额)；(2)ROE 24.9%行业最高；(3)增速+14.7%仍处于全球电池企业最快梯队。但需注意EPS增速放缓（从+24%降至+15%）可能限制PE扩张空间。

| 审计维度 | 核心审计因子 | **审慎思考(估值影响力过滤)** | 调节建议 | 深度理由 |

| **内部：企业底层特质** | ①技术迭代速度领先行业2年以上 ②全产业链垂直整合能力 ③客户转换成本极高(2-3年验证周期) | 技术领先直接转化为定价权和利润率优势；客户锁定效应在买方市场中极为稀缺；全产业链布局降低上游波动风险。**这些特质确实能提升估值中枢**——如同Intel在PC时代享受的"技术+客户"双溢价。 | **+10%** | 技术领先增强长期确定性→PE中"确定性溢价"+10%。参考Intel在1990s的25-30x PE vs 行业均值15-20x。 |

### 2.3 逻辑纯净性校验（新增锂价分配说明）

**E vs PE 的分配原则检查**：

| 因素 | 在E(利润)中已体现 | 在PE(估值)中体现 | 理由 |
| :--- | :--- | :--- | :--- |
| 碳酸锂价格上涨 | ✓(毛利率假设从25.5%→24.0%) | ✗ | 可量化为成本冲击，进E |
| 储能政策支持 | ✓(营收增长) | ✗ | 可量化为收入，进E |
| 动力电池需求增长 | ✓(销量假设) | ✗ | 可量化为销量，进E |
| 海外毛利率优势 | ✓(利润率假设) | ✗ | 可量化为利润，进E |
| 地缘政治风险 | ✗ | ✓(-15%) | 无法量化具体损失，进PE |
| 技术领先确定性 | ✗(已隐含在量价假设) | ✓(+10%) | 确定性溢价→PE |
| 治理/资本配置 | ✗ | 0% | 不改变估值中枢 |

**校验通过**：无重复计算。锂价冲击已充分在E中体现（毛利率-1.5pct→EPS-7.5%），不应在PE中再次扣除。

## 三、目标价计算与盈亏比

### 3.1 修正目标市值

**修正目标市值 = 20,700亿 × (1 - 5%) = 19,665亿元**

对应目标价 = 19,665亿 ÷ 45.64亿股 = **约431元/股(不复权市价)**

较当前股价411.16元的潜在上行空间：**约4.8%**

### 3.2 三态逻辑加权与合成目标价

| 情景 | EPS 2026E | 目标PE | 目标价 | 概率权重 | 加权贡献 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **保守(Bear)** | 14.24元 | 18x | 256元 | 25% | 64元 |
| **中性(Base)** | 18.14元 | 23.75x | **431元** | 50% | 215元 |
| **乐观(Bull)** | 21.47元 | 28x | 601元 | 25% | 150元 |
| **概率加权目标价** | — | — | **429元** | 100% | — |

**盈亏比计算**：

`盈亏比 = (目标价 - 现价) / (现价 - 保守价)`  
`= (429 - 411) / (411 - 256)`  
`= 18 / 155`  
`= 0.12 : 1`

**核心判断**：当前股价411元已较大程度地消化了锂价上涨带来的盈利下修（PE从26x降至24x，EPS从19.6降至18.1）。但盈亏比仍偏低(0.12:1)。**好消息是**：股价从上次研究时的446元回落至411元(-7.8%)，已进入了上次报告建议的"350-400元买入区间"的上沿附近。
