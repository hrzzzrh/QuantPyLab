# 研报章节五：增长驱动与盈利预测

**研究日期**：2026年05月31日  
**研究标的**：宁德时代（300750.SZ）  
**前序四大模块核心结论**：①CATL财务强劲(ROE 24.9%)，2026Q1营收+52.5%加速增长；②行业从去库转向补库，终端需求保持+15-30%增长，存在中等需求前置风险；③全球动力电池市占率40.7%（Q1 2026再创新高），竞争壁垒四重；④美国市场实质关闭，匈牙利模块组装投产但电芯生产延后。

## 二、利润矩阵的边际穿透

### 2.1 毛利率前瞻（重大修正：锂价飙升至20万元/吨）

**锂价核心变量审计**：

自2026年5月以来，碳酸锂价格出现超预期飙升，从约8万元/吨一路攀升至约20万元/吨（5月20日期货主力合约最高触及20.88万元/吨），较年初约7.5万低位累计涨幅超过160%。驱动本轮锂价暴涨的核心因素包括：

| 供给收缩因素 | 影响量级 | 详情 |
| :--- | :--- | :--- |
| 江西宜春锂云母矿停产 | -4.8~6.6万吨LCE | 6座主力矿山换证停产(5月1日起)，占全国锂云母产能80%+ |
| 津巴布韦锂精矿出口限制 | -2.0~3.0万吨LCE | 出口禁令后改为配额制，实际发运延后至7月 |
| 澳大利亚锂矿运输受阻 | -1.7万吨LCE | 极端天气+柴油短缺影响港口发运 |
| Greenbushes下调产量指引 | -2.0万吨LCE | 全球最大硬岩锂矿产量下调13% |
| CATL枧下窝矿持续停产 | ~20万吨LCE年产能闲置 | 自2025年8月起停产，2026年内复产希望渺茫 |

| 需求爆发因素 | 详情 |
| :--- | :--- |
| 储能电池Q1出货215GWh(+139%) | 储能需求非线性的爆发式增长 |
| 动力电池Q1出货225GWh(+45%) | 磷酸铁锂动力市场维持高增速 |
| AI数据中心储能新增需求 | 24/7绿电要求推动长时储能配置 |
| 全球锂库存降至极低位 | 不足1个月需求量，库销比历史最低 |
| 2026年供给缺口 | 约11.5万吨LCE（兴业期货测算） |

**综合锂价预判**：

| 机构 | 2026年中枢预判 | 关键假设 |
| :--- | :--- | :--- |
| 瑞银 | 20万元/吨（均价），5-6月冲击25万 | 储能需求爆发+供给持续受限 |
| 花旗 | 25-27万元/吨（近3月） | 约7.5万吨LCE缺口 |
| 华创证券 | 16-18万元/吨（全年均价） | 下半年供给释放后回落 |
| 高盛 | ~10.1万元/吨（下半年均值） | 下半年供给过剩 |
| 天齐锂业 | 紧平衡 | 供需全年偏紧 |

**本报告采用**：Q2锂价20-23万元/吨，Q3回落至15-18万元/吨，Q4在12-15万元/吨寻找均衡，**全年均价约16-18万元/吨**（华创路径）。核心逻辑：津巴布韦配额矿7月集中到港+盐湖提锂进入高产期+新项目集中在下半年释放（25-30万吨LCE有效增量），下半年供需将从紧平衡转向宽松。

**CATL的缓冲机制评估**：

| 对冲手段 | 有效性 | 详情 |
| :--- | :--- | :--- |
| **长协价格联动条款** | ★★★★☆ | 与VW、BMW、Tesla等大客户合同中包含原材料价格联动条款，可部分传导成本，但存在1-2个季度议价滞后期 |
| **3个月库存缓冲** | ★★★☆☆ | 存货周转约89天，Q1的低价库存可缓冲短期冲击，但Q2-Q3将逐步消化 |
| **海外矿源自供** | ★★☆☆☆ | 非洲/南美参股锂矿逐步释放产能，但宜春主力矿（枧下窝，20万吨LCE/年）持续停产，自供比例有限 |
| **产品结构升级** | ★★★★☆ | 神行Pro、骁遥双核等高端产品ASP更高，可部分消化成本上涨 |
| **钠电池替代** | ★★☆☆☆（2026年） | 2026Q4量产，2026年对锂需求替代有限，2027年起才有实质影响 |
| **储能高毛利对冲** | ★★★☆☆ | 储能毛利率26.7%高于动力23.8%，储能占比提升有拉高效应，但储能对锂价同样敏感 |

**毛利率情景修正**：

基于锂价全年均价16-18万元/吨的核心判断，结合CATL的成本缓冲与传导能力，对各情景毛利率进行修正：

| 情景 | 原假设(锂价8-9万) | 修正假设(锂价16-18万) | 毛利率影响 | 核心逻辑 |
| :--- | :--- | :--- | :--- | :--- |
| **保守(Bear)** | 毛利率23.0% | **毛利率22.0%** | -1.0pct | 锂价维持20万+全年，竞争压价叠加，成本传导受阻 |
| **中性(Base)** | 毛利率25.5% | **毛利率24.0%** | -1.5pct | 锂价全年均价16-18万，Q1低价库存消化后Q2-Q3承压，Q4随锂价回落+长协调价改善 |
| **乐观(Bull)** | 毛利率26.5%+ | **毛利率25.5%** | -1.0pct | 锂价下半年快速回落至12-15万，长协传导顺畅+产品高端化对冲 |

**关键判断**：
1. **2026Q1毛利率24.8%已反映部分锂价上涨**（Q1锂均价约12-15万元/吨），Q2锂价跳升至20万+将带来额外的100-150bps毛利率压缩。
2. **全年毛利率中枢从此前预估的25.5%下调至24.0%**，下调的主要原因：(a)锂价总体涨幅超预期；(b)价格传导存在1-2季度滞后期，Q2-Q3将是利润率承压最重的阶段；(c)虽然储能出货爆发（+139%）结构上有利，但储能ASP低于动力，量增难以完全对冲价降。
3. **Q2业绩窗口是关键验证点**：若Q2毛利率能维持在24%以上，说明CATL的成本传导能力超预期；若跌破23%，需警惕全年毛利率进一步下修至22%的风险。
4. **锂价对毛利率的边际影响测算**：碳酸锂每上涨1万元/吨（全年均价），估算对CATL综合毛利率的影响约-0.15~0.25pct（考虑传导机制后）。本次锂均价从8-9万上调至16-18万（+8万），对应毛利率下调约1.2-2.0pct，与上述情景修正一致。
