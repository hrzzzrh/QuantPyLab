# 研报章节二：产业周期与终端穿透

**研究日期**：2026年05月31日  
**研究标的**：宁德时代（300750.SZ）  
**前序模块核心结论回顾**：模块一确认宁德时代营收从2024年-9.7%扭转为2025年+17%，2026Q1加速至+52.5%；毛利率恢复至26.3%历史高位；Q1 2026全球动力电池市占率突破40%至40.7%连冠10年。需警惕锂价飙升（5月触及20万元/吨）对毛利率的冲击。  
**本次更新要点**：IEA 2026年全球EV展望发布，预计2026年全球EV销量2,340万辆(+14%)；Q1全球EV销量受中美补贴政策退坡影响下滑8%，但亚太（中国除外）+欧洲市场大幅增长。

## 一、产业生命周期与库存节奏

### 1.1 周期定位：动力电池进入"成熟增长期"，储能电池仍在"高速成长期"

锂电池行业的核心终端市场正处不同生命周期阶段：

| 细分市场 | 生命周期阶段 | 核心特征 | 增长驱动力 |
| :--- | :--- | :--- | :--- |
| **中国新能源乘用车** | 成熟增长期 | 渗透率已超54%（2025），向61%（2026E）→80%（2030E）演进 | 以旧换新补贴延续、产品降价、充换电设施完善 |
| **海外新能源乘用车** | 加速成长期 | 欧洲渗透率29.1%（2025），新兴市场销量增速超60% | CO2排放法规、补贴恢复（德国/意大利/西班牙） |
| **储能电池** | 高速成长期 | 2025年全球出货550GWh(+79%)，中国新增并网189.5GWh(+73%) | 风光装机高增、电力市场化改革、AI数据中心拉动 |
| **商用车电动化** | 起步加速期 | 中国新能源商用车渗透率26.9%（2025），重卡电动化刚起步 | 换电模式推广、运营成本优势凸显 |
| **新兴领域（eVTOL/船舶/换电）** | 萌芽期 | 处于商业验证阶段，尚未形成规模化收入 | 政策支持、技术突破 |

**核心判断**：全球动力+储能电池总需求已跨过1TWh门槛（2024年），2025年达约1.6TWh，McKinsey预测2030年达4.2TWh。宁德时代面对的总可寻址市场（TAM）正以年均约20%的速度扩张，行业仍具备显著的长期增长空间。但增速将从2021-2023年的爆发期斜率（动力电池使用量CAGR约60%）回落至15%-25%的可持续增长区间。

### 1.2 库存动力学：从"主动去库"转向"主动补库"，行业景气度触底回升

锂电池产业链库存周期经历了完整的"主动累库→被动累库→主动去库→被动去库"循环：

**阶段一（2022H1-2023H1）：主动累库→被动累库**
- 上游碳酸锂价格暴涨至60万元/吨，全产业链恐慌性备货
- 下游车企预期过高导致订单虚增，渠道端"牛鞭效应"显著放大需求信号
- 2023年Q1-Q2行业库存积累至历史高位

**阶段二（2023H2-2024H1）：主动去库**
- 碳酸锂价格暴跌至10万元/吨以下，产业链进入惨烈去库存
- LFP正极材料产能利用率降至50%，正极材料价格从17.3万元/吨跌至3.4万元/吨
- 宁德时代营收在2024年出现-9.7%的下滑，是去库存周期的直接反映

**阶段三（2024H2-2025H1）：被动去库→主动补库**
- 2025年H2，EV和ESS需求超预期增长，电池厂和正极材料厂排产持续上升
- 上游锂盐库存从占总库存的45%大幅降至20%，进入快速去库阶段
- 行业总库存天数已降至不足1个月，为2019年以来最低水平

**当前阶段（2025Q4-2026年）：主动补库初期**
- 头部电池制造商订单已排满至2026年全年（据财新报道），产能利用率显著提升
- 碳酸锂市场从供给过剩转向紧平衡，2025年全年去库1-2万吨
- 下游正极材料厂春节前积极补库，市场可流通库存较高点下降约50%

**库存周期的关键启示**：
1. **去库存周期已经结束**。宁德时代2025年营收+17%、2026Q1营收+52.5%不仅仅是基数效应，更包含了渠道补库的"真实需求"回归。
2. **补库需求可能额外拉动2026年出货5%-10%**。当前产业链库存处于历史低位，下游为了保障交付安全，有较强的补库动力。这部分需求与终端消费无关，属于"库存需求"——如果终端需求增速放缓，补库需求可能快速消失。
3. **区分"牛鞭效应"与"真实需求"**：Q4 2025-Q1 2026的强劲出货可能部分源于：（i）海外客户为应对关税不确定性提前备货；（ii）国内车企为应对购置税减半政策提前生产。这两者均属于需求前置，可能在2026H2造成需求真空。

## 二、终端需求穿透与跨期断层审计

### 2.1 最终产品画像

宁德时代的电池产品最终负载于以下终端形态：

| 最终产品 | 销量/装机（2025年） | 电池需求 | 趋势 |
| :--- | :--- | :--- | :--- |
| **新能源乘用车** | 全球2,147万辆(+21.5%) | ~1,000GWh | 强增长 |
| **新能源商用车** | 中国87.1万辆(+63.7%) | ~90GWh | 爆发初期 |
| **公用事业储能** | 全球装机~200GWh | ~300GWh | 爆发增长 |
| **工商业/家庭储能** | 分布式需求 | ~150GWh | 稳定增长 |
| **AI数据中心** | 全球数据中心耗电大幅增长 | ~50GWh（增量） | 新兴高增长 |
| **电动船舶** | 全球~900艘累计（CATL配套） | ~5GWh | 萌芽期 |
| **eVTOL飞行器** | 峰飞航空获全球首张三证 | <1GWh | 概念验证期 |
| **换电网络** | 1,300+站 | ~3GWh | 模式验证期 |

### 2.2 终端景气度审计：未来1-3年需求趋势

#### （1）新能源乘用车：中国正处从"政策驱动"到"内生驱动"的过渡关键期

**确定性增长逻辑**：
- 中国新能源渗透率已从2020年的5.4%飙升至2025年的54%，2026年预计61%。根据IEA预测，2030年中国NEV渗透率将达约80%。
- 产品力全面超越燃油车：2025年同级纯电动车售价已低于燃油车（电池成本$74/kWh→电芯约0.5元/Wh），经济性拐点已过。
- 换购需求成为主力：中国乘用车保有量超3亿辆，每年自然报废+置换需求约1,500-2,000万辆，其中置换为新能源的比例快速提升。

**不确定性/风险**：
- **补贴退坡冲击**：2026年购置税减半（从0%升至5%），叠加以旧换新补贴均值下降21%-30%（从固定金额改为价格比例），实际购车补贴力度显著下降。2026年1月前18天全国乘用车零售同比-28%、新能源车-16%，显示政策换档已造成短期阵痛。
- **2025Q4存在明确的"需求前置"**：2025年12月新能源车渗透率飙至59.1%，是购置税免征末班车效应的直接体现。这部分"透支"可能在2026年Q1-Q2形成需求真空。
- **2026年全年预测**：乘联会预计乘用车零售同比+1%，新能源车零售同比+13%。这说明补贴退坡虽拖累总量增速，但新能源替代燃油车的结构性趋势不变。

#### （2）储能电池：数据中心的"电力黑洞"正成为最大增量变量

**确定性增长逻辑**：
- 中国2025年新增风光装机438GW(+22.3%)，新能源累计装机首超煤电。随着新能源占比提升，电力系统对储能的灵活性需求指数级增长。
- AI数据中心用电需求爆发：2025年美国数据中心耗电量预计增长超40%，谷歌、微软、亚马逊等科技巨头大规模采购清洁能源+储能。
- 国家政策加码：中国发布《新型储能规模化建设专项行动方案（2025-2027年）》，目标2027年新型储能装机1.8亿千瓦（180GW），带动直接投资约2,500亿元。
- 成本快速下降：电芯价格降至$74/kWh，系统成本降至$125/kWh，使储能项目经济性全面改善。

**不确定性/风险**：
- **"取消强制配储"政策**：2025年1月起，发改委/能源局明确"不得将配置储能作为新建新能源项目核准、并网、上网等的前置条件"。这一政策的初衷是推动储能由"被动配置"转向"主动需求"，但短期可能减少部分政策驱动的储能装机。
- **美国FEOC限制**：中国电池企业在美国市场面临"外国关注实体"（FEOC）限制，但美国本土储能电池产能严重不足，韩国企业（LG/SK/Samsung）正在填补缺口，CATL通过技术授权（如与福特合作）曲线进入。

#### （3）数据中心储能：2026年后最具爆发力的增长极

AI算力需求激增正催生一个新的储能市场：
- 全球数据中心电力需求2026年预计增长30%+，单座超大规模数据中心耗电可达100MW+
- 数据中心的电力保障需求（UPS+应急备电）天然适配储能系统
- 宁德时代已推出数据中心专用储能解决方案（Unic系列），并与头部云厂商建立合作关系
- 这一市场目前规模尚小（<50GWh），但增速极快，可能在2028-2030年成为独立的百GWh级别市场

### 2.3 动能溯源与透支审计

#### 增长动能的性质分析：

| 增长驱动力 | 内生/外生 | 可持续性评分 | 分析 |
| :--- | :--- | :--- | :--- |
| 中国新能源车渗透率提升 | 内生为主 | ★★★★★ | 产品力已全面超越燃油车，经济性拐点已过 |
| 海外新能源车市场扩张 | 内生+政策 | ★★★★☆ | 政策（CO2法规）支撑，但关税壁垒增加不确定性 |
| 中国储能市场爆发 | 内生+政策 | ★★★★☆ | "强制配储"取消短期有冲击，但电力市场化改革创造真实需求 |
| 海外储能市场增长 | 内生为主 | ★★★★☆ | AI数据中心需求是真实增量，非政策驱动 |
| 以旧换新补贴刺激 | 外生为主 | ★★☆☆☆ | 2026年补贴力度下降，已出现前期透支 |
| 购置税减半过渡 | 外生为主 | ★☆☆☆☆ | 2026年后再减半/取消的具体方案未明确 |

#### "跨期需求移位"（需求前置）审计：

**强制审计结论：存在中等程度的跨期需求移位风险，但影响可控。**

**证据链**：
1. **2025Q4的"末班车效应"**：2025年12月新能源车渗透率飙至59.1%（全年均值54%），连续3个月销量下滑的异常走势，明确指向消费者抢在购置税免征到期前购车。
2. **2026年1月数据**：前18天新能源车零售同比-16%，远低于乘联会全年预测的+13%。这说明部分需求被提前到2025Q4消耗。
3. **但移位幅度有限**：2024年Q4也存在类似透支（2024年12月渗透率49.6%），但2025年全年仍实现+13%的增长。这说明：（i）新能源车需求的"蓄水池"足够深（大量燃油车主等待换购）；（ii）补贴退坡的影响是渐进的而非断崖的。

**对宁德时代的影响评估**：
- 电池需求滞后于整车销售约1-2个季度（车企备货周期），因此2026H1电池出货受整车需求前置的影响有限
- 2026H2可能面临需求增速放缓的挑战，届时将验证：Q1的52.5%营收增速有多少是"真实需求"，有多少是"补库效应"

## 三、供需边际与压力传导

### 3.1 供需平衡审计：结构性过剩与结构性紧缺并存

**全局视角：总产能严重过剩**

| 指标 | 数值 | 来源 |
| :--- | :--- | :--- |
| 2025年全球锂电需求 | ~1.6 TWh | McKinsey |
| 2025年全球名义产能 | ~4.0 TWh | McKinsey（含在建） |
| 2025年全球有效产能 | ~2.5 TWh | 估算（考虑利用率） |
| 2025年实际过剩 | ~900 GWh | McKinsey |
| 2030年全球名义产能 | ~6.7 TWh | McKinsey预测 |
| 2030年全球需求 | ~4.2 TWh | McKinsey预测 |

**关键辨析：总量过剩 ≠ 有效供给过剩**

1. **产能质量分化**：全球电池产能中，约50%位于中国，而中国产能中至少30%-40%属于落后/低效产能（小厂、旧产线）。有效高端产能（满足车规级要求的优质产线）并不严重过剩。
2. **头部集中度持续提升**：CATL+BYD合计占全球动力电池装机量的55.6%。2025年Q4，头部企业产能利用率已接近满产，订单排满至2026年。而中小电池厂仍在为生存挣扎。
3. **行业出清正在进行**：正极材料价格从17.3万元/吨跌至3.4万元/吨，LFP正极产能利用率仅50%，已导致大批中小企业亏损退出。工信部已于2026年1月召见电池企业讨论优化产能管理。
4. **海外产能严重不足**：欧洲和美国本土电池产能占全球不到10%，且生产成本比中国高50%。CATL的海外工厂（匈牙利、印尼等）面临的是供不应求的市场。

**供需预判**：
- **2026年**：行业整体仍处于结构性过剩，但头部企业有效产能将维持紧张。CATL的产能利用率有望维持在85%+。
- **2027-2028年**：随着落后产能出清+需求持续增长（储能+CAGR 30%+），行业供需格局将逐步改善。届时可能从"总量过剩"过渡到"高端紧平衡"。

### 3.2 压力传导特征分析

#### （1）终端价格波动向电池环节的传导

**传导路径**：终端车企降价 → 压降电池采购价 → 电池企业毛利率承压

**CATL的防御机制**：
- **成本加成型定价**：CATL与大客户（VW、BMW、Tesla等）签订的长期合同中通常包含原材料价格联动条款，能将碳酸锂等成本波动部分传导给下游
- **规模效应降本**：超级拉线（Super Line）技术使单GWh投资成本和制造成本持续下降，对冲了部分降价压力
- **产品结构升级**：高附加值新产品（神行Pro、骁遥双核、天恒储能等）ASP更高，拉高了综合均价

**传导时滞**：约1-2个季度。碳酸锂价格变化→电池成本变化→与车企重新议价→毛利率体现。因此2025年H2碳酸锂价格反弹的影响将在2026年H1的毛利率中体现。

#### （2）波动放大效应评估

- **下游微小的需求波动是否会被显著放大**？——**是，但正在减弱**。
- 2022-2023年期间，"牛鞭效应"极为显著：终端车企基于乐观预测大量下单（甚至重复下单），导致电池厂和材料厂严重误判需求，最终酿成全产业链的库存危机。
- 2025年以来，随着行业成熟度提升和数字化供应链管理工具的应用，"牛鞭效应"有所减弱，但远未消除。2025Q4的"末班车效应"就是最新的例证。

#### （3）CATL的博弈筹码

- **不可替代性**：CATL在全球高端动力电池市场的份额超过39%，且其最新产品（神行Pro 12C超充、骁遥双核等）具有至少1-2年的技术领先期。车企短期内难以找到完全替代的供应商。
- **刚性需求属性**：电池是电动车不可或缺的核心部件，占整车成本的30%-40%。即使车企面临降价压力，也无法绕过电池这个核心环节。
- **多客户分散风险**：CATL的前五大客户集中度估计在30%-40%，任何单一客户的波动不会对整体造成致命冲击。
- **储能对冲**：储能业务（2025年占营收14.7%）的增速远高于动力电池（+29% vs +25%），且储能客户更加分散化，提供了天然的对冲。

## 四、强制复核确认

1. **终端产品景气度预测引证**：✓ 已穿透中国新能源乘用车（2026E: +13%增长/61%渗透率）、全球储能（2026E: +30%+增长）、AI数据中心等终端的需求预测。

2. **跨期需求移位审计**：✓ 已确认存在中等程度的"需求前置"风险——2025Q4末班车效应+2026年1月销量暴跌。该风险对宁德时代2026H1的影响可控，但H2可能面临增速放缓验证。

3. **供需边际演变与压力传导**：✓ 已基于行业产能（4.0TWh名义产能 vs 1.6TWh需求）分析了结构性过剩与结构性紧缺的并存格局，并评估了CATL在价格传导中的博弈地位和时滞特征。
