# 中际旭创 (300308.SZ) 投资跟踪手册
## 建立日期：2026年04月16日

### 1. 核心逻辑验证点 (Thesis Validators)
随财报及月度调研重点验证：
- [ ] **毛利率门槛**：综合毛利率是否维持在 **45%** 以上。若 1.6T 占比提升但毛利停滞，说明行业进入恶性价格战。
- [ ] **1.6T 出货节奏**：2026 年底前 1.6T 累计出货是否突破 **1,000 万只**，份额是否维持在全球第一。
- [ ] **财务议价权**：**应付/应收比率**是否持续大于 1.0。反映其在产业链中的霸权地位是否动摇。
- [ ] **硅光渗透率**：自研硅光芯片在 1.6T 模块中的使用占比是否如期突破 **60%**。

### 2. 关键外部变量 (Exogenous Watchlist)
- [ ] **英伟达 Blackwell 部署**：关注 GB200/GB300 的大规模交付节奏。若终端服务器延期，光模块订单将同步后移。
- [ ] **出口管制动态**：关注美商务部对“200G/400G PAM4 芯片”的出口管制修正案。这是最核心的地缘博弈点。
- [ ] **汇率波动**：美元/人民币汇率波动若超过 5%，需重新计算财务费用的汇兑损益补偿。

出现以下信号，必须考虑下调评级或强制止损：

- **🔴 信号二：毛利大幅下行**。毛利率连续两个季度环比下滑超过 **3 个百分点**，且无法被规模效应解释。
