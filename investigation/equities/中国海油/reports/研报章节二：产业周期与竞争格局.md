# 研报章节二：产业周期与竞争格局

## 1. 产业周期定位：能源转型的“余晖”与“压舱石”
当前全球油气行业正处于复杂的“错位周期”：
*   **需求端（达峰压力）**：随着电动汽车（EV）在全球（尤其是中国）渗透率突破 50%，交通燃料需求已进入增长平稳期。预计 2026-2027 年将见证全球石油需求的物理峰值。
*   **供应端（存量博弈）**：OPEC+ 仍维持减产纪律，但美国页岩油及圭亚那、巴西等新兴海上项目的产能正在加速释放。
*   **周期判断**：行业处于**成熟期向减量期过渡的拐点**。此时，企业的竞争力不再取决于扩张速度，而取决于**“边际成本”**。中国海油凭借极低的桶油成本，在行业下行期依然具备极强的“掠夺性盈利”能力。

## 2. 终端需求穿透与跨期断层审计
*   **终端产品画像**：中海油的原油主要负载于国内炼厂，最终转化为汽油（35%）、柴油（25%）、航煤（10%）及石化原料（30%）。
*   **景气度审计**：
    *   **红灯**：汽柴油受 EV 替代影响，未来两年需求斜率向下。
    *   **绿灯**：航煤随国际航线恢复保持增长；化工原料（乙烯、丙烯）随高端制造业发展维持刚性。
*   **需求预见性**：目前未发现严重的“需求前置”风险，原油消费属于刚性消耗，不存在政策刺激导致的跨期透支。

## 3. 竞争格局扫描：全球第一梯队的“成本之王”
中国海油在全球油气版图中具备独特的竞争生态：

### 3.1 成本对标（核心护城河）
| 企业 | 桶油完全成本 (USD/BOE) | 成本分位 |
| :--- | :---: | :--- |
| **中国海油 (CNOOC)** | **$27 - $29** | **全球 Top 5%** |
| ExxonMobil | $35 - $45 | 第一梯队 |
| Shell / BP | $45 - $50 | 第二梯队 |
| 中石油 / 中石化 | $45 - $55 | 陆上老油田拖累 |
| 美国页岩油平均 | $55 - $70 | 边际成本线 |

**结论**：中国海油拥有约 **10-15 美元/桶** 的结构性成本优势。这意味着即使布伦特油价跌至 50 美元，海油仍能维持 40% 以上的毛利率，而大量同行将进入亏损边缘。

### 3.2 技术壁垒与资源禀赋
*   **深水霸权与技术输出**：公司已完全掌握 1500 米水深（“深海一号”）的开发技术。2026 年，公司正加速向“一带一路”国家（如尼日利亚、莫桑比克）输出深水钻完井及标准化 FPSO 方案，实现从“资源开发者”向“能源技术系统运营商”的身份跨越。
*   **特许经营权**：拥有中国海上油气资源的对外合作独家经营权，在国内海上市场份额 >75%。
*   **海外奇兵（Stabroek 稳固化）**：持有的圭亚那 Stabroek 区块 25% 权益。随着 2025 年 7 月埃克森、雪佛龙与中海油三方仲裁的圆满落幕，雪佛龙的正式入局显著增强了该项目的资金与技术底座，大幅降低了资产的运营风险。

## 4. 供需压力传导特征
*   **压力传导滞后性**：由于中海油多为长期合同且具备完整的上游采掘权，下游炼厂的需求波动主要通过价格机制传导。
*   **博弈筹码**：作为国家能源安全的“压舱石”，公司在海上资源的获取与开发节奏上拥有极强的议价权与抗周期能力。

## 5. 模块二、三研究结论
中国海油在供需过剩的预期中，凭借其**“不可复制的资源禀赋”**与**“极致的成本控制”**，实现了从周期股向“高分红红利股”的蜕变。在全球油价 2026 年预计中枢下移至 60 美元附近的背景下，海油依然是全球最具确定性的获利者。
