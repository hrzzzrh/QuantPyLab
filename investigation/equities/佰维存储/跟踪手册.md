## 三、逻辑证伪警报（Invalidation Triggers）

**以下信号出现任意一条，即触发"逻辑证伪"，必须执行止损或重大评级下调：**

### 🔴 一级证伪（触发即清仓）

- [ ] **毛利率连续两个季度低于25%** → 意味产品结构性升级逻辑破灭，公司回归传统低毛利存储模组商
- [ ] **年化经营现金流净流出超过50亿，且公司无法获得新的股权/债务融资** → 流动性危机
- [ ] **被美国商务部加入实体清单** → 核心晶圆供应中断，海外客户流失
- [ ] **Meta/Google等核心AI眼镜客户宣布更换存储供应商** → 公司核心α被证伪
- [ ] **年审会计师出具非标审计意见** → 财务真实性存疑（存货/应收账款的减值计提是否充分）

### 🟡 二级证伪（触发即大幅减仓至观察仓）

- [ ] **存储晶圆合同价连续两个季度环比下跌超过10%** → 行业景气拐点确认
- [ ] **存货账面价值/年化营收 >130%** → 存货积压严重，减值风险急剧上升
- [ ] **资产负债率突破80%** → 杠杆超出安全边界
- [ ] **公司核心技术人员（主控设计/封测负责人）离职** → 技术壁垒受损
- [ ] **AI眼镜全年出货增速低于+50%** → 行业逻辑弱化

### 🟢 三级证伪（触发即下调评级至"中性"）

- [ ] **连续两个季度营收QoQ下降超过20%** → 增长动能显著衰减
- [ ] **PE(TTM)突破100倍且基本面无超预期支撑** → 估值泡沫
- [ ] **大股东（国家集成电路基金二期）大规模减持** → 信号负面
- [ ] **江波龙/宏芯宇/德明利增速持续超越佰维** → 市场地位弱化
