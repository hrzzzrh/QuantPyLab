# 研报章节二：产业周期与终端穿透

**研究日期**：2026年06月02日

---

## 核心结论摘要（前序模块红线继承）

模块一揭示：佰维存储是在存储超级周期中实现"非线性跳跃"的强周期企业。2025年营收113亿（+69%）、净利8.5亿（+430%），2026年Q1单季营收68亿、净利29亿、毛利率53.3%。但存货高达121亿（占2025全年营收107%），有息负债111亿，经营现金流持续为负（2026Q1净流出27亿）。公司盈利本质高度绑定两个变量：**存储晶圆价格方向**和**高附加值产品占比**。本章需回答：当前的超级周期能持续多久？终端需求是否稳固？供给端何时打破紧平衡？

## 一、产业周期定位：AI驱动的结构性超级周期

### 1.1 周期位置判断

存储行业正处于一场**与历史周期本质性不同**的超级周期（Supercycle）早中期阶段。根据TrendForce、Nomura、IDC、Sigmaintell等多家机构的最新研判，本轮周期具有以下不同于2016-2018年周期的核心特征：

| 维度 | 2016-2018周期 | 当前2025-2027周期 |
|:---|:---|:---|
| 核心驱动力 | 通用服务器换机+手机升级 | AI训练/推理+数据中心基础设施 |
| 需求主体 | 消费电子OEM | CSP云巨头（价格不敏感） |
| 供给特征 | 正常产能扩张 | 晶圆产能向HBM/服务器严重倾斜 |
| 价格弹性 | 季度涨幅最高35% | 季度涨幅60-70%+ |
| 持续时间 | 约9个季度 | 预计至少延续至2027/2028 |
| 消费电子影响 | 量价齐升 | 量缩价涨（被挤出） |

### 1.2 市场规模加速膨胀

全球存储市场规模正在经历指数级增长：
- **2025年**：2,354亿美元（+46% YoY）
- **2026年**：预计5,516亿美元（+134% YoY），其中DRAM 4,043亿（+144%）、NAND 1,473亿（+112%）
- **2027年**：预计8,427亿美元（+53% YoY）
- **2025-2027年化增速**：89.2%

这一规模膨胀的核心动能来自AI服务器的DRAM和NAND需求。AI服务器出货量预计从2025年的235万台增长至2026年的370万台（+51%），2029年有望接近500万台。单台AI服务器搭载的DRAM容量是传统服务器的数倍，且HBM的晶圆消耗量极大——2026年三大供应商约22%的DRAM晶圆投入被HBM占据，2027年将升至30%。同时HBM仅贡献总DRAM bit供应的约9%，形成"消耗大量晶圆、产出极少bit"的供给黑洞。

### 1.3 库存动力学：前所未有的卖方市场

当前库存动力学与历史上任何一次周期都不同：

1. **供应商库存为零**：SK海力士明确表示"可用产能实际为零，几乎没有额外分配给特定客户的空间"。三星、美光同样以利润最大化为导向，持续限制消费级产品的出货量。

2. **CSP买方恐慌性锁单**：北美云巨头（微软、谷歌、Meta、亚马逊）不仅接受涨价，还纷纷主动提出为SK海力士的工厂建设和设备采购提供资金，从长期协议（LTA）升级为"基础设施投资竞赛"。这是存储器行业史上首次出现下游客户主动出资帮供应商建厂。

3. **渠道端囤货前置**：2025年Q4和2026年Q1，PC和手机OEM厂商为规避后续涨价，提前大量采购并堆积渠道库存。IDC数据显示Q4 2025和Q1 2026出货量显著高于正常水平，形成"需求前置"效应，可能意味着2026年下半年真实终端需求将弱于出货数据所展示的。

4. **消费级产品被迫降配**：由于存储成本上涨，OEM厂商开始削减低端产品的DRAM和SSD配置。Gartner预计到2028年低于500美元的入门级PC市场将消失。TrendForce报告指出，部分消费级DRAM产品已出现"存储成本超过终端售价"的情况。

## 二、终端需求穿透：多元终端景气度审计

### 2.1 最终产品画像

佰维存储的产品最终负载于以下六大终端形态：

| 终端形态 | 对应产品 | 需求等级 |
|:---|:---|:---|
| AI服务器/数据中心 | 企业级SSD、RDIMM、CXL模组 | ★★★★★ 最强 |
| 智能手机 | LPDDR、eMMC、UFS、uMCP、eMCP | ★★★ 结构分化 |
| AI PC/个人电脑 | SSD、DDR5模组、BGA SSD、LPCAMM2 | ★★ 短期承压 |
| AI智能眼镜/AR | ePOP、超小尺寸eMMC | ★★★★★ 爆发增长 |
| 智能汽车 | 车规LPDDR、eMMC、UFS、BGA SSD | ★★★★ 稳健增长 |
| 可穿戴/物联网 | ePOP、eMCP | ★★★ 稳步渗透 |

### 2.2 各终端景气度深度审计

#### （A）AI服务器/数据中心 ——【最强增量，持续至2027+】

- **AI服务器出货**：2026年全球预计约370万台（+51% YoY），2028年接近500万台（Sigmaintell）。AI服务器在整体服务器出货中的占比从2025年的18.8%提升至2026年约25%。
- **单机存储强度提升**：AI推理场景要求更大的内存容量和更高带宽。NVIDIA Rubin Ultra平台单GPU将搭载384GB HBM。同时AI服务器的系统DRAM（DDR5 RDIMM）容量也在持续扩容，单台AI服务器的DDR5需求是传统服务器的3-5倍。
- **企业级SSD**：Nomura预计2026年eSSD需求将翻倍（+100%），受AI推理对实时数据存取需求以及HDD产能短缺的双重推动。TrendForce预期2026全年企业级SSD将持续供不应求，供应商拥有绝对定价权。
- **对佰维的影响**：公司企业级PCIe Gen5 SSD和RDIMM产品直接受益。虽然公司不直接参与HBM市场，但高端企业级存储市场的供不应求为公司提供了极高的利润空间和增长确定性。

#### （B）智能手机 ——【总量承压、结构升级】

- **总量**：IDC最新预测2026年全球智能手机出货量将下降13.9%至10.9亿部，创历史最大年度降幅。Gartner预期下降8.4%。Omdia数据显示美国市场Q1 2026已下降3%。
- **量缩价涨**：iPhone 17系列将最低存储配置从128GB提升至256GB；华为Mate 80系列推动512GB规格普及。旗舰机型的存储容量升级是结构性趋势（受端侧AI大模型驱动，系统需40-60GB作为缓存空间），但中低端市场面临严重的规格降配风险。
- **ASP跃升**：2026年智能手机ASP预计达550美元，较2025年上升100美元，创历史新高。
- **对佰维的影响**：公司ePOP、LPDDR5X、UFS3.1等高端产品主要面向旗舰机型，受中低端市场萎缩的冲击相对有限。但uMCP/eMCP等中端产品线将面临显著压力。2026年Q1营收68亿中，若很大部分来自手机端的"渠道前置"囤货，则下半年可能面临需求走弱。**关键不确定性**：Q1的强劲营收中多少是真实终端消费、多少是渠道库存？

#### （C）PC/个人电脑 ——【强逆风，但结构性机会在AI PC】

- **总量**：Omdia预测2026年全球PC出货下降12%至2.45亿台，IDC预测下降11.3%，Gartner预测下降10.4%。低于500美元的入门级PC市场预计在2026年萎缩28%（Omdia）。
- **AI PC受阻**：Gartner预测AI PC渗透率达到50%的时间点推迟至2028年（此前预期2026年）。存储成本飙升导致Copilot+ PC所需的16GB最低内存配置变得更加昂贵，OEM面临"加内存则涨价、减内存则失去AI PC标签"的两难局面。
- **但内存配置结构性上移**：尽管销量下降，单台PC的DRAM需求预计仍增长15%。GPU/显存（GDDR）受供应限制价格持续上涨。
- **对佰维的影响**：PC预装市场（惠普、联想、宏碁等）是公司核心营收来源之一。PC出货下降12%将直接冲击公司SSD和DRAM模组的预装订单量。但公司在后装/电竞市场的品牌（HP、Acer、Predator授权品牌）以及Mini SSD等差异化产品可能提供一定缓冲。

#### （D）AI智能眼镜/AR ——【最强劲的增量引擎，2026年爆发元年】

- **出货爆发**：2025年全球AI眼镜出货约850万副（+330% YoY），Smart Analytics Global预计2026年达2,200万副（+157%），营收达60亿美元（+178%）。TrendForce预测AR眼镜2026年出货95万副（+53%），2030年达3,211万副。
- **苹果入场**：苹果AI眼镜（代号N401）预计2026年底发布，将彻底改变市场格局。三星+谷歌Android XR平台也将在2026年规模化出货。Meta在6个月内将Ray-Ban显示眼镜的组件订单从8万调升至15万（+87.5%），并设定了两年内累计出货50万副的目标。
- **存储需求特征**：AI眼镜对存储要求极高——需要超小尺寸、超薄封装、低功耗的ePOP/eMCP/LPDDR方案，以支持本地AI推理（本地运行Llama-3.2-1B等轻量级模型）。
- **对佰维的影响**：公司ePOP方案（0.54mm超薄，行业最轻薄方案之一）和超小尺寸eMMC（7.16×7.16×0.73mm）精准契合AI眼镜的极致小型化需求。年报中明确提到超小尺寸eMMC已送样客户，正在研发的uPOP3.1（UFS3.1+LPDDR5X）将进一步卡位高端。**AI眼镜是佰维存储未来2-3年最核心的α增量来源。**

#### （E）智能汽车 ——【稳健增长，国产生态受益】

- **单车DRAM需求增长36%**（2026年），受智能座舱大屏化、自动驾驶感知融合、V2X通信等驱动。
- 车规存储的门槛极高（AEC-Q100认证、-40°C至105°C宽温、IATF16949产线），竞争格局较好。
- 比亚迪、华为等国内车企加速智能化，为国产车规存储解决方案提供大量导入机会。
- 公司车规eMMC采用自研SP1800主控，是目前少数可提供全自研全国产车规方案的企业。

## 三、供需边际审计与压力传导

### 3.1 供给端：结构性"赤字"至少延续至2027年

| 厂商 | 2026-2027产能动作 | 有效供给时间点 |
|:---|:---|:---|
| 三星 | P4提前至Q4 2026完成，P5预计2028年3月投产，P6提前至2026年7月动工（2029投产） | 2027年Q1有增量 |
| SK海力士 | M15X已于2026年2月投产（7万wpm），龙仁Y1提前至2027年2月（3万wpm） | 2026年Q2有增量 |
| 美光 | 2026年资本开支超250亿美元（设施/洁净室为主），新加坡NAND厂2028年H2投产 | 2027-2028年 |
| 长江存储 | 三期2026年底运营（5万wpm/2027），四期/五期规划中，三期50%产能转向DRAM | 2027年起 |
| 长鑫存储 | 聚焦工艺升级，向先进制程迁移 | 2026年有限增量 |

**核心结论**：尽管所有主要厂商都在加速扩产，但由于以下结构性因素，2026-2027年内的供给增量难以填补需求缺口：1）扩产周期长达18-36个月（从动工到产出）；2）工艺迁移阶段（如向1c纳米过渡）导致10-15%的产能暂时性损失；3）HBM消耗大量晶圆但产出极少的可用bit；4）设备交货周期因ASML等供应瓶颈而延长。**市场主流共识是供给紧张将至少持续至2027年底，部分机构如Hyundai Motor Securities认为可能延续至2028年。**

### 3.2 需求端：AI正在重塑DRAM产业的"需求版图"

根据Sigmaintell的研究，AI服务器对全球DRAM需求的结构性重塑正在加速：
- **2024年**：AI服务器占全球DRAM需求的约17%
- **2026年**：预计超越40%，首次超过消费电子
- **2027年**：预计攀升至49%，占全球DRAM近半壁江山
- **2028年**：预计突破50%，正式主导DRAM行业需求

与此同时，智能手机占DRAM需求的比例从2024年的43%剧降至2027年的23%，消费电子正在被AI服务器"挤出"。这种结构性转变对佰维存储的启示是：**消费电子端的周期性波动对公司的影响权重正在系统性下降，AI/服务器/企业级的高端需求正在成为更重要的定价锚点。**

### 3.3 供需边际演变方向：从"紧平衡"到"持续紧缺"

**供给侧增量**：2026年DRAM bit产出增速约16%（TrendForce），低于正常的20-25%。2027年随着M15X和Y1产能释放，可能升至20%左右，但HBM将消耗其中的大部分。

**需求侧增量**：AI服务器DRAM需求增长105%+，整体DRAM需求增长约23%（远超供给增速16%）。

**供需缺口判断**：2026年DRAM供需缺口约5-7个百分点，意味着价格具备持续上涨的基础。在供给不能显著改善的背景下，唯一能制衡价格上涨的因素是"需求破坏"——即终端消费者因价格上涨而放弃购买设备。这正是2026年PC和智能手机市场正在发生的事。

### 3.4 压力传导特征分析

从终端市场向佰维存储的压力传导路径如下：

```
上游存储晶圆涨价 → 佰维采购成本上升 → 库存升值（利好毛利率）
                                     → 但需要更多现金囤货（利空现金流）
下游OEM需求变化 → 总订单量下行风险（PC/手机）→ 出货量承压
                 → 但高端产品需求刚性 → 结构性增长（AI眼镜/企业级/车规）
                 → 旗舰机存储容量升级 → 单机价值量提升
```

**波动放大效应**：由于存储晶圆价格是佰维营业成本和库存价值的核心变量，晶圆端10%的价格波动可能在毛利率端放大为20-30%的波动（在低毛利率区域更剧烈）。2023年晶圆价格下行将公司毛利率压至1.71%，而2026年Q1晶圆价格上行则将毛利率推至53.3%，充分展示了这种非对称的放大效应。

**传导时滞**：公司"按需采购+策略性储备"模式意味着：1）低价库存约1-2个季度后释放，形成利润缓冲；2）高价库存约1-2个季度后体现在成本端，形成利润率下行风险。**当前121亿存货中若大部分为相对低价采购，则2026年Q2-Q3毛利率有望继续维持高位；反之，若Q1已大量补充高价晶圆，则下半年毛利率将面临压力。**

## 四、强制复核项

### 4.1 终端产品未来两年景气度预测

| 终端 | 2026年 | 2027年 | 趋势判断 |
|:---|:---|:---|:---|
| AI服务器 | ↑↑ 强劲（+51%出货） | ↑ 持续增长 | 最确定的增量 |
| 智能手机 | ↓↓ 大幅萎缩（-14%） | → 微弱回升（+1.9%） | 量缩、价涨、结构分化 |
| PC | ↓↓ 大幅萎缩（-12%） | → 持平 | 受存储成本严重挤压 |
| AI眼镜 | ↑↑↑ 爆发（+157%） | ↑↑ 持续高增 | 最强α增量 |
| 智能汽车 | ↑↑ 稳健（+36% DRAM） | ↑ 持续增长 | 国产替代空间大 |

### 4.2 跨期需求透支审计

**存在显著的"需求前置"风险**。IDC和Omdia均明确指出了2025年Q4至2026年Q1的渠道前置囤货行为——OEM为规避后续涨价，在价格大幅上涨前集中采购并将成品推入渠道。这意味着：

1. **2026年上半年终端出货数据可能高估了真实消费需求**
2. **2026年下半年品牌商和渠道可能面临库存去化压力**，导致新的采购订单减少
3. **佰维存储Q1 2026的68亿营收中，可能有相当比例属于这种前置需求**，而非可持续的终端消费

这是当前看多逻辑中必须正视的核心风险：**如果2026年下半年去库存叠加终端需求疲软，公司可能面临"量价双杀"——订单减少同时库存成本上升。**

### 4.3 供需边际演变与压力传导审计

**当前状态**：供给严重不足，价格持续上行，卖方掌握绝对定价权。  
**12个月内演变方向**：供给紧张难以根本缓解，价格高位运行甚至继续上行。  
**关键转折点监测**：SK海力士M15X满产进度、三星P4投产进度、长江存储三期DRAM产能释放节奏。  
**对佰维的传导**：在晶圆持续涨价阶段，公司库存持续升值，毛利率和利润表非常好看，但经营现金流持续承压（需更多资金囤货）。**真正的风险点在晶圆价格见顶回落的那一天**，届时逾120亿存货将面临方向性逆转。

## 五、产业周期综合判断

**当前周期阶段**：AI驱动的存储超级周期的**早中期阶段**（类似2017年位置）。距离周期顶部仍有2-4个季度的时间窗口，但涨幅最大、斜率最陡的阶段（2025年Q4-2026年Q2）可能已经或正在接近尾声。

**核心判断**：存储价格在2026年全年大概率维持高位上行，2027年可能趋于平稳但仍高于历史均值。供给端大量新产能的释放时间点在2027年下半年至2028年，届时才是真正的供需拐点信号。

**对佰维盈利预测的关键约束（传递至模块五）**：
1. 2026年营收增速极高（全年有望冲击250-300亿+），但下半年增速将显著放缓
2. 毛利率在Q1的53.3%大概率是周期阶段性高位，Q2-Q4可能温和回落但不低于30%
3. 存货减值风险在2026年内较低（价格上涨），但必须在2027年情景中予以考虑
4. 消费电子终端萎缩（-12%至-14%）将通过订单量传导，部分抵消价格红利
5. AI眼镜带来的增量具有极高的确定性溢价，可给予更高的估值容忍度
