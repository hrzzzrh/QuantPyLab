## 1. 投资摘要 (Investment Summary)

### 1.1 核心投资逻辑

天顺风能正处于**"陆上衰退出清 + 海工爆发爬坡 + 发电稳定支撑"**的三重业务结构切换关键期。**2025年首次年度亏损的本质是"先出清、后起飞"的战略定力**：

1. **海工装备（25.51%营收占比，+220%增速）**—— 2025年成为公司核心增长引擎，"7+1"全球基地网络成型（射阳/通州湾/揭阳/汕尾/阳江/德国等），**国内份额前三、与大金重工形成"双寡头"**。
2. **零碳发电（24.44%营收占比，毛利率62.87%）**—— 1.78GW并网装机提供稳定现金流，已核准储备830MW+建设指标1030MW保障未来增长。
3. **陆上风能装备（45.55%营收占比，-15.68%增速）**—— 主动收缩，2026年1月停产6家子公司，是"战略出清"而非"被动萎缩"。

### 1.2 关键财务结论

| 指标 | 2025A | 2026E | 2027E | 2028E |
|---|---|---|---|---|
| 营业收入(亿) | 53.88 | 55.20 | 67.80 | 78.80 |

| 毛利率 | 18.94% | 23.00% | 25.00% | 27.00% |

**关键判断**：
- **2025年是周期低点**（首次年度亏损+一次性减值）
- **2026年扭亏为盈**（减值压力释放+海工业务放量）
- **2027-2028年**进入"利润弹性兑现期"

**评级：减持 / 观望（中性偏空）**

**核心理由**：
1. **赔率不利**：目标价6.43元 vs 当前股价9.77元 = -34%下行空间，盈亏比仅0.30
2. **估值已偏贵**：当前PB 2.03倍处于历史8%分位，但**没有体现"海工高增长"应有的溢价**
3. **技术面弱势**：跌破MA20/MA60+MACD空头+RSI 25.75超卖（未现底背离）

**反向触发（重新评估的触发点）**：

## 2. 风险因素 (Risk Factors)

按对股价影响的烈度进行优先级排序：

### 2.1 高烈度风险（可能导致股价下跌30%+）

#### 风险1：欧盟反倾销税续期或扩大范围

- **触发条件**：2026年12月17日日落复审后，欧盟对塔筒维持14.4%税率，或扩大至单桩/导管架
- **影响**：
  - **维持现状**：海外塔筒出口成本+14.4%，对天顺影响有限（已主动收缩陆上）
  - **扩大范围**：海外业务受损**3-5亿**
- **当前状态**：欧委会已发布即将到期通知（2026年3月20日）
- **跟踪时点**：2026-12-17

#### 风险2：商誉减值风险

- **触发条件**：海工业务或电站资产现金流不达预期
- **影响**：
  - 商誉19.05亿占归母净资产22%
  - 若计提5-10亿减值，**单次冲击EPS 0.28-0.56元**
- **当前状态**：2025年未计提（已剔除商誉19.05亿），但**仍存隐患**
- **跟踪时点**：每年年报商誉减值测试

#### 风险3：海工业务重大延期

- **触发条件**：阳江基地/射阳二期/德国库克斯港因审批、设备、人员问题延期1-2年
- **影响**：
  - 2026-2027年海工销量下调30-40%
  - 德国基地利用率不足，固定成本摊销压力
  - SOTP估值下调25-30%
- **当前状态**：阳江2026Q2投产+射阳二期2026调试+德国2026Q4-2027Q1投产
- **跟踪时点**：季度披露的产能爬坡数据

#### 风险4：欧洲AR7项目落空

- **触发条件**：2026-2027H1供应商锁定中，天顺未获主流订单
- **影响**：
  - 德国基地50万吨产能利用率不足
  - 失去"出海"逻辑
  - 估值修复"看涨期权"（30亿）归零
- **当前状态**：正在密切推进
- **跟踪时点**：2026-2027H1定标结果

### 2.2 中烈度风险（可能导致股价下跌10-30%）

#### 风险5：钢材价格反弹

- **触发条件**：若2026H2需求复苏+地缘风险，铁矿石/钢材价格反弹10%+
- **影响**：
  - 钢材占成本65-70%
  - 毛利率下降1.5-2.0pct
  - 净利下调0.8-1.2亿
- **当前状态**：钢材价格2026年低位企稳（3320-3360元/吨）
- **跟踪时点**：Mysteel中厚板价格指数

#### 风险6：陆上风电招标量持续低迷

- **触发条件**：2026年陆上招标量<100GW（2025年同比-26%）
- **影响**：
  - 陆塔订单减少
  - 行业"价格战"可能复发
- **当前状态**：2025年陆上招标量已下降26%
- **跟踪时点**：金风科技/每日风电月度招标数据

#### 风险7：人民币兑欧元大幅贬值

- **触发条件**：人民币对欧元贬值超过10%
- **影响**：
  - 德国基地成本上升
  - 汇兑损失
  - 2026Q1大金已确认汇兑损失
- **当前状态**：1 EUR ≈ 7.85 CNY，2025-2026年相对稳定
- **跟踪时点**：人民币汇率

#### 风险8：应收账款持续高位

- **触发条件**：下游EPC资金链恶化，2025年信用减值损失（1.22亿）大幅增加
- **影响**：
  - 2025年应收账款47.59亿（占营收88%）
  - 周转天数310-636天
  - 若信用减值损失翻倍至2.4亿+，净利下调1.2亿
- **当前状态**：高位震荡
- **跟踪时点**：半年报/年报应收账款及信用减值数据

### 2.3 低烈度风险（影响有限）

#### 风险9：A股市场风格切换

- 若市场从"成长/出海"切换至"红利/价值"，天顺可能受冲击
- 但公司**发电业务**（毛利率62.87%）具备"准红利"属性——具有防御性

#### 风险10：管理层稳定性

- 公司治理结构：严俊旭（董事长）+ 上海天神投资（29.52%股权）+ Real Fun Holdings（20.77%股权）
- 家族控股结构：严俊旭与金亮（金亮拥有Real Fun 100%表决权，金亮系严俊旭妻弟）**不构成一致行动人**
- **未发现重大治理风险**

#### 风险11：定增进展不及预期

- 2025年12月披露定增预案（合计19.5亿元），预计最快2026Q3-2026年底完成
- 若定增失败，海工扩产+船舶购置+码头建设将受资金约束
- **影响**：定增失败 → 海工扩产延期 → 2027-2028年增长延后

## 4. 风险/收益的不对称性分析

### 4.1 当前赔率

| 情景 | 目标价 | 概率 | 期望收益 |
|---|---|---|---|
| 保守 | 1.84元 | 25% | 0.46元 |
| 中性 | 5.96元 | 50% | 2.98元 |
| 乐观 | 11.96元 | 25% | 2.99元 |
| **总期望** | **6.43元** | - | - |
| **当前价** | 9.77元 | - | - |
| **期望收益** | - | - | **-3.34元（-34%）** |

### 4.2 盈亏比计算

```
盈亏比 = (乐观目标价 - 当前价) / (当前价 - 保守目标价)
       = (11.96 - 9.77) / (9.77 - 1.84)
       = 2.19 / 7.93
       = 0.28
```

**盈亏比 0.28 < 1**：**典型的不利赔率**

**对比标准**：
- **优秀投资**：盈亏比 ≥ 2.0
- **可接受投资**：盈亏比 1.0-2.0
- **不良投资**：盈亏比 < 1.0 ← **天顺当前位置**

### 4.3 与同行的赔率对比

| 公司 | 当前股价 | 中性目标价 | 盈亏比 | 评级 |
|---|---|---|---|---|
| 大金重工 | 41元 | 45-50元 | 1.0-2.0 | 增持 |
| 天顺风能 | 9.77元 | 5.96元 | 0.28 | 减持 |
| 海力风电 | 75元 | 80-90元 | 0.5-1.0 | 观望 |

**核心判断**：**大金的赔率明显优于天顺**——这反映了"领跑者 vs 追赶者"的市场定价。
