**P端趋势修正说明**：铜价自上次研究（5/8）以来大幅上修——UBS将2026年铜价预测提升13%至$6.0/磅（$13,200/吨），Cochilco上修至$5.55/磅（$12,200/吨）。此水平远超原研报假设（$10,000-12,650/吨）。由于铜占海缆材料成本40-55%，高铜价对新签合同毛利率形成结构性压力。但公司通过①在手订单100%套期保值覆盖存量敞口、②海外项目（毛利率38.5%）占比提升、③产品结构向高附加值柔直海缆升级，可在2026年有效对冲。我们将2026年海缆毛利率假设从33.0%下调至32.0%，以反映新签国内订单铜成本中枢的上移。

**Q端确定性溯源**：
- **内生动力**：南部（阳江）基地2025年全面建成投产（[证据E5]2025年报），海缆产能瓶颈解除；**北海20亿扩产**（2026年5月公告）为2028年后新增产能铺路（[证据E5b]2026/5公司公告）

- **博弈空间**：欧洲Tier 1产能锁定至2028/2029（[证据E7]模块二全球供需分析），中国企业出口窗口持续扩大；Prysmian海底电缆在手订单€12.6B（Q1 2026）验证产能紧张

## 四、分部估值（SOTP）利润分配算法

公司三大业务板块属性差异显著，适用SOTP分部模拟净利润：

### 4.1 SOTP计算参数

| 板块 | 2026E营收（亿） | 毛利率 | K系数 | 分配逻辑 |
| :--- | ---: | ---: | ---: | :--- |
| 海缆与高压电缆 | 68.0 | 32.0% | 1.2 | 高研发强度（超高压+直流技术），需持续高研发投入 |
| 陆缆 | 52.0 | 8.5% | 1.0 | 成熟标准化制造，研发强度低，铜成本敏感度最高 |
| 海洋装备 | 13.0 | 27.0% | 1.2 | 工程服务属性，管理难度高 |
| 其他 | 0.1 | 50.0% | 0.5 | 极小规模 |

总期间费用率 = 6.70%，有效税率 = 13.0%

### 4.2 SOTP分部利润模拟

```
板块模拟净利_海缆 = 68.0 * [32.0% - (6.70% * 1.2)] * (1 - 13.0%)
                  = 68.0 * [32.0% - 8.04%] * 0.87
                  = 68.0 * 23.96% * 0.87
                  = 14.17亿

板块模拟净利_陆缆 = 52.0 * [8.5% - (6.70% * 1.0)] * (1 - 13.0%)
                  = 52.0 * [8.5% - 6.70%] * 0.87
                  = 52.0 * 1.80% * 0.87
                  = 0.81亿

板块模拟净利_海洋装备 = 13.0 * [27.0% - (6.70% * 1.2)] * (1 - 13.0%)
                       = 13.0 * [27.0% - 8.04%] * 0.87
                       = 13.0 * 18.96% * 0.87
                       = 2.14亿

板块模拟净利_其他 ≈ 0.04亿
```

### 4.3 勾稽校验

| 项目 | 金额（亿元） |
| :--- | ---: |
| 海缆分部净利润 | 14.17 |
| 陆缆分部净利润 | 0.81 |
| 海洋装备分部净利润 | 2.14 |
| 其他分部净利润 | 0.04 |
| **∑分部净利润** | **17.16** |
| 公司预测总净利润（Base） | 18.07 |
| **偏差** | **-0.91亿（-5.0%）** |

偏差在可接受范围内（约5%）。SOTP的核心启示不变：公司约83%的净利润来自海缆与高压电缆板块（14.17/17.16）。陆缆由于毛利率极薄（8.5%）+铜成本上行，仅贡献约5%，验证了东方电缆本质上是一家"海缆公司"。

## 八、模块五核心结论对后续模块的传递

1. **传导至模块六（估值定价）**：2026E EPS 2.63元（Base）是估值锚定的核心分母。SOTP揭示了"东方电缆≈海缆公司"的本质——应优先适用PE和DCF，而非PB-ROE。

2. **估值弹性来源**：Bull情景下的EPS 3.20元较Base高出21.7%，主要弹性来源于海外订单超预期和毛利率上行。若欧洲出海逻辑兑现，公司可能享受更高的估值溢价。

3. **关键风险提示**：2026年净利润42.2%的高增速中，约12个百分点的贡献来自"毛利率修复"（2024年低基数效应），这一部分在2027-2028年不可持续。真正的可持续增速中枢约为20-25%。铜价是2026-2027年最大的"盈利预测下行风险"——铜价每持续上涨$1,000/吨，新签合同毛利率约被侵蚀0.5-1pct。

4. **北海扩产的估值含义**：20亿投资（7亿固投）的北海项目虽然增强2028年后增长可见度，但短期将增加Capex和资本开支强度（自由现金流率短期承压），需在DCF模型中体现。
