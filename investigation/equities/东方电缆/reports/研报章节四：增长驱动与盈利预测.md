## 四、分部估值（SOTP）利润分配算法

公司三大业务板块属性差异显著，适用SOTP分部模拟净利润：

### 4.1 SOTP计算参数

| 板块 | 2026E营收（亿） | 毛利率 | K系数 | 分配逻辑 |
| :--- | ---: | ---: | ---: | :--- |
| 海缆与高压电缆 | 68.0 | 33.0% | 1.2 | 高研发强度（超高压+直流技术），需持续高研发投入 |
| 陆缆 | 52.0 | 9.0% | 1.0 | 成熟标准化制造，研发强度低 |
| 海洋装备 | 13.0 | 27.0% | 1.2 | 工程服务属性，管理难度高 |
| 其他 | 0.1 | 50.0% | 0.5 | 极小规模 |

总期间费用率 = 6.70%，有效税率 = 13.0%

### 4.2 SOTP分部利润模拟

```
板块模拟净利_海缆 = 68.0 * [33.0% - (6.70% * 1.2)] * (1 - 13.0%)
                  = 68.0 * [33.0% - 8.04%] * 0.87
                  = 68.0 * 24.96% * 0.87
                  = 14.76亿

板块模拟净利_陆缆 = 52.0 * [9.0% - (6.70% * 1.0)] * (1 - 13.0%)
                  = 52.0 * [9.0% - 6.70%] * 0.87
                  = 52.0 * 2.30% * 0.87
                  = 1.04亿

板块模拟净利_海洋装备 = 13.0 * [27.0% - (6.70% * 1.2)] * (1 - 13.0%)
                       = 13.0 * [27.0% - 8.04%] * 0.87
                       = 13.0 * 18.96% * 0.87
                       = 2.14亿

板块模拟净利_其他 ≈ 0.04亿
```

### 4.3 勾稽校验

| 项目 | 金额（亿元） |
| :--- | ---: |
| 海缆分部净利润 | 14.76 |
| 陆缆分部净利润 | 1.04 |
| 海洋装备分部净利润 | 2.14 |
| 其他分部净利润 | 0.04 |
| **∑分部净利润** | **17.98** |
| 公司预测总净利润（Base） | 18.89 |
| **偏差** | **-0.91亿（-4.8%）** |

偏差在可接受范围内（<5%），SOTP模型通过勾稽校验。K系数的合理区间为海缆1.0-1.5、陆缆0.8-1.2。若将海缆K调整至1.1，∑分部净利润≈18.35亿，偏差进一步缩小。

**SOTP的核心启示**：公司约82%的净利润来自海缆与高压电缆板块（14.76/17.98），陆缆仅贡献约6%。东方电缆本质上是一家"海缆公司"——陆缆和海工是附属业务。这一结论对模块六的估值方法选择具有决定性影响。

## 八、模块五核心结论对后续模块的传递

1. **传导至模块六（估值定价）**：2026E EPS 2.75元（Base）是估值锚定的核心分母。SOTP揭示了"东方电缆≈海缆公司"的本质——应优先适用PE和DCF，而非PB-ROE。

2. **估值弹性来源**：Bull情景下的EPS 3.29元较Base高出19.6%，主要弹性来源于海外订单超预期和毛利率上行。若欧洲出海逻辑兑现，公司可能享受更高的估值溢价。

3. **关键风险提示**：2026年净利润48.6%的高增速中，约15个百分点的贡献来自"毛利率修复"（2024年低基数效应），这一部分在2027-2028年不可持续。真正的可持续增速中枢约为20-25%。
