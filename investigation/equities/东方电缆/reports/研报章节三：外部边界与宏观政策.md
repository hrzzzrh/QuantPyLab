# 研报章节三：外部边界与宏观政策

**研究日期**：2026年05月31日

## 一、宏观环境变量

### 1.1 铜价：结构性高位但有回落预期

铜是公司最大的原材料成本项（陆缆尤为敏感）。2026年全球铜价展望：

| 机构 | 最新预测（2026年） | 方向判断 |
| :--- | :--- | :--- |
| 高盛 | $10,000-11,000/吨 | 结构性看空短期（近月过剩），长期看多至$15,000（2035年） |
| **摩根大通** | **Q2 $13,500 → Q3 $13,000 → Q4 $12,500 → Q1 2027 $11,800** | **Q2后阶梯式回落**（地缘冲突降温+过剩修正） |
| 瑞银 | 2026年赤字约520kt | 矿山供给中断推动严重赤字 |
| Cochilco（智利铜委会） | $5.55/磅≈$12,200/吨（2026/5上修） | 大幅上修（2月预测$4.95/磅），供给紧张+需求旺盛 |
| ICSG（国际铜研究组） | **2026年盈余96kt（4月上修）** | **从150kt赤字大幅上修为盈余**（需求增速1.6%→低于预期+废铜回收+6%） |
| Reuters 30家分析师调查 | $10,500/吨（2026年） | 一致看多，预测区间$9,200-11,500 |

**2026年5月铜价更新（5/31审计）**：自上次审计（5/24）以来，铜价继续超预期运行，已从$13,000区间实质性突破至$14,000区间。**5月13日LME 3个月铜触及$14,140/吨（当日高点），5月21日盘中再破$14,000至$14,153**，之后回落至约$13,600-13,700区间（5月29日3个月收于$13,636，现金$13,615）。COMEX铜（5月29日）报$6.4195/磅（≈$14,150/吨）。**当前铜价中枢约$13,600-14,200/吨，较5/24审计时的$13,000-13,600中枢上移约$600-800/吨。**

**铜价韧性超出所有预测模型的三重结构性支撑**：
1. **硫酸危机**：中国自5月起禁止硫酸出口（年减少约300万吨海运量），智利约37%进口硫酸来自中国→约110万吨浸出铜产量面临风险（Morgan Stanley估算），约占全球精铜产量15%。
2. **美国关税抢跑**：6月30日Commerce Department将决定是否从2027年1月起加征15%普遍关税。COMEX库存从2025年2月至今暴增550%至57.5万吨，LME可用库存骤降至89,725吨（扣除注销仓单）。全球可见库存约150万吨看似充裕，但存在严重地理错配（集中在美国）。
3. **TC/RC暴跌至负值**：2026年现货加工费已跌至负$90-120/吨，中国冶炼厂减产超10%，精铜产量收缩独立于矿产供应。

**但需关注压制因素**：ICSG 4月将2026年从150kt赤字大幅上修为96kt盈余（246kt净变动），主因需求增速下调（1.6% vs 2.1%）+废铜回收增长（+6%）。若伊朗局势缓和+关税抢跑结束，铜价存在从$14,000快速回落至$11,000-12,000的可能（JPM/Q4预测$12,500）。

**对公司的影响评估**：

- **存量订单（正面/中性）**：海缆/高压陆缆项目制订单铜敞口实现**100%套期保值**，中低压陆缆套保超80%。已签合同的毛利率面临极低的铜价波动风险。
- **增量订单（偏负面）**：若铜价持续高于10,000美元/吨，新签合同的铜成本基数被结构性抬高。套保只解决合同签订后的波动，无法消除高铜价对利润率的侵蚀。
- **综合判断（5/31更新）**：铜价$14,000+已经事实性触及此前判定为Bear场景的阈值。**铜价是当前最大的"盈利预测下行风险"**——铜价每持续上涨$1,000/吨，新签合同毛利率约被侵蚀0.5-1pct。考虑到公司存量订单100%套保覆盖+海外高毛利订单占比提升的对冲效应，我们预计2026年全年海缆毛利率将从此前Base假设的32%进一步下移至31%。若铜价在H2维持$13,500+水平（JPM预测Q3 $13,000、Q4 $12,500），全年毛利率影响约-1pct；若铜价突破$15,000并持续，影响将扩大至-2~3pct。**关键缓解因素**：①欧洲海缆订单积压创纪录（NKT €135亿、Prysmian €168亿→中国出口溢价维持）；②公司海外项目毛利率38.5%可对冲部分国内毛利率压力。

**铜价敏感性简化测算**（2026E，5/31更新）：
- 铜价$10,000/吨场景 → 海缆毛利率≈34%（与2025年接近）
- 铜价$12,000/吨场景 → 海缆毛利率≈32%（5/24审计Base假设）
- **铜价$13,600/吨场景（当前实际）→ 海缆毛利率≈31.0%（本次Base假设，-1pct）**
- 铜价$14,000/吨场景 → 海缆毛利率≈30%（Bear场景方向，**已接近触发**）
- 铜价$15,000/吨场景 → 海缆毛利率≈28%（逼近红灯阈值）
- 铜价$16,000/吨场景 → 海缆毛利率≈27%（严重侵蚀）

### 1.2 汇率：海外业务双刃剑

2025年海外营收12.53亿元（+70.83%），毛利率高达38.50%。随着海外业务占比快速提升：

- **人民币升值风险**：海外收入以美元/欧元/英镑结算，人民币升值将直接压缩折算后的人民币收入。
- **公司对冲机制**：已推出**3亿美元（合约价值）外汇套期保值计划**，覆盖远期结售汇、外汇掉期、外汇期权等工具。但3亿美元（约21亿人民币）的覆盖上限vs.未来海外营收可能超过20-25亿的增长轨迹，覆盖可能不完全。
- **综合判断**：汇率是可控但不可忽视的风险。公司已建立制度化对冲体系，但若人民币出现超预期大幅升值（如美国大选后美元走弱），超出套保覆盖的部分将产生汇兑损失。关键在于海外业务增速与套保规模的动态匹配。

### 1.3 利率环境：影响有限但需关注

- 中国处于降息周期（LPR持续下行），有利于降低公司有息负债融资成本。但公司当前有息负债率仅约6%，货币资金远大于有息负债，利率波动的敏感性极低。
- 全球利率环境影响海上风电项目的IRR：若欧洲央行/美联储维持高利率，将增加海上风电项目融资成本，可能延缓部分项目的FID（最终投资决策），间接影响海缆招标节奏。
- **综合判断**：利率对东方电缆的直接影响可忽略，间接影响通过下游项目IRR传导，在目前欧洲海缆供不应求的格局下，这种影响被供给短缺所掩盖。

## 二、地缘政治与贸易环境

### 2.1 重大利好：英国取消风电组件进口关税

2026年4月1日起，英国取消**33项风电进口组件关税**，其中包括海缆（税率从2%降至0%），覆盖范围涵盖海上风电电缆、转子、叶片、变电站辅助系统等。此举旨在释放约**220亿英镑**的北海8.4GW海上风电项目投资。

**对东方电缆的实质影响**：关税清零直接提升了中国海缆在英国市场的价格竞争力。公司2025年已成功交付首个英国超高压海缆项目（Inch Cape），关税取消将进一步降低后续投标的总成本，扩大中标概率。**该政策已于2026年4月1日生效（上次研究时为"即将执行"），利好已落地。**

### 2.1b 重大验证：欧洲海缆巨头订单积压创历史记录（5/31新增）

**Q1 2026欧洲三巨头财报验证产业格局空前紧张**：

| 企业 | 海底电缆/Transmission积压 | 环比变动 | 关键信号 |
| :--- | :--- | :--- | :--- |
| **NKT** | **€135亿**（Transmission） | +€33亿（Q1 2026单季新增订单>€42亿） | 创历史最高积压；Q1获€22亿EGL3（史上最大单缆合同）+€20亿SSEN双项目；HVDC产线锁定至2029年；2027年Karlskrona新产能+第二艘敷设船投产 |
| **Prysmian** | **€168亿**（Transmission总额）/ **€126亿**（海底电缆） | — | HVDC积压约€40亿；DolWin4/BorWin4/Ijmuiden等多项目在建；新工厂美国/意大利/法国扩建中 |
| **Nexans** | **€79亿**（PWR-Transmission） | +2.6% QoQ | 第三艘敷设船Nexans Electra预计2026年中投产；Great Sea Interconnector（€12亿）重谈中 |

**核心结论（强化）**：欧洲三巨头HVDC产线已锁定至2029年，新增订单交付周期36-48个月，且产能扩张最快2027年才能上线。TGS|4C报告指出HVDC turnkey合同成本较2024年预估上升30%（铜价+产能稀缺驱动），海缆供应商定价权处于历史最强时期。这为中国海缆企业（东方电缆、中天科技）提供的出口战略窗口从原先判断的"3-5年"延长至至少"5-7年"——产能缺口不是暂时的，而是结构性的。**东方电缆海外营收从12.53亿（2025）向25亿+（2026E）的跃迁逻辑得到产业端最有力的背书。**

### 2.2 欧洲能源安全焦虑→海缆需求刚性化

中东局势升级（霍尔木兹海峡紧张）+乌克兰危机余波（无人机袭击"土耳其溪"管道）共同加剧了欧洲的能源安全焦虑。第三届北海峰会《汉堡宣言》将北海海上风电联合装机目标定为100GW（2050年），欧盟《清洁能源投资战略》计划2026-2030年每年投入6,600亿欧元。

**核心传导机制**：能源安全焦虑 → 海上风电从"按计划推进"升格为"国家能源安全必选项" → 建设节奏转为"加速抢装" → 海缆需求弹性增加。

### 2.3 地缘政治风险矩阵

| 风险因子 | 概率 | 影响程度 | 具体影响路径 |
| :--- | :---: | :---: | :--- |
| 欧美对中国海缆加征关税 | 中 | 中 | 仿效光伏"双反"逻辑，将海缆纳入贸易限制范围 |
| 英国"国家安全"审查否决中资项目 | 中 | 高 | 明阳智能15亿英镑苏格兰工厂已被传出国家安全审查，海缆可能同样面临政治化审查 |
| 欧盟碳边境调节机制（CBAM） | 高（已确定） | 低-中 | 增加出口合规成本，但海缆碳足迹相对较小，影响可控 |
| 台海/南海地缘冲突 | 低 | 极高 | 可能影响海上风电项目海域使用权审批及保险成本 |
| 中欧关系恶化 | 低-中 | 高 | 全面影响公司欧洲出海战略 |

**核心判断**：地缘政治风险是东方电缆长期估值中的**最大不确定性来源**。当前欧洲海缆供给缺口为中国企业提供了"不可替代性"的护身符，但一旦欧洲本土产能扩张到位（预计2028-2030年），地缘政治因素可能从"背景噪音"升级为"核心制约"。公司选择以属地化运营（荷兰、英国子公司）来降低地缘敏感度的战略是正确的。

## 三、监管与产业政策

### 3.1 国家战略高度契合

东方电缆的业务线与国家战略形成高度共振：

| 国家战略 | 与公司的关联 | 确定性 |
| :--- | :--- | :---: |
| **"双碳"目标（3060）** | 海上风电是能源结构转型的核心载体 | 极高 |
| **海洋强国战略** | 深海脐带缆、海底观测网等产品直接服务国家海洋权益 | 极高 |
| **新型电力系统建设** | 国网4万亿+南网1万亿投资拉动特高压及跨海联网海缆需求 | 极高 |
| **新质生产力/科技自立自强** | 公司被认定为国家高新技术企业、技术创新示范企业 | 高 |
| **"十五五"规划** | 海上风电深远海化、高端海缆超5,000公里新增需求 | 高 |

### 3.2 "单30"政策：强制深远海化的供给侧利好

2024年底发布的"单30"政策（海上风电项目须位于离岸30千米以外或水深大于30米海域），是海缆行业最重要的供给侧政策：

- **对行业的影响**：强制推动海上风电向深远海发展 → 单项目离岸距离大幅增加 → 海缆长度和电压等级要求跳升 → 大幅提升了行业进入壁垒
- **对东方电缆的影响**：深远海化意味着海缆从"阵列缆为主"转向"外送缆为主"，单项目海缆价值量从几亿元跃升至10-30亿元级别。只有三家龙头（东方/中天/亨通）具备500kV/±535kV海缆供应能力，政策壁垒进一步强化了寡头格局
- **政策节奏风险**：深远海开发缺乏明确的规范与标准，5年内大规模铺开的难度较大。但即使年均10GW的保守装机量，也足以支撑高端海缆的持续增长

### 3.3 政策成本与合规

- **环保约束**：海缆路由需通过海洋环评，涉及生态保护区、渔业资源、航道安全等多重审批。近年来审批周期有所延长，但不构成实质性障碍
- **行业准入**：海缆制造不涉及特殊的行政许可壁垒（区别于军工、核电等领域），进入壁垒主要由技术认证和市场业绩构成
- **补贴退坡**：海上风电已进入平价上网时代，对补贴的依赖极低。公司政府补助2025年仅3,782万元（占利润总额的2.48%），基本不受补贴政策退坡影响

## 四、模块四核心结论对后续模块的传递

1. **传导至模块五（盈利预测）**：铜价高位运行应在盈利预测的成本假设中体现（保守情景假设毛利率中枢略微下移），但产品结构升级（高端海缆占比提升）可在量价模型中部分对冲。海外业务毛利率38.50%应作为分部预测的独立假设。

2. **传导至模块六（估值定价）**：地缘政治风险是估值折价的主要来源——需在审计矩阵中量化"欧洲出海战略的政治风险折价"。同时，"十五五"电网投资的极高确定性构成估值溢价的支撑。

3. **定价权判断**：政策环境对东方电缆整体友好。行业不受补贴依赖、不受出口管制（目前）、受国家战略直接支撑，"政策红利窗口"可持续至"十五五"末（2030年）。
