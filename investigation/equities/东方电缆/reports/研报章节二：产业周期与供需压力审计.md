# 研报章节二：产业周期与供需压力审计

**研究日期**：2026年05月24日

---

## 一、产业生命周期与时空坐标定位

### 1.1 行业所处的生命周期阶段

高端海缆（尤其是海上风电用海底电缆）行业当前处于**成长期中段向成熟早期过渡**的关键阶段。判断依据如下：

| 生命周期指标 | 当前状态 | 判断 |
| :--- | :--- | :--- |
| 渗透率 | 全球海上风电在总发电量中占比不足2% | 极低，成长空间巨大 |
| 增速特征 | 2024-2030年全球海风新增装机CAGR约27% | 典型成长期增速 |
| 技术路线 | 500kV交流→±525kV柔直，66kV替代33kV | 技术仍在快速迭代 |
| 竞争格局 | CR3≈87%，三家国内企业进入全球前10 | 已形成寡头，但格局未固化 |
| 进入壁垒 | 需DNV认证、EPC总包能力、历史业绩验证 | 极高，新进入者几乎不可能 |

**核心判断**：行业远未进入存量博弈阶段。渗透率驱动（海上风电从近海走向深远海、从中国走向全球）是未来5-10年最根本的增长逻辑。但需区分：海缆制造环节已进入"寡头成长期"（CR3高度集中），而终端海上风电运营仍处于"导入期"（渗透率极低）。

### 1.2 库存周期动力学定位

结合产业链调研数据，当前海缆产业链库存周期定位如下：

| 阶段 | 时间区间 | 特征 | 东方电缆表现 |
| :--- | :--- | :--- | :--- |
| **主动去库** | 2022H2-2023H1 | 海上风电抢装潮后需求骤降，产业链消化库存 | 存货从15.18亿降至13.21亿 |
| **被动去库** | 2023H2-2024H1 | 需求回暖但企业谨慎，存货维持低位 | 存货从21.48亿降至18.34亿 |
| **主动补库** | 2024H2-至今 | 合同负债暴增，在手订单积压，备货加速 | 存货从18.34亿跳增至39.50亿 |

**核心辨析：订单波动是需求驱动还是牛鞭效应？**

当前在手订单193亿、存货39.50亿的"双高"局面的驱动力拆解：
- **真实终端需求（60%）**：海上风电从近海走向深远海（单项目海缆用量从几十公里增至上百公里），全球海风装机确定性增长。欧洲海缆产能缺口（2025年中压/高压需求约10,000km vs产能约5,700km）已形成订单外溢。
- **渠道备货放大（25%）**：客户（能源央企）为确保海缆产能排期而提前下单并支付大比例预付款（合同负债23.70亿），本质上是供应链安全焦虑驱动的"抢产能"行为。
- **统计口径放大（15%）**：在手订单包含含税金额，且部分为框架协议而非确定合同。

**结论**：当前存在一定程度的"温和牛鞭效应"，即下游客户为确保产能而提前锁定订单，但需求并非虚假——海缆制造产能的稀缺性决定了这种"抢跑"具有合理性。真正的风险不在于需求虚增，而在于项目实际建设进度延迟导致订单积压而非取消。

## 二、终端需求穿透与跨期断层审计

### 2.1 最终产品画像

东方电缆的海缆产品最终负载于以下终端形态：

| 终端产品 | 需求占比（估算） | 海缆类型 | 单项目海缆价值量 |
| :--- | :---: | :--- | :--- |
| 海上风电场（阵列缆+外送缆） | ~65% | 35kV/66kV阵列缆+220kV/500kV外送缆 | 5-30亿元 |
| 跨海电力互联/岛屿供电 | ~15% | 220kV/500kV交流+±500kV柔直 | 3-15亿元 |
| 深海油气田（脐带缆+动态缆） | ~12% | 脐带缆、动态电力缆 | 2-10亿元 |
| 国际间电力互联（HVDC海缆） | ~8% | ±525kV直流海缆 | 10-50亿元 |

### 2.2 终端景气度审计：2026-2028年

**（1）中国海上风电——政策定调明确，但落地存在折扣**

| 指标 | 预测值 | 来源 |
| :--- | :--- | :--- |
| 2026年新增装机 | 10-13GW（产业界） / 15GW（券商乐观） | 行业调研vs券商研报 |
| 2027年新增装机 | 12-15GW（十五五新增项目释放） | 行业调研 |
| 2028年新增装机 | 14-18GW（深远海项目规模化） | 趋势外推 |
| "十五五"年均中枢 | 10-12GW（产业界合理判断） | 一线调研 |

**上限制约**：施工安装资源（吊装船、敷设船）是硬瓶颈，海缆产能虽然也在扩张但相对柔性更大。年均15GW需配套约3-4艘专用海缆敷设船——当前国内仅2-3艘具备大型海缆敷设能力。

**（2）欧洲海上风电——确定性更强，且存在产能缺口红利**

| 年份 | 欧洲装机量 | 海缆需求（km） | 产能缺口 |
| :--- | :---: | :---: | :--- |
| 2025 | 4.5GW | ~11,000 | 缺口~5,300km |
| 2026 | 8.4GW | ~15,000 | 缺口~7,000km |
| 2027 | ~10GW | ~18,000 | 缺口~8,000km |

欧洲Tier 1海缆厂（Prysmian/NKT/Nexans）HVDC产能已预订至2028/2029年，这是中国海缆企业（东方电缆、中天科技）出海的历史性窗口。东方电缆2025年已交付首个英国海上风电超高压海缆项目，验证了产品认证与项目执行能力。

**（3）电网投资——确定性最强的第二支柱**

- 国网"十五五"规划投资约4万亿元（较十四五+40%），南网配套约1万亿元
- 特高压主网架投资超1.2万亿元，超高压电缆市场2026年预计逼近2000亿元
- 跨海联网项目（海南500kV送出、广东柔直工程等）直接拉动高端海缆需求
- 电网投资具有较强的"逆周期"属性，可在海风装机波动时提供需求缓冲

### 2.3 动能溯源与透支审计

**增长动能分解：**

| 驱动力 | 性质 | 持续性评估 | 透支风险 |
| :--- | :--- | :--- | :--- |
| 海上风电从近海→深远海 | 内生技术驱动 | 10年+长周期 | 低 |
| 欧洲海缆产能缺口→中国出海 | 外生供给缺口 | 3-5年窗口期 | 中（缺口终将填补） |
| "单30"政策强制深远海化 | 政策驱动 | 确定性极高但节奏受审批制约 | 低（增加需求而非前置） |
| 国网特高压投资加码 | 政策+基建驱动 | "十五五"期间高度确定 | 低 |
| 浙江/福建军事管制松动预期 | 政策博弈 | 极高不确定性 | 高（若松动将释放被压抑需求） |

**跨期断层审计结论**：

当前增长逻辑中**不存在显著的"跨期需求透支"风险**。核心原因：
1. 海上风电渗透率极低（不足2%），不存在"市场饱和后的需求塌陷"。
2. 海缆的物理寿命（30-40年）远超项目建设周期，不存在"提前安装后多年不再需要"的逻辑。
3. 与光伏、陆上风电的可贸易属性不同，海上风电的"地域锚定"特征决定了不存在"全球产能过剩后倾销"的机制。

**但需警惕的微妙风险**：2021年海上风电"抢装潮"后2022年需求断崖的前车之鉴表明，政策驱动的短期激增可能透支未来1-2年的有效需求。若2026-2027年出现超预期的集中并网要求，2028-2029年可能重现"潮后低谷"。

## 三、供需边际与压力传导

### 3.1 供需平衡审计

**海缆制造环节**：

| 维度 | 当前状态 | 12个月展望 | 24个月展望 |
| :--- | :--- | :--- | :--- |
| 国内有效产能 | 东方/中天/亨通三足鼎立，扩产谨慎 | 南部基地投产、北部在建 | 产能爬坡完成 |
| 国内需求 | 2026年海缆招标≥12GW | 稳步增长 | 取决于深远海政策 |
| 欧洲缺口 | 约5,300-8,000km | 持续扩大 | Tier 1扩产后收窄 |
| 供需判断 | **紧平衡偏紧** | **供给略不足（高端）** | **趋于均衡** |

**关键预判**：产业链供需边际正在向"量增价稳"方向演变。与市场普遍预期的"量价齐升"不同，我们认为：
- **量的增长确定性极高**（订单可见度高、政策支撑强）。
- **价的上涨空间有限**（三强竞争下的价格纪律约束，以及下游能源央企的持续降本诉求）。毛利率回升更多来自产品结构升级（高附加值海缆占比提升）而非单品类涨价。

### 3.2 压力传导特征分析

**从终端市场波动向海缆制造环节的传导路径：**

```
海上风电项目IRR下滑 → 开发商降本诉求增强 → 海缆招标更注重性价比
                                           → 高附加值产品仍有溢价（技术壁垒决定）
                                           → 标准化产品价格承压（陆缆部分）
```

**波动放大效应评估**：
- 海缆在海上风电总投资中占比约8-12%，不属于最大的成本项（风机占40-50%），因此下游降本对海缆的挤压可控。
- 但海缆的"定制化+长交付周期"特征放大了需求信号的波动：下游项目延迟1年 → 海缆厂损失1年排产窗口 → 为填补产能而接低毛利订单 → 毛利率承压。

**传导时滞与弹性**：
- **滞后约6-12个月**：从海缆招标到收入确认通常耗时12-18个月，因此当前高增长的营收反映的是2023-2024年的中标项目。2025年的新中标将在2026H2-2027年转化为营收。
- **博弈筹码**：东方电缆的"不可替代性"（海缆EPC总包能力、高端产品认证壁垒）赋予其较强的议价地位。合同负债/营收从2022年的7.3%跃升至2025年的21.9%，是议价能力提升的直接证据。

## 四、强制复核确认

在进入模块三（竞争格局）之前，严格执行三项强制复核：

| # | 复核项 | 结论 |
| :---: | :--- | :--- |
| 1 | **我是否已经穿透并引用了该行业最终终端产品的未来两年景气度预测？** | 是。已覆盖2026-2028年中国海上风电装机预测（10-15GW/年）、欧洲海风（4.5→10GW）、电网投资（年均万亿级）。 |
| 2 | **我是否审计了当前增长逻辑中是否存在由于刺激政策导致的"跨期需求移位"？** | 是。识别出2021年抢装潮的历史教训，判断当前不存在显著的跨期透支，但警惕2026-2027年集中并网后2028-2029年的节奏性低谷。 |
| 3 | **我是否基于行业产能与终端需求，分析了供需边际的演变方向及其对公司的压力传导特征？** | 是。判断供需边际为"紧平衡偏紧"，价格上行空间有限但产品结构升级可持续推动毛利率改善。压力传导存在6-12个月滞后期，公司的不可替代性构成有效缓冲。 |

## 五、模块二对后续模块的核心传递

1. **传导至模块五（盈利预测）**：海缆量增确定性强（+30-40%），但价增应保守假设（毛利率维持32-35%即可，不宜线性外推）。
2. **传导至模块六（估值定价）**：全球海缆供需缺口提供的3-5年"出口红利窗口"是估值溢价的核心逻辑，而非简单的国内海风增长故事。
3. **传导至模块三（竞争格局）**：CR3≈87%的寡头格局是毛利率稳定性的最重要保障，需在竞争分析中聚焦"三强博弈"而非行业整体。
