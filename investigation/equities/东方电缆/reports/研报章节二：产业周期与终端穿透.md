# 研报章节二：产业周期与终端穿透

**研究日期**：2026年06月14日

---

## 一、逻辑继承：模块一核心结论关照

在进入产业周期分析前，回顾模块一识别的关键财务信号：
1. **海缆业务 2025 年营收 +65.6%，毛利率 33.36%**——这是产业景气度的直接映射。以下需验证该景气度是周期性脉冲还是结构性长坡。
2. **毛利率从 30%→22% 中枢下移**——2025 年的修复（22.11%）能否持续，取决于供需边际走向。
3. **在手订单 193.12 亿，合同负债 23.7 亿**——需从终端需求角度审计其"含金量"，是否存在需求前置透支。

## 二、产业生命周期与库存节奏

### 2.1 海上风电：从成长期迈向加速成长期

**生命周期定位**：中国海上风电处于**成长前期向成长中期过渡**阶段。核心判断依据：

| 维度 | 当前状态 | 生命周期阶段特征 |
| :--- | :--- | :--- |
| 渗透率 | 累计装机 47GW，占可开发资源 3,000GW 仅 **1.57%** | 成长前期 |
| 技术成熟度 | 16MW 风机已成标配，20MW 下线，±500kV 柔直海缆商用化 | 成长中期 |
| 政策强度 | "十五五"累计 100GW+ 量化目标，首次纳入规划纲要 | 强政策驱动→市场驱动过渡 |
| 经济性 | LCOE 降至 0.33-0.38 元/kWh，全面平价，低于沿海省煤电 | 已具备内生经济性 |
| 竞争格局 | 海缆 CR3≈87%，整机集中度高，产业链自主可控 | 成熟期特征（局部） |

**关键时间节点**：
- **2025 年**："十四五"收官年，全年海风新增并网 **659 万千瓦（6.59GW）**，创历史新高
- **2026 年**："十五五"开局年 + **国管项目开发元年**，机构预测新增装机 **11-16GW**（同比增长 60-140%）
- **2030 年**：累计并网目标 **1 亿千瓦以上**，"十五五"期间新增约 **70-100GW**

### 2.2 阶段转换的深层逻辑：从"补贴抢装"到"市场内生"

回顾历史，中国海上风电经历过典型的"补贴抢装→断崖下跌"周期：2021 年国补退出前迎来海风抢装潮，2022 年装机骤降。2025 年再次出现"136 号文"引发的短期抢装（5 月单月装机 2632 万千瓦），但**本轮景气的本质已根本不同**：

| 对比维度 | 2020-2021 抢装周期 | 2025-2030 本轮周期 |
| :--- | :--- | :--- |
| 驱动力 | 补贴退坡前的"末班车效应" | "十五五"规划量化目标 + 市场交易机制 |
| 持续性 | 1-2 年脉冲 | 5 年长坡（十五五 70-100GW 新增） |
| 经济基础 | 依赖补贴，LCOE 0.5-0.7 元/kWh | 全面平价，LCOE 0.33-0.38 元/kWh |
| 需求真实性 | 部分"为拿补贴而建"的低效项目 | 竞价上网、绿电直连，需求由电力缺口和碳减排驱动 |
| 产业链格局 | 产能分散，技术依赖进口 | 国产龙头垄断，全链条自主可控 |

**结论**：当前海上风电已跨越"补贴驱动的伪需求"陷阱，进入由**国家规划刚性约束 + 经济性内生驱动 + 电力市场化交易**三重力量支撑的真成长周期。

### 2.3 库存动力学：海缆产业链的去/补库判断

海缆产业库存有其特殊性——**以销定产（Make-to-Order）**，不存在传统制造业的渠道库存。因此"库存周期"应从以下维度重新定义：

| 库存类型 | 当前状态 | 含义 |
| :--- | :--- | :--- |
| **海缆厂商存货**（东方电缆 2025 年末 39.5 亿） | 高位（+115% YoY） | **主动备产**：多项目同时推进，为交付高峰期做准备，非"滞销" |
| **合同负债**（23.7 亿，+106.5% YoY） | 历史最高 | 客户预付款锁定产能——需求侧在主动"抢产能" |
| **在手订单**（193.12 亿） | 覆盖 1.78 年营收 | 绝对高位，需求侧的刚性锁定量 |
| **开发商风机招标量**（2025 全年 15-20GW 预期） | 历史高位 | 需求端在加速布局 |

**核心辨析——"牛鞭效应"审计**：
牛鞭效应（Bullwhip Effect）指终端需求微小波动经过层层放大传导至上游。对于海缆而言，需要判断合同负债和在手订单的激增是否过度反映了远期的乐观预期：

1. **下游需求真实性**：海上风电开发商（国家能源集团、中广核、华能等央企）的发包行为并非投机性囤货，而是对应明确的电网消纳承诺和"十五五"装机指标。这些项目的最终消费者是电网调度下的电力用户，不存在"层层加库存"的商业链条。
2. **非投机性**：海缆是高度定制化产品（每个风电场的海缆方案都不同），不存在标准化产品的"囤积居奇"前提。
3. **退订风险极低**：合同负债意味着客户已支付 10-30% 预付款，退订成本极高，且在建项目已投入沉没成本。

**判断**：当前海缆产业链的"库存高位"反映的是**需求拉动下的健康扩张**，而非牛鞭效应的虚假繁荣。但需要持续跟踪——如果未来合同负债增速持续放缓或转为下降，则可能是需求拐点的先行信号。

## 三、终端需求穿透与跨期断层审计

### 3.1 最终产品画像

东方电缆产品的最终负载终端主要为：

| 终端产品 | 对应业务板块 | 需求占比 |
| :--- | :--- | :--- |
| **海上风电场**（阵列缆 + 送出缆） | 海底电缆（最主要） | 海缆业务的 80%+ |
| 岛屿电网互联（跨海输电） | 海底电缆 | 海缆业务的 5-10% |
| 海洋油气开发平台（脐带缆） | 海洋装备 | 海洋装备板块 |
| 城市电力网络、特高压工程 | 陆缆 | 陆缆业务的 60%+ |
| 数据中心、轨道交通、建筑 | 陆缆 | 陆缆业务的剩余部分 |

**核心终端**：海上风电场是决定公司中长期增长天花板的最关键变量。

### 3.2 终端景气度审计（1-3 年需求展望）

#### 中国市场：刚性翻倍

"十五五"规划纲要明确："在渤海、黄海、东海、南海海域建设海上风电基地，规范有序推进深远海风电开发，**海上风电累计并网装机规模达到 1 亿千瓦以上**。"

| 指标 | 2025 年末（实际） | 2030 年（目标） | 增量 |
| :--- | :--- | :--- | :--- |
| 累计装机 | 47GW（0.47 亿千瓦） | ≥100GW（1 亿千瓦） | ≥53GW（翻倍+） |
| 中国海缆市场（2025） | 249.80 亿元 | — | CAGR 15-20% |
| 年均新增装机 | 2025 年 6.6GW | 十五五年均 10-15GW | +50-130% |

关键政策节点：
- 2026 年：政府工作报告再提"海上风电集群开发"，发改委明确"十五五"将推进万亿级以上能源重大工程
- 国家电网"十五五"投资 4 万亿（较十四五 +40%），南方电网约 1 万亿
- 广东、江苏、浙江、山东、福建五大千万千瓦级基地已形成

#### 欧洲市场：供需缺口加速中国海缆出海

欧洲海上风电正经历"能源安全 + 算力需求"的双击：

| 指标 | 数据 |
| :--- | :--- |
| 2026-2030 年欧洲海风新增装机 | 34.5GW（欧洲风能协会预测） |
| 2030-2040 年年均装机 | 有望翻倍至 15GW+（数据中心绿电需求驱动） |
| 欧洲海缆供需缺口 | 2024 年起已出现，26-27 年更显著 |
| 中国海缆企业出海机会 | 2022 年以来东方电缆海外订单约 28.3 亿+，2025 年进一步加速 |

**海外海缆供需缺口的量化估算**：参照 4C Offshore 的测算，仅靠海外海缆产能，2024 年起已出现供需缺口且逐年扩大。以 2026 年为例，海外海缆需求 8000km+，按 NKT 320kV 直流海缆 870 万元/公里的价格计算，对应市场规模达 696 亿元——而欧洲本土三强（普睿司曼、耐克森、NKT）的合计产能无法覆盖。

#### 新需求来源：数据中心绿电直连（增量看点）

2025 年绿电直连政策正式落地，欧盟计划出台《数据中心能效法案》，新建数据中心或强制 100% 可再生能源供电。2025-2035 年欧洲有望新增 17GW 数据中心——每一个数据中心园区都可能催生独立的专用海缆输电线路。这是传统海上风电需求之外的**增量变量**。

### 3.3 动能溯源与透支审计

#### 增长动能分解

| 驱动因素 | 性质 | 持续性 |
| :--- | :--- | :--- |
| "十五五"规划量化装机目标（100GW） | **外生政策驱动** | 至 2030 年（中确定性） |
| 海风 LCOE 低于煤电（0.33-0.38 元/kWh） | **内生经济性驱动** | 长期（高确定性） |
| AI 数据中心绿电需求 | **外生技术变革驱动** | 长期（中确定性） |
| 欧洲能源安全焦虑（中东冲突催化） | **外生地缘驱动** | 中期（高波动） |
| 海上风电"竞价上网"→刺激 2025 抢装 | **外生政策窗口驱动** | 短期脉冲（已基本消化） |
| 风机大型化→单位成本持续下降 | **内生技术驱动** | 长期（高确定性） |

**核心判断**：当前动能以**内生经济性 + 长期规划刚性约束**为主，政策补贴驱动的"伪需求"成分已显著降低。但 2025 年 5 月"136 号文"引发的抢装是一个警示——政策窗口期的短期脉冲效应仍然存在。

#### 强制审计：跨期需求透支风险评估

**审计问题**：当前的火爆需求是否透支了未来？

**审计证据**：

1. **"十四五"完成度低，递延而非透支**：沿海各省"十四五"海风规划约 50-60GW，实际完成仅约 26GW（完成度约 50%）。约 20GW 项目**递延至"十五五"**。这意味着当前的需求增量一部分是"补课"——并非透支未来，而是补给过去。
2. **已核准未开工 33GW**：巨大的项目储备意味着需求确定性高，而非提前消耗。
3. **深远海才刚开始**：当前开发集中在近海（离岸 <50km），深远海（>50km、水深 >50m）仅处于起步阶段。深远海可开发资源约 1,582GW（漂浮式），远大于近海 1,400GW。开发重心转向深远海才是真正的增量空间。
4. **全球渗透率极低**：中国海风仅占可用资源的 1.57%，全球海上风电技术可开发潜力 71TW。这不是一个随时会被"开发完毕"的市场。

**审计结论**：**不存在显著的跨期需求透支风险**。当前的旺盛需求更多是"十四五积压 + 十五五加速"的叠加，而非提前消耗未来。真正的风险在于项目审批和建设进度是否如期（该风险不属于需求端，属于执行端）。

## 四、供需边际与压力传导

### 4.1 供需平衡审计：紧平衡→偏紧

#### 海缆供给端格局

**竞争格局**：中国海缆市场高度集中——CR3≈87%（中天科技 36-38%、东方电缆 32-34%、亨通光电 16-18%），且该格局因**三重壁垒**（技术认证周期 1 年+、码头独占性、业绩门槛）极难被打破。

**产能扩张节奏**：
| 企业 | 现有基地 | 扩产进度 |
| :--- | :--- | :--- |
| 东方电缆 | 东部（北仑）+ 南部（阳江，2025 年投产） | 北部（烟台，建设中） |
| 中天科技 | 南通 + 汕尾 | 持续扩产 |
| 亨通光电 | 常熟 + 射阳 | 持续扩产 |

**供需平衡判断**：
- **短期（2026-2027）**：偏紧。需求端年均海缆新增招标 15-20GW（对应海缆需求 5000-8000km），而头部三家合计产能在快速扩张中但仍需产能爬坡。南部基地（阳江）2025 年刚投产，正在经历产能利用率爬升，尚不能满产。
- **中期（2028-2030）**：产能释放追赶需求。随着三大头部企业新建产能完全达产，供给端将改善，但欧洲供需缺口可能持续吸引中国产能外溢，稀释国内供给端的宽松效应。
- **长期风险**：若所有规划产能如期达产，2029 年后可能出现局部产能过剩。但海缆的高定制化属性（非标品）决定了"过剩"的含义不同于标准制造业——过剩的是中低端产能，高端（500kV+、柔直）产能大概率长期紧平衡。

#### 毛利率走势预判

**关键问题（继承模块一的核心关注）**：海缆毛利率 33.36% 是否可持续？

| 压力方向 | 因素 | 影响 |
| :--- | :--- | :--- |
| **↑ 上行** | 高电压等级（500kV+）产品占比提升 | 高附加值 → 毛利率上行 |
| **↑ 上行** | 海外订单（外销毛利率 38.5% vs 内销 19.9%）放量 | 结构性改善 |
| **↑ 上行** | 产能利用率提升→单位固定成本下降 | 规模效应 |
| **↓ 下行** | 中天科技、亨通光电同步扩产，竞标竞争加剧 | 价格压力 |
| **↓ 下行** | 铜价波动（原材料占成本 89%） | 成本端冲击 |
| **→ 中性** | 陆缆毛利率 8.65%（低但稳定），占比可能重新回升 | 稀释但不致命 |

**预判**：2026-2028 年海缆毛利率有望维持在 **30-35%** 的高位区间。支撑理由：1）高端产品（500kV 柔直等）占比快速提升，此类产品毛利率可达 40%+；2）欧洲项目毛利率显著高于国内；3）产能偏紧环境下议价能力较强。但 2029 年以后，若国内产能释放+欧洲本土扩产完成，毛利率可能回落至 25-30%。

### 4.2 压力传导特征分析

#### 从终端波动到海缆厂商的传导路径

```
终端电力价格波动 → 风电开发商 IRR 变化 → 招标/开工节奏 → 海缆订单 → 海缆厂商产能利用率和定价
```

**传导时滞**：约 1-2 年（从开发商决策到海缆招标再到交付确认收入）

**波动放大效应**：
- **存在但不剧烈**：风电开发商的招标节奏确实具有周期性（如抢装年集中释放），但海缆环节的放大效应有限，因为：1）海缆是"瓶颈环节"（产能有限），开发商必须提前锁定；2）合同负债机制提供了需求先行指标和现金流缓冲。
- **方向判断**：当前处于"向上波动"阶段——开发商为锁定远期产能提前下单，东方电缆 2025 年合同负债同比增长 106.5% 即为明证。

#### 公司博弈筹码

| 博弈维度 | 东方电缆的筹码 | 强弱评估 |
| :--- | :--- | :--- |
| **对下游（开发商）** | 稀缺产能 + 高端资质 + EPC 总包能力 | ★★★★★ 极强 |
| **对上游（铜材供应商）** | 大额采购量（年采购 66 亿+），期货套保对冲 | ★★★ 中等 |
| **对竞争对手** | 技术领先（500kV 柔直率先商用）+ 码头资源 + 南部基地投产 | ★★★★ 强 |

核心结论：东方电缆在产业链中处于**强势地位**。2025 年合同负债的大幅增长，本质是下游客户在"抢海缆产能"而非公司在"抢订单"——这构成了定价权向公司倾斜的有利环境。

## 五、强制复核项

1. ✅ **终端产品未来两年景气度预测已引用**：中国海风十五五目标 100GW+，2026 年新增 11-16GW，欧洲 2026-2030 新增 35GW
2. ✅ **跨期需求移位审计已完成**：当前需求源自"十四五积压递延 + 十五五刚性目标 + 内生经济性"，不存在透支
3. ✅ **供需边际演变方向已分析**：短期偏紧利好公司议价，中长期需关注产能释放节奏和高端 vs 低端的分化

## 六、模块二结论摘要

海上风电产业处于**加速成长期**，终端需求由"十五五"规划刚性约束（2030 年 100GW+）、内生经济性（LCOE 全面平价）及欧洲能源安全焦虑三重力量驱动，"补贴抢装"的旧逻辑已被"市场+规划"的新范式取代。当前海缆产业链处于**供需偏紧**状态，中国三强（CR3≈87%）垄断格局稳固且短期无法打破，东方电缆依托技术壁垒和产能扩张在产业链中具备强议价权。不存在显著的跨期需求透支风险——"十四五"低完成率意味着当前需求部分是历史积压的补课，而非对未来的提前消耗。

**对模块三的指引**：需深入审计东方电缆 vs 中天科技 vs 亨通光电的差异化竞争优势，判断三家同步扩产是否会导致竞争格局恶化。重点关注海外订单获取能力的分化——这是决定毛利率能否维持高位的关键变量。
