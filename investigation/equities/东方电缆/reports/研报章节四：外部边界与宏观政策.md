# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月25日

## 一、逻辑继承：前序模块核心结论关照

1. **模块一**：2025 年外销营收 12.53 亿（+70.8%），外销毛利率 38.5%（远高于内销 19.9%）——以下需审查汇率损益对海外利润的侵蚀程度及贸易壁垒变化。
2. **模块一**：原材料（铜）占营业成本 89%，毛利率受铜价高度敏感——以下需评估铜价中长期走势及其对成本端的传导。
3. **模块二**：欧洲海缆供需缺口是中国企业出海的核心机遇——以下需审计地缘政治风险对这一机遇窗口的威胁。

#### 铜价对东方电缆的传导机制

**直接材料成本**：铜占营业成本 89%（2025 年，直接材料 75.30 亿/总营业成本 84.43 亿）。

**传导效率**：铜价上涨对毛利率的侵蚀存在 1-2 个季度的滞后，因为：
1. 公司通过期货套期保值（2026 年最高持仓 5500 吨铜期货）对冲短期波动
2. 海缆订单的定价机制通常包含原材料价格调整条款
3. 部分长期合同为"闭口合同"（固定价格），铜价急涨时利润被压缩

**2025 年的正向意外**：尽管铜价在 2025 年大幅上涨，公司整体毛利率从 18.83%（2024）反弹至 22.11%（2025），原因是：
- 海缆毛利率从 ~27.8% 跃升至 33.36%（产品高端化对冲了成本上涨）
- 陆缆毛利率仅从 8.19% 微升至 8.65%（陆缆对铜价更敏感，但占比下降）

**中长期预判**：铜价大概率维持 10 万+/吨的高位中枢（超级周期逻辑未破），东方电缆的毛利率韧性取决于：
- 高端产品占比能否继续提升（高附加值 → 高容忍度）
- 套保策略的有效性
- 是否能在合同中成功传导上涨成本

## 三、地缘政治风险审计

### 3.1 中欧贸易摩擦升温

**风险信号（6月更新——显著升级）**：
- 欧盟执委会明确表态"当前欧中贸易和投资关系不可持续"
- 法国、西班牙、意大利等五国联合提议更强硬的对华贸易机制
- 欧盟已对金风科技（2026年2月）等中国风电企业启动《外国补贴条例》（FSR）深入调查
- **【新增】欧盟"Team Europe"海缆安全战略**（2026年3月24日正式启动）：将海底电缆提升为"关键欧洲资产"，建立统一安全框架。战略明确：13个"Cable Projects of European Interest"优先使用本地供应商，CEF Digital拨款5.33亿欧元（含3.47亿欧元战略性项目拨款）支持本土海缆产能
- **【新增】欧盟拟推出"供应链多元化工具"**（2026年6月EU峰会）：要求战略行业企业减少对单一供应来源的依赖，可能强制性地限制中国海缆在欧盟关键基础设施项目中的份额
- **【新增】中国启动《阻断外国法律与措施不当域外适用办法》反制**（2026年5月）：将欧盟FSR调查定性为"不当域外管辖"，首次实际运用反制框架

**对东方电缆的实质影响分析（重大更新）**：

| 风险维度 | 当前状态（6月25日更新） | 对东方电缆的影响 |
| :--- | :--- | :--- |
| 海缆关税 | **英国已取消33类风电组件进口关税（2026.04）** | ✅ 利好（英国为东方电缆当前最主要海外市场） |
| 欧盟反补贴调查 | **已扩展至风电行业**——金风科技（2026.02）被正式调查；欧盟21项活跃贸易防御调查中18项指向中国 | 🔴 风险从"未波及"升级为"已波及风电行业" |
| **EU海缆安全战略** | **Team Europe框架正式运行**，海缆被纳入关键基础设施安全审查，欧洲制造商获91%历史订单份额 | 🔴 结构性壁垒——非关税但效果等同"软性配额" |
| **EU多元化工具** | 正在制定中，可能设置中国供应商份额上限 | 🟡 政策方向明确但实施时间和强度待定 |
| 碳边境调节（CBAM） | 逐步实施 | ✅ 中国新能源产业链的碳足迹优势反而有利 |
| 供应链安全审查 | 欧洲关注关键基础设施依赖 | 🔴 海缆已被明确定义为"关键欧洲资产"，安全化趋势不可逆 |
| 中国反制措施 | 阻断办法首次实施（2026年5月） | ⚠️ 可能使中国企业在欧盟的合规操作陷入两难 |

**核心判断（重大修正）**：原6月14日研报判断"短期（1-2年）海缆尚不在中欧贸易摩擦的靶心"，这一判断需要修正。欧盟Team Europe海缆安全战略（2026年3月）已将海缆正式纳入关键基础设施安全审查框架，其性质已从"贸易摩擦"升级为"安全审查"——后者的规避难度远高于前者。

**具体影响路径**：
1. **实质性市场准入限制**：欧洲市场91%的海缆合同授予本地制造商（Prysmian/Nexans/NKT），Team Europe战略通过安全审查机制进一步固化了这一偏好。尽管中国海缆在技术和价格上具备竞争力，但"安全审查"标签构成无形的制度性壁垒
2. **当前影响有限——英国市场是缓冲**：东方电缆当前海外营收主要来自英国（Inch Cape等项目），英国已取消风电组件进口关税且不在Team Europe安全审查框架的直接约束范围内。欧盟大陆市场的准入壁垒暂时不影响英国/东南亚/中东等出海方向
3. **中长期风险显著上升**：若欧盟多元化工具落地并设置中国供应商份额上限，将直接限制东方电缆在欧洲大陆的远期扩张空间。不过荷兰子公司和英国孙公司（本地注册实体）可能提供部分规避路径——关键在于欧盟对"本地企业"的定义是看注册地还是看最终控制方

**概率量化**：
- 2年内欧盟正式限制中国海缆参与关键基建项目：概率从10%→**30%**（Team Europe战略已搭好制度框架）
- 英国跟进类似限制：概率 **<10%**（英国自身海缆产能为零，严重依赖进口，且已取消关税）
- 2026年东方电缆欧洲订单受实质影响：概率 **<15%**（现有英国订单不受影响，欧洲大陆新订单尚未大规模启动）

### 3.2 美国 FCC 海底电缆监管收紧

**关键事件**：2026 年 6 月 3 日，FCC 宣布加强对海底通信电缆的监管，拟扩大限制中国企业的设备使用范围。

**影响辨析**：
- 此次 FCC 新规主要针对**海底通信光缆**（互联网数据传输），非东方电缆主营的**海底电力电缆**（海上风电输电）
- 但 FCC 的监管思路有可能延伸至电力海缆领域——需持续跟踪
- 东方电缆目前的海外布局集中在欧洲（荷兰、英国）和东南亚，不涉及美国市场，短期内直接影响有限

### 3.3 全球供应链重构（重大更新）

**趋势**："去风险化（De-risking）"——欧美企业寻求供应链多元化，减少对中国依赖。这一趋势在过去两周出现**显著加速信号**：

**新证据**：
- **TGS/4C Offshore 白皮书**（2026年6月16日发布）：欧洲2026-2040年累计海缆供需缺口22,800km，但**欧洲制造商占据91%的本地市场份额**，中国制造商面临技术和监管双重门槛
- **华盛顿邮报WP Intelligence**（2026年3月）：警告中国可能控制全球80%的海缆过剩产能，"美国和盟友应将HVDC海缆视为关键基础设施"
- **欧盟多元化工具**（制定中）：首次将"去风险"从战略表述转为可执行的制度设计
- **TGS/4C**报告明确提及：即使中国厂商低价竞争，欧洲项目出于电网合规（Grid Code）、保险要求和维护考虑，仍倾向于选择欧洲供应商——这不是关税壁垒，而是**制度惯性和标准排他性**的结果

**对东方电缆的辩证分析（更新）**：
- **不利面（升级）**：Team Europe安全战略 + 欧盟多元化工具 + 91%本地偏好 → 欧洲大陆海缆市场的制度性壁垒正在从"软性"变为"半硬性"。东方电缆进入欧盟大陆市场的难度显著高于6月14日研报中的评估
- **有利面（不变）**：当前过渡期内（2026-2029），欧洲本土产能严重不足——TGS/4C报告确认年缺口2,200-4,500km（27%）。这一刚性缺口是中国企业唯一的结构性机遇
- **关键变量**：英国市场——英国自身无海缆产能，且已取消风电零部件关税，不受Team Europe安全框架直接约束。东方电缆的英国订单（Inch Cape等）验证了英国仍是开放市场
- **对冲策略（更新）**：东方电缆的荷兰子公司和英国孙公司（本地注册）是应对制度性壁垒的核心筹码。但需关注欧盟对"本地企业"最终控制方的审查标准是否会收紧

## 五、通胀传导与成本转移效率

### 5.1 铜价→海缆价格的传导路径

```
铜价上涨 → 直接材料成本上升 → 海缆产品报价调整 → 下游接受 → 毛利率影响
```

**传导时滞**：1-6 个月（取决于订单定价模式）
**传导效率**：取决于产品差异化和市场供需

### 5.2 东方电缆的传导能力评估

| 企业特征 | 对成本传导的影响 |
| :--- | :--- |
| 海缆产品差异化强（非标品） | ✅ 有利于向下游传导 |
| 海缆供应商稀缺（仅 3 家可做 500kV+） | ✅ 议价能力强 → 传导效率高 |
| 部分订单为闭口合同 | ⚠️ 铜价急涨时利润承压 |
| 已建立期货套保体系 | ✅ 缓冲短期波动 |
| 陆缆产品同质化严重、竞争激烈 | ❌ 传导困难，毛利率低且敏感 |

**结论**：铜价上涨对海缆业务的毛利率影响相对可控（产品溢价空间大 + 供应商议价权强），但对陆缆业务构成持续压制。这从一个侧面解释了为何公司毛利率从 30% 降至 22%——低毛利陆缆占比的波动是主因，而非海缆的成本传导不畅。

## 六、模块四结论摘要

东方电缆面临的外部宏观环境呈现"**国内强劲顺风 + 海外机遇窗口边际收窄 + 成本端高位震荡**"的复杂格局。

**利好**：1）"十五五"海上风电被提升至国家战略高度，100GW+ 累计装机目标刚性约束，6月发改委42号令和NEA调度会进一步强化消纳制度保障；2）国家电网4万亿投资提供确定性需求基座；3）欧洲海缆供给缺口（TGS/4C测算年缺口27%）仍然巨大，供需面利好未变。
