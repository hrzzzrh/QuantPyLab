# 研报章节七：投资摘要与风险因素

**研究日期**：2026年03月06日

## 1. 投资摘要 (Investment Summary)

*   **业绩爆发前夜**：东方电缆正处于“十五五”规划红利与历史最高量级订单（195.5 亿）的交汇点。2026 年预计营收将达 160 亿（+52%），EPS 预计 3.26 元，实现跨越式增长。
*   **核心竞争力护城河**：公司通过“500kV 超高压技术领先 + 自有专业船队 EPCI 一体化”构建了极高的准入门槛。在 2026 年全球海缆铺设船结构性紧缺的背景下，自有船队将成为溢价的核心来源。
*   **估值定价结论**：当前 60.75 元的股价对应 2026F 仅 18.6x PE，显著低于历史中枢（38.8x）。保守目标 PE 39.9x 对应目标价 **130.00 元**，具备 **+110%** 的预期上涨空间。

## 2. 风险因素 (Risk Factors)

1.  **原材料价格波动（极高）**：全球铜价若长期突破 $15,000/吨，且下游成本传导受阻，将直接侵蚀毛利空间。
2.  **地缘政治与贸易壁垒（高）**：欧盟 NZIA 与 CBAM 政策可能限制海外高溢价订单的获取速度及利润。
3.  **项目交付进度风险（中）**：海上风电受天气、军事、环保审批等外部因素影响大，若大项目交付推迟至 2027 年，将导致 2026 年业绩不及预期。
4.  **行业竞争加剧（中）**：起帆、汉缆等二线厂商产能释放，若 35kV/66kV 等中低端产品毛利崩塌，将拉低整体盈利能力。

## 3. 研究结论象限图 (Final Evaluation Matrix)

本报告将东方电缆精准定位于：
**第一象限 (High-Conviction Buy) —— 高确定性 + 高预期收益**

*   **确定性来源**：195.5 亿已签约在手订单提供业绩底色。
*   **预期收益空间**：估值严重错配，业绩增长与估值修复有望实现“戴维斯双击”。
