# 研报章节五：增长驱动与盈利预测

**研究日期**：2026年07月02日
**研究对象**：杭州天地数码科技股份有限公司（300743.SZ）
**目标年份**：2026年（当前为Q3，锚定当年EPS）

---

## 一、核心增长驱动力解构

### 1.1 驱动力一：中高端产能释放——维森二期投产

**【证据链】**

| 时间节点 | 事件 | 确定性 |
|:---|:---|:---|
| 2022年 | 收购安徽维森100%股权 | 已完成 |
| 2023-2024年 | 维森一期技改（1.5→5.6亿平米） | 已完成 |
| 2025年 | 二期项目建设（主体完工+设备安装调试） | 已完成 |
| 2026年2月 | 取得排污许可证，进入试生产 | **已实现** |
| 2026年H2 | 预计完成安全/环保验收→正式量产 | 高概率 |

维森二期规划产能2.3亿平米，专注TTO高速边压碳带（属于中高端产品）。这是天地数码最确定的短期增长引擎。

**销量（Q）预测**：
- 2026年为试生产→量产过渡年，假设有效利用率30-40%
- 新增有效产量：2.3亿 × 35% ≈ **0.8亿平米**
- 2025年总销量18.9亿平米，2026年自然增长约5-6%（约1.0亿平米）
- **2026年总销量预测：18.9 + 1.0 + 0.8 = 20.7亿平米**（+9.5%）

**确定性评分：4分（高）**
已取得排污许可证并进入试生产，实物资产已就位。不确定性在于试生产到量产的时间节奏和良率爬坡速度。

### 1.2 驱动力二：CALOR/RTT全年并表贡献

2025年2月28日完成对德国CALOR和法国RTT的收购（对价594万欧元），2025年仅并表10个月。2026年将实现全年并表。

**收入增量估算**：
- 2025年CALOR+RTT并表收入估计约0.3-0.5亿（10个月）
- 2026年全年收入估计：0.4-0.6亿
- 增量贡献：**约0.1-0.2亿**

这部分收入主要来自彩色碳带和欧洲本地化服务，毛利率预计在35-40%，高于公司平均水平。

**确定性评分：4分（高）**
已完成交割，团队和产线已就位。

### 1.3 驱动力三：新兴市场渠道扩张

2025年新增新加坡、印尼子公司。越南子公司（2023年设立）和泰国子公司（2024年设立）仍处于早期爬坡阶段。

**收入增量估算**：
- 越南+泰国+新加坡+印尼四国的渠道收入：2025年估计约0.3-0.5亿
- 2026年在低基数上实现高增长（50-100%）：增量约**0.2-0.4亿**

**确定性评分：3分（中等）**
方向确定（东南亚制造业扩张带来增量需求），但收入规模爬升节奏存在不确定性。

### 1.4 驱动力四：产品结构升级（混合基/树脂基占比提升）

根据行业数据，蜡基碳带ASP约2-3元/平米，混合基约5-8元/平米，树脂基约10-15元/平米。维森二期和CALOR/RTT的产品均为中高端碳带，天然推升ASP。

2025年天地数码ASP（按总营收/总销量）：8.85亿/18.93亿平米 = **4.67元/平米**。这一ASP在行业中处于中低水平（大量蜡基产品拉低均值），隐含的混合基+树脂基占比可能仅25-30%。随着维森产能释放和产品升级，ASP有望温和提升。

**ASP预测**：
- 保守情景：ASP维持4.67元/平米（竞争加剧，价格承压）
- 基准情景：ASP升至4.75元/平米（结构优化+1.7%）
- 乐观情景：ASP升至4.85元/平米（中高端放量+3.9%）

**确定性评分：3分（中等）**
方向明确但速度取决于市场开拓进度。

### 1.5 驱动力五：成本端改善（原材料价格回落）

2026年Q2起，BOPET等石油化工原材料价格从高位回落（隆众资讯数据）。天地数码直接材料成本占比71.4%，PET薄膜是其中最大单项。

**毛利率改善空间**：
- 2025年综合毛利率32.0%，2026Q1进一步降至30.6%
- 若原材料价格持续下行，综合毛利率有望在2026H2修复至31-33%

**确定性评分：3分（中等）**
方向有利但受制于地缘局势和原油价格波动。

| 管理费用率 | 5.3% | 5.0% | -0.3ppt（规模效应） |

**EPS计算细节**：

- 加权平均流通股：约1.47亿股
- EPS = 1.27/1.47 ≈ **0.86元**（加权平均口径）
- 按期末总股本：EPS = 1.27/1.51 ≈ **0.84元**

## 三、多态逻辑情景推演

### 3.1 保守情景（Bear Case，概率25%）

**触发条件**：原油价格因地缘冲突再度大幅上涨→原材料成本反弹；维森二期量产进度延迟→新增产能未能有效贡献；人民币升值→出口收入压缩。

| 参数 | 保守值 | vs基准 |
|:---|:---|:---|
| 营收(亿) | 9.20 | -6.4% |
| 毛利率 | 29.5% | -2.0ppt |
| 费用率 | 17.5% | +0.2ppt |
| **净利润(亿)** | **0.95** | **-25.2%** |
| **EPS(元)** | **0.63** | -25.0% |
| **YoY净利润增速** | **-12.0%** | 利润下滑 |

**情景特征**：原材料成本飙升+产能爬坡受阻双重压力叠加，净利润出现近年首次下滑。若此情景兑现，当前PE(TTM)约20.9倍将被重估至~23-24倍（盈利下滑+估值收缩双杀）。

### 3.2 基准情景（Base Case，概率50%）

参数见上文P*Q模型，EPS = **0.84元**。

**情景特征**：维森二期平稳爬坡、产品结构温和改善、原材料价格保持当前水平。净利润同比增17.6%，增速保持。

### 3.3 乐观情景（Bull Case，概率25%）

**触发条件**：维森二期提前完成验收→从Q3即开始规模量产；BOPET价格持续走低→毛利率修复至34%+；海外市场超预期增长（东南亚制造业回流带动标签需求激增）。

| 参数 | 乐观值 | vs基准 |
|:---|:---|:---|
| 营收(亿) | 10.60 | +7.8% |
| 毛利率 | 34.0% | +2.5ppt |
| 费用率 | 16.5% | -0.8ppt |
| **净利润(亿)** | **1.60** | **+26.0%** |
| **EPS(元)** | **1.06** | +26.2% |
| **YoY净利润增速** | **+48.1%** | 重回高增长 |

**情景特征**：量价齐升+"成本下降+费用率优化"双击，重回50%左右的高增长轨道。

## 五、SOTP分部估值利润分配

天地数码的业务板块高度集中于热转印碳带（占营收89.5%），其他业务（占10.5%）主要是外采产品的贸易，不具备显著的板块差异特征。因此不适用严格的分部估值（SOTP），整体法估值更为合适。

但可以粗略拆分：

| 板块 | 收入(亿) | 毛利率 | K系数 | 费用分摊 | 模拟净利(亿) |
|:---|:---|:---|:---|:---|:---|
| 核心TTR（蜡基+混合基） | 7.50 | 28% | 1.0 | 17.5%×7.50×1.0 | 0.79 |
| 高端TTR（树脂基+TTO+彩色） | 1.30 | 42% | 1.3 | 17.5%×1.30×1.3 | 0.35 |
| 其他业务（贸易等） | 1.03 | 16% | 0.6 | 17.5%×1.03×0.6 | 0.09 |
| **合计/验证** | **9.83** | **31.5%** | - | - | **1.23** ≈ **1.27** ✓ |

勾稽校验通过（模拟值1.23 vs 预测值1.27，差异来自简化假设）。

## 七、模块五结论摘要

2026年天地数码面临"产能释放+成本改善"的有利组合。基准情景下，预计营收9.83亿（+11.1%）、净利润1.27亿（+17.6%）、EPS 0.84元。增长的核心驱动来自维森二期投产（确定性最高）和原材料成本边际改善（确定性中等）。三态情景加权EPS为0.84元，与基准情景一致。需要高度警惕的风险是：若原材料价格反弹叠加产能爬坡不及预期，净利润可能下滑至0.95亿（保守情景），届时将触发估值收缩的负反馈。
