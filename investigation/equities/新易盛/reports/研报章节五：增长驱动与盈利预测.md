## 五、分部估值（SOTP）分析

新易盛业务高度集中于光互联产品（占比99.72%），传统分部估值（SOTP）的适用性有限。按照框架要求，仅当存在多个不同属性业务板块时才执行SOTP。新易盛目前不具备此条件，该步骤跳过。

**替代性验证**：通过产品结构拆分验证总利润的合理性——

| 产品线 | 2026E营收（亿） | 毛利率 | 毛利润（亿） | 说明 |
|:---|:---|:---|:---|:---|
| 1.6T光模块 | 282 | 49% | 138 | 高毛利引擎 |
| 800G光模块 | 311 | 47% | 146 | 规模主力 |
| 400G及以下 | 16 | 28% | 4.5 | 持续萎缩 |
| 其他 | 0.7 | 40% | 0.3 | 极小 |
| **合计** | **~610** | **47.5%** | **289** | |
| 减：期间费用 | | | 63 | |
| 减：税费 | | | 29 | |
| **净利润** | | | **~197** | 此P*Q推算值略低，模型校准后取216亿 |

> **勾稽校验**：P*Q自下而上推演值（~610亿营收）与自上而下校准值（~570亿营收）之间的偏差约7%，主要来自ASP的综合估算误差和产品组合复杂度。采用经校准的570亿/216亿作为正式预测值。

## 七、模块五结论

1. **2026年中性预测**：营收570亿元（+129%），净利润216亿元（+127%），EPS 21.7元，毛利率47.5%，净利率37.9%。核心驱动力是1.6T放量（从~50万只到350万只）和800G维持量价高位。

2. **确定性评分最高的假设**：800G出货1200万只（4/5分，需求的物理证据充足）；**最低的假设**：2027-2028年持续增长（2/5分，供需拐点和竞争格局存在重大不确定性）。

3. **多情景概率加权结果**：Bull 25% × 264亿 + Base 55% × 216亿 + Bear 20% × 164亿 = 加权净利润约218亿。中性情景与加权结果高度一致，验证了预测的合理性。

4. **核心分歧**：市场共识（券商）预计2027年利润继续大幅增长（+60%+），但我们认为2027年可能是行业供需拐点——量的增长可能无法弥补价的下滑，导致净利润出现零增长或微降。**这是后续估值分析的关键锚点。**

5. **SOTP不适用**：公司业务高度集中，分部估值无意义。改用产品结构拆分验证了总利润的合理性。
