# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月30日  
**研究标的**：宏发股份 (600885.SH)  
**前序核心结论继承**：
- 模块一：毛利率被银铜成本侵蚀，Q1 2026毛利率32.09%为年内低点。海外业务毛利率40.20%显著高于国内32.74%
- 模块二：NEV增速换挡、光伏首次下滑、家电国补退坡
- 模块三：宏发全球份额23-24%，日美对手战略收缩，护城河宽但原材料敞口大
- **宏观变量更新（6月30日）**：银价进一步回落至$58-59/盎司（较5月26日$72.5再降19%，较1月$121峰值腰斩，YTD-17.6%）；铜ICSG确认2026年9.6万吨盈余、2027年扩大至37.7万吨（LME仍在$12,000-13,800高位震荡，但6月30日COMEX单周回撤5.86%）；NEV 5月总量+14.4%但国内零售-4.1%（YTD国内-16.5%），出口+110%持续爆发；中美Board of Trade启动、Section 122 7月24日到期（仅剩24天）——关税呈结构性缓和趋势
- 本章核心问题不变

## 一、宏观环境变量

### 1.1 利率与折现率环境

**中国货币政策：适度宽松**

| 指标 | 当前值（2026Q1） | 2026年展望 |
|:---|:---|:---|
| 央行政策利率（7天逆回购） | 1.40% | 预计再降20bp至1.20%（宏利投资预测） |
| 1年期LPR | 3.00% | 连续11个月不变，下行空间存在 |
| 5年期LPR | 3.50% | 维持低位，房贷利率仍有下行压力 |
| 10年期国债收益率 | ~1.70-1.90% | 低位波动，PBOC恢复国债购买 |
| 实际利率 | 1.50-2.00%（仍偏紧） | 通缩压力下需要进一步宽松 |

**对宏发的影响**：
- **贴现率下行支撑估值**：无风险利率持续走低，理论上支撑宏发PE维持在中位数以上（25-28倍），与当前27.4倍基本匹配
- **融资成本降低**：宏发自身有息负债极低（短期借款6亿+长期借款1.7亿），直接受益有限。但下游客户（车企、家电厂商）融资成本降低有利于资本开支和设备采购，间接利好继电器需求
- **理财收益率下行**：宏发账面金融资产（交易性金融资产3亿+）收益承压

### 1.2 汇率损益分析

**人民币汇率：双向波动，小幅升值**

2026年初以来人民币对美元升值约1.3%，对欧元升值3.7%，对日元升值3.2%。央行表态"保持人民币汇率基本稳定，防范汇率超调风险"。

**对宏发的双重影响**：

| 影响维度 | 方向 | 程度 | 分析 |
|:---|:---|:---|:---|
| 海外收入折算 | 负面 | 中等 | 海外收入49.18亿（占比30%），人民币升值1%约减少归母净利0.3-0.5% |
| 原材料进口 | 正面 | 部分对冲 | 铜、银以美元计价，人民币升值降低进口原料的人民币成本 |
| 海外竞争力 | 负面 | 有限 | 竞争对手松下/欧姆龙以日元计价，日元也升值，相对竞争力变化不大 |
| 汇兑损益 | 小幅 | 财务费用0.84亿，占总收入0.49%，影响可控 |

**综合判断**：汇率对宏发整体影响中性偏负面，但不构成核心变量。海外收入30%的占比提供了天然的对冲——汇率波动在收入端和成本端部分抵消。

### 1.3 通胀传导效率

中国2026年3月PPI同比+0.5%（三年多来首次转正），CPI同比+1.0%。全球通胀环境正在从"输入性通缩"向"温和再通胀"过渡。

**对宏发成本端的影响路径**：

```
全球再通胀 → 铜/银/工程塑料涨价 → 继电器原材料成本上升
           → 劳动力成本上升 → 人工成本上升
           → 能源/运输成本 → 制造费用上升
```

2025年宏发原材料成本暴涨33.44%已是通胀传导的实证。但**2026年6月出现重大边际变化**：

**银价趋势**：COMEX白银从2026年1月峰值$121/盎司持续回落，6月30日现货价约$58-59/盎司，较5月26日的$72.5再降约19%，YTD累计下跌17.6%。驱动因素包括：伊朗停火协议缓解能源通胀预期、Warsh表态政策利率跟随核心通胀（5月核心CPI 2.9%）、以及COMEX注册库存增加（截至6月15日约8,520万盎司）缓解了实物紧缺。J.P. Morgan仍预测2026年全年均价$81，但当前现货远低于该水平。若银价维持$58-65区间，宏发Q2-Q3原材料成本将显著低于Q1。

**铜价趋势**：ICSG 6月30日发布最新预测——2026年全球精炼铜盈余9.6万吨（此前预测短缺15万吨），2027年盈余扩大至37.7万吨。供给端：全球矿产增速+1.6%（2026）/+2.3%（2027），精炼产量增速+0.4%（2026）/+3.0%（2027）。需求端：全球精炼铜需求增速+1.6%（2026）/+2.0%（2027），中国需求增速仅+1.9%。LME铜价仍在$12,000-13,800/吨高位震荡，但COMEX 6月30日单周回撤5.86%，关税前置库存已达创纪录的65.4万吨。**铜的"供给盈余+库存高企"格局正在形成，中期价格下行压力加大**。

**对宏发毛利率的综合判断**：银价腰斩（$121→$59）是毛利率修复的最强催化剂。Q1毛利率32.09%对应的是银价$70-90区间采购的库存去化。Q2起低价银材逐步进入成本，叠加第二轮顺价落地，H2毛利率修复路径较5月预期**显著增强**。铜价虽有盈余信号但绝对水平仍高，部分对冲了银价下行的利好。综合看，2026年全年毛利率有望从基准预测的33.4%上移至34-35%区间。

**宏发的对冲能力**：
- 规模采购优势：年采购铜银量极大，议价能力强于中小对手
- 产品提价：部分传导（海外毛利率仅降0.36pp，说明海外提价能力较强）
- 效率提升：自动化率持续提高，制造费用率持平
- **边际改善**：银价腰斩是2026年内出现的最大成本端利好，成本传导时滞约3-6个月，Q2末起逐步体现在报表中

## 二、地缘政治风险

### 2.1 中美贸易摩擦：深度结构化的新常态

**关税演进路径（2025-2026年）**：

| 时间节点 | 事件 | 对华关税水平 |
|:---|:---|:---|
| 2025年1月 | Trump就职 | ~20% |
| 2025年2-3月 | IEEPA加征20% | ~40% |
| 2025年4月2日 | "解放日"全球关税 | 73%（峰值后164%） |
| 2025年5月 | 中美谈判降级 | 大幅下调 |
| 2025年10月 | Xi-Trump会晤 | 再降10pp |
| 2026年2月 | 最高法院裁决限制总统关税权 | 加权平均22.3%（对中国） |
| 2026年2月24日 | Section 122全球10%关税 | 维持22-36%水平 |
| 2026年7月后 | Section 122到期 | 不确定性高 |

**关键判断**：中美关税已从2025年的极端峰值（164%）回归至可管理水平（22-36%），且**呈现结构性缓和趋势**：

**2026年5月Trump-Xi北京峰会**（5月13-16日）达成"管理贸易"框架：
- 建立**美中贸易委员会（U.S.-China Board of Trade）**，管理非敏感产品双边贸易
- 目标：约$300亿双边贸易额享受关税减让（中方提议从$30亿扩大至$300亿）
- USTR已启动公众意见征询（6月2日-7月10日），征集可降低关税的非敏感中国商品清单
- 中国商务部6月25日表态"继续与美方就互惠关税削减进行磋商"

**Section 122关键时间节点**：
- 当前10%全球附加关税2026年2月24日生效，150天后——**2026年7月24日自动到期**
- 到期后，非232条款（钢铝汽车铜）且非中国商品可能降至MFN税率（降幅约10-15pp）
- 中国商品叠加Section 301关税（约12-26%）将继续维持
- 美国最高法院判决（Learning Resources案）及CIT判决已实质性削弱总统单边关税权
- 商务部必须在6月30日前提交铜关税建议报告（今日到期）

**综合判断**：关税环境正在经历从"极端峰值"向"管理缓和"的结构性转变。7月24日Section 122到期是近期最关键催化剂——若如期到期且未被新关税替代，宏发海外业务的关税成本有望降低约10个百分点。

### 2.2 供应链重构 (China+1) 的实质影响

**全球供应链转移趋势**：
- 美国自中国进口占比已从2018年的21.6%降至2025年的约13%
- 中国2025年对美出口下降20%，但总贸易顺差创新高（$1.2万亿）
- 中国出口转向：非洲+25.8%、拉美+7.4%、东南亚+13.4%、欧盟+8.4%

**对宏发的影响——机遇大于威胁**：

1. **宏发已提前布局全球化制造**：
   - 德国工厂2025年投产（高压直流继电器全自动产线）
   - 印尼宏发：2025年营收+64%，60亩新工业园2025年11月奠基
   - 越南工厂：2026年1月与VinFast合资签约（宏发持股80%）
   - **"中国+东南亚+欧洲"三极制造体系已初步成型**

2. **竞争对手替代逻辑不成立**：
   - 继电器认证周期2-4年（车规级），短期不可替代
   - 全球仅宏发、松下、泰科具有大批量稳定供货能力
   - 即使客户想"去中国化"，也找不到同等级别的替代供应商

3. **"China+1"反而为宏发提供了海外建厂的商业合理性**：
   - 欧洲客户：德国工厂就近供应，规避关税和物流成本
   - 东南亚客户：印尼/越南工厂辐射东盟市场
   - 美国客户：可通过东南亚基地间接进入（Section 301对东盟国家仍有关税优惠）

**唯一实质风险**：若中美关系极端恶化至"全面脱钩"（100%+关税维持），宏发海外收入的30%将直接受到冲击。但考虑到宏发已启动全球产能布局，这一风险正在被主动管理。

### 2.3 出口管制与稀土反制

中国对稀土、关键矿产的出口管制是应对美国关税的核心战略武器。中国控制全球约70%的稀土产量和85%的精炼能力。2026年2月中国宣布对7种稀土元素实施出口管制。

**对宏发的影响**：间接有限。继电器不直接使用稀土元素，但下游客户（电动汽车、风电、工业机器人）受稀土供应影响，可能间接减缓部分终端需求。但这种影响是第二阶的，优先级远低于关税和原材料价格。

## 三、监管与产业政策

### 3.1 "十五五"规划定位

2026年3月13日正式发布的《十五五规划纲要》与宏发相关度极高：

| 政策要点 | 原文 | 对宏发的含义 |
|:---|:---|:---|
| **元器件国产化** | "推进电子信息、机械装备等全产业链创新，发展高端、短缺产品，加快突破关键零部件、元器件和专用材料" | 宏发作为国产继电器绝对龙头，是政策直接受益者 |
| **新能源汽车** | "智能网联新能源汽车"列入战略性新兴产业 | NEV继续高速发展，高压直流继电器需求有政策保障 |
| **新型储能** | "扎实推进新型储能等关键技术创新" | 储能继电器是增量市场，政策背书 |
| **智能制造** | "发展智能制造、绿色制造" | 宏发获评"卓越级智能工厂"，符合政策导向 |
| **科技自立自强** | "全链条推动集成电路...关键核心技术攻关取得决定性突破" | 元器件自主可控上升到国家安全高度 |

### 3.2 电子信息制造业稳增长行动方案（2025-2026）

工信部、发改委、数据局联合发布的行动方案明确提出：
- 规模以上电子信息制造业增加值年均增速约7%
- 加上锂电池、光伏及元器件制造后，年均营收增速达到5%以上
- **电子元器件被明确纳入稳增长重点领域**

宏发作为中国电子元器件百强企业TOP4（2025年），充分受益。

### 3.3 新能源政策的持续性

| 政策领域 | 当前状态 | 对宏发影响 |
|:---|:---|:---|
| NEV购置税减免 | 2026年逐步退坡 | 短期负面（增速放缓），但行业趋势不可逆 |
| "以旧换新"补贴 | 2026年第二批625亿元已拨付 | 支撑家电/汽车更新需求 |
| 光伏管理办法 | 2026年新政落地，短期观望 | 光伏继电器需求短期承压 |
| 储能政策 | 强制配储政策延续 | 储能继电器需求高增（+62% YOY） |
| 智能电网 | 国网投资持续增长 | 电力继电器需求稳定 |

### 3.4 行业准入与环保约束

继电器行业不存在特殊的行政准入壁垒（非牌照管制行业），但事实上的认证壁垒极高：
- 汽车继电器需通过AEC-Q、IATF 16949
- 出口需UL（美国）、VDE（德国）、TÜV（欧洲）、CQC（中国）认证
- 国网/南网有独立的供应商准入体系

宏发同时拥有UL&CUL、VDE、TÜV、CQC等全部主流认证，且是"国内首家获国网2025版电流互感器资质的制造商"，在认证层面建立了事实上的护城河。

环保约束方面，欧盟RoHS和REACH法规对继电器的有害物质（如镉）使用有严格限制，AgCdO（银氧化镉）触点材料已被AgSnO₂（银氧化锡）替代。宏发已完成技术切换，环保合规成本已经消化。

## 四、宏观变量对估值的传导机制

### 4.1 折现率（无风险利率）路径

```
PBOC宽松 → 10Y国债低位（1.6-1.9%）→ 权益风险溢价稳定 → 
DCF分母下降 → 理论PE中枢抬升 → 支撑当前27x PE
```

当前中国10年期国债1.70-1.90%对应约5-5.5%的股权成本（加3.5%风险溢价），支持25-30x PE的中枢水平。宏发当前27.4x PE处于这一区间的中位。

### 4.2 汇率路径

人民币升值→海外收入折算减少→EPS小幅下调→PE估值保持不变（分子分母同向调整）→对估值倍数无实质影响

### 4.3 地缘政治风险溢价

最需要警惕的情景：**中美关系恶化 → 全面关税回归 → 海外收入断崖 → EPS骤降30% → PE杀跌至15-18倍**

这一情景在2025年4月曾短暂出现（股价最低触及约21.70元，PE约18.8倍），但随后随中美谈判缓和而修复。该风险在2026年7月（Section 122到期）和Trump-Xi再次会晤前后仍可能扰动股价。

## 五、政策契合度与战略价值评估

### 5.1 与国家战略的契合度矩阵

| 国家战略 | 宏发契合度 | 政策红利转化为业绩的确定性 |
|:---|:---|:---|
| 元器件国产替代 | ★★★★★ | 高——宏发已是龙头，主要受益形式是份额持续提升 |
| 新能源汽车 | ★★★★☆ | 高——高压直流继电器市占率40%+，与行业同步增长 |
| 智能制造/数字化转型 | ★★★★★ | 高——工信部"卓越级智能工厂"，运营效率持续改善 |
| 新型储能 | ★★★☆☆ | 中——储能继电器仍在渗透初期，体量尚小 |
| 碳中和/绿色制造 | ★★★☆☆ | 中——间接受益，非直接政策驱动 |

### 5.2 政策风险：反垄断与价格监管

宏发全球继电器份额23-24%，国内份额可能超过50%，接近反垄断关注线（《反垄断法》规定单一经营者市场份额达到50%可被推定为具有市场支配地位）。但继电器属于全球性竞争市场，欧姆龙、泰科、松下等外资对手的存在降低了单一市场支配地位的认定风险。
