# 研报章节二：产业周期与终端穿透

**研究日期**：2026年05月12日  
**公司名称**：惠州亿纬锂能股份有限公司（300014.SZ）  
**关联前序模块核心结论**：毛利率从21.57%→14.04%持续压缩、营收高速增长但净利停滞、ROIC降至6.80%，本章将从产业周期和供需角度解释这一"量增价跌"困境的根源。

---

## 一、产业生命周期与阶段性定位

亿纬锂能横跨三大电池赛道，各自处于不同的生命周期阶段，需分层审视：

### 1.1 动力电池：成长后期→成熟早期过渡

| 指标 | 数据 | 来源 |
|:---|:---|:---|
| 2025全球新能源汽车销量 | 2,260万辆（+23.7%YoY） | GGII |
| 2025全球电动化渗透率 | 23.7% | GGII |
| 2026全球NEV销量预测 | ~2,650万辆 | GGII |
| 2025全球动力电池出货量 | 1,495.1GWh（+42.2%YoY） | EVTank |
| 2025全球乘用车动力电池装机量 | 1,039.6GWh（+24.9%YoY） | SNE Research |
| 2029出货量预测 | 3,548.3GWh | 弗若斯特沙利文 |
| 2025-2029 CAGR | 26.7% | 弗若斯特沙利文 |

**生命周期判断**：动力电池处于**成长后期**。核心依据：
1. 渗透率23.7%仍处于S曲线加速段（通常20%-50%为最快渗透期），但增速已从52.8% CAGR（2020-2024）降至26.7%（2025-2029预测）。
2. EV-Volumes明确指出"全球EV增长正在放缓"，2025年21.6M→2026年22.7M仅+5%，受制于补贴退坡和贸易摩擦。
3. **中国领先、欧美追赶、新兴市场起步**的三层结构，决定了需求仍有结构性增量，但整体增速将边际递减。

### 1.2 储能电池：高速成长期

| 指标 | 数据 | 来源 |
|:---|:---|:---|
| 2020-2024全球储能出货量CAGR | 87.4%（23.7→292.2GWh） | 弗若斯特沙利文 |
| 2025全球储能电池出货量增速 | +80%（YoY） | GGII |
| 2025中国储能企业海外订单 | 366GWh（+144%YoY） | CNESA DataLink |
| 2025-2029集中式储能CAGR | 23.1% | 弗若斯特沙利文 |
| 2025-2029分布式储能CAGR | 25.5% | 弗若斯特沙利文 |

**生命周期判断**：储能电池处于**高速成长期**，渗透率仍低（全球储能对可再生能源的渗透率远低于10%），政策驱动+经济性改善双重共振。但需警惕：行业增速已从87%→80%→23-25%的急剧放缓，意味着"野蛮生长"期正在结束，进入竞争淘汰阶段。

### 1.3 消费电池：成熟期中的结构性机会

消费电池整体处于**成熟期**（CAGR约15-16%），但其中锂原电池为亿纬锂能的现金牛（连续9年国内第一），新兴应用（机器人、BBU/AIDC备电、低空经济、医疗植入式电池）提供了结构性高增长点。具体而言：
- **人形机器人**：2025年全球出货量约1.8万台（+508%YoY），IDC数据
- **低空经济**：2026年中国市场规模预计10,645亿元（+24%YoY），赛迪智库
- **BBU电池**：2030年全球需求预计超6亿只，EVTank数据

## 二、库存周期与供应链动力学

### 2.1 当前库存阶段判断

| 信号维度 | 观察 | 判断 |
|:---|:---|:---|
| 亿纬锂能存货周转天数 | 从78天（2023）→50天（2025） | **主动去库完成，库存回归健康** |
| 储能出货量增速 | 80%YoY（2025），满产满销 | **被动去库/主动补库早期** |
| 动力电池价格趋势 | 持续走低 | **竞争性降价而非库存出清** |
| 上游碳酸锂价格 | 从60万/吨→8万/吨企稳 | **成本端已去泡沫，价格传导压力转至制造端** |
| 终端车企库存 | 大部分正常化 | **牛鞭效应风险较低** |

**综合判断**：经历2022-2023年的剧烈去库后，产业链库存已回归健康水平。当前产品价格的持续下行并非库存出清驱动，而是**产能过剩竞争**的结果。这一点至关重要——传统的"去库→补库"周期无法解决当前行业的结构性价格压力。

### 2.2 牛鞭效应辨析

亿纬锂能的主要客户为整车厂和大型储能集成商（Wärtsilä、ABB、台达电子等），属于**B2B直供模式**，渠道层级少，牛鞭效应的放大风险相对有限。真正的需求信号失真风险在于：
- **车企竞争性备货**：为争夺市场份额，车企可能下过剩订单
- **储能项目"抢装潮"**：政策窗口期前的集中采购可能导致需求前置

## 三、终端需求穿透与跨期断层审计 (MANDATORY)

### 3.1 最终产品画像

| 亿纬锂能产品 | 最终终端形态 | 需求驱动因子 |
|:---|:---|:---|
| 动力电池（方形LFP/NCM） | 新能源乘用车/商用车/工程机械 | 电动化渗透率、排放法规、TCO经济性 |
| 储能电池（大方形LFP） | 电网侧储能电站/工商业储能/户储 | 可再生能源装机、电力市场化、电网调节需求 |
| 消费电池（锂原/小锂电/圆柱） | 智能表计/汽车电子/电动工具/TWS/机器人/无人机 | 智能化渗透、消费电子周期、新兴应用 |

### 3.2 终端景气度审计（未来1-3年）

#### A. 新能源汽车（1-3年展望）: ★★★☆☆（谨慎乐观）

**正向因素**：
- 全球电动化渗透率23.7%→27%（2026E），仍有提升空间
- 商用车电动化加速（亿纬锂能市占率全国第二的核心战场）
- 中国继续引领全球，内需基本盘稳固
- 电池成本下降提升电动车TCO经济性

**负向因素**：
- 全球EV增速放缓（EV-Volumes预测2026年仅+5%至22.7M）
- 美国政策逆转：特朗普政府放松排放法规，北美市场不确定性上升
- 欧洲增速受经济疲软拖累
- 中国EV渗透率已超50%（含PHEV），纯增量空间在收窄

**核心矛盾**：行业出货量增速（~25%）远超终端车辆销售增速（~5%），这中间的差额意味着**单车带电量提升、海外市场拓展和库存补库**是主要驱动力，而非单纯的终端销量增长。

#### B. 全球储能（1-3年展望）: ★★★★★（高度景气）

**正向因素**：
- 国家级行动方案开启未来十年储能新增量（中国储能网2026年5月报道）
- 中国"双碳"战略+欧盟"绿色协议"双引擎驱动
- 电力市场化改革（容量电价、现货市场）使储能盈利模式闭环
- AI数据中心（AIDC）驱动备电需求爆发
- 新兴市场（中东、南美）GWh级项目招标活跃
- 2025年中国储能企业海外订单366GWh（+144%），出海势头强劲

**风险因素**：
- 系统价格持续走低（储能项目EPC中标价已至0.42元/Wh以下）
- 多地取消固定分时电价，储能"躺赚"模式终结
- 欧洲负电价频发，可能影响储能项目IRR
- 钠离子电池等新技术路线对LFP形成替代威胁

**核心判断**：储能是亿纬锂能三大板块中**确定性最高的增长赛道**，但利润端的兑现取决于产能规模优势向成本优势的转化效率。

#### C. 消费电池新兴应用（1-3年展望）: ★★★★☆（结构性强劲）

人形机器人2025年出货量同比增长508%，低空经济2026年市场规模超万亿，BBU电池2030年需求超6亿只——三大新兴赛道为消费电池业务提供了远超传统消费电子增速的新增量。亿纬锂能在机器人（已与头部客户建立合作）、医疗植入式电池（国内唯一实现能力）、无人机（首家通过AS9100D航天认证）的先发布局，有望在消费电池板块开辟第二增长曲线。

### 3.3 动能溯源与跨期透支审计

#### 增长动能拆解：

| 增长来源 | 性质 | 持续性评估 |
|:---|:---|:---|
| 全球电动化渗透率提升 | **内生**（技术迭代+成本下降） | 可持续，但增速边际递减 |
| 中国"双碳"政策 | **外生**（政策驱动） | 2030年前方向确定，但力度可能边际调整 |
| 欧盟碳排放法规 | **外生**（监管驱动） | 长期确定，2035年禁燃目标 |
| 储能电价市场化 | **外生/内生**（政策+经济性） | 盈利模式正从政策依赖转向市场驱动 |
| 美国补贴（IRA） | **外生**（短期刺激） | 高度不确定，政策风险大 |
| AI数据中心备电 | **内生**（技术需求） | 强持续性，刚需属性 |
| 机器人/低空经济 | **内生**（新兴技术） | 早期，爆发力强但基数低 |

#### 跨期需求透支审计（强制项）：

**（1）新能源汽车**：存在**局部透支风险**。
- 中国2025年新能源商用车渗透率已达26.9%（+63.7%YoY），部分由政策考核驱动
- 全球数据：EV-Volumes显示PHEV增速从55.2%骤降至11.1%，表明补贴退坡后真实需求显露
- **结论**：短期内（2026-2027）不会出现整体性需求断层，但增速放缓已是确定趋势。亿纬锂能若能维持商用车和大圆柱技术领先，可在增速放缓中通过份额提升对冲

**（2）储能**：存在**"政策抢装"型透支**。
- 2025年储能出货量同比+80%，部分由各地容量电价政策窗口期前的"抢装潮"驱动
- 多地取消固定分时电价后，部分项目的经济性将重新评估
- 海外订单366GWh（+144%）的增长中，部分为长期框架协议而非常态化交付
- **结论**：2026年储能增速将从80%显著回落至合理水平（预计30-50%），但由于行业渗透率仍极低，不构成需求断层

**（3）消费电池**：**基本无透支风险**。消费电子需求随换机周期自然波动，新兴应用（机器人、低空经济）仍处于早期供给创造需求阶段。

## 四、供需边际与压力传导审计

### 4.1 供需平衡预判

#### 动力电池：从紧平衡滑向结构性过剩

| 维度 | 判断 |
|:---|:---|
| **需求端** | 2025-2029 CAGR 26.7%，绝对增量依然可观 |
| **供给端** | 全球主要电池企业2025年末规划产能远超需求预测；宁德时代、比亚迪、中创新航、亿纬锂能等头部企业均在大幅扩产 |
| **产能利用率** | 行业整体产能利用率从2022年的~80%降至2025年的~50-60% |
| **结构性分化** | 高端/大圆柱/商用车电池供给不足；低端LFP严重过剩 |
| **预判** | **中低端产能过剩持续恶化，高端差异化产能仍享受溢价** |

**对亿纬锂能的影响**：公司大圆柱电池在行业中率先量产装车（超9万台），商用车电池市占率全国第二，在结构性过剩中处于**相对有利位置**。但储能LFP电池面临最激烈的价格竞争，毛利率仅12.28%且同比下降2.44pp即是明证。

#### 储能电池：高增长对冲出清中，但价格战白热化

储能领域的需求增长（2025年+80%）暂时消化了新增产能，但供给端扩张速度同样惊人。2025年全球储能电池出货量约526GWh，而中国主要企业规划产能在2026-2027年将超1,000GWh。**产能过剩的拐点可能在2027年前后来临**。

### 4.2 压力传导特征分析

```
终端市场波动 → 电池价格 → 电池企业利润
    ↓              ↓              ↓
  EV销量增速   ASP持续下降   毛利率持续压缩
  放缓至+5%   （储能系统低    （16.17%→14.04%）
              至0.42元/Wh）
```

#### 波动放大效应：
- **下游整车降价→电池降价**：整车厂价格战直接向上游传导，且存在**放大效应**——车企压价10%，电池企业可能面临15-20%的毛利率压缩（因为制造成本中固定成本占比高）
- **储能招标价格走低**：2026年5月储能系统EPC中标价已至0.42元/Wh以下（中国储能网数据），较2023年下降超50%。亿纬锂能储能电池毛利率从2024年的14.72%降至2025年的12.28%，印证了这一趋势

#### 传导时滞与弹性：
- **压力传导约滞后1-2个季度**：电池企业与整车厂的年度价格谈判通常发生在Q4或Q1，价格锁定后执行一个周期
- **亿纬锂能的博弈筹码**：
  - **优势**：大圆柱电池产能稀缺（率先量产），具备一定议价权；商用车市场份额领先（全国第二），客户粘性较强；全产业链布局（镍钴锂→电池→回收）提供成本缓冲
  - **劣势**：储能电池面临海辰储能、比亚迪等强劲对手的低价竞争，差异化程度有限（628Ah大电芯虽有技术优势，但终端客户更看重价格）

## 五、强制复核项确认

### ✅ 复核项1：终端产品景气度预测
已穿透并引用了：
- 新能源汽车：2025年2,260万辆 → 2026年2,650万辆（GGII），2025年21.6M→2026年22.7M（EV-Volumes）
- 储能：2025年+80%（GGII），2029年CAGR 23-26%（弗若斯特沙利文）
- 低空经济：2026年10,645亿元（赛迪智库）
- 机器人：2025年1.8万台（+508%）（IDC）

### ✅ 复核项2：跨期需求移位审计
已识别以下透支风险：
- 储能"抢装潮"可能导致2025年数据提前透支2026年部分需求
- 新能源商用车政策驱动下存在局部透支
- 整体判断：短期无重大断层风险，但增速放缓确定

### ✅ 复核项3：供需边际与压力传导
已分析：
- 动力电池：中低端产能过剩持续，高端差异化仍有溢价
- 储能电池：高增长对冲中，但拐点可能在2027年前后
- 压力传导放大效应显著（整车降价10%→电池利润压缩15-20%）
- 亿纬锂能在大圆柱和商用车领域具备局部议价优势
