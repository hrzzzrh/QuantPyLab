## 三、逻辑证伪警报 (Invalidation Triggers)

以下信号出现时，必须承认**投资逻辑已出现实质性裂痕**，应立即触发止损或下调评级：

| 级别 | 证伪信号 | 触发动作 | 逻辑理由 |
| :---: | :--- | :--- | :--- |
| **🔴 一级警报（立即止损）** | 公司被列入美国实体清单，FPGA芯片无法采购 | **无条件清仓** | 模块四核心风险兑现，商业模式面临根本性动摇 |
| **🔴 一级警报（止损或大幅减仓）** | 连续两个季度营收增速<+10%且毛利率<73% | **减仓50-70%** | 量价双杀，意味着竞争格局或行业需求发生重大不利变化 |
| **🟡 二级警报（减仓30-50%）** | 年报披露中低功率市占率首次跌破50% | **减仓并重新评估** | 核心基本盘被动摇，公司议价能力将下降 |
| **🟡 二级警报（减仓30-50%）** | 大客户（大族/宏石等）宣布自研控制系统并切换 | **大规模减仓** | 如果头部设备商走向垂直整合，行业格局将根本性改变 |
| **🟠 三级警报（下调评级至观望）** | 年度高功率份额增长<2pct | **暂停加仓，观望** | 替代速度放缓意味着增长逻辑中最核心的驱动力减弱 |
| **🟠 三级警报（下调盈利预测）** | 智能焊接年出货<1500套 | **下调中长期盈利预测20%** | 最重要的第二增长曲线可能不及预期 |
| **🟠 三级警报（下调盈利预测）** | 连续两季应收账款周转天数>45天 | **关注现金流恶化风险** | 设备端压力可能正在向控制系统传导 |

### 5.2 Q1 评估结论

| 评估项 | 判断 | 说明 |
| :--- | :---: | :--- |
| 营收增速骤降 | ⚠️ **需密切关注** | Q1 +10.53% vs 预警线+15%。但Q1 2025基数极高（+30.12%），且Q4 2025年末集中交付可能透支Q1需求。暂不触发警报，但若Q2继续低于+15%，则需下调盈利预测。 |
| 毛利率继续下滑 | ⚠️ **趋势延续** | 74.51%较2025全年77.66%再降3.15pct，主要系智能硬件占比持续提升。但Q1通常为毛利率最低季度（季节性因素），后续季度有望修复。 |
| 费用控制超预期 | ✅ **积极信号** | 三费比率从10.03%骤降至6.27%，研发/销售费用增速显著低于营收增速，规模效应加速释放。这是净利润增速（+23.88%）显著高于营收增速（+10.53%）的核心原因。 |
| 分析师预期对比 | ✅ **符合预期** | 分析师一致预期2026年净利13.69亿，Q1完成3.16亿（占全年23.1%），与季节性规律一致，全年实现概率较高。 |
| 整体评估 | **中性偏谨慎** | 营收减速是真实性信号，但利润端韧性超预期。Q2（8月半年报）将是全年展望的最关键验证节点。 |
