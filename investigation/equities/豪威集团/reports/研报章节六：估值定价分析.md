# 研报章节六：估值定价分析

**研究日期**：2026年06月07日
**当前不复权市价**：95.39元（2026年6月5日收盘价）

## 一、价值锚定：适配性方法选择

### 1.1 方法论选择与理由

豪威集团是Fabless半导体设计公司，兼具周期成长属性。选择以下三种方法进行交叉验证：


| 方法                         | 适用理由                                     | 权重 |
| ---------------------------- | -------------------------------------------- | ---- |
| **PE估值法**（主导）         | 盈利已进入稳态，PE是科技成长股最主流的定价锚 | 50%  |
| **PEG估值法**                | 公司处于成长阶段，需要纳入增速考量           | 25%  |
| **历史PE分位法**（参考校验） | 提供长周期坐标                               | 25%  |

不使用DCF的理由：半导体行业周期性显著，远期自由现金流预测不确定性过高。不使用PB的理由：Fabless公司轻资产属性，PB参考意义有限。

### 1.2 基准PE与市值

**当前估值锚点**：2026年EPS（加权）= 3.56元/股

**可比公司PE参照**：


| 公司         | 2025年净利（亿） | 2026E增速 | 当前PE(TTM) | 特征对比                         |
| ------------ | ---------------- | --------- | ----------- | -------------------------------- |
| 索尼         | ~5000亿日元      | ~5%       | ~17x        | 技术龙头，低增速，给予"垄断溢价" |
| 思特威       | ~9亿(2025E)      | ~43%      | ~45x        | 高成长，A股溢价                  |
| 格科微       | ~5亿(2025E)      | ~25%      | ~50x+       | 转型期，高估值不稳定             |
| **豪威集团** | **40.45亿**      | **~11%**  | **32.7x**   | **平台型龙头，增速适中**         |

**基准PE判断**：

- 豪威的盈利规模（40亿+）远大于思特威和格科微，理应享受部分规模&龙头溢价
- 但增速（11%）低于思特威，高于索尼
- 合理PE区间：**30-38x**
- **基准PE = 34x** → 基准目标市值 = 45亿 × 34 = **1,530亿**
- 基准目标价（不复权）= 1,530亿 / 12.61亿股 = **121.3元**

**修正目标市值 = 1,530亿 × (1+0%) = 1,530亿**
**修正目标价 = 121.3元/股（不复权市价）**

### 交叉验证1：PEG估值

加权EPS 3.56元 × 增速11.3% → PEG = 当前PE 32.7 / 11.3 = 2.89。若给予PEG=3.0（科技平台型龙头合理水平），对应PE=3.0 × 11.3 = 33.9 → 目标价 = 3.56 × 33.9 = **120.7元**。与PE法高度一致。

### 交叉验证2：历史PE分位


| 分位          | PE值      | 说明                                        |
| ------------- | --------- | ------------------------------------------- |
| P10           | 32.5x     | 仅高于历史10%的交易日的估值水平             |
| P25           | 42.1x     | —                                          |
| P50（中位数） | 70.2x     | 历史中位数，但受2021-2023年极致高估期的影响 |
| 当前          | **32.7x** | **接近P10（历史极端低位）**                 |

**审慎判断**：A股CIS公司（豪威/思特威/格科微）的PE历史中枢可能因以下原因不可简单套用：①2021年半导体超级周期将历史PE虚高至异常水平；②当前注册制改革+经济增速中枢下行，A股整体估值中枢在下移；③投资者结构从散户向机构化转变，估值泡沫空间收窄。但即便是考虑这些因素，**32.7x PE TTM在历史10%分位以下，属于"极端低估"区间**。给予中性PE=35-40x → 目标价126-144元。

## 四、极限压力测试（黑天鹅）

**定义事件**：中美全面科技脱钩，中国CIS被列入实体清单，无法获取台积电等主要代工厂的晶圆供应。

**情景推演**：

- 设计业务产能骤降至30%（仅靠国内代工厂如中芯国际、晶合等接续，但良率/产能/先进节点不匹配）
- 汽车CIS收入下降70%（海外客户全部流失，仅剩国内部分）
- 2026年归母净利降至约5亿（参考2023年行业低谷水平）
- 当年PE炒作至60x+（供给短缺预期），市值约300亿
- 但此后若代工彻底断绝，长期归零风险

**极限支撑位**（基于资产价值）：

- 归母净资产：330.81亿 ÷ 12.61亿股 = **26.2元/股**（净现金价值锚）
  - 货币资金：163.87亿 → 每股净现金约**13.0元**
- **地狱价格 = 净现金价值 = 13.0元/股**（极端清算情景，概率<1%）
