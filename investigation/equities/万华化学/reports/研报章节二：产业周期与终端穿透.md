# 研报章节二：产业周期与终端穿透

**研究日期：2026年05月09日**

## 一、产业生命周期与库存节奏

### 1.1 三大主业所处的产业时空坐标

万华化学横跨三大产业，各自处于截然不同的生命周期位置，需分开定位：

| 业务板块 | 产业阶段 | 核心特征 | 周期位置 |
| :--- | :--- | :--- | :--- |
| 聚氨酯 (MDI/TDI) | 成熟期（有结构性增量） | 高集中度寡头格局（CR5≈91%），量增价稳 | 被动去库向主动补库过渡 |
| 石化（乙烯/丙烯/PO等） | 衰退期/底部出清 | 严重供过于求，盈利低迷，产能出清进行中 | 下行周期的底部区域 |
| 磷酸铁锂（电池材料） | 成长期（高速扩张） | 需求爆发（+60% YoY），低端过剩但高端紧缺 | 主动补库阶段 |

#### 聚氨酯（MDI）：成熟期的"现金牛"出现边际修复

全球MDI总产能约1,100万吨，2025年全球需求约850万吨，产能利用率约77%，属于结构性偏松。但需要穿透表层数据——全球仅万华、巴斯夫、科思创、亨斯迈、陶氏五大企业掌握约91%产能，属典型的寡头垄断格局。在寡头格局下，产能利用率并非衡量供需平衡的最佳指标，供给协同能力才是核心。

2025年11月至12月，全球MDI巨头罕见地实现协同提价，万华、巴斯夫、亨斯迈、陶氏在三周内相继发布调价通知，涨幅200-350美元/吨。此轮提价源于三重供给冲击：（1）万华宁波MDI二期（100万吨/年）55天检修；（2）亨斯迈荷兰28万吨/年装置意外停车；（3）欧洲能源成本攀升。

更重要的结构性变化发生在2026年3月以来的地缘政治事件。中东地区冲突导致陶氏中东40万吨MDI+20万吨TDI装置宣布不可抗力（预计停产2-3个月），韩国锦湖61万吨装置因原料短缺可能降负延长。受此影响，国内聚合MDI价格从14,500元/吨飙升至20,500-21,000元/吨，出口价从1,800美元/吨涨至3,000美元/吨。预计4-6月供需格局偏紧，全球贸易流向向中国倾斜——中东需求的供给缺口将由万华填补。

**当前周期判断**：MDI已从此前2年的被动去库（需求疲软、价格承压）进入被动去库向主动补库过渡阶段。催化剂是地缘政治驱动的海外供给收缩叠加中国出口替代。需要警惕的是，若地缘冲突缓和（海外供给恢复），涨价逻辑将迅速瓦解。

#### 石化：底部出清中的"负资产"

乙烯、丙烯等基础化工品正处于史无前例的产能过剩周期。2025年中国乙烯新增产能近900万吨（为2020-2024年年均水平的近2倍），丙烯新增产能911万吨。2026年仍将有乙烯超400万吨、丙烯约471万吨新增产能投放。

供需剪刀差持续扩大：乙烯消费量增速不足4%（2026年预计增至7,100万吨），丙烯当量消费增速回落至4.1%（消费量约6,570万吨）。丙烯过剩规模较上年增长179%，全行业亏损，部分独立PDH企业深陷"生产即亏损、停产则淘汰"的困局。

万华的石化板块面临类似挑战——2025年营收842.04亿元但毛利率仅0.58%，扣除期间费用后全面亏损。唯一亮点是万华乙烯一期装置已完成原料多元化技改（低成本乙烷原料替代高价石脑油），2026Q1优势初显。万华蓬莱POCHP装置的成功开车也带来一定的差异化竞争力。

**周期判断**：石化行业处于本轮下行周期的底部区域，预计2025-2027年为盈利低谷期，此后随着老旧产能退出、新增产能放缓，供需将缓慢再平衡。关键在于万华能否利用一体化优势和低成本路线熬过这轮出清。

#### 磷酸铁锂：成长期的"混乱繁荣"

磷酸铁锂产业处于爆发式增长阶段，但增长质量高度分化。2025年中国LFP出货约375-387万吨，同比增长58-60%，全球动力电池产量1,421GWh（+36%），储能出货量前三季度同比增长98.5%。但行业面临"量增利减"矛盾——低端产品产能严重过剩（总产能688万吨vs产量375万吨，产能利用率仅54%），而四代高压实产品实际供应仅100-120万吨，远低于150万吨需求，供需偏紧。

万华定位明确——通过"资源-中间体-成品"一体化碾压式切入，规划总产能超125万吨（包括莱州65万吨、海阳50万吨、四川等），利用氯碱副产氢气还原铁源，自主生产磷酸铁核心中间体，构建显著成本优势。2026年底有望形成80-90万吨有效产能。

**周期判断**：产业处于快速成长与产能出清并存的混乱期，未来1-2年将加速洗牌。万华作为后发但具备全产业链优势的竞争者，有望在高端市场建立地位。

### 1.2 库存动力学：各板块行为分化

- **MDI**：经历2023-2024年的库存消化后，2025年末进入补库阶段。万华自身2025年聚氨酯库存量49万吨，同比减少13.48%，处于正常偏低水平。

- **石化**：产业链下游聚丙烯、聚乙烯库存高位，独立生产商库存压力大（2025年9月全国丙烯厂家库存同比+26.55%），行业处于被动累库阶段。

- **精细化学品**：产销量高速增长，产销基本平衡，库存小幅下降5.27%。

- **电池材料**：行业整体库存水平受下游储能季节性影响较大，Q3通常是全年补库旺季。

## 二、终端需求穿透（MANDATORY）

### 2.1 最终产品画像

万华的主要产品最终负载于以下终端形态：

| 公司产品 | 直接下游 | 终端产品 |
| :--- | :--- | :--- |
| 聚合MDI | 硬质聚氨酯泡沫 | **建筑保温材料**（占MDI用量的36%） |
| 纯MDI | TPU/合成革/氨纶 | **鞋服、汽车内饰** |
| 聚醚多元醇 | 软质聚氨酯泡沫 | **家具/床垫（27%）、汽车座椅（17%）** |
| TDI | 软质聚氨酯泡沫 | **汽车、家具** |
| 聚烯烃/PC/PMMA | 改性塑料 | **家电外壳（18%）、汽车零部件、电子** |
| POE | 光伏胶膜 | **光伏组件封装膜** |
| 磷酸铁锂 | 锂电池正极 | **新能源汽车（动力）、储能系统** |

### 2.2 终端景气度审计（未来1-3年）

#### （一）建筑保温材料：强力政策驱动，核心增量源

中国建筑领域聚氨酯硬泡渗透率持续提升——从2024年的34%升至2025年的38%，华北、华东需求增速分别达23%和18%。全球层面，各国推进建筑节能改造以应对气候变化，三井化学预计全球MDI需求将以每年约5%的速度增长，核心驱动力即来自建筑保温。

**增长动能溯源**：
- **内生力量**：中国城镇化持续推进，城市人口占比已达58%（世界银行2024年数据），绿色建筑标准趋严（LEED/BREEAM认证驱动），世界绿色建筑市场预计到2030年将创造1.8万亿美元
- **外生干预**：中国"双碳"政策强制提升建筑能效标准，老旧小区节能改造持续推进

**需求前置/透支审计**：
- 中国建筑保温是典型的政策驱动需求，但不同于补贴驱动（如光伏抢装），建筑保温需求具有持续性和不可逆转性——墙体一旦建成，保温层即永久性安装
- 短期存在地产下行周期对新建住宅的量价冲击，但存量建筑节能改造（旧改）正在补位
- **结论**：不存在显著的跨期需求前置风险，建筑保温需求具有韧性

#### （二）汽车：新能源汽车的结构性增量

2025年中国汽车产销3,453万辆/3,440万辆，同比+10.4%/+9.4%，新能源汽车产销1,662.6万辆/1,649万辆，同比+29%/+28.2%，渗透率逼近50%。全球汽车产量中超5,000万辆使用MDI基PU泡沫。

汽车轻量化趋势利好聚氨酯的应用扩展——约61%的现代车辆含有PU部件（座椅、仪表板、隔音材料、电池封装等）。新能源车对NVH（噪声、振动）要求更高，PU隔音材料需求更强。

**动能溯源**：主要来自内生力量（汽车电动化、轻量化趋势），但也受到以旧换新补贴等政策催化。2026年汽车销量增速预计回落至2.4%，政策刺激效果边际递减。

**需求前置审计**：2025年汽车销量高增长部分归因于以旧换新补贴政策的刺激效应。若政策退坡，2026-2027年可能存在需求小幅回落的跨期风险。但对MDI的影响有限——单车用PU量约20-30kg，在MDI总需求中占比约18%。

#### （三）储能系统：最强劲的增长极

2025年前三季度中国储能锂电池出货430GWh，全年预计580GWh（+75%），全球电化学储能市场前三季度出货同比+98.5%。中金预计2026年磷酸铁锂正极材料产量将继续50%以上增速。

储能是全球能源转型的"硬需求"，受各国可再生能源渗透率提升和电网调峰需求驱动。中国强制新能源配储政策、美国IRA法案、欧盟可再生能源指令等构成多层政策支撑。

**动能溯源**：储能需求兼具内生（电力系统灵活性需求）和外生（配储政策强制要求）双重驱动。核心在于技术降本——LFP电芯成本持续下降正在开启商业化正循环。

**需求前置审计**：存在一定程度的"抢装"风险（风光项目强制配储导致需求集中释放），但储能系统具有持续运营需求且电池寿命有限（6-10年），初期安装将为后续替换需求提供基础。总体前置风险可控。

#### （四）家电：存量博弈中的结构性亮点

约59%的冰箱和47%的家电制造商使用PU泡沫。2025年中国家电出口表现亮眼，但国内房地产市场低迷对大家电新品需求形成拖累。预计2026年家电行业增速将回落。

**动能溯源**：出口驱动为主，国内更新换代需求为辅。存在贸易摩擦导致的出口需求不确定性。
**需求前置审计**：家电是以旧换新补贴的重点领域，存在一定政策透支风险，但影响范围有限。

## 三、供需边际与压力传导

### 3.1 供需平衡审计：边际在向何处演变？

**MDI/PU**：
- 全球产能1,100万吨，需求850万吨——表象供需偏松
- 但考虑到①CR5=91%寡头协同控量能力；②欧洲高能源成本已实质性地压制了约100-150万吨产能的有效供给弹性；③地缘政治导致的区域性供需失衡
- **预判**：2026年MDI供需格局大概率较2025年趋紧，尤其是Q2-Q3期间。海外有效产能收缩+中国出口替代=万华的"供给红利"窗口期
- 风险：若地缘冲突结束，海外供给恢复，2026下半年供需可能回归宽松

**石化**：
- 乙烯、丙烯、苯乙烯等均处于严重供过于求，2026年新增产能仍将持续释放
- **预判**：供需边际继续恶化，产能出清尚未真正开始，底部将持续2-3年
- 万华的相对优势在于一体化（自用丙烯生产PO/POE/PC等高附加值产品）和低成本原料（乙烯裂解多元化改造）

**LFP**：
- 低端严重过剩，但四代及以上高端产品"供不应求"（供需缺口约30-50万吨/年）
- **预判**：2026年高端产品持续紧缺（储能大电芯对能量密度要求升级），万华若能成功量产四代产品，将享有议价权

### 3.2 压力传导特征分析

**（1）波动放大效应**

石化产业存在典型的"牛鞭效应"——下游微小的需求波动在库存层面被放大数倍。以丙烯为例，独立PDH企业库存2025年9月同比增长26.55%，下游聚丙烯开工率长期低迷，形成了"需求弱→库存积累→降价促销→亏损加剧→被迫减产→但仍无法出清"的负循环。

相比之下，MDI因其寡头结构和万华的领先地位，避免了库存的剧烈波动（万华2025年聚氨酯库存降13.48%），压力传导较为温和。

**（2）传导时滞与博弈筹码**

- **MDI对原材料（纯苯）传导**：上游苯价格波动对MDI毛利率的影响存在约1-3个月的滞后期，万华通过"浮动价+合同价"组合采购模式部分平滑波动。2025年MDI毛利率逆势提升0.73个百分点，表明万华具备较强的成本转嫁能力。

- **石化对原油价格的传导**：呈即时传导模式（石脑油裂解路线直接受原油影响）。万华2026Q1通过原料多元化（乙烷替代石脑油）改善了成本结构，这是突破常规传导路径的成功尝试。

- **LFP对碳酸锂价格的传导**：碳酸锂占LFP成本约60-70%。2025年LFP行业正从"原料+加工费"捆绑定价向分离结算转型，这将降低碳酸锂价格波动对加工企业利润的冲击。万华由于自主生产磷酸铁，成本端受外部因素扰动更小。

## 四、强制复核确认

基于以上分析，以下三个强制复核项已逐一落实：

### ✅ 复核项1：终端产品景气度预测

已穿透并引用了万华核心终端的未来两年景气度预测：
- **建筑保温**：全球聚氨酯保温材料市场预计CAGR 4.2%（2026-2035），MDI基PU市场从2,121亿美元增至2,739亿美元（2030年）
- **新能源汽车**：2025年渗透率近50%，2026年仍将增长但增速回落，2027-2028年继续渗透
- **储能系统**：2026年全球LFP产量预计560-580万吨（+55%），储能出货量增速超75%

### ✅ 复核项2：跨期需求置换审计

已对各终端进行了需求前置审计：
- 建筑保温：**不存在**明显的跨期需求前置风险（政策驱动具有持续性）
- 汽车：**存在**以旧换新补贴退坡后的需求回落风险（影响有限，MDI用量占比小）
- 储能：**存在**配储政策导致的短期抢装效应，但长期替换需求可对冲
- 家电：**存在**以旧换新补贴导致的透支风险

### ✅ 复核项3：供需边际演变与压力传导

已基于产能增量与终端需求增速，分析了各板块的供需边际演变：
- MDI偏向**从松转紧**（短期地缘驱动+中期需求增长）
- 石化处于**底部寻底**（产能出清刚开始，还将持续2-3年）
- LFP高端**紧俏**但低端**过剩出清**（结构性分化加剧）
