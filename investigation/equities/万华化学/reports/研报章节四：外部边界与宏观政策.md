# 研报章节四：外部边界与宏观政策

**研究日期：2026年05月09日**

## 一、贸易摩擦与关税壁垒：直接冲击已基本Price-In

### 1.1 美国MDI反倾销案：终裁落地，最坏已过

2026年4月8日，美国商务部对进口自中国的MDI反倾销作出终裁，万华化学获85.11%倾销税率（较初裁376.12%大幅下调291个百分点），其他中国企业为159.04%。5月1日，美国国际贸易委员会（USITC）发布损害终裁，认定中国MDI对美国产业造成实质性损害，反倾销税正式开征。

**叠加现有关税后的综合税率**：
- 基础关税：6.5%
- 301条款关税：25%
- "对等关税"：10%（24%部分暂停至2026年11月）
- 反倾销税：85.11%
- **综合税率约126.61%**

**量化影响评估**：

| 指标 | 数据 | 影响程度 |
| :--- | :--- | :---: |
| 2024年中国对美MDI出口 | 约26.9万吨（占比美国进口50-60%） | — |
| 2025年1-10月对美出口 | 同比下降82.6%，全年仅约4.63万吨 | 重大 |
| 美国市场占万华MDI总产能 | <8% | 有限 |
| 万华匈牙利基地可替代对美供应 | 30万吨/年（非中国原产地） | 对冲有效 |
| Fitch评级判断 | "对万华信用状况影响相对有限" | 确认 |

**核心判断**：终裁税率虽高达85.11%，但①较初裁"封杀级"511.75%已大幅下降；②万华通过匈牙利基地实现"绕开关税"；③对美出口占比极低。此项风险已基本Price-In，且终裁落地反而消除了不确定性。

### 1.2 其他贸易摩擦风险

- **中美关税升级风险**：特朗普近期警告若中国向伊朗提供军事援助，将对所有中国商品加征高达50%报复性关税。此风险具有高度不确定性但影响面广泛。
- **欧盟反倾销风险**：目前未针对MDI发起反倾销，但CBAM碳边境税的间接影响正在显现。
- **全球贸易多极化趋势**：中国MDI出口结构已从"依赖美国"转向"多元化市场"，2026年2月前五大出口国（土耳其、韩国、阿联酋、印度、越南）占比超50%。

## 二、欧盟碳边境调节机制（CBAM）：远期成本压力显著

### 2.1 CBAM实施进展

CBAM于2026年1月1日结束过渡期、正式进入征收阶段。首期证书价格为75.36欧元/吨CO₂当量（约588元/吨），2026年按季度更新，2027年起按周动态调整。首批清缴时间为2027年（清缴2026年度进口碳排放）。

### 2.2 对万华的传导路径

**短期（当前—2027年）**：**影响较小**

CBAM当前仅覆盖钢铁、铝、水泥、化肥、电力、氢六大行业。万华的主要产品MDI、TDI、聚醚、PC、PMMA、POE等均不在首批管控清单中。唯一可能受影响的是化肥相关产品线，但占万华营收比例极低。

**中期（2028年起）**：**影响中等**

欧盟已提出立法草案，计划2028年起将CBAM扩展至有机化学品（HS code 29开头产品），这将涵盖万华的大部分精细化学品及石化出口产品。届时若中国碳市场仍不被欧盟认可（当前因两大短板：①缺乏总量约束，②有偿配额仅10-15% vs 欧盟90%），万华对欧出口将面临双重碳成本。

**中国企业面临的核心困境**：
- 中国碳价约77元/吨（~10欧元），仅为欧盟碳价（75.36欧元/吨）的13%
- 国内已缴碳费不被欧盟抵扣
- 若使用欧盟默认排放值（通常比实测值高20-50%），成本将进一步放大

### 2.3 万华的对冲优势

万华在应对CBAM方面具备相对优势：
- **绿电布局**：莱州绿电产业园、蓬莱基地绿电配套
- **技术降碳**：煤浆提浓、废CO₂利用（甲烷-二氧化碳联合重整中试成功）、废塑料回收
- **匈牙利基地**：位于欧盟境内，不受CBAM约束
- **乙烷原料替代**：乙烯装置原料多元化降低碳足迹

### 2.4 CBAM影响量化预估

假设2028年有机化学品纳入CBAM，万华对欧出口约占总营收的8-10%（~160-200亿元），平均碳排放强度预计2-3吨CO₂/吨产品：

- CBAM证书成本（以75欧元/吨估算）：150-225欧元/吨 → 约1,170-1,755元/吨
- 对于毛利率仅10.53%的精细化学品（ASP约2-3万元/吨），碳成本占产品价值的4-6%
- 对于毛利率0.58%的石化产品，碳成本将完全抹去微薄利润

**结论**：CBAM对万华的核心冲击不在当期而在远期（2028年后），且万华具备一定的应对工具（绿电、匈牙利基地、低碳技术）。当前可将此视为中长期观察变量而非短期风险因子。

## 三、宏观环境变量

### 3.1 利率与折现率环境

2026年中国继续实施"适度宽松的货币政策"，政策利率已下调10个基点（2025年），社综合融资成本处于低位运行。2026年赤字率拟按4%安排，超长期特别国债1.3万亿元，地方政府专项债券4.4万亿元。

对万华的影响：
- **融资成本降低**：2026Q1万华短期借款601亿元、长期借款485亿元，融资成本下降10-20个基点，年化节省约1-2亿元利息支出
- **资产折现率影响**：在DCF估值框架下，较低的折现率将抬升万华的合理估值。但万华当前ROIC仅7.28%仍低于WACC（估计8-10%），估值折现率变动对公允价值的影响有限

### 3.2 汇率风险

万华2025年境外营收963.21亿元，占总营收47.4%，且境外毛利率（11.50%）低于境内（15.45%）。人民币汇率变动的影响呈现双重性：

- **人民币升值**：以人民币计价的境外收入折算缩水，对利润表构成不利影响
- **人民币贬值**：境外收入折算增加，但同时进口原材料（乙烷、LPG、纯苯等）成本上升

2025年人民币对美元汇率在7.0-7.25区间波动，2026年政策定调"保持在合理均衡水平上基本稳定"。万华作为大型跨国企业，已建立相对完善的外汇对冲机制（衍生金融工具公允价值变动2.66亿元/2025年）。

**敏感性测算**：人民币每贬值1%，境外营收折算增加约9.6亿元，但同时原材料进口成本增加约7-8亿元（假设进口依赖度约40%），综合正面影响约1.5-2亿元税后利润。汇率风险可控。

### 3.3 通胀与大宗商品传导

万华的核心原材料（纯苯、LPG、煤、盐等）均为大宗商品，价格受全球供需与地缘政治影响剧烈。2026年3月以来，地缘政治冲突推动原油、纯苯等价格上涨，但对万华而言是"双刃剑"：

- **成本推升**：纯苯价格上涨直接抬升MDI原料成本
- **产品涨价**：MDI价格从14,500元/吨飙升至20,500-21,000元/吨，涨幅远超成本增幅

从2025年年报看，原材料占总成本的79.46%，但万华通过成本转嫁能力（MDI全球定价权）和一体化原料自供（苯胺、氯碱等自产），在涨价周期中通常享有"剪刀差"优势——产品涨价快于成本上升。

## 四、国内产业政策：方向性利好

### 4.1 "十五五"规划与化工行业

《"十五五"规划纲要》明确了化工行业的四大方向：
1. **重点领域安全保障**：提升化肥、能源化工的战略供给能力
2. **综合整治"内卷式"竞争**：健全产能监测预警机制，采取规划引导、产能调控、价格治理、行业自律等措施
3. **新材料国产替代**：培育新兴产业和未来产业，推动化工新材料国产化
4. **绿色低碳转型**：碳排放双控制度，扩大碳市场覆盖，建设零碳园区

对于万华而言：
- **"反内卷"政策利好**：若石化行业产能调控落地，将直接缓解万华石化板块的亏损压力
- **新材料国产替代**：万华的POE、MS树脂、XLPE、特种胺等均属于进口替代方向
- **绿色低碳**：万华已有较好基础（煤浆提浓、CO₂利用、废塑料回收），绿电产业园布局先行一步

### 4.2 设备更新与消费品以旧换新政策

2026年安排2000亿元超长期特别国债支持设备更新，对万华的影响：
- 大型化工设备的更新换代可获政策补贴
- 消费品以旧换新（家电、汽车等）刺激MDI下游需求

### 4.3 财政与货币政策宽松

2026年"更加积极的财政政策+适度宽松的货币政策"组合，直接影响：
- 基础设施投资加速 → 建筑保温材料（MDI）需求增长
- 消费刺激政策持续 → 家电、汽车（PU）需求维持
- 融资成本降低 → 万华每年节省财务费用1-2亿元

## 七、核心宏观定价因子

在对万华进行估值时，以下宏观因子必须被纳入考量：

1. **折现率（WACC）**：预计8-10%，其中无风险利率约2.5-3%（中国10年期国债）、股权风险溢价约5-7%
2. **终端增长率**：预计2-3%（与全球GDP增速和化工品需求长期增长率匹配）
3. **有效税率**：当前约15%（高新技术企业所得税优惠），政策稳定性较高
4. **汇率假设**：基准假设人民币对美元7.0-7.3区间波动
