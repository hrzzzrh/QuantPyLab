# 赣锋锂业 (002460) 动态监控仪表盘 (跟踪手册)
**创建日期**：2026年5月8日
**最后更新**：2026年5月26日

本手册旨在列出未来 6-12 个月内足以影响核心投资逻辑的关键变量，为后续跟踪与调仓提供机械的执行纪律。

## 1. 核心逻辑验证点 (Thesis Validators)
需要在每季财报中紧密验证的定量指标：
*   **毛利率与净利率（盈利质量核心）**：单季销售毛利率必须维持在 **20% 以上**（Q1实际29.72%，验证了低成本自有矿的降本效应）。**更新**：随着锂价从底部大幅回升，核心关注点转向：毛利率能否在高锂价环境下维持或继续改善（验证公司定价权）。
*   **海外项目产能释放进度**：重点跟踪马里 Goulamina 锂辉石项目（2026年目标满产50.6万吨精矿，Q1出货顺利验证）、阿根廷 Cauchari-Olaroz（2026年指引3.5-4万吨LCE）和 Mariana 盐湖的实际产出量。
*   **经营性现金流 (OCF)**：单季经营性现金流净额需保持强劲的正向流入。Q1 OCF为7.87亿元（同比+150%），但较Q4环比下降，需持续跟踪回款质量。

## 2. 关键外部变量 (Exogenous Watchlist)
*   **碳酸锂现货/期货价格（高频监控）**：每日跟踪广期所碳酸锂期货主力合约及SMM/长江有色现货报价。**更新**：当前价格区间已上移至 **17-21万元/吨**。重点关注：(1) 18万关口能否守住；(2) 津巴布韦出口配额发放进展；(3) 广期所仓单变动量；(4) 下游正极材料厂对高价的接受度变化。
*   **津巴布韦锂精矿出口禁令进展（新增优先跟踪）** 🔴：这是当前全球锂供给最大的不确定性来源。关键跟踪节点：(1) 津巴布韦政府对出口配额的新发放情况；(2) 中资矿企（华友钴业、中矿资源等）在当地锂盐冶炼厂的建设进度；(3) 首批锂精矿/锂硫酸盐的实际到港时间（预计最早7月）。
*   **江西宜春锂云母矿停产换证进展（新增优先跟踪）** 🔴：跟踪枧下窝矿区及其他主力矿山的采矿权换证审批进度。若集中复产，可能对锂价构成短期供给冲击。
*   **储能电池装机量数据**：每月跟踪BNEF、InfoLink、国家能源局发布的全球及国内储能装机/出货数据。**更新**：2026年Q1全球储能出货已超预期，全年维持600GWh高位预期。验证点：每月排产数据是否维持环比增长趋势。
*   **全球利率环境**：美联储等核心央行的降息步伐（直接影响储能电站的融资成本和内部收益率IRR）。

## 3. 逻辑证伪警报 (Invalidation Triggers) 🚨
当以下任一信号出现时，必须承认现有逻辑已死，**强制触发减仓或下调评级**：
1.  **锂价中枢崩塌**：碳酸锂现货价格**连续两周低于12万元/吨**（意味着供给缺口逻辑彻底被证伪，全年均价将大幅低于当前预期）。
2.  **地缘资产暴雷**：公司核心海外资产——**马里Goulamina矿出现所在国国有化、实质性长期停产或矿业权被收回**的任何信号（这是公司2026年利润增量的核心引擎，一旦出事直接摧毁盈利预测）。
3.  **需求端崩塌**：全球储能电池单月/单季出货量**同比增速跌破15%**（储能接力逻辑被证伪）。
4.  **技术替代加速**：钠离子电池在LCOE（平准化储能成本）上实现对磷酸铁锂的全面超越，且头部企业宣布大规模替代锂电路线图（CATL 60GWh钠电协议需持续跟踪其实际交付进度和成本表现）。

*   **2026年6月底**：广期所碳酸锂主力合约LC2609交割情况（仓单压力释放窗口）。

*   **每月初**：国内外重要新能源车交付数据、中国储能并网规模数据发布日。
