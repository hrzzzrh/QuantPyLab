# 赣锋锂业 (002460) 动态监控仪表盘 (跟踪手册)
**创建日期**：2026年5月8日

本手册旨在列出未来 6-12 个月内足以影响核心投资逻辑的关键变量，为后续跟踪与调仓提供机械的执行纪律。

## 1. 核心逻辑验证点 (Thesis Validators)
需要在每季财报中紧密验证的定量指标：
*   **毛利率与净利率（盈利质量核心）**：单季销售毛利率必须维持在 **15% 以上**（验证公司低成本自有矿的降本效应和产业链议价权）。
*   **海外项目产能释放进度**：重点跟踪马里 Goulamina 锂辉石项目、阿根廷 Cauchari-Olaroz 和 Mariana 盐湖的实际产出量（LCE吨数），验证销量增长逻辑。
*   **经营性现金流 (OCF)**：单季经营性现金流净额需保持强劲的正向流入，验证下游回款健康度。

## 2. 关键外部变量 (Exogenous Watchlist)
*   **储能电池装机量数据**：每月跟踪国家能源局及第三方机构（如高工锂电）发布的全球及国内大储/户储实际并网装机数据，验证储能增长是否达到40%以上预期。
*   **碳酸锂现货价格**：高频监控 Fastmarkets 等平台报价，8万元/吨为全行业的成本安全垫，10-12万元/吨为公司利润爆发舒适区。
*   **全球利率环境**：美联储等核心央行的降息步伐（直接影响储能电站的融资成本和内部收益率 IRR）。

## 3. 逻辑证伪警报 (Invalidation Triggers) 🚨
当以下任一信号出现时，必须承认现有“紧平衡+量价齐升”逻辑已死，**强制触发减仓或下调评级**：
1.  **产品端失守**：连续两个季度单季**销售毛利率跌破 10%**（说明公司降本速度不敌锂价下跌或价格战传导加剧）。
2.  **需求端崩塌**：全球储能电池单月/单季出货量**同比增速跌破 15%**（储能接力逻辑彻底被证伪）。
3.  **地缘资产暴雷**：公司任何一个处于运营期或爬坡期的核心海外主力矿山（如 Goulamina, Cauchari-Olaroz）遭遇所在国**国有化或实质性长期停产**制裁。

## 4. 关键时间节点 (Calendar)
*   **2026年8月底**：公司2026年半年度报告披露（检验Q2储能旺季拉动效应及半年度EPS兑现度）。
*   **每月初**：国内外重要新能源车交付数据、中国储能并网规模数据发布日。
*   **不定期**：墨西哥 Sonora 仲裁案 ICSID 的中期裁决或关键进展通报。
