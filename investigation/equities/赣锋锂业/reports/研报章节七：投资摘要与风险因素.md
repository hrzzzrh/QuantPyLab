# 研报章节七：投资摘要与风险因素
**研究日期**：2026年6月13日

## 1. 投资摘要 (Investment Summary)

赣锋锂业（002460）正处于其产业生命周期中罕见的"**盈利暴增 + 股价深度回调后反弹**"的极端价值收敛窗口。自2026年5月26日上期研究以来，以下核心变量发生了关键性变化：

1. **锂价经历"V型"走势，全年均价中枢维持不变**：
   *   碳酸锂期货从5月26日的18.44万元/吨持续杀跌至6月4日的15.76万元/吨（周跌幅10.7%），随后强劲反弹至6月12日的17.46万元/吨（+10.8%）。Q2均价约17万元/吨，仍高于Base情景假设的16.5万元/吨。
   *   高盛6月12日发布报告，预测2026年Q4锂价可能触及25万元/吨，将当前周期类比2020-2021年。
   *   津巴布韦锂精矿出口禁令部分解除（6月10日多家公司确认获配额），但恢复受配额管控，首个显著增量窗口在7月中下旬。
   *   反向供给扰动：智利5月对华碳酸锂出口骤降40.8%（预计6-7月进口显著回落）+ Greenbushes CGP3火灾 + 枧下窝用地预审被注销（复产预期进一步下降）。

2. **公司经营全线确认强劲（6月8日投资者交流）**：
   *   Goulamina 1-4月产出16.14万吨锂精矿（年化约48.4万吨，接近满产），二期已完成可研。
   *   Cauchari-Olaroz 1-4月产出1.26万吨LCE，已入选阿根廷RIGI激励计划。
   *   储能电芯**满产满销、产能利用率接近100%**，588Ah大电芯产线Q3启动爬坡。

3. **盈利预测维持不变，但股价提供了更深的折扣**：
   *   2026年Base EPS维持 **4.06元**（全年锂均价16.5万元/吨，净利85亿元）。
   *   股价从5月26日的75.00元最低跌至6月8日的61.18元（-18.4%），随后反弹至71.16元（6月12日）。6月8日低点对应2026E PE仅**15.1倍**，为周期反转股中极为罕见的低估。
   *   当前71.16元对应2026E PE仅 **17.5倍**，处于全历史PE分布的极度低估区间（P25约35x，P50约54x）。

4. **估值结论**：给予2026E 25x Forward PE（较历史中枢折价54%，体现周期股合理估值收缩），经全变量对冲矩阵+5%修正，**得出目标价106.50元（对应2026E PE 26.2x），较当前市价71.16元有49.7%上行空间**。三态概率加权综合目标价约 **104.50元**（+46.9%），盈亏比3.17。

**三态目标价**：
*   保守 Bear（概率20%）：**60.00元**（2.39元EPS × 25x）
*   中性 Base（概率60%）：**106.50元**（4.06元EPS × 26.2x）
*   乐观 Bull（概率20%）：**143.00元**（5.73元EPS × 25x，PE从28x下调，因储能IRR天花板压缩成长溢价）

**当前现价**：71.16元（不复权，2026年6月12日收盘）

## 2. 风险因素 (Risk Factors)
按对股价影响的烈度由高到低排序：

1.  **锂价短期剧烈回调风险（烈度：极高）**：
    *   当前锂价（17.0万元/吨）已较5月12日高点（20.99万）累计回调19%。5月26日至6月4日期间，锂价从18.2万暴跌至15.76万（周跌幅10.7%），直接触发了股价6月8日暴跌至61.18元（2026E PE 15.1x）。若津巴布韦精矿7月集中到港叠加下游淡季，锂价可能在Q3再次考验15万关口。
    *   **对冲因素**：Q1-Q2已锁定较高均价（~16万元/吨），全年Base EPS 4.06元安全垫厚实。即使H2回落至12万，全年EPS仍可维持在2.39元以上（Bear情景）。

2.  **储能IRR天花板制约锂价上行空间（烈度：高）** 🆕：
    *   根据36氪一线调研与光大证券储能IRR敏感性测算，碳酸锂180,000元/吨时多数中国独立储能项目IRR跌破6%决策红线，200,000元/吨时大面积延期/取消。储能已占锂电排产42.3%，其高价格敏感度构成锂价上行的刚性天花板——与2021-2022年电动车驱动牛市的本质区别在于，储能投资商对每分钱成本精算到底。
    *   **但支撑底线依然坚实**：即使200,000元+时中国储能边际需求（约占全球储能30-40%）收缩，欧美大储（IRR 12%+）+动力电池+AI数据中心备电仍可提供约160-180万吨LCE的需求底线。这意味着锂价处于150,000-180,000元/吨时供需最为均衡，赣锋的全成本优势在此区间内最能充分释放。

3.  **海外资产地缘政治风险（烈度：高）**：
    *   津巴布韦禁令的"制度化升级"（5月22日关键矿产宣言+国家强制持股+10%出口税）具有一定的政策示范效应。马里已于2023年通过类似矿业法（政府持股上限35%），但赣锋已与马里政府签署Goulamina项目新矿业法协议，当前运行正常。需持续警惕：若有其他非洲资源国效仿津巴布韦的激进政策。
    *   墨西哥Sonora仲裁仍在推进中，2026年5月8日仲裁庭完成重组，但实质裁决尚无时间表。

4.  **津巴布韦锂精矿集中到港风险（烈度：中高）** 🆕：
    *   6月10日确认多家企业已获出口配额并启动发运，预计7月中下旬出现首个显著到港增量。若到港量超预期（月均1.5万吨LCE以上），可能对Q3锂价形成阶段性压力。
    *   **对冲因素**：智利5月对华出口骤降40.8%意味着6-7月进口缺口更大于津巴布韦增量。且津巴布韦出口受配额管控+10%出口税+2027年硫酸锂厂deadline等多重约束，实际增量可能低于市场预期。中矿资源Bikita首批仅0.5-0.7万吨精矿，量级有限。

5.  **技术路线颠覆风险（烈度：中）**：
    *   2026年6月，CATL进一步确认将于9月交付首批钠离子储能系统、2026年实现GWh级出货，并与海辰储能签署60GWh三年期协议。钠电商业化步伐持续加速。预计钠电成本2026年底与LFP平价。
    *   **对冲因素**：钠电目前主要适用于对能量密度要求不高的场景（大型地面电站），在动力电池和高端储能领域仍无法替代锂电。CATL自身也表示钠电是锂电的"补充"而非"替代"。赣锋锂业的固态电池布局具备差异化竞争壁垒。

6.  **在建工程转固带来的折旧压力（烈度：中）**：
    *   高达154亿的在建工程一旦转固，若产能爬坡不及预期，巨额的固定成本摊销将严重拖累当期利润。需要持续跟踪产能利用率。

7.  **市场风格切换风险（烈度：中低）**：
    *   当前市场对周期股的"见顶焦虑"可能导致估值持续承压。尽管基本面强劲，但市场持续给予周期股低PE估值（15-20x），已导致股价从91.45元跌至61.18元（-33.1%）。但极端低估值（2026E PE 17.5x）本身已充分甚至过度反映了这一风险。

## 3. 研究结论象限图 (Final Evaluation Matrix)

基于当前**极高确定性的基本面改善**（Q1 EPS 0.88元已兑现 + Q2锂价17万元/吨已验证 + 公司6月8日投资者交流全线确认经营强劲）和**进一步扩大的预期收益空间**（目标价106.50元，+49.7%上行空间，盈亏比3.17）：

**公司精确定位于：第一象限 (High-Conviction Buy) —— 高确定性 + 高预期收益空间**。

*   **象限转换说明**：维持第一象限定位。较上期（盈亏比2.10）进一步强化，盈亏比飙升至3.17。核心驱动因素：(1) Base EPS维持4.06元不变，基本面改善逻辑未受动摇；(2) 股价从75元进一步回调至71.16元（6月8日盘中一度触及61.18元的恐慌极值）；(3) 2026E PE从18.5倍压缩至17.5倍，估值安全边际更为深厚。6月8日极端低点61.18元对应的2026E PE仅15.1倍，几乎是"破产定价"（市场假设锂价将崩回8万元以下，与基本面现实严重背离），为前瞻性布局者提供了罕见的"恐慌性错杀"买入窗口。
*   **注意**：虽然当前价位已从61.18元反弹15.9%至71.16元，但2026E PE仍仅17.5倍，且上行空间仍有46.9%。从盈亏比3.17和概率加权目标价104.50元来看，当前价位依然具备极高的配置价值。

*   **投资建议**：**积极买入/增持**。在当前70-72元区间积极布局，首仓建议目标仓位的50%。若出现二次回踩MA200（63.76元附近）或缩量整理至64-68元区间，可加仓至80%。止损位下调至**61.00元**（基于6月8日技术底+估值底双重确认，-14.3%），若基本面出现逻辑证伪信号（如锂价持续跌破12万、Mali政策突变）则无条件离场。
