# 研报章节六：技术面分析
**研究日期**：2026年6月13日

**重大更新说明**：自2026年5月26日上期研究以来，股价经历了"加速下跌→击穿200日均线→暴力反弹"的极端波动。5月26日收盘价75.00元，至6月8日最低触及61.18元（累计跌幅18.4%），随后四个交易日反弹至71.16元（+15.9%）。200日均线（长期趋势生命线）被短暂有效击穿后收复，趋势结构需要审慎重评估。

## 1. 趋势架构分析
当前赣锋锂业（002460）处于**长期上升趋势中的深度回调完成→反弹修复**阶段。趋势结构出现局部破坏但未发生根本性反转。

*   **均线系统（关键变化）**：
    *   60日均线（MA60）：当前位于 **77.18元**（后复权1,091.43元）。股价（71.16元）仍处于MA60下方约7.8%，但自6月8日低点已反弹15.9%，收复MA60的动能正在积聚。
    *   200日均线（MA200）：当前位于 **63.76元**（后复权901.67元）。**6月8日盘中最低61.18元、收盘62.76元，为2026年2月本轮上涨以来首次有效击穿200日均线。** 但随后连续三个交易日放量反弹收复MA200（6月9日收盘67.94元），该关键支撑位的技术意义依然成立。
    *   均线系统：5日/10日/20日短期均线仍呈空头排列，但5日均线已在62元附近拐头向上，即将金叉10日均线（67元附近）。MA60仍高于MA200，长期牛市的均线多头排列格局未被破坏。
*   **价格行为（完整下跌-反弹结构回顾）**：
    *   5月6日：本轮上涨高点 **91.45元**（后复权1,293.03元），52周最高价。
    *   5月13日-6月8日：伴随碳酸锂期货从20.99万元/吨持续杀跌至15.76万元/吨（-24.9%），股价进入加速下跌通道，累计跌幅 **31.4%**（91.45 → 62.76），共计19个交易日。
    *   **6月8日：极端情绪低点**。盘中最低触及 **61.18元**（后复权865.20元），收盘62.76元。当日碳酸锂期货一度跌至15.76万元/吨，市场恐慌情绪达到本轮下跌的极致。
    *   6月9日-12日：伴随碳酸锂期现货反弹（15.76→17.46万元/吨，+10.8%），股价爆发式反弹。**4个交易日反弹+13.4%**（62.76 → 71.16），成交量从50.5万手逐级放大至87.7万手（6月11日），呈现典型的"放量反弹、缩量回踩"格局。
*   **阶段定性**：本轮极端下跌已完成。核心驱动逻辑：碳酸锂从20.99万→15.76万的恐慌性杀跌，叠加广期所仓单高企（一度超5.6万手）的隐性库存显性化担忧，触发了技术面与情绪面的共振踩踏。但6月8日锂价15.76万元/吨与股价61.18元同时见底，形成了明确的**量价同步拐点**。当前反弹仍在进行中，量价配合良好。

## 2. 关键支撑与阻力位（以不复权市价标注）
*   **强阻力位（上行目标）**：**91.45元**（后复权1,293.03元，52周前高）。突破需锂价稳定站上20万+或Q2业绩超预期催化。
*   **中期阻力位（第一目标）**：**77.18元**（后复权1,091.43元，MA60当前位置）。收复此位将确认中级调整结束。这也是前期4月底-5月初的成交密集区上沿。
*   **短期阻力位**：**75.00元**（后复权1,060.44元，5月26日研究锚点）。6月9-12日的反弹在此价位附近遭遇了部分获利了结压力。
*   **核心支撑位（中期生命线，已受考验但守住）**：**63.76元**（后复权901.67元，200日均线）。**该位在6月8日已被短暂击穿（收盘62.76元），但次日即放量收复。** 这意味着63.76元虽非"不可跌破的铁底"，但其有效性已经过一次实战检验。若未来二次回踩且放量击穿此位，则中期趋势将实质性转弱。
*   **极端支撑位（底线）**：**61.18元**（后复权865.20元，6月8日盘中低点）。此为本轮下跌的绝对极值，基于估值底（对应2026E PE约15x）和情绪底的双重确认。

## 3. 量价行为与未来走势演绎
*   **量价结构分析**：
    *   **下跌末期放量**：6月5日（65.65元）和6月8日（62.76元）成交量分别为56.2万手和50.5万手，较5月底的40-45万手明显放大。在长期下跌后的低位放量，往往意味着恐慌盘（止损盘+融资盘）的出清，这是底部形成的必要条件之一。
    *   **反弹放量确认**：6月11日反弹至69.94元时成交量达87.7万手，为6月最高，且明显高于下跌过程中的日均量（约45-55万手）。放量反弹 + 缩量回踩是趋势反转的早期信号。
    *   **锂价-股价联动**：本轮下跌中，碳酸锂期货价格与赣锋锂业股价的相关系数极高（15.76万 vs 61.18元同时见底）。这进一步证明本轮下跌主因是"锂价暴跌→盈利预期下修→估值压缩"，而非公司基本面恶化。
*   **短期走势（1-3个月）**：
    *   预计股价将在 **[63.76, 77.18]** 区间内震荡筑底并尝试向上突破。当前71.16元处于该区间的约55%分位。
    *   **核心观察点**：未来2周内，若股价能放量突破并站稳MA60（77.18元），则可确认中级调整结束，上升趋势回归，目标指向84-91元区间。
    *   **下行风险**：若碳酸锂价格再度回落并跌破15万元/吨，股价可能二次回踩63.76元（MA200）。但鉴于锂价已从15.76万强劲反弹至17.46万且基本面坚实，该概率较低。
*   **长期走势（6-12个月以上）**：
    *   200日均线当前63.76元仍是多头趋势的生命线。公司在6月8日投资者交流会上确认：Goulamina（1-4月产出16.14万吨精矿）、Cauchari-Olaroz（1-4月产出1.26万吨LCE）均运行正常，储能电芯满产满销。
    *   基于2026E Base EPS 4.06元，当前71.16元对应2026E PE仅**17.5倍**。中长期基本面目标价106.50元不变，对应2026E PE 26.2x。技术面与基本面的剪刀差已达到历史极端水平。
*   **交易策略建议**：
    *   **已持有仓位**：MA200（63.76元）上方坚定持有。6月8日虽短暂击穿但已收复，止损纪律已被触发但随即被市场修复，建议将止损线下移至**61.00元**（日内低点以下），给予更宽的波动容忍度。
    *   **新建仓位**：当前71.16元处于"已从底部反弹+15.9%但仍在MA60以下"的中间位置。激进策略可在69-72元区间分批建仓（50%仓位），保守策略等待放量突破MA60（77.18元）后右侧追入。
    *   **加仓信号**：放量站上MA60（77.18元）或缩量回踩MA200（63.76元）不破。
    *   **止损纪律**：若收盘价连续两日低于61.00元，基本面改善逻辑可能已被市场完全否定，需无条件离场重新评估。

## 4. 估值与技术的交叉验证
*   当前动态PE（2026E Base EPS 4.06元）仅 **17.5倍**，处于公司全历史PE分布的极端低位（P25约35x，P50约54x）。
*   技术面深度回调（-31.4%）vs 基本面持续改善（公司确认Q2各项经营指标向好），形成了更为极端的"**恐慌性错杀窗口**"：6月8日低点61.18元对应的2026E PE仅**15.1倍**，为周期反转股中极为罕见的低估水平。
*   52周价格区间：不复权最低价30.60元 → 最高价91.45元，当前71.16元处于该区间的约66%分位。6月8日低点61.18元处于约50%分位，已回吐本轮上涨（30.60→91.45）的约49.7%，符合技术性回调（38.2%-61.8%黄金分割）的经典幅度。
