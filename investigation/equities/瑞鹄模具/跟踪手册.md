## 三、逻辑证伪警报 (Invalidation Triggers)

**以下信号出现时，必须承认当前投资逻辑已死或严重受损，直接触发止损或下调评级：**

### 3.1 致命级（一旦出现→立即清仓）

| 警报信号 | 判定标准 | 逻辑破坏路径 |
| :--- | :--- | :--- |
| **奇瑞大幅削减订单** | 单季度来自奇瑞的营收同比下降>30% | 第一大客户流失→生存基础动摇 |
| **核心客户更换供应商** | 奇瑞公告或行业消息确认模具/零部件主供切换 | 竞争壁垒被证伪 |
| **财务造假或重大违规** | 被证监会立案调查/审计师出具否定意见 | 信任基础崩塌 |

### 3.2 严重级（一旦出现→减仓至半仓以下）

| 警报信号 | 判定标准 | 逻辑破坏路径 |
| :--- | :--- | :--- |
| **综合毛利率跌破18%** | 连续2个季度综合毛利率<18% | 盈利模式陷入困境，价格战全面传导 |
| **零部件毛利率跌破5%** | 连续2个季度零部件毛利率<5% | 零部件业务不具经济可行性 |
| **经营现金流持续为负** | 连续4个季度OCF/NP < 0 | 利润质量崩溃，可能面临流动性危机 |
| **在手订单大幅萎缩** | 装备业务在手订单同比下降>20% | 未来收入确定性被破坏 |
| **存货大额减值** | 单季度计提存货减值>净利润的20% | 订单取消或客户违约集中爆发 |

### 3.3 警示级（一旦出现→重新评估持仓，密切跟踪）

| 警报信号 | 判定标准 | 逻辑破坏路径 |
| :--- | :--- | :--- |
| **奇瑞市占率连续下滑** | 奇瑞在中国乘用车市场市占率连续2季下降>1pct | 核心客户基本面恶化 |
| **新能源车终端增速降至个位数** | 中国新能源车月度销量同比<5% | 行业增长逻辑弱化 |
| **毛利率连续3季环比下降** | 每个季度毛利率环比下降>1pct | 毛利率无底线下滑 |
| **可比公司估值系统性下移** | 行业平均PE降至10-12x | 板块杀估值风险 |
| **管理层大规模减持** | 董事长/总经理/控股股东减持>5% | 内部人信号 |

## 六、2026年5月24日审计更新

### 6.1 已跟踪事项更新

**奇瑞2026年Q1及4月销量核实**：
- Q1总销量56.61万辆（同比-3.1%），营收658.7亿元（同比-3.45%），归母净利润41.7亿元（同比-10.32%）。Q1整体弱于年初目标节奏（320万辆÷4≈80万辆/季度）。
- 4月总销量25.14万辆（同比+25.2%），新能源10.03万辆，出口17.76万辆（创单月出口新纪录）。4月强劲反弹部分对冲Q1弱势。
- 4月国内销量仅7.38万辆（占29%），出口占比71%，内需疲软但出口强劲。
- Q1主要弱势品牌：星途（-39.9%）、捷途（-22.6%）、智界（-75.7%）、iCAR（-17.2%）；奇瑞主品牌销量40.68万辆（+15.5%）表现逆势增长。
- 前四个月累计约81.75万辆，年化约245万辆，距320万辆目标仍有差距。但Q2通常为全年销量旺季，且新车集中上市（瑞虎9、QQ3、EX7、智界V9等）有望驱动H2加速。
- **判断**：奇瑞320万辆年目标达成概率从此前的高确定性下调至中等确定性。Q1弱势+4月反弹的组合尚未改变长期增长逻辑，但需密切跟踪Q2累计销量来重新校准。

**可转债进度更新**：
- 2026年1月29日公司回复深交所审核问询函，此后无实质性进展公告。目前审核状态仍为"已问询"。
- 原报告假设的"2026年5-6月完成发行"未能兑现。最早发行窗口推迟至2026年Q3-Q4。
- **影响**：短期资金到位延后，但公司截至2026Q1仍有货币资金6.21亿元+尚未使用的首发募集资金，短期流动性无虞。募投项目建设可通过自筹资金先行投入。

**美伊和平协议框架（新增外部变量）**：
- 2026年5月23日，特朗普宣布美伊和平协议"largely negotiated"，框架包括：60天停火延期、霍尔木兹海峡复航、伊朗可自由出售石油、后续谈判限制伊朗核计划。
- 伊朗方面对部分细节（如交出浓缩铀）存在争议，但谈判方向积极。巴基斯坦、沙特、阿联酋等多国参与斡旋。
- **若协议最终落地**：霍尔木兹海峡复航→中东铝产能恢复→全球铝供给改善→铝价回落至$2,800-3,000/t→瑞鹄模具零部件业务毛利率可能回升至12-14%（乐观情形），构成2026年最重要的正向催化剂。
- **风险**：谈判仍可能破裂。若协议未达成而战事重燃，铝价可能飙升至$4,000+，构成重大下行风险。
- **跟踪**：未来1-2周关注协议正式签署情况，以及霍尔木兹海峡实际通行恢复的声明。
