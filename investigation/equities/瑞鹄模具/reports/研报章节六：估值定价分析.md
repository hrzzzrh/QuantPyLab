# 研报章节六：估值定价分析

**研究日期**：2026年05月26日

## 一、价值锚定：适配性方法选择

### 1.1 方法论选择与理由

瑞鹄模具属于 **"成长型装备+零部件制造商"**，具有以下特征：
- 轻资产中资产运营（固定资产12.5亿，总资产72亿）
- 利润正增长、现金流改善中
- 装备业务具备较高可预见性（订单驱动），零部件业务处于快速成长期
- 横跨两个业务板块，盈利能力差异显著

**综合判断**：选择 PE（市盈率）+ SOTP（分部估值）组合交叉验证。理由：
1. PE是最直接、市场最通用的估值方法，适合盈利增长稳定的公司
2. SOTP能更精准反映装备和零部件业务的不同估值逻辑
3. PB不适用（重资产程度低且未进入清算阶段）；DCF因零部件业务波动大、远期参数高度不确定而不可靠；PS可作为辅助参考

### 1.2 历史估值分位审计

| 时间维度 | 当前PE（16.76x）对应分位 | 当前PB（2.66x）对应分位 |
| :--- | :--- | :--- |
| 近3年（2023-2026） | **<P10**（16.44为P10） | **<P10**（2.87为P10） |
| 近5年（2021-2026） | **<P10** | **<P25** |

**当前估值处于上市以来的历史最低区间**，但需审慎评估这一"低估"是周期性的均值回归机会还是结构性的估值中枢下移：

- **支持均值回归的因素**：利润持续增长、订单历史新高、产能扩张带来规模效应
- **支持中枢下移的因素**：零部件毛利率趋势性下降→盈利质量弱化、客户集中度过高→风险溢价上升、成长性从30%+降至15-20%→成长溢价收缩

## 二、估值全变量对冲审计矩阵（强制性输出）

**基准市值计算**：
- 基准2026E净利润：4.7亿
- 合理PE（参考历史中位数+可比估值+增长前景）：18-22x
- **基准市值区间：84.6亿 - 103.4亿**
- 基准市值中值：94亿

**审计矩阵**：

| 审计维度 | 核心审计因子 | **审慎思考与估值影响力过滤** | 调节建议 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质** | "装备+零部件"垂直整合模式稀缺性 | **辨析**：这一模式在A股确实独特（无同类可比），属于能产生溢价的少数特质。但需注意这种模式的经济价值尚未充分证明（零部件毛利率仅10%） | **+5%** | 全产业链闭环模式降低了交易成本、提升了良品率和交付确定性，长期应享有稀缺性溢价 |
| **内部：治理透明度** | 深度绑定奇瑞、股权关系复杂（芜湖宏博、奇瑞科技为股东） | **辨析**：关联交易占比高（79.8%），治理层面存在信息不对称风险。这不是简单的"平庸关联"，而是可能引发少数股东利益担忧的实质风险 | **-5%** | 高关联交易占比增加了治理折价，市场给予15-20%的"关联方折价"是合理的 |
| **内部：资本配置效率** | 可转债募投项目（ROIC目标较高）、分红稳定 | **辨析**：资本配置中规中矩，既无明显浪费也无出色回报，不属于改变估值中枢的因子 | **0%** | 中性。ROIC约15%处于行业中上水平，但不能构成溢价 |
| **宏观：政策与战略对齐** | "十五五"智能制造、轻量化、国产替代均属政策鼓励方向 | **辨析**：政策利好确实存在，但模具/装备行业历史上政策利好并未转化为定价权（行业高度分散、非标属性制约）。属于"平庸的政策关联"，对估值影响力为零 | **0%** | 政策利好已在行业Beta中体现，对公司Alpha无额外贡献 |
| **宏观：地缘与外部约束** | 出口占比仅7.5%，中东冲突推高铝价是双刃剑 | **辨析**：出口占比低意味着公司不受地缘直接冲击（正面），但铝价上涨带来的成本压力可能持续1-2年，已在利润预测（E）中部分体现，但铝价的不确定性应计入估值折价 | **-3%** | 铝作为公司最大成本项，其价格受到中东地缘政治的不可预测影响。这种"无法定价的宏观噪音"应从PE中扣除 |
| **市场：流动性与风格** | 自由流通盘约2.09亿股全流通、市值65.66亿属中小市值、近期机构增仓 | **辨析**：中小市值在当前市场风格下不被青睐（资金偏向大盘红利/价值），但公司在汽车零部件赛道中属于成长属性，风格契合度一般。不能产生可持续溢价 | **0%** | 风格不构成估值溢价或折价，属于市场中性因子 |
| **综合调节系数** | | | **-3%** | |

**修正目标市值** = 94亿 × (1 + 0%) = **91.2亿**

**修正后隐含PE** = 91.2亿 / 4.7亿 = **19.4x**

## 三、逻辑纯净性校验

### 3.1 "进E还是进PE"的区分

| 因子 | 归入E（利润） | 归入PE（估值倍数） | 理由 |
| :--- | :---: | :---: | :--- |
| 零部件毛利率从10.45%→10% | ✅ | — | 可量化的成本变化 |
| 铝价上涨成本冲击 | ✅ | 部分 | 可量化的入E，不确定性部分入PE（-3%） |
| 客户集中度风险 | — | ✅ | 无法量化，影响信心 |
| 装备+零部件协同 | 部分 | ✅ | 成本节省入E，模式稀缺性入PE |
| 治理关联交易折价 | — | ✅（-5%） | 非量化治理风险 |

### 3.2 锚点回测

修正后PE 19.4x vs 近3年PE中位数24.1x——仍低于历史中位数，但高于当前P10水平（16.4x）。这一折价反映了零部件毛利率下行和奇瑞集中度的结构性风险，具有商业合理性。

## 四、三态逻辑加权目标价

| 情景 | 概率 | 2026E EPS | 合理PE | 目标价 | 市值 |
| :--- | :---: | :---: | :---: | :---: | :---: |
| 保守 | 25% | 1.82元 | 17x | **30.94元** | 64.7亿 |
| 基准 | 60% | 2.25元 | 19.5x | **43.88元** | 91.9亿 |
| 乐观 | 15% | 2.68元 | 22x | **58.96元** | 123.4亿 |
| **加权** | **100%** | — | — | **41.53元** | **86.9亿** |

**当前价格（不复权）：31.37元**

**盈亏比计算**：
- 上行空间：43.88 - 31.37 = 12.51元（+39.9%）
- 下行风险：31.37 - 30.94 = 0.43元（-1.4%）
- **盈亏比 = 12.51 / 0.43 = 29.1:1**

高盈亏比主要源于当前价格已接近保守情景目标价，下行空间极为有限（主要是估值已处于历史最低区间）。

## 五、逻辑证伪与极限压力测试

### 5.1 黑天鹅场景模拟

**"一击毙命"事件**：奇瑞汽车因重大质量/安全事故导致品牌声誉崩塌，销量大幅下滑60%以上，且瑞鹄无法在短期内找到替代客户。

在此场景下：
- 零部件业务营收可能归零（奇瑞独家配套）
- 装备业务营收可能萎缩40-50%（奇瑞占比50%+，其他客户部分切换需时间）
- 公司可能陷入亏损

**极限支撑位审计（"地狱价格"）**：
- 最坏情景净利润：-1亿（考虑折旧和固定成本刚性）
- 净资产：25.77亿（2026Q1）
- **清算价值支撑位**：25.77亿 / 2.09亿股 = **12.33元**（PB=1x）
- 考虑极端恐慌，股价可能跌至PB 0.8x = **9.86元**

### 5.2 逻辑证伪警报

- 若奇瑞汽车市占率连续两个季度下滑超2%→警惕客户风险
- 若零部件毛利率连续两个季度低于8%→盈利模式证伪
- 若装备在手订单同比下降超15%→增长逻辑证伪
