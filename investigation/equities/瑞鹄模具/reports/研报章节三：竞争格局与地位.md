## 一、竞争格局全景扫描

瑞鹄模具横跨**汽车模具/装备**和**轻量化零部件**两大赛道，面对的竞争对手群体和行为逻辑截然不同。以下分赛道展开分析。

### 1.1 赛道一：汽车模具/装备

#### 行业集中度

汽车冲压模具行业高度定制化（非标产品），天然具有低集中度特征。2022年数据显示，中国汽车冲压模具行业CR5合计仅占6.36%，天汽模以2.75%的市场份额位列第一（按模具业务口径）。全国汽车模具制造企业约300家，但规模以上企业仅占15%——行业呈现"大而不强、玩家林立"的格局。

### 1.2 赛道二：轻量化零部件

#### 竞争格局

铝合金压铸行业是"军备竞赛"式竞争。参与者分为三个层级：

**第一梯队（超大型压铸，已量产）：**
- **文灿股份**：行业内最早布局，9000T压铸机领先，2025年营收约61亿（含百炼集团），净利约0.14元/股处于亏损边缘
- **拓普集团**：底盘轻量化龙头，7200T*6台，深度绑定比亚迪+吉利+新势力，2025年营收约198亿，净利约15.77亿
- **广东鸿图**：6800T→12000T布局，营收约80亿

**第二梯队（快速追赶，部分量产）：**
- **旭升集团**：特斯拉深度绑定，2025年营收约14.65亿，净利约0.32元/股
- **爱柯迪**：中小件精密压铸龙头，6100T*2+8400T*2，净利约6亿+
- **瑞鹄模具**：10.72亿零部件收入，一体压铸+冲焊并行，处于放量初期

**第三梯队（新进入者/Laggards）：**
- 多利科技、泉峰汽车、嵘泰股份等，压铸产能布局中，尚未形成规模效应

#### 瑞鹄的差异化竞争力

| 维度 | 文灿/旭升等纯压铸厂 | 瑞鹄模具 | 差异化分析 |
| :--- | :--- | :--- | :--- |
| **模具能力** | 需外购大型模具（1000-2000万/套） | 自研自制，成本降30%+ | 核心成本优势 |
| **良品率** | 行业80-93% | 行业顶尖（自研模具+工艺协同） | 质量+成本双重优势 |
| **装备+零部件协同** | 无 | 独有"模具→零部件"垂直闭环 | 难以复制的护城河 |
| **客户绑定** | 依赖单一/少数主机厂 | 奇瑞深度绑定+其他自主品牌拓展中 | 客戶忠诚度高 |
| **盈利模式** | 赚加工费，靠规模 | 赚装备+零部件双重利润 | 抗周期性强 |

**核心护城河识别**："自研模具+自制零部件"的垂直整合能力是瑞鹄在铝合金压铸赛道中独特且难以被复制的竞争壁垒。纯压铸厂（如文灿、旭升）需要外购模具，单套超大型模具价值1000-2000万元，且模具性能直接影响良品率和生产效率。瑞鹄凭借二十余年覆盖件模具的技术积累，将模具设计与零部件工艺深度融合，形成良品率、成本和交付周期的三重优势。

## 二、护城河深度评估

### 2.1 成本侧护城河（★★★★☆）

| 护城河要素 | 强度 | 证据 |
| :--- | :--- | :--- |
| **模具自研降本** | 高 | 自研模具使零部件成本降低30%+，纯压铸厂难以短期内建立模具能力 |
| **规模效应** | 中 | 装备业务32.92亿营收已经越过盈亏临界点，但相较拓普等仍差距较大 |
| **工艺Know-How** | 高 | 累计为340+款车型开发模具、200+条焊装产线，数据库积累是时间壁垒 |
| **原材料采购** | 低 | 钢铁/铝价被动接受，可转债募投项目投产后关键材料自制率提升，部分对冲 |

### 2.2 品牌/客户侧护城河（★★★☆☆）

- **正面**：国家企业技术中心、国家重点新产品、AAA级两化融合、"中国汽车覆盖件模具重点骨干企业"——技术品牌壁垒深厚
- **负面**：客户集中于奇瑞系（50.49%），尚未建立广泛的客户多元化品牌认知。而天汽模客户覆盖奔驰、大众、蔚来、路虎等全球车企，品牌国际化程度更高

### 2.3 转换成本护城河（★★★★☆）

- **装备业务**：白车身模具+焊装线是整车制造的核心工装，供应商更换成本极高（重新磨合6-12个月），客户粘性强
- **零部件业务**：定点开发周期长（12-18个月），一旦进入配套体系即形成长期锁定

### 2.4 专利/技术护城河（★★★★☆）

- 主持制定多项行业标准
- 中国机械工业科学技术奖一等奖
- AI技术应用（DeepSeek大模型"瑞鹄智汇库"）——行业先行者
- 100%再生铝免热材料开发成功——技术差异化
- 智能协作机器人联合安川电机开发——前瞻布局

## 三、产业链定价权分析

### 3.1 对上游的议价能力

2025年应付账款周转天数约184.85天——公司对上游供应商具有显著的占款能力，这是装备行业龙头企业的典型特征。装备业务采用"以销定产、以产订购"模式，在原材料采购上具有一定议价权（大宗材料通过框架协议锁定）。

零部件业务中公司正推进"关键材料自制率提升"（可转债募投项目覆盖模具铸件自制），将进一步增强对上游的掌控力。

### 3.2 对下游的议价能力——核心软肋

| 客户 | 销售内容 | 议价能力判断 |
| :--- | :--- | :--- |
| **奇瑞汽车（50.49%）** | 装备+零部件全品类 | ⚠️ 极度弱势。单一客户占比过高，定价权在客户手中。关联交易定价原则为"市场公允价"，但实际谈判中议价筹码有限 |
| **其他自主品牌** | 装备为主 | 中等。技术+认证壁垒有一定议价空间 |
| **海外客户** | 装备 | 较强。出口毛利率42.98%，显著高于国内22.63%，说明海外客户对价格敏感度更低 |

**关键数据验证**：出口业务毛利率42.98% vs 国内22.63%，差距高达20.35个百分点——这雄辩地证明了公司在国内市场（尤其对奇瑞）的议价能力存在结构性短板。

### 3.3 合同负债与营运资金——正面信号

2025年末合同负债18.64亿（预收款），远超应收账款6.23亿。这意味着公司在产业链中的资金占用模式是"先收钱、后交货"，营运资金效率优秀。但需注意合同负债中相当部分来自奇瑞的预付款，这更多反映的是双方的关联信任关系，而非市场化的议价权。

## 四、外部数据交叉验证

### 4.1 真实订单验证

- **在手订单**：47.72亿（截至2025年末），同比增长23.76%，为历史最高水平
- **关联交易预计**：2026年向奇瑞系销售合计26.26亿（模检具产线10.38亿+零部件9.7亿+成飞瑞鹄5.5亿等），较2025年实际发生额（约19.35亿，1-11月口径推算）增长约+35.7%
- **奇瑞自身验证**：奇瑞汽车2025年销量和市占率快速提升，新车型密集投放（智界R7热销等），对瑞鹄的需求增长是实质性的而非虚假的

### 4.2 供应链生态穿透

- **下游（奇瑞）侧面验证**：奇瑞子公司芜湖奇瑞科技是瑞鹄股东之一（股权绑定），双方合作深度远超普通供应关系。在投资者互动中，瑞鹄确认"为奇瑞多款车型提供一体化压铸、冲压、焊接等零部件"，配套车型包括智界系列
- **上游设备验证**：可转债募投项目"大型精密覆盖件模具关键材料智能增材制造项目"投产后将实现模具铸件自制，侧面说明公司正通过纵向整合强化产业链控制力

### 4.3 技术代差实证

- 公司开发的"汽车模具三维实体CAD/CAE/CAM集成一体化技术"经省级科技成果鉴定，综合技术水平达到国内领先
- DeepSeek大模型"瑞鹄智汇库"储备超10万条技术知识点，检索准确率超93%——属于行业内的AI先行者
- 100%再生铝免热材料已成功搭载多款整车进行耐久路试——若成功量产，在环保合规和成本端均有优势

### 4.4 实物产能校验

- 覆盖件模具智能工厂：2023年启动建设，2025年7月结项，已全面投产（扩产信号）
- 新能源汽车轻量化车身部件项目：一期已投产，二期预计2026Q2投产
- 三个可转债募投项目累计投资6.36亿，产能扩张实物进展明确

## 五、模块一&二核心结论在本模块的验证

### 从模块一继承的核心关注

1. **零部件业务毛利率仅10.45%且下滑** → 本模块揭示：行业普遍面临毛利率压缩（文灿、旭升毛利率从25-30%降至15-20%），瑞鹄凭借模具自研获得成本优势，但10.45%的水平在行业中确实偏低（纯压铸同行约15-18%），与模块二判断的"行业供给过剩→毛利率中枢下移"一致

2. **奇瑞客户集中度过高** → 本模块量化：公司2026年向奇瑞系关联交易预计26.26亿，占2025年营收比例高达79.8%（含部分成飞瑞鹄销售）。虽然6.00亿来自奇瑞控股（非合并范围内），但与奇瑞系的深度绑定是双刃剑

3. **装备业务毛利率稳定（30.03%）** → 本模块验证：装备业务的"非标定制+长周期交付+技术认证壁垒"使其在下游价格战中保持了独特的抗压能力，是公司利润的压舱石

### 从模块二继承的核心关注

1. **零部件供给侧过剩** → 本模块通过竞争对手产能扩张数据验证：文灿、拓普、旭升、爱柯迪等均在大幅扩产，行业产能正在经历"大跃进"，将在2026-2027年集中释放，供需平衡将进一步向过剩倾斜
2. **装备业务紧平衡** → 本模块确认：覆盖件模具行业有效产能增长有序（定制化壁垒、技术人才稀缺），瑞鹄47.72亿在手订单为其提供了约2年的收入能见度
