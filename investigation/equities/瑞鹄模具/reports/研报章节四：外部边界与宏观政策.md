# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月24日

---

## 一、宏观环境变量

### 1.1 利率与折现率环境

当前中国10年期国债收益率在1.6-1.8%区间低位运行，无风险利率处于历史低位。这对于DCF估值意味着更低的折现率、更高的终值权重。对于瑞鹄模具（PE约15.87x），低利率环境提供了估值支撑。

2026年美联储6月议息会议释放强烈鹰派信号，联邦基金利率维持3.50-3.75%，点阵图显示加息前置倾向。美元指数突破101（逾一年新高），对有色金属整体构成系统性压制——这也部分解释了近期铝价的下跌。若下半年美联储开启降息，将有利于全球风险资产估值修复。

### 1.2 原材料成本：铝价出现重大边际改善

**铝价分析（关键更新）**：自2026年6月8日上次研究以来，铝价经历了显著的"地缘溢价退潮"。

- **LME铝价**：从6月初的~3700美元/吨峰值（逼近2022年历史高位），大幅回落至6月23日的3,251.5美元/吨，跌幅约12%。盘中一度触及3,225.5美元/吨，为3月26日以来最低
- **下跌驱动**：美伊达成和平协议，霍尔木兹海峡恢复通航预期升温，前期"海湾电解铝永久停产"的极端地缘溢价快速出清
- **LME库存**：31.17万吨，仍处历史低位；现货从升水104美元大幅回落至升水4.5美元，back结构收窄
- **沪铝**：从6月初的24,200-24,800元/吨高位区间，回落至23,520元/吨（6月23日），跌幅约3.2%。长江现货A00铝锭报23,800元/吨
- **国内库存**：铝锭社会库存124.2万吨（6月22日），周度去库4.8万吨，淡季去库拐点确立，高库存压力边际缓解
- **中期展望**：海湾铝厂产能恢复周期需6-18个月（Alba、EGA、Qatalum当前仅以30-60%开工率运营），短期内LME低库存仍构成支撑。机构预计LME铝价回调后在2,850-2,950美元/吨区间企稳，7-9月运行于2,900-3,050美元/吨。但2027-2028年印尼、印度、沙特在建产能的远期过剩预期仍是中长期压制因素

**对瑞鹄模具的影响——边际显著改善**：

| 影响方向 | 机制 | 强度变化 |
| :--- | :--- | :--- |
| **成本端（利空→减弱）** | 零部件业务直接材料占比26.17%（2025），铝价回落12%直接缓解成本压力 | ⚠️ 从"高"降至"中高" |
| **需求端（利好）** | 铝价下跌短期可能降低车企轻量化紧迫性，但长期趋势不变 | 中（不变） |
| **零部件毛利率修复** | Q1毛利率-0.9%的核心压力来自铝价上涨10+pct，当前铝价显著回落，若年降谈判顺利，H2毛利率回升至盈亏平衡以上的概率大幅增加 | **正向边际** |

**关键判断**：铝价从3700→3250的回落是本次审计周期内对瑞鹄基本面最重要的边际改善。此前零部件毛利率转负的风险因子正在系统性减弱。但需注意沪铝回撤幅度（-3.2%）远小于LME（-12%），国内成本改善幅度较国际有限，且23000+元/吨仍处历史偏高水平。

钢铁价格方面，国内钢材价格相对平稳，对装备业务成本影响可控。

### 1.3 国内汽车需求：总量深度收缩，新能源渗透率继续攀升

2026年1-5月国内乘用车市场持续深度承压：

- 1-5月乘用车累计零售709.9万辆，同比-19.5%；5月零售151万辆，同比-22.1%
- 乘联会将2026年全年零售增速预期从-1%大幅下调至-11%
- 5月新能源零售渗透率达62.9%（创历史新高），6月第一周进一步攀升至66.7%
- 新能源乘用车5月零售95万辆，同比-7.5%；1-5月累计-15.1%
- 燃油车5月零售同比-39%，1-5月燃油车份额仅37.1%
- 5月乘用车零售销量前十车型首次全部被新能源车包揽

**奇瑞集团逆势增长**：
- 5月总销量24.78万辆（含集团口径），同比+20.5%；1-5月累计110万辆，同比+7.2%
- 奇瑞汽车（含五大品牌）5月销售23.19万辆，同比+20.7%；1-5月累计103.45万辆，同比+7.5%
- 新能源5月10.03万辆，同比+58.8%，连续两月破10万
- 出口5月18.19万辆，同比+80.5%，连续三月刷新中国汽车单月出口纪录，占集团总销量的73.4%
- 智界品牌5月销量3,428辆，同比-31.4%；1-5月累计1.38万辆，同比-67.5%，仅完成全年目标4.6%

需求端总量收缩对瑞鹄的影响呈现结构性分化：装备业务（依赖新车型开发周期和奇瑞整体扩张）受冲击相对有限；零部件业务（依赖配套车型销量）受智界销量崩塌的冲击仍在持续，但智界V9（6月6日开启交付，大定超1.8万台）和EHV车型（5月小批量供件）正在形成新的增量来源。

### 1.4 汇率损益

- 公司出口销售占比7.51%（2025年），外销毛利率42.98%
- 美元指数走强至101+、人民币偏弱有利于出口业务毛利率改善
- 海外订单占比已达44-45%（2026Q1），汇率弹性将进一步增大
- 公司目标使海外业务收入和订单量占总体50%以上

## 二、监管与产业政策

### 2.1 新能源汽车购置税减半（2026-2027）

2026年起新能源汽车购置税从免征调整为减半征收（5%），单车上限1.5万元。标志行业从"政策驱动"向"市场驱动"的实质性切换。短期内可能压制部分价格敏感型消费者的购车意愿，但中高端车型（16.7万以上）影响有限。

### 2.2 以旧换新政策延续

2026年继续实施汽车报废更新和置换更新补贴，从"定额补贴"调整为"按车价比例补贴"。新能源乘用车报废更新补贴车价12%（上限2万），置换更新补贴8%（上限1.5万）。

### 2.3 "十五五"规划利好

- 模具行业作为高端装备列入重点发展领域
- 智能制造、工业机器人国产替代获得政策重点扶持
- 轻量化材料（铝镁合金）被列为战略性新兴产业

### 2.4 欧盟CBAM碳关税

2027年起欧盟要求进口产品再生铝比例不低于30%且碳足迹需符合PEF标准。瑞鹄已研发100%再生铝免热材料并通过量产验证，在碳合规方面具有先发优势。

## 三、地缘政治风险

### 3.1 中东冲突与铝供应链——重大边际改善

2026年2月爆发的中东冲突曾导致霍尔木兹海峡航运受阻、全球铝供应出现结构性缺口。**6月美伊达成和平协议是本次审计周期内最重要的地缘变量变化**：霍尔木兹海峡恢复通航预期升温，LME铝价地缘溢价快速出清（从升水104美元降至升水4.5美元）。但海湾铝厂（Alba、EGA、Qatalum）当前仅以30-60%开工率运营，全面恢复需6-18个月，全球铝供应偏紧格局短期不会根本扭转。

对瑞鹄而言：成本端压力边际缓解，但行业层面上中小压铸厂因铝价急涨退出市场的"供给出清"效应对公司利好（竞争减少）。

### 3.2 中美贸易摩擦

当前美国对中国汽车零部件征收25%关税，但对瑞鹄直接影响有限（出口占比仅7.51%，且主要面向非美市场）。若贸易摩擦进一步升级至欧洲、东南亚市场，需重新评估。

### 3.3 全球供应链重构

"China+1"战略驱动车企在东南亚、拉美建厂，瑞鹄作为设备供应商正在间接受益——奇瑞海外扩张（5月出口18.19万辆，占销量73%）为瑞鹄提供了伴随"出海"的增量订单。海外订单占比已达44-45%。

---

## 四、结论：政策环境中性偏利好，铝价是当前最大边际改善变量

1. **铝价显著回落**（LME -12%/沪铝 -3.2%）是本次审计的最大边际利好。零部件毛利率修复的宏观条件正在改善，H2回升至盈亏平衡以上的概率显著增加
2. **购置税减半**是短期利空（影响终端消费），但以旧换新补贴延续提供了托底
3. **奇瑞出口爆发式增长**（5月+80.5%）为瑞鹄装备业务的海外订单提供强劲支撑
4. **国产替代政策**是长期利好（智能制造、机器人国产化）
5. **碳关税**对瑞鹄是差异化优势（再生铝技术领先）
