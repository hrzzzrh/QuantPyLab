# 研报章节六：估值定价分析

**研究日期**：2026年06月26日

---

## 一、价值锚定：适配性估值方法选择

伊利股份具有稳定盈利、高分红、低增长的成熟消费品龙头特征，适用以下估值路径：

### 方法一：PE估值（主锚，权重50%）

**理由**：盈利稳定且可预测，PE是最直观的消费品估值锚。

| 基准 | PE倍数 | 对应市值（亿元） | 对应股价（元） | 逻辑 |
| :--- | :--- | :--- | :--- | :--- |
| 历史中位（2019-2026） | 23.93x | — | — | 仅作参考，A股估值中枢下移 |
| 全球乳业龙头平均 | 18-22x | — | — | 雀巢~20x，达能~18x |
| **基准PE** | **18x** | **2,196** | **34.73** | 盈利确定性溢价+消费龙头溢价 |
| 保守PE | 15x | 1,830 | 28.93 | 接近当前PE，市场悲观延续 |
| 乐观PE | 22x | 2,684 | 42.44 | 消费复苏+龙头发掘 |

**PE定价的审慎思考**：
- 当前PE仅13.27x，处于近5年<5%分位，显著低于历史中位数23.93x
- 但超低PE的原因并非市场"误判"——2024年业绩暴雷（Q4亏损）、液态奶持续萎缩、人口结构担忧均是合理折价因素
- 随着2025年业绩回升和行业拐点确认，PE有望向15-18x修复，但要回到23x+的历史中位需要消费信心和人口预期的根本性逆转
- **给予18x PE的基准估值**：对应2026年EPS 1.93元，即35元目标价（不复权市价）

### 方法二：PB-ROE模型（交叉验证，权重25%）

**理由**：ROE稳定在20%+，PB-ROE框架适用。

- 2025年ROE：20.87%，2026E ROE：~19.5%
- 可持续ROE估值：`(ROE - g) / (r - g) = (19.5% - 2%) / (8% - 2%) = 2.92x PB`
- 当前BVPS：9.42元
- 理论合理股价：9.42 × 2.92 = **27.51元**

但PB-ROE模型对ROE假设高度敏感，如果ROE持续维持在20%以上（伊利历史上ROE波动区间15-21%），合理PB可上升至3.0-3.3x。考虑到股本回购注销持续进行，BVPS有上行趋势。

### 方法三：股息贴现（DDM，权重25%）

**理由**：承诺2025-2027年分红比例≥75%，适合DDM估值。

- 2025年每股分红：1.38元（0.90年末+0.48中期）
- 2026E EPS × 75%：1.93 × 0.75 = 1.45元
- 折现率假设8%，永续增长率2%
- 内在价值：1.45 / (8% - 2%) = **24.17元**

DDM得出的价值偏低（24.17元），主要是因为伊利虽然分红慷慨，但股息增长率有限（公司处于成熟期，利润增速仅中个位数）。

### 综合估值锚定

| 估值方法 | 权重 | 对应股价 | 加权贡献 |
| :--- | :--- | :--- | :--- |
| PE估值（18x 2026E EPS） | 50% | 34.73元 | 17.37 |
| PB-ROE模型 | 25% | 27.51元 | 6.88 |
| DDM股息贴现 | 25% | 24.17元 | 6.04 |
| **综合基准市值/股价** | — | **1,915亿 / 30.29元** | — |

## 二、估值全变量对冲审计矩阵

| 审计维度 | 核心审计因子 | 审慎思考（估值影响力过滤） | 调节幅度 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质** | 战略定力+业务多元化的抗风险能力 | 伊利多元化布局（液态奶→奶粉→冷饮→深加工）在行业低谷期已验证其抗风险价值，这是普通乳企不具备的稀缺能力 → **溢价** | **+5%** | 业务结构多元化降低了盈利波动性，增加了长期现金流的可预见性，可给予确定性溢价 |
| **内部：治理与资本配置** | 75%+分红+回购注销+管理层稳定 | 高分红承诺和回购注销是A股罕见的"股东友好"行为，潘刚团队20年稳定经营→ **溢价** | **+3%** | 高分红相当于为估值提供了一个"股息底"，降低了下行风险 |
| **宏观：政策与战略对齐** | 灭菌乳国标+反补贴税+育儿补贴+消费刺激 | 三大利好同时落地，但多数政策的实际业绩弹性有限（灭菌乳国标行业已基本符合、育儿补贴金额温和）→ **微幅溢价** | **+2%** | 政策利好更偏向"信心催化"而非"实质增量"，给予轻微溢价而非大幅上调 |
| **宏观：地缘与外部约束** | 供应链国际化风险 | 伊利海外营收占比<5%，地缘风险敞口小。但进口关键原料（乳清蛋白等）若受限可能影响成本 → **中性** | **0%** | 风险敞口极小，不足以构成折价因子 |
| **市场：流动性与风格** | 机构持仓集中度+红利风格契合度 | 当前市场风格偏向红利/价值，伊利5.69%股息率+低PE+稳定现金流高度契合 → **溢价** | **+3%** | 低利率时代红利资产享受流动性溢价，短期可能维持，但中期风格轮动是必然 |
| **综合调节系数** | Σ = +13% | — | | |
| **修正目标市值** | 1,915亿 × (1+13%) = **2,164亿** | | | |
| **修正目标股价** | = **34.21元**（不复权市价） | | | |

| 情景 | 概率 | 2026E EPS | 目标PE | 目标股价（元） | 市值（亿元） |

**当前价格**：24.27元（不复权市价，2026年6月26日收盘）

> **除权说明**：2026年6月4日为2025年度分红股权登记日（10派9元，含税），6月5日除权除息。当日收盘价反映除权后的新价格水平。6月26日收盘价24.27元较6月4日除权前收盘价26.56元下跌-8.6%，其中除权贡献约-3.4%（0.90/26.56），超额下跌约-5.2%。

**盈亏比计算**：

- **盈亏比** = 9.94 / 1.03 = **9.65 : 1** ✅

盈亏比较6月8日的4.15:1大幅改善，主因是股价进一步下跌扩大了上行空间，同时保守情景的目标价仍提供充分的下行保护。

## 五、极限压力测试：逻辑证伪场景

**黑天鹅事件定义**："三重打击同时发生"——(1) 中国爆发重大乳制品食品安全事故（类似于2008年三聚氰胺级别的行业信任危机）；(2) 伊利作为行业龙头首当其冲，营收暴跌50%+；(3) 经销商体系崩溃，大规模退货和坏账。

**极端情景估值**：
- 营收：1,210亿 → 600亿（-50%）
- 净利润：清零或巨亏（品牌重建需3-5年）
- 清算价值：净资产595.88亿 - 商誉21.73亿 - 无形资产46.77亿 ≈ 527亿
- 每股清算价值：527亿 / 63.25亿股 ≈ **8.34元**
- **地狱价格（净资产折价30%）**：8.34 × 0.7 ≈ **5.84元**

*概率评估：发生概率 < 1%。2008年后中国乳制品安全监管体系全面升级，伊利MSCI ESG评级AA级，全产业链溯源体系完善。但此类事件一旦发生，杀伤力是毁灭性的。*

## 六、估值结论

伊利股份当前24.27元的价格，对应2026E PE仅12.6x（24.27/1.93），处于历史极端低位。综合PE估值、PB-ROE模型和DDM模型，考虑全变量对冲审计修正后，**基准目标股价34.21元（不复权市价），对应市值约2,164亿元，上行空间约41.0%**。保守/基准/乐观三态盈亏比9.65:1，风险收益比极佳。近期除权（每股0.90元）后股价的进一步下跌扩大了安全边际。
