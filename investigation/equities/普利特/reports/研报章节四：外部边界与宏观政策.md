# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月13日

---

## 一、宏观环境变量

### 1.1 原油价格——改性塑料成本的"达摩克利斯之剑"

**当前状态**（2026年6月）：
- WTI原油：约92美元/桶（较年初60美元+53%）
- 布伦特原油：约100美元/桶
- 核心驱动：中东地缘冲突导致霍尔木兹海峡通行中断，影响约2000万桶/日原油及产品出口（占全球海运贸易25%）

**2026年油价预期**（多空分歧剧烈）：

| 机构 | 2026年均价预测 | 逻辑 |
|:---|:---|:---|
| 摩根大通 | >100美元/桶（至年末） | 海峡中断供给冲击超1956年苏伊士危机 |
| 国泰海通 | 偏强震荡 | 全年供应缺口扩大，高油价粘性 |
| 高盛 | 维持高位 | Q4锂+油均偏紧 |
| 北京理工大学 | 53-63美元（布伦特） | 供需宽松格局（年初预测，已失效） |
| 申万宏源 | 80-120美元区间 | 去库+中东复产缓慢 |

**对普利特的影响机制**：

```
油价上涨10% → PP/ABS单体上涨15-20% → 改性料成本上涨8-12% → 毛利率压缩3-5pct
```

定量测算：普利特改性材料业务年营业成本约54亿元（2025年），其中原材料成本占70%即~38亿元。若油价中枢从60美元升至90美元（+50%），原料成本可能上升15-20%，即增加5.7-7.6亿元，对应毛利率压缩6-8个百分点（若无涨价传导）。

**关键不确定性**：6月美伊谈判若达成协议，霍尔木兹海峡重新开放，油价可能从当前92美元回落至75-85美元区间，对下半年毛利率形成正向修复。反之，若冲突持续，油价维持100美元+，Q3-Q4毛利率将继续承压。

### 1.2 碳酸锂价格——电池业务的双刃剑

**当前状态**（2026年6月）：
- 电池级碳酸锂：15.5-17.5万元/吨
- 2026Q1曾冲高至17.9万元，目前回落但仍在高位
- 从2025年6月低点5.99万元已上涨约175%

**2026年预期**：

| 情景 | 价格区间 | 触发条件 |
|:---|:---|:---|
| 基准 | 12-18万元/吨 | 储能需求高增+津巴布韦/江西供给扰动 |
| 乐观（对锂盐企业） | 18-25万元/吨 | Q4补库+供给缺口扩大 |
| 悲观 | 10-15万元/吨 | 澳矿复产+津巴布韦恢复出口+需求不及预期 |

**对普利特的三重影响**：

1. **锂电业务（负面）**：碳酸锂占电芯成本约40-50%。碳酸锂从6万涨至15万（+150%），电芯成本增加约35-50%。但向下游传导涨价需3-6个月（合同约束），Q1-Q2 2026的电池毛利率可能跌至盈亏平衡线以下。

2. **钠电业务（正面）**：碳酸锂越贵，钠电池的相对成本优势越突出。目前钠电芯成本0.45-0.50元/Wh，已与LFP（0.48-0.55元/Wh）基本持平。若碳酸锂涨至18万+，钠电的成本优势将明确凸显，有利于加速下游客户切换。

3. **改性材料业务（中性）**：碳酸锂价格通过新能源汽车销量间接影响改性材料需求。当前锂价尚未抑制NEV销量（Q1 2026 NEV销量仍+15%+），但若锂价突破20万并维持，可能抑制中低端电动车的经济性。

### 1.3 汇率——对出海产能的隐性影响

普利特有马来西亚、墨西哥、德国、美国等海外基地。人民币汇率波动影响：
- 人民币贬值（如2025年趋势）：利好出口竞争力，海外收入折算人民币增加
- 人民币升值：压缩海外基地利润

2026年人民币在7.1-7.3区间波动，总体稳定，汇率风险可控。

## 二、地缘政治与贸易壁垒

### 2.1 中美贸易战：电池出口的"定向狙击"

**关税升级路径**（2025-2026年）：

| 时间节点 | 事件 | 储能电池关税 |
|:---|:---|:---|
| 2025年初 | 基础关税 | 10.9% |
| 2025年4月 | 对等关税125% + 芬太尼20% | **114.9%** |
| 2026年1月 | 301条款储能电池关税生效（25%） | **139.9%** |
| 2026年4月 | 再加征50% | **173%** |

**对普利特的具体影响**：

1. **直接出口**：海四达对美国直接出口储能电池的比例不详，但整个行业对美出口占比已从25%（2024年）降至9.7%（2026年1-2月）。高关税已基本关闭美国市场大门。

2. **转口贸易**：马来西亚基地（2.5GWh，2026年3月投产）成为关键布局——美国对马来西亚仅加征24%关税，远低于对中国的173%。通过马来西亚转口至美国，关税成本可大幅降低至可接受水平。

3. **市场替代**：欧盟已取代美国成为中国锂电池最大出口市场（占43.1%），2026年1-2月对欧出口+69%。海四达可通过马来西亚基地和国内基地同时供应欧洲市场。

4. **改性材料出口**：汽车用改性材料未被单独针对，但汽车零部件（含材料）出口若受到整车关税波及，间接受影响。目前普利特改性材料以内销为主，出口敞口较小。

### 2.2 地缘政治：中东冲突的"蝴蝶效应"

2026年Q1-Q2的中东冲突对普利特构成了一系列间接但深刻的影响链：

```
中东冲突 → 霍尔木兹海峡中断 → 原油供给缺口
    ├─→ 油价飙升 → 改性材料原料涨价 → 毛利率压缩（直接利空）
    ├─→ 能源安全焦虑 → 欧洲加速新能源部署 → 储能需求增加（间接受益）
    ├─→ 石脑油涨价 → 化工品全线上涨 → PP/ABS成本全面抬升（直接利空）
    └─→ 航运保险费飙升 → 全球贸易成本增加 → 出口竞争力受损
```

**净影响评估**：对普利特而言，中东冲突的利空因素（原料涨价）远大于利好因素（储能需求）。公司是原料的消费者而非生产者，高油价周期天然不利。

### 2.3 China+1与供应链重构

"去中国化"的供应链转移趋势对普利特而言是**双面效应**：

- **威胁**：若汽车主机厂（宝马、奔驰等）要求供应链本地化（如欧洲本地采购改性材料），普利特可能失去部分出口订单。
- **机遇**：普利特已在德国（Pret Europe GmbH）和美国（PRET Advanced Materials, LLC）布局产能，具备跟随客户本地化生产的能力。马来西亚、墨西哥等基地进一步分散了地缘风险。

## 三、监管与产业政策

### 3.1 "十五五"新材料规划——LCP的国家级背书

"十五五"规划纲要（2026年3月发布）将新材料列为**战略性新兴产业的核心方向**，明确提出：

> "聚焦战略必争领域和产业链关键薄弱环节，采取超常规措施，全链条推动……关键材料……攻关取得决定性进展。"

**LCP直接受益的政策条款**：

| 政策方向 | 具体措施 | 对普利特LCP的影响 |
|:---|:---|:---|
| 关键材料自主可控 | 国产替代率目标80%+ | LCP作为"卡脖子"材料，享受采购优先 |
| 首批次应用保险补偿 | 国产新材料首批次应用享受政策保险 | 降低下游客户试用风险，加速验证 |
| 重大工程优先采购 | 政府采购和重大工程支持国产创新产品 | 5G基站、卫星等项目可能指定国产LCP |
| 研发专项支持 | 100个研发专项，每个2000-8000万元 | LCP产业化可能获得直接资金支持 |
| 时间表 | 2027-2028突破关键技术，2029装备国产化 | 与普利特LCP放量时间节奏吻合 |

**政策确定性评估**：★★★★★。新材料国产替代是中国"十五五"最明确、最不可逆的政策方向之一，LCP作为被美日垄断的"卡脖子"材料，政策加持持续性极高。

### 3.2 新能源汽车补贴退坡——双刃剑

2026年新能源汽车购置税从0%提高至5%，国补政策存在退坡可能。影响：
- **利空销量端**：可能抑制2026年NEV销量增速（中汽协预计+15.2%，但部分券商预测仅+6-11%）
- **利好格局端**：补贴退坡加速行业洗牌，头部车企（普利特的主要客户比亚迪、吉利等）市占率提升
- **净影响**：若NEV销量增速放缓至个位数，改性材料的需求增量将收窄，产能消化压力增大

### 3.3 新型储能政策——钠电的政策顺风

《新型储能制造业高质量发展行动方案》等政策明确：
- 到2027年累计装机达1.8亿千瓦（180GW）以上
- 培育3-5家生态主导型企业
- 支持钠离子电池等多元化技术路线

普利特（海四达）的钠电池储能业务处于"政策顺风+技术差异"叠加的有利位置。南网30MWh标杆项目的落地也获得了电网侧的政策认可。

### 3.4 环保与安全约束

- 改性塑料行业面临日益严格的VOC排放标准和阻燃剂环保要求
- 电池行业面临安全生产检查和危化品管理趋严
- 这些约束对头部合规企业是利好（淘汰不规范小厂），但合规成本也在上升

## 五、章节结论

**宏观环境对普利特构成"短期显著承压、中长期结构性利好"的不对称格局**：

- **短期（6-12个月）**：高油价和高锂价双重压制毛利率，是Q1 2026业绩低于预期的根本原因。这一压力在霍尔木兹海峡问题解决前难以根本缓解。
- **中期（1-3年）**：十五五新材料政策对LCP国产替代的强力支持、储能政策对钠电池的背书、以及欧盟市场对锂电池的旺盛需求，构成了三重结构性利好。
- **核心矛盾**：原材料成本压力（利空）vs 政策红利释放节奏（利好）的时间错配——利空即刻兑现，利好需要2-3年孕育。这决定了普利特在未来12个月内可能经历"盈利底→政策催化→估值修复"的三阶段演进。

在下一模块的盈利预测中，我们将把这些宏观变量量化为具体的毛利率假设、营收增速预期和利润弹性。
