**研究日期：2026年07月01日**

## 一、价值锚定：适配性方法选择

### 1.1 方法论选择与理由

心脉医疗的商业模式特征：**高毛利（70%+）、高ROIC（13.7%）、轻资产（固定资产仅0.94亿）、稳定净利率（~41%）、确定性的中速增长（15-20% CAGR）**。

| 估值方法 | 适配度 | 理由 | 权重 |
|:---|:---|:---|:---|
| **PE（市盈率）** | ★★★★★ | 盈利稳定且可预测，行业横向可比性强，是最核心的估值锚 | 50% |
| **PEG（成长性估值）** | ★★★★☆ | 公司处于15-20%增速区间，PEG框架可有效判断估值性价比 | 25% |
| **PB-ROE** | ★★★☆☆ | 公司ROE仅14%（被商誉/现金拉低），PB框架参考价值有限 | 10% |
| **DCF** | ★★★☆☆ | 现金流稳定但医疗政策变量多，远期预测精度低 | 15% |
| **PS/EV-EBITDA** | ★★☆☆☆ | 医疗器械公司PS波动大，且公司有息负债近乎为零 | 0% |

**主要估值路径**：**PE（为核心）+ PEG（为辅助）+ PB-ROE（为参考）**

### 1.2 历史估值分位审计（2022年-至今）

基于前序模块0的统计数据：

| 分位 | PE (TTM) | PB | 对应市值（约） |
|:---|:---|:---|:---|
| P1（极低） | 16.45 | 2.50 | ~100亿 |
| P5 | 18.58 | 2.77 | ~113亿 |
| P10 | 20.18 | 2.88 | ~123亿 |
| P25 | 23.60 | 3.07 | ~144亿 |
| P50（中位） | 31.56 | 3.98 | ~193亿 |
| P75 | 35.44 | 7.61 | ~216亿 |

| P95 | 45.79 | 9.76 | ~279亿 |
| **当前（2026.7.1）** | **19.03** | **2.66** | **~112亿** |
| **历史分位** | **6.1%** | **3.2%** | -- |

**解读**：当前PE和PB均处于历史极低分位——PE仅高于6.1%的历史交易日，PB仅高于3.2%。然而，历史分位参考意义需审慎对待：

1. **结构性变化**：公司历史上的高PE（40-57x）对应的是30%+增速的黄金期，当前增速中枢已下移至15-20%，PE中枢理应对应下调
2. **政策环境质变**：2024年以前主动脉支架未受集采/价格治理影响，此后该"政策溢价"已永久性消失
3. **ROE中枢下移**：ROE从26.4%（2023）降至14.3%（2025），拖累PB中枢

**结论**：不能简单以"历史极低分位=低估"来指导买入。需要独立判断当前增速中枢对应的合理PE。

## 二、基准估值定价

### 2.1 PE框架：基于2026E EPS

```
基准PE取值逻辑：
- 可比公司平均PE（先健科技~30x，归创通桥~40x，但两者增速较低/波动大）
- 心脉2026E利润增速：~7%（过渡年低增速），合理PE应为15-22x
- 心脉2027E利润增速：~24%（恢复高增），合理前瞻PE应为20-28x

采用2027E前瞻PE进行定价（动态锚点）：
2027E EPS = 6.13元（Base）
2027E合理PE = 18-25x（对应增速24%，PEG约0.75-1.0x）
→ 2027E目标价（不复权）= 110-153元
→ 折算当前（1年贴现10%）= 100-139元

采用2026E PE（保守）：
2026E EPS = 4.94元
2026E合理PE = 20-26x
→ 2026E目标价 = 99-128元
```

### 2.2 PEG框架

```
2027E归母净利增速 = 24%
合理PEG = 0.8-1.2x（考虑到确定性中等偏高）
→ 2027E PE = 19.2-28.8x
→ 2027E目标价 = 118-177元

综合PE框架：2026E目标价（不复权）= 100-140元
```

### 2.3 PB-ROE参考

```
2026E ROE ≈ 14.5%（净利6.03/净资产~41.6亿）
合理PB = ROE / (COE - g) ≈ 14.5% / (9% - 5%) ≈ 3.6x
2026E BVPS ≈ 34元
→ 合理PB目标价 ≈ 123元
```

### 2.4 基准市值区间

| 方法 | 2026E市值（亿） | 2026E股价（元，不复权） | 2027E市值（亿） | 2027E股价（元） |
|:---|:---|:---|:---|:---|
| PE（Base EPS×18-25x） | 109-151 | 89-124 | 135-187 | 110-153 |
| PEG（合理PEG 0.8-1.2x） | 95-142 | 78-116 | 118-177 | 97-145 |
| PB-ROE（ROE 14.5%） | ~150 | ~123 | -- | -- |
| **综合基准** | **120-140** | **98-115** | **140-170** | **115-139** |

**基准目标市值：130亿（对应2026E PE ~21.5x，股价~107元）**

**当前市值：112亿（股价90.86元），潜在上行空间：~18%**

## 三、核心：估值全变量对冲审计矩阵

### 3.1 审计原则与影响力过滤

在基准市值130亿的基础上，逐维度审计以下因子。**审慎思考**：识别真正能改变估值中枢的"稀有因子"，剔除"平庸的关联"。

### 3.2 全变量审计矩阵

| 审计维度 | 因子挖掘 | 审慎思考（影响力过滤） | 调节建议 | 深度理由 |
|:---|:---|:---|:---:|:---|
| **内部：企业DNA** | Fontus/Hector创新平台→术中支架+多分支支架独占性 | **辨析**：术后支架垄断是真的稀缺性，直接带来定价权和利润增量。但"平台型企业"的提法有夸大——公司本质上是一个主动脉+外周产品组合，不具备真正的平台网络效应 | **+5%** | Fontus在A型夹层市场的独占性将持续5年以上，A型夹层手术量5年CAGR 45%意味着确定性极高的增量利润。这赋予了公司一个"被低估的增长期权" |
| **内部：资本配置** | Lombard并购→首年盈利+全球化通道 | **辨析**：这是已验证的并购整合能力（非空谈），海外渠道对于中国医疗器械企业是稀缺资产。4.65亿商誉可视为品牌+渠道的合理溢价 | **+3%** | Lombard从亏损到盈利的验证，降低了商誉减值概率，同时为长期海外增长提供了可复制的模板。这是"证实过的战略执行力"带来的估值溢价 |
| **内部：治理透明度** | 2024年被医保局公开问询，暴露定价不透明问题；股权激励彰显信心 | **辨析**：被问询是负面信号（治理瑕疵），但公司已整改。股权激励（2024年员工持股计划）提供短期业绩底线 | **-2%** | 定价不透明事件在一定程度上损害了公司在监管层和公众中的信誉。虽然已整改，但"微创系"的分拆上市结构（控股股东香港心脉）本质上存在公司治理复杂性 |
| **宏观：政策对齐** | 国产替代+创新医疗器械绿色通道+集采规则优化 | **辨析**：政策大方向利好国产龙头，但绝大多数医疗器械公司都能享受这些政策红利，因此这不是心脉独有的估值溢价来源。只有能证明"比别人受益更多"的才值得计入 | **0%** | 政策红利是行业性的，心脉作为国产龙头享受最多，但同行（先健、归创）同样受益。这是"涨潮抬升所有船"，不是心脉独有的溢价来源 |
| **宏观：集采风险** | 河北主动脉支架集采待落地→价格最大不确定性 | **辨析**：这是实质性的、公司特异性的利空——主动脉支架占心脉收入的73%，远高于先健的外周为主结构。河北集采的降价幅度直接影响利润中枢 | **-3%** | 冠脉支架国采降价90%+的前车之鉴，虽然主动脉支架因技术壁垒更高不太可能同等幅度降价，但10-25%的降价足以对73%的收入基础产生实质性利润冲击。市场尚未充分定价这一风险（Q1 2026的良好数据可能让投资者放松警惕） |
| **宏观：地缘与供应链** | ePTFE进口依赖+中美贸易摩擦 | **辨析**：ePTFE虽然高端品种依赖进口，但目前不属于出口管制范围；公司国内备货+中科院联合研发已提供部分对冲。这是一个"墙上的裂缝"，但还不到"墙要塌"的程度 | **-1%** | 当前节点，地缘政治对心脉的影响属于"可管理"范畴。但如果中美关系进一步恶化导致医用级材料被纳入出口管制，折价幅度需大幅上调。目前给予-1%作为预防性折价 |
| **市场：流动性与风格** | 当前市场风格偏向红利/低估值/防御 | **辨析**：心脉医疗作为科创板医疗器械成长股，在当前防御性市场风格下不属于资金偏好方向。但风格切换是短期因素，不应改变长期估值中枢 | **0%** | 市场风格波动应在交易层面处理，而非估值层面。任何基于"风格契合"给出的溢价都是泡沫，基于"风格不契"给出的折价都是噪音 |
| **市场：自由流通盘** | 总股本1.23亿，微创医疗控股约45%→自由流通市值约60亿 | **辨析**：自由流通盘偏小（~60亿），机构进出存在流动性折价。同时微创医疗母公司近年经营困难→潜在减持压力 | **-2%** | 微创医疗（0853.HK）近年持续亏损，存在出售心脉医疗股份以回笼资金的动机。2025年微创医疗曾减持心脉部分股份。这一潜在的"减持悬顶"构成了真实的估值压力 |
| **综合调节系数 Σ** | -- | -- | **0%** | 正负因子基本对冲：内部优势（+5%+3%）≈ 宏观/市场压力（-2%-3%-1%-2%） |

**最终：修正目标市值 = 基准市值130亿 × (1+0%) = 130亿，修正目标股价 ≈ 107元**

## 五、逻辑证伪与极限压力测试

### 5.1 "一击毙命"事件定义

**黑天鹅场景**：河北省主动脉支架集采以冠脉支架模式落地——胸/腹主支架降价70-80%，且全国联采联动执行。

在此场景下：
- 主动脉支架收入从9.88亿暴跌至~3亿（量增无法弥补价跌）
- 主动脉GPM从79%降至30-40% → 毛利从7.8亿缩至~1.2亿
- 外周业务（3.6亿）不足以弥补主动脉缺口
- 公司年度净利润可能降至1-2亿（甚至亏损）

### 5.2 极限支撑位审计（地狱价格）

```
极限情景假设：
- 主动脉支架收入归零（极端假设市场崩塌）
- 外周业务维持3.6亿，GPM 50%
- 停止所有研发资本化，费用化冲减利润
- 商誉4.65亿全额减值
- 清算价值 = 现金10.89亿 - 总负债5.54亿 = 5.35亿
- 加上外周业务的持续经营价值（~15亿，按2x PS）
- 极限市值 ≈ 20亿，股价 ≈ 16元（不复权）

但更现实的"地狱价格"（非极端清算）：
- 河北集采大幅降价50%，主动脉收入跌至5亿，GPM降至50%
- 总营收降至8.6亿，净利约2亿
- 市场给予PE 12x → 市值24亿 → 股价 ~20元
```

**地狱价格：~20元（不复权），较当前价格有78%的下行空间。**

这看似可怕，但需注意：①河北集采以极端降幅落地的概率极低（主动脉支架非标准化产品，不同于冠脉支架的高度同质化）；②即使发生，公司还有外周+海外业务作为生存支撑。
