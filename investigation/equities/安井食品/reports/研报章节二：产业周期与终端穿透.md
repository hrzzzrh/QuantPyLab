# 研报章节二：产业周期与终端穿透

**研究日期**：2026年06月30日
**研究对象**：安井食品（603345）
**目标年份**：2026年

---

## 一、产业生命周期与库存节奏

### 1.1 周期定位：成长期中后段，结构性分化明显

中国速冻食品行业整体处于**成长期中后段**，但子行业生命周期分化显著：

| 子行业 | 生命周期阶段 | 2024年市场规模 | 2024-2029 E CAGR | 核心驱动力 |
| :--- | :--- | :--- | :--- | :--- |
| 速冻调制食品 | **成长期** | ~565亿元 | 7.1% | 火锅连锁化、麻辣烫、烧烤场景扩容 |
| 速冻菜肴制品 | **起步期/成长早期** | ~868亿元 | **16.1%** | 餐饮B端标准化、预制菜国标规范、家庭便捷需求 |
| 速冻面米制品（传统） | **成熟期** | ~771亿元（含新兴） | 低个位数 | 存量替代，现包饺子冲击 |
| 速冻面米制品（新兴） | **成熟前期** | — | 高个位数 | 手抓饼/烧麦等场景化创新 |
| 冷冻烘焙 | **成长早期** | — | >15% | 烘焙连锁化、冻转鲜技术 |

_数据来源：弗若斯特沙利文、安井年报_

**总体判断**：中国速冻食品行业2024年市场规模约2,212亿元（沙利文口径），2025年约2,042-2,131亿元（不同机构口径），2024-2029年预计CAGR 9.4%。中国人均速冻食品消费量仅10.0公斤，远低于美国（62.2公斤）、英国（45.9公斤）和日本（25.3公斤），**行业远未触及天花板**。

### 1.2 库存动力学——小龙虾的"牛鞭效应"警报

安井2025年产销数据暴露了库存问题的严重分化：

| 品类 | 2025年产量同比 | 2025年销量同比 | 库存量同比 | 库存/销量比 |
| :--- | :--- | :--- | :--- | :--- |
| 速冻调制食品 | +3.93% | +5.93% | +0.76% | 14.6%（健康） |
| 速冻面米制品 | -1.94% | +1.57% | -12.28% | 14.2%（去库） |
| 速冻菜肴制品 | **+18.84%** | **+7.19%** | **+54.84%** | **18.6%（危险）** |

**核心辨析**：速冻菜肴制品（主要是小龙虾及虾滑系列）出现严重的**产大于销**——产量暴增18.84%，但销量仅增7.19%，导致库存同比暴增54.84%。这究竟是终端需求减弱，还是渠道"牛鞭效应"？

综合判断更倾向于**需求侧结构性问题 + 渠道囤货放大效应**：

1. **小龙虾消费增速放缓**：经过2020-2023年爆发式增长后，小龙虾品类逐步进入成熟期，消费者新鲜感减退。
2. **产量被动扩张**：安井通过收购新柳伍等上游资产保证原料供应，导致产量刚性增长，但终端需求未能同步吸收。
3. **渠道牛鞭效应**：经销商在2024年小龙虾旺季预期乐观大量备货→终端销售不及预期→库存积压→2025年继续压货→库存进一步堆积。

**借鉴教训（模块一关联）**：2024年资产减值损失0.52亿→2025年飙升至2.14亿（含商誉减值和小龙虾存货跌价）。若2026年小龙虾行情持续低迷，存货减值风险仍未释放完毕。

## 二、终端需求穿透与跨期断层审计

### 2.1 最终产品画像——安井食品"最终负载"于何处？

安井的产品最终负载在以下终端消费形态：

```
B端（约70-75%）                    C端（约25-30%）
├─ 火锅店 → 鱼丸、肉丸、虾滑      ├─ 家庭火锅 → 锁鲜装、火锅料
├─ 麻辣烫/关东煮 → 丸子系列        ├─ 空气炸锅 → 烤肠、小酥肉
├─ 连锁中餐 → 小酥肉、牛肉卷       ├─ 早餐场景 → 手抓饼、烧麦
├─ 团餐/食堂 → 标准化菜肴          ├─ 休闲零食 → 烤肠、薯条
├─ 西式快餐 → 冷冻烘焙(鼎味泰)     └─ 快手烹饪 → 预制菜肴
└─ 烧烤店 → 烤肠系列
```

**核心依赖路径**：安井食品与火锅/麻辣烫产业链深度绑定，其B端营收约70%依赖餐饮渠道。火锅料（鱼丸、肉丸、虾滑等）和面向餐饮的标准化菜肴是关键业绩变量。

### 2.2 终端景气度审计——增速放缓但结构性机遇并存

#### （1）火锅行业：存量博弈，但不改B端基本盘

- 2025年中国火锅市场规模约6,390亿元，同比增长3.5%；2026年预计约6,700亿元
- 火锅门店数从2024年高峰52万家回落至2025年的44.8万家（-15.4%），**关店潮显现**
- 连锁化率从23%提升至25%，CR5头部品牌市占率从7.9%提升至8.1%

**对安井的影响**：火锅门店总数减少在短期内压制火锅料需求增速，但连锁化率提升实际上**有利于安井**——连锁品牌更依赖标准化速冻半成品，且安井已与海底捞、呷哺呷哺、围辣火锅、半天妖等头部品牌建立稳定供应关系。门店端的"优胜劣汰"（淘汰小散非连锁店）反而利好安井这样能服务大B客户的龙头供应商。

#### （2）餐饮连锁化率：核心结构性驱动，确定性极强

- 中国餐饮连锁化率从2019年13%提升至2025年25%，2024年501-1000家规模品牌门店数同比暴涨93.6%
- 万店品牌已达9家，千店品牌近40个
- 连锁餐饮的标准化需求 → 中央厨房/速冻半成品 → **安井食品的直接增量市场**

**这一逻辑具有极强的不可逆性**：一家全国有1000+门店的连锁品牌，不可能在每个门店配专业厨师从零加工——中央厨房预制+门店复热是唯一具备规模可行性的解决方案。安井作为速冻半成品供应商，本质上是"连锁餐饮的体外中央厨房"。

#### （3）团餐/西式餐饮：被低估的增量引擎

- 团餐市场2024年规模2.07万亿元，预计2024-2029年CAGR 9.1%
- 西式餐饮2024年规模9,175亿元，预计CAGR 9.8%
- 这两个赛道对食品安全、出品一致性、标准化程度的要求极高，与速冻食品特性天然契合

#### （4）C端：空气炸锅+一人食的结构性红利

- 空气炸锅保有量快速攀升 → 催生"速冻小吃"品类爆发（烤肠、小酥肉、薯条从偶尔吃到高频零食）
- 家庭小型化（户均人数持续下降）→ 小包装、单人份速冻产品增速显著高于传统家庭装
- 2025年Q2速冻食品是食品饮料酒类中**唯一**实现销售额同比增长的子类目（+1.3%）

### 2.3 动能溯源与透支审计

#### 增长动能来源：

| 驱动因素 | 性质 | 可持续性 |
| :--- | :--- | :--- |
| 餐饮连锁化率提升 | **内生结构性** | ★★★★★（长期不可逆） |
| 冷链基础设施完善 | **外生基础设施** | ★★★★（稳步推进） |
| 家庭小型化/一人食 | **内生人口结构** | ★★★★★（长期不可逆） |
| 空气炸锅普及 | **外生设备渗透** | ★★★（短期爆发，中期饱和） |
| 预制菜国标出台 | **外生政策规范** | ★★★★（利好龙头，出清小散） |
| 火锅出海国际化 | **外生市场拓展** | ★★★（中长期渐进） |
| 消费补贴/刺激政策 | **外生短期干预** | ★（不可持续） |

#### 跨期需求透支审计：

**结论：安井食品的增长逻辑中不存在显著的"跨期需求透支"风险。**

审计要点：
1. **火锅料消费是高频重复性消费**，不存在"买了今年就不买明年"的跨期替代——这与家电/汽车的购置补贴导致的透支性质截然不同。
2. **连锁化率的提升是单向上行的产业结构变化**，不是短期政策催化的"伪需求"。
3. **唯一值得警惕的透支信号**：小龙虾品类的库存积压可能反映了2022-2024年过度扩张后需求回归常态，但实际上这更像是品类生命周期从"爆发期"进入"成熟期"的自然减速，而非政策性透支。
4. **预防性警示**：需持续关注火锅门店关店潮是否加速（2025年已关15.4%），若关店率持续超20%，火锅料B端需求可能面临阶段性收缩。

## 三、供需边际与压力传导

### 3.1 供需平衡审计——产能释放vs需求增长

**供给侧**：
- 安井2025年设计产能129.07万吨（+15.36万吨），实际产量116.63万吨（+6.15万吨），产能利用率90.37%
- 2026年计划推进河南三期、四川三期、厦门安井、泰州三期及鼎益丰烘焙项目建设
- 行业层面：龙头企业普遍在扩张（三全、千味央厨等），中小企业加速出清

**需求侧**：
- 速冻食品行业整体需求增速：2024-2029 CAGR 9.4%
- 安井自身营收增速：2024年+7.7%，2025年+7.1%，2026Q1+30.8%（有低基数效应）

**供需边际判断**：

| 品类 | 供需关系 | 未来1-2年趋势 |
| :--- | :--- | :--- |
| 速冻调制食品 | 紧平衡偏松 | 产能扩张（+15万吨）+火锅门店收缩 → 价格竞争加剧，毛利率承压 |
| 速冻菜肴制品 | 结构性过剩 | 小龙虾供给过剩 + 预制菜赛道拥挤 → 低价竞争，盈利改善待国标落地后行业出清 |
| 速冻面米制品 | 存量博弈 | 新兴品类（烧麦/手抓饼）增量有限，传统水饺/汤圆受现包店冲击 |
| 冷冻烘焙 | 供不应求 | 成长早期，鼎味泰并表后有望成为新增长极 |

**核心预判**：行业正从"供需两旺"向"供给略大于需求"过渡。2025年安井产能利用率90.37%（较前几年估计下降），暗示产能消化压力加大。龙头企业通过规模优势仍能维持盈利，但中小企业面临生存压力，**行业整合加速期到来**——这长期利好安井这样的龙头。

### 3.2 压力传导特征分析

#### 原材料价格传导：直接且即时

安井直接材料成本占比64.20%，2025年核心原料价格全面上涨：

| 核心原料 | 2025年采购吨价变动 | 对毛利率的影响 |
| :--- | :--- | :--- |
| 牛肉 | **+16.19%** | 速冻调制食品（含牛丸等） |
| 鱼糜 | **+13.14%** | 鱼丸系列、虾滑系列 |
| 鲜虾类 | **+43.75%** | 虾滑、小龙虾产品 |
| 猪肉 | 2026年同比下跌22.30% | 利好2026年成本端（肉制品） |

**传导机制**：原材料价格上涨直接压缩毛利率（2025年毛利率-1.70pct）。公司通过集中采购招标、淘汰低效供应商来对冲，但无法完全抵消。好消息是**猪肉价格2026年进入下行周期**（5月全国猪肉均价20.03元/kg，同比-22.30%），鱼糜和虾类价格走势将成为2026年盈利弹性的关键变量。

#### 终端需求波动传导：B端具缓冲、C端更敏感

- **B端波动**：餐饮客户通常签订年度/季度框架协议，需求波动相对平稳。安井前五大客户销售占比仅7.04%，客户高度分散，不存在单一客户的"断崖式"需求冲击风险。
- **C端波动**：家庭消费对价格更敏感，但速冻食品的"低价高频"属性使其在经济下行期反而具有"口红效应"——消费者减少外出就餐转向家庭火锅/速冻食品。2025年Q2速冻食品是食品饮料领域唯一正增长子类目印证了这一点。

#### 波动放大效应评估

**安井面临的波动放大效应相对可控**，原因：
1. 经销商库存周期短（速冻食品保质期通常12个月以内，预制菜国标要求不超过12个月），不存在长鞭效应的极端放大。
2. 唯一例外：小龙虾品类的季节性备货（Q2-Q3旺季前大量生产入库）→若旺季销售不及预期→形成大量过期库存→引发减值。这是需要持续监控的核心风险点。

#### 传导时滞与弹性

- **成本传导时滞**：原材料涨价→毛利率受损存在约1-2个季度滞后（库存原料消耗周期）。
- **定价弹性**：安井在B端对连锁餐饮客户具有一定议价能力（产品定制化、供应稳定性），在C端通过"高质中价"策略维持性价比优势。但行业整体供大于求的趋势将限制其提价空间。

## 四、强制复核项确认

### ✅ 复核1：终端产品未来两年景气度预测

| 终端形态 | 2025-2027年景气度 | 依据 |
| :--- | :--- | :--- |
| 火锅市场 | ★★★☆☆（温和增长） | 规模3.5%增速，门店结构调整，连锁化率提升利好供应链龙头 |
| 连锁中餐 | ★★★★☆（稳健向上） | 连锁化率25%→持续提升，标准化需求刚性 |
| 团餐+西式餐饮 | ★★★★☆（确定性高） | 团餐CAGR 9.1%，西式CAGR 9.8% |
| C端家庭消费 | ★★★☆☆（缓慢增长） | 一人食/空气炸锅驱动，但人口结构长期减速 |
| 预制菜（B端） | ★★★★☆（高增速） | 国标规范后龙头获益，行业出清小散 |

### ✅ 复核2：跨期需求移位审计

未发现由刺激政策导致的显著跨期需求透支。速冻食品的核心驱动（连锁化、冷链、家庭小型化）均为结构性因素，不存在"补贴退坡后需求崩塌"的风险。唯一关注点是小龙虾品类自身的生命周期回调。

### ✅ 复核3：供需边际演变与压力传导

行业供给增速（龙头产能扩张+新进入者）阶段性超过需求增速（9.4%），供需边际向宽松方向演变。安井凭借规模效应和品牌壁垒仍具竞争优势，但毛利率中枢可能从2023年的23.21%进一步下移至20-22%区间。压力传导最敏感的是小龙虾/预制菜赛道，其次是火锅料。

## 五、模块二与模块一的"红线"贯穿

**从模块一继承的核心约束**：
1. 模块一发现2025年净利润首次下滑（-8.46%）、毛利率降至21.60% —— **模块二确认**：这背后是原材料成本上涨+小龙虾库存积压+行业竞争加剧三重因素叠加，而非偶然。
2. 模块一发现资产减值损失暴增2.14亿 —— **模块二深化**：小龙虾业务的牛鞭效应是主因，2026年仍需警惕进一步减值。
3. 模块一发现2026Q1显著复苏（营收+30.84%，净利+42.74%） —— **模块二的解释**：猪肉价格下行贡献成本红利+低基数效应+产品结构升级，但需在Q2验证可持续性。
4. 模块一发现ROE持续下行 —— **模块二补充**：行业资本开支竞赛（产能过剩）是ROE下行的产业层面原因，行业整合完成后龙头ROE有望回升。
