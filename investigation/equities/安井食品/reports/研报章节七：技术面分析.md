### 3.2 量价配合分析

近期呈现**典型的"缩量下跌"特征**：
- 6月跌幅-18.5%（近1个月），成交量持续低于20日均量
- 缩量下跌通常有两种解读：① 卖方力量衰竭，底部信号（看好）；② 市场无人接盘，流动性枯竭（看空）

**结合安井的基本面判断，倾向于第①种解读**。理由：
- Q1业绩已证明基本面改善（+42.74%净利润）
- PE TTM 17.5x处于历史低位，估值泡沫已充分挤出
- 港股安井（02648.HK）6月29日收盘约82.81港元（约等同A股82.81元），A/H溢价极小

**但需警惕**：若未来1-2周出现"放量跌破年线"的走势——即单日成交量暴增至10万手以上且价格收于年线之下超过3%——则将否定底部判断，信号将转为"趋势破位"。

## 四、未来走势演绎

### 4.1 短期走势（1-3个月）

**基准路径（概率55%）**：年线附近震荡筑底

```
场景：价格在93-100（后复权）/ 80-86（市价）区间反复测试
时间：2026年7-8月
触发条件：Q2业绩预告/半年报确认趋势向好
目标：价格站稳MA20（99.3）以上，结束短期下跌通道
风险：大盘系统性下跌拖累
```

**偏空路径（概率25%）**：年线失守，继续探底

```
场景：跌破年线（94.84/81.44）后加速下探
时间：2026年7-8月
触发条件：Q2业绩不及预期、大盘破位、消费板块恐慌蔓延
目标：测试52周低点81.05（69.60）甚至更低
风险：趋势从调整恶化为熊市
```

**偏多路径（概率20%）**：V型反转

```
场景：年线附近放量企稳后快速反弹
时间：2026年7月
触发条件：Q2业绩大超预期+市场风格切换至消费
目标：快速收复MA60（108.64/93.30），进入中期上升通道
```

### 4.2 长期走势（6-12个月以上）

**核心判断**：当前价格行为正在构建**"头肩底"的右肩或"双底"的第二只脚**。

历史结构分析：
- **左肩**：2025年H1在85-95区间反复测试（后复权）
- **头部**：2025年Q3 H股上市前短暂探至81.05（52周低点）
- **右肩**：2026年6月回踩年线93-95区间

若右肩构建成功（在年线附近缩量企稳并放量突破颈线115-120），则：
- 头肩底形态高度 ≈ 120 - 81 = 39（后复权）
- 形态目标价 ≈ 120 + 39 = 159（后复权）≈ 137（市价）

**但需注意**：技术形态的完成需要基本面的确认。安井当前的右肩构建成功与否，取决于2026年H2（Q3）业绩能否延续Q1的改善趋势。如果H2确认趋势性反转，则技术面+基本面共振将形成强力上行动能。

**技术面核心结论**：短期偏弱（下跌趋势中），中期待变（年线保卫战），长期看多（头肩底形态+低估值）。技术面给出的操作建议是：**等待年线附近放量企稳信号出现后再介入，而非在当前加速下跌阶段盲目抄底。**

**关键监测信号**：
1. 年线（MA250，94.84后复权/81.44市价）是否有效守住（连续3日收盘在之上）
2. 成交量是否从缩量转为温和放量（量比重回1.0以上）
3. 短期均线MA20（99.30）是否被重新站上
