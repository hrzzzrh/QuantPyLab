# 研报章节二：产业周期与终端穿透

**研究日期**：2026年06月07日
**研究标的**：中国西电（601179.SH）
**前序模块核心结论回顾**：营收/利润双增且利润增速>收入增速；毛利率持续扩张（16.4%→22.9%）；合同负债+22.3%印证订单景气；但ROE仅5.64%、AR高压（96.5亿）、海外毛利率低迷（12%）、2025年投资现金净流出54.6亿为关键隐忧。

---

## 一、产业生命周期与时空坐标定位

### 1.1 行业生命周期判定

输变电设备制造行业在技术维度已进入成熟期，但在**需求维度正处于新一轮超级上行周期**。不同于一般制造业的"成熟即衰退"宿命，电网设备行业正经历由能源转型驱动的"二次成长曲线"：

| 观察维度 | 现状判定 | 核心证据 |
| :--- | :--- | :--- |
| 技术成熟度 | 成熟期 | 特高压技术全球领先，交流1000kV/直流±1100kV已商业化运营超15年 |
| 需求增速 | **加速成长期** | "十五五"电网投资4万亿元（+40%），年均超8000亿元 |
| 渗透率 | 结构性扩张 | 新能源占比从20%→30%+，驱动电网重建需求 |
| 全球化阶段 | 出海初期 | 海外收入仅占18%，巴西/中东/南美/欧洲机会巨大 |

**研判结论**：行业处于"技术成熟+需求加速"的黄金交叉期——技术成熟意味着竞争壁垒稳固、供给格局清晰；需求加速意味着增量蛋糕正在变大。这一组合在制造业中较为罕见。

### 1.2 库存周期动力学

电网设备行业与一般周期品行业的库存逻辑有本质区别：

- **生产模式**：以销定产（MTO），非备货式生产
- **需求特征**：项目制驱动，Q4集中交付（Q4营收占比约28%-30%）
- **库存含义**：存货52.8亿（+11.9%）并非"滞销积压"，而更多是"在制品"——即在执行但尚未达到收入确认节点的在手合同
- **合同负债信号**：57.78亿（+22.3%）是比存货更前瞻的景气度指标，直接反映预收款/已签约未交付订单

**牛鞭效应辨析**：电网投资以国网/南网招标→设备商中标→生产交付的链条传导，**下游客户（电网公司）的投资计划基于电力需求预测和系统规划**，非渠道库存博弈。因此，订单波动真实反映终端电网建设需求，牛鞭效应放大风险较低。

## 二、终端需求穿透与跨期断层审计（强制审计）

### 2.1 最终产品画像

中国西电的产品最终负载于**电力输配电网**——具体形态为特高压/超高压变电站、换流站、输电线路中的核心一次设备。其终端"消费者"不是个人或企业，而是**电网基础设施的物理扩张与升级**。因此，需求分析的终点是电网投资规模与结构。

#### 跨期透支强制审计

**审计问题一：是否因刺激政策导致"需求前置"？**

国家电网4万亿"十五五"规划是**五年结构性投资计划**，基于电力系统物理需求（新能源消纳、负荷增长、安全冗余），而非短期经济刺激方案。佐证：Q1 2026投资同比+37%，但配网投资占比达55%——说明投资结构均衡，非突击式集中投放。**前置风险：低**。

**审计问题二：特高压建设是否存在"抢装潮"后的断崖风险？**

"十五五"投产15条直流（vs "十四五"约12条），条数增加但增幅温和（+25%），且工期长达3-4年，设备交付平滑分布于各年。长周期项目属性使需求具有天然平滑性。**断崖风险：低**。

**审计问题三：海外需求是否为可持续趋势？**

①欧洲电网升级是十年维度的存量替换周期，非景气波动；②AI/数据中心电力需求处于爆发初期（IEA预测2030年数据中心用电量约950TWh，翻倍）；③全球变压器产能扩张落后于需求增长（花旗缺口模型验证）。**趋势确定性：高**。

## 三、供需边际与压力传导

### 3.1 供需平衡审计

**供给侧（行业产能）**：

| 环节 | 格局特征 | 扩产节奏 |
| :--- | :--- | :--- |
| 特高压变压器 | CR3>80%（中国西电、特变电工、保变电气） | 壁垒极高，新进入者不可能 |
| 高压开关（GIS） | CR3>70%（中国西电、平高电气、新东北电气） | 产能扩张有序 |
| 换流阀 | CR3>90%（国电南瑞、许继电气、中国西电） | 柔直技术迭代中 |
| 全球层面 | 扩产速度落后于需求增速 | 花旗：2025-2028产能+53%仍不足 |

中国西电智慧产业基地已于2025年转固（在建工程-69.5%，固定资产+48.8%），**产能扩张的物理信号已兑现**，意味着2026年起将进入新产能释放期。

**需求侧（终端拉动）**：

- 国内特高压：在建8条+规划10+条，年设备需求约300-500亿
- 国内配网/超高压：国网110kV+项目Q1开工+42%
- 海外：公司海外收入同比+54%（FY2025），在手订单持续增长

**供需边际预判**：

当前全球高压变压器市场处于**供不应求的紧平衡**（花旗缺口模型：2025年缺口~30%）。中国西电凭借新释放的智慧产业基地产能，有望在"供给硬约束"的窗口期实现量价齐升。**预判：供需格局在未来2-3年内有利于设备制造商，产能扩张的红利正在兑现而非已被透支。**

### 3.2 压力传导特征分析

#### 从原材料到设备商的成本传导

- 铜/硅钢片价格上涨→变压器成本上升→能否转嫁给下游？
- **中国西电的转嫁能力：中等偏强**。理由：①特高压设备技术壁垒高、供应商有限，议价权优于低端产品；②毛利率从16.4%（2022）持续攀升至22.9%（2025），表明成本转嫁+产品升级双轮驱动有效
- 但需关注原材料成本占比83.25%，若铜价持续大幅上涨，毛利率扩张节奏可能受阻

#### 从终端需求到设备商订单的波动传导

- **传导时滞**：电网投资计划→招标（1-3个月）→中标→排产交付（6-18个月）。国网Q1投资+37%将在Q3-Q4体现为设备商交付高峰
- **波动放大效应**：较低。因为：①B2G模式，订单直接来自电网集中招标，中间环节少；②合同负债57.8亿（+22.3%）提前锁定需求，无需猜测
- **博弈筹码**：中国西电作为央企龙头，在国网招标中享有"稳定的份额+合理利润"的双重保障

## 五、模块二核心结论

1. **产业周期定位**：输变电设备行业处于"技术成熟+需求二次加速"的黄金交叉期，驱动力来自能源转型（新能源接入）、电网升级（特高压/配网）、新兴用电需求（AI/数据中心/EV）三大引擎的叠加共振，属于超长周期景气而非短期波动。

2. **终端需求确定性极高**：国网4万亿+南网持续加码+全球电网投资进入超级周期，三重共振下未来2-3年需求下行风险极低。全球变压器供给缺口（30%）为中国设备商提供了扩产红利窗口。

3. **跨期断层风险可控**：特高压项目周期长、交付平滑；投资驱动来自物理需求（非政策刺激）；配网投资占比55%说明投资结构均衡。不存在类似光伏/新能源汽车"抢装后断崖"的跨期透支模式。

4. **压力传导偏向有利**：供给侧缺口赋予设备商议价权，成本转嫁能力经受住了铜价波动的考验（毛利率持续扩张）。但需持续监控原材料价格和国内竞争烈度。
