## 三、逻辑证伪警报（Invalidation Triggers）

> **出现以下任一信号，投资逻辑即宣告失效，必须触发止损或下调至"卖出"评级。**

### 🚨 一级警报（触发即止损）

| 编号 | 证伪信号 | 阈值 | 逻辑说明 |
| :--- | :--- | :--- | :--- |
| **T1** | 奇瑞/吉利（前两大客户）转向自研或更换制动供应商 | 任一客户丢失>20%订单 | 将直接导致15亿+年化收入归零，且引发连锁反应 |
| **T2** | 毛利率连续两季跌破17% | <17% | 说明公司已丧失技术定价权，沦为纯代工企业 |
| **T3** | EMB量产出现重大安全事故或召回 | 任一批次召回 | 安全件召回将直接摧毁EMB商业信誉，且可能波及WCBS业务 |
| **T4** | 单季营收负增长且净利腰斩 | 营收YoY<-10% 且 净利YoY<-50% | 增长逻辑彻底破裂 |
| **T5** | **【新增】坐标系诉讼失败且客户大规模转向** | EMB丢失2个以上核心客户 | EMB护城河瓦解，先发优势归零 |

### ⚠️ 二级警报（触发即下调至保守情景+减持）

| 编号 | 证伪信号 | 阈值 |
| :--- | :--- | :--- |
| **T6** | WCBS市占率连续两季下降 | One-Box份额跌破10% |
| **T7** | 新增定点年化收入断崖式下滑 | <50亿元/年（较2025年腰斩） |
| **T8** | OCF持续为负 | 连续两季经营现金流为负（含豫北转向并表） |
| **T9** | 大额应收账款坏账 | 单项坏账>1亿元 |
| **T10** | 豫北转向整合失败 | 并表后净利大幅低于收购预期（<1亿元）或OCF持续为负 |
| **T11** | **【新增】USMCA审查结果严重不利** | 北美市场实质性关闭或墨西哥工厂被迫停产 |
