# 研报章节二：产业周期与终端穿透

**研究日期**：2026年05月26日  
**逻辑继承**：模块一发现毛利率5年压缩6.78pp至19.65%，Q1 2026营收近乎零增长，且新曝光治理隐忧。本章节将向下穿透至终端汽车市场，结合2026年4-5月最新数据审计增长动能真伪与供需边际演变。  

---

## 一、产业生命周期与库存节奏定位

### 1.1 行业三维坐标定位

| 维度 | 定位 | 证据 |
| :--- | :--- | :--- |
| **汽车整体市场** | **成熟期（存量博弈，需求显著承压）** | 2026年1-4月乘用车累计零售560.4万辆，同比-18.5%；4月单月零售138.4万辆，同比-21.5%。全年预期已从此前+0.5%下修至负增长 |
| **新能源汽车** | **成长期后半段（增速放缓，但渗透率创新高）** | 4月NEV渗透率61.4%（历史首破60%），预计5月达62.5%。但零售量同比仍为负（1-4月NEV零售同比-17.2%），绝对销量受拖累 |
| **线控制动（EHB）** | **成长期中段（规模化放量）** | 2025年渗透率突破50%，One-Box占比86.6%。2026年预计达60%+，市场规模预计180.84亿元，CAGR 16.45% |
| **电子机械制动（EMB）** | **导入期→早期量产（量产元年确认）** | 2026年为首个量产年。伯特利3月小批量交付，4月批量生产，5月8日第1万件下线。法规已实施。预计2026-2030年CAGR超70% |

**核心定位**：伯特利横跨三个生命周期阶段——成熟期（盘式制动器/轻量化零部件）、成长期（EHB线控制动/EPB）、导入期（EMB/线控转向/悬架/机器人零部件）。公司的整体成长性取决于成长期和导入期业务能否接续成熟期业务的增长放缓。

### 1.2 库存周期动力学——牛鞭效应预警强化

**产业链库存状态判断（更新至2026年4月）**：

| 环节 | 信号 | 方向判断 |
| :--- | :--- | :--- |
| **终端零售** | 4月乘用车零售138.4万辆，同比-21.5%（更深于Q1的-17.4%）；5月1-17日零售68万辆，同比-23% | **需求持续恶化，未见好转** |
| **主机厂批发** | 4月批发212.9万辆，同比仅-4.2%。零售-批发剪刀差扩大（内销-25.5% vs 批发-4.2%） | **批发与零售严重背离，渠道被动累库** |
| **经销商库存** | 4月月末行业库存354万辆，较上月+9万辆。库存持续增长，经销商预警指数偏高 | **被动累库信号明确** |
| **Tier1供应商（伯特利）** | 存货18.07亿元（+27.3%），增速远超营收增速20.9% | **被动累库信号明确** |

**牛鞭效应辨析（强化版）**：

2025年主机厂在"两新"补贴刺激下大幅扩产（批发+8.8%），带动Tier1供应商大量备货。但2026年一季度终端零售同比-17.4%，**4月进一步恶化至-21.5%，5月1-17日同比-23%**——需求侧下滑不仅在持续，而且在加深。而供给侧的产能仍在惯性释放（4月批发仅-4.2%，远好于零售）。

这种**终端-批发-零部件之间的增速错配**，正是牛鞭效应的典型特征。库存链条从终端（经销商累库）→主机厂（批发不减但零售崩塌）→供应商（被动累库+新增定点兑现延迟），压力正在逐级向上传导。

> **核心辨析**：伯特利Q1 2026营收仅增1.38%，但终端需求在Q2的进一步恶化（4月零售-21.5%）意味着——Q2营收增速可能由正转负。2025年新增定点605项/年化95亿元是未来收入的领先指标，但其兑现高度依赖下游主机厂的实际产销量。**终端需求若持续承压，新增定点的兑现率将显著下降。**

## 二、终端需求穿透与跨期断层审计（MANDATORY）

### 2.1 最终产品画像

伯特利的产品最终负载于**乘用车**（占比近100%），具体为：
- 盘式制动器/EPB/WCBS/EMB → 乘用车制动系统
- 轻量化铸铝转向节/控制臂 → 乘用车底盘
- 机械转向/EPS → 乘用车转向系统
- ADAS → 乘用车智能驾驶

**终端景气度直接影响因子** = 中国乘用车产量 + 新能源汽车产量 + 出口量

### 2.2 终端景气度审计（2026-2028年）——更新至5月数据

#### 2.2.1 中国乘用车内需：承压下行加深

| 年度/时段 | 乘用车销量 | YoY | 驱动力 |
| :--- | :--- | :--- | :--- |
| 2025全年 | 3010万辆 | +9.2% | 补贴加码+报废更新放量 |
| 2026 Q1 | 零售同比-17.4% | — | 补贴退坡+高基数 |
| **2026年4月** | **零售138.4万辆** | **-21.5%** | 三重压力叠加，恶化加深 |
| **2026年5月1-17日** | **零售68.0万辆** | **-23%** | 需求延续大幅下滑 |
| **2026年5月全月** | **预计152万辆** | **-21.6%** | 同比仍深跌，但环比+9.9%显示边际修复 |
| **2026E全年** | **下调至约2800-2900万辆** | **约-4%~-7%** | 此前中汽协预测+0.5%已严重失效 |

**需求下滑的结构性特征**：
- **燃油车崩塌**是主因：1-4月燃油车零售同比下滑74万辆（-40%），贡献了绝大部分下滑
- **NEV韧性偏强**：尽管NEV零售同比也下降（1-4月-17.2%），但渗透率仍在创新高（4月61.4%，5月预计62.5%）
- **出口是唯一亮点**：1-4月汽车出口312.7万辆，同比+61.5%

#### 2.2.2 新能源汽车：渗透率创新高但绝对量承压

| 时段 | 新能源零售/批发 | YoY | 渗透率 |
| :--- | :--- | :--- | :--- |
| 2026年4月 | 零售84.9万辆/批发122.5万辆 | 零售-6.8%/批发+7.5% | **61.4%（首破60%）** |
| 2026年5月1-17日 | 零售40.0万辆 | -12% | 58.9% |
| **2026年5月E** | **零售95万辆** | — | **62.5%（预计新高）** |
| **2026E全年** | **下调至约1700-1800万辆** | **约+5~10%** | 此前预测+15.2%难以实现 |

新能源仍是增长动力，但增速显著不及预期。渗透率飙升主要源于燃油车基数的坍塌性下滑（"被动渗透率提升"），而非新能源绝对增速的强劲。

#### 2.2.3 出口：结构性增量仍在

1-4月出口312.7万辆（+61.5%），其中4月乘用车出口76.9万辆（+80.7%），新能源占出口52.7%。出口仍是伯特利海外业务（轻量化+EPB）的确定性增长点，但需注意USMCA审查（7月）的不确定性。

### 2.3 动能溯源与跨期透支审计（强制）——更新

#### 增长动能分类

| 驱动力 | 类型 | 持续性判断 |
| :--- | :--- | :--- |
| 新能源汽车渗透率提升 | **内生** | 持续但以"被动提升"为主——渗透率升但绝对量降 |
| "两新"以旧换新政策 | **外生（政策）** | **边际效应显著递减**——2026年从"定额制"改为"比例制"，低价车补贴大幅缩水（微型车从2万降至6000元） |
| 购置税减免 | **外生（政策）** | **2026年减半**，购车成本上升约8800元/辆 |
| 智能化升级（线控制动渗透） | **内生** | 确定性较高——EHB渗透率持续提升（50%→60%→70%） |
| 国产品牌替代外资 | **内生** | 确定性较高——伯特利在EHB领域持续获得份额 |
| 出口扩张 | **内生+外生** | 高增长（+61.5%）但USMCA审查（7月）存在骤停风险 |

#### 跨期断层审计：风险已从"预警"升级为"现实"

**需求透支证据链（强化版）**：

| 证据 | 来源 | 最新数据 |
| :--- | :--- | :--- |
| 2025年以旧换新超1150万辆 | 商务部 | 透支效应正在释放 |
| 2026年Q1零售-17.4% | 乘联分会 | 一季度深跌 |
| **2026年4月零售-21.5%（恶化）** | 乘联分会 | **二季度延续且加深** |
| **2026年5月1-17日零售-23%** | 乘联分会 | **下滑趋势未见拐点** |
| 经销商库存354万辆（4月末） | 乘联分会 | **被动累库持续，去化压力大** |
| NEV渗透率61.4%（4月） | 中汽协 | 历史新高但燃油车加速退出是主因 |
| 行业利润率Q1仅3.2% | 中汽协 | 逼近盈亏线，价格战空间已封死 |

**"跨期断层"判断（升级）**：2024-2025年两轮"两新"刺激实质上已**提前透支了2026-2027年的置换需求**。2026年4-5月的终端数据证实了这种透支效应不仅存在，而且比一季度更为严重（-21.5%/-23% vs -17.4%）。5月环比改善（+9.9%）提供了一丝希望，但同比深跌的格局短期内难以逆转。

**对伯特利的业务含义**：
1. **2026年乘用车市场将负增长（而非零增长）**，销量重心需从此前的3025万辆下调至2800-2900万辆
2. **Q2营收可能由正转负**，2025年新增定点的大规模兑现将被推迟至Q3-Q4甚至2027年
3. **出口（+61.5%）是少有的确定性增量**，但占比仍小（<15%）

## 三、供需边际与压力传导——最新信号

### 3.1 行业价格战的变化：从"持续恶化"到"戛然而止"

**2026年5月出现了一个转折性信号**（来源：36氪5月26日报道《汽车价格战，突然停了》）：

| 变化 | 说明 |
| :--- | :--- |
| **碳酸锂价格暴涨** | 从7.5万元/吨→17万元/吨（+130%），电池成本增3000-5000元/车 |
| **芯片价格暴涨** | 车规DRAM涨价180%，DDR5涨300% |
| **车企利润被双向挤压** | 上游成本暴涨+终端需求疲软→降价空间被彻底封死 |
| **政策整治"内卷"** | 市场监管总局发布《汽车行业价格行为合规指南》，划定低于成本销售红线 |
| **多数车企提价** | 涨价+增配调价成为新常态 |

**对伯特利的含义**：这可能是毛利率压力的**边际拐点**。如果行业价格战真正停止，主机厂对Tier1的年降压力将缓解。伯特利的毛利压缩（5年-6.78pp）在很大程度上源于这种传导压力。但这一转机的兑现仍需2-3个季度的数据验证。

### 3.2 EMB竞争格局：量产窗口的"卡位战"白热化

最新信息（36氪5月7日报道+各公司公告）揭示：

| 竞争者 | EMB进度 | 竞争态势 |
| :--- | :--- | :--- |
| **伯特利** | 3月小批量交付，4月批量生产，5月8日第1万件下线，120万年产能 | 量产进度领先 |
| **坐标系** | 80万件/年EMB产线已建成，计划2026年Q1量产，已获2个EMB合作项目 | **最直接竞争者，定价3488元/套（极低）** |
| 理想汽车 | 自研EMB，计划2026年大规模量产 | 主机厂自研威胁 |
| 比亚迪/弗迪科技 | 自研EMB推进中 | 体系内替代风险 |
| 利氪科技 | 2026年头部车企首发量产 | 科创公司追赶 |
| 采埃孚 | 已获500万辆级EMB订单 | 外资巨头参与 |

**伯特利 vs 坐标系——正面对抗**：坐标系将EMB系统定价3488元/套，业内评价"打到地板线"。伯特利已起诉坐标系（前员工团队），保护核心知识产权。这场法律+商业的双重竞争将决定伯特利在EMB领域的市场份额和盈利空间。

> **核心辨析**：EMB量产窗口打开的同时，竞争强度远超市场预期。价格战从整车蔓延至零部件（EMB套价3488元），先发优势可能难以转化为定价权优势。

## 四、强制复核项确认（Mandatory Audit Checks）

> ✅ **复核1**：已穿透并引用终端产品最新景气度——4月乘用车零售-21.5%，5月1-17日-23%（恶化加深）。全年预期从此前+0.5%大幅下调至-4%~-7%。

> ✅ **复核2**：已审计跨期需求透支风险——4-5月终端数据证实透支效应比Q1更严重。2025年超1150万辆以旧换新实质性地前置了置换需求。

> ✅ **复核3**：已分析供需边际演变方向——EHB从"供不应求"转向"供需平衡"，EMB竞争白热化（15+竞争者+价格战3466元/套）。行业价格战戛然而止暗示毛利率底可能已过，但需数据验证。

## 五、模块二核心结论

1. **终端需求恶化超预期**：Q1零售-17.4%→4月零售-21.5%→5月1-17日-23%，下滑在加深而非企稳。全年乘用车销量预期需从此前+0.5%大幅下调至-4%~-7%。

2. **但5月出现了两个边际转机信号**：①价格战突然停止（成本上涨+政策整治），可能缓解伯特利的年降压力；②5月全月零售预计环比+9.9%，出现微弱的季节性修复。

3. **EMB量产元年确认但竞争白热化**：伯特利5月8日第1万件下线+120万年产能，量产进度领先。但坐标系以3488元/套的"地板价"竞争，叠加伯特利对其的知识产权诉讼，EMB可能从"技术溢价"提前进入"价格竞争"。

4. **"跨期需求透支"风险已从预警升级为现实**：2025年超1150万辆以旧换新透支了2026-2027年需求。2026年全年车市大概率负增长，对伯特利而言意味着——**2025年新增定点605项的大规模兑现将被推迟**。

5. **产业链位置提供缓冲但力度减弱**：作为安全件供应商的刚性需求属性未变，但在终端市场整体收缩的背景下，这种缓冲的边际效用正在减弱。
