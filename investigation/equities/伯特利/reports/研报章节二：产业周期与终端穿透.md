# 研报章节二：产业周期与终端穿透

**研究日期**：2026年05月09日  
**逻辑继承**：模块一发现伯特利毛利率5年压缩6.78pp至19.65%，Q1 2026营收近乎零增长。本章节将向下穿透至终端汽车市场，审计增长动能真伪与供需边际演变。  

---

## 一、产业生命周期与库存节奏定位

### 1.1 行业三维坐标定位

| 维度 | 定位 | 证据 |
| :--- | :--- | :--- |
| **汽车整体市场** | **成熟期（存量博弈）** | 2026年中汽协预测总销量3475万辆，仅微增1%；乘用车3025万辆，微增0.5% |
| **新能源汽车** | **成长期后半段（增速放缓）** | 2025年同比+28.2%（1649万辆），2026年预计+15.2%（1900万辆），渗透率从47.9%向55%+迈进，但增速腰斩 |
| **线控制动（EHB）** | **成长期中段（规模化放量）** | 2025年渗透率突破50%，2026年预计达60%+，市场规模预计180.84亿元，CAGR 16.45%（2022-2026） |
| **电子机械制动（EMB）** | **导入期（量产元年）** | 2026年为首个量产年，伯特利3月首次小批量交付，4月批量生产。法规（GB21670-2025）2026年1月实施。预计2026-2030年CAGR超70% |
| **线控转向（SBW）** | **导入期（渗透率<5%）** | 仅部分豪华车型后轮线控，预计2025年渗透率约5%，2030年达10% |

**核心定位**：伯特利横跨三个生命周期阶段——成熟期（盘式制动器/轻量化零部件）、成长期（EHB线控制动/EPB）、导入期（EMB/线控转向/悬架/机器人零部件）。公司的整体成长性取决于成长期和导入期业务能否接续成熟期业务的增长放缓。

### 1.2 库存周期动力学——牛鞭效应预警

**产业链库存状态判断**：

| 环节 | 信号 | 方向判断 |
| :--- | :--- | :--- |
| **终端零售** | Q1 2026狭义乘用车零售422.6万辆，同比-17.4%；4月预计142万辆，环比-13.8% | **需求疲软，被动累库** |
| **主机厂批发** | 2026年乘用车批发3009万辆预测（+2%），高于零售预测（-1%），批发-零售剪刀差扩大 | **主动加库，可能误判终端需求** |
| **Tier1供应商（伯特利）** | 存货18.07亿元（+27.3%），增速远超营收增速20.9% | **被动累库信号明确** |

**牛鞭效应辨析**：

2025年主机厂在"两新"补贴刺激下大幅扩产（批发+8.8%），带动Tier1供应商大量备货。但2026年一季度终端零售同比骤降17.4%——需求侧信号已经转负，而供给侧的产能仍在惯性释放。这种**终端-批发-零部件之间的增速错配**，正是牛鞭效应的典型特征。

> **核心辨析**：伯特利Q1 2026营收仅增1.38%，但2025年全年在研项目571项（+32.8%）、新增定点605项（+45.4%）、新定点年化收入95亿元（+45.4%）。**新增定点是未来收入的领先指标，但其兑现取决于下游主机厂的实际产销量**。如果主机厂因终端需求疲软而推迟或缩减量产计划，这些"在手定点"可能出现兑现率下降。

## 二、终端需求穿透与跨期断层审计（MANDATORY）

### 2.1 最终产品画像

伯特利的产品最终负载于**乘用车**（占比近100%），具体为：
- 盘式制动器/EPB/WCBS/EMB → 乘用车制动系统
- 轻量化铸铝转向节/控制臂 → 乘用车底盘
- 机械转向/EPS → 乘用车转向系统
- ADAS → 乘用车智能驾驶

**终端景气度直接影响因子** = 中国乘用车产量 + 新能源汽车产量 + 出口量

### 2.2 终端景气度审计（2026-2028年）

#### 2.2.1 中国乘用车内需：承压下行

| 年份 | 乘用车销量 | YoY | 驱动力 |
| :--- | :--- | :--- | :--- |
| 2024 | 约2758万辆 | +5.3% | "两新"补贴刺激+新能源渗透加速 |
| 2025 | 3010万辆 | +9.2% | 补贴加码+报废更新放量（全年以旧换新超1150万辆） |
| **2026E** | **约3025万辆** | **+0.5%** | 补贴边际效应递减+购置税减半+高基数 |
| 2027E | — | 大概率负增长 | 补贴退坡+透支效应显现+自然置换周期下行 |

**三大利空叠加**（2026年）：
1. **新能源汽车购置税减半**：从全额免征→减半征收，20万元车型购车成本增加约1万元
2. **以旧换新补贴方式调整**：从"定额制"变为"比例制"，低价车型补贴缩水
3. **高基数效应**：2025年Q1零售基数极高（同比+12.8%），2026年Q1已暴跌-17.4%

#### 2.2.2 新能源汽车：增速换挡

| 年份 | 新能源销量 | YoY | 渗透率 |
| :--- | :--- | :--- | :--- |
| 2024 | 1287万辆 | +35.5% | 42% |
| 2025 | 1649万辆 | +28.2% | 47.9% |
| **2026E** | **1900万辆** | **+15.2%** | **54.7%** |
| 2027E | ~2050万辆 | ~+8% | ~60% |

新能源仍是增长动力，但增速斜率显著放缓（28.2%→15.2%→~8%）。增量绝对值仍在扩大（+361万→+251万→+150万），但已从"爆发期"进入"稳健增长期"。

#### 2.2.3 出口：结构性增量

2025年出口740万辆（+4.3%），增速同样放缓。伯特利的海外业务主要依赖墨西哥工厂（二期建设中），面向通用、Stellantis、福特等全球客户。出口仍是确定性较高的增量来源，但受关税和地缘政治影响存在不确定性。

### 2.3 动能溯源与跨期透支审计（强制）

#### 增长动能分类

| 驱动力 | 类型 | 持续性判断 |
| :--- | :--- | :--- |
| 新能源汽车渗透率提升 | **内生** | 持续但减速——从28%→15%→8%逐年放缓 |
| "两新"以旧换新政策 | **外生（政策）** | **正在退坡**——2025年为高峰期，2026年补贴方式调整+多地暂停 |
| 购置税减免 | **外生（政策）** | **2026年减半**，购车成本上升 |
| 智能化升级（线控制动渗透） | **内生** | 确定性较高——EHB渗透率持续提升（50%→60%→70%） |
| 国产品牌替代外资 | **内生** | 确定性较高——伯特利在EHB领域持续获得份额 |
| 出口扩张 | **内生+外生** | 需关注关税和地缘政治风险 |

#### 跨期断层审计：风险真实存在

**需求透支证据链**：

| 证据 | 来源 | 严重程度 |
| :--- | :--- | :--- |
| 2025年以旧换新超1150万辆，占零售50%以上 | 商务部数据 | **极高**——政策驱动贡献了一半以上需求 |
| 2026年Q1零售-17.4%，新能源-14.1% | 乘联分会 | **极高**——政策退坡后真实需求显露 |
| 多地"地补"资金耗尽暂停（九成省份） | 多地商务厅公告 | **高**——地方财政承压，补贴持续性存疑 |
| 2025年9月后新能源渗透率连续4个月超50%后，2026年3月回落至51.7% | 中汽协/乘联分会 | **中**——渗透率提升边际放缓 |
| 补贴从"定额制"改"比例制"后低价车受冲击 | 2026年实施细则 | **中**——结构影响大于总量 |

**"跨期断层"判断**：2024-2025年两轮"两新"刺激实质上已**提前透支了部分2026-2027年的置换需求**。政策的"边际效应递减"叠加"购置税减半"将形成双重压力。2026年Q1已经验证了这种压力——零售暴跌17.4%，即使在补贴延续的背景下。

对伯特利而言，这意味着：
1. **2026年整体乘用车市场增量接近零**，营收增长将主要依赖产品渗透率提升（线控制动/EMB/ADAS）+ 市场份额扩大（国产替代）+ 收购并表（豫北转向）
2. **Q1的增长骤停并非偶然**，而是政策退坡+高基数+需求透支的必然结果

## 三、供需边际与压力传导

### 3.1 线控制动（EHB/EMB）：从紧平衡向供给释放演进

**需求端**：
- 2026年EHB前装出货预计1231.69万套，市场规模180.84亿元（+16.45% CAGR）
- EMB 2026年小规模量产，2026-2030年CAGR超70%，2030年市场规模超115亿元

**供给端**：
- 博世仍占53.68%份额（2024H1），但正在下降
- 伯特利+亚太股份+弗迪动力等国内厂商合计占30%+
- EMB竞争者众多：伯特利、亚太股份、利氪科技、拿森科技、万安科技、格陆博等

**供需边际判断**：

| 维度 | 判断 |
| :--- | :--- |
| EHB当前状态 | 从"供不应求"进入"供需基本平衡"，但国产替代仍在进行 |
| EHB 2027年前景 | 渗透率接近天花板（70%+）时，增长将完全依赖汽车总量，**量增逻辑弱化** |
| EMB当前状态 | 需求远大于供给（仅少数厂商具备量产能力），伯特利具备显著先发优势 |
| EMB 2028年前景 | 竞争者涌入→供给快速增加→价格竞争→利润率承压（复刻EHB路径） |

### 3.2 压力传导特征分析

**传导路径**：终端车市疲软 → 主机厂价格战 → Tier1供应商年降压力 → 伯特利毛利率压缩

| 传导环节 | 具体表现 | 数据验证 |
| :--- | :--- | :--- |
| **终端→主机厂** | 价格战白热化，比亚迪2026年5月发动"限时一口价"，吉利、奇瑞跟进 | 车企利润率低位（行业利润率4.5%），以价换量 |
| **主机厂→Tier1** | 年度降价要求（3-5%/年），采购策略从"高质高价"转向"高质低价" | 伯特利毛利率5年下降6.78pp |
| **波动放大效应** | 主机厂销量目标激进（部分车企目标+30%以上），倒逼Tier1备货。若终端不及预期→渠道库存积压→砍单 | 伯特利存货增速27.3% vs 营收增速20.9% |
| **传导时滞** | 新增定点（年化95亿元）到实际量产通常需12-18个月。2025年的定点高峰预计在2026H2-2027年释放收入 | Q1 2026增速骤停可能是短期扰动而非趋势逆转 |

### 3.3 公司的博弈筹码评估

| 博弈维度 | 评价 | 依据 |
| :--- | :--- | :--- |
| **技术壁垒** | ★★★★☆ | One-Box自主龙头，EMB全球首张ASIL-D认证，IMC前馈控制技术 |
| **客户粘性** | ★★★★☆ | 制动系统验证周期长（2-3年），替换成本高；覆盖全球前十大车企中的8家 |
| **规模优势** | ★★★☆☆ | 年产智能电控700万+套，规模效应初显但仍在爬坡 |
| **议价能力** | ★★★☆☆ | 对下游：承受年降压力但有一定技术定价权；对上游：应付账款29.96亿，占款能力增强 |
| **刚性需求属性** | ★★★★★ | 制动系统是安全件，法规强制，无法被省略或降级 |

**综合判断**：伯特利作为**安全件供应商**，在产业链中拥有一定的不可替代性，这构成了对抗下游传导压力的核心缓冲。但**线控制动的国产替代加速意味着竞争格局从"外资垄断"向"多家竞争"演变**，定价权在中长期可能受到蚕食。

## 四、强制复核项确认（Mandatory Audit Checks）

> ✅ **复核1**：已穿透并引用终端产品（中国乘用车/新能源汽车）2026-2028年景气度预测——中汽协预测2026年乘用车+0.5%，新能源+15.2%；Q1实际零售-17.4%，景气度承压。

> ✅ **复核2**：已审计跨期需求透支风险——2025年以旧换新超1150万辆（占零售50%+），2026年购置税减半+补贴方式调整+高基数构成"政策退坡三部曲"，Q1数据已验证"跨期断层"风险。

> ✅ **复核3**：已分析供需边际演变方向——EHB正从"供不应求"转向"供需平衡"，EMB处于"供给稀缺"阶段但竞争者即将涌入。伯特利的先发优势窗口期为2-3年（2026-2028），之后将面临与EHB类似的竞争加剧局面。

## 五、模块二核心结论

1. **终端需求正在经历"政策退坡阵痛期"**：2026年乘用车市场近乎零增长，Q1零售-17.4%证实了增长动能转换的阵痛。这是解释伯特利Q1营收近乎零增长的最根本宏观背景。

2. **线控制动行业自身处于高成长期**：EHB渗透率仍将从50%→60%+提升，EMB刚开启量产元年。行业β为正值，但增速在放缓。

3. **"跨期需求透支"风险真实且已开始显现**：2024-2025年超1150万辆以旧换新实质性地前置了置换需求，2026年面临"政策退坡+高基数+需求透支"三重压力。

4. **供需边际方向不利于供应商利润率**：EHB竞争者增多、EMB即将面临同样的竞争格局。成本传导压力将在未来2-3年持续存在，伯特利的毛利率可能继续承压。

5. **但产业链位置提供缓冲**：作为安全件供应商，伯特利的制动产品具有刚性需求属性，且技术壁垒和客户粘性构成护城河。2025年新增定点605项（年化95亿元）为2026H2-2027年收入提供了一定保障。
