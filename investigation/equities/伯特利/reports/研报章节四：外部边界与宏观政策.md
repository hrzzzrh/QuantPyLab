# 研报章节四：外部边界与宏观政策

**研究日期**：2026年05月26日  
**逻辑继承**：模块三发现EMB竞争白热化（坐标系3488元价格战+专利诉讼）、产业链地位边际弱化。模块四评估外部不可控变量，包括新出现的治理风险、H股上市进展和USMCA审查临近对公司估值中枢的影响。  

---

## 一、宏观环境变量

### 1.1 折现率与估值环境

| 变量 | 现状 | 对伯特利影响 |
| :--- | :--- | :--- |
| **中国10年期国债** | ~1.7%（历史低位） | 降低折现率，理论上提升成长股估值中枢 |
| **A股市场风格** | 价值/红利优于成长（2025-2026） | 汽车零部件成长股遭遇风格逆风 |
| **流动性环境** | 适度宽松但信用传导偏弱 | 中小企业融资困难，但伯特利作为上市公司有优势 |

### 1.2 汇率影响

| 币种 | 方向 | 对伯特利影响 |
| :--- | :--- | :--- |
| **人民币/美元** | 人民币贬值压力（美国关税+中美利差） | **利好出口**：海外收入换算人民币增厚。2025年轻量化零部件出口为主的海外收入受惠 |
| **人民币/欧元** | 相对稳定 | 影响中性 |
| **墨西哥比索** | 受USMCA谈判不确定性影响 | 墨西哥工厂成本端存在波动风险 |

### 1.3 通胀与大宗商品传导

| 原材料 | 趋势 | 对毛利率影响 |
| :--- | :--- | :--- |
| 铝（轻量化零部件主材） | 2025年高位震荡 | 成本端持续承压；公司可通过规模采购部分对冲 |
| 钢材 | 产能过剩，价格偏弱 | 利好盘式制动器成本端 |
| 芯片/半导体 | 国产替代加速，价格下行 | 利好电子电控产品成本端 |
| 稀土永磁（电机材料） | 价格稳定 | 机器人零部件成本可控 |

**综合判断**：原材料成本端整体中性偏正面，不是当前毛利率压缩的主要驱动力。

## 二、地缘政治风险——供应链重构的实质性冲击

### 2.1 北美市场：USMCA重谈与关税壁垒

| 风险因子 | 程度 | 影响链路 |
| :--- | :--- | :--- |
| **美国对华汽车零部件关税** | 极高 | 叠加301条款+芬太尼关税+对等关税，中国产零部件对美综合税率高达70%+ |
| **USMCA重谈（2026年）** | 极高 | 要求汽车零部件"北美制造"比例从75%提至85%，堵截中国借道墨西哥转口 |
| **墨西哥对华关税** | 高 | 2026年起对华汽车征50%关税（整车），零部件10-50%不等 |
| **供应链"去中国化"** | 高 | 通用/Tesla要求供应商2027年前剔除中国零部件 |

**伯特利的应对策略与风险评估**：

| 策略 | 进展 | 效果评估 |
| :--- | :--- | :--- |
| **墨西哥工厂** | 一期已投产，二期推进中 | 满足USMCA原产地规则，可规避美国关税。但墨西哥可能对中国资本投厂施加限制 |
| **H股上市计划** | 筹划中（2025年12月公告） | 建立国际资本平台，助力海外产能扩张与品牌国际化 |
| **客户结构全球化** | 覆盖通用、Stellantis、福特、沃尔沃等 | 降低单一市场依赖，但北美仍是最大海外市场 |
| **本地化供应链** | 墨西哥本地采购+全球平台供货 | 需持续加大本地化率以满足USMCA要求 |

### 2.2 欧洲市场：关税壁垒相对可控

- 中欧达成MIP（最低进口价格）协议替代高额反补贴税
- 对零部件（非整车）关税相对温和（约10%基础税率）
- 伯特利欧洲客户（Stellantis、沃尔沃、雷诺）以本地化供货为主，受关税影响较小

### 2.3 地缘风险综合评级

| 风险维度 | 评级 | 说明 |
| :--- | :--- | :--- |
| 北美市场准入 | **高风险** | USMCA重谈+墨西哥摇摆立场构成最大不确定性 |
| 海外资产安全 | 中等 | 墨西哥工厂体量尚小（占收入<10%），可承受最坏情境 |
| 技术脱钩风险 | 低 | 制动系统核心技术自主可控，不依赖美国技术 |
| 供应链安全 | 低 | 上游芯片/传感器国产化率持续提升 |

## 三、产业政策——"双刃剑"效应

### 3.1 利好伯特利的政策

| 政策 | 影响 |
| :--- | :--- |
| **GB 21670-2025（新增EMB法规）** | 2026年1月实施，为EMB量产扫清法规障碍，伯特利作为全球首家获ASIL-D认证的企业率先受益 |
| **GB 17675-2025（线控转向法规）** | 2026年7月实施，删除机械连接强制要求，利好伯特利线控转向布局 |
| **"十五五"智能底盘标准化** | 工信部明确推进智能底盘标准制修订，行业标准化有利于龙头 |
| **新能源汽车产业支持** | 国家战略方向不变，线控制动作为标配受益 |
| **反"内卷"政策** | 中汽协等机构倡议反对无底线价格战，有利于稳定产业链利润预期 |

### 3.2 对伯特利不利的政策

| 政策 | 影响 |
| :--- | :--- |
| **新能源汽车购置税减半（2026年起）** | 增加购车成本，压降终端销量预期 |
| **以旧换新补贴方式调整** | 从"定额制"改"比例制"，低价车补贴缩水，压制汽车消费总量 |
| **多地补贴暂停** | 九成省份"地补"暂停，短期终端需求承压 |
| **反补贴调查风险** | 若EMB出口放量，可能引发海外反补贴调查 |

### 3.3 政策影响净评估

政策组合对伯特利的影响呈现"**短期压力+中长期利好**"的格局：
- **短期（2026年）**：购置税减半+补贴退坡压制终端需求、毛利率承压
- **中期（2027-2028年）**：EMB/线控转向法规红利释放、标准统一有利于龙头份额集中
- **长期（2028+）**："十五五"智能底盘战略支持、"软件定义底盘"产业升级

## 四、模块四核心结论

1. **北美地缘风险是最大外部不确定性——且7月USMCA审查迫在眉睫**：USMCA三国联合审查将于2026年7月1日正式启动。美国预计将寻求更严格的区域含量要求、限制中国零部件通过墨西哥转口、甚至可能要求美国特有含量条款。墨西哥已对华汽车征收50%关税、对部分零部件征收5-50%关税，立场趋硬。伯特利墨西哥工厂2025年录得全年亏损，海外扩张叙事正面临双重压力（业绩亏损+政策风险）。

2. **国内产业政策方向利好伯特利——豫北转向反垄断通过是关键里程碑**：EMB/线控转向法规落地为伯特利提供了合规先发优势。2026年5月6日，豫北转向收购获得反垄断审查通过，收购可正式实施——这是补全转向能力、实现底盘全域布局的决定性一步。5月21日，伯特利正式向港交所递交H股上市申请（中金独家保荐），从"筹划中"进入正式程序，募资将用于研发、产能扩张及全球化布局。

3. **短期政策（购置税减半+补贴退坡）构成需求逆风，且4-5月数据证实恶化**：4月零售-21.5%，5月1-17日-23%，需求下滑在加深。

4. **汇率和原材料环境——出现边际变化**：人民币贬值利好出口；但碳酸锂暴涨130%（7.5→17万/吨）、芯片涨价180%正在推高整车制造成本，迫使主机厂停止价格战（参见模块二）。这对伯特利而言可能是双刃剑——短期年降压力缓解，但终端需求因购车成本上升而进一步受抑。

5. **治理风险——新增外部边界因子**：每日经济新闻2026年5月25日报道揭示：①董秘陈忠喜因关联交易信披违规多次被上交所警告；②公司账面资金充裕（17.82亿现金+28亿可转债），却额外举债7.8亿推进豫北转向收购；③墨西哥工厂2025年亏损；④豫北转向OCF为-1.71亿。治理层面的多重信号增加了公司层面的不确定性溢价。

6. **H股上市从"筹划"推进至"递表"**：5月21日正式递交申请是战略性进展，但短期将面临：①股本稀释（预计10-15%），②A/H价差不确定性，③跨境监管合规成本。
