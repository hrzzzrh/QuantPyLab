# 研报章节三：竞争格局与地位

**研究日期**：2026年06月09日

---

## 前序模块核心结论继承

模块一发现：客户高度集中（前两大57.9%），毛利率趋势性下滑，海外为增长引擎
模块二发现：特高压高端市场供需紧平衡，中低端过剩，复合绝缘子加速替代

本章需验证：公司的市占率优势是否可持续？护城河有多深？定价权几何？

---

## 一、竞争格局扫描：寡头格局的形成与强化

### 1.1 特高压瓷绝缘子：四家分食的寡头市场

特高压瓷绝缘子是整个绝缘子行业壁垒最高的细分领域，当前国内具备投标资格的企业**仅约4家**：

| 企业 | 特高压瓷绝缘子市占率（估） | 核心优势 | 产能规模 |
| :--- | :--- | :--- | :--- |
| **大连电瓷** | **~50%** | 全系列、全电压等级覆盖，百年品牌，唯一一家具备±800kV以上直流批量供货能力 | 年产能近千万只 |
| 苏州电瓷厂 | ~15-20% | 历史悠久，部分交流线路 | 中等 |
| NGK（日本） | ~10-15% | 国际品牌，技术高端但成本劣势 | 全球布局 |
| 内蒙古精诚 | ~10-15% | 区域优势，特定标段 | 较小 |

**竞争格局核心特征**：
1. **高度集中**：CR4接近90%，大连电瓷一家独大
2. **格局稳定**：过去10年竞争格局未发生根本变化，新进入者极难突破（电网供应商认证周期3-5年）
3. **份额持续提升**：2024年特高压1-2批中标占比28%，2025年同期提升至46%（+18pct），体现龙头地位强化
4. **直流领域更强**：历史最高中标金额占比达70%，直流特高压对产品质量要求更高，竞争更少

### 1.2 输变电批次市场（中高压）：寡头+追随者格局

2026年5月最新国网第二次输变电设备招标数据——支柱绝缘子市场：

| 企业 | 中标金额（万元） | 占比 | 
| :--- | :--- | :--- |
| 唐山高压电瓷 | 2401 | 26.1% |
| **大连电瓷** | **2357** | **25.7%** |
| 湖南阳东电瓷 | 1410 | 15.4% |
| 西安西电高压电瓷 | 1226 | 13.3% |
| 醴陵华鑫电瓷 | 1005 | 10.9% |
| 中材江西电瓷 | 787 | 8.6% |
| **合计（6家）** | **9185** | **100%** |

**解读**：在支柱绝缘子领域，公司与唐山高压电瓷形成双寡头格局（CR2=51.8%），差距极小（仅44万元）。公司在悬瓷领域的绝对优势未完全延伸至支柱绝缘子，但支柱绝缘子体量尚小（2025年营收仅0.65亿），对公司整体格局影响有限。

### 1.3 中低压及复合绝缘子：红海竞争

- 中低压瓷绝缘子：竞争对手超1200家，同质化严重，平均毛利率<15%，非公司主战场
- 复合绝缘子：渗透率持续提升（新增采购占比>55%），但竞争主体多，公司该品类2025年毛利率仅23.65%
- 玻璃绝缘子：山东瑞泰、浙江泰伦、南京电气等，在中低压线路替代效应显著

## 二、护城河识别：四维壁垒体系

### 2.1 资质壁垒（最强壁垒，☆☆☆☆☆）

**"准入即壁垒"**：
- 国家电网供应商资质认证周期长达3-5年，需要通过型式试验、资格预审、业绩考核等多重环节
- 特高压领域要求具有同电压等级的运行业绩，形成"先有业绩才能投标，不中标永远没有业绩"的封闭循环
- 国内具备±800kV及以上特高压直流瓷绝缘子批量供货能力的企业，**大连电瓷为唯一一家本土企业**
- NGK（日本）虽技术实力雄厚，但在国网招标中受国产化率要求限制，只能作为补充供应商

### 2.2 技术壁垒（核心壁垒，☆☆☆☆☆）

**百年技术沉淀难以复制**：
- 瓷绝缘子配方是核心know-how（瓷、釉配方、粘合剂），涉及材料科学、热工工艺等多学科交叉
- 公司自主研制的840kN大吨位盘形悬式瓷绝缘子已实现量产（行业最大吨级之一）
- 累计获国家科学技术进步奖特等奖和一等奖各1项
- 2025年新增发明专利5件，实用新型16件，参与编制2项国网企业标准
- 实验室通过国际CNAS认证，具备国际标准测试能力

**技术代际领先的证据**：
- 公司自主研发的圆柱头产品结构优化项目已结项并批量生产，成功保障"甘肃-浙江"特高压工程
- 悬式棕釉新配方、高耐电弧性能混炼胶配方等研发项目结项
- 与清华大学、大连理工大学等联合推进高性能水泥胶合剂等前沿项目

### 2.3 规模与成本壁垒（☆☆☆☆）

**三基地协同的产能优势**：
- 大连（总部+高端制造）、江西（智能化新基地）、福建三大基地，年产能近千万只
- 江西工厂二期2025年全面达产，为数字化智能工厂，获评"省工业互联网平台"和"省级智能制造标杆"
- 新建产能的规模效应持续释放，单位固定成本有望下降
- **但需注意**：原材料（62%成本为直接材料）价格波动仍是主要变量，规模优势无法完全对冲

### 2.4 品牌与客户粘性壁垒（☆☆☆☆）

- 百年"三箭"品牌，国内绝缘子行业最具影响力品牌
- 深度参与所有标志性特高压工程：首条1000kV交流、世界首条±800kV直流、±1100kV昌吉-古泉
- 客户关系深度绑定：2025年国网收入6.26亿元（占营收35.16%），连续多年中标率第一
- 海外已与近百个国家建立业务往来，出口历史超60年

## 三、产业链定价权审计

### 3.1 对上游：议价能力中等偏弱

| 上游要素 | 供应商集中度 | 公司议价能力 | 备注 |
| :--- | :--- | :--- | :--- |
| 氧化铝/高岭土/长石 | 分散（矿企） | **中等** | 属大宗商品，难以锁定长期低价，但可通过配方调整替代 |
| 钢材（铁帽/钢脚） | 分散 | **中等** | 通过子公司亿德金具自供部分 |
| 天然气 | 垄断性（燃气公司） | **弱** | 价格受国际能源市场影响，2025年燃气成本占比约11.4% |
| 硅橡胶 | 中等集中 | **中等偏弱** | 受石油价格波动影响，年均涨幅6.2% |

**验证——财报科目审计**：
- 应付账款2.63亿（2025年末），应付票据2.75亿，合计5.38亿，对上游存在一定占款能力
- 前五大供应商占比22.73%，集中度适中，降低单一供应商依赖风险

### 3.2 对下游：议价能力偏弱但存结构性差异

| 下游客户 | 特征 | 公司议价空间 |
| :--- | :--- | :--- |
| 国家电网（~35%营收） | 超级买方、集中招标 | **弱**——但特高压领域因技术垄断具一定溢价能力 |
| 南方电网（~23%营收） | 超级买方、集中招标 | **弱** |
| 海外电网公司 | 分散、标准各异 | **中**——毛利率35.29%显著高于国内31.14%，证明海外议价能力更强 |
| EPC总包商 | 中等集中 | **中** |

**验证——财报科目审计**：
- 应收账款6.43亿（2025年末），占营收36.1%，应收账款周转天数约130天，**回款周期偏长**
- 预收账款/合同负债1.13亿（2026Q1），有一定预收款但规模不大
- 核心结论：在国内两大电网面前，公司议价能力天然不足，但**特高压领域的技术垄断性提供了一定的议价缓冲**

## 四、外部数据交叉验证

### 4.1 真实订单验证（招标公示穿透）

**最新中标动态（2026年6月9日）**：
- 国家电网2026年特高压项目第二次材料招标：大瓷材料预中标瓷绝缘子、支柱绝缘子，合计约**1.90亿元**，占2025年营收10.66%
- 参考：2025年5月特高压第二次材料招标中标约2.90亿元，占2024年营收19.38%
- 2026年5月输变电第二次招标合计约0.90亿元，占2025年营收5.03%

**订单趋势**：单次特高压中标金额从2025年的2.90亿降为2026年的1.90亿（-34%），但需注意特高压招标具有明显的批次波动性，不宜以单次数据线性外推。2025年全年公司特高压中标超5亿元、批次中标超4亿元，预期2026年维持或略增。

### 4.2 实物产能校验

**江西工厂产能扩张验证**：
- 江西工厂二期项目于2025年全面达产，总投资规模可查
- 2025年悬瓷绝缘子生产量116,766吨（+16.43%），销售量110,111吨（+27.96%）
- 产能扩张已转化为实际产量增长，验证了扩产逻辑的物理落地

### 4.3 供应链生态穿透

**国网采购金额结构（2025年1-2批）**：
- 材料+设备共计18.59亿元
- 瓷绝缘子占比约60%（~11.2亿）、复合绝缘子16%（~3.0亿）、玻璃绝缘子23%（~4.3亿）
- 瓷绝缘子在特高压领域的主导地位明确

### 4.4 竞争对手动态

- **中国西电**（601179）：全产品线电力设备巨头，年营收超200亿，绝缘子仅为其业务之一，与大连电瓷在支柱绝缘子领域直接竞争
- **苏州电瓷厂**：专注瓷绝缘子，规模较小（非上市，数据不透明），为大连电瓷在特高压领域的主要国内竞争对手
- **NGK（日本）**：全球技术标杆，但在国内受国产化政策限制，份额持续收缩
- **行业整合加速**：2025年电瓷行业12起并购涉及30亿元，小型企业加速退出

## 五、模块三核心结论

1. **龙头地位稳固且强化**：特高压瓷绝缘子市占率~50%，且2025年中标占比从28%提升至46%，份额加速集中
2. **护城河极深**：资质壁垒（3-5年认证周期+业绩闭环）+技术壁垒（配方know-how+110年积淀）+规模壁垒（三基地协同）形成三重保护
3. **定价权存在结构性差异**：对国网议价弱，但对特高压品类有一定技术溢价；海外市场议价能力明显更强（毛利率高4pct）
4. **行业格局利好龙头**：中小竞争者加速退出（1200家→集中度提升），高端市场寡头格局将持续强化
5. **核心风险点**：如果国网招标规则从"最低价中标"进一步向"技术评分"倾斜，公司将显著受益；反之，若价格权重维持高位，毛利率可能继续承压
