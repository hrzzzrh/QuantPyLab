# 研报章节四：外部边界与宏观政策

**研究日期**：2026年06月09日

---

## 前序模块核心结论继承

模块一发现：海外收入占比23.56%、毛利率高于国内，但客户集中度高
模块二发现：十五五4万亿确定性极强，全球电网投资共振
模块三发现：公司在特高压领域护城河极深，但对国网议价能力弱

本章需评估：宏观变量（汇率、通胀）如何冲击盈利？地缘冲突如何影响海外战略？政策环境边际变化方向？

---

## 一、宏观环境变量

### 1.1 人民币升值：海外利润的"隐形税"

**当前汇率环境**：
- 2025年4月以来，人民币兑美元从7.35附近持续升值，至2026年6月已突破6.82，累计升值超7%
- 2026Q1，A股非金融上市公司汇兑损失达497亿元，较去年同期159亿元收益同比多损失657亿元
- 出海企业普遍"增收不增利"：比亚迪汇兑损失~21亿元，三一重工~8亿元

**对大连电瓷的传导分析**：

公司2025年海外收入4.19亿元，主要以美元报价和结算。年报明确将"汇率变动风险"列为经营风险之一，并表示"积极运用各类结汇工具，尽量规避风险损失"。

**定量估算**：
- 假设海外收入中70%以美元结算（~2.93亿元）
- 若人民币升值5%（如7.0→6.65），则折算损失约1500万元
- 占2025年归母净利润2.12亿元的约7%
- **影响可控，但2026Q1已有汇兑损失迹象**：财务费用从上年同期23万元飙升至877万元（+3568%），年报解释为"利息费用增加及汇兑收益减少"

**核心判断**：汇率风险对大连电瓷属中等影响，原因如下：
1. 海外收入占比约24%，低于纯出口型企业（通常>50%）
2. 公司正推进海外本地化生产（沙特建厂），将形成"收入成本同币种"的自然对冲
3. 可通过远期结汇等工具部分对冲，但公司目前套保规模可能不足
4. **积极面**：人民币升值降低了公司进口原材料的成本（氧化铝等以美元计价的大宗商品），存在一定自然对冲

### 1.2 利率与折现率：估值锚定影响

- 当前中国10年期国债收益率约2.7%，处于历史低位
- 美联储因中东战事推高通胀，降息预期推迟至2027年
- 低利率环境有利于DCF估值，但无风险利率下行空间已有限

**对估值的影响**：中性偏正面。低利率支撑成长股估值，但PE中枢的进一步抬升需要盈利增速配合。

### 1.3 通胀与成本传导

**原油/天然气价格**：
- 布伦特原油因中东战事维持在93-96美元/桶（战前~70美元），若局势恶化可能升至130-150美元
- 天然气价格同步上涨，直接影响公司燃料成本（天然气占悬瓷成本11.4%）
- 船用燃油成本暴涨70%，推升海运费，公司出口物流成本上升

**成本传导效率**：
- 年报已披露：2025年财务费用大幅增加，"利息费用增加及汇兑收益减少"
- 年报"中东战争风险"部分明确："全球油价上涨，预计未来海运成本会有所增加，库存商品短时间会增加"
- 公司通过光伏绿电（自发自用35-80%）部分对冲能源成本

## 二、地缘政治风险

### 2.1 美以-伊朗战争：直接冲击沙特战略

**冲突概况**（截至2026年6月9日）：
- 2月28日战争爆发，已持续100+天
- 霍尔木兹海峡航运量骤降90%+，美伊双方形成"双重封锁"
- 和平谈判反复，4月8日停火协议脆弱，6月初伊朗再次向以色列发射导弹

**对公司沙特业务的冲击**（年报原文）：
> "目前美以—伊朗战争致使沙特位于霍尔木兹海峡的达曼港暂时无法通行，后续或计划改道其他航线通行，会增加一定的内陆运输成本，同时由于全球油价上涨，预计未来海运成本会有所增加，库存商品短时间会增加。"

**定量影响评估**：
| 影响维度 | 冲击方向 | 影响程度 | 持续时间预判 |
| :--- | :--- | :--- | :--- |
| 沙特订单交付 | 延迟/改道 | 中等 | 短期（3-6个月） |
| 海运成本 | 大幅上升 | 中等偏大 | 中期（战争持续期间） |
| 中东新增订单 | 推迟但不取消 | 中等 | 短期 |
| 全球油价→原材料成本 | 上升 | 中等 | 中期 |

**辩证视角**：
- **短期利空**：交付延迟、成本上升、在途库存增加
- **中长期利好**：高油价→中东国家财政充裕→电网投资加速→绝缘子需求增长。沙特"2030愿景"中电网投资计划庞大，战后重建将释放更大需求
- **竞争格局优化**：中小出口商因运费/保险成本激增而退出，有利于龙头集中份额

### 2.2 全球供应链重构（China+1）

- 部分国家推动供应链多元化，但特高压设备领域中国具有绝对技术和产能优势
- 公司已通过沙特本地化生产（沙特设立子公司DACI SAUDI）应对"去中国化"趋势
- 美国FERC发布《Energized for 2026》，加快电网建设，但对中国产品存在关税和准入壁垒

### 2.3 贸易摩擦与关税

- 特朗普关税政策虽有反复（部分被最高法院裁定违宪），但对华加征关税趋势未根本逆转
- 公司出口以中东、东南亚为主（非美市场），受中美贸易摩擦直接影响有限
- 风险点：若美国进一步限制中国电力设备进口，将影响公司开拓美国市场的潜力（当前美国占中国绝缘子出口13.43%）

## 三、监管与产业政策

### 3.1 国家发展规划契合度：极高

大连电瓷处于国家能源战略的**核心受益位置**：

| 政策/规划 | 内容要点 | 对公司影响 |
| :--- | :--- | :--- |
| 十五五电网投资 | 国网4万亿（+40%），投产15项特高压直流 | **极大利好**——直接拉动订单 |
| "西电东送" | 2030年规模超4.2亿千瓦 | 特高压线路建设→绝缘子需求 |
| 新型电力系统 | 新能源年均新增2亿千瓦 | 新能源并网配套→增量需求 |
| 零碳园区/绿电直连 | 84个绿电直连项目获批，3259万千瓦 | 配网升级→中低压绝缘子需求 |
| "两重""两新" | 电网基础设施更新改造 | 存量替换需求 |
| 南方电网 | 2026年固投1800亿元，更新投资≥506亿元 | 第二增长曲线 |

### 3.2 环境监管：利好龙头出清

- 环保法规趋严：窑炉废气排放标准提升、高能耗产线受限
- 2025年已有15%以上小型绝缘子企业因环保不达标被关停
- 公司率先完成绿色工厂认证、碳足迹认证、ESG报告，形成"合规溢价"
- 欧盟CBAM（碳边境调节机制）实施，公司的碳足迹认证成为出口欧洲的"通行证"

### 3.3 行业准入：壁垒加固

- 国网供应商分级管理制度趋严，引入全生命周期成本（LCC）评估
- 对绝缘子的机械强度、污秽等级、使用寿命提出更高要求
- 特高压资质维持严格的"业绩闭环"，新进入者极难突破
- 结论：**政策方向持续利好龙头，加速行业集中度提升**

---

## 四、模块四核心结论

1. **汇率风险可控但需关注**：海外收入占比24%，汇兑损失对利润影响约5-8%，公司需加强套保
2. **中东战事短期阵痛、中长期利好**：当前影响交付和成本，但战后重建将释放更大需求，沙特本地化布局具有战略远见
3. **十五五是最大确定性**：4万亿电网投资政策锁定，公司处于核心受益位置
4. **环保和准入政策持续利好龙头**：中小竞争者加速退出，合规溢价逐渐显现
5. **核心外部风险排序**：中东地缘>汇率波动>贸易摩擦>原材料涨价
