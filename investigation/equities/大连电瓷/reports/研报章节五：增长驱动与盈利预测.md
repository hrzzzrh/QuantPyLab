## 一、分业务 P×Q 模型的证据链锚定

### 1.1 悬瓷绝缘子（收入占比82.8%，核心利润来源）

**量（Q）——确定性评分：5/5**

| 证据维度 | 证据内容 | 证据来源 |
| :--- | :--- | :--- |
| 产能上限 | 江西二期2025年末全面达产，三基地总产能~120,000吨 | 2025年报（P23-24） |
| 2025年产量 | 悬瓷生产量116,766吨（+16.43%），销售量110,111吨（+27.96%） | 2025年报（P29） |
| 2026年生产计划 | 子公司江西大瓷"2026年生产计划基本排满" | 2026年1月公告 |
| 在手订单 | 2025年特高压中标>5亿+批次>4亿+海外>10亿，大量订单结转至2026年 | 公司公告 |
| 最新中标 | 2026年6月9日特高压二次材料预中标~1.90亿元 | 公司公告 |

**2026E销量**：在产能利用率已接近极限（97%）的背景下，量增主要来自：
- 江西二期完整年度运行（2025年仅部分月份达产）
- 三基地协同排产优化
- **预计2026年悬瓷销量125,000-130,000吨（+13-18%）**

**价（P）——确定性评分：3/5**

| 证据维度 | 分析 |
| :--- | :--- |
| 产品结构升级 | 840kN大吨位产品量产，ASP高于普通产品约15-20%；2025年甘肃-浙江特高压工程使用大吨位 |
| 国网招标压价 | 2025年批次招标均价同比-4.8%，但特高压产品具技术溢价，受影响较小 |
| 海外均价更高 | 海外毛利率35.29% vs 国内31.14%，海外占比提升将拉高综合ASP |

**2026E ASP**：悬瓷综合ASP（万元/吨）：
- 2023年：约0.99（8.40亿/估算）
- 2024年：约1.42（12.26亿/8.61万吨）
- 2025年：约1.34（14.73亿/11.01万吨）
- **2026E：~1.38（+3%），产品结构升级部分对冲了批次压价**

---

### 1.2 海外业务（收入占比23.6%，增速引擎）

**量（Q）——确定性评分：4/5**

- 2025年海外收入4.19亿（+50.8%），新签海外订单超10亿元
- 沙特、菲律宾、孟加拉、韩国、意大利、英国等重点市场持续放量
- **制约因素**：中东战事影响沙特达曼港通行，部分订单交付延迟

**2026E海外收入**：5.5-6.0亿（+31-43%）
- 假设在手订单10亿中约50-60%在2026年确认收入
- 再考虑新增订单和交付延迟，保守取中值

**价（P）——确定性评分：3/5**

- 海外毛利率35.29%，但人民币升值+海运费上升将挤压
- 2026E海外毛利率：**32-34%**（下行风险，但显著高于国内）

## 四、2027E 初步推演

**核心逻辑**：
- 十五五特高压建设2026-2028为高峰期（高盛预测2026年增速24%，2027-2030年4条/年）
- 海外沙特本地化工厂预计2027年投产，打开中东市场增量
- 产能无新增大型资本开支→折旧平稳，利润弹性增大

**2027E中性假设**：
- 营收：~28亿（+19%）
- 归母净利润：~3.5亿（+23%）
- EPS：~0.81元

## 五、证据链强度总结

1. **销量增长（Q）**：确定性极高（★★★★★），产能已落地、订单已锁定
2. **价格趋势（P）**：确定性中等（★★★☆☆），特高压技术溢价存在但国网压价持续
3. **毛利率回升**：确定性中等（★★★☆☆），Q1信号积极但需后续季度验证
4. **海外增长**：确定性较高（★★★★☆），在手订单支撑，但地缘风险不可忽视
5. **2026E EPS 0.65元（加权）/0.66元（中性）**，对应现价13.77元的Forward PE约**20.9x**
