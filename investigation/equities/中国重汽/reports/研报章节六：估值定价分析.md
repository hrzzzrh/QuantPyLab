# 研报章节六：估值定价与全变量风险对冲

**研究日期**：2026年05月09日

**前置研究红线继承**：
- 模块五：基准 EPS 2026E = 1.63 元（PE 分位 31%，PB 分位 38%）——以此为估值起点
- 模块一：OCF 与净利润严重背离，应收/存货双重膨胀——必须通过估值折扣体现
- 模块二/三：行业周期性 + 市占率提升 + 出口护城河——但利润质量极差，估值溢价空间有限

## 一、价值锚定——适配性方法选择

### 1.1 方法论适用性论证

中国重汽是一家**高营收、低毛利、强周期、重资产**的制造业企业，其商业模式决定了对估值方法的特定要求：

| 估值方法 | 适用性 | 理由 |
| :--- | :---: | :--- |
| **PE（市盈率）** | ★★★★★ 核心 | 制造业最通用；公司盈利为正且可预测；A 股以 PE 为主的定价范式 |
| **PB（市净率）** | ★★★ 辅助 | ROE 10.58% > 资金成本 7-8%，PB 应 >1.0；可校验 PE 定价 |
| **PEG** | ★★ 参考 | 2026E EPS 增速 ~15%，PEG ≈ 0.91（当前 PE 13.7 / 增速 15%），低估区间 |
| **DCF** | ★ 不推荐 | 重卡行业周期性极强，10 年自由现金流预测的可靠性几乎为零 |
| **PS（市销率）** | ★ 辅助 | 当前 PS 仅 0.40，在同行业极低；可交叉验证 |
| **EV/EBITDA** | ★★ 参考 | 公司零有息负债，EV ≈ Market Cap - Net Cash ≈ 261 - 85 = 176 亿 |

**选择路径**：**PE 为主（含 PEG 校验）+ PB-ROE 交叉验证 + PS 底部信号**

### 1.2 历史分位——当前估值坐标

| 指标 | 当前值 | 历史区间 | 当前分位 | 分位定性 |
| :--- | :---: | :---: | :---: | :--- |
| PE(TTM) | 14.43 | 8.26 - 2062.12 | **31.0%** | 明显偏低 |
| PB | 1.57 | 0.97 - 32.42 | **38.0%** | 偏低 |
| PS(TTM) | 0.40 | - | - | 历史极低值附近 |

**分位解读注意事项**：
- PE 历史最低 8.26 发生在 2013 年和 2022 年——两次均对应行业大衰退，当前行业基本面远好于彼时
- 但 PE 均值 17.16 是长期中枢，包含了 2020 年国六切换前的景气巅峰（PE 显著压缩但仍高位）
- A 股整体估值体系在 2024-2025 年经历了系统性下移，历史分位的参考意义有所削弱

## 二、基准市值推演

### 2.1 PE 估值法（主力方法）

| 情景 | 2026E EPS | 合理 PE 区间 | 目标价区间 | 逻辑依据 |
| :--- | :---: | :---: | :---: | :--- |
| 熊 | 1.20 元 | 13-15x | 15.6-18.0 | 景气下行期的自然压缩 |
| 基 | 1.63 元 | 15-18x | **24.5-29.3** | 中枢对冲 OCF 风险后合理区间 |
| 牛 | 1.92 元 | 16-20x | 30.7-38.4 | 出口加速 + 市占率跃升情境 |

**基准情景 PE 区间推导**：
- 行业平均 PE 中枢 ~15-17x（考虑了 A 股估值中枢下移趋势）
- 中国重汽龙头溢价 +1-2x（市占率 #1 + 出口壁垒）
- OCF 质量折价 -1x
- 关联交易风险折价 -0.5x
- **综合：15-18x**
- 取中值 **16.5x → 目标价 26.9 元 → 市值为 314 亿**

### 2.2 PB-ROE 交叉验证

ROE 2025 = 10.58%，2026E 基准 ROE ≈ 19.1 亿 / (161.63 + 19.1/2) ≈ 11.2%

合理 PB = ROE / COE：
- 若 COE = 8%（无风险利率 2.5% + 股权风险溢价 5.5%）：PB = 11.2%/8% = **1.40x**
- 对应目标价 = 1.40 × 17.5 元（2026E BVPS）≈ **24.5 元**

PB-ROE 框架给出的目标价 24.5 元与 PE 框架区间下沿 24.5 元完美重合——**24.5 元可作为本轮分析的"硬底线"定价**。

### 2.3 PEG 校验

2026E EPS 增速 15%（基准），现价 PE 14.43 → PEG = 14.43/15 = **0.96**

PEG < 1.0 通常被视为低估。若 PEG 回到 1.0 且增速维持 15%，合理 PE = 15 → 对应目标价 24.5 元。这与 PE 和 PB-ROE 框架高度一致。

## 三、全变量风险对冲审计矩阵（强制输出）

**基准市值**：314 亿（基准目标价 26.9 元 × 11.69 亿股）

### 「估值影响力过滤」审慎思考

在进入具体调节前，必须首先执行影响力过滤：发掘出的因子绝大多数只是市场杂音，不能改变估值中枢。以下因子中，**仅少数真正具备定价权**：

| 因素 | 是否进入调节？ | 理由 |
| :--- | :---: | :--- |
| 出口连续 21 年第一 | ❌ 不进 | 已体现在 EPS 增速中——出口增长本身就是利润预测的一部分 |
| 市占率第一 | ❌ 不进 | 同上，已体现在 V（量）假设中 |
| 新能源布局全面 | ❌ 不进 | 远期 OPTIONALITY，当前对估值无实质影响 |
| OCF 质量极差 | ✅ 进入 | 不可量化为具体成本（何时能修复、修复到何种程度均不确定），必须进 PE |
| 关联交易 55% | ✅ 进入 | 治理风险影响折现率/风险溢价 |
| 零有息负债 | ✅ 进入 | 资产负债表质量好 → 降低尾部风险 → 小幅溢价 |

### 对冲审计矩阵

| 审计维度 | 核心因子 | 审慎过滤（估值影响力） | 调节建议 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质** | 零有息负债 + 充足现金(84.84亿) | ✅ 该特质降低破产风险、为逆周期扩张储备弹药——具定价权 | **+5%** | 在强周期行业中，零有息负债意味着穿越周期的生存能力远超同行——这是一种"反脆弱性"，应享受小幅估值溢价 |
| **内部：企业底层特质** | 关联交易占比 55% vs 山东重工 | ✅ 独立定价权存疑——治理风险需要溢价补偿 | **-5%** | 大股东同时是最大客户和供应商，中小股东利益保护存在结构性风险。这种"非独立性"应对估值给予合理折扣 |
| **宏观：政策与战略对齐** | "以旧换新"补贴无缝续期至 2026 年底 | ❌ 不进 | 政策利好已在 EPS 预测的 V 假设中充分体现（行业 110 万基本盘），无需重复计算进 PE | **0%** | |
| **宏观：政策与战略对齐** | 2027 年国四补贴退出风险 | ❌ 不进 | 属于远期权衡——若 2027 年政策退出导致 EPS 下滑，届时估值自然会压缩。2026 年可暂时不扣减 | **0%** | |
| **宏观：地缘与外部约束** | 出口地缘多元化有效对冲俄罗斯下滑 | ❌ 不进 | 多元化进展已反映在出口 V 假设的增长中（+14.4%） | **0%** | |
| **市场：流动性与风格** | PE 31%分位 + PB 38%分位（历史低估区） | ❌ 不进 | 当前估值本身就是结果而非原因——不能因"处于分位低位"而机械地"上修"，此为由果推因的逻辑错误 | **0%** | |
| **市场：流动性与风格** | OCF/NP = 0.23x（利润无现金含量） | ✅ 进入——信心层面的持续性折价 | **-8%** | 2025 年 OCF 仅 5.63 亿 vs NP 23.98 亿。投资者无法确定利润是否"真实"。这种不确定性必须通过更低 PE 来补偿——因为如果现金永远收不回来，利润数字只是纸上富贵。这是本报告最重要的估值折价因子 |
| **市场：流动性与风格** | 中金/中信建投/开源等多家机构给予正面评级 | ❌ 不进 | 卖方评级属市场情绪，不应作为估值锚定依据 | **0%** | |
| **综合调节系数** | —— | —— | **Σ = -8%** | — |

### 修正目标市值与目标价

修正目标市值 = 314 亿 × (1 + 0.05 - 0.05 - 0.08) = **314 × 0.92 = 289 亿**

对应目标价 = 289 亿 / 11.69 亿股 = **24.7 元**

**逻辑纯净性校验**：
- E（利润）中已包含：出口增长、市占率提升、补贴续期、钢价温和反弹
- PE 中调整的仅三项：零有息负债溢价(+5%)、关联交易折价(-5%)、OCF 质量折价(-8%)
- 无重复计算 ✓ ，E 与 PE 分工清晰 ✓

## 四、确定性与胜率矩阵

| 情景 | 目标价 | 概率 | 2026E EPS | 隐含 PE |
| :--- | :---: | :---: | :---: | :---: |
| 乐观 | 34.6 元 | 20% | 1.92 元 | 18.0x |
| 基准 | 26.9 元 | 50% | 1.63 元 | 16.5x |
| 悲观 | 16.8 元 | 30% | 1.20 元 | 14.0x |
| **加权目标价** | **25.5 元** | 100% | | |

当前股价：**22.34 元**（2026-05-08 收盘）

盈亏比（基准情景）：
- 上行空间 = (26.9 - 22.34) / 22.34 = **+20.4%**
- 下行风险 = (22.34 - 16.8) / 22.34 = **-24.8%**
- **盈亏比 = 20.4% / 24.8% = 0.82x**（不理想）

盈亏比（乐观情景）：
- 上行空间 = (34.6 - 22.34) / 22.34 = **+54.9%**
- 下行风险 = -24.8%
- **盈亏比（乐观）= 2.21x**（较有吸引力）

## 五、极限压力测试——逻辑证伪与"地狱价格"

### 5.1 黑天鹅情景定义

**"三重打击"情景**（概率 ~5%）：
1. **2027 年国四补贴全面退出**：行业销量骤降 20% 至 ~88 万辆
2. **钢材价格暴涨 30%**（中东冲突全面升级 ⇒ 能源成本飙升 ⇒ 钢铁供给收缩）
3. **出口遭遇系统性贸易壁垒**（主要出口国集体加征关税 15-25%）

### 5.2 黑天鹅下的财务推演

| 科目 | 基准 2026E | 黑天鹅 | 变动 |
| :--- | ---: | ---: | :--- |
| 营收（亿） | 665 | **430** | -35.3% |
| 毛利率 | 6.77% | **3.5%** | -3.27pct |
| 归母净利（亿） | 19.1 | **净亏损 ~5-8 亿** | — |
| 资本市场反应 | PE 14.4x | PE 极度压缩至历史极端 10x（假设仍有微利）或 PB 定价 | — |

### 5.3 "地狱价格"测算

在极端情景下，估值按 PB 清算逻辑：
- 净资产（2025 年底）= 161.63 亿
- 扣除商誉(0) + 扣除存货减值（保守折 20%：74.93 × 20% = 15 亿）+ 扣除应收账款坏账（保守折 10%：202 × 10% = 20 亿）
- 调整后净资产 = 161.63 - 15 - 20 ≈ **127 亿**
- 极限股价 = 127 / 11.69 = **10.9 元/股**（极限支撑位）

**对比当前价格 22.34 元，极限下行空间约 -51%**。这是在"公司不会破产但利润归零"的假设下的终极锚点。

## 六、综合定价结论

| 定价维度 | 目标价/区间 | 对应市值 |
| :--- | :---: | :---: |
| PE 估值（基准，16.5x） | **26.9 元** | 314 亿 |
| 全变量对冲后 | **24.7 元** | 289 亿 |
| PB-ROE 硬底线 | **24.5 元** | 286 亿 |
| PEG 合理 PE(=15x) | **24.5 元** | 286 亿 |
| 中金目标价 | **29.9 元** | 350 亿 |
| **本报告综合目标价** | **25.0-27.0 元** | **292-316 亿** |
| **上行空间（vs 22.34）** | **+12% 至 +21%** | — |

**核心结论**：中国重汽当前股价 22.34 元处于低估区间，合理价格应在 25-27 元之间，有 **12%-21% 的上行空间**。但盈亏比仅 0.82x（基准情景），风险收益结构不具强烈吸引力——**仅在乐观情景（概率 20%）下才呈现高质量投资机会**（54.9% 上行/24.8% 下行 = 2.2x）。
