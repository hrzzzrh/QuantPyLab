## 一、价值锚定——适配性方法选择

### 1.1 方法论适用性论证

中国重汽是一家**高营收、低毛利、强周期、重资产**的制造业企业，其商业模式决定了对估值方法的特定要求：

| 估值方法 | 适用性 | 理由 |
| :--- | :---: | :--- |
| **PE（市盈率）** | ★★★★★ 核心 | 制造业最通用；公司盈利为正且可预测；A 股以 PE 为主的定价范式 |
| **PB（市净率）** | ★★★ 辅助 | ROE 10.58% > 资金成本 7-8%，PB 应 >1.0；可校验 PE 定价 |
| **PEG** | ★★ 参考 | 2026E EPS 增速 ~15%，PEG ≈ 0.91（当前 PE 13.7 / 增速 15%），低估区间 |
| **DCF** | ★ 不推荐 | 重卡行业周期性极强，10 年自由现金流预测的可靠性几乎为零 |
| **PS（市销率）** | ★ 辅助 | 当前 PS 仅 0.40，在同行业极低；可交叉验证 |
| **EV/EBITDA** | ★★ 参考 | 公司零有息负债，EV ≈ Market Cap - Net Cash ≈ 261 - 85 = 176 亿 |

**选择路径**：**PE 为主（含 PEG 校验）+ PB-ROE 交叉验证 + PS 底部信号**

**分位解读注意事项** **[2026-06-08更新]**：
- 当前PE 15.02较5月31日的13.88显著上升，分位从27.5%升至44.8%——估值已从"深度低估"区间移向"合理偏低"区间。上行主要来自股价上涨(+8.2%)而非盈利增长的主动驱动
- PE历史中位数约15.55，当前折价约3.5%——估值安全边际已大幅收窄

- **重要警示**：5月31日研报中"深度低估"的判断已不再成立，当前估值处于历史中枢附近，进一步上行需要基本面的实质性改善（如Q2 OCF转正、半年报超预期）而非估值修复

## 二、基准市值推演

### 2.1 PE 估值法（主力方法）

| 情景 | 2026E EPS | 合理 PE 区间 | 目标价区间 | 逻辑依据 |
| :--- | :---: | :---: | :---: | :--- |
| 熊 | 1.15 元 | 13-15x | 15.0-17.3 | 景气下行期的自然压缩 |
| 基 | **1.69 元** | 14-17x | **23.7-28.7** [下修EPS+PE区间持平] | 中枢对冲 OCF 风险后合理区间 |
| 牛 | 1.97 元 | 16-20x | 31.5-39.4 | NE登顶#1 + 出口加速情境 |

**基准情景 PE 区间推导** **[2026-06-08更新]**：
- 行业平均 PE 中枢 ~15-17x（考虑了 A 股估值中枢下移趋势）
- 中国重汽龙头溢价 +1-2x（市占率 #1 + NE登顶+ 出口壁垒）
- OCF 质量折价 -1x（Q1 TTM OCF 32.78亿已大幅修复，但单季Q1仍为负）
- 关联交易风险折价 -0.5x
- **综合：14-17x** [维持不变]
- 取中值 **15.5x → 目标价 26.2 元 → 市值为 306 亿**

### 2.2 PB-ROE 交叉验证

ROE 2025 = 10.58%，2026E 基准 ROE ≈ 19.7 亿 / (161.63 + 19.7/2) ≈ 11.5%

合理 PB = ROE / COE：
- 若 COE = 8%（无风险利率 2.5% + 股权风险溢价 5.5%）：PB = 11.5%/8% = **1.44x**
- 对应目标价 = 1.44 × 17.2 元（2026E BVPS）≈ **24.8 元**

PB-ROE 框架给出的目标价 24.8 元与 PE 框架中值 26.2 元接近——交叉验证一致性良好。

### 2.3 PEG 校验

2026E EPS 增速 19%（基准），现价 PE 15.02 → PEG = 15.02/19 = **0.79**

PEG < 0.8 仍属低估区间，但较 5 月 31 日的 0.60 显著上升。若 PEG 回到 1.0 且增速维持 19%，合理 PE = 19 → 对应目标价 32.1 元。**PEG 框架仍然指向低估，但低估幅度已较上月明显收窄**。

## 三、全变量风险对冲审计矩阵（强制输出）

**基准市值**：306 亿（基准目标价 26.2 元 × 11.69 亿股）[更新自317亿]

### 「估值影响力过滤」审慎思考

在进入具体调节前，必须首先执行影响力过滤：发掘出的因子绝大多数只是市场杂音，不能改变估值中枢。以下因子中，**仅少数真正具备定价权**：

| 因素 | 是否进入调节？ | 理由 |
| :--- | :---: | :--- |
| 出口连续 21 年第一 | ❌ 不进 | 已体现在 EPS 增速中——出口增长本身就是利润预测的一部分 |
| 市占率第一 | ❌ 不进 | 同上，已体现在 V（量）假设中 |
| 新能源布局全面 | ❌ 不进 | 远期 OPTIONALITY，当前对估值无实质影响 |
| OCF 质量边际改善 | ✅ 进入 | TTM OCF 已从5.63亿修复至32.78亿——但单季Q1 OCF仍为负，修复持续性待验证 |
| 关联交易 55% | ✅ 进入 | 治理风险影响折现率/风险溢价 |
| 零有息负债 | ✅ 进入 | 资产负债表质量好 → 降低尾部风险 → 小幅溢价 |

### 对冲审计矩阵 [2026-05-31更新]

| 审计维度 | 核心因子 | 审慎过滤（估值影响力） | 调节建议 | 深度理由 |
| :--- | :--- | :--- | :---: | :--- |
| **内部：企业底层特质** | 零有息负债 + 充足现金(84.84亿) | ✅ 该特质降低破产风险、为逆周期扩张储备弹药——具定价权 | **+5%** | 在强周期行业中，零有息负债意味着穿越周期的生存能力远超同行——这是一种"反脆弱性"，应享受小幅估值溢价 |
| **内部：企业底层特质** | 关联交易占比 55% vs 山东重工 | ✅ 独立定价权存疑——治理风险需要溢价补偿 | **-5%** | 大股东同时是最大客户和供应商，中小股东利益保护存在结构性风险。这种"非独立性"应对估值给予合理折扣 |
| **宏观：政策与战略对齐** | "以旧换新"补贴无缝续期至 2026 年底 | ❌ 不进 | 政策利好已在 EPS 预测的 V 假设中充分体现（行业 118 万基本盘），无需重复计算进 PE | **0%** | |
| **宏观：政策与战略对齐** | 2027 年国四补贴退出风险 | ❌ 不进 | 属于远期权衡——若 2027 年政策退出导致 EPS 下滑，届时估值自然会压缩。2026 年可暂时不扣减 | **0%** | |
| **宏观：地缘与外部约束** | 出口地缘多元化有效对冲俄罗斯下滑 | ❌ 不进 | 多元化进展已反映在出口 V 假设的增长中（+27.5%） | **0%** | |
| **市场：流动性与风格** | PE 27.5%分位 + PB ~33%分位（历史低估区） | ❌ 不进 | 当前估值本身就是结果而非原因——不能因"处于分位低位"而机械地"上修"，此为由果推因的逻辑错误 | **0%** | |
| **市场：流动性与风格** | OCF 边际改善（TTM 32.78亿 vs 2025年底5.63亿） | ✅ 进入——边际改善但持续性待验证 | **-3%** | Q1 TTM OCF已大幅修复至32.78亿（vs 2025年底仅5.63亿），但Q1单季OCF仍为-4.73亿。OCF质量折价从原来的-8%下调至-3%，反映边际改善但未根本解决（证据：应收202亿+存货75亿仍在膨胀） |
| **市场：流动性与风格** | 开源/国泰海通等多家机构给予正面评级并上调盈利预测 | ❌ 不进 | 卖方评级属市场情绪，不应作为估值锚定依据 | **0%** | |
| **综合调节系数** | —— | —— | **Σ = -3%** | [上修自-8%] |

### 修正目标市值与目标价 [2026-05-31更新]

修正目标市值 = 306 亿 × (1 + 0.05 - 0.05 - 0.03) = **306 × 0.97 = 297 亿**

对应目标价 = 297 亿 / 11.69 亿股 = **25.4 元**

**逻辑纯净性校验**：
- E（利润）中已包含：出口增长、市占率提升、补贴续期、钢价温和运行
- PE 中调整的仅三项：零有息负债溢价(+5%)、关联交易折价(-5%)、OCF 质量折价(-3%)
- 无重复计算 ✓ ，E 与 PE 分工清晰 ✓

当前股价：**23.26 元**（2026-06-08 收盘）

盈亏比（基准情景）：
- 上行空间 = (26.2 - 23.26) / 23.26 = **+12.6%**
- 下行风险 = (23.26 - 16.7) / 23.26 = **-28.2%**
- **盈亏比 = 12.6% / 28.2% = 0.45x** [vs 5/31的1.56x，大幅恶化]

盈亏比（加权）：
- 上行空间 = (24.0 - 23.26) / 23.26 = **+3.2%**
- 下行风险 = -28.2%
- **盈亏比（加权）= 0.11x** [vs 5/31的1.56x]

**核心警示** **[2026-06-08]**：股价从21.49元反弹至23.26元(+8.2%)后，盈亏比从1.56x急剧恶化至0.45x(基准)甚至0.11x(加权)。当前价位已接近加权目标价24.0元，上行空间极为有限。**5月31日研报中"风险收益结构具有吸引力"的判断已不再成立**——当前价格需要基本面出现超预期改善（如Q2 OCF转正、NE毛利改善超预期）才能支撑进一步上行。

## 五、极限压力测试——逻辑证伪与"地狱价格"

### 5.1 黑天鹅情景定义

**"三重打击"情景**（概率 ~5%）：
1. **2027 年国四补贴全面退出**：行业销量骤降 20% 至 ~88 万辆
2. **钢材价格暴涨 30%**（中东冲突全面升级 ⇒ 能源成本飙升 ⇒ 钢铁供给收缩）
3. **出口遭遇系统性贸易壁垒**（主要出口国集体加征关税 15-25%）

### 5.2 黑天鹅下的财务推演

| :--- | ---: | ---: | :--- |

| 资本市场反应 | PE 15.0x | PE 极度压缩至历史极端 10x（假设仍有微利）或 PB 定价 | — |

### 5.3 "地狱价格"测算 [2026-06-08更新]

在极端情景下，估值按 PB 清算逻辑：
- 净资产（2025 年底）= 161.63 亿
- 扣除商誉(0) + 扣除存货减值（保守折 20%：74.93 × 20% = 15 亿）+ 扣除应收账款坏账（保守折 10%：202 × 10% = 20 亿）
- 调整后净资产 = 161.63 - 15 - 20 ≈ **127 亿**
- 极限股价 = 127 / 11.69 = **10.9 元/股**（极限支撑位）

**对比当前价格 23.26 元，极限下行空间约 -53%**。这是在"公司不会破产但利润归零"的假设下的终极锚点。
