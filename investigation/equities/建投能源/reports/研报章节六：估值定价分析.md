## 一、价值锚定：适配性方法选择

### 1.1 方法论工具箱与选择理由

根据建投能源的商业模式、资产属性、现金流特征及所处生命周期，选择以下三种估值方法进行交叉验证：

| 估值方法 | 选择理由 | 适用性评级 | 权重 |
| :--- | :--- | :--- | :--- |
| **PE（市盈率）** | 火电企业盈利相对稳定（除2021年极端情况），PE是最直观的估值工具 | ★★★★★ | 40% |
| **PB-ROE（市净率-ROE模型）** | 火电是重资产行业，PB可反映资产价值；结合ROE可判断资产盈利能力 | ★★★★☆ | 30% |
| **DCF（现金流折现）** | 公司现金流质量高（经营现金流/净利润>2），且未来现金流可预测性较强 | ★★★★☆ | 30% |

**不采用的方法及理由**：
- **PS（市销率）**：电力是同质化商品，不同企业的成本结构差异远大于收入差异，PS无法反映盈利能力的分化。不适用。
- **EV/EBITDA**：虽然可用于重资产行业，但火电企业的折旧金额巨大（2025年折旧费16.30亿元），EV/EBITDA会高估企业价值。不适用。
- **SOTP（分部估值）**：公司虽有多个业务板块，但火电业务贡献94%的利润，其他板块占比极小，SOTP的必要性不足。已在模块五中使用SOTP进行利润拆分，但估值层面以整体法为主。

**近五年PE/PB走势与分位**：

| :--- | :--- | :--- | :--- | :--- | :--- | :--- |

| PB | 约1.0 | 约0.9 | 约0.9 | 约1.2 | 约1.4 | **1.76** |

**历史分位分析**：
- **PE**：当前10.34倍，处于近五年盈利期的较低水平（2022年PE约93倍、2023年约50倍、2024年约24倍）。但需注意，这是盈利暴增后的被动降低，而非估值压缩。
- **PB**：当前1.76倍，处于近五年最高位。2021-2023年PB均在1.0倍以下，2024年升至1.2倍，2025年进一步升至1.4倍，当前1.76倍已创历史新高。

**审慎思考**：当前PE看似处于历史低位，但已充分反映了2025年的业绩爆发。若2026年盈利回落（保守情景EPS仅0.61元），PE将被动抬升至20倍以上。当前PB处于历史高位，反映出股价涨幅（2024年低点4.01元→当前12.55元，涨幅213%）超过了净资产增速。**单纯看历史分位容易产生误导**，需结合盈利可持续性综合判断。

### 2.2 PB-ROE估值法

**PB-ROE模型公式**：合理PB = ROE / (要求回报率 - 永续增长率)

假设要求回报率（股权成本）10%，永续增长率2%：
- 保守ROE（保守情景归母净利润11亿÷归母净资产128.57亿）= 8.6% → 合理PB = 8.6% / (10%-2%) = **1.08倍**
- 中性ROE（中性情景归母净利润19亿÷128.57亿）= 14.8% → 合理PB = 14.8% / 8% = **1.85倍**
- 乐观ROE（乐观情景归母净利润29亿÷128.57亿）= 22.6% → 合理PB = 22.6% / 8% = **2.83倍**

| 情景 | BVPS | 合理PB | 目标价（元） | 市值（亿元） |
| :--- | :--- | :--- | :--- | :--- |
| 保守 | 6.58 | 1.08x | **7.11** | 128 |
| 中性 | 6.58 | 1.85x | **12.17** | 220 |
| 乐观 | 6.58 | 2.83x | **18.62** | 336 |

**PB-ROE与PE估值对比**：
- 保守情景：PE法4.88元 vs PB法7.11元 → 取较低者，**4.88元**。
- 中性情景：PE法10.50元 vs PB法12.17元 → 取较低者，**10.50元**。
- 乐观情景：PE法19.32元 vs PB法18.62元 → 取较低者，**18.62元**。

**基准市值确定**：综合PE和PB-ROE两种方法，取交叉验证的较低值：
- 保守基准市值：**88亿元**（对应股价4.88元）
- 中性基准市值：**189亿元**（对应股价10.50元）
- 乐观基准市值：**336亿元**（对应股价18.62元）

### 2.3 DCF估值法（交叉验证）

**核心假设**：
- 预测期：10年（2026-2035年）
- 永续增长率：2%
- WACC：9%（无风险利率3%+风险溢价6%，β取0.9， Debt/Equity约1.0，税后债务成本4.5%）
- 2026-2030年：中性情景下自由现金流约10-15亿元/年（经营现金流45-55亿-资本开支35-45亿）
- 2031-2035年：增速放缓至3%，资本开支下降

**DCF粗略测算**：
- 10年预测期自由现金流现值：约100-120亿元
- 永续价值现值：约80-100亿元
- **企业价值（EV）**：约180-220亿元
- 减：净负债（有息负债约300亿-现金32亿）= 268亿
- **股权价值**：约-48至-88亿元

**DCF估值陷阱识别**：
> 由于公司有息负债规模庞大（约300亿元），远超当前市值（226亿元），标准DCF模型测算的股权价值为负。这反映了一个重要事实：**火电行业的重资产、高负债属性使得DCF估值对WACC和资本开支假设极度敏感**。WACC每变化0.5%，股权价值可波动30-50亿元。因此，DCF在此案例中的可信度较低，仅作为参考，不作为主要定价依据。

**基准市值综合确定**：
- 以PE法为主（权重40%），PB-ROE法为辅（权重30%），DCF作为参考（权重30%但可信度低）。
- 保守基准市值：**90亿元**（取PE法88亿与PB法128亿的加权平均偏下值）
- 中性基准市值：**200亿元**（取PE法189亿与PB法220亿的加权平均）
- 乐观基准市值：**340亿元**（取PE法349亿与PB法336亿的加权平均）

## 四、逻辑纯净性校验

**严禁重复计算校验**：

| 利好/利空因子 | 是否已在利润预测（E）中体现 | 是否应在估值倍数（PE）中体现 | 结论 |
| :--- | :--- | :--- | :--- |
| 容量电价提升（+5.7亿利润） | **是**（模块五已纳入中性预测） | **否**（具体的钱已进入E，不重复进PE） | 正确区分 |
| 煤价上涨（成本压力） | **是**（模块五已按750元/吨假设） | **否**（具体的钱已进入E） | 正确区分 |
| 电价下行压力 | **是**（模块五已按435元/吨假设） | **否**（具体的钱已进入E） | 正确区分 |
| 新增装机投产 | **是**（模块五已纳入2.5%电量增量） | **否**（具体的钱已进入E） | 正确区分 |
| 公司治理改善（股权激励+高分红） | **否**（难以量化为具体利润） | **是**（在估值矩阵中+2%） | 正确区分 |
| 行业周期位置（成熟期向转型期过渡） | **否**（影响长期盈利稳定性，非单年利润） | **是**（在估值矩阵中+3%） | 正确区分 |
| 市场风格（红利策略契合度） | **否**（不影响企业内在盈利能力） | **是**（在估值矩阵中+3%） | 正确区分 |

**锚点回测**：
- 修正后中性隐含PE = 12.31元 ÷ 1.05元 = **11.7倍**
- 修正后保守隐含PE = 5.54元 ÷ 0.61元 = **9.1倍**

这一PE区间（9.1-13.0倍）与可比公司平均PE（10-12倍）基本吻合，且与火电行业"周期+公用事业"的混合属性相匹配。纯周期股PE通常在6-8倍，纯公用事业股PE在12-15倍，建投能源作为转型中的火电企业，11.7倍的中性PE具有商业逻辑的合理性支撑。

### 5.2 盈亏比计算

| 指标 | 数值 | 说明 |
| :--- | :--- | :--- |
| 当前股价 | 12.55元 | 2026年5月29日收盘价 |
| 概率加权目标价 | 12.00元 | — |

| 下行空间 | (12.55-5.54)/12.55 = **55.8%** | 保守场景下的最大跌幅 |
| **盈亏比** | **-0.08** | (目标价-现价)/(现价-保守价) = -0.55/7.01 |

**核心结论**：当前股价12.55元已略高于概率加权目标价12.00元，**盈亏比为负**，表明当前价位不具备足够的风险补偿。若股价在保守情景下（5.54元）买入，盈亏比将显著改善至约0.9（(12.00-5.54)/(5.54-保守情景价，假设保守情景价为底部)），具备较高的安全边际。

## 六、逻辑证伪与极限压力测试

### 6.1 黑天鹅场景模拟：足以推翻当前看好逻辑的"一击毙命"事件

**场景一：煤价失控暴涨**
- **触发条件**：地缘政治冲突升级（如中东战争全面爆发）+ 印尼完全禁止煤炭出口 + 国内安全检查导致大规模停产。
- **煤价情景**：秦皇岛5500K动力煤价格突破1,200元/吨，较2025年均价上涨70%以上。
- **对公司影响**：标煤单价若涨至1,000元/吨以上，燃料成本将增加约90亿元，公司可能重新陷入亏损（类似2021年情景）。
- **概率评估**：低概率（<5%），但后果极其严重。

**场景二：电力市场化改革超预期**
- **触发条件**：现货市场价格完全放开，火电电价在竞争中暴跌至300元/兆瓦时以下，且容量电价机制被取消。
- **对公司影响**：收入端崩溃，即使有煤价下降也无法弥补。
- **概率评估**：极低概率（<3%）。容量电价机制已写入国家政策，短期不会取消。

**场景三：新能源技术革命**
- **触发条件**：固态电池、钙钛矿光伏或核聚变技术取得突破性进展，储能成本骤降90%，火电完全失去存在价值。
- **对公司影响**：资产大规模减值，牌照价值归零。
- **概率评估**：极低概率（<1%），但长期（2030年后）不可忽视。

### 6.2 极限支撑位审计："地狱价格"测算

在最极端的负面情景下（煤价暴涨+电价暴跌+利用小时骤降），公司的"地狱价格"应基于以下底线价值：

**底线价值一：重资产清算价值**
- 2025年末固定资产270.87亿元，在建工程24.06亿元，长期股权投资63.17亿元。
- 假设固定资产清算价值为账面价值的50%（火电设备专用性强，二手市场有限）：270.87×50% = 135.4亿元
- 加：在建工程24.06亿+长期股权投资63.17亿+货币资金32.31亿+存货7.83亿 = 127.4亿元
- 减：负债301.33亿元
- **清算价值 ≈ -38.5亿元**
- **结论**：清算价值为负，重资产清算不能作为有效支撑。

**底线价值二：净现金价值**
- 2025年末货币资金32.31亿元，交易性金融资产10.13亿元。
- 减：短期借款44.73亿元。
- **净现金 ≈ -2.29亿元**
- **结论**：净现金为负，无法提供有效支撑。

**底线价值三：容量电价年金价值**
- 假设公司控股火电装机维持1,177万千瓦不变，容量电价维持165元/千瓦。
- 年容量电费收入（含税）：19.42亿元，税后约14.6亿元。
- 假设容量电价永续存在，折现率10%，永续增长率2%。
- 年金价值 = 14.6亿 / (10%-2%) = **182.5亿元**
- 对应每股价值 = 182.5亿 ÷ 18.03亿股 = **10.12元**

**极限支撑位综合判断**：
> 在极端负面情景下，公司的有效底线价值来源于**容量电价年金的折现价值**，约为 **10.12元/股**。这一价格远低于当前股价12.55元，意味着若发生"一击毙命"事件，股价有下跌至10元以下的风险。但10元附近构成了基于政策红利的强支撑——只要容量电价机制不被取消，火电企业的估值就不会归零。
