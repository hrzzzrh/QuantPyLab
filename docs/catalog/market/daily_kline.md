# 日线行情表 (daily_kline)

存储股票的每日原始价格与成交数据，以 Parquet 格式存储在数据湖中。

## 1. 字段定义

| 序号 | 字段名 | 类型 | 说明 | 样例值 |
| :--- | :--- | :--- | :--- | :--- |
| 1 | `symbol` | VARCHAR | 股票代码 (纯数字，分区键) | 600519 |
| 2 | `date` | DATE | 交易日期 | 2024-01-02 |
| 3 | `open` | DOUBLE | **不复权**开盘价 | 1715.00 |
| 4 | `high` | DOUBLE | **不复权**最高价 | 1718.19 |
| 5 | `low` | DOUBLE | **不复权**最低价 | 1678.10 |
| 6 | `close` | DOUBLE | **不复权**收盘价 | 1685.01 |
| 7 | `volume` | DOUBLE | 成交量 (手) | 32156 |
| 8 | `amount` | DOUBLE | 成交额 (元) | 5440082944.0 |
| 9 | `adj_factor` | DOUBLE | **后复权因子** (用于计算后复权价) | 5.577083 |

## 2. 数据来源
- 接口: `ak.stock_zh_a_hist`
- 逻辑: 每天增量同步，同时获取 Raw 和 HFQ 价格以计算复权因子。
