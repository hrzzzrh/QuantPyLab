# QuantPyLab 数据资产目录 (Data Catalog)

本项目的数据资产说明书已按模块拆分，以提供更清晰的字段定义和维护体验。请点击下方链接查看各模块详情：

## 1. 基础数据
- **[基础行情与个股信息](catalog/market_data.md)**: 包含股票代码、名称、行业、地域等基础索引。

## 2. 财务数据
- **[财务报表 (三大会计报表)](catalog/financial_statements.md)**: 包含资产负债表、利润表、现金流量表的原始数据字段。
- **[财务指标 (高阶计算因子)](catalog/financial_indicators.md)**: 包含由东财提供的 140+ 项专业财务指标。
- **[滚动财务 (TTM 估值分母)](catalog/valuation_ttm.md)**: 包含自主计算的滚动十二个月 (TTM) 数据，专门用于估值还原。

## 3. 计算规范
- 所有金额单位如无特殊说明，均为 **元 (CNY)**。
- 比例指标单位通常为 **%**。
- 日期格式统一为 **YYYYMMDD**。
